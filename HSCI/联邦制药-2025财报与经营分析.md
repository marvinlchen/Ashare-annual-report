# 联邦制药（03933.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

联邦制药2025年实现营业收入RMB 132.11亿元（-4.0%），股东应占溢利RMB 20.86亿元（-21.6%）。PBT RMB 25.28亿（-23.1%）。毛利率从44.2%降至41.7%（-2.5ppt）。EPS RMB 1.1065（-24.4%）。DPS RMB 0.42（含中期0.16+末期0.26），派息比率38.0%。

### 1.2 增长与拖累因素

**最大亮点**：与诺和诺德（Novo Nordisk）达成UBT251（GLP-1R/GIPR/GCGR三靶点激动剂）独家许可协议，收取RMB 14.42亿预付款，推动制剂分部溢利暴增455.3%。这笔交易标志着联邦制药创新研发能力获得国际一流药企认可。

**主要拖累**：中间体产品收入-31.4%、分部溢利-79.4%；原料药收入-19.8%、分部溢利-53.4%——均受产品市场价格大幅下跌拖累。销售成本+0.3%几乎刚性（收入-4%但成本不降），压缩毛利率-2.5ppt。

**本质故事**：如果剔除诺和诺德RMB 14.42亿许可费，调整后收入仅RMB 117.69亿（实际下降-14.4%），说明核心中间体+原料药业务面临显著价格压力。许可费的一次性巨额贡献掩盖了传统业务的困境。

### 1.3 趋势判断

联邦制药正处于**从原料药企业向创新制剂企业转型的关键期**——诺和诺德合作验证了GLP-1/胰岛素研发能力，PP&E +32.5%反映大规模产能扩建（胰岛素/GLP-1），现金RMB 106亿创历史新高为转型提供充裕资金。但中间体/原料药的价格下行周期尚未见底，短期内传统业务利润仍将承压。

---

## 2. 主营业务

### 2.1 公司简介

联邦制药是中国领先的综合性制药企业，业务覆盖抗生素中间体、原料药和制剂产品的全产业链，近年来积极布局胰岛素和GLP-1类创新药。

### 2.2 主营业务构成

1. **制剂产品**：2025年收入+41.7%，分部溢利+455.3%，受诺和诺德许可费RMB 14.42亿推动
2. **原料药**：抗生素原料药（阿莫西林、6-APA等），2025年收入-19.8%，分部溢利-53.4%
3. **中间体产品**：抗生素中间体（7-ACA等），2025年收入-31.4%，分部溢利-79.4%
4. **创新药研发**：UBT251（三靶点GLP-1激动剂，已授权诺和诺德）、胰岛素类似物等

### 2.3 核心收入来源

传统业务（中间体+原料药）仍是收入基本盘，但利润贡献快速向制剂转移。诺和诺德许可费使制剂成为2025年利润贡献最大的分部。长期看，胰岛素和GLP-1类产品将成为核心增长引擎。

### 4.2 分部溢利

### 4.3 营收分析

**主要增长来源**

制剂产品是唯一增长的分部（+41.7%），主要受诺和诺德UBT251许可费RMB 14.42亿推动。剔除许可费后制剂基础业务仍在增长（胰岛素和口服制剂放量）。

**主要拖累因素**

中间体（-31.4%）和原料药（-19.8%）受全球抗生素中间体/原料药价格下行周期影响，7-ACA、6-APA等产品价格持续走低。成本端刚性（销售成本+0.3%）导致毛利率-2.5ppt。

**结构变化**

利润重心正加速从中间体/原料药向制剂转移。制剂分部溢利从极小占比跃升至主要利润贡献者（+455%）——这是联邦制药战略转型的关键里程碑。

## 6. 未来可能的利好

### 利好 1：诺和诺德合作——GLP-1赛道国际认可

**依据**

UBT251（GLP-1R/GIPR/GCGR三靶点激动剂）独家许可给诺和诺德，已收取RMB 14.42亿预付款。后续还有里程碑付款和销售分成。

**解读**

诺和诺德是全球GLP-1领域绝对龙头（司美格鲁肽年销超$200亿）。获得诺和诺德认可并签署独家许可，是对联邦制药创新药研发能力的最高级别背书。后续里程碑付款和潜在销售分成将持续贡献收入。

### 利好 2：胰岛素/GLP-1产能大扩张——PP&E +32.5%

**依据**

PP&E从RMB 87.25亿增至RMB 115.58亿（+RMB 28.33亿），承担RMB 21.47亿。推测主要用于胰岛素类似物和GLP-1类产品的新产线。

**解读**

产能扩建为未来收入增长提供硬件基础。胰岛素/GLP-1是全球增长最快的糖尿病/减肥药赛道，联邦制药拥有从原料药到制剂的全产业链优势，产能投产后可释放巨大收入弹性。

### 利好 3：现金RMB 106亿创新高——转型资金充裕

**依据**

现金+67.6%至RMB 106.06亿，净现金RMB 56.15亿。应收帐款-26.7%，回款改善。

**解读**

充裕的现金为产能扩建、研发投入和潜在并购提供了强大的财务支撑，同时维持38%的派息比率。

## 7. 未来可能的利空

### 利空 1：中间体/原料药价格周期下行未见底

**依据**

中间体收入-31.4%、溢利-79.4%；原料药收入-19.8%、溢利-53.4%。7-ACA、6-APA等全球抗生素原料药价格持续走低。

**解读**

中间体和原料药仍是联邦制药的收入基本盘。如果价格周期继续下行，传统业务利润可能进一步萎缩。制剂转型尚需时间，中间过渡期的利润压力不容忽视。

### 利空 2：诺和诺德许可费的一次性效应——2026年盈利基数压力

**依据**

RMB 14.42亿预付款是一次性收入。2026年如无类似规模交易，制剂分部溢利将大幅回落（-455%的反面）。

**解读**

市场可能已对2025年利润中的许可费有预期，但2026年如果回落，仍可能引发盈利预期下调。关键看后续里程碑付款时间表。

### 利空 3：重资产扩张的折旧与回报风险

**依据**

PP&E +32.5%至RMB 115.58亿。D&A已+5.5%至RMB 6.50亿，新产能投产后折旧将继续增加。

**解读**

大规模资本开支在产能利用率爬坡期会拉低ROE。如果胰岛素/GLP-1产品放量速度不及预期，固定成本压力将加大。

### 利空 4：成本刚性——收入-4%但成本+0.3%

**依据**

销售成本RMB 76.97亿（+0.3%），收入-4%，毛利率-2.5ppt。

**解读**

制药企业（尤其是中间体/原料药）存在较强的成本刚性（产能固定成本、能源、人工等），价格下跌时利润弹性向下。

## 8. 总结判断

**一句话结论**

联邦制药2025年收入-4%、净利-21.6%至RMB 20.86亿，表面业绩下滑但暗含转型突破——诺和诺德UBT251许可费RMB 14.42亿使制剂分部溢利+455%，掩盖了中间体(-79.4%)和原料药(-53.4%)利润的大幅下滑；PP&E +32.5%和现金RMB 106亿创新高体现向胰岛素/GLP-1创新药的重资产转型加速。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| PBT | RMB 25.28亿（-23.1%） | 表面下滑 |
| 毛利率 | 41.7%（-2.5ppt） | 价格下跌+成本刚性 |
| PP&E | RMB 115.58亿（+32.5%） | ⭐大规模扩产 |
| 现金 | RMB 106.06亿（+67.6%） | ⭐含诺和诺德预付款 |
| 净现金 | RMB 56.15亿 | 财务稳健 |
| P/B | HK$10.31 / NBV RMB 8.815≈HK$9.70，约1.06x | |
| P/E | HK$10.31 / EPS RMB 1.107≈HK$1.22，约8.5x | |

**核心看点**

1. **中间体/原料药价格何时触底反弹**：决定传统业务利润是否见底
2. **诺和诺德后续里程碑付款时间表**：弥补预付款一次性效应
3. **胰岛素/GLP-1新产能投产和放量**：PP&E扩张的回报兑现
4. **创新药管线进展**：UBT251之外是否有新的授权/合作机会

**核心风险**

1. **中间体/原料药价格持续下跌**：传统业务利润可能进一步萎缩
2. **许可费一次性效应**：2026年盈利基数压力
3. **重资产扩张的折旧和回报风险**：产能利用率爬坡期拉低ROE
4. **GLP-1竞争加剧**：全球多家药企布局，市场竞争日趋激烈
