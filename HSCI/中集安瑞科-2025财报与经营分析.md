# 中集安瑞科（03899.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

中集安瑞科2025年实现总收入RMB 263.26亿元，同比 +6.3%。股东应占溢利RMB 11.35亿元，同比 +3.7%。但PBT RMB 13.93亿实际同比 -3.5%——净利增长完全来自税率大幅下降（有效税率20.8%→15.9%，-4.9ppt）。每股盈利RMB 0.561元（+3.5%）。末期股息HK$0.31，全年股息RMB 0.27，派息率48.1%。经营现金流RMB 71.34亿（+0.5%）。

### 1.2 增长与拖累因素

增长方面，清洁能源分部新签订单RMB 222.29亿创历史新高，水上清洁能源收入RMB 64亿（+38%），LNG车用瓶收入RMB 12亿。在手订单合计RMB 297.48亿提供收入可见性。拖累方面，销售成本+6.8%快于收入+6.3%导致毛利率下降（14.0%→13.6%？待确认），其他公平值/减值从+RMB 0.17亿转为-RMB 0.27亿（-RMB 0.44亿），PBT实际下降3.5%。

### 1.3 趋势判断

中集安瑞科受益于全球清洁能源转型（LNG、氢能、碳中和装备需求），清洁能源新签订单创历史新高，在手订单RMB 297亿提供未来1-2年收入保障。整体判断：**收入端增长稳健（+6.3%，在手订单近300亿），但毛利率持续收窄和PBT实际下降（-3.5%）是隐忧，净利微增（+3.7%）靠税率下降支撑，需关注盈利质量**。

## 2. 主营业务

### 2.1 公司简介

中集安瑞科是中集集团旗下专注于能源、化工及液态食品装备制造和工程服务的上市平台，产品涵盖LNG储运装备、化工装备、氢能装备、液态食品设备等。

### 2.2 主营业务构成

1. **清洁能源**：LNG储运设备、加注站、水上清洁能源（LNG/氨燃料罐）、LNG车用瓶等；新签订单占比最大
2. **化工环保**：化工压力容器、储罐、反应器等；在手订单RMB 1,276亿（+36.3%）
3. **液态食品**：不锈钢罐、发酵罐等食品饮料生产设备
4. **氢能**：加氢站设备、高压储氢瓶等

### 2.3 核心收入来源

清洁能源是最大的收入和订单来源（新签RMB 222亿，约84%），其中水上清洁能源（LNG/氨燃料罐用于船舶）增长最快（+38%）。化工环保在手订单RMB 1,276亿，为未来确认收入的主要储备。

## 6. 未来可能的利好

### 利好 1：在手订单RMB 297亿创历史新高——收入可见性强

**依据**

累计在手订单RMB 297.48亿（约1.1倍年收入），清洁能源新签RMB 222亿创历史新高。化工环保在手订单RMB 1,276亿（+36.3%）。

**解读**

近300亿的在手订单为未来1-2年的收入增长提供了较高可见性，即使新增订单放缓，存量订单也足以支撑收入稳定增长。

### 利好 2：水上清洁能源+38%——航运脱碳的核心受益者

**依据**

水上清洁能源收入RMB 64亿（+38%）创历史新高，受益于IMO航运脱碳法规（2030年碳强度-40%目标）推动的LNG/氨燃料罐需求。

**解读**

全球航运业正面临碳排放法规的结构性收紧，LNG和氨燃料动力船舶是主要过渡方案。中集安瑞科作为全球领先的LNG储运装备制造商，拥有技术壁垒和规模优势，将持续受益于这一长期趋势。

### 利好 3：氢能业务布局——长期增长期权

**依据**

氢能新签订单RMB 7.45亿，产品涵盖加氢站设备、高压储氢瓶等。

**解读**

氢能处于产业化早期，当前收入贡献较小但长期空间巨大。中集安瑞科在气体储运装备领域的技术积累可顺畅延伸至氢能储运，属于低成本布局未来的战略举措。

## 7. 未来可能的利空

### 利空 1：毛利率持续收窄——成本增速快于收入

**依据**

毛利率从14.36%降至13.95%（-0.4ppt），销售成本+6.8%快于收入+6.3%。PBT在收入+6.3%的情况下反而下降3.5%。

**解读**

制造业的毛利率收窄通常来自原材料涨价、竞争加剧或产品组合变化。如果趋势持续，即使收入增长也可能无法带来利润改善，特别是在订单快速增长导致产能紧张、外包比例上升的情况下。

### 利空 2：利润增长靠税率——有效税率的可持续性

**依据**

有效税率从20.8%大幅降至15.9%（-4.9ppt），所得税-26.3%是净利增长的唯一桥梁。PBT实际下降3.5%。

**解读**

税率下降可能来自地方政府税收优惠或收入结构变化（海外低税率地区占比提升）。如果优惠到期或结构回归，净利润增速将面临回落压力。

### 利空 3：或然负债RMB 49.45亿——担保风险

**依据**

2024年末或然负债RMB 49.45亿（较2023年的39.37亿+25.6%），主要为对外担保。

**解读**

或然负债接近净利润的4.5倍，如果被担保方出现违约，可能产生实质性损失。需持续关注此项风险。

## 8. 总结判断

**一句话结论**

中集安瑞科2025年收入+6.3%（在手订单近300亿创历史新高），但PBT实际下降3.5%（毛利率收窄+费用增长），净利+3.7%完全依赖有效税率大幅下降（20.8%→15.9%，-4.9ppt），盈利质量需审视——水上清洁能源+38%和巨额在手订单提供收入可见性，但利润率承压趋势是中期隐忧。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| PBT | RMB 13.93亿（-3.5%） | ⚠️实际下降 |
| 有效税率 | 15.9%（-4.9ppt） | ✅净利增长的唯一桥梁 |
| 毛利率 | 13.95%（-0.4ppt） | ⚠️持续收窄 |
| 现金 | RMB 77.52亿（+6.7%） | ✅ |
| 净现金 | RMB 39.84亿 | ✅ |
| 应收帐款 | RMB 35.52亿（-1.0%） | ✅控制良好 |
| NBV | RMB 5.982（+5.5%） | 当前HK$10.77≈RMB 9.68，P/B≈1.62x |

**核心看点**

1. **毛利率能否止跌回升**：13.95%是否触底，还是将继续下滑
2. **有效税率的可持续性**：15.9%能否维持，还是会回升至20%+
3. **在手订单转化为收入的节奏**：RMB 297亿订单如何在2026-2027年释放
4. **水上清洁能源能否持续高增长**：IMO法规执行力度和船东改造意愿

**核心风险**

1. **PBT实际下降而净利靠税率支撑**：盈利增长质量存疑
2. **毛利率持续收窄**：成本增速>收入增速的趋势若延续将压缩利润
3. **或然负债RMB 49.45亿**：担保风险不可忽视
4. **制造业周期性**：全球经济放缓可能影响能源和化工装备需求
