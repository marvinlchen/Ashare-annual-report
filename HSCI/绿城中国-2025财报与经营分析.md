# 绿城中国（03900.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

绿城中国2025年实现营业收入RMB 1,549.66亿元，同比 -2.3%。股东应占溢利仅RMB 0.71亿元，同比暴跌95.6%（2024年：RMB 15.96亿），几乎归零。PBT RMB 48.89亿（-30.9%），仍有盈利但税（RMB 26.03亿）和NCI（RMB 22.15亿）合计吃掉PBT的98.5%，留给股东仅1.5%。EPS RMB 0.03。**上市20年来首次零派息。**

### 1.2 增长与拖累因素

拖累方面，三大因素叠加导致利润崩塌：（1）净财务支出从RMB 2.95亿暴增至RMB 12.53亿（+325%），利息费用化比例大幅增加；（2）联营/合营公司亏损从RMB -6.33亿扩大至-11.34亿（+79%），合营项目盈利恶化；（3）其他公平值/减值RMB -49.36亿（虽同比微降2.5%但绝对值仍巨大）。毛利率从12.8%降至11.9%（-0.9ppt），主动推动长库存去化压低结转利润。正面方面，合同销售额RMB 2,519亿排名全国第二，现金RMB 632亿创历年新高（但etnet显示591.60亿，-14.1%），加权平均融资成本3.3%（-60bps）。

### 1.3 趋势判断

绿城中国虽然合同销售全国第二、品牌溢价能力强，但利润结构极度脆弱——PBT近RMB 49亿看似尚可，但税和NCI合计吃掉98.5%，留给股东的空间极薄。整体判断：**"高收入、高销售但极低股东利润"的典型困境——净财务支出暴增+联营亏损扩大+NCI占比过高是核心矛盾，短期内盈利修复困难，首次零派息反映管理层对现金流的保守态度**。

## 2. 主营业务

### 2.1 公司简介

绿城中国是中交集团旗下的高端房地产开发企业，以品质住宅开发著称，业务涵盖物业开发、物业投资、项目管理服务和生活服务，布局集中于长三角和一二线核心城市。

### 2.2 主营业务构成

1. **物业开发及销售（自投项目）**：核心业务，2025年自投销售RMB 1,534亿（占总合同销售61%），面积466万平方米
2. **代建/项目管理**：轻资产第二曲线，2025年代建销售RMB 985亿（占39%），面积742万平方米；绿城管理（09979.HK）收入RMB 31.2亿，毛利RMB 12.39亿，毛利率~40%，归母净利润RMB 4.19亿
3. **物业投资**：持有并运营投资性物业，投资物业RMB 117.28亿
4. **生活服务及其他**：物业管理、生活配套等

### 2.3 核心经营模式

绿城采用**"小股操盘"**模式——在合作项目中仅持有少量股份但具有决策权（管理控制权），项目利润合并入报表后按股权比例分配给合作方。这使得绿城以较低风险扩大规模（合同销售全国第二），但NCI长期分走绝大部分利润。截至2025年末，股东资金仅占资产净值总额的33.3%（RMB 351.84亿/1,057.14亿），远低于同类浙系房企滨江集团的54.5%。

### 2.4 核心收入来源

物业开发及销售是绝对主导收入来源。合同销售区域集中于长三角（杭州/宁波/上海为重镇）和一二线核心城市，销售均价RMB 32,924元/平方米，远高于行业平均，体现品牌溢价。2025年新增50个项目，预计新增货值RMB 1,355亿，一二线货值占比86%。

⚠️**或然负债**：2024年末或然负债RMB 242.95亿，主要为对合营企业的担保。

## 6. 未来可能的利好

### 利好 1：合同销售全国第二——品牌竞争力依然强劲

**依据**

2025年总合同销售额RMB 2,519亿排名全国第二，销售均价RMB 32,924元/平方米远高于行业。自投项目销售RMB 1,534亿，权益销售RMB 1,043亿排名全国第五。

**解读**

在房地产深度调整期仍能保持全国第二的销售规模，体现绿城在一二线城市的品牌溢价能力和产品竞争力。高销售额为未来收入确认提供保障。

### 利好 2：融资成本下降——3.3%且继续降息空间

**依据**

加权平均融资成本3.3%（-60bps），短债占比降至18.5%。受益于央企背景和宽松货币政策。

**解读**

对于有息负债RMB 1,338亿的重资产企业，融资成本每降低10bps约节省RMB 1.3亿利润。低利率环境延续将持续减轻利息负担。

### 利好 3：存货去化加速——资产瘦身降低减值风险

**依据**

存货从RMB 2,520亿降至RMB 2,148亿（-14.8%），短期借贷-21.9%。

**解读**

主动去库存虽然压低当期毛利率，但减少了未来减值风险，同时回笼现金改善流动性。

### 利好 4：代建第二曲线——轻资产高毛利

**依据**

代建业务毛利率~40%，远高于物业开发。绿城管理（09979.HK）2025年收入RMB 31.2亿，归母净利RMB 4.19亿。代建销售额RMB 985亿占总合同销售的39%。

**解读**

代建是绿城的轻资产转型方向，不占用资金、不承担库存风险，且享受绿城品牌溢价。如果代建占比持续提升，将改善整体利润质量和ROE。

### 利好 5：土储聚焦一二线——新增货值86%在高能级城市

**依据**

2025年新增50个项目，预计新增货值RMB 1,355亿，一二线货值占比86%。

**解读**

高能级城市需求韧性更强、减值风险更低，聚焦一二线的土储策略有利于未来毛利率企稳。

## 7. 未来可能的利空

### 利空 1：股东利润结构极度脆弱——PBT的98.5%被税和NCI吃掉

**依据**

PBT RMB 48.89亿，税占53.2%+NCI占45.3%=98.5%，股东仅分到1.5%（RMB 0.71亿）。NCI权益RMB 705亿是股东资金RMB 352亿的2倍。

**解读**

这是绿城中国最核心的结构性问题——大量合作开发项目导致NCI长期分走巨大利润份额。即使PBT恢复增长，股东能分到的比例也极为有限。除非减少合作项目比例或提高自投项目占比，否则股东回报将长期被稀释。

### 利空 2：净财务支出暴增+325%——利息费用化加速

**依据**

净财务支出从RMB 2.95亿暴增至RMB 12.53亿（+325%），可能因为存货结转加速导致原资本化的利息进入损益表。

**解读**

随着更多项目结转，资本化利息将持续释放为费用，对PBT形成持续压力。有息负债RMB 1,338亿即使按3.3%的低利率，年利息约RMB 44亿，其中越来越多将进入损益表。

### 利空 3：联营/合营亏损扩大——合作项目盈利恶化

**依据**

联营/合营公司亏损从RMB -6.33亿扩大至-11.34亿（+79.2%）。联营公司权益RMB 368.71亿（占总资产8.2%）。

**解读**

联营/合营亏损反映合作开发项目（通常位于二三线城市或利润较薄的项目）盈利能力恶化，可能需要进一步减值。

### 利空 4：代建质量投诉与举报事件——品牌溢价面临挑战

**依据**

2025年绿城代建的上海多个项目遭遇业主集体质量投诉（开云锦悦华庭车库漏水、摩登江南地下车库装修质量差）。此外，业绩发布当天港交所收到天鸿房地产的举报信，指控绿城高管在唐山"绿城·桂语江南"项目中存在不当利益输送，将上市公司业务转移至关联公司。绿城回应称已进入司法程序。

**解读**

代建是绿城品牌变现的核心第二曲线，质量口碑是代建模式的基石。频发质量投诉将损害品牌声誉和未来代建订单。举报事件如被查实则涉及公司治理风险。

### 利空 5：有效税率异常偏高——53.2%

**依据**

2025年有效税率53.2%（PBT RMB 48.89亿，税RMB 26.03亿），远高于正常水平（2024年41.4%）。

**解读**

房地产企业面临增值税、土增税、企业所得税等多重税负，且部分联营亏损可能无法抵税。高有效税率叠加高NCI占比，使股东利润空间被双重压缩。

## 8. 总结判断

**一句话结论**

绿城中国2025年收入-2.3%、PBT -30.9%至RMB 49亿，但税（53.2%）+NCI（45.3%）合计拿走PBT的98.5%，股东应占溢利仅RMB 0.71亿（-95.6%），上市20年首次零派息——净财务支出+325%（利息费用化）、联营亏损+79%和巨额减值RMB 49亿是PBT下降主因，NCI权益=2倍股东资金的结构性问题导致股东利润极度脆弱。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| PBT | RMB 48.89亿（-30.9%） | 仍有盈利但大幅下降 |
| 净财务支出 | RMB 12.53亿（+325%） | ⚠️⚠️⚠️最大拖累 |
| 联营亏损 | RMB -11.34亿（+79%） | ⚠️恶化 |
| 减值 | RMB -49.36亿 | 仍巨大 |
| 税+NCI占PBT | 98.5% | ⚠️⚠️⚠️股东仅得1.5% |
| 现金 | RMB 591.60亿（-14.1%） | 仍充裕 |
| 存货 | RMB 2,148亿（-14.8%） | ✅去化中 |
| NBV | RMB 13.85（-3.4%） | 当前HK$9.20≈RMB 8.27，P/B≈0.60x |

**核心看点**

1. **NCI占比能否下降**：是否减少合作项目、提高自投比例以改善股东利润分配
2. **净财务支出是否见顶**：利息费用化节奏何时放缓
3. **联营亏损能否收窄**：合作项目盈利是否触底
4. **毛利率能否企稳**：11.9%是否为阶段性底部

**核心风险**

1. **股东利润结构极度脆弱**：税+NCI合计吃掉PBT的98.5%，任何PBT波动都可能导致亏损
2. **净财务支出暴增趋势**：+325%，利息费用化将持续侵蚀PBT
3. **NCI=2倍股东资金（"小股操盘"代价）**：股东资金仅占总权益33.3%，合作开发结构性稀释股东回报
4. **减值RMB 49亿仍巨大**：房地产市场未见底则减值将持续
5. **首次零派息**：股息投资者流失风险
6. **代建质量投诉+唐山举报事件**：品牌溢价和公司治理面临挑战
