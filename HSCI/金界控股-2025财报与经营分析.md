# 金界控股（03918.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

金界控股（NagaCorp）2025年实现营业收入$7.097亿（+26.2%），股东应占溢利$3.099亿（+183%），近乎三倍增长。EBITDA $4.044亿，EBITDA利润率57.0%。毛利率81.3%（极高，典型赌场业务特征）。PBT $3.106亿（+182%）。EPS $0.0701（+183%）。恢复派息DPS $0.021（派息比率30%），上年和2023年均未派息。

### 1.2 增长与拖累因素

增长驱动：（1）博彩总收入（GGR）$6.916亿（+27.4%），大众市场赌桌+27.2%至$3.424亿、高端VIP +32.1%至$1.362亿、转介VIP +57.2%至$7,040万；（2）2024年有$8,911万减值/公允值损失（投资物业-$4,885万+其他-$4,026万），2025年为零——仅此一项贡献约$8,900万利润改善；（3）"侧注"游戏贡献额外约$3,900万收入；（4）高端大众（premium mass）区域占比由33.9%提升至38.5%。现金大增209%至$3.721亿。拖累方面，净利息收入从$607万降至$288万（现金虽增但利率环境变化），销售成本+25.4%与收入增速基本同步。

### 1.3 趋势判断

金界控股2025年业绩爆发式增长，但需区分**一次性因素（2024年减值$8,900万消失）和结构性改善（GGR +27.4%、大众市场客量+12.6%、中国赴柬游客+51%）**。整体判断：**"柬埔寨博彩垄断龙头复苏势头强劲，大众市场和VIP双线增长，现金充裕+超低杠杆——但Naga 3项目融资协议终止带来重大不确定性，公司治理（陈家控股）仍是长期隐忧"**。

## 2. 主营业务

### 2.1 公司简介

金界控股（NagaCorp）是柬埔寨唯一持有合法赌场牌照的博彩娱乐运营商，在金边运营NagaWorld综合度假村，享有柬埔寨首都200公里范围内博彩独家经营权至2035年、赌税豁免至2035年。

### 2.2 主营业务构成

1. **大众市场博彩（Mass Market）**：赌桌+电子博彩机，2025年GGR $4.850亿（占总GGR约70%），毛利率~89%
2. **高端VIP博彩（Premium VIP）**：高端贵宾厅，2025年GGR $1.362亿（+32.1%）
3. **转介VIP博彩（Referral VIP）**：第三方中介VIP，2025年GGR $7,040万（+57.2%）
4. **非博彩业务**：酒店、餐饮、娱乐等，收入约$1,800万

### 2.3 核心收入来源

博彩收入占总收入97%+，其中大众市场是绝对主力（~70%）。NagaWorld受益于柬埔寨旅游复苏（2025上半年国际旅客340万人次+6%，中国游客+51%），且享有独家经营权的垄断地位。

### 4.3 营收分析

**主要增长来源**

## 6. 未来可能的利好

### 利好 1：柬埔寨旅游持续复苏——中国游客+51%

**依据**

2025年上半年经金边机场抵达的国际游客恢复至疫情前98%水平，大中华地区航班大幅增加46%，中国赴柬游客+51%。柬埔寨FDI +77%至$60亿。

**解读**

中国是NagaWorld最重要的客源市场。中国经济复苏+航班增加将持续推动客量增长。大众市场客量+12.6%尚未完全恢复到疫情前水平，仍有增长空间。

### 利好 2：垄断牌照+零税率——独特竞争壁垒

**依据**

NagaWorld享有金边200公里范围内博彩独家经营权至2035年、赌税豁免至2035年。有效税率仅0.2%。

**解读**

这是全球极为罕见的博彩垄断+零税率组合。在其他博彩市场（澳门税率39%、新加坡15-22%、菲律宾5%+）面临税率上升压力时，金界的零税率优势更加突出。

### 利好 3：现金大增+恢复派息——财务状况大幅改善

**依据**

现金从$1.204亿增至$3.721亿（+209%），由净负债转为净现金$1.882亿。恢复派息（DPS $0.021，30%派息比率）。

**解读**

财务状况的显著改善为未来发展（包括Naga 3）和股东回报提供了坚实基础。

### 利好 4：高端大众占比提升——结构性升级

**依据**

Premium mass区域占比从33.9%提升至38.5%。大众市场毛利率~89%远高于VIP。

**解读**

高端大众客群的客单价高、波动性低、利润率优于VIP，占比提升是收入质量的结构性改善。

## 7. 未来可能的利空

### 利空 1：Naga 3融资协议终止——最大不确定性

**依据**

2025年12月，金界控股终止了与控股股东陈家的Naga 3项目融资协议（原计划陈家提供50%资金）。Naga 3是总投资约$35-40亿的超大型综合度假村项目。2024年末项目承担已达$31.41亿。

**解读**

这是金界控股面临的最大不确定性。融资协议终止意味着：（1）Naga 3的资金来源需要重新安排；（2）可能需要更多外部融资（增发/借贷），稀释股东或增加负债；（3）项目进度可能延迟。考虑到总投资远超公司总资产（$27.69亿），融资方案将决定公司未来的资本结构和股东回报。

### 利空 2：2024年减值消失的一次性效应——2026年增速将回归正常

**依据**

2025年净利润+183%中约44%来自2024年$8,911万减值的消失。2026年将没有这一正面基数效应。

**解读**

投资者应预期2026年利润增速将大幅放缓至正常水平。如果2026年出现新的减值（如Naga 3相关），利润可能反向受冲击。

### 利空 3：公司治理风险——陈家控股+单一资产

**依据**

控股股东陈志勤家族对公司有绝对控制权。业务100%依赖柬埔寨金边的NagaWorld单一物业。曾发生劳资纠纷（2022年大规模员工抗议事件）。

**解读**

单一资产+控股家族治理是典型的公司治理风险。任何柬埔寨政策变化、牌照续期问题、自然灾害或地缘政治事件都可能对公司产生致命打击。陈家融资协议的终止也引发对其与上市公司关系的疑虑。

### 利空 4：VIP业务的中国监管风险

**依据**

中国对跨境赌博的打击力度持续加大。VIP业务（高端+转介合计占GGR约30%）的客源高度依赖中国大陆。

**解读**

中国持续加码反跨境赌博执法，可能影响VIP客量和转介中介。虽然大众市场（占70%）受影响较小，但VIP是高贡献客群。

## 8. 总结判断

**一句话结论**

金界控股2025年收入+26.2%至$7.10亿、净利+183%至$3.10亿，业绩爆发式增长——但需区分一次性因素（2024减值$8,900万消失贡献+183%中的44%）和结构性改善（GGR +27%、大众市场升级、中国游客+51%）；垄断牌照+零税率+净现金是核心优势，但Naga 3融资协议终止是最大悬念。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| PBT | $3.106亿（+182%） | 含一次性减值消失$8,900万 |
| 可比PBT增速 | +56% | 剔除2024减值影响 |
| 毛利率 | 81.3% | 极高且稳定 |
| 有效税率 | 0.2% | 赌税豁免至2035 |
| 现金 | $3.721亿（+209%） | ⭐大幅改善 |
| 资产负债率 | 13.3% | ⭐极低 |
| P/B | HK$4.20 / NBV $0.543≈HK$4.23，约1.0x | |

**核心看点**

1. **Naga 3融资方案**：终止陈家协议后如何安排50%资金
2. **大众市场客量恢复**：能否完全恢复并超越疫情前水平
3. **VIP业务vs中国反赌**：高端和转介VIP能否维持增长
4. **派息政策**：30%派息比率是否会提升

**核心风险**

1. **Naga 3融资不确定性**：总投资超公司总资产，融资方案决定未来资本结构
2. **单一资产+控股家族治理**：100%依赖NagaWorld，公司治理透明度不足
3. **2024减值消失的基数效应**：2026年增速将回归正常
4. **中国反跨境赌博政策**：可能冲击VIP客源
5. **柬埔寨政治/政策风险**：牌照续期、税率变化等
