# 中国船舶租赁（03877.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

中国船舶租赁2025年实现营业收入HK$40.44亿元，同比 +0.24%（基本持平）。净利润HK$19.81亿元；股东应占溢利HK$18.45亿元，同比 -12.37%。每股基本盈利HK$0.298。利润下降的核心原因是OECD全球最低税（Pillar Two）计提HK$1.86亿元税务拨备——若剔除此项，利润同比实际增长0.6%至HK$21.67亿元。全年股息合计HK$0.16/股（中期5仙+特别6仙+末期5仙），派息率53.7%。

### 1.2 增长与拖累因素

增长方面，综合航运服务（经营租赁为主）收入HK$25.33亿元（+11.6%），受益于新增8艘集装箱船投放运营；净财务支出大幅下降28.8%（-HK$2.80亿），综合融资成本-62bps至2.91%。拖累方面，联营公司溢利腰斩50.6%（-HK$2.42亿，从HK$4.78亿→2.36亿），融资服务收入-14.4%（部分到期），所得税暴增846%（含Pillar Two HK$1.86亿），NCI+174%。利润下降非单一因素所致。

### 1.3 趋势判断

中国船舶租赁拥有"中国船厂系唯一上市船舶租赁平台"的独特定位，A-级国际信用评级提供低成本融资优势（2.91%远低于同业），绿色船舶占比91%契合全球航运脱碳趋势。新签10艘订单（$5.19亿，100%中高端船型）为未来收入提供增量。整体判断：**稳健的"收息+成长"组合——12.6% ROE、53.7%派息率、4.5年平均船龄的年轻船队，核心竞争力在于低融资成本和船厂资源协同**。

## 2. 主营业务

### 2.1 公司简介

中国船舶租赁是中国船舶集团（CSSC）旗下唯一的上市船舶租赁平台，专注于船舶融资租赁和经营租赁，同时提供船舶贷款和经纪服务。

### 2.2 主营业务构成

1. **综合航运服务（经营租赁为主）**：自有船舶出租给航运公司，收取租金；收入占比约62.6%
2. **融资服务（融资租赁+贷款）**：为船东提供船舶融资租赁和贷款服务，收取利息和租金；收入占比约37.4%
3. **船舶经纪**：提供船舶买卖和租赁经纪服务；收入占比约1%

### 2.3 核心收入来源

综合航运服务（经营租赁）贡献62.6%收入（HK$25.33亿），且增速最快（+11.6%）。融资服务贡献37.4%（HK$15.11亿），受到部分到期合同影响有所下降。经营租赁的资产端直接暴露于航运市场周期，融资租赁则收取利差。

### 4.3 营收分析

**主要增长来源**

综合航运服务收入+11.6%，核心驱动是2025年新投放运营的8艘集装箱船的全年贡献。航运市场整体景气（集装箱运价和油轮运价维持高位）支撑经营租赁租金水平。

**主要拖累因素**

融资服务收入-14.4%，部分融资租赁合同到期船东赎回船舶，导致融资租赁资产规模阶段性下降。

**结构变化**

经营租赁占比从约56%提升至62.6%，融资租赁占比下降。这意味着公司对航运市场周期的直接暴露度在增加。海上清洁能源装备占比37.7%为第一大类，反映公司积极布局绿色航运。

## 6. 未来可能的利好

### 利好 1：全球航运脱碳——绿色船舶需求持续增长

**依据**

公司节能船舶占比91%（行业前两名）。海上清洁能源装备（LNG船、FLNG等）占投融资金额37.7%，为第一大类。全球航运业面临IMO碳排放法规日趋严格（2030年碳强度目标-40%）。2025年新签10艘订单全部为中高端船型。

**解读**

全球现有船队中大量老旧高排放船舶需要更新换代，绿色船舶需求将持续增长至少到2030年。中国船舶租赁作为CSSC旗下平台，可优先获得中国船厂的绿色船舶订单和优先交付权，这是独特的竞争壁垒。

### 利好 2：融资成本持续下降——A-级评级的核心优势

**依据**

综合融资成本2.91%（-62bps），远低于同业平均水平。惠誉A-、标普A-国际评级+国内AAA评级。已建立30亿美元中期票据框架，发行10亿人民币境外债券。

**解读**

船舶租赁是资金密集型业务，融资成本是利润的核心决定因素。2.91%的融资成本在同业中处于领先水平（同业通常4-5%+），每节省100bps约对应数亿港元的利润改善。低利率环境延续+A-级评级将持续为公司提供融资成本优势。

### 利好 3：在建21艘船舶——未来2-3年收入增量可预见

**依据**

截至2025年末，在建船舶21艘（融资租赁8艘+经营租赁13艘），将在未来2-3年陆续交付投运。新签10艘订单金额$5.19亿。

**解读**

在建船舶提供了清晰的收入增长路径。每艘大型船舶投运后年贡献租金收入约HK$2,000-5,000万，21艘在建船舶全部交付后可增加年收入约HK$5-10亿元（增量25-50%）。

### 利好 4：高股息+稳定ROE——收息投资者的优质标的

**依据**

全年股息HK$0.16/股，派息率53.7%，ROE 12.6%。自上市以来持续派息。

**解读**

以当前股价估算，股息率约5-7%，加上ROE 12.6%的增长潜力，总股东回报颇具吸引力。稳定的租金收入+高派息率使其具备"类债券"的收息属性。

## 7. 未来可能的利空

### 利空 1：OECD Pillar Two全球最低税——永久性利润侵蚀

**依据**

2025年首次计提OECD Pillar Two税务拨备HK$1.86亿元，这是一个新的、持续性的税负。该税制要求跨国企业在每个管辖区的有效税率不低于15%。

**解读**

与一次性费用不同，Pillar Two是永久性的税负增加。在公司利润约HK$20亿的规模下，每年HK$1.86亿的额外税负相当于削减了约9%的净利润。这将成为持续侵蚀股东回报的因素。

### 利空 2：航运周期性风险——经营租赁敞口增大

**依据**

经营租赁收入占比从约56%提升至62.6%，公司对航运市场租金水平的直接暴露度在增加。航运业具有强周期性，集装箱运价和散货运价均处于历史高位区间。

**解读**

如果全球贸易增速放缓（如中美贸易摩擦加剧）或航运市场供给大幅增加，租金水平可能回落，直接影响经营租赁收入。2025年新增的8艘集装箱船如果遇到运价下行周期，利润贡献将不及预期。

### 利空 3：利率风险——融资成本回升可能性

**依据**

当前融资成本2.91%处于历史低位。如果全球利率环境因通胀反弹而逆转上升，公司的融资成本将随之增加。

**解读**

虽然A-级评级提供了一定的利率缓冲，但利率每上升50bps对应数千万至上亿港元的成本增加。公司总资产规模约HK$400-500亿，债务占比较高，对利率变动敏感。

## 8. 总结判断

**一句话结论**

中国船舶租赁2025年PBT微增1.8%但股东应占净利-12.4%，差距来自三大因素——联营公司溢利腰斩（-50.6%）、所得税暴增（+846%，含Pillar Two）、NCI大增（+174%），融资成本大降28.8%是最大正面因素但被抵消。凭借2.91%的低融资成本、A-级国际评级、91%绿色船舶占比和53.7%高派息率，仍是"航运+高股息"板块中兼具成长和防御特征的标的——但利润增长质量需审视。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| PBT | HK$22.19亿（+1.8%） | 经营层面基本稳定 |
| 联营公司溢利 | HK$2.36亿（-50.6%） | ⚠️⚠️腰斩，最大拖累 |
| 所得税 | HK$2.38亿（+846%） | ⚠️⚠️Pillar Two为主 |
| NCI | HK$1.35亿（+174%） | ⚠️分走更多利润 |
| 净财务支出 | HK$6.93亿（-28.8%） | ✅✅最大利好 |
| 现金 | HK$37.07亿（+109%） | ✅翻倍 |
| 资产负债率 | 65.0%（-2.4ppt） | ✅改善 |
| NBV | HK$2.421（+5.6%） | 当前股价HK$2.47，P/B ≈ 1.02x |

**核心看点**

1. **联营公司溢利能否恢复**：2024年HK$4.78亿→2025年HK$2.36亿的腰斩是否为一次性因素
2. **在建21艘船舶的交付节奏**：未来2-3年将带来确定性的收入增量
3. **Pillar Two税负的实际影响**：是否固定在~HK$1.86亿/年还是会随利润变化
4. **融资成本能否继续下降**：当前2.91%已很低，还有空间但幅度有限

**核心风险**

1. **联营公司溢利波动性**：占PBT约10%，但变动幅度巨大（-50.6%），不可预测
2. **OECD Pillar Two永久性税负**：每年~HK$1.86亿+，有效税率从1.2%提升至10.7%
3. **NCI持续增长**：子公司利润分流加剧，需关注NCI比例趋势
4. **航运市场下行周期**：经营租赁占比62.6%意味着对运价/租金敏感度高
5. **地缘政治对全球贸易的影响**：航运需求与全球贸易量直接挂钩
