# 中海石油化学（03983.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

中海石油化学2025年实现营业收入RMB 120.34亿（+0.7%），股东应占溢利RMB 9.74亿（-9.0%）。PBT RMB 13.66亿（-6.9%）。毛利RMB 15.71亿（-7.9%），毛利率13.05%（-1.22ppt）。EPS RMB 0.21（-8.7%）。DPS RMB 0.112（-7.3%），派息比率53.3%。NBV RMB 4.085（+2.3%）。

### 1.2 增长与拖累因素

拖累方面：（1）尿素均价下跌247元/吨，减少收入RMB 4.76亿；（2）甲醇均价下跌120元/吨，减少收入RMB 1.82亿；（3）毛利率-1.22ppt至13.05%，因产品价格下降而成本端（天然气/煤炭）未同步下降；（4）新增投资物业减值RMB 0.40亿；（5）NCI+77.3%（子公司盈利增加导致少数股东分走更多利润）。增长驱动：（1）磷肥/复合肥销量+2.4万吨，收入+10.4%；（2）甲醇销量+9.1万吨抵消部分价格下跌；（3）联营公司盈利+153%至RMB 1.24亿；（4）有效税率-3.2ppt至18.3%（节省RMB 0.66亿）；（5）八所港棚改征收资产处置收益贡献其他收入增长155%。

### 1.3 趋势判断

中海石油化学2025年"量增价跌"——销量增长被价格下跌吃掉，收入几乎零增长而利润下降9%。整体判断：**"化肥周期下行中的'以量补价'——尿素/甲醇价格下跌是核心拖累，磷肥增长和联营盈利对冲部分压力，但超低资产负债率（19.6%）和RMB 144亿关联方存款提供了极强的安全垫"**。

## 2. 主营业务

### 2.1 公司简介

中海石油化学（China BlueChem）是中国海油集团旗下的化肥及化工业务平台，主要从事化肥（尿素、磷肥、复合肥）和化工产品（甲醇）的生产和销售。

### 2.2 主营业务构成

1. **尿素**：核心产品，以天然气/煤为原料，2025年销量增加3.4万吨但均价下跌247元/吨
2. **甲醇**：第二大产品，2025年销量增加9.1万吨但均价下跌120元/吨
3. **磷肥及复合肥**：2025年销量增加2.4万吨，收入RMB 30.47亿（+10.4%）
4. **其他**：精细化工等

### 2.3 核心收入来源

尿素和甲醇合计贡献主要收入，磷肥/复合肥是增长亮点。公司的天然气原料部分来自母公司中国海油（关联交易），具有成本优势。海南富岛和海南基地是主要生产基地。

## 4. 营收分布和增长

### 4.1 按产品划分

| 产品 | 2025年收入 | 同比增速 | 量变 | 价变 | 备注 |
|---|---:|---:|---|---|---|
| 甲醇 | RMB 31.06亿 | — | 销量+9.1万吨 | 均价-120元/吨 | 以量补价 |
| 磷肥及复合肥 | RMB 30.47亿 | +10.4% | 销量+2.4万吨 | 价格相对稳定 | ✅亮点 |
| 尿素 | — | — | 销量+3.4万吨 | 均价-247元/吨 | ⚠️价跌幅度大 |

### 4.3 营收分析

**量增价跌格局**

三大产品（尿素/甲醇/磷肥）销量均有增长，合计增量超12.5万吨。但尿素和甲醇价格分别下跌247和120元/吨，导致收入几乎零增长。磷肥/复合肥+10.4%是唯一的收入端亮点。

**价格下跌原因**

尿素价格下跌主要因国内产能释放（新疆等煤制尿素项目投产）和出口受限（窗口期调控）。甲醇价格受国际油价波动和国内产能过剩影响。

## 6. 未来可能的利好

### 利好 1：净现金公司——RMB 151亿广义现金+19.6%资产负债率

**依据**

关联方存款+现金合计约RMB 151亿，有息负债仅RMB 21亿，净现金约RMB 130亿。每年净利息收入RMB 2.81亿。

**解读**

极低的资产负债率和充裕的现金意味着：（1）化肥周期下行时有极强的抗风险能力；（2）有能力在行业低谷进行扩张或收购；（3）每年利息收入贡献约29%的PBT。

### 利好 2：磷肥/复合肥+10.4%——产品结构优化

**依据**

磷肥及复合肥收入RMB 30.47亿（+10.4%），销量+2.4万吨。

**解读**

复合肥附加值高于尿素和甲醇，产品结构向磷肥/复合肥倾斜有利于提升毛利率。

### 利好 3：联营公司盈利+153%——投资布局见效

**依据**

联营公司盈利RMB 1.24亿（+153%），联营权益增至RMB 6.22亿（+35%）。

**解读**

公司通过联营参股方式布局产业链上下游，盈利贡献增加说明投资布局开始见效。

### 利好 4：高派息——53.3%派息率

**依据**

DPS RMB 0.112，派息比率53.3%。按HK$2.78股价计，股息率约RMB 0.112/RMB 2.53≈4.4%（按港元计约4.4%）。

**解读**

稳定的高派息率（50%+）对于收入型投资者有吸引力。

## 7. 未来可能的利空

### 利空 1：尿素/甲醇价格持续下跌风险

**依据**

尿素均价-247元/吨，甲醇均价-120元/吨。国内尿素产能仍在扩张（新疆等煤制项目），甲醇受油价和产能过剩影响。

**解读**

如果产品价格继续下跌而成本端（天然气价格）不同步下降，毛利率将进一步压缩。化肥行业具有明显的周期性，当前可能处于下行周期。

### 利空 2：毛利率持续下滑——13.1%且在恶化

**依据**

毛利率从2021年21.9%→2024年14.3%→2025年13.1%，持续下滑。

**解读**

毛利率已从高点下降近9个百分点，反映了化肥行业景气度下行。如果降至10%以下，公司可能面临盈利能力的实质性恶化。

### 利空 3：关联方存款集中度风险

**依据**

RMB 143.89亿存放于中国海油财务公司，占总资产58%。

**解读**

虽然中国海油是央企、信用风险极低，但如此高的关联方存款集中度意味着：（1）资金使用效率可能不高（相当于"帮母公司存钱"）；（2）如果财务公司出现问题（极端情况），将对公司产生重大影响。此外，存款利息收入将随利率下降而减少。

### 利空 4：ROE偏低——约5.2%

**依据**

净利润RMB 9.74亿/平均股东资金约RMB 186亿 = ROE约5.2%。

**解读**

ROE偏低主要因为大量现金"闲置"在关联方存款中拉低了资产收益率。如果公司能更积极地使用这些资金（分红/投资/回购），ROE可以显著提升。

## 8. 总结判断

**一句话结论**

中海石油化学2025年收入+0.7%至RMB 120亿、利润-9.0%至RMB 9.74亿——"量增价跌"格局下尿素/甲醇价格下跌拖累毛利率-1.2ppt至13.1%，但净现金RMB 130亿+资产负债率19.6%提供极强安全垫，53%高派息率是亮点。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| 毛利率 | 13.05%（-1.22ppt） | ⚠️持续下滑 |
| PBT | RMB 13.66亿（-6.9%） | |
| 联营 | RMB 1.24亿（+153%） | ✅ |
| 有效税率 | 18.3%（-3.2ppt） | ✅ |
| 关联方存款 | ~RMB 144亿 | 占资产58% |
| 净现金 | ~RMB 130亿 | ⭐ |
| P/B | HK$2.78/NBV RMB 4.09≈HK$4.50，约0.62x | 深度破净 |
| P/E | HK$2.78/EPS RMB 0.21≈HK$0.23，约12.1x | |
| 股息率 | DPS RMB 0.112≈HK$0.123/HK$2.78=4.4% | |

**核心看点**

1. **尿素/甲醇价格走势**：是否见底回升
2. **毛利率能否企稳**：13.1%是否为底部
3. **关联方存款用途**：能否提高资金使用效率
4. **磷肥/复合肥增长持续性**

**核心风险**

1. **产品价格下跌周期**：尿素/甲醇产能过剩
2. **毛利率持续下滑**：4年从22%降至13%
3. **关联方存款集中度**：58%资产存放于母公司财务公司
4. **ROE偏低**：约5.2%
