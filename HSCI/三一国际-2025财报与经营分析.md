# 三一国际 2025 年财报整理

> 三一國際發展有限公司（股票代码：0631.HK），三一集团旗下港股上市平台，主营矿山装备、物流装备、油气装备和新兴产业装备，全球化布局持续推进。
> 本报告基于公司2025年12月31日止年度全年业绩公告，币种为人民币（除特别注明外）。

# 【三一国际】2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

三一国际2025年营业收入人民币243.34亿元，同比 +11.1%；归母净利润人民币17.79亿元，同比 +61.5%。国际收入92.34亿元（+13.0%）持续驱动增长，矿山装备和物流装备双轮驱动格局稳固。

- 营业收入：人民币243.34亿元，同比 +11.1%（2024年：约人民币219亿元）
- 归母净利润：人民币17.79亿元，同比 +61.5%（2024年：约人民币11.01亿元）
- 经营性现金流：人民币24.6亿元，同比 +14%
- 每股基本盈利：人民币0.51元
- 末期股息：每股0.35港元

### 1.2 增长与拖累因素

利润增速（+61.5%）远超收入增速（+11.1%），体现出强劲的经营杠杆和费用管控能力。国际化和产品结构优化是核心驱动力。

- 主要增长驱动：国际收入+13%（物流装备国际+14.6%）、经营杠杆效应、产品结构优化
- 主要挑战：H1毛利率小幅下降0.7ppt至23.7%、矿山装备国际增速放缓至+6.8%
- 利润增速远超收入增速，经营效率显著改善

### 1.3 趋势判断

公司正从"中国矿山设备商"向"全球化综合装备集团"转型。国际收入占比持续提升，电动化和智能化是技术升级方向，目标2030年成为全球一流矿山设备供应商。

- 未来主要利好：全球矿山电动化趋势、物流自动化需求增长、国际化深入
- 未来主要压力：毛利率下行压力、美国关税政策对出口影响、矿山行业周期性
- 整体趋势判断：收入和利润双增长趋势确立，国际化战略成效显著

## 2. 主营业务

### 2.1 公司简介

三一国际是三一集团旗下港股上市公司，专注于矿山装备、物流装备、油气装备和新兴产业装备的研发、制造和销售。产品涵盖掘进机、采煤机、矿用卡车、港口起重机、正面吊、空港设备、油气钻井设备等，销售覆盖亚洲、欧洲、非洲、美洲等全球市场。

### 2.2 主营业务构成

1. **矿山装备**：掘进机、采煤机、矿用卡车等井工和露天矿山设备，收入占比约50%
2. **物流装备**：港口起重机、正面吊、叉车、空港设备等，收入占比约33%
3. **新兴产业装备**：包括微电网、储能等新业务，收入占比约9%
4. **油气装备**：石油天然气钻采设备，收入占比约9%

### 2.3 核心收入来源

矿山装备和物流装备合计贡献约83%的收入，是绝对核心业务。国际收入占比显著（全年92.34亿，约38%），且增速快于国内。

### 4.3 营收分析

**矿山装备稳健增长**

矿山装备收入108.98亿（约占50%），国际收入38.23亿（+6.8%）。公司深耕井工设备领域，坚持电动化动力总成研发。国际增速放缓至6.8%（低于整体13%），可能受到部分海外矿山投资周期影响。

**物流装备国际增长亮眼**

物流装备收入72.56亿（约占33%），国际收入51.25亿（+14.6%），是增长最快的板块。物流装备的国际收入占比达70.6%，高度国际化。推进港口设备电动化和无人化是核心竞争力。

**新兴产业和油气装备成为增长补充**

新兴产业装备18.89亿（含微电网、储能等）和油气装备18.67亿合计贡献约17%收入，是多元化布局方向。

**国际化战略成效显著**

全年国际收入92.34亿（+13.0%），约占总收入38%。产品销售覆盖全球，物流装备国际占比超70%，矿山装备国际占比约35%。

## 5. 利润分布和增长

### 5.1 整体利润表现

| 归母净利润 | 人民币17.79亿 | 约人民币11.01亿 | 同比 +61.5% | |
| 经营性现金流 | 人民币24.6亿 | 约人民币21.6亿 | 同比 +14% | |
| 净利率 | 7.3% | 5.0% | +2.3ppt | |

### 5.2 利润分析

**利润高增长驱动因素**

(1) 收入增长11.1%带来规模效应，固定成本被更多收入覆盖（经营杠杆）。(2) 产品结构优化——高毛利产品占比可能提升。(3) 费用管控有效——销售和管理费用增速可能低于收入增速。(4) 净利率从5.0%提升至7.3%，改善显著。

**毛利率微降值得关注**

H1毛利率23.7%（-0.7ppt），可能受到竞争加剧和产品价格压力影响。但净利率大幅提升说明公司在毛利率小幅下降的情况下通过费用控制实现了利润率的整体改善。

**现金流健康**

经营性现金流24.6亿（+14%），远超归母净利润17.79亿，现金流转化率优秀。这为持续投资研发和国际扩张提供了资金保障。

## 6. 未来可能的利好

### 利好 1：全球矿山电动化趋势

**依据**

公司坚持电动化动力总成研发，通过"源网荷储"微电网解决方案支持矿山装备电动化。全球矿山行业正在加速脱碳，电动矿卡和电动井工设备需求快速增长。

**解读**

矿山电动化是未来十年的确定性趋势，三一国际在电动矿山装备领域的布局将带来产品溢价和市场份额提升。微电网业务可以形成"设备+能源"的捆绑销售模式。

### 利好 2：物流自动化和港口无人化

**依据**

物流装备国际收入+14.6%，国际占比超70%。公司推进中国电动化、无人化港口设备向海外拓展。全球港口自动化改造需求持续增长。

**解读**

港口自动化是全球物流基础设施升级的核心方向。三一国际的港口起重机、AGV等产品在全球市场具有性价比优势。随着更多港口启动自动化改造，物流装备订单有望持续增长。

### 利好 3：国际化持续深入

**依据**

2025年国际收入92.34亿（+13.0%），约占总收入38%。公司目标2030年成为全球一流矿山设备供应商，持续拓展亚洲、欧洲、非洲、美洲市场。

**解读**

国际化是三一国际区别于国内竞争对手的核心优势。海外市场利润率通常高于国内，国际收入占比的提升将改善整体盈利能力。

## 7. 未来可能的利空

### 利空 1：毛利率下行压力

**依据**

H1毛利率从24.4%降至23.7%（-0.7ppt）。矿山装备和物流装备均面临价格竞争加剧。

**解读**

毛利率下滑若持续，可能侵蚀利润增长的基础。目前利润高增长依赖经营杠杆和费用控制，若收入增速放缓+毛利率继续下降，利润增速将大幅回落。

### 利空 2：美国关税政策对出口的潜在冲击

**依据**

公司H1对美国销售2.86亿元。美国关税政策频繁变动，对中国制造的工程机械和装备加征关税。

**解读**

虽然美国收入占比不高（~2.3%），但关税政策的溢出效应可能影响其他市场（如客户转向非中国品牌）。地缘政治风险是全球化企业的长期挑战。

### 利空 3：矿山行业周期性风险

**依据**

矿山装备占收入约50%，煤炭和矿产价格波动直接影响矿山企业资本开支。全球经济增速放缓可能抑制矿产需求。

**解读**

矿山行业具有明显的周期性。若大宗商品价格大幅下跌或矿山企业削减资本开支，矿山装备订单将受到直接冲击。目前国际矿山装备增速已放缓至+6.8%，需关注趋势变化。

## 8. 总结判断

**一句话结论**

三一国际2025年收入+11.1%、利润+61.5%——国际化和经营杠杆驱动利润高增长，矿山装备和物流装备双轮驱动格局稳固，净利率从5.0%提升至7.3%，但毛利率微降需关注。

**核心看点**

1. 归母净利润17.79亿（+61.5%），利润增速远超收入增速
2. 国际收入92.34亿（+13.0%），物流装备国际占比超70%
3. 经营性现金流24.6亿（+14%），现金流转化率优秀
4. 矿山电动化+港口无人化双重技术趋势受益
5. 末期股息0.35港元/股

**核心风险**

1. H1毛利率从24.4%降至23.7%（-0.7ppt），价格竞争压力
2. 矿山装备国际增速放缓至+6.8%，需关注周期拐点
3. 美国关税政策对出口的潜在影响
4. 矿山行业周期性——大宗商品价格波动传导
