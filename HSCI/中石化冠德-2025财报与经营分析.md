# 【中石化冠德】2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

中石化冠德2025年实现营业收入约6.59亿港元，同比 -1.27%；毛利约3.19亿港元，同比 -1.38%；税前利润约11.30亿港元，同比 -12.20%；公司权益持有人应占溢利约10.28亿港元，同比 -12.67%。每股基本盈利41.36港仙。全年每股现金股息25港仙（中期10港仙+末期15港仙），与上年持平。股息派付比率由52.79%提升至60.45%。

- 全年营业收入约6.59亿港元（同比 -1.27%）
- 全年税前利润约11.30亿港元（同比 -12.20%）
- 全年归母净利润约10.28亿港元（同比 -12.67%）

### 1.2 增长与拖累因素

利润同比下滑12.67%的核心原因是联营/合营公司投资收益大幅减少——应占联营及合营公司业绩从8.55亿港元降至6.04亿港元（同比 -29.32%）。国内六家码头公司吞吐量下降（上半年 -14.97%至8,503万吨）是最大拖累，主要受新能源替代加速、国内原油需求走弱及部分炼厂检修影响。LNG船舶投资收益亦因一次性财务调整和维保费用增加而下滑（上半年 -39.06%）。

- 主要增长驱动：华德石化石脑油接卸项目试运行、Vesta欧洲业务投资收益增长（上半年 +15.40%）、派息政策上调
- 主要拖累因素：国内码头吞吐量下降、LNG船舶一次性调整、富查伊拉折旧费用增加
- 增长或承压主要来自联营公司投资收益的变动，自营华德石化业务基本稳定

### 1.3 趋势判断

中石化冠德是典型的"收费站"型资产——利润主要来自码头吞吐费、仓储费和船舶运输费。收入规模不大（6.59亿港元），但利润高达10.28亿港元，因为大量利润来自联营/合营公司的投资收益（如六家码头公司、富查伊拉、Vesta等），这些不计入收入但计入利润。

- 未来主要利好方向：石脑油接卸新增收入、3艘新LNG船投产（预计2027-2028年）、派息政策≥30%+每年不低于上年
- 未来主要压力方向：新能源替代导致原油需求下降、码头吞吐量恢复不确定
- 整体趋势判断：业务稳定但增长有限，核心看码头吞吐量恢复和新项目投产时间

## 2. 主营业务

### 2.1 公司简介

中石化冠德是中国石化集团旗下唯一的红筹上市附属公司，主要经营原油码头装卸及储运服务、LNG船舶运输、海外油品仓储等业务，致力于成为世界一流国际石化仓储物流公司。

### 2.2 主营业务构成

1. **原油码头及储运（自营）**：全资子公司华德石化运营大亚湾马鞭洲码头，提供原油及石脑油接卸、输送服务
2. **国内码头投资（联营/合营）**：持有六家国内码头公司权益——曹妃甸实华（90%）、天津港实华（50%）、青岛实华（50%）、日照实华（50%）、宁波实华（50%）、湛江港石化（50%）
3. **海外仓储**：富查伊拉油品码头（阿联酋中东油品仓储中转）、Vesta（欧洲多品种油品仓储）
4. **LNG船舶运输**：通过中国能源运输投资有限公司运营8艘LNG船舶，另有3艘在建

### 2.3 核心收入来源

公司合并报表的收入（6.59亿港元）全部来自华德石化的原油码头及储运服务。但公司的核心利润来源是六家国内联营码头公司的投资收益，以及海外仓储和LNG运输的投资收益——2025年联营及合营公司业绩贡献约6.04亿港元，占税前利润的53.5%。利息及定期存款收入约3.08亿港元亦是重要利润组成部分。

### 4.3 营收分析

**主要增长来源**

华德石化马鞭洲码头石脑油接卸项目于2025年开始试运行，上半年接卸4艘石脑油船、28.44万吨石脑油，为新增收入来源。Vesta在欧洲成功接卸首船航空煤油，油品覆盖范围扩展至石脑油、汽油、柴油、生物柴油、航煤及燃料油，增强了市场竞争力。

**主要拖累因素**

国内六家联营码头吞吐量大幅下降是最大拖累——上半年合计约8,503万吨（-14.97%），主要受新能源替代加速导致国内原油需求走弱、部分炼厂缩减进口计划及检修停产影响。华德石化自身原油接卸量亦下降12.45%。LNG运输因中国能源对CESI QINGDAO设备故障损失进行一次性财务调整及长期维保费用摊销增加，投资收益下滑39.06%。

**结构变化**

收入结构保持单一——合并收入全部来自华德石化。但利润贡献结构正在微调：石脑油接卸业务的试运行有望增加华德石化自营利润的占比；Vesta油品品种覆盖更全面提升了海外仓储竞争力。联营/合营公司投资收益占比因国内码头下滑而有所下降。

## 6. 未来可能的利好

### 利好 1：派息政策上调——≥30%派息率+三年递增承诺

**依据**

公司制定2025-2027年股息分配方案：任何一年全年现金股息不低于上一年；现金股息派付比率由≥20%提高至≥30%。2025年实际派息率已达60.45%。

**解读**

对于收费站型资产，提高派息率和"永不下降"承诺直接提升了回报确定性。当前全年股息每股25港仙，以约4.11港元股价计算，股息率约6.1%，在利率下行环境中具有吸引力。即使利润短期承压，派息金额不降的承诺为投资者提供了底线保护。

### 利好 2：马鞭洲石脑油接卸项目+Vesta航煤业务拓展

**依据**

华德石化马鞭洲码头石脑油接卸项目已进入试运行，业务量预计可提升20%以上。Vesta顺利接卸首船航煤，油品品种覆盖已扩展至六大品种，并已启动绿色氨卸载和仓储项目可行性研究。

**解读**

石脑油接卸是华德石化的增量业务，利用现有码头富余产能，边际成本较低，有望提升华德石化分部业绩。Vesta油品多元化增强了抗周期能力，绿色氨等新业务为中长期增长埋下伏笔。

### 利好 3：3艘新LNG船舶建设有序推进

**依据**

中国能源运输投资有限公司3艘LNG船舶建设有序推进，预计2027-2028年陆续建成下水。目前运营8艘LNG船。

**解读**

新增3艘LNG船舶将使运力提升37.5%，在长期合约模式下可带来稳定的增量投资收益。中国LNG进口需求长期增长趋势为运力扩张提供了需求基础。

## 7. 未来可能的利空

### 利空 1：新能源替代加速导致国内原油码头吞吐量结构性下行

**依据**

2025年上半年六家国内码头吞吐量同比 -14.97%，华德石化接卸量同比 -12.45%。管理层在中报中明确提到"新能源替代加速"和"国内原油需求走弱"。

**解读**

中国交通领域电动化转型正在加速削弱成品油需求，进而影响上游原油进口和码头吞吐量。这不是短期波动而可能是长期趋势。联营/合营公司投资收益高度依赖码头吞吐量，如果下行趋势持续，将持续压制公司利润。

### 利空 2：增长空间有限，业务天花板明显

**依据**

收入仅下降1.27%但利润下降12.67%——体现了利润对联营公司经营的高杠杆敏感性。收入6.59亿港元的体量下，增长空间有限——码头是存量资产，新项目（石脑油、LNG船）投产周期较长。

**解读**

作为收费站型资产，中石化冠德的增长空间受限于码头产能和新项目投产节奏。在没有重大收购或新建项目的情况下，利润可能长期处于平台期。公司账上现金超过77亿港元但缺乏高回报的再投资机会，大量资金沉淀在定期存款中。

### 利空 3：高度依赖中石化集团关联交易

**依据**

公司主要客户为中石化集团及其关联公司。华德石化的码头服务、国内六家码头的主要客户均为中石化体系。

**解读**

业务高度依赖母公司中石化集团——码头服务定价和业务量受母公司采购及物流策略决定。如果中石化调整炼油布局、改变原油进口港口分配或削减进口计划，将直接影响冠德的业务量和利润。公司缺乏独立的市场化客户拓展能力。

## 8. 总结判断

**一句话结论**

中石化冠德2025年收入6.59亿港元（-1.27%），归母净利10.28亿港元（-12.67%）——国内码头吞吐量下降导致联营公司投资收益大幅减少是核心拖累；但资产负债率仅2.05%、账上现金超77亿港元、派息率提升至60.45%且承诺三年递增，作为高股息收费站型资产的防御价值仍在。

**核心看点**

1. 国内码头吞吐量能否止跌回升——决定联营公司投资收益走向
2. 石脑油接卸项目正式运营后的收入和利润增量
3. 3艘新LNG船2027-2028年投产后的利润贡献
4. 派息政策执行——≥30%派息率和每年不低于上年的承诺

**核心风险**

1. 新能源替代导致原油码头吞吐量结构性下行
2. 利润对联营公司经营高度敏感——联营收益下降29%而归母净利仅下降12.67%
3. 高度依赖中石化集团关联交易，缺乏独立客户拓展
4. 巨额现金沉淀但缺乏高回报再投资机会
