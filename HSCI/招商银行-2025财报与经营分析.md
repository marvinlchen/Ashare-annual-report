# 招商银行（03968.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

招商银行2025年实现营业收入RMB 3,375.32亿（+0.01%），归母净利润RMB 1,501.81亿（+1.2%）。H股口径扣除优先股/永续债利息后股东应占溢利RMB 1,448.70亿（+0.7%）。净利息收入RMB 2,155.93亿（+2.0%），非利息收入RMB 1,219.39亿（-3.4%）。ROAE 13.44%，ROAA 1.19%。EPS RMB 5.70（+0.7%）。全年DPS RMB 2.016（含中期1.013+末期1.003），派息比率35.4%。

### 1.2 增长与拖累因素

增长驱动：（1）净利息收入+2.0%，受益于负债端成本大幅下降（利息支出-16.7%），对冲了资产端利息收入-6.1%的压力；（2）大财富管理收入RMB 440.13亿（+16.9%），是非息收入中的亮点；（3）净手续费及佣金收入RMB 752.58亿（+4.4%）；（4）联营公司盈利+108%至RMB 54.73亿；（5）资产质量优异——不良率0.94%（-1bp）、拨备覆盖率391.79%。

拖累方面：NIM持续收窄至1.87%（-11bps）；非利息收入-3.4%（投资收益/汇兑等下降）；其他经营收入-5.7%。经营收入总额实际-0.8%（etnet口径），利润增长几乎完全依赖拨备节省和税收优化。

### 1.3 趋势判断

招商银行2025年在NIM持续承压的行业大背景下实现了"营收零增长、利润微增长"，展现了中国最优质零售银行的韧性。整体判断：**"净息差压力是短期核心矛盾（1.87%且仍在收窄），但大财富管理+16.9%是转型亮点，资产质量（不良0.94%/拨备392%）和ROE（13.4%）仍远优于同业——低增长但高质量"**。

## 2. 主营业务

### 2.1 公司简介

招商银行是中国最大的股份制商业银行，以零售银行业务见长（被誉为"零售之王"），业务涵盖零售金融、公司金融、金融市场和财富管理，旗下招银国际、招商基金、招银理财等子公司覆盖综合金融。

### 2.2 主营业务构成

1. **零售金融**：贡献超60%营收，涵盖个人存贷款、信用卡、按揭、消费贷和私人银行
2. **公司金融**：对公存贷款、贸易金融、现金管理和供应链金融
3. **金融市场/其他**：债券投资、做市、同业业务、衍生品
4. **大财富管理**：2025年收入RMB 440.13亿（+16.9%），涵盖代销基金/保险/理财、托管、资管

### 2.3 核心收入来源

零售金融贡献超60%营收和利润，是招行最核心的竞争壁垒。大财富管理是转型方向和增长引擎（+16.9%），包括AUM管理费/代销佣金/托管费。招行拥有1.97亿零售客户（含私人银行），MAU超1亿的招商银行App是零售获客的核心入口。

## 6. 未来可能的利好

### 利好 1：大财富管理+16.9%——转型核心引擎

**依据**

大财富管理收入RMB 440.13亿（+16.9%），涵盖代销基金/保险/理财、AUM管理费、托管费。招行AUM超13万亿，私人银行客户超15万户。

**解读**

财富管理是"轻资本、低风险、高ROE"的业务模式，代表银行从利差依赖向服务费转型。招行在这一领域的领先地位（App超1亿MAU+海量零售客户）是最核心的长期竞争优势。

### 利好 2：资产质量极优——不良0.94%/拨备392%

**依据**

不良贷款率0.94%（-1bp），拨备覆盖率391.79%，远优于同业（行业平均不良~1.6%，拨备~200%）。

**解读**

低不良+高拨备意味着：（1）未来有释放拨备支撑利润的空间；（2）即使经济下行，也有足够的安全垫。这是招行能在行业中持续获得估值溢价的核心原因。

### 利好 3：负债端降本——存款成本下降支撑NII

**依据**

利息支出-16.7%至RMB 1,358亿，使得NII在资产端收益下降6.1%的情况下仍实现+2.0%增长。

**解读**

招行的活期存款占比（~60%以上）远高于同业，是其最大的负债端优势。在降息周期中，活期存款成本下降更快，有助于对冲NIM压力。

## 7. 未来可能的利空

### 利空 1：NIM持续收窄——1.87%且仍在下行

**依据**

NIM 1.87%（-11bps），已从2021年的2.48%持续下降至当前水平。LPR仍有下调空间，存量房贷利率调整持续影响。

**解读**

NIM收窄是中国银行业面临的系统性压力。如果NIM进一步降至1.7%以下，即使招行的活期存款优势也难以完全对冲。NIM每降10bps约影响利润RMB 100亿+。

### 利空 2：核心经营利润实际收缩——增长依赖非经营因素

**依据**

经营收入-0.8%，经营溢利-1.4%。利润正增长（+1.2%）靠联营公司+108%、税收优化-4.2%和拨备释放来实现。

**解读**

这种增长模式不可持续——联营公司利润波动大，拨备释放有底限（不良率已很低），税率优化空间有限。如果核心经营利润持续收缩，未来净利润增长可能转负。

### 利空 3：非利息收入-3.4%——手续费以外的收入下降

**依据**

非利息净收入-3.4%至RMB 1,219亿。虽然净手续费+4.4%和大财富管理+16.9%表现良好，但其他非息收入（投资收益、汇兑等）下降。

**解读**

非利息收入的波动性较大，受资本市场环境影响。如果2026年资本市场走弱，财富管理和手续费增长也可能放缓。

### 利空 4：ROE下降趋势——13.44%且在压缩

**依据**

ROAE从2021年的约16.9%持续降至13.44%（-0.56ppt）。ROAA从1.36%降至1.19%。

**解读**

ROE下降反映了NIM收窄和收入增速放缓。如果ROE降至10%以下，招行相对同业的估值溢价将面临压力。

## 8. 总结判断

**一句话结论**

招商银行2025年营收零增长（+0.01%）、归母净利+1.2%至RMB 1,502亿——NIM收窄-11bps至1.87%是核心压力，经营利润实际-1.4%，正增长靠联营+108%/税收优化/拨备释放勉强实现；但大财富管理+16.9%是转型亮点，资产质量（不良0.94%/拨备392%）和ROE（13.4%）仍远优于同业——"低增长但高质量"的零售银行龙头。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| NII | RMB 2,156亿（+2.0%） | 负债降本支撑 |
| NIM | 1.87%（-11bps） | ⚠️核心压力 |
| 不良率 | 0.94%（-1bp） | ⭐行业最优 |
| 拨备覆盖率 | 391.79% | ⭐安全垫极厚 |
| 大财富管理 | RMB 440亿（+16.9%） | ⭐转型亮点 |
| ROAE | 13.44%（-0.56ppt） | 仍远优于同业 |
| P/B | HK$51.50/NBV RMB 44.52≈HK$48.97，约1.05x | |
| P/E | HK$51.50/EPS RMB 5.70≈HK$6.27，约8.2x | |
| 股息率 | DPS RMB 2.016≈HK$2.22/HK$51.50=4.3% | |

**核心看点**

1. **NIM何时企稳**：1.87%是否为阶段性底部
2. **大财富管理增速持续性**：+16.9%能否延续
3. **核心经营利润能否恢复正增长**：而非依赖联营/拨备/税收
4. **资产质量能否持续优异**：不良0.94%是否为真实底部

**核心风险**

1. **NIM持续收窄**：LPR下调+存量房贷利率调整
2. **核心经营利润收缩**：收入零增长/经营溢利-1.4%
3. **ROE下降趋势**：从~17%降至13.4%
4. **房地产风险**：虽然当前资产质量优异，但房地产贷款仍有潜在风险
