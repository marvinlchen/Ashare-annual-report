# 中国银行（03988.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

中国银行2025年实现营业收入RMB 6,583.1亿（+4.5%，A股口径），经营收入总额RMB 6,598.66亿（+4.3%，etnet/IFRS口径）。归母净利润RMB 2,430.2亿（+2.2%），H股口径扣除优先股/永续债利息后股东应占溢利RMB 2,291.87亿（+3.5%）。NIM 1.26%（-14bps）。净利息收入RMB 4,407.05亿（-1.8%），非利息收入RMB 2,176.1亿（+20.1%）。ROAE 8.94%，ROAA 0.70%。成本收入比27.84%（-0.93ppt）。EPS RMB 0.74（-1.3%）。DPS RMB 0.2263，派息比率30.6%。

### 1.2 增长与拖累因素

增长驱动：（1）非利息收入+20.1%至RMB 2,176亿，是经营收入增长的核心引擎；（2）利息支出-10.0%（负债端成本大幅下降）；（3）贷款减值-11.3%至RMB 1,072亿（拨备释放）；（4）优先股/永续债利息-15.1%。拖累方面：（1）NIM持续收窄-14bps至1.26%；（2）利息收入-6.6%；（3）经营支出+8.8%（显著高于收入增速+4.3%）；（4）其他减值回拨大幅收窄-77.4%。

### 1.3 趋势判断

中国银行2025年在NIM持续承压的环境下实现了收入+4.5%、利润+2.2%的稳健增长，得益于非利息收入+20.1%的强劲表现。整体判断：**"NIM承压但非息收入接力——1.26%的NIM是四大行中偏低的（受外币资产拖累），但非息+20.1%补偿了净利息收入-1.8%的缺口；费用增速+8.8%远超收入是隐忧，拨备释放-11.3%继续支撑利润——国际化大行的'低NIM、高非息'特征明显"**。

## 2. 主营业务

### 2.1 公司简介

中国银行是中国四大国有商业银行之一，也是中国国际化程度最高的银行，在全球60多个国家和地区设有分支机构。业务涵盖商业银行、投资银行、保险、直接投资、证券、基金等综合金融服务。

### 2.2 主营业务构成

1. **公司金融**：对公贷款、贸易融资、现金管理、供应链金融等
2. **个人金融**：个人贷款（含按揭）、存款、信用卡、财富管理
3. **金融市场**：外汇交易、利率衍生品、债券投资、做市业务
4. **其他/全球化业务**：海外分行贡献约25%资产，是中行区别于其他大行的核心特征

### 2.3 核心收入来源

净利息收入占经营收入的66.8%（RMB 4,407亿），非利息收入占33.2%（RMB 2,192亿）。中行的国际化业务使其外币资产占比较高，NIM受外币市场利率影响较大。非利息收入中包含较多金融市场交易收入和汇兑收益。

## 4. 营收分布和增长

### 4.1 按收入类型划分

| 收入项 | 2025年 | 2024年 | 同比增速 | 占比 | 备注 |
|---|---:|---:|---:|---:|---|
| 净利息收入 | RMB 4,407亿 | RMB 4,489亿 | -1.8% | 66.8% | |
| 非利息收入 | RMB 2,176亿 | RMB 1,812亿 | +20.1% | 33.2% | ⭐ |

### 4.3 营收分析

**非利息收入+20.1%是核心亮点**

非利息收入大幅增长得益于：（1）金融市场交易收入增长（资本市场回暖+债券交易）；（2）汇兑收益（中行外汇业务优势）；（3）代理业务手续费增长。非息收入占比从28.7%提升至33.2%，收入结构优化。

**净利息收入-1.8%——量增价跌**

利息收入-6.6%（资产端收益率下行49bps），但利息支出-10.0%（存款付息率下行37bps）。以量补价——客户贷款+8.7%，但NIM-14bps拖累了净利息收入。

## 6. 未来可能的利好

### 利好 1：非利息收入+20.1%——收入多元化

**依据**

非息收入RMB 2,176亿（+20.1%），占比从28.7%提升至33.2%。

**解读**

中行的国际化布局使其在外汇交易、跨境结算等领域具有天然优势。非息收入的强劲增长有助于对冲NIM下行压力，是大行中收入多元化最突出的。

### 利好 2：负债端降本——利息支出-10.0%

**依据**

利息支出RMB 5,602亿（-10.0%），存款付息率下降37bps。

**解读**

利息支出降幅远超利息收入降幅（-6.6%），说明负债端成本管控有效。存款重定价效应仍在持续，有望继续释放。

### 利好 3：成本收入比改善——27.84%

**依据**

成本收入比27.84%（-0.93ppt），在大行中处于较优水平。

**解读**

虽然经营支出+8.8%增速偏快，但A股口径下成本收入比仍下降了0.93ppt，说明收入端的增长更快（非息大增带动）。

### 利好 4：国际化业务——独特优势

**依据**

中行海外资产占比约25%，是中国最国际化的银行。全球60多个国家和地区设有机构。

**解读**

人民币国际化和一带一路推进将持续利好中行的跨境业务。外汇清算、贸易融资等业务中行具有不可替代的地位。

## 7. 未来可能的利空

### 利空 1：NIM持续收窄——1.26%是四大行偏低水平

**依据**

NIM 1.26%（-14bps），低于工行、建行、农行。NIM从2021年持续下降。

**解读**

中行NIM偏低部分因外币资产收益率低于人民币资产。如果外币市场利率进一步下行（美联储降息），外币NIM将继续承压。LPR持续下调也将拖累人民币NIM。

### 利空 2：经营支出+8.8%——增速远超收入

**依据**

经营支出RMB 2,566亿（+8.8%），而经营收入仅+4.3%。

**解读**

费用增速约为收入增速的2倍，这是不可持续的。虽然A股口径成本收入比仍在改善，但etnet口径下经营支出增速过快可能反映了IT投入、合规成本或人力成本上升。如果费用增长不能带来对应的收入回报，将侵蚀利润。

### 利空 3：其他减值回拨-77.4%——不可持续

**依据**

其他减值回拨从+RMB 181亿大幅收窄至+RMB 41亿。

**解读**

2024年的高额回拨是一次性的，2025年已大幅正常化。这意味着2026年此项目难以继续贡献利润增量。

### 利空 4：ROE偏低——8.94%

**依据**

ROAE 8.94%，在大行中处于中等偏低水平。

**解读**

较低的ROE限制了内生资本增长能力。如果要维持当前贷款增速（+8.7%），可能需要外部资本补充（如永续债/配股），进一步稀释股东回报。

## 8. 总结判断

**一句话结论**

中国银行2025年收入+4.5%至RMB 6,583亿、归母净利+2.2%至RMB 2,430亿——非利息收入+20.1%是核心亮点（占比提升至33%），但NIM -14bps至1.26%（四大行偏低）、经营支出+8.8%增速偏快、ROE 8.94%偏低——"国际化大行的低NIM高非息特征，收入结构优化但盈利能力有待提升"。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| NII | RMB 4,407亿（-1.8%） | NIM承压 |
| 非息收入 | RMB 2,176亿（+20.1%） | ⭐核心亮点 |
| NIM | 1.26%（-14bps） | ⚠️偏低 |
| 经营支出 | RMB 2,566亿（+8.8%） | ⚠️增速快 |
| 贷款减值 | RMB 1,072亿（-11.3%） | ✅拨备释放 |
| ROAE | 8.94% | 偏低 |
| P/B | HK$5.13/NBV RMB 8.67≈HK$9.54，约0.54x | 深度破净 |
| P/E | HK$5.13/EPS RMB 0.74≈HK$0.81，约6.3x | |
| 股息率 | DPS RMB 0.2263≈HK$0.249/HK$5.13=4.9% | |

**核心看点**

1. **NIM何时见底**：1.26%是否有进一步下行空间
2. **非息收入增长可持续性**：+20.1%能否延续
3. **费用增速能否放缓**：+8.8%远超收入增速
4. **国际化业务贡献**：人民币国际化红利

**核心风险**

1. **NIM持续收窄**：LPR下调+外币利率下行
2. **经营支出增速过快**：+8.8%不可持续
3. **ROE偏低**：8.94%限制内生增长
4. **国际地缘政治风险**：海外业务面临制裁/合规风险
