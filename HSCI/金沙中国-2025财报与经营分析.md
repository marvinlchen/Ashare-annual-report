# 金沙中国（01928.HK）2025年财报整理

> 注：金沙中国有限公司以美元为报告货币。母公司为Las Vegas Sands Corp.（LVS），持股约74.80%。

## 1. 业绩总结

### 1.1 收入与利润表现

金沙中国2025年度实现净收益总额74.4亿美元，同比 +5.1%（2024年~70.8亿美元）。年度净利润8.96亿美元，同比 -14.3%（2024年~10.45亿美元）。经调整物业EBITDA 23.1亿美元（-0.7%）。Q4表现显著改善——Q4净收益20.5亿美元（+16.4%），EBITDA 6.08亿美元。经营开支62.1亿美元（+8.7%），其中娱乐场开支增加3.11亿美元是利润下滑的主因。经营活动现金净额21.1亿美元。年末现金15.05亿美元。建议派发末期股息每股0.50港元。

### 1.2 增长与拖累因素

核心驱动：(1) 澳门游客超4,000万人次，旅游消费持续恢复；(2) 伦敦人二期（The Londoner Phase 2）扩张完成——Q4伦敦人娱乐场收入+35.4%，客房收入+54.41%，带动Q4整体收入+16.4%；(3) 总净收益+5.1%体现量的增长。主要拖累：(1) 经营开支+8.7%（+5亿美元），远超收入增速——娱乐场开支+3.11亿（伦敦人新增产能的运营成本）、企业开支+0.4亿、折旧+0.38亿；(2) 市场竞争加剧——澳门6大博彩运营商均加大投入争夺市场份额；(3) 澳门金沙娱乐场收入-12.82%。

### 1.3 趋势判断

金沙中国处于"大规模投资→产能爬坡→利润释放"的过渡期。伦敦人二期的Q4爆发（+35%+）暗示2026年将迎来完整年度的产能贡献。利好方向：伦敦人二期全面运营+澳门旅游复苏+非博彩收入增长；压力方向：竞争加剧推高费用+净利润率下滑+高资本开支持续。

## 2. 主营业务

### 2.1 公司简介

金沙中国是澳门领先的综合度假村开发和运营商，旗下运营五大物业——威尼斯人、伦敦人、巴黎人、百丽宫/四季及澳门金沙，涵盖博彩、酒店、零售、餐饮、会展（MICE）等业务。

### 2.2 主营业务构成

1. **博彩/娱乐场（核心）**：贵宾厅和中场博彩，收入主力
2. **酒店客房**：旗下约13,000+间客房（含伦敦人二期新增）
3. **购物中心/零售**：金沙购物城邦等大型零售Mall
4. **餐饮及会展**：餐厅、MICE设施
5. **娱乐及其他**：演出、展览等非博彩内容

### 2.3 核心收入来源

博彩收入为绝对主力（占比~75%+），酒店/零售/餐饮/会展等非博彩收入占比约25%。伦敦人和威尼斯人为两大旗舰物业。

## 4. 营收分布和增长

### 4.1 按物业划分（以Q4数据为参考）

伦敦人成为增长最快的物业（Q4娱乐场+35.4%，客房+54.4%），威尼斯人保持稳健（+9.36%），澳门金沙表现疲弱（-12.82%）。全年物业级别收入分布年报未单独披露完整数据。

### 4.3 营收分析

**主要增长来源**

(1) 伦敦人二期扩张完成——新增约2,400间高端客房+赌场空间，Q4贡献爆发式增长；(2) 澳门旅游复苏——全年游客超4,000万人次；(3) 非博彩收入（酒店、零售、餐饮、会展）受益于旅游消费增长。

**主要拖累因素**

(1) 澳门金沙（半岛物业）收入-12.82%，可能受益于客流向路氹新物业转移和市场竞争加剧；(2) 上半年伦敦人二期尚在爬坡期，全年平均贡献低于Q4水平。

## 5. 利润分布和增长

### 5.1 整体利润表现

| 指标 | 2025年 | 2024年 | 同比增速 | 备注 |
|---|---:|---:|---:|---|
| 净收益 | 74.4亿美元 | ~70.8亿美元 | +5.1% | |
| 经营开支 | 62.1亿美元 | ~57.1亿美元 | +8.7% | |
| 净利润 | 8.96亿美元 | ~10.45亿美元 | -14.3% | |
| EBITDA | 23.1亿美元 | ~23.3亿美元 | -0.7% | |

### 5.3 利润分析

**增收不增利的核心原因**

收入+5.1%但费用+8.7%——费用增速是收入增速的1.7倍。娱乐场开支+3.11亿美元（伦敦人新物业的运营成本、市场推广费用增加）、企业费用+0.4亿、折旧+0.38亿。这是典型的"投资期+产能爬坡期"特征——收入尚未完全释放但成本已全额计入。

**Q4利润回升信号**

Q4 EBITDA 6.08亿美元，年化约24.3亿——高于全年23.1亿。说明伦敦人二期进入全面运营后，利润率正在恢复。2026年有望实现EBITDA的正增长。

## 6. 未来可能的利好

### 利好 1：伦敦人二期全面运营——2026年利润释放

**依据**

伦敦人二期Q4娱乐场+35.4%、客房+54.4%。至2025年5月约2,405间客房全部投入使用。

**解读**

伦敦人二期的全面运营意味着2026年将是完整年度的产能贡献——预计将带动集团净收益增长10%+，更重要的是边际成本递减（固定成本已在2025年一次性计入），EBITDA利润率将恢复上行。

### 利好 2：澳门旅游持续复苏——游客超4,000万

**依据**

2025年澳门游客超4,000万人次。大湾区基础设施持续完善。

**解读**

澳门旅游业仍在向疫情前峰值（2019年3,940万，2025年已超越）靠拢和突破。大湾区互联互通、内地消费升级和国际旅游恢复将持续为澳门带来客流。金沙中国作为路氹最大的综合度假村运营商，直接受益。

### 利好 3：非博彩收入增长——多元化降低周期性

**依据**

伦敦人客房收入+54.4%，餐饮+54.55%。金沙购物城邦零售持续增长。MICE设施扩展。

**解读**

澳门政府鼓励博彩业向"非博彩"转型。金沙中国在酒店、零售、餐饮、会展方面拥有澳门最大的非博彩基础设施。非博彩收入的增长不仅降低了对博彩收入的依赖，还提供了更稳定的利润率。

## 7. 未来可能的利空

### 利空 1：竞争加剧——6大运营商同步扩张

**依据**

经营开支+8.7%（远超收入+5.1%），娱乐场开支+3.11亿。澳门6家博彩运营商均在加大投入。

**解读**

新博彩合约（十年期，至2032年）要求各运营商大规模投资非博彩设施，导致全行业资本开支和营销费用急剧上升。在市场总量增长有限的情况下，费用竞争可能持续压缩利润率——"更多投入不一定带来更多利润"。

### 利空 2：净利润率下滑——增收不增利

**依据**

净利润率从~14.8%降至12.0%（-2.8pp）。EBITDA利润率从~32.9%降至31.0%。

**解读**

伦敦人二期的产能爬坡期是2025年利润率下降的主因，但如果2026年利润率未能恢复，可能反映的是结构性成本压力（竞争加剧推高营销费用、人工成本上升、合规成本增加等）而非暂时性因素。

### 利空 3：高资本开支——计划至2032年再投$45亿

**依据**

金沙中国计划至2032年再投入约45亿美元用于澳门资本及运营投资。

**解读**

45亿美元的长期投资承诺（年均约6.4亿/年）将持续消耗自由现金流，限制分红和股东回报空间。如果投资回报不及预期（例如非博彩设施利用率低），将造成资本浪费。

---

## 8. 总结判断

**一句话结论**

金沙中国2025年收入+5.1%但净利润-14.3%，处于伦敦人二期"投资→爬坡→释放"的过渡期；Q4爆发增长（+16.4%）暗示2026年将迎来利润拐点，澳门旅游持续复苏（游客超4,000万）是核心支撑。

**核心看点**

1. 伦敦人Q4爆发——娱乐场+35.4%、客房+54.4%，2026年全年释放
2. 澳门游客超4,000万——旅游复苏突破疫情前峰值
3. Q4 EBITDA 6.08亿（年化24.3亿>全年23.1亿）——利润率恢复中
4. 经营现金流21.1亿——现金生成能力强劲

**核心风险**

1. 增收不增利——经营费用+8.7%远超收入+5.1%
2. 6大运营商竞争——费用战可能持续
3. 至2032年再投$45亿——高资本开支压制自由现金流
