# 中金公司（03908.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

中金公司2025年实现营业收入RMB 284.81亿元（A股口径），同比增长33.5%；IFRS口径收入及其他收益总额约RMB 405.18亿（+29.0%）。归母净利润RMB 97.91亿（+71.9%），扣非净利润RMB 92.05亿（+75.6%）。H股口径扣除永续债利息后股东应占溢利RMB 90.54亿（+81.1%）。EPS RMB 1.88（+80.8%）。全年合计派现每10股RMB 3.20（含中期RMB 0.90+末期RMB 2.30），累计派现超RMB 111亿。

### 1.2 增长与拖累因素

增长驱动：各业务板块全面发力——股票业务+65.5%（全球交易能力提升）、投资银行+62.6%（引领境内外重大资本运作，IPO 56单排名第一）、财富管理+35.9%（买方投顾转型见效）、资产管理+30.8%（规模突破）。净利润增速（+71.9%）远超营收增速（+33.5%），体现费用管控效率提升和高毛利业务占比优化——毛利率40.75%（+8.62ppt），净利率34.41%（+7.81ppt）。拖累较少，主要为研发费用+42.2%（数字化投入加大）、其他公平值/减值RMB -3.23亿（+235%），但绝对值不大。

### 1.3 趋势判断

中金公司2025年业绩实现强劲V型复苏（2023-2024年为低谷），受益于资本市场回暖和全业务线协同。整体判断：**"周期性强复苏之年，全业务线高增长+费用管控改善推动利润弹性远超收入弹性——但券商盈利高度依赖市场环境，2026年能否持续取决于资本市场景气度"**。

## 2. 主营业务

### 2.1 公司简介

中金公司（CICC）是中国领先的综合性投资银行，提供投资银行、股票交易、固定收益、资产管理、财富管理和私募股权等全牌照金融服务，具有显著的跨境业务优势。

### 2.2 主营业务构成

1. **财富管理**：买方投顾转型中，2025年收入RMB 94.89亿（占营收33.3%，+35.9%），是第一大收入来源
2. **股票业务**：全球交易服务，2025年收入RMB 73.45亿（占25.8%，+65.5%），增速最快
3. **投资银行**：IPO、债券承销、并购，2025年收入RMB 50.31亿（+62.6%），IPO 56单排名第一、债券承销RMB 8,250亿（+16.1%）、并购64宗/$820.81亿排名第一
4. **固定收益**：债券交易及做市，收入占比13.4%
5. **资产管理**：公募/私募/专户，收入RMB 15.82亿（+30.8%）
6. **私募股权**：股权投资，收入占比3.9%

### 2.3 核心收入来源

财富管理+股票业务+投资银行合计贡献约77%的营收。中金的核心竞争优势在于：（1）投行领域的头部地位（IPO和并购均#1）；（2）跨境业务能力（中资企业全球IPO融资$267.83亿）；（3）财富管理的高净值客户基础。

### 5.2 利润弹性分析——净利增速远超营收增速

**收入→PBT利润桥**

净利润增速（+71.9%）远超营收增速（+33.5%），核心驱动因素：

| 因素 | 说明 |
|---|---|
| ✅收入增长 | 全业务线高增长，营收+33.5% |
| ✅毛利率大幅提升 | +8.62ppt至40.75%，高毛利业务占比提升 |
| ✅费用管控 | 销售费用+28.7%、管理费用+21.4%，均低于营收增速 |
| ✅财务费用改善 | 财务费用RMB -8.23亿（净收入），同比减亏12.17亿 |
| ✅净财务支出-25.3% | 资金管理优化 |
| ⚠️研发费用+42.2% | 高于营收增速，数字化投入 |

**利润弹性的核心原因**

券商是高经营杠杆行业——固定成本（人力、IT、租金）比例高，收入增长时边际利润率大幅提升。2025年毛利率从32.1%升至40.8%，完美体现了这一特征。反面意味着市场下行时利润下滑也会更快。

## 6. 未来可能的利好

### 利好 1：投行头部地位稳固——IPO和并购双双第一

**依据**

2025年中资企业全球IPO 56单（$267.83亿，#1）、并购64宗（$820.81亿，#1）、债券承销RMB 8,250亿（+16.1%）。

**解读**

投行是券商收入质量最高的业务之一（手续费收入，不承担资本风险）。中金在IPO和并购领域的绝对领先地位具有较强的护城河，尤其是跨境业务能力在中资券商中独一无二。

### 利好 2：费用管控+高经营杠杆——利润弹性巨大

**依据**

毛利率+8.62ppt至40.75%，净利率+7.81ppt至34.41%。销售/管理费用增速均低于营收增速。财务费用改善RMB 12.17亿。

**解读**

高经营杠杆特征意味着如果2026年资本市场维持景气，利润率仍有进一步提升空间。同时费用端展现了较好的管控纪律。

### 利好 3：数字化转型投入——RITAS平台+中金点睛大模型

**依据**

研发费用RMB 18.96亿（+42.2%），研发人员1,286人（占11.3%），重点投向RITAS数字化平台和中金点睛AI大模型。

**解读**

金融科技投入有望提升运营效率和客户服务精准度，中长期降低边际服务成本，提高ROE。

### 利好 4：财富管理买方转型——长期增长第二曲线

**依据**

财富管理收入RMB 94.89亿（+35.9%），是第一大收入来源。买方投顾转型持续推进。

**解读**

财富管理具有客户粘性强、收入稳定性好的特征，是从交易型券商向综合金融服务商转型的关键。

## 7. 未来可能的利空

### 利空 1：高度依赖市场周期——交易型收入占比上升

**依据**

股票业务占比从约21%提升至25.8%（+65.5%）。券商行业PBT与资本市场交投量/指数水平高度相关。2023-2024年市场低迷时净利润已大幅下滑。

**解读**

2025年的高增长部分来自2024年的低基数效应和市场回暖。如果2026年资本市场转冷，交易型收入将首当其冲，利润可能大幅回落。高经营杠杆是双刃剑。

### 利空 2：债务短期化——短债占比从47%升至67%

**依据**

短期借贷+47.1%至RMB 1,466亿，长期借贷-34.5%至RMB 724亿，短债占比从47.4%升至66.9%。现金/短债覆盖率0.42倍。

**解读**

虽然券商短期借贷多为回购/同业拆借（可滚动续期），但在市场流动性收紧时可能面临再融资成本上升或续期困难的风险。

### 利空 3：监管与合规风险——跨境业务+ESG要求趋严

**依据**

公司年报提示跨境业务合规风险、境内外监管政策趋严、ESG相关监管要求提升。

**解读**

中金的跨境业务优势也意味着更大的合规暴露面。中美关系变化、数据安全法规、ESG披露要求等外部环境变化可能增加合规成本或限制业务发展。

### 利空 4：人才竞争与薪酬压力

**依据**

金融行业人才竞争激烈，核心高管薪酬增长28-35%。研发人员占比提升。

**解读**

券商核心竞争力在人，人才流失风险是持续性问题。近年来头部券商间人才争夺加剧，薪酬成本可能持续上升。

## 8. 总结判断

**一句话结论**

中金公司2025年业绩强劲复苏，收入+33.5%、净利润+71.9%（A股口径），全业务线高增长+费用管控改善推动利润弹性远超收入——IPO和并购双双市场第一，毛利率+8.6ppt至40.8%，经营现金流+45.7%至156亿，但高度依赖市场周期（股票业务占比升至26%）、债务短期化（短债占比67%）是主要风险。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| PBT | RMB 117.13亿（+72.1%） | 强劲复苏 |
| 股东应占溢利 | RMB 90.54亿（+81.1%） | 扣永续债利息后 |
| 毛利率 | 40.75%（+8.62ppt） | 盈利能力大幅提升 |
| 经营现金流 | RMB 156.28亿（+45.7%） | ✅造血能力强 |
| 短债占比 | 66.9%（+19.5ppt） | ⚠️债务短期化 |
| P/B | HK$19.26/NBV RMB 20.71≈0.93x | 略低于1倍 |
| 派息 | RMB 0.32（+77.8%） | 派息比率17% |

**核心看点**

1. **2026年资本市场景气度**：直接决定业绩能否持续高增长
