# 中国东方集团 2025 年财报整理

> 中國東方集團控股有限公司（股票代码：0581.HK），主要从事钢铁产品制造及销售、钢铁产品/铁矿石及相关原材料贸易、电力设备销售和房地产业务。
> 本报告基于公司2025年12月31日止年度全年业绩公告，币种为人民币（除特别注明外）。

# 【中国东方集团】2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

中国东方集团2025年总收入人民币404.13亿元，同比 -5.9%；年内溢利（净利润）人民币2.92亿元，同比 +30.2%；股东应占溢利约人民币2.24亿元，同比约 +50%。每股基本盈利人民币0.06元。

- 总收入：人民币404.13亿元，同比 -5.9%（2024年：人民币429.57亿元）
- 年内溢利：人民币2.92亿元，同比 +30.2%（2024年：人民币2.25亿元）
- 股东应占溢利：约人民币2.24亿元，同比约 +50%（2024年：人民币1.49亿元）
- 每股基本盈利：人民币0.06元（2024年：人民币0.04元）

### 1.2 增长与拖累因素

收入下降但利润逆势增长，核心驱动力是"成本降幅大于售价降幅"。铁矿石等主要原材料价格大幅回落，叠加精益管理降本增效，推动毛利改善。

- 主要增长驱动：原材料成本下降快于钢铁产品售价下降、精益管理降本增效、产销量回升
- 主要拖累因素：钢铁行业下游需求持续低迷、钢铁产品平均售价持续下跌
- 净利率从约0.52%提升至约0.72%，利润率仍处于极低水平

### 1.3 趋势判断

2023年亏损 → 2024年扭亏（+1.49亿） → 2025年利润增长（+2.24亿），盈利修复趋势确立。但收入连续两年下滑（2024年 -7.1%，2025年 -5.9%），反映行业需求仍未见底。

- 未来主要利好：原材料成本端持续改善空间、钢铁行业供给侧改革和产能出清
- 未来主要压力：钢铁产品价格持续下行、房地产下游需求疲弱、行业产能过剩
- 整体趋势判断：盈利在低基数上修复，但行业结构性压力未消，低利润率使业绩对价格波动高度敏感

## 2. 主营业务

### 2.1 公司简介

中国东方集团是一家以钢铁为主业的综合性企业集团，总部位于河北省唐山市，主要从事钢铁产品制造、钢铁及铁矿石贸易、电力设备销售和房地产开发。主要钢铁产品包括H型钢、带钢及带钢类产品、冷轧板、镀锌板、钢坯和螺纹钢等。

### 2.2 主营业务构成

1. **钢铁产品制造及销售**：自产H型钢、冷轧板、镀锌板、带钢、钢坯等，占H1收入约58%
2. **钢铁产品/铁矿石及原材料贸易**：钢铁产品和铁矿石等原材料贸易，占H1收入约33%
3. **电力设备销售**：电力设备产品销售，占H1收入约9%
4. **房地产**：唐山、苏州及澳大利亚悉尼房地产项目，占H1收入约0.1%

### 2.3 核心收入来源

钢铁制造和贸易合计贡献超过91%的收入，是绝对核心业务。公司位于唐山——中国最大的钢铁生产基地，区位优势明显但也面临同质化竞争。

### 4.3 营收分析

**收入下降原因**

(1) 钢铁产品平均售价持续下跌，H1售价同比 -14.5%至2,998元/吨；(2) 下游需求持续疲弱，尤其房地产和基建领域；(3) 收入已连续两年下滑（2024年 -7.1%，2025年 -5.9%）。

**结构特征**

**季度节奏**

H1收入约199亿（占全年49%），H2收入约205亿（占全年51%）。但H1净利润2.43亿（占全年83%），H2利润仅约0.49亿，利润分布严重前高后低。

## 5. 利润分布和增长

### 5.1 整体利润表现

| 指标 | 2025年 | 2024年 | 同比增速 | 备注 |
|---|---:|---:|---:|---|
| 年内溢利 | 人民币2.92亿 | 人民币2.25亿 | 同比 +30.2% | |
| 股东应占溢利 | 约人民币2.24亿 | 人民币1.49亿 | 同比约 +50% | |
| H1毛利 | 人民币9.38亿 | 人民币6.43亿 | 同比 +45.9% | |
| 净利率 | 0.72% | 0.52% | +0.2pct | |

### 5.2 利润分析

**利润增长驱动**

(1) 原材料成本（铁矿石、焦炭等）价格大幅回落，降幅快于钢铁产品售价降幅，推动吨钢毛利改善。Q3每吨毛利约200-250元。(2) 精益管理策略持续推进，降本增效、优化管理及技术改造。(3) 钢铁产品产销量按年回升。

**利润压力因素**

(1) 净利率仅0.72%，极低的利润率意味着微小的价格波动就能导致亏损。(2) H2利润仅约0.49亿，较H1的2.43亿大幅下滑——可能源于H2原材料成本反弹或钢铁价格进一步下跌。(3) 少数股东权益占净利润约23%（2.92亿-2.24亿=0.68亿），摊薄了归母利润。

**盈利修复路径**

2023年亏损1.6亿 → 2024年盈利1.49亿（扭亏） → 2025年盈利2.24亿（+50%），三年修复趋势明确。但修复主要依赖成本端改善而非需求回暖。

## 6. 未来可能的利好

### 利好 1：原材料成本改善空间持续存在

**依据**

2025年利润增长核心驱动力是铁矿石等原材料价格下降快于钢铁产品售价。全球铁矿石供给增加（澳洲、巴西产量持续扩张），需求端因中国房地产调整而疲弱。

**解读**

若铁矿石价格继续下行或维持低位，公司吨钢成本将进一步优化。作为唐山地区的中型钢企，原材料成本占收入比极高，成本端每降1%对利润的杠杆效应显著。

### 利好 2：钢铁行业供给侧改革和产能出清

**依据**

中国钢铁行业已进入产能过剩和淘汰落后产能阶段。政策持续推动行业整合、兼并重组和环保限产，中小钢企面临淘汰。

**解读**

若行业产能有效出清，供需格局将改善，钢铁产品价格有望企稳回升。中国东方集团作为存续企业，市场份额可能提升。

### 利好 3：派息稳定，股东回报持续

**依据**

2023-2025年连续三年派息（0.05、0.06、0.07港元/股），且持续派发特别股息。2025年总派息0.07港元，较2024年增长17%。

**解读**

在行业低迷期仍能保持甚至增加派息，显示公司现金流管理较为稳健，有利于维护股价和吸引价值投资者。

## 7. 未来可能的利空

### 利空 1：钢铁产品价格持续下行风险

**依据**

H1钢铁平均售价2,998元/吨，同比 -14.5%。收入连续两年下滑（2024年 -7.1%，2025年 -5.9%）。下游房地产、基建需求持续低迷。

**解读**

净利率仅0.72%，几乎没有安全边际。若钢铁产品售价再降2-3%而成本未能同步下降，公司将再次陷入亏损。H2利润仅0.49亿已显示出盈利脆弱性。

### 利空 2：行业产能过剩和同质化竞争

**依据**

唐山是中国最大的钢铁生产基地，区域内钢企众多，产品同质化严重。公司主要产品H型钢、冷轧板、镀锌板等均为通用型钢铁产品，议价能力有限。

**解读**

在需求收缩期，同质化竞争将加剧价格战，进一步压缩本已微薄的利润空间。缺乏差异化产品和高附加值钢种是长期竞争劣势。

### 利空 3：房地产下游需求结构性萎缩

**依据**

中国房地产行业深度调整持续，新开工面积和钢铁需求持续萎缩。建筑用钢（H型钢、螺纹钢）是公司核心产品，直接受到影响。

**解读**

房地产行业的结构性调整可能持续数年，相关钢铁需求难以回到历史高点。虽然基建和制造业提供部分替代需求，但难以完全弥补房地产需求的缺口。

## 8. 总结判断

**一句话结论**

中国东方集团2025年"降收增利"——收入-5.9%但净利+30%，盈利修复主要依赖原材料降价而非需求回暖，0.72%的极低净利率使业绩对价格波动高度敏感。

**核心看点**

1. 盈利三年修复路径：2023年亏损 → 2024年扭亏 → 2025年利润+50%
2. 原材料成本改善是核心驱动力，管理优化和产销量回升为辅
3. 持续派息：2025年合计派息0.07港元（末期息0.02+特别股息0.05）
4. H1利润占全年83%，盈利节奏前高后低

**核心风险**

1. 净利率0.72%，几乎没有安全边际，售价波动2-3%即可导致亏损
2. 收入连续两年下滑（-7.1%、-5.9%），需求端尚未见底
3. 产品同质化严重，唐山区域竞争激烈，议价能力弱
4. 房地产下游需求结构性萎缩，建筑用钢长期承压
