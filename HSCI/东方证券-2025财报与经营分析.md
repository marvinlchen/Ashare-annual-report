# 东方证券（03958.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

东方证券2025年实现收入及其他收益RMB 254.77亿（A股口径，+23.3%）；IFRS口径收入RMB 181.16亿（+16.5%）。归母净利润RMB 56.34亿（A股口径，+68.2%）；H股口径普通股股东应占溢利RMB 54.55亿（+75.3%，扣除优先股/永续债利息后）。PBT RMB 66.44亿（+81.6%）。加权平均ROE 6.99%（+2.85ppt）。EPS RMB 0.65（+75.7%）。DPS RMB 0.32（中期+末期合计，+82.9%），派息比率49.2%。连续17年获评A类券商、连续5年AA级。

### 1.2 增长与拖累因素

增长驱动：（1）财富及资产管理收入RMB 134.73亿占52%，是第一大收入来源；（2）机构及销售交易RMB 88.28亿（34%）受益于市场交投活跃；（3）其他公平值/减值损失大幅缩窄58%至-RMB 2.84亿（2024年-RMB 6.76亿）；（4）联营公司盈利+65.7%至RMB 7.59亿（核心联营为汇添富基金）。拖累方面，有效税率从8.4%大幅升至15.2%（+6.8ppt），税项+227%至RMB 10.10亿。

### 1.3 趋势判断

东方证券2025年业绩大幅复苏，与中金公司同为受益于资本市场回暖的券商。整体判断：**"周期性强复苏+减值收窄双重推动——财富管理（含汇添富基金）是核心优势，但券商盈利高度依赖市场环境，负经营现金流和税率跳升值得关注"**。

## 2. 主营业务

### 2.1 公司简介

东方证券是上海国资背景的综合性证券公司（A+H上市），业务涵盖财富管理、资产管理、机构销售交易、投行和国际业务，旗下汇添富基金是中国领先的公募基金公司。

### 2.2 主营业务构成

1. **财富及资产管理**：2025年收入RMB 134.73亿（占52%），包含经纪、融资融券、资管和汇添富基金
2. **机构及销售交易**：2025年收入RMB 88.28亿（占34%），包含自营投资和做市交易
3. **国际及其他**：2025年收入RMB 21.18亿（占8%），含东方证券（香港）
4. **投行及另类投资**：2025年收入RMB 15.59亿（占6%），含IPO、债券承销和私募投资

### 2.3 核心收入来源

财富及资产管理是最大且最稳定的收入来源，其中汇添富基金是核心盈利引擎（联营公司贡献RMB 7.59亿利润，+65.7%）。机构及销售交易是第二大但波动性较大的收入源。

## 4. 营收分布和增长

### 4.1 按业务划分

| 业务板块 | 2025年收入 | 收入占比 | 备注 |
|---|---:|---:|---|
| 财富及资产管理 | RMB 134.73亿 | 52.9% | 第一大分部 |
| 机构及销售交易 | RMB 88.28亿 | 34.6% | 受益市场交投 |
| 国际及其他 | RMB 21.18亿 | 8.3% | |
| 投行及另类投资 | RMB 15.59亿 | 6.1% | |
| **合计** | **RMB 254.77亿** | **100%** | **A股口径** |

### 4.3 营收分析

**主要增长来源**

全业务线增长，受益于2025年A股和港股市场回暖。财富及资产管理（52.9%）是最大收入来源，受益于汇添富基金规模增长和经纪/融资融券业务放量。机构及销售交易受益于自营投资收益和做市业务。

**结构特点**

财富管理+资管的占比超50%，是东方证券最具竞争力的业务线（汇添富基金是核心壁垒）。相比中金公司的投行优势，东方证券更偏向资管和财富管理。

### 5.2 利润增长分析——减值收窄和联营改善是PBT超越收入增速的关键

**PBT增量分解**

| 因素 | 影响 | 说明 |
|---|---:|---|
| 收入增长 | 基础推动 | +16.5%（IFRS） |
| ✅减值损失大幅缩窄 | +RMB 3.92亿 | -6.76亿→-2.84亿（-58%） |
| ✅联营公司盈利改善 | +RMB 3.01亿 | +4.58亿→+7.59亿（+66%） |
| ⚠️税率跳升 | -RMB 7.01亿 | 有效税率8.4%→15.2%（+227%） |
| 永续债利息减少 | +RMB 0.59亿 | 赎回RMB 20亿永续债 |

PBT +81.6%远超收入增速+16.5%，核心驱动是减值损失缩窄（+RMB 3.92亿）和联营公司改善（+RMB 3.01亿），合计贡献约RMB 6.93亿的利润增量，加上收入增长带来的经营杠杆效应。

⚠️但税率从8.4%跳升至15.2%部分抵消了PBT的高增长，导致净利润增速（+68.2%）低于PBT增速（+81.6%）。

## 6. 未来可能的利好

### 利好 1：汇添富基金——核心盈利引擎

**依据**

联营公司（主要为汇添富基金）贡献RMB 7.59亿利润（+65.7%）。汇添富是中国Top 10公募基金，管理规模超万亿。

**解读**

公募基金行业受益于居民财富向权益类资产转移的长期趋势。汇添富的品牌和规模优势为东方证券提供稳定的利润来源。

### 利好 2：财富管理+资管——收入超50%的核心竞争力

**依据**

财富及资产管理收入RMB 134.73亿占52.9%，是行业中财富管理占比最高的券商之一。

**解读**

财富管理具有客户粘性强、收入可预测性好的特征。买方投顾转型和公募基金费率改革长期利好大型资管平台。

### 利好 3：派息比率近50%——高分红回报

**依据**

DPS RMB 0.32（+82.9%），派息比率49.2%。2025年累计分红RMB 26.99亿，占归母净利47.9%。

**解读**

近50%的派息比率在券商行业属于较高水平，反映管理层对盈利持续性的信心和回馈股东的意愿。

## 7. 未来可能的利空

### 利空 1：高度依赖市场周期——券商固有属性

**依据**

机构及销售交易收入占34%，受市场交投量和指数波动直接影响。2022-2024年市场低迷时盈利承压。

**解读**

与中金公司类似，2025年高增长部分来自低基数和市场回暖。如果2026年资本市场转冷，利润可能大幅回落。

### 利空 2：有效税率跳升——8.4%→15.2%

**依据**

税项+227%至RMB 10.10亿，有效税率从8.4%大幅升至15.2%。

**解读**

2024年的8.4%低税率可能有一次性税收优惠/递延税抵扣，2025年回归正常化。如果税率进一步向名义25%靠拢，将持续侵蚀净利润。

### 利空 3：经营现金流由正转负

**依据**

每股现金流从RMB +2.686转为-RMB 1.973。流动资产净值从-RMB 87.70亿恶化至-RMB 315.34亿。

**解读**

负经营现金流可能因业务扩张（金融资产+19.5%）占用了大量资金。短期内现金RMB 1,120亿仍充裕，但持续的负现金流需要依赖外部融资。

### 利空 4：应收帐款+154%——信用风险

**依据**

应收贸易帐款从RMB 9.73亿增至RMB 24.72亿（+154%）。

**解读**

应收帐款大增可能反映融资融券/客户交易结算的增加，也可能意味着回款速度变慢。需关注潜在信用风险。

## 8. 总结判断

**一句话结论**

东方证券2025年PBT +81.6%至RMB 66.44亿、普通股股东溢利+75.3%至RMB 54.55亿，受益于收入增长（+16.5%）、减值缩窄（-58%）和汇添富基金利润+66%——财富管理+资管占比超52%是核心竞争优势，但税率跳升（8.4%→15.2%）和负经营现金流需关注，券商盈利高度依赖市场周期。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| PBT | RMB 66.44亿（+81.6%） | 强劲复苏 |
| 联营盈利 | RMB 7.59亿（+65.7%） | 汇添富基金 |
| 减值损失 | -RMB 2.84亿（-58%） | ✅大幅改善 |
| 有效税率 | 15.2%（+6.8ppt） | ⚠️跳升 |
| 每股现金流 | -RMB 1.973 | ⚠️由正转负 |
| P/B | HK$5.97 / NBV RMB 9.45≈HK$10.40，约0.57x | 深度折价 |
| 派息比率 | 49.2% | ✅高分红 |

**核心看点**

1. **2026年资本市场景气度**：直接决定业绩持续性
2. **汇添富基金规模和利润增长**：核心盈利引擎
3. **有效税率正常化水平**：15%是否为新常态
4. **财富管理买方转型进展**：能否提高收入稳定性

**核心风险**

1. **市场周期性风险**：券商盈利弹性大、波动也大
2. **税率跳升**：从8.4%到15.2%，可能继续正常化上升
3. **负经营现金流**：业务扩张消耗现金
4. **流动性结构恶化**：流动资产净值-RMB 315亿
