# 李宁（02331.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

李宁2025年实现营业收入295.98亿元人民币（创历史新高），同比 +3.2%。毛利144.89亿元，同比 +2.4%，毛利率49.0%（同比 -0.4ppt）。归母净利润29.36亿元，同比 -2.6%，净利率9.9%（-0.6ppt）。经营利润率13.2%（+0.4ppt）。这是公司连续第三年"收入增长但利润下滑"。

### 1.2 增长与拖累因素

主要增长驱动为跑步品类（零售流水 +10%，占零售销售31%，专业跑鞋销量超2,600万双）和羽毛球品类（+30%，占收入约7%创新高）。渠道层面，特许经销商（+6.3%至137.73亿元）和电商（+5.3%至87.43亿元）驱动增长。主要拖累为篮球品类（-19%，此前定价策略失误导致竞争力减弱）和运动休闲（-9%，国潮热度消退）。直营渠道收入下降3.3%（关闭59家直营店）。毛利率下滑主要因直营渠道折扣加大和税率上升。

### 1.3 趋势判断

管理层给出2026年高单位数收入增长指引，核心信心来自跑步品类持续高增、夺回中国奥委会合作伙伴权益（冬奥曝光度提升）以及2026年"李宁YOUNG"渠道发力。但篮球和运动休闲品类的恢复时间、直营渠道的盈利提升速度、以及行业竞争加剧（Nike/Adidas反攻+安踏多品牌围攻）是主要不确定性。整体趋势为"专业化转型初见成效，但利润修复仍需时间"。

## 2. 主营业务

### 2.1 公司简介

李宁是中国领先的体育用品公司，以"李宁"品牌为核心，提供专业运动及运动休闲鞋类、服装及配件产品，定位"专业运动品牌"，覆盖跑步、篮球、综训、羽毛球、运动休闲等品类。

### 2.2 主营业务构成

1. **跑步**：第一大品类（零售占比31%），核心产品线"飞电""赤兔""超轻"年销超1,100万双，"Boom"技术平台持续升级
2. **篮球**：曾经的核心品类，但因此前定价策略问题竞争力减弱，2025年零售流水 -19%，正在调整中
3. **综合训练**：零售流水 +5%，保持稳健
4. **羽毛球**：高增长品类（+30%），年销球拍超550万支，器材类产品占比已达85%
5. **运动休闲**：零售流水 -9%，受国潮退潮影响，正在重新定位
6. **李宁YOUNG**：儿童运动品牌，门店从2024年末50家激增至1,518家

### 2.3 核心收入来源

收入按渠道分布：特许经销商~47%、电商~30%、直营~22%。按产品类型，鞋类收入占比49.5%（首次超越服装），反映公司"以鞋带牌"的专业化策略生效。跑步品类已成为绝对核心增长引擎。

### 4.3 营收分析

**主要增长来源**

跑步品类（+10%，占比从16%提升至31%仅用5年）是绝对核心引擎。羽毛球（+30%）虽然规模小但增速最快。特许经销商渠道（+6.3%）增长最稳健，反映品牌对渠道端的赋能效果。鞋类收入占比首次超过服装（49.5%），"以鞋带牌"策略正在兑现。

**主要拖累因素**

篮球品类（-19%）和运动休闲（-9%）合计拖累显著。篮球此前因激进定价和库存管理失误导致竞争力下降，目前正在"控量收缩"。运动休闲受国潮热度消退影响，公司正试图将其重新绑定到专业运动DNA上。直营渠道收入下降（-3.3%）反映关店优化策略。

**结构变化**

专业化转型加速：专业产品收入占比超56%，三大专业品类零售占比64%。鞋类首超服装，标志着"产品力>品牌溢价"的竞争模式转变。李宁YOUNG门店从50家爆增至1,518家，是潜在的增量渠道。

### 5.1 整体利润表现

| 归母净利润 | 29.36亿元 | ~30.14亿元 | 同比 -2.6% | 连续第三年下滑 |
| 毛利率 | 49.0% | 49.4% | -0.4ppt | |
| 净利率 | 9.9% | 10.5% | -0.6ppt | |

### 5.2 利润补充分析

**年报是否披露分部利润**

未按品类披露分部利润。仅披露渠道收入和品类零售流水增速。

**可替代观察指标**

- 销售及经销开支/收入比：31%（-1.1ppt），费用控制成效显著
- 销售及经销开支绝对额同比 -0.1%
- 经营利润率13.2%（+0.4ppt），说明核心经营层面在改善
- 净利润率下滑主要因有效税率上升（非经营因素），而非经营恶化

**利润增长驱动**

1. 费用率大幅改善：销售经销费用率-1.1ppt至31%，绝对额微降，反映渠道效率提升
2. 经营利润率提升：从12.8%升至13.2%，说明收入增长带来的经营杠杆效应为正
3. 产品结构优化：鞋类占比提升（通常高毛利）+专业品类占比提升
4. 库存健康：周转64天持平，<6月库存占85%，减少打折清库存需求

**利润承压因素**

1. 毛利率下滑-0.4ppt至49.0%：直营渠道折扣加大、行业竞争加剧（Nike/Adidas反攻）
2. 有效税率上升：这是净利润率-0.6ppt的主要原因（经营利润率实际+0.4ppt）
3. 总部大楼减值：2023年购入后两年减值超6亿，对利润持续构成拖累
4. 篮球和运动休闲收缩导致部分品类可能亏损

## 6. 未来可能的利好

### 利好 1：跑步品类持续高增，"以鞋带牌"专业化战略兑现

**依据**

跑步零售流水+10%（连续高增），品类占比5年内从16%提升至31%。专业跑鞋销量超2,600万双（其中三大旗舰系列超1,100万双）。"Boom"技术平台持续升级推出"超级Boom"。

**解读**

跑步是全球运动品牌最核心的竞技场。李宁在中国跑步市场已建立差异化优势（飞电竞速系列多次在马拉松赛事中助力破纪录），这一优势正在从竞技圈向大众消费者扩散。31%的品类占比意味着跑步已取代篮球成为公司新的核心支柱。

### 利好 2：夺回中国奥委会合作权益+2026冬奥曝光，品牌势能提升

**依据**

2025年李宁从安踏手中夺回中国奥委会合作伙伴权益。冬奥会期间中国代表团穿着李宁品牌出席开幕式，获得高曝光。北京太古里三里屯旗舰店开业推出"荣耀金标"系列。

**解读**

奥委会合作权益是运动品牌最顶级的营销资源。此前安踏凭借这一权益在2022年北京冬奥期间大幅提升品牌势能。李宁夺回这一权益后，叠加2028年洛杉矶奥运会，将获得持续的品牌曝光机会。"荣耀金标"系列是首次将奥委会商业标志与李宁品牌结合，可能开辟高端产品线。

### 利好 3：经营效率持续提升+200亿净现金提供战略投资空间

**依据**

销售经销费用率-1.1ppt至31%，经营利润率+0.4ppt至13.2%。净现金199.74亿元（+18.10亿）。库存周转64天保持健康。管理层2026年指引"高单位数"收入增长。

**解读**

200亿净现金在中国运动品牌中仅次于安踏（~317亿），提供了充裕的战略投资弹药——无论是海外扩张、品类孵化还是营销投入。经营效率改善说明公司正在从"规模扩张"转向"质量提升"。如果2026年收入增长加速（高单位数=7-9%）且费用率继续优化，利润有望止跌回升。

### 利好 4：李宁YOUNG门店爆发式增长，打开儿童运动市场

**依据**

"李宁YOUNG"门店从2024年末50家激增至1,518家（+2,936%）。

**解读**

中国儿童运动用品市场是增长最快的细分赛道之一。1,518家门店意味着"李宁YOUNG"已完成初步渠道铺设，2026年有望开始贡献可见的收入增量。安踏旗下的FILA KIDS已证明儿童运动品牌的高盈利能力。

## 7. 未来可能的利空

### 利空 1：利润连续三年下滑，"增收不增利"困局尚未破解

**依据**

2023-2025年，李宁收入连续增长但归母净利润连续下滑：2023年31.87亿→2024年~30.14亿→2025年29.36亿。毛利率三年下滑趋势（49.4%→49.0%），净利率从2022年高点约15%降至9.9%。

**解读**

"增收不增利"持续三年意味着公司的增长质量存在问题。收入增长主要靠低利润的经销商渠道拉动，而高利润的直营渠道在收缩。如果2026年毛利率继续下滑或税率进一步上升，利润可能第四年下滑，市场耐心将被进一步消耗。

### 利空 2：篮球品类-19%深度调整，品类平衡面临挑战

**依据**

篮球零售流水同比 -19%，管理层承认此前定价策略失误导致竞争力减弱。公司正在控量清库存，短期内篮球品类占比将继续下降。

**解读**

篮球曾是李宁的核心品类（韦德之道系列），目前的深度调整意味着公司在篮球赛道上已从"进攻"转入"防守"。竞争对手（Nike Jordan、安踏Amer Sports旗下品牌）仍在积极争夺篮球市场份额。如果篮球无法在1-2年内恢复增长，李宁的品类多元化将面临"跑步一枝独大"的风险。

### 利空 3：行业竞争加剧，Nike/Adidas反攻中国市场

**依据**

Adidas 2025年跑步品类全球增长30%（Adizero系列），Nike推出Vomero旗舰缓震跑鞋。国内安踏集团收入突破800亿、特步跑步品类高增。国际品牌正通过"以价换量"策略恢复中国市场份额。

**解读**

2021-2023年国潮红利期，国际品牌在中国市场处于被动。但随着国潮热度消退和国际品牌战略调整，竞争正在回归"产品力+性价比"的本质。李宁毛利率49%已出现下滑，如果被迫加大折扣力度应对竞争，利润空间将进一步被压缩。

## 8. 总结判断

**一句话结论**

2025年李宁"跑步强劲+经营提效"推动收入创新高，但利润连续第三年下滑（-2.6%至29.4亿），核心矛盾是"专业化转型的长期价值"与"短期利润压力"之间的博弈，2026年是验证利润能否止跌回升的关键年。

**核心看点**

1. 跑步品类+10%（占比31%）+羽毛球+30%，"以鞋带牌"专业化战略正在兑现
2. 夺回中国奥委会合作权益+冬奥曝光，品牌势能获得关键提升
3. 200亿净现金+经营利润率+0.4ppt至13.2%，经营基本面在改善

**核心风险**

1. 归母净利润连续三年下滑（+3.2%收入增长无法转化为利润增长），"增收不增利"困局待解
2. 篮球（-19%）和运动休闲（-9%）品类深度调整，品类集中度风险上升
3. Nike/Adidas反攻中国+安踏多品牌围攻，行业竞争加剧压制毛利率
