# 中车时代电气（03898.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

中车时代电气2025年实现营业收入RMB 287.03亿元，同比 +15.2%。归母净利润RMB 40.97亿元，同比 +10.6%；扣非净利润RMB 38.99亿元，同比 +20.9%（核心盈利质量提升）。PBT RMB 48.74亿（+12.9%）。每股盈利RMB 2.99元（+14.1%）。末期股息RMB 0.68元，全年股息RMB 1.12元，派息率37.5%。

### 1.2 增长与拖累因素

增长方面，新兴装备板块收入RMB 127.80亿（+26.4%），其中半导体+30.4%（RMB 53.60亿）、汽车+27.2%（RMB 32.68亿）、工业+34.6%为主要驱动；轨道交通板块收入RMB 158.06亿（+8.0%），稳健增长。拖累方面，其他公平值/减值损失扩大36.1%至RMB -4.79亿，所得税大幅增长53.5%（有效税率从8.5%→11.6%），D&A +25.8%至RMB 15.54亿（重资产扩张期）。非经常性损益从RMB 4.77亿降至1.97亿，使得报表净利增速（+10.6%）低于扣非增速（+20.9%）。

### 1.3 趋势判断

中车时代电气正从**轨交电气设备龙头**加速向**"轨交+半导体+新能源+汽车"多元化平台**转型——新兴装备占比已达44.5%（从2024年40.6%提升），半导体（IGBT/SiC）和新能源车电驱系统是高增长引擎。整体判断：**收入端双位数增长可持续（新兴装备+26%，轨交稳健），扣非净利+21%体现核心盈利改善，但重资产扩张期（PP&E +36%、D&A +26%）和应收/存货膨胀需关注现金流质量**。

## 2. 主营业务

### 2.1 公司简介

中车时代电气是中国中车集团旗下的高端装备制造企业（A+H双重上市），核心业务包括轨道交通电气系统、功率半导体（IGBT/SiC）、新能源汽车电驱系统、工业变频器等。

### 2.2 主营业务构成

1. **轨道交通装备**：牵引变流器、网络控制系统、信号系统等轨交核心电气设备；收入占比约55.1%
2. **半导体**：IGBT模块、SiC器件等功率半导体，面向轨交、新能源、工业等领域；收入占比约18.7%
3. **汽车电驱**：新能源汽车电驱系统和控制器；收入占比约11.4%
4. **新能源/海洋/工业**：风电变流器、海洋装备、工业变频器等；收入占比约14.8%

### 2.3 核心收入来源

轨道交通装备仍是第一大收入来源（55%），但新兴装备板块（半导体+汽车+新能源+海洋+工业）增速远快于轨交，占比已达45%，且半导体（IGBT/SiC）是最具战略价值的高毛利业务。

## 6. 未来可能的利好

### 利好 1：半导体（IGBT/SiC）国产替代加速——第二增长曲线

**依据**

半导体板块收入RMB 53.60亿（+30.4%），IGBT模块和SiC器件受益于新能源汽车、光伏逆变器、储能变流器等下游需求爆发。公司是国内最大的IGBT模块制造商之一。

**解读**

全球功率半导体市场由英飞凌等海外巨头主导，国产化率仍较低。中车时代电气背靠中车集团的体量和技术积累，有望在IGBT/SiC领域持续突破，特别是在新能源车800V高压平台和光伏/储能领域。半导体fab的大规模资本开支（PP&E +36%）正在为未来产能释放铺路。

### 利好 2：新兴装备板块高速增长——占比有望超50%

**依据**

新兴装备收入+26.4%至RMB 127.80亿（占比44.5%），汽车电驱+27.2%、工业+34.6%均保持高增长。2-3年内有望超过轨交成为第一大收入来源。

**解读**

新兴装备的毛利率通常高于传统轨交设备，占比提升将改善整体利润率。多元化布局也降低了对单一轨交市场的依赖风险。

### 利好 3：扣非净利润增速+21%——核心盈利质量持续提升

**依据**

扣非净利润增速（+20.9%）远超收入增速（+15.2%）和报表净利增速（+10.6%），体现规模效应和经营杠杆。

**解读**

当非经常性损益波动消除后，核心业务的盈利增长能力更为突出。如果2026年非经常性损益恢复正常水平，报表净利增速有望向扣非增速靠拢。

## 7. 未来可能的利空

### 利空 1：重资产扩张期——现金消耗与D&A压力

**依据**

PP&E +36.2%至RMB 148.58亿，其他非流动资产+156%至RMB 127.85亿，D&A +25.8%至RMB 15.54亿。现金从RMB 106.52亿下降37%至RMB 67.14亿。

**解读**

公司正处于半导体fab和新产能建设的高峰期，大量资本支出导致现金消耗和D&A持续增长。如果新产能的产出和营收贡献不及预期，D&A的持续增长将压缩利润率。

### 利空 2：应收帐款和存货膨胀——营运资本压力

**依据**

应收贸易帐款+18.3%至RMB 175.52亿（增速>收入+15.2%），存货+23.2%至RMB 87.06亿（增速>收入），流动资产净值-37.6%至RMB 172.90亿。

**解读**

应收帐款和存货增速均超过收入增速，说明营运资本管理有所松弛。应收帐款占总资产24.3%（主要为轨交/政府客户应收，回款周期较长），如果经济放缓导致客户付款延迟，可能带来减值风险。

### 利空 3：有效税率上升——利润增长的隐形障碍

**依据**

有效税率从8.5%上升至11.6%（+3.1ppt），所得税+53.5%。

**解读**

此前较低的有效税率可能受益于高新技术企业优惠和研发加计扣除，税率回升趋势若持续，将对净利润增速形成持续压力。

## 8. 总结判断

**一句话结论**

中车时代电气2025年收入+15.2%、扣非净利+20.9%体现了"轨交+半导体+新能源"多元化战略的高质量增长——半导体+30.4%、汽车+27.2%、工业+34.6%三大引擎强劲，新兴装备占比已达45%，但报表净利+10.6%被非经常性损益减少和税率上升拉低，同时处于重资产扩张高峰期（PP&E +36%、现金-37%、D&A +26%），需关注资本回报和营运资本管理。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| PBT | RMB 48.74亿（+12.9%） | 经营面稳健增长 |
| 扣非净利 | RMB 38.99亿（+20.9%） | ✅✅核心盈利质量好 |
| 有效税率 | 11.6%（+3.1ppt） | ⚠️回升中 |
| D&A | RMB 15.54亿（+25.8%） | ⚠️重资产扩张 |
| 现金 | RMB 67.14亿（-37.0%） | 投入产能建设 |
| 净现金 | RMB 61.84亿 | ✅仍为净现金 |
| NBV | RMB 31.29（+6.0%） | 当前HK$38.2≈RMB 34.3，P/B≈1.10x |

**核心看点**

1. **半导体产能释放**：PP&E大幅扩张后的产能利用率和收入贡献
2. **新兴装备占比何时超50%**：结构性转型的里程碑
3. **扣非净利增速能否维持20%+**：核心盈利增长的持续性
4. **营运资本管理**：应收帐款和存货增速能否回归与收入同步

**核心风险**

1. **重资产扩张期的资本回报不确定性**：PP&E +36%、D&A +26%，若产能利用率不足则利润率承压
2. **应收帐款RMB 176亿（占总资产24%）**：大客户回款周期风险
3. **有效税率持续上升**：从8.5%→11.6%，若继续回升将侵蚀利润增长
4. **非经常性损益波动**：2025年减少RMB 2.80亿，影响报表净利增速
