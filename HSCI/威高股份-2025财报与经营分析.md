# 【威高股份】2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

威高股份2025年实现收入133.89亿元（同比 +2.3%），股东应占纯利16.12亿元（同比 -22.0%），每股基本盈利0.36元（vs 2024年0.46元）。收入温和增长但利润大幅下滑——毛利率从50.9%（H1 2024）降至49.7%（H1 2025），全年趋势类似。

### 1.2 增长与拖累因素

收入+2.3%的驱动：（1）介入产品和血液管理产品维持较快增长；（2）威高骨科收入+5.95%（尤其H2改善）；（3）总体集采降价影响趋于"见底"。利润-22%的核心拖累：（1）通用医疗器械板块集采降价影响高于预期——直接压缩毛利率；（2）海内外销售体系搭建带来的费用投入增加；（3）部分产品（冲管式注射器等）新一轮集采冲击。

### 1.3 趋势判断

威高股份正处于"集采阵痛期"——短期利润受压，但管理层表示集采影响已基本反映，预计2026年起医疗器械板块销售收入将缓步回升。骨科和介入业务是增长亮点，海外布局为长期提供想象空间。2026年的核心看点是：利润能否触底反弹。

---

## 2. 主营业务

### 2.1 公司简介

威高股份是中国领先的一次性医疗器械和医用高分子制品制造商，产品涵盖注射器、输液器、血液净化、骨科植入物、介入器械和药品包装等，是中国医疗器械行业的龙头企业之一。

### 2.2 主营业务构成

1. **一次性使用医疗器械**：注射器、输液器、采血管等——收入占比~50%，核心传统业务
2. **药品包装产品**：预灌封注射器、药用胶塞等——占比~18%
3. **介入产品**：心脏支架、导管等——占比~15%，增长较快
4. **骨科产品**：关节、脊柱、创伤植入物——子公司威高骨科（688161.SH）运营
5. **血液管理产品**：采血袋、血液净化产品等
6. **其他**：金融服务等

### 2.3 核心收入来源

一次性医疗器械是最大收入来源（~50%），但受集采降价影响最大。介入和骨科是增长引擎。药品包装是稳定贡献。

## 4. 营收分布和增长

### 4.1 按业务划分（H1参考）

| 业务 | H1 2025收入 | 占比 | 趋势 |
|---|---:|---:|---|
| 一次性医疗器械 | 33.00亿 | 49.7% | 集采见底 |
| 药品包装 | 11.66亿 | 17.6% | 冲管器集采影响 |
| 介入产品 | 9.90亿 | 14.9% | 较快增长 |
| 骨科（H1推算） | ~7.0亿 | ~10% | H2大幅改善 |
| 血液管理等 | ~5亿 | ~7% | 增长较快 |

### 4.2 营收分析

**主要增长来源**

介入产品和血液管理维持较快增长——受益于心血管介入和血液净化的需求扩大。骨科全年+5.95%——H2在经销模式转型完成后加速增长（Q3+10%）。

**主要拖累因素**

一次性医疗器械受集采降价影响最大——低值耗材集采直接压低售价。药品包装受冲管式注射器集采冲击——H1收入几乎持平。

**结构变化**

集采推动收入结构变化——低值耗材占比可能逐步下降，介入、骨科等高值耗材占比有望提升。

### 5.1 整体利润表现

| 毛利率 | ~49-50% | ~51% | 下降 | |
| 股东应占纯利 | 16.12亿 | 20.67亿 | -22.0% | |
| 净利率 | ~12.0% | ~15.8% | -3.8ppt | |

### 5.2 利润分析

**利润大幅下降的原因**

1. **集采降价压缩毛利率**：一次性医疗器械和冲管式注射器的集采降价直接压缩了产品售价——毛利率从~51%降至~49-50%
2. **销售费用增加**：海内外销售体系搭建（尤其是海外渠道建设）带来前期费用投入
3. **总体费用率上升**：收入+2.3%但费用增长快于收入——经营杠杆为负

**利润亮点**

威高骨科净利+19.17%——经销模式转型完成后盈利能力显著改善。

## 6. 未来可能的利好

### 利好 1：集采影响"见底"——2026年收入有望缓步回升

**依据**

管理层表示通用医疗器械板块集采影响已基本反映。预计2026年起该板块销售收入将缓步回升。骨科H2加速增长（Q3+10%）是"见底回升"的信号。

**解读**

集采降价是一次性冲击——一旦新的价格体系稳定，企业可以通过"以量补价"和成本优化恢复增长。威高作为行业龙头，在集采中通常能获得较大份额——市场份额提升可以部分弥补价格下降。2026年利润"触底反弹"的概率较大。

### 利好 2：介入和骨科——高值耗材增长引擎

**依据**

介入产品维持较快增长。威高骨科收入+5.95%、净利+19.17%。

**解读**

介入器械和骨科植入物是医疗器械行业的高价值领域——技术壁垒高、利润率好。威高在这两个领域持续投入研发和销售渠道，有望成为未来的核心增长引擎。集采对高值耗材的降价幅度通常小于低值耗材——利润韧性更强。

### 利好 3：海外市场布局——长期增长空间

**依据**

公司加大海外销售体系搭建投入。

**解读**

中国医疗器械企业的国际化是大趋势——海外市场的产品定价通常高于国内（不受集采影响）。虽然短期内增加了费用，但长期来看海外收入将成为重要的增量来源和利润贡献。

---

## 7. 未来可能的利空

### 利空 1：集采范围可能继续扩大——新品类纳入

**依据**

冲管式注射器已被纳入集采。医疗器械集采范围持续扩大。

**解读**

如果更多威高的核心产品被纳入集采（如更多规格的注射器、输液器等），收入和利润将面临新一轮压力。集采常态化是中国医疗器械行业的大趋势——企业需要持续承受降价压力。

### 利空 2：利润率恢复不确定——费用增长未放缓

**依据**

利润-22%但收入+2.3%——费用增长快于收入。海外渠道建设仍在投入期。

**解读**

如果海外市场的收入增长不达预期，前期投入的费用将无法被覆盖——导致利润率持续承压。销售费用率的上升如果成为结构性的（而非一次性的），将长期影响利润水平。

### 利空 3：绩后暴跌14%——市场信心不足

**依据**

2025年报发布后股价暴跌超14%。机构指出净利润低于市场预期。

**解读**

市场的大幅负面反应反映了投资者对威高"增收不增利"的失望。如果2026年利润未能如预期反弹，股价可能面临进一步下行压力。

---

## 8. 总结判断

**一句话结论**

威高股份2025年营收134亿（+2.3%）、净利16.1亿（-22%）——集采降价+销售费用投入双重挤压利润；骨科净利+19%是亮点；管理层表示集采影响"已基本反映"，2026年有望缓步回升；50%派息率维持。

**核心看点**

1. 2026年医疗器械板块收入能否恢复增长——集采"见底"验证
2. 介入和骨科能否持续高增长——成为新利润引擎
3. 海外市场收入何时放量——前期费用投入能否获得回报
4. 毛利率能否企稳——成本优化的效果

**核心风险**

1. 集采范围继续扩大——新品类纳入降价
2. 利润率恢复不确定——费用增长快于收入
3. 市场信心不足——绩后暴跌14%
4. 海外拓展不达预期
