# 中远海控（01919.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

中远海控2025年度实现营业收入人民币2,195.04亿元，同比 -6.14%（2024年~2,338.59亿元）。归母净利润308.68亿元，同比 -37.24%（2024年~491.72亿元）——利润降幅远大于收入降幅。基本每股收益1.99元。集装箱航运业务收入2,107.31亿元（-6.74%），毛利率19.44%（-9.79pp）。港口码头收入120.41亿元（+11.39%）。资产负债率41.42%（20年最低）。拟派发末期股息每股0.44元，全年现金分红约154.1亿元，分红率50%。

### 1.2 增长与拖累因素

核心拖累：(1) 全球集装箱运价大幅回落——SCFI（上海出口集装箱运价指数）同比-37%，CCFI同比-23%，红海复航+贸易不确定性导致运价从2024年高位急剧下滑；(2) 集装箱航运成本+6.05%（1,709.98亿元），"量增价跌+成本刚性"导致毛利率-9.79pp至19.44%；(3) Q4单季归母利润仅37.99亿（-65.39%），运价加速下行。正面因素：(1) 集装箱货运量2,743.45万TEU（+5.76%），运输量保持增长；(2) 港口吞吐量1.53亿TEU（+6.22%），韧性较强；(3) 供应链业务（不含远洋）收入~448.9亿（+9.64%），成为新增长极。

### 1.3 趋势判断

2025年是集装箱航运从"超级周期"回归常态的一年——运价大幅回落但仍远高于疫情前水平。利好方向：全球贸易量长期增长+港口业务韧性+供应链多元化+超低负债率+50%高分红；压力方向：运价持续下行风险+成本刚性+2026年全球新船交付高峰→运力过剩+Q4利润急剧萎缩暗示2026年压力加大。

## 2. 主营业务

### 2.1 公司简介

中远海控是全球领先的集装箱航运和港口运营商，旗下运营COSCO SHIPPING Lines和OOCL（东方海外）两大集装箱航运品牌，同时控股运营全球多个集装箱码头，是OCEAN Alliance成员。

### 2.2 主营业务构成

1. **集装箱航运（核心）**：全球集装箱班轮运输服务，运营约500艘船舶，运力约320万TEU，收入占比~96%
2. **港口码头**：全球集装箱码头的投资和运营，总吞吐量1.53亿TEU，收入占比~5%
3. **供应链及物流**：货运代理、仓储、多式联运等综合物流服务

### 2.3 核心收入来源

集装箱航运业务贡献约96%收入，业绩高度依赖全球集装箱运价水平。港口业务提供稳定的费率收入，与运价周期的相关性较低。

### 4.3 营收分析

**正面因素**

港口码头（+11.39%）和供应链业务（+9.64%）展现韧性，成为运价周期下行中的稳定器。集装箱运量+5.76%说明全球贸易量仍在增长。

## 5. 利润分布和增长

### 5.1 整体利润表现

| 营业收入 | 2,195.04亿元 | ~2,338.59亿元 | -6.14% | |
| 营业成本 | 1,768.02亿元 | ~1,661.75亿元 | +6.40% | |
| 归母净利润 | 308.68亿元 | ~491.72亿元 | -37.24% | |
| 集装箱航运毛利率 | 19.44% | ~29.23% | -9.79pp | |

### 5.3 利润分析

**利润降幅远大于收入降幅的原因**

典型的"量增价跌+成本刚性"剪刀差效应：
- 运价下降导致单箱收入-11%
- 但成本端（燃油、租船、港口费、人工）具有刚性，同比+6.05%
- 收入-6.74%但成本+6.05%→毛利率从~29.23%暴跌至19.44%
- 毛利从~661亿降至~397亿（-40%），直接传导至净利润-37.24%

**Q4利润暴跌是核心警信**

Q4单季归母利润仅37.99亿（-65.39%），暗示运价在Q4加速下行。如果2026年运价继续走低，利润可能面临更大压力。

## 6. 未来可能的利好

### 利好 1：全球贸易量长期增长——运量保持正增长

**依据**

2025年集装箱运量+5.76%至2,743万TEU，港口吞吐量+6.22%。

**解读**

尽管运价下跌，但全球贸易的物理量仍在增长。中远海控作为全球第三大集装箱航运公司（运力~320万TEU），将持续受益于全球化的长期趋势。只要运量保持增长，运价回到合理水平时的利润弹性依然巨大。

### 利好 2：资产负债率20年最低+50%高分红——股东回报确定性

**依据**

资产负债率41.42%（20年最低）。全年分红约154.1亿元（50%分红率）。

**解读**

中远海控在上一轮航运超级周期中积累了大量现金，并持续降杠杆。41.42%的负债率给予公司极大的财务灵活性——可以承受运价周期的波动而不面临偿债风险。50%的分红率在周期性行业中极为罕见，体现了管理层对现金流的信心。

### 利好 3：港口+供应链业务——平滑周期波动

**依据**

港口收入+11.39%，供应链收入（不含远洋）+9.64%。

**解读**

港口码头和供应链物流业务与运价的相关性较低，且增长趋势稳定。随着中远海控持续投资供应链业务，这些业务将逐步平滑集装箱航运的周期性波动，降低集团利润的波动性。

## 7. 未来可能的利空

### 利空 1：运价持续下行——2026年可能进一步承压

**依据**

SCFI -37%，CCFI -23%。Q4归母利润-65.39%。

**解读**

2024年的高运价是红海危机的"意外红利"，随着局势缓和，运价正在向常态化水平回归。如果2026年全球经济放缓+红海复航+新船交付高峰叠加，运价可能进一步下跌——Q4的利润暴跌已预示了这一趋势。集装箱航运毛利率可能跌至15%甚至更低。

### 利空 2：全球新船交付高峰——运力过剩风险

**依据**

2026-2027年全球集装箱船订单交付量处于历史高位。

**解读**

大量在建新船将在2026-2027年集中交付，显著增加市场运力供给。如果需求增长无法匹配运力增长，将导致运力过剩→运价下行→利润率承压。这是航运行业的经典周期性风险，中远海控也无法独善其身。

### 利空 3：成本刚性——"量增利减"可能持续

**依据**

营业成本+6.40%，集装箱航运成本+6.05%。毛利率-9.79pp。

**解读**

船舶折旧、人工、港口费等固定/半固定成本难以随运价下降而大幅压缩。燃油成本虽有波动，但整体仍处高位。在运价下行但成本上升的"剪刀差"环境中，利润率可能进一步收缩。

## 8. 总结判断

**一句话结论**

中远海控2025年收入-6.14%但归母利润-37.24%，集装箱运价暴跌（SCFI -37%）叠加成本刚性导致毛利率-9.79pp；运量+5.76%和41.42%的超低负债率说明基本面稳健，50%分红率提供确定性回报。

**核心看点**

1. 集装箱运量+5.76%——全球贸易量仍在增长
2. 资产负债率41.42%——20年最低，财务极度稳健
3. 50%分红率——全年约154.1亿元，高股息确定性
4. 港口+供应链——平滑周期波动的新增长极

**核心风险**

1. SCFI -37%/CCFI -23%——运价暴跌，Q4利润-65.39%
2. 全球新船交付高峰——2026-2027年运力过剩风险
3. 集装箱航运毛利率-9.79pp至19.44%——"量增利减"
