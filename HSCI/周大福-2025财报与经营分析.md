# 周大福（01929.HK）2025年财报整理

> 注：周大福珠宝集团财政年度截止日为**3月31日**。本报告基于截至2025年3月31日止的财政年度（FY2025）年报数据。

## 1. 业绩总结

### 1.1 收入与利润表现

周大福珠宝FY2025实现营业收入港币896.56亿元，同比 -17.5%（FY2024约1,086.7亿港元），收入大幅下滑。但经营溢利147.46亿港元，同比 +9.8%（FY2024~134.3亿），经营溢利率16.4%（+400bp），盈利质量显著提升。股东应占溢利59.16亿港元，同比 -9.0%（FY2024~65.0亿）。每股盈利0.59港元。毛利率29.5%（+550bp）。全年每股股息0.52港元（中期0.20+末期0.32），派息率87.8%。

### 1.2 增长与拖累因素

核心拖累：(1) 同店销售大幅下降——内地-19.4%，港澳-26.1%，高金价抑制消费者购买量；(2) 收入-17.5%，黄金价格飙升导致"克重减少、金额缩减"（消费者在高金价下购买更轻的产品）；(3) 内地门店数净减少。核心亮点：(1) 定价黄金产品收入+105.5%，占比从7.1%飙升至19.2%——这是毛利率暴增550bp的核心驱动力；(2) 经营溢利+9.8%，经营利润率+400bp——"收入跌但赚钱更多"；(3) 传福系列和故宫系列各实现约40亿港元销售额，品牌IP化策略成功。

### 1.3 趋势判断

周大福正在经历"从量到质"的战略转型——收入虽然下降17.5%，但经营利润率+400bp至16.4%、毛利率+550bp至29.5%，说明产品组合向高毛利定价产品的转型正在奏效。利好方向：定价黄金占比持续提升+品牌IP化+新零售转型+高派息率（87.8%）；压力方向：同店销售持续负增长+高金价抑制消费量+门店收缩期。

## 2. 主营业务

### 2.1 公司简介

周大福珠宝集团是全球最大的珠宝零售商之一，以黄金珠宝产品为核心，涵盖足金产品、镶嵌珠宝、铂金/K金饰品及钟表等，通过遍布中国内地、港澳和海外的超6,300个零售点销售。

### 2.2 主营业务构成

1. **黄金产品（核心）**：足金饰品（按克计价+定价产品）和投资金条，为收入主力
2. **镶嵌珠宝**：钻石、翡翠、宝石等镶嵌首饰
3. **铂金/K金饰品**：铂金和K金首饰
4. **钟表及其他**：品牌钟表和配饰

### 2.3 核心收入来源

黄金产品占收入绝大部分（约80%+）。内地市场占总零售值约90%。定价黄金产品（如传福系列、故宫系列）正快速替代传统按克计价产品成为利润贡献核心。

## 4. 营收分布和增长

### 4.1 按产品划分

定价黄金产品是FY2025的明星品类——收入飙升105.5%，占内地黄金珠宝零售值比例从7.1%跃升至19.2%。传福系列和故宫系列各约40亿港元。镶嵌珠宝和其他品类受消费降级和金价高企挤压。

### 4.2 按地区划分

| 地区 | FY2025同店销售 | 门店数 | 备注 |
|---|---:|---:|---|
| 中国内地 | -19.4% | 6,274 | 按季改善 |
| 香港 | -23.1% | ~87（含澳门） | |
| 澳门 | -35.3% | 含上 | 内地游客消费低迷 |
| 其他（泰国/马来/日本） | — | 净+3 | |

### 4.3 营收分析

**收入下降的核心原因**

(1) 高金价抑制消费量——国际金价FY2025飙升至历史新高，消费者倾向于购买更轻的产品或延迟购买；(2) 同店销售大幅负增长（内地-19.4%，港澳-26.1%）；(3) 内地门店数净减少——关闭低效门店。

**毛利率暴增的核心原因**

定价黄金产品（如传福系列）毛利率远高于传统按克计价产品（后者本质上是"黄金+加工费"，毛利率极低，约5-10%）。定价产品通过IP化、设计溢价实现30-50%的毛利率。随着定价产品占比从7.1%跃升至19.2%，整体毛利率大幅提升550bp至29.5%。

### 5.3 利润分析

**经营溢利增长但归母利润下降的原因**

经营溢利+9.8%但归母-9.0%，差距来自经营层面以下：(1) 联营/合营企业投资收益变化；(2) 融资成本；(3) 所得税费用。经营层面的盈利质量（+400bp利润率）是极强的积极信号。

**利润质量提升**

"收入-17.5%但经营利润+9.8%"——这是典型的"提质增效"成功案例。核心策略：(1) 定价黄金产品占比提升→毛利率+550bp；(2) 低效门店关闭→成本优化；(3) 新形象店生产力更高；(4) 电商占比提升→固定成本降低。

## 6. 未来可能的利好

### 利好 1：定价黄金产品占比持续提升——利润率升级引擎

**依据**

定价产品占比从7.1%飙升至19.2%（+12.1pp），收入+105.5%。传福系列、故宫系列各40亿。

**解读**

定价黄金是周大福"从量到质"转型的核心。如果占比继续提升至30-40%（可类比行业领先水平），毛利率有望进一步提升至32-35%，经营利润率可达18-20%。金价越高，消费者对"设计+IP"的溢价接受度越高（买黄金饰品而非投资金条），有利于定价产品渗透。

### 利好 2：品牌IP化和新零售转型——提升消费者粘性

**依据**

故宫系列、传福系列、CTF PET宠物珠宝等IP产品线持续扩展。5间新形象店生产力超一般门店。港澳电商+91%。

**解读**

周大福正从"黄金零售商"转型为"珠宝生活方式品牌"。IP化产品具有更高的品牌溢价和复购率，新形象店提升了品牌形象和客单价。电商渠道的快速增长（港澳+91%）也在改变传统的门店依赖模式。

### 利好 3：高派息率87.8%——确定性股东回报

**依据**

全年股息0.52港元/股，派息率87.8%。

**解读**

87.8%的派息率在消费品公司中极为慷慨，体现了管理层对现金流的信心。即使收入端承压，高派息率为投资者提供了确定性回报。周大福的轻资产加盟模式（约75%门店为加盟）也支持了高分红的可持续性。

## 7. 未来可能的利空

### 利空 1：同店销售持续负增长——消费需求疲弱

**依据**

内地同店-19.4%，港澳-26.1%。

**解读**

尽管"按季改善"是正面信号，但两位数的同店下滑反映了结构性压力——高金价抑制消费量、消费降级、房地产财富效应消退。如果同店销售在FY2026未能转正，收入端可能继续萎缩，即使利润率改善也难以完全弥补。

### 利空 2：高金价的双刃剑——量缩可能加速

**依据**

国际金价持续创历史新高。消费者购买更轻的产品。

**解读**

金价上涨虽然提升了定价产品的毛利率，但也显著抑制了消费者的购买量和克重——这是FY2025收入-17.5%的重要原因。如果金价继续飙升，可能出现"量缩>价升"的局面，收入下滑幅度可能加大。

### 利空 3：门店收缩——规模扩张红利结束

**依据**

内地门店数净减少。过去几年从7,000+高峰回落至6,274。

**解读**

周大福在内地的门店扩张已达到饱和——低线城市的门店效率下降导致关店。门店收缩意味着通过开新店驱动增长的模式已不可持续，未来增长只能依赖同店提升和产品升级——在同店-19.4%的环境下，这是挑战性的。

---

## 8. 总结判断

**一句话结论**

周大福FY2025收入-17.5%但经营溢利+9.8%（利润率+400bp），定价黄金产品+105.5%的爆发和毛利率+550bp至29.5%验证了"从量到质"的转型成功；但同店-19.4%和门店收缩说明收入端仍在寻底。

**核心看点**

1. 定价黄金+105.5%——占比从7.1%升至19.2%，利润率升级引擎
2. 毛利率+550bp至29.5%——产品组合优化效果显著
3. 经营溢利+9.8%——"收入跌但赚钱更多"
4. 派息率87.8%——确定性股东回报

**核心风险**

1. 同店销售-19.4%（内地）/-26.1%（港澳）——消费需求持续疲弱
2. 高金价双刃剑——量缩可能加速收入下滑
3. 内地门店净减少——规模扩张红利结束
