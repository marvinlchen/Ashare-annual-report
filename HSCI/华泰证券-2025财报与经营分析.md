# 华泰证券/HTSC（06886.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

2025年华泰证券CAS口径营收RMB 358.1亿（-13.6%），归母净利润RMB 163.8亿（+6.7%），扣非净利润RMB 162.7亿（+80.1%）。IFRS口径收入RMB 473.07亿（+4.6%）。收入下降而利润增长的背离说明2024年有大额非经常性损益（可能含AssetMark相关减值）拖低了基数。扣非净利+80%才是核心经营利润的真实改善幅度。PBT为RMB 184.05亿（+19.9%），但税项从RMB +1.67亿（退税/递延税）骤变为RMB -20.32亿，吃掉了大部分PBT增长，导致归母净利仅增6.7%。

### 1.2 增长与拖累因素

**增长驱动**：(1) 财富管理/经纪净收入RMB 91亿（+41%），受益于A股交易量放大；(2) 投行净收入RMB 31亿（+48%），香港IPO保荐数量全市场第三；(3) 联营公司利润RMB 33.57亿（+43%），AssetMark等海外布局贡献显著；(4) 国际业务利润贡献18%，ROE 13%，境外资产+38%。**拖累因素**：(1) 税项从退税+1.67亿变为-20.32亿（有效税率从-1.1%升至11.0%）；(2) 公允价值损失-RMB 5.94亿（扩大141%）；(3) 短期借贷+94%至RMB 2,026亿，杠杆扩张激进。

### 1.3 趋势判断

华泰证券是中国头部综合券商中业务最均衡的一家——财富管理（+41%）、投行（+48%）、国际业务（18%利润贡献）三大亮点形成差异化优势。相较中国银河等"靠行情吃饭"的同行，华泰在轻资产和国际化业务上走在前列。ROE 9.2%在行业中处于领先水平。H股P/B 0.79x的估值与同行相近，但基本面质量更高。

## 2. 主营业务

### 2.1 公司简介

华泰证券（HTSC）是中国头部综合类券商，A+H+伦敦GDR三地上市（A股601688，H股06886，伦敦HTSC）。注册地江苏南京，实际控制人为江苏省国资委（通过江苏省国信集团）。旗下拥有华泰联合证券（投行）、南方基金（参股）、华泰紫金投资、AssetMark（美国财富管理平台）等。

### 2.2 主营业务构成

1. **财富管理业务**（经纪净收入RMB 91亿，+41%）：代理买卖、席位租赁、基金代销等，涨乐财富通APP用户领先
2. **机构服务/投行业务**（净收入RMB 31亿，+48%）：IPO保荐、再融资、债券承销，香港IPO保荐第三
3. **投资管理业务**（资管净收入RMB 18亿）：华泰资管子公司收入RMB 22亿、净利RMB 11亿
4. **国际业务**（境外收入RMB 59.2亿）：华泰国际金控ROE 13%，利润贡献18%，境外资产RMB 1,932亿（+38%）
5. **自营投资/交易业务**：含固收、权益、衍生品做市
6. **其他**：净利息收入RMB 43.96亿（+62.5%），期货、直投等

### 2.3 核心收入来源

华泰证券的业务结构相对均衡，国际化程度（18%利润贡献）和科技化程度（涨乐财富通）在中国券商中领先。财富管理和投行等轻资产业务增速（+41%、+48%）显著快于整体，业务结构在优化中。

### 4.3 营收分析

**与中国银河的关键差异**

## 5. 利润分布和增长

### 5.1 PBT增长拆解

PBT从RMB 153.52亿增至RMB 184.05亿（+RMB 30.53亿，+19.9%）：

| 因素 | 影响 | 说明 |
|---|---:|---|
| 收入增长（IFRS） | +RMB 20.73亿 | +4.6% |
| 联营公司利润增加 | +RMB 10.03亿 | +42.6%，AssetMark等 |
| 公允价值损失增加 | -RMB 3.47亿 | 从-2.46→-5.94 |
| 经营效率改善 | +~RMB 3.24亿 | |
| **PBT增加** | +RMB 30.53亿 | |

PBT到净利的转化被严重侵蚀：
- 税项从+RMB 1.67亿（退税/递延税）→-RMB 20.32亿，净影响-RMB 21.99亿
- NCI从-1.68→+0.11，净影响+RMB 1.79亿
- 永续债利息-RMB 7.96亿（+12.4%）
- 最终普通股东净利仅增RMB 9.45亿（+6.4%），远低于PBT的+19.9%

### 5.2 扣非净利+80%的意义

扣非净利从RMB 90.4亿→RMB 162.7亿（+80.1%），说明2024年约有RMB 63亿的非经常性亏损（可能包括AssetMark goodwill减值——无形资产从2023年RMB 93.66亿骤降至2024年RMB 5.61亿）。2025年回归正常，核心经营利润的真实改善是显著的。

## 6. 未来可能的利好

### 利好 1：国际化业务领先同行，海外收入持续扩张

**依据**

华泰国际金控ROE 13%，利润贡献18%，境外资产RMB 1,932亿（+38%）。AssetMark为美国领先的turnkey asset management平台。

**解读**

华泰是中国券商中国际化最成功的，海外业务不仅提供收入多元化，还能对冲A股周期性。境外资产+38%的增速说明国际化仍在加速。如果中概股回流香港上市的趋势延续，华泰在香港IPO保荐领域的竞争力将进一步凸显。

### 利好 2：轻资产业务高增长（财富管理+41%，投行+48%）

**依据**

财富管理经纪净收入RMB 91亿（+41%），投行净收入RMB 31亿（+48%），均大幅领先行业平均。

**解读**

轻资产业务增速远超重资产业务，说明华泰在业务结构优化上取得实质进展。财富管理和投行不需要大量资本金支撑，有助于提升ROE和降低业绩波动性。

### 利好 3：扣非净利+80%揭示核心经营实力

**依据**

2024年因非经常性损失（可能含AssetMark减值）导致扣非净利仅RMB 90亿，2025年回归RMB 163亿。

**解读**

去除非经常性干扰后，华泰的核心经营能力在行业中处于领先（ROE 9.2%），扣非基础上的持续增长更值得关注。

## 7. 未来可能的利空

### 利空 1：杠杆扩张激进（总资产+32%，短期借贷+94%）

**依据**

总资产从RMB 8,143亿扩张至RMB 10,773亿（+32%），短期借贷从RMB 1,045亿→RMB 2,026亿（+94%），杠杆倍数从4.25x升至5.20x。

**解读**

快速扩表可能源于做市交易、融资融券等业务需求，但也带来资产负债期限错配风险。一旦市场大幅下跌导致交易对手违约或资产减值，高杠杆将放大损失。

### 利空 2：税率波动大，利润可预测性降低

**依据**

有效税率从2024年的-1.1%（退税）升至2025年的11.0%，税项从+RMB 1.67亿变为-RMB 20.32亿。

**解读**

税率的剧烈波动（可能与递延税资产确认/转回、跨境税务安排相关）使得利润的可预测性降低。如果2026年税率进一步正常化至25%标准税率，将额外侵蚀约RMB 26亿利润。

### 利空 3：A股市场依然是核心驱动因素

**依据**

尽管华泰业务更均衡，但财富管理（+41%是因为A股交易量放大）和自营业务仍是主要收入来源。

**解读**

华泰虽然国际化程度领先，但国际业务仅贡献18%利润，A股市场走势仍是业绩的决定性因素。如果A股在2026年转冷，经纪和自营收入将明显回落。

## 8. 总结判断

**一句话结论**

华泰证券2025年扣非净利+80%至RMB 163亿、ROE 9.2%，是中国券商中业务最均衡（国际18%利润贡献+投行HK IPO第三+财富管理+41%）的头部标的——但杠杆从4.25x升至5.20x、短期借贷+94%的激进扩表需要关注。H股P/B 0.79x、P/E 8.8x的估值合理，基本面质量优于多数同行。

**核心看点**

1. **扣非净利+80%**：去除2024年非经常性损失后的真实经营改善
2. **国际业务18%利润贡献，ROE 13%**：中国券商国际化标杆
3. **财富管理+41%，投行+48%**：轻资产业务高增长
4. **ROE 9.2%**：行业领先水平

**核心风险**

1. **杠杆扩张激进**：总资产+32%，短期借贷+94%
2. **税率波动大**：有效税率-1.1%→11.0%
3. **A股市场依然是核心驱动**
