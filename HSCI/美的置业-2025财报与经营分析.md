# 美的置业（03990.HK）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

美的置业2025年实现营业收入RMB 41.49亿（+11.4%），毛利RMB 13.65亿（+2.5%），毛利率32.9%。股东应占溢利RMB 5.59亿（2024年亏损RMB 20.58亿，实现扭亏为盈）。PBT RMB 8.51亿（+24.1%）。核心净利润RMB 6.01亿（+19.3%）。EPS RMB 0.39（2024年-1.44）。DPS HK$0.34（中期0.15+末期0.19），派息比率76.4%。ROE 11.4%。NBV RMB 3.418。

### 1.2 增长与拖累因素

**2025年是重组后转型深化首年**。2023-2024年，美的置业完成了重大战略重组——将物业开发业务（含大量存货和负债）剥离，转型为轻资产服务平台。2024年因已终止经营业务亏损RMB 28.37亿导致全年巨亏。2025年不再有已终止业务影响，持续经营业务全面盈利。

增长驱动：（1）开发服务收入RMB 9.58亿（+227.1%），大幅增长；（2）毛利率32.9%（轻资产模式远高于原来的开发商15%毛利率）；（3）净财务由支出RMB 0.92亿转为收入RMB 0.20亿；（4）联营公司由亏转盈。拖累方面：（1）有效税率从29.1%升至32.3%；（2）NCI由正（+RMB 2.93亿，即NCI曾"分担"亏损）转为-RMB 0.16亿。

### 1.3 趋势判断

美的置业2025年完成了从重资产地产开发商向轻资产服务平台的华丽转身——收入RMB 41.5亿（仅为重组前RMB 736亿的5.6%），但毛利率32.9%远超原开发业务，ROE 11.4%、资产负债率47.7%、净现金RMB 8亿+。整体判断：**"脱胎换骨的轻资产转型——从736亿收入15%毛利率的重资产开发商变成41亿收入33%毛利率的服务平台，净现金+76%派息率+11.4%ROE，关键在于能否持续规模化增长"**。

## 2. 主营业务

### 2.1 公司简介

美的置业原为美的集团旗下的房地产开发商，2023-2024年完成重大重组，剥离了物业开发业务，转型为以开发服务、物管服务、资产运营和房地产科技为核心的轻资产服务平台。

### 2.2 主营业务构成

1. **开发服务**：2025年收入RMB 9.58亿（+227.1%），为房地产项目提供代建、工程管理等服务
2. **物管服务**：物业管理和社区增值服务
3. **资产运营**：投资性物业的运营管理（投资物业RMB 48.33亿）
4. **房地产科技**：智慧建筑、数字化解决方案

### 2.3 核心收入来源

重组后，公司从"卖房子"转变为"提供房地产相关服务"。开发服务（代建模式）是增长最快的板块（+227%），物管服务是稳定收入来源，资产运营提供租金收入。总收入RMB 41.5亿中，毛利率32.9%远高于传统开发商。

### 4.3 营收分析

**开发服务+227.1%是核心增长引擎**

代建模式下，公司为其他开发商或政府代建项目提供管理服务，收取管理费。这是轻资产、高毛利的商业模式——不需要承担土地和建设成本，只收取服务费。2025年开发服务从RMB 2.93亿暴增至9.58亿，说明代建业务快速拓展。

**毛利率32.9%——轻资产模式的优势**

相比重组前的物业开发业务（毛利率15%），轻资产服务业务的毛利率高达32.9%，盈利质量大幅提升。

## 6. 未来可能的利好

### 利好 1：轻资产转型成功——33%毛利率+11%ROE+净现金

**依据**

毛利率32.9%，ROE 11.4%，净现金RMB 8.16亿，资产负债率47.7%。

**解读**

重组后的美的置业已经不是传统地产股，而是一家轻资产服务公司。33%毛利率、11%ROE、净现金的财务特征更接近物管/服务公司而非开发商。这意味着估值方式应从P/B转向P/E。

### 利好 2：开发服务+227%——代建模式快速扩张

**依据**

开发服务收入RMB 9.58亿（+227.1%）。

**解读**

代建模式在行业低谷期具有独特优势——许多中小开发商缺乏管理能力，需要有品牌和管理能力的公司代建。美的品牌（美的集团旗下）提供了强大的品牌背书。

### 利好 3：高派息——76.4%派息率+HK$0.34 DPS

**依据**

DPS HK$0.34（+36%），派息比率76.4%。按HK$3.52股价计，股息率9.7%。

**解读**

76%的派息率极为慷慨，说明公司对盈利可持续性有信心，且轻资产模式下资本支出需求很低，可以将大部分利润回馈股东。

### 利好 4：美的集团品牌赋能

**依据**

美的集团是公司控股股东，在智能家居、暖通空调等领域具有强大的技术和品牌优势。

**解读**

"房地产科技"板块可以利用美的集团的IoT、智能家居技术，形成差异化竞争优势（智慧社区、绿色建筑解决方案）。

## 7. 未来可能的利空

### 利空 1：收入规模偏小——RMB 41.5亿能否持续放大

**依据**

总收入仅RMB 41.5亿，在重组前为RMB 736亿。公司需要证明轻资产模式可以持续规模化。

**解读**

代建/开发服务的天花板取决于行业代建需求的增长。如果房地产市场持续低迷，新开工面积下降，代建需求也可能受限。

### 利空 2：投资物业RMB 48亿——占总资产50%

**依据**

投资物业RMB 48.33亿，占总资产50%。

**解读**

这是剩余最大的"重资产"。如果商业地产市值下跌（零售/办公需求走弱），可能面临公允价值减值。投资物业的租金收入回报率也是关注焦点。

### 利空 3：毛利率下降趋势——32.9%（-2.8ppt）

**依据**

毛利率从35.7%下降至32.9%。

**解读**

虽然仍远高于传统开发业务，但方向是向下的。如果代建市场竞争加剧（更多公司进入代建领域），毛利率可能继续承压。

### 利空 4：流动资产净值大幅缩水——仅RMB 0.38亿

**依据**

流动资产净值从RMB 4.20亿缩至RMB 0.38亿（-90.9%）。短期借贷+802%至RMB 6.07亿。

**解读**

虽然整体净现金为正，但流动资产净值几乎为零，短期借贷大幅增加（长转短），需关注短期偿债压力。

## 8. 总结判断

**一句话结论**

美的置业2025年完成轻资产转型首年——收入RMB 41.5亿（+11.4%）、核心净利RMB 6.01亿（+19.3%），33%毛利率+11%ROE+净现金+76%派息率，从重资产开发商蜕变为轻资产服务平台。开发服务+227%是核心增长引擎，关键在于能否持续规模化。

**核心数据验证（etnet）**

| 指标 | 数值 | 关键判断 |
|---|---|---|
| PBT | RMB 8.51亿（+24.1%） | 持续经营强劲 |
| 毛利率 | 32.9%（-2.8ppt） | 仍远优于开发 |
| ROE | 11.4% | ✅ |
| 净现金 | RMB 8.16亿 | ⭐ |
| 投资物业 | RMB 48.33亿（占资产50%） | ⚠️重资产残余 |
| P/B | HK$3.52/NBV RMB 3.42≈HK$3.76，约0.94x | |
| P/E | HK$3.52/EPS RMB 0.39≈HK$0.43，约8.2x | |
| 股息率 | DPS HK$0.34/HK$3.52=9.7% | ⭐极高 |

**核心看点**

1. **开发服务能否保持高增长**：+227%后能否继续拓展
2. **毛利率能否企稳**：32.9%是否有进一步下行压力
3. **投资物业估值和租金回报**
4. **新业务（房地产科技）何时贡献规模收入**

**核心风险**

1. **收入规模偏小**：RMB 41.5亿能否持续放大
2. **投资物业减值风险**：占资产50%
3. **毛利率下滑**：-2.8ppt
4. **行业代建需求不确定性**
