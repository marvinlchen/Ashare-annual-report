# 瑞声科技（02018.HK）2025年财报整理

> 注：瑞声科技以人民币为报告货币。公司是全球领先的智能设备解决方案提供商，核心业务涵盖声学、光学、电磁传动及精密结构件、散热等，正从硬件供应商向AI感知与交互系统全球领导者转型。

## 1. 业绩总结

### 1.1 收入与利润表现

瑞声科技2025年度实现营业收入318.2亿元，同比 +16.4%，创历史新高。净利润25.1亿元，同比 +39.8%。毛利70.2亿元，毛利率22.1%（同比持平）。经营性现金流71.8亿元（+38.1%，创历史新高）。每股基本盈利2.18元。全年每股股息0.35港元（+45.8%），派息比率15%。

### 1.2 增长与拖累因素

核心驱动：(1) 电磁传动及精密结构件收入117.7亿（+21.3%）——智能手机+AI终端需求增长；(2) 散热业务16.7亿（+410.9%！）——AI服务器/端侧散热需求爆发；(3) 光学业务57.3亿（+14.5%）——毛利率持续改善至11.5%；(4) 车载声学41.2亿（+16.1%）。增长较慢：声学业务整体+1.7%——传统手机声学增长趋缓。

### 1.3 趋势判断

2025年是瑞声科技"AI驱动全线增长"的一年。利好方向：AI终端创新周期+散热业务爆发（+411%）+光学盈利持续改善+车载声学+16%+经营现金流创新高+净负债率仅2.1%；压力方向：传统声学增长趋缓（+1.7%）+毛利率22.1%持平未提升+资本开支28.3亿仍较大。核心看点：AI眼镜、人形机器人等新硬件形态能否驱动下一轮增长。

## 2. 主营业务

### 2.1 公司简介

瑞声科技是全球领先的智能设备解决方案提供商，为智能手机、汽车、AI终端等提供声学、光学、电磁传动、精密结构件和散热等核心零组件。

### 2.2 主营业务构成

1. **电磁传动及精密结构件**：触觉马达、金属/塑料机壳、精密结构件等
2. **光学**：摄像头模组、WLG晶圆级玻璃镜头、光学镜头
3. **声学**：扬声器、麦克风、车载声学系统
4. **散热（新兴）**：AI服务器、端侧设备散热解决方案
5. **传感器及半导体**：MEMS器件、半导体等

### 2.3 核心收入来源

电磁传动及精密结构件是第一大收入来源（117.7亿），得益于智能手机和AI终端对精密零件的需求。光学（57.3亿）和声学（~57亿）为第二和第三大业务。散热业务虽然规模尚小（16.7亿）但增速惊人（+411%），是未来增长引擎。

## 4. 营收分布和增长

### 4.1 按业务划分

| 业务板块 | 2024年收入 | 2025年收入 | 同比增速 | 备注 |
|---|---:|---:|---:|---|
| 电磁传动及精密结构件 | ~97.0亿 | 117.7亿 | +21.3% | 第一大业务 |
| 光学 | ~50.0亿 | 57.3亿 | +14.5% | GM 11.5% |
| 声学（消费+车载） | ~56亿 | ~57亿 | +1.7% | 车载41.2亿+16.1% |
| 散热（新兴） | ~3.3亿 | 16.7亿 | +410.9% | 爆发 |
| 传感器及半导体等 | ~67亿 | ~69.5亿 | — | |
| **合计** | 273.4亿 | 318.2亿 | +16.4% | |

### 4.2 营收分析

**主要增长来源**

(1) 电磁传动及精密结构件+21.3%至117.7亿——智能手机+AI终端精密零件需求旺盛；(2) 散热+410.9%至16.7亿——AI服务器散热需求爆发，大客户合作+出货放量；(3) 光学+14.5%至57.3亿——WLG技术突破+毛利率改善；(4) 车载声学+16.1%至41.2亿——汽车智能化趋势。

**主要拖累因素**

声学整体仅+1.7%——传统手机声学进入成熟期，增长空间有限。消费声学增长放缓，需要车载声学（+16.1%）来弥补。

**结构变化**

散热业务从微不足道（~3.3亿）跃升至16.7亿，占比从~1%升至~5%。光学六年复合增长率超32%。公司正从"声学为核心"向"多元化+AI感知"转型。

## 5. 利润分布和增长

### 5.1 整体利润表现

| 指标 | 2025年 | 2024年 | 同比增速 | 备注 |
|---|---:|---:|---:|---|
| 毛利润 | 70.2亿 | ~60.4亿 | +16.2% | |
| 净利润 | 25.1亿 | 17.95亿 | +39.8% | |
| 毛利率 | 22.1% | 22.1% | 持平 | |

### 5.2 利润分析

**净利增速（+39.8%）远超收入增速（+16.4%）的原因**

(1) 光学业务盈利性持续改善——毛利率提升至11.5%（此前接近盈亏平衡或亏损）；(2) 高盈利水平业务（声学、电磁传动）收入增长——这些业务毛利率远高于光学；(3) 运营效率提升——存货周转天数保持历史最佳水平。

**光学业务的利润转折**

光学是利润增长的最大边际贡献者——从亏损/微利到毛利率11.5%，意味着57.3亿收入中约6.6亿转化为毛利。这种盈利改善趋势如果延续（目标毛利率提升至15-20%），将显著增厚集团利润。

## 6. 未来可能的利好

### 利好 1：散热业务+410.9%——AI硬件的"卖水人"

**依据**

散热业务收入从~3.3亿飙升至16.7亿（+410.9%）。大客户合作+出货放量。AI服务器和端侧设备对散热需求爆发。

**解读**

AI训练和推理对算力的需求推动服务器/GPU散热成为高增长赛道。瑞声凭借精密制造能力切入散热市场，与大客户（如英伟达供应链）的合作已开始放量。如果2026年AI算力投资继续增长，散热业务有望达到30-50亿规模，成为第四大利润支柱。

### 利好 2：光学业务毛利率持续改善——从"拖累"变"增长极"

**依据**

光学收入57.3亿（+14.5%），毛利率提升至11.5%。WLG晶圆级玻璃镜头技术突破。六年复合增长率超32%。

**解读**

光学曾经是拖累利润的业务（低毛利率甚至亏损），现在毛利率已提升至11.5%且仍在改善中。如果达到15-20%的目标毛利率，光学将从"拖累"完全转变为"利润增长极"。WLG技术是差异化竞争壁垒。

### 利好 3：AI终端创新周期——眼镜、机器人、汽车

**依据**

公司战略定位为"AI感知与交互系统全球领导者"。智能眼镜、人形机器人、汽车智能化正在开启新硬件周期。车载声学+16.1%至41.2亿。

**解读**

AI终端（智能眼镜、AR/VR、人形机器人）的兴起为瑞声的声学、光学、触觉、散热等多元技术能力提供了全新的应用场景。这些新品类的单机价值量（ASP）远高于手机，如果成功渗透，将打开远超手机市场的增长空间。

## 7. 未来可能的利空

### 利空 1：声学业务仅+1.7%——传统核心增长趋缓

**依据**

声学整体+1.7%。传统手机声学市场进入成熟期。

**解读**

声学是瑞声的起家业务和传统利润来源，但全球智能手机市场增长放缓意味着声学的增长天花板已经接近。虽然车载声学（+16.1%）提供了增量，但整体声学增速仅1.7%。

### 利空 2：毛利率22.1%持平——规模效应未显现

**依据**

毛利率22.1%连续两年持平。收入+16.4%但毛利率未提升。

**解读**

收入创新高但毛利率未提升，表明新增收入（特别是散热和光学）的毛利率仍然低于整体平均水平，拉低了综合毛利率。如果未来高增业务（散热、光学）的毛利率提升不达预期，整体毛利率可能面临下行压力。

### 利空 3：资本开支28.3亿——持续高投入

**依据**

资本开支28.3亿元。经营现金流71.8亿元。

**解读**

虽然自由现金流充裕（~43.5亿），但28.3亿的资本开支反映了公司需要持续大规模投资以维持技术领先和产能。如果行业需求不达预期或技术路线判断失误，大额资本开支可能成为负担。

## 8. 总结判断

**一句话结论**

瑞声科技2025年收入318.2亿元（+16.4%创新高）、净利润25.1亿元（+39.8%）、经营现金流71.8亿元（+38.1%创新高），电磁传动+21.3%、散热+410.9%、光学+14.5%驱动全线增长，光学毛利率持续改善至11.5%，净负债率仅2.1%——"AI驱动全线增长"，散热业务爆发是最大亮点，从硬件供应商向AI感知系统领导者转型。

**核心看点**

1. 散热业务+410.9%至16.7亿——AI硬件需求爆发
2. 光学毛利率改善至11.5%——从"拖累"变"增长极"
3. 电磁传动+21.3%至117.7亿——第一大业务强劲
4. 经营现金流71.8亿+38.1%——创历史新高
5. 净负债率2.1%+现金86.1亿——极稳健

**核心风险**

1. 声学仅+1.7%——传统核心增长趋缓
2. 毛利率22.1%持平——规模效应未显现
3. 资本开支28.3亿——持续高投入
4. AI终端（眼镜、机器人）商业化节奏不确定
