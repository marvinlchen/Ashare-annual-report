# 烽火通信2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

烽火通信（600498.SH）**2025年年度报告尚未正式披露**（预计2026年4月），但**Q1-Q3已披露**：营业收入**170.72亿元，同比-19.30%**；归母净利润**5.13亿元，同比+30.28%**；扣非归母净利润**5.01亿元，同比+35.75%**。**Q3单季营收5.95亿元（-18.48%）+归母净利2.26亿元（+28.14%）**。**毛利率22.42%（+0.29pct，连续5年上涨）**+净利率2.96%（+1.13pct）+EPS 0.43元（+30.30%）+ROE 3.59%（+0.32pct）+资产负债率**61.42%（-3.55pct）**。**研发费用同比-15.67%+财务费用-31.78%**推动利润率提升。**呈现"营收-19%大幅下滑+利润+30%逆势增长+毛利率连升+费用优化"格局**——业务结构+成本控制驱动利润质量提升。2024年基数：营收285.49亿（-8.29%）+归母7.03亿（+39.05%）+扣非6.53亿（+43.70%）+每10股派1.79元。

### 1.2 增长与拖累因素

**主要增长驱动**：（1）**毛利率连续5年上涨至22.42%**（+0.29pct）；（2）**费用控制优化**——研发费-15.67%+财务费-31.78%+销售费+2.44%+管理费+3.04%；（3）**净利率提升1.13pct至2.96%**；（4）**ROE+0.32pct至3.59%**；（5）**AI算力+电力+金融等行业AI算力服务器规模销售**；（6）**400G/800G高速光模块规模交付**；（7）**"万兆光网+东数西算+数字中国"**国家重大工程参与；（8）**泰国+匈牙利研发制造基地启动试投产**；（9）**每股收益+30.30%**。**主要拖累**：（1）**营收Q1-Q3 -19.30%**大幅下滑——光通信市场投资放缓；（2）**Q3单季营收-18.48%**持续下滑；（3）**数据网络产品-27.44%（2024）**+**光纤及线缆-12.77%**；（4）**资产负债率虽降但仍偏高61.42%**；（5）**市场份额+行业竞争激烈**；（6）**运营商资本开支下降**；（7）**研发费用-15.67%虽然利于利润但可能影响长期竞争力**；（8）**财务费用大幅下降需关注原因**。

### 1.3 趋势判断

烽火通信是**中国信科集团（武汉邮科院）旗下光通信+ICT基础设施龙头**。整体呈现**"营收持续下滑+利润逆势高增+毛利率连升+费用优化+AI算力新赛道+国际化布局"**格局。**核心矛盾**：**营收-19.30%**反映**光通信+运营商传统市场萎缩**——5G投资周期见顶+运营商集采价格压力+光纤光缆市场疲弱。但**归母+30.28%+扣非+35.75%**显示**业务结构优化+费用控制+毛利率提升**成效——从量转向质。**AI算力+400G/800G光模块+数据中心**是未来核心增长点——**全球AI驱动数据中心光器件需求暴增**+**数据中心光纤光缆需求+75.9%**。**泰国+匈牙利基地**国际化布局是应对地缘政治的关键。**"万兆光网+东数西算+数字中国"**国家战略参与提供订单可见度。但**营收持续下滑+行业竞争+资本开支周期**是长期压力。**关键看**2026年营收能否企稳+AI算力业务放量+国际化突破+光模块收入增长+研发投入恢复。2024年分红每10股1.79元。

## 4. 营收分布和增长

### 4.1 按业务划分

**2024年数据**：
- **通信系统设备223.13亿（78.2%）-6.19%**
- **光纤及线缆46.99亿（16.5%）-12.77%**

4. **运营商集采价格压力**
5. **5G建设周期见顶+过渡期**

**利润增长原因**

1. **毛利率提升**（结构优化+高毛利产品占比提升）
2. **费用控制**（研发-15.67%+财务-31.78%）
3. **降杠杆+现金流改善**
4. **AI算力业务高毛利贡献**
