# 珂玛科技2025年财报整理

> 说明：截至 2026-04，珂玛科技 2025 年年报**尚未正式披露**。本文基于**2025年度业绩预告(2026-01披露)+2025年三季报(2025-10披露)+苏州铠欣收购案+机构研报**整理。

## 1. 业绩总结

### 1.1 收入与利润表现

- **2025年预告营业收入 10.6-10.8亿元，同比 +23.63%~+25.96%**⚡⚡；
- **2025年预告归母净利润 2.86-3.36亿元，同比 -8.00%~+8.00%**（波动区间）；
- **2025年预告扣非归母净利润 2.85-3.34亿元，同比 -6.92%~+9.35%**；
- **Q1-Q3 营业收入 7.94亿元，同比 +28.86%**⚡⚡；
- **Q1-Q3 归母净利润 2.45亿元，同比 +8.29%**⚡；
- **Q1-Q3 毛利率 53.94%（-5.48pct同比）**⚠️；
- **Q1-Q3 净利率 30.56%（-6.09pct同比）**；
- **Q3单季营收 2.74亿(+18.10%)**⚡；
- **Q3单季归母 0.7266亿(-16.16%)**⚠️⚠️ 增速放缓；
- **Q3单季毛利率 51.12%**⚠️；
- **H1营收 5.15-5.25亿(+33.93%~+36.53%)**⚡⚡；
- **H1扣非归母 1.635-1.745亿(+19.88%~+27.94%)**⚡⚡。

### 1.2 增长与拖累因素

- **主要增长驱动**：①**陶瓷加热器批量供应**（2023Q4起+SACVD/PECVD/LPCVD/ALD/PVD/激光退火设备装配）；②**静电卡盘+超高纯碳化硅套件小批量出货**（12英寸ICP/CCP Monopolar静电卡盘通过验证）；③**全球半导体设备市场复苏+国产替代加速**；④**H1收入+33.93-36.53%+H1扣非+19.88-27.94%**；⑤**6英寸非渗硅套件通过北方华创+Fab端验证+取得多套批量订单**；⑥**先进陶瓷材料零部件91.74%占比**+**半导体领域结构件H1+66.06%**；⑦**收购苏州铠欣73%股权布局碳化硅**。
- **主要拖累因素**：①**收入结构变化**—陶瓷结构件（毛利较低）增速高于模块化产品（毛利较高）→**毛利率同比-5.48pct**；②**研发投入加大+研发团队扩充**；③**苏州铠欣并表**（2025下半年完成）产生的整合成本；④Q3归母-16.16%增速明显放缓。
- 公司整体呈现"**半导体陶瓷零部件国产替代+北方华创/Fab客户验证+陶瓷加热器批量+静电卡盘突破+碳化硅收购+毛利率承压+业绩短期放缓**"的高端制造+国产替代长期成长格局。

### 1.3 趋势判断

- **未来主要利好方向**：①**静电卡盘（12英寸Monopolar）小批量→批量爆发**（2026年关键）；②**超高纯碳化硅套件通过北方华创验证+批量订单加速**；③**苏州铠欣（CVD碳化硅涂层+块体陶瓷）贡献增量**（Si/SiC/GaN外延规模化应用）；④陶瓷加热器持续放量+产线搬迁完成；⑤**国产替代大潮**（国内半导体设备+晶圆厂扩产）；⑥全球半导体设备复苏+2026年周期向上；⑦北方华创/中微等国产设备龙头合作深化。
- **未来主要压力方向**：①毛利率-5.48pct下行压力能否稳住；②研发费用+并购整合成本持续上升；③国内半导体设备竞争加剧（富乐德/神工股份/中材高新）；④海外厂商（应用材料/Lam/TEL）竞争；⑤半导体周期不确定性。
- **整体趋势判断**：**2026年在静电卡盘批量+碳化硅套件爆发+苏州铠欣并表完整年+国内晶圆厂扩产持续**驱动下+**营收有望继续保持25-35%增长**+**扣非净利增速或重回20-30%**+**毛利率随模块化产品占比回升**+**国产半导体陶瓷零部件龙头地位进一步巩固**。

## 2. 主营业务

### 2.1 公司简介

珂玛科技(301611.SZ)是**国内半导体先进陶瓷零部件国产替代龙头+北方华创核心供应商+深度绑定Fab+先进陶瓷材料+表面处理服务一体化**。主营**高纯度氧化铝+高导热氮化铝+静电卡盘（ESC）+陶瓷加热器+超高纯碳化硅套件+半导体设备表面处理+苏州铠欣CVD碳化硅涂层+块体陶瓷零部件**。下游覆盖**半导体刻蚀/薄膜沉积/光刻/扩散/退火**全工艺。

### 2.2 主营业务构成

1. **先进陶瓷材料零部件（核心91.74%）**：
   - **静电卡盘（ESC）**：12英寸ICP/CCP Monopolar静电卡盘
   - **陶瓷加热器**：SACVD/PECVD/LPCVD/ALD/PVD/激光退火
   - **超高纯碳化硅套件**：6/8/12英寸
   - 高纯氧化铝+氮化铝+石英等结构件
2. **表面处理服务（7.20%）**：
   - 半导体设备零部件表面处理+清洗
3. **其他（1.06%）**：
   - 模块化产品+其他
4. **苏州铠欣（2025H2新增）**：
   - CVD碳化硅涂层+块体陶瓷零部件
   - 碳化硅刻蚀环+喷淋头+12吋外延石墨基座

### 2.3 核心收入来源

- **先进陶瓷材料零部件**为绝对核心（91.74%）+**陶瓷加热器**（2023Q4起批量+产线搬迁）+**静电卡盘**（小批量进爆发前夜）+**超高纯碳化硅套件**（北方华创验证通过）；
- **北方华创**为核心客户+**国内晶圆厂直接供应**+**半导体设备复苏**带动需求；
- **半导体结构件H1+66.06%**驱动+**模块化产品H1+15.66%**稳健；
- **苏州铠欣H2并表**→**2026年贡献完整年收入**。
