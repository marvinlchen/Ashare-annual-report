# 华峰化学2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

华峰化学2025年实现营业收入241.98亿元，同比 -10.15%；归母净利润18.58亿元，同比 -16.32%；扣非归母净利润17.78亿元，同比 -14.65%。经营活动现金流37.40亿元，同比 +23.66%。典型的"量增价跌"格局——三大板块销量均增长，但产品价格普遍下滑拖累收入和利润。

- 全年营业收入241.98亿元，同比 -10.15%（2024年约269.33亿元）
- 全年营业利润23.69亿元
- 全年归母净利润18.58亿元，同比 -16.32%（2024年约22.20亿元）
- 全年扣非归母净利润17.78亿元，同比 -14.65%（2024年约20.83亿元）

### 1.2 增长与拖累因素

收入和利润双降的核心不是产销崩塌，而是化工品价格持续承压。三大板块（化学纤维+7.65%、化工新材料+11.12%、基础化工产品+1.73%）销量均增长，但年报明确指出"报告期内售价走势均为下降"。己二酸板块受冲击最大，收入-17.01%、毛利率仅3.92%。现金流反而改善（+23.66%），说明经营质量并未同步恶化。

- 主要拖累因素：行业供给偏强+价格竞争激烈，己二酸板块尤其承压
- 积极因素：销量全面增长+现金流改善+一体化优势维持
- 利润质量：扣非占归母95.70%，非经常性损益仅0.80亿元

### 1.3 趋势判断

华峰化学仍深陷化工周期底部，"量增价跌"格局短期难以彻底扭转。但公司作为氨纶/聚氨酯原液/己二酸全球产量第一的龙头，一旦行业出清+价格企稳，利润弹性会很大。

- 未来主要利好方向：行业出清+差异化产品+新应用场景+产能扩张巩固龙头
- 未来主要压力方向：产品价格承压+己二酸低毛利+氨纶产能释放+库存上升
- 整体趋势判断：龙头竞争力未损，但需等待周期回暖

## 2. 主营业务

### 2.1 公司简介

华峰化学（002064.SZ）是全球领先的聚氨酯材料一体化企业，氨纶纤维、聚氨酯原液和己二酸产量均为全球第一，以一体化产业链对冲周期波动。

### 2.2 主营业务构成

1. **化学纤维（氨纶）**：纬编、经编、包纱、纸尿裤等全用途系列，产能产量全球第一/中国第一，收入占比36.30%
2. **基础化工产品（己二酸）**：应用于尼龙66和聚氨酯，国内市场份额超40%、产量全球第一，收入占比35.92%
3. **化工新材料（聚氨酯原液）**：应用于鞋材、汽车配件、高铁垫片等，产量全球第一，收入占比22.30%
4. **其他业务**：物流服务等，收入占比5.48%

### 2.3 核心收入来源

化学纤维（87.84亿，36.30%）和基础化工产品（86.93亿，35.92%）体量接近，合计超七成收入。化工新材料（53.95亿，22.30%）是第三大板块，盈利能力最好。公司以国内市场为主（82.11%），海外占17.89%。

化学纤维和基础化工产品体量接近，各占约36%。化工新材料占比22%，虽规模最小但毛利率最高（20.63%），是利润的重要稳定器。"量增价跌"格局下，收入结构变化不大。

## 6. 未来可能的利好

### 利好 1：行业出清后头部集中度提升

**依据**

氨纶和聚氨酯原液行业面临中小产能出清。公司三大主产品产量均为全球第一，国内己二酸市场份额超40%。

**解读**

龙头企业在行业洗牌中受损最小、恢复最快。一旦中小产能退出+价格企稳，份额和盈利同步改善。

### 利好 2：一体化产业链增强抗周期能力

**依据**

公司拥有氨纶→聚氨酯原液→己二酸的联动布局，既能内部消化部分原料，也能在景气切换时对冲单一产品波动。

**解读**

一体化结构使公司在深度下行周期中仍能保持正利润和强现金流，穿越周期能力强于单品企业。

### 利好 3：下游新场景拓展+差异化产品

**依据**

氨纶向医疗、汽车内饰等新领域拓展。聚氨酯原液向汽车、高铁、3D打印、新能源电池等场景延伸。己二酸受益于尼龙66国产化和TPU/PBAT扩产。

**解读**

新场景需求增量有望部分抵消传统下游景气波动。差异化产品（高回弹经编氨纶、耐高温氨纶等）提升产品附加值。

### 利好 4：现金流改善提供财务安全垫

**依据**

2025年经营现金流37.40亿元，同比 +23.66%，远高于净利润18.58亿元。资产负债率仅25.88%。

**解读**

低杠杆+强现金流意味着公司可安全穿越周期底部，甚至在行业出清期通过扩产巩固龙头地位。

## 7. 未来可能的利空

### 利空 1：化工周期尚未反转，量增价跌可能延续

**依据**

2025年三大板块售价均下降。氨纶产能同比增长13.1%，供给持续释放。年报提示市场变化和行业周期风险。

**解读**

如果供给继续偏强+需求恢复缓慢，价格可能继续承压，销量增长无法弥补价格下跌对利润的拖累。

### 利空 2：己二酸板块盈利极弱

**依据**

己二酸收入86.93亿但毛利率仅3.92%（毛利3.41亿）。价格先扬后抑，苯己差维持低位。库存同比增长47.19%。

**解读**

己二酸占收入36%但利润贡献仅12%，是最大的利润短板。如果价格继续走弱或库存消化不畅，可能进一步拖累整体盈利。

### 利空 3：氨纶产能集中释放+行业竞争加剧

**依据**

2025年中国大陆氨纶产能同比增长13.1%。公司自身重庆氨纶扩建项目持续投产（2025年新增15万吨）。

**解读**

产能释放速度快于需求增长，行业价格战可能延续。虽然龙头成本优势明显，但价格底部持续时间存在不确定性。

### 利空 4：原材料波动+海外需求偏弱

**依据**

纯苯、PTMG、MDI在采购成本中占比较高。海外收入-16.74%，下滑快于国内（-8.57%）。

**解读**

原材料反弹而产品价格无法跟涨会压缩利润。海外需求偏弱意味着出口无法有效对冲国内压力。

## 8. 总结判断

**一句话结论**

华峰化学2025年收入-10.15%、归母-16.32%，典型"量增价跌"——三大板块销量均增长但价格普遍下滑，己二酸板块尤其承压。但公司作为全球产量第一的一体化龙头，现金流+23.66%、资产负债率仅26%，穿越周期能力突出。

**核心看点**

1. 氨纶/己二酸价格能否在行业出清后企稳回升
2. 聚氨酯原液在新应用场景（汽车、高铁、新能源）的放量进度
3. 己二酸高库存（+47.19%）能否随需求恢复消化
4. 重庆氨纶扩建项目产能释放后的市场消化能力

**核心风险**

1. 化工周期持续底部，价格反弹时间不确定
2. 己二酸毛利率仅3.92%，进一步恶化可能接近亏损
3. 氨纶产能快速释放（+13.1%），价格战延续
4. 海外需求偏弱，出口无法有效对冲
