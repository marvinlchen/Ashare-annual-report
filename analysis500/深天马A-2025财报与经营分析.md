# 天马微电子股份有限公司 (000050.SZ) 2025年财报整理

> 数据口径说明：本报告数据来源于天马微电子股份有限公司（深交所主板，证券简称"深天马A"，证券代码 000050.SZ）2025 年年度报告及摘要（披露日 2026 年 3 月，通过巨潮资讯网/证券之星/同花顺/新浪财经获取）。金额披露币种为人民币（CNY），以亿元或万元为单位。利润口径以企业会计准则为基础，归母净利润口径为"归属于上市公司股东的净利润"。同比增速统一写为"同比 +X.X% / -X.X%"。公司财年为自然年（1 月–12 月），"2025 年"指截至 2025 年 12 月 31 日的财年。天马是航天科工集团旗下主要的中小尺寸显示面板龙头，坚持"2+1+N"战略（两大核心事业 + 一条新技术曲线 + N 个专业显示赛道），以车载显示（连续 6 年全球出货量第一、车载 HUD 2024 年登顶全球第一）和柔性 AMOLED（国内第二、全球第三）为核心，业务覆盖车载 / 手机 / IT 及运动健康 / 专业显示 / Micro-LED / MPG 非显示应用等多条产品线，拥有多条 LTPS、AMOLED、LCD、Micro-LED 产线。

## 1. 业绩总结

### 1.1 收入与利润表现

深天马 A 2025 年实现营业收入 362.27 亿元（同比 +8.16%），归母净利润 1.67 亿元（2024 年亏损 6.69 亿元，扭亏为盈，同比 +125.03%），扣非归母净利润 -5.74 亿元（2024 年 -22.20 亿元，亏损同比收窄约 74%），基本每股收益 0.07 元（同比 +125.07%）；毛利率 16.24%（同比 +22.32% 相对，即 +~2.96pp 绝对），净利率 0.31%（同比 +114.81%）；经营活动现金流净额 69.88 亿元（同比 +21.49%），每股经营现金流 2.84 元；加权平均 ROIC 约 1.4%（10 年中位数 2.22%）；每股净资产 11.09 元。Q4 单季营收 95.64 亿元（+0.88%）但归母净亏损 1.46 亿元（同比减亏 25.63%），年末集中计提减值拖累全年盈利表现。2025 年度不派发现金红利、不送红股、不以公积金转增股本；管理层同期推出《估值提升计划》、控股股东自愿承诺不减持、回购股份方案等市值管理措施。

整体看，归母扭亏为盈但扣非仍亏—归母转正主要依赖约 7.41 亿元的非经常性收益（政府补贴、资产处置、税收等），核心主业经营层面仍处于亏损，但亏损大幅收窄（从 -22.2 亿 → -5.74 亿，改善 16.5 亿元），主营盈利拐点已近在咫尺。经营现金流远超报表利润，净经营资产收益率从 2023 年的 -3.9%、2024 年的 -1.3% 回升至 2025 年的 +0.2%，反映主业现金创造能力持续修复。

### 1.2 增长与拖累因素

**收入端增长驱动**（分业务同比增速来自 2025 年全年/前三季度披露）：(1) 车载业务全年销售额同比 +~18%（前三季度 +24%），连续 6 年全球出货量第一，传统车显 + 汽车电子 + 新能源车业务三线并举；(2) 专业显示业务前三季度 +22-25%，工控、医疗、航空等应用规格升级驱动；(3) IT 及运动健康显示营收 + 利润同比双增长；(4) 手机显示前三季度收入基本稳定，但柔性 AMOLED 占比提升、产品结构改善，毛利率修复；(5) Micro-LED 产品 2025 年实现小批量出货，并成功点亮首款车载标准产品。

**利润端改善驱动**：(1) 毛利率从 ~13.28% 提升至 16.24%（+~2.96pp），受益于高毛利车载占比提升 + LTPS 技术加速渗透 + 产品尺寸升级 + 上游关键材料国产化 + 全维度降本；(2) 三费（销售 + 管理 + 财务）合计 20.06 亿元，占营收比 5.54%（同比 -1.2pp，-11.28%），费用管控成效显著；(3) 汽车电子业务 2025 年前三季度实现盈利突破。

**利润端仍承压因素**：(1) 面板行业产能过剩（国内多条 OLED 产线投产）导致价格竞争激烈，柔性 AMOLED 手机产品价格阶段性承压；(2) AMOLED 产线折旧和研发投入大（公司业绩主要依靠研发及营销驱动）；(3) Q4 单季亏损 1.46 亿元，年末集中计提减值损失；(4) 扣非净利润仍为 -5.74 亿元，核心主业盈利尚未转正。

### 1.3 趋势判断

显示面板行业正处于从 LCD 向 OLED / Micro-LED 的技术转型期，公司作为国内中小尺寸面板龙头之一，在车载显示领域全球第一的地位稳固，车载 + 专业显示 + 汽车电子 + Micro-LED 四线并举构成中长期结构性增长主线。短期看扣非扭亏仍需时间，但毛利率改善（+2.96pp）+ 亏损收窄（-74%）+ 经营现金流持续高增（+21.49% 至 69.88 亿元）三大积极信号已汇聚，若 2026 年车载毛利率继续提升 + 消费类面板价格企稳，扣非有望接近盈亏平衡。核心风险在于面板价格下行、AMOLED 产线折旧与产能爬坡进度、Micro-LED 商业化节奏、以及行业竞争加剧下的毛利率弹性。

## 2. 主营业务

### 2.1 公司简介

天马微电子股份有限公司（深天马 A，000050.SZ）成立于 1983 年，1995 年深交所 A 股上市，是中国航天科工集团旗下以中小尺寸显示面板为核心业务的国家级高新技术企业。公司主营研发、设计、生产和销售各类显示产品及模组，产品基本实现中系 / 欧系 / 美系 / 日韩系等国际主流品牌全覆盖，服务奔驰、宝马、奥迪、大众、丰田、通用、福特、比亚迪、吉利、理想、蔚来、小鹏等主流整车厂商。公司坚持"2+1+N"发展战略：车载显示 + 柔性 AMOLED 两大核心、Micro-LED 新技术曲线、N 个专业显示赛道。在全国运营多条 LTPS / LCD / AMOLED / Micro-LED 产线（主要基地位于上海、厦门、武汉、成都、深圳）。2025 年深化多元化布局，MPG（多项目玻璃基板技术平台）推动面板制造技术向智能天线、微流控、智能调光、大面积指纹识别、柔性传感器等非显示应用延伸。

### 2.2 主营业务构成

1. **车载显示（核心战略业务）**：2025 年全年销售额同比 +~18%（前三季度 +24%），连续 6 年全球车规 TFT-LCD、车载仪表显示出货量全球第一，车载抬头显示（HUD）2024 年登顶全球第一。车载业务分为三大板块：(a) **传统车载显示**（仪表盘、中控屏、副驾屏）稳中有升；(b) **汽车电子**（2025 年前三季度同比 +30%，前三季度实现盈利突破）；(c) **新能源汽车业务**（国内 NEV 渗透率提升快速受益，出货增速高于整体车载）。公司主打 LTPS 车载 + 多屏联动 + 屏上集成解决方案，定制化程度高。
2. **柔性 AMOLED 手机显示（核心战略业务）**：国内第二、全球第三，拥有多条柔性 AMOLED 产线。2025 年受行业价格竞争影响营收相对稳定，但柔性 OLED 占比持续提升，产品结构持续向高端化（中尺寸、穿戴）升级，产线能力 + 全维度降本推进毛利率修复。
3. **专业显示**：工控（工业控制显示）、医疗（医疗诊断显示）、航空（座舱/机载显示）、安防 / POS 等专业显示领域，2025 年前三季度营收同比 +22-25%，规格升级驱动 ASP 提升。
4. **IT 及运动健康显示**：笔记本电脑、平板、智能穿戴、运动手表等；2025 年前三季度营收、利润同比双增长，出货量大幅增长。
5. **Micro-LED（新技术曲线）**：2017 年开始布局，2022 年合资投建一条从巨量转移到显示模组的全制程 Micro-LED 产线，2024 年点亮，2025 年实现标准化显示单元模块小批量出货并点亮首款车载标准产品；在大尺寸拼接（108 寸 / 135 寸 4K+ 无缝拼接）、极致对比度、宽广色域、超高亮度领域突破。
6. **MPG 非显示应用平台**：稳步推进面板级超材料智能天线、微流控、智能调光、大面积指纹识别、柔性传感器等孵化 + 部分小批量，将面板制造技术赋能更广泛应用。

### 2.3 核心收入来源

(1) 按业务结构：车载业务（车载显示 + 汽车电子 + 新能源车）是公司最核心的收入增长和利润贡献来源，叠加专业显示、IT 及运动健康显示合计占比估计超 55-60%，已成为抗消费电子周期的"护城河"；柔性 AMOLED + LCD 消费类手机显示仍占较大比例（估计 35-40%），但受价格竞争影响利润贡献有限。(2) 按客户结构：车载客户覆盖中 / 欧 / 美 / 日韩系主流整车厂；消费类客户覆盖国内外一线品牌。(3) 按地区结构：年报未单独披露境内 / 境外收入占比，但车载业务海外占比估计 50%+，公司长期"国际化"战略持续深入。(4) 战略定位："2+1+N"战略下，车载显示和柔性 AMOLED 双核心 + Micro-LED 新曲线 + N 个专业显示赛道，多元化布局降低单一业务波动。

**资产负债率变化**

合并资产负债率约 62%，有息资产负债率 38.08%（推算有息负债规模约 292 亿元）。总资产规模同比 -5.5%，主要源于固定资产折旧消耗 + 应收账款周转优化 + 部分资产减值（AMOLED 产线折旧计提）。对于以晶圆 + 蒸镀 + 模组为主的重资产面板制造行业，负债率水平属正常范围，略低于京东方（~55-60%）、TCL 科技（~65%）。

**偿债压力判断**

经营现金流 69.88 亿元（远超 1.67 亿元净利润）说明主业现金创造能力良好；然而证券之星财报体检指出 **货币资金 / 流动负债仅 25.67%** 显示短期流动性偏紧，**有息资产负债率 38.08%** 说明刚性财务费用压力较大，**应收账款 / 利润高达 6,459%** 说明客户回款效率对公司经营影响极大。整体看：长期偿债能力稳定（航天科工背景 + 银行授信充足）；短期偿债压力需关注流动性指标改善。扣非持续亏损意味着公司盈利尚不足以完全覆盖折旧和研发投入，需关注未来两年盈利能否持续修复。

**商誉风险判断**

年报未单独披露具体商誉规模。公司历史上通过收购（如与台湾地区 LCD 技术合作、与海外 OLED 技术合作）完成过跨境整合，账面商誉规模历史上较小。当前 Micro-LED 合资产线形成的长期股权投资 + 少量商誉可能存在，但 Micro-LED 已进入小批量出货阶段，减值风险可控。

**数据口径说明**

以上数据采用 `中国会计准则` 合并口径，金额单位 `人民币亿元`。净利润默认采用 `归属于上市公司股东的净利润`。"毛利润"按毛利率反推估算；"净经营利润 / 净经营资产 / 营运资本"来自证券之星基于公开财报的财务分析工具。部分 2024 年对比值根据同比增速或历史披露反推。2024 年营收数据以证券之星披露的 334.94 亿元为准，此前估算的 335.00 亿元为接近值。

## 6. 未来可能的利好

### 利好 1：车载显示龙头地位稳固 + 汽车智能化带来结构性增量 + HUD 全球第一

**依据**

(1) 车载显示出货量连续 6 年全球第一（TFT-LCD + 车载仪表），车载 HUD 2024 年首次登顶全球第一；(2) 2025 年车载业务全年销售额 +~18%（前三季度 +24%），汽车电子业务 +30%，新能源车业务持续快速增长；(3) 客户覆盖中 / 欧 / 美 / 日韩系主流品牌；(4) LTPS 技术加速渗透 + 产品尺寸升级 + 屏上集成 + 模组复杂度提升不断推高 ASP。

**解读**

车载显示是面板行业中增长最确定、壁垒最高的细分赛道，受益于全球汽车智能化（单车屏数量从 1-2 屏提升至 5-10 屏）、电动化（新能源车渗透率加速提升）、规格升级（从 LCD → LTPS → OLED）三重驱动。公司在车规级认证（AEC-Q100 / IATF 16949）、客户关系（长达 10+ 年供应）和产品可靠性方面具备深厚积累，竞争对手（JDI、LG 显示、友达、群创）在车载领域规模和定制化能力均不及天马。展望 2026-2028 年，全球车载显示面板市场规模有望从 2025 年约 150 亿美金增长至 200 亿美金（CAGR ~10%），公司作为龙头有望持续受益。

### 利好 2：毛利率持续改善 + 扣非亏损大幅收窄 + 2026 年接近盈亏平衡

**依据**

(1) 2025 年毛利率从 ~13.28% 提升至 16.24%（+2.96pp）；(2) 扣非亏损从 -22.20 亿元收窄至 -5.74 亿元（改善 74%）；(3) 净经营利润从 -21.1 亿 → -6.91 亿 → +1.11 亿（三年完成主业盈利修复）；(4) 三费占营收比从 6.74% 降至 5.54%（-1.2pp）。

**解读**

若毛利率改善趋势延续（车载 + 专业显示占比提升 + LTPS 技术渗透 + 全维度降本 + 上游材料国产化），扣非有望在 2026-2027 年接近盈亏平衡。经营现金流持续为正且增长（69.88 亿元，+21.49%），说明现金层面已经实现良性循环，主业经营拐点已近。这将使公司市净率（当前约 0.7-0.8x）和估值逻辑发生重大变化。

### 利好 3：柔性 AMOLED 产能爬坡 + Micro-LED 商业化起步，打开第二成长曲线

**依据**

(1) 柔性 AMOLED 手机显示保持国内第二、全球第三，中尺寸（平板、穿戴、电竞显示器）和车载 OLED 应用持续拓展；(2) Micro-LED 产线 2024 年点亮，2025 年实现标准化显示单元模块小批量出货 + 点亮首款车载标准产品；(3) 已发布 108 寸 / 135 寸 4K+ 无缝拼接、透明可调、超低反透明、超高 PPI 等行业领先 Micro-LED 显示产品，全激光巨量转移技术实现突破；(4) Micro-LED 重点布局车载显示和拼接显示应用，同步探索大尺寸屏幕、穿戴、专业显示。

**解读**

AMOLED 是未来中高端显示的主流技术路线，若 2026 年手机面板价格企稳 + 公司产线良率提升 + 产能利用率提高，柔性 AMOLED 有望从当前亏损转向接近盈亏平衡。Micro-LED 被视为下一代显示技术（亮度 + 对比度 + 寿命全面超越 OLED），公司作为国内较早布局的面板厂，有望在 2027-2030 年 Micro-LED 大规模商业化阶段占据先发优势，特别是在车载和大尺寸拼接显示领域。

### 利好 4：航天科工背景 + 市值管理措施 + 估值修复空间大

**依据**

(1) 公司为中国航天科工集团旗下主要中小尺寸显示面板龙头企业，具备国资央企背景和战略协同价值；(2) 2025 年公司将市值管理指标纳入考核体系，先后发布《估值提升计划》、《关于控股股东自愿承诺不减持公司股份的公告》、《关于回购公司股份方案的公告》；(3) 当前 PB 约 0.7-0.8x，历史低位。

**解读**

国资央企背景为公司在银行授信、政府补贴（2025 年非经常性损益 ~7.41 亿元）、研发项目支持、客户拓展等方面提供综合支持。管理层市值管理承诺（控股股东不减持 + 回购）传递对公司基本面信心。若 2026 年扣非接近盈亏平衡叠加 Micro-LED 商业化进展，当前 0.7-0.8x PB 估值有显著修复空间。

### 利好 5：MPG 非显示应用平台孵化新赛道

**依据**

公司建立 MPG（多项目玻璃基板技术平台），稳步推进面板制造技术在超材料智能天线、微流控、智能调光、大面积指纹识别、柔性传感器等领域的孵化，部分技术已小批量出货。

**解读**

面板级超材料智能天线可应用于 5G/6G 通信、卫星通信、低空经济（无人机）等领域；微流控芯片在生物医疗检测（IVD、POCT）具有巨大应用前景；智能调光可应用于建筑玻璃、汽车车窗、航空航天等领域。MPG 平台将公司面板制造经验延伸至高附加值非显示应用，打开长期增量空间，有望成为公司"第三增长曲线"。

## 7. 未来可能的利空

### 利空 1：扣非仍然亏损 + 归母转正依赖非经常性收益 + 不分红

**依据**

(1) 2025 年扣非净利润 -5.74 亿元（2024 年 -22.20 亿元），连续第 3 年扣非亏损；(2) 归母转正依赖约 7.41 亿元非经常性收益（含政府补贴、资产处置等）；(3) 2025 年不派发现金红利、不送红股、不以公积金转增股本；(4) 10 年 ROIC 中位数 2.22%，ROIC 在国内面板龙头中偏低；(5) 上市 30 年来亏损 5 次。

**解读**

核心主业尚未实现盈利意味着公司的收入规模和毛利率水平仍不足以完全覆盖折旧、研发和运营成本。如果 2026 年政府补贴、税收优惠、资产处置收益减少或消失，且主业扣非修复不及预期，归母利润可能再次转亏。当前非经常性收益占归母净利润比例超过 400%，盈利质量偏弱。不分红政策反映经营现金流需优先支撑 AMOLED / Micro-LED 产线持续投入。价值投资者需谨慎：历史财报非常一般的特征决定了估值天花板。

### 利空 2：面板行业产能过剩 + 价格竞争 + 柔性 OLED 价格承压

**依据**

(1) 面板行业处于产能扩张周期：国内京东方、TCL 科技（华星光电）、维信诺、和辉等多条 OLED 产线投产；(2) 2025 年柔性 OLED 手机显示产品市场价格阶段性承压；(3) 消费类 LCD 面板价格波动受整机（手机、笔记本）需求影响较大。

**解读**

行业供给过剩会持续压制消费类面板价格和毛利率。虽然天马在车载领域有差异化优势（定制化 + 车规认证 + 长周期供应），但消费类手机面板仍占约 40%，价格战风险无法完全回避。若 2026 年京东方柔性 OLED 产能继续扩张（A4/B7/B15 产线）+ 维信诺 V3 产线提升稼动率，公司可能面临手机 OLED 价格进一步下探的压力。

### 利空 3：重资产投入与折旧压力 + 流动性指标偏紧

**依据**

(1) 总资产 767.93 亿元（同比 -5.5%），资产负债率约 62%，有息资产负债率 38.08%；(2) 面板生产线需持续高额资本开支（Micro-LED 产线、AMOLED 扩产、LTPS 升级）；(3) **货币资金 / 流动负债仅 25.67%** 显示短期流动性偏紧；(4) **应收账款 / 利润高达 6,459%** 显示客户账期风险集中。

**解读**

面板制造是典型的重资产行业，产线折旧是利润的重大压力（估算每年 40-50 亿元级别），需要持续盈利改善才能消化。若技术路线切换（如从 LCD 到 OLED、从 OLED 到 Micro-LED）不及预期，前期投入可能面临减值风险。货币资金 / 流动负债 25.67% 的流动性指标需重点关注，若经营现金流出现短期波动或银行授信收紧，可能引发短期偿债压力。

### 利空 4：车载市场竞争加剧 + 毛利率阶段性承压

**依据**

(1) 车市需求增长和规格提升趋势不变，但"整车市场竞争激烈、价格挑战"压力会向上游供应链传导；(2) 公司车载显示业务以模组出货为主，需要定制化 + 技术方案升级维持毛利率；(3) 京东方、华星光电、TCL 等国内同行加速切入车载显示赛道；(4) 韩国 LG Display、三星 Display 在高端车载 OLED 领域仍有技术优势。

**解读**

公司在业绩说明会中承认车载业务毛利率受整车价格战传导，"会阶段性影响毛利率"。若 2026 年国内整车价格战延续（比亚迪、特斯拉降价引发连锁反应），车载供应链的降价压力将加大；同时，竞争对手加速扩张可能削弱公司全球第一的市占率优势。公司需要通过 LTPS 渗透 + 尺寸升级 + 模组复杂度提升 + 高附加值产品占比提升来对冲，但节奏存在不确定性。

### 利空 5：海外关税 + 地缘政治扰动 + 客户集中风险

**依据**

(1) 车载业务海外占比较高（覆盖中 / 欧 / 美 / 日韩系主流品牌），受关税政策、地缘政治影响较大；(2) 手机显示业务主要客户包括华为、小米、OPPO、vivo、荣耀等头部品牌，客户集中度较高；(3) 美国对华半导体、显示面板相关制裁可能影响高端 AMOLED / Micro-LED 产品的出口。

**解读**

若美国进一步收紧对华显示面板出口管制，或欧盟 / 日本跟进关税壁垒，可能影响公司海外车载客户的订单稳定性。地缘政治带来的供应链分裂可能迫使海外车厂分散采购，降低对单一供应商的依赖。同时，手机显示客户一旦出货放缓（如华为、小米 2026 年销量不及预期），将直接影响公司手机面板订单。

## 8. 总结判断

**一句话结论**

2025 年深天马 A 成功扭亏为盈（归母 +1.67 亿元，同比 +125.03%），但扣非仍亏 -5.74 亿元——真正的盈利拐点尚未完全到来；车载显示全球第一的龙头地位（+18%）、毛利率持续改善（+2.96pp）、扣非亏损大幅收窄（-74%）三大积极信号汇聚，叠加 Micro-LED 小批量出货、MPG 非显示应用孵化，盈利修复路径清晰，2026-2027 年有望实现扣非盈亏平衡。但不分红 + 对政府补贴的依赖 + 消费类面板价格压力仍构成阶段性制约。

**核心看点**

1. **车载显示全球第一龙头地位稳固**：连续 6 年全球出货量第一，HUD 2024 年登顶全球第一，2025 年车载业务 +18%，汽车电子 +30%，受益全球汽车智能化 / 电动化 / 多屏化三重驱动
2. **毛利率改善 + 扣非亏损收窄 74%**：毛利率 16.24%（+2.96pp），扣非亏损 -22.2 亿 → -5.74 亿，净经营利润连续三年改善（-21.1 → -6.91 → +1.11 亿），盈利拐点接近
3. **经营现金流优异**：69.88 亿元（+21.49%），远超 1.67 亿归母净利润，每股经营现金流 2.84 元，现金层面已实现良性循环
4. **Micro-LED 商业化起步 + MPG 新赛道孵化**：Micro-LED 标准化模块小批量出货 + 首款车载标准产品点亮；MPG 平台孵化智能天线、微流控、智能调光等非显示应用
5. **航天科工央企背景 + 市值管理**：估值提升计划 + 控股股东不减持承诺 + 回购股份方案传递管理层信心，当前 PB ~0.7-0.8x 存在修复空间

**核心风险**

1. **扣非仍亏损 + 归母转正依赖非经常性收益**：扣非 -5.74 亿元（连续 3 年扣非亏损），约 7.41 亿元非经常性收益（占归母 >400%）若减少将导致归母再次转亏；2025 年不分红
2. **面板行业产能过剩 + 柔性 OLED 价格承压**：国内多条 OLED 产线投产导致消费类面板价格持续下行，公司手机面板业务利润空间有限
3. **重资产折旧 + 流动性偏紧**：货币资金 / 流动负债仅 25.67%，应收账款 / 利润 6,459%，有息资产负债率 38.08%，流动性指标偏紧，需持续融资支撑产线投入
4. **车载竞争加剧 + 整车价格战传导**：公司承认车载毛利率受整车价格战阶段性影响，京东方、华星光电加速切入车载赛道削弱市占率优势
5. **海外关税 + 地缘政治 + 客户集中**：车载海外占比较高 + 手机客户集中于头部国产品牌，若整机厂出货放缓或关税政策收紧将直接冲击订单稳定性
