# 协创数据2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

协创数据2025年实现营业收入122.36亿元，同比 +65.13%；归母净利润11.64亿元，同比 +68.32%；扣非归母净利润11.35亿元，同比 +69.07%。经营活动现金流11.00亿元，同比 +1011.23%（增长超10倍）。ROE（加权）30.03%。基本每股收益3.38元。总资产237.98亿元（+225.80%）。分红每10股3.40元转增4股。

收入+65%和利润+68%双高增长，经营现金流暴增10倍至11亿元（与净利润基本匹配），ROE 30%极为优秀。算力集群项目交付验收+存储量价齐升是核心驱动。

### 1.2 增长与拖累因素

**主要增长驱动**：①多个算力集群项目完成交付验收，算力相关收入高速增长；②企业级存储产品"量价齐升"——出货规模增长+存储产品价格上行；③服务器再制造业务规模扩大+运营效率提升；④经营现金流+1011%至11亿元，回款大幅改善；⑤ROE 30.03%，资本回报极为优秀。

**主要拖累因素**：①总资产+226%至238亿元，资产负债率可能大幅上升（累计投入212亿元采购服务器用于算力租赁）；②应收账款上升（年报简析提到）。

### 1.3 趋势判断

协创数据2025年是算力+存储双轮驱动的爆发之年。公司正从"IoT智能终端"向"AI算力基础设施+数据存储"快速转型。累计212亿元算力投入规模巨大，算力租赁商业模式能否持续创造高回报是核心看点。

## 2. 主营业务

### 2.1 公司简介

协创数据（300857）是AI算力基础设施和数据存储解决方案提供商，主营业务包括智能算力（算力集群建设和租赁）、数据存储（企业级存储产品）、服务器再制造、AIoT智能终端等。公司在深圳、缅甸、泰国、新加坡、美国和菲律宾等地设有子公司。

### 2.2 主营业务构成

1. **智能算力**：算力集群建设、运维和租赁
2. **数据存储**：企业级存储产品
3. **服务器再制造**：二手服务器翻新
4. **AIoT智能终端**：智能物联网终端设备

### 2.3 核心收入来源

算力业务（集群项目交付+租赁）和数据存储是核心增长引擎。

## 6. 未来可能的利好

### 利好 1：AI算力需求持续增长

**依据**

公司累计投入约212亿元建设算力集群，多个项目完成交付验收。

**解读**

全球AI大模型训练和推理需求持续增长，算力基础设施投资回报期长且确定性高。

### 利好 2：存储业务量价齐升

**依据**

企业级存储产品出货规模增长+价格上行。

**解读**

AI训练数据爆发式增长驱动存储需求。NAND/DRAM价格周期上行利好存储产品。

### 利好 3：ROE 30%+经营现金流与净利润匹配

**依据**

ROE 30.03%，经营现金流11亿≈净利11.64亿元。

**解读**

极高的资本回报和优质的现金流质量，说明商业模式运转良好。

## 7. 未来可能的利空

### 利空 1：总资产暴增226%的风险

**依据**

总资产从~73亿增至238亿元（+226%），累计投入212亿元采购服务器。

**解读**

大规模投资算力集群意味着高杠杆，若算力租赁需求不及预期或价格下降，可能面临资产减值和债务压力。

### 利空 2：应收账款上升

**依据**

年报简析提到应收账款上升。

**解读**

大客户集中度和回款周期需关注。

### 利空 3：算力租赁商业模式的可持续性

**依据**

大量CAPEX投入算力基础设施。

**解读**

算力租赁市场竞争加剧、GPU技术迭代（新一代算力效率提升可能导致旧设备贬值）等因素可能影响投资回报。

## 8. 总结判断

**一句话结论**

协创数据2025年AI算力+存储双轮驱动爆发——营收122.36亿元（+65%）/净利11.64亿元（+68%）/扣非+69%，ROE 30%+经营现金流+1011%至11亿元（与净利匹配），多个算力集群项目交付+存储量价齐升。总资产+226%至238亿元（累计算力投入212亿）需关注杠杆风险。

**核心看点**

1. 算力集群项目持续交付和租赁收入增长
2. 存储产品量价趋势的持续性
3. 212亿元算力投入的长期回报率
4. ROE 30%能否维持

**核心风险**

1. 总资产+226%带来的杠杆风险
2. 算力设备技术迭代导致贬值
3. 应收账款增长和大客户集中度
4. 算力租赁市场竞争加剧
