# 申能股份（600642.SH）2025年财报整理

> 说明：截至本文件撰写时，申能股份2025年年度报告尚未正式披露，业绩预告/快报也暂未披露。公司已披露**2025年三季报**及**2025年度控股发电量完成情况公告**。本文件基于三季报、发电量公告、2024年年报进行整理。所有数据以原始披露为准，不作主观换算。

## 1. 业绩总结

### 1.1 收入与利润表现

- **全年营业收入**：年报未披露。2025年前三季度营业收入**209.32亿元，同比 -6.44%**。
- **全年营业利润**：年报未披露。前三季度净利率**19.81%（+2.22pct）**，毛利率**22.84%（+3.48pct）**，均明显提升。
- **全年归母净利润**：年报未披露。前三季度归母净利润**33.15亿元，同比 +1.04%**；扣非归母净利润**31.69亿元，同比 +7.85%**（扣非增速远高于归母）。
- **2025全年发电量576.54亿千瓦时（-1.7%）**，Q4新增装机206.72万千瓦显著扩张。
- 呈现"**营收下滑但利润稳健/核心经营改善**"的格局，公司通过燃料成本优化+新能源扩张实现增长。

### 1.2 增长与拖累因素

- **主要增长驱动**：
  1. **燃料成本下降**：煤价持续下行带动毛利率 +3.48pct；
  2. **风电+光伏爆发式增长**：前三季度风电发电量 +21.2%，光伏 +8.4%；全年风电 +24.3%；
  3. **经营现金流 +19.39%至73.19亿元**：现金流大幅改善；
  4. **财务费用 -17.72%至7.18亿元**：有息负债优化或利率下行；
  5. **Q4新增装机206.72万千瓦**：装机扩张加速；
  6. **油气管输业务稳定贡献**：占收入13.34%。
- **主要拖累因素**：
  1. **煤电发电量 -6.1%**（前三季度 -5.0%）：电量替代压力；
  2. **气电发电量 -10.0%**（前三季度-10.0%，全年 -0.1%）：气价压力或调度减少；
  3. **总发电量 -1.7%**：结构性下行；
  4. **营收 -6.44%**：电量与价格双重影响。

### 1.3 趋势判断

- **未来主要利好方向**：①煤价中枢仍处下行通道；②新能源装机持续扩张（风电/光伏+21%/+8%）；③Q4新增206.72万千瓦装机将在2026年贡献电量；④上海国资+申能集团背景+长江电力战略股东；⑤持续高分红传统（2024年每10股4.50元）。
- **未来主要压力方向**：①煤电电量持续替代；②气电需求波动；③电力市场化改革带来电价不确定性。
- **整体趋势判断**：申能股份是"**煤电+气电+新能源+油气管输"四位一体的综合能源平台**，2025年延续"量减利增"态势，新能源装机快速扩张，煤价下行带来毛利率显著改善。上海本地电力保供龙头+多元能源+高分红的独特定位，属于**稳健型电力蓝筹**。

## 2. 主营业务

### 2.1 公司简介

申能股份（600642.SH）是**上海申能（集团）有限公司**旗下综合能源上市平台，主营业务覆盖**电力（煤电/气电/风电/光伏）+油气管输+煤炭销售**四大板块。公司是上海本地电力保供核心企业，同时是A股少数的"火电+气电+风光+油气"四位一体综合能源标的。

### 2.2 主营业务构成

根据2025年三季报的分业务收入占比：

1. **煤电业务**（43.09%）：核心基本盘，装机840万千瓦（占45.18%）。受煤价下行+电量减少影响，但毛利率改善。
2. **气电业务**（18.92%）：装机342.56万千瓦（占18.42%）。2025年前三季度电量 -10.0%，承压明显。
3. **油气管输业务**（13.34%）：西气东输项目股权等，毛利率稳定，现金流良好。
4. **风电业务**（11.88%）：装机343.02万千瓦（占18.45%），发电量 +21.2%（Q1-Q3），+24.3%（全年）。
5. **煤炭销售**（8.06%）：煤炭贸易业务，受煤价影响。
6. **光伏发电业务**（4.54%）：装机244.06万千瓦（占13.13%），发电量 +8.4%。

### 2.3 核心收入来源

**煤电（43.09%）+气电（18.92%）**是两大主力电源，合计占62%；**油气管输（13.34%）+风电（11.88%）**合计占25%是重要补充；**煤炭销售（8.06%）+光伏（4.54%）**是辅助。公司装机结构已逐步向"**火电+气电+风光**"多元化演进，新能源占比（风电+光伏）超31%。

## 6. 未来可能的利好

### 6.1 利好 1：**多元能源结构优势，全国罕见的"四合一"电力标的**

**依据**

装机结构：煤电45.18% + 气电18.42% + 风电18.45% + 光伏13.13%，新能源占比31.58%，是A股少数实现"**煤电+气电+风光+油气"全覆盖**的电力标的。

**解读**

多元化的能源结构使公司具有最强的抗周期能力——煤价下行时煤电毛利扩张；煤价上行时可借气电和新能源对冲；新能源装机持续扩张提供长期增长曲线；油气管输提供稳定现金流。

### 6.2 利好 2：**新能源装机爆发式扩张，Q4新增206.72万千瓦**

**依据**

2025年Q4新增控股装机206.72万千瓦，占截至Q3装机1859万千瓦的**11.1%**。前三季度风电发电量+21.2%、光伏+8.4%，全年风电+24.3%、光伏+7.9%。

**解读**

Q4新增装机将在2026年全年贡献电量，叠加新能源装机持续扩张，未来2-3年公司有望看到更快的业绩增长。

### 6.3 利好 3：**煤价持续下行+燃料成本优化**

**依据**

2025年前三季度毛利率+3.48pct至22.84%。财务费用-17.72%。净利率+2.22pct。

**解读**

煤价中枢仍处下行通道，煤电板块盈利弹性释放。若2026年煤价维持当前水平或继续下行，煤电板块利润有望继续改善。

### 6.4 利好 4：**申能集团+长江电力战略股东+上海国资**

**依据**

申能集团持股53.67%为实控人，**长江电力持股9.38%为第二大股东**，实际控制人为上海市国资委。

**解读**

长江电力作为A股电力龙头入股，形成强大的战略协同+资源对接。上海国资改革+国企市值管理考核+SMG体系下资产整合预期多重利好。

### 6.5 利好 5：**高分红传统+类公用事业**

**依据**

2024年每10股派4.50元（22.02亿元），分红率约55.8%。2025年利润稳健增长有望延续或提升分红。

**解读**

公用事业属性+稳定分红+现金流73亿对股息型投资者具有吸引力。

## 7. 未来可能的利空

### 7.1 利空 1：**煤电发电量持续下滑**

**依据**

2025年前三季度煤电发电量-5.0%，全年-6.1%。

**解读**

煤电电量下滑是长期结构性趋势（新能源替代），未来或继续承压。虽然毛利率改善部分对冲，但营收和利润基数受限。

### 7.2 利空 2：**气电电量下滑10%**

**依据**

2025年前三季度气电发电量-10.0%。

**解读**

气电电量下滑可能反映气价压力+调度减少。若气价持续高企，气电盈利能力可能受损。

### 7.3 利空 3：**电力市场化改革不确定性**

**依据**

电力市场化改革持续推进，电价波动性可能加大。

**解读**

上海本地电价受市场化改革影响，若电价下行，将直接影响公司营收。容量电价机制虽有保障，但不能完全对冲。

### 7.4 利空 4：**营收-6.44%反映总量下行**

**依据**

前三季度营收209.32亿（-6.44%）。

**解读**

总量下行反映电量和价格的双重压力。若长期延续，会制约公司增长空间，虽然利润通过成本优化实现增长，但可持续性值得关注。

### 7.5 利空 5：**煤价反弹风险**

**依据**

当前煤价处于相对低位，若2026年煤价反弹，将直接侵蚀煤电毛利率。

**解读**

煤电利润对煤价敏感度极高（煤炭成本占60%+）。煤价任何上行都会快速反映在业绩中。

## 8. 总结判断

**一句话结论**

2025年申能股份"**量减利增、结构优化**"：前三季度营收209.32亿（-6.44%）、归母33.15亿（+1.04%）、扣非31.69亿（+7.85%）。毛利率+3.48pct（煤价下行红利）+新能源爆发（风电+21.2%/光伏+8.4%）+Q4新增装机206.72万千瓦，公司"煤电+气电+风光+油气"四位一体综合能源平台优势显著，高分红+低波动，是A股稳健型电力蓝筹代表。

**核心看点**

1. **装机结构最均衡**：煤电45%+气电18%+风电18%+光伏13%的"四合一"结构。
2. **新能源爆发**：风电+21.2%、光伏+8.4%、Q4新增206.72万千瓦。
3. **煤价下行**：毛利率+3.48pct。
4. **申能集团+长江电力战略股东+上海国资**：顶级股东背景。
5. **高分红传统**：2024年每10股4.50元，分红率55.8%。
6. **经营现金流+19.39%至73.19亿元**：现金流健康。
7. **油气管输稳定贡献**：公用事业属性。

**核心风险**

1. 煤电发电量持续下滑（-6.1%）。
2. 气电电量下滑-10%。
3. 电力市场化改革不确定性。
4. 营收-6.44%反映总量压力。
5. 煤价反弹风险。
