# 中国国贸2025年财报整理

> 2025年年度报告已于2026年4月1日正式披露。本文基于**2025年年报+Q3财报+2025业绩快报+2026年经营预测**整理。

## 1. 业绩总结

### 1.1 收入与利润表现

- **2025年营业收入 37.70亿元，同比 -3.63%**⚠️ 连续第二年下滑；
- **2025年归母净利润 12.02亿元，同比 -4.76%**⚠️；
- **2025年利润总额 16.05亿元，同比 -4.74%**⚠️；
- **2025年基本每股收益 1.19元/股，同比 -4.76%**⚠️；
- **加权平均净资产收益率 12.64%，同比 -0.71pct**⚠️；
- **经营活动现金流净额 15.75亿元，同比 -12.32%**⚠️；
- **财务费用 -838.47万元（净收益）+174.83%**⚡（偿还债券+银行借款）；
- **总资产 114.54亿元，同比 -7.07%**；
- **Q3单季营收 9.31亿(-5.47%)**⚠️；
- **Q3单季归母 3.08亿(-6.98%)**⚠️；
- **2025年分红 每10股10.7元现金红利+合计10.78亿**⚡⚡⚡（分红率**89.64%**）。

### 1.2 增长与拖累因素

- **主要增长亮点**：①**财务费用转负**（偿还债券+银行借款+财务成本优化）⚡；②**公寓+1.02%**+**其他+0.20%**细分正增长；③**每股分红10.7元+分红率89.64%**豪气分红；④**现金流净额15.75亿**虽下降但基本盘稳定；⑤商城稳定（-1.38%降幅最小）+**商城租金1308元/㎡/月出租率95.5%**。
- **主要拖累因素**：①**写字楼-6.32%**收入9554万元减少**（主力业务下滑）**；②**酒店-5.98%**+**酒店毛利率从7.27%→5.17%大幅下滑**；③**北京写字楼市场新增需求疲弱+租金持续承压**；④**高端酒店业竞争加剧+CBD去化压力**；⑤**写字楼平均租金639→609元/㎡/月下降30元**；⑥**写字楼平均出租率93.1%→91.8%下降1.3pct**；⑦**经营现金流-12.32%**；⑧**总资产-7.07%**。
- 公司整体呈现"**写字楼+酒店双下滑+市场需求疲弱+CBD去化压力+商城韧性+分红率高+持续回馈股东**"的优质物业龙头+行业下行期+高分红防御属性格局。

### 1.3 趋势判断

- **未来主要利好方向**：①**每10股派10.7元+分红率89.64%高股息**；②财务费用转负+**净现金+无息负债率下降**；③**商城租金韧性**（1308元/㎡/月+95.5%出租率）；④**公寓/其他细分正增长**；⑤**长期核心资产**（北京CBD国贸中心不可复制性）；⑥**长期稳定现金流**+**典型REITs标的**。
- **未来主要压力方向**：①**公司自身预计2026年营收36.1亿+利润总额14.5亿**均低于2025**——明确下滑指引**⚠️⚠️⚠️；②北京写字楼供过于求+新增需求疲弱；③高端酒店竞争加剧；④CBD区域去化压力；⑤美联储降息节奏+中国经济周期；⑥租金持续承压。
- **整体趋势判断**：**2026年公司自身指引营收-4.2%+利润总额-9.7%**+**短期无反转催化**+**但89.64%分红率+高股息+低估值+稳定分红+核心资产+长期稀缺**+**适合防御性+高股息策略投资**+**2027年后有望随经济回暖反转**。

## 2. 主营业务

### 2.1 公司简介

中国国贸(600007.SH)是**北京CBD核心地带中国国际贸易中心经营管理公司+中国高端物业龙头+国贸中心一二三期+国贸商城/国贸大酒店/中国大饭店/国贸公寓+商业物业租赁+酒店经营一体化**。**核心资产位于北京CBD核心地带**+**不可复制的地段稀缺性**+**国家/央企/跨国企业总部聚集地**。

### 2.2 主营业务构成

1. **写字楼（核心37.55%）**：
   - 国贸中心一期/二期/三期A座/三期B座
   - 35.62万㎡出租面积（Q3）
   - 平均租金609元/㎡/月（2025年）
   - 平均出租率91.8%（2025年）
2. **商城（33.65%）**：
   - 国贸商城18.05万㎡
   - 平均租金1308元/㎡/月
   - 出租率95.5%
3. **酒店（13.11%）**：
   - 中国大饭店+国贸大酒店+公寓酒店
   - 毛利率5.17%（同比下滑）
4. **公寓（5.01%）**：
   - 国贸公寓8.01万㎡
   - 平均租金363元/㎡/月+出租率90.4%
5. **其他（10.68%）**：
   - 物业管理+停车场+会议展览+其他租赁

### 2.3 核心收入来源

- **写字楼+商城为两大支柱**（合计70%+）；
- **国贸中心一二三期**+**核心CBD地段稀缺性**；
- **租约长期化+外资企业+央企总部**客户；
- **国贸商城**高端定位+**奢侈品+餐饮+娱乐**全业态。

## 6. 未来可能的利好

### 利好 1：每股10.7元分红+分红率89.64%⚡⚡⚡

**依据**

- **2025年每10股派现金红利10.7元**
- **合计分红10.78亿**
- **分红率89.64%**
- 连续多年高分红政策

**解读**

中国国贸**分红率极高+股息率4-5%+**+**高股息防御属性极强**+**价值投资首选**+**如果2026年维持85%+分红率+每股分红仍可达9-10元**+**长期现金奶牛**。

### 利好 2：财务费用转负+偿债优化⚡⚡⚡

**依据**

- **财务费用-838.47万元（净收益）**
- **同比+174.83%改善**
- **偿还债券+银行借款**

**解读**

**财务费用转净收益**+**无息负债率大幅下降**+**现金净头寸+利息净收入**+**长期财务健康度领先**+**抗周期性强**。

### 利好 3：核心资产地段稀缺性⚡⚡⚡

**依据**

- **北京CBD核心地带国贸中心**
- **一二三期A/B座合计35.62万㎡写字楼+18.05万㎡商城+8.01万㎡公寓**
- **国家级金融街+CBD+国贸商圈**
- **外资企业+央企总部聚集**

**解读**

中国国贸**核心资产不可复制**+**北京CBD最核心地段**+**物业+地段双重稀缺**+**长期价值稳固**+**即使短期经营压力，中长期价值基石稳固**+**经济复苏后反弹弹性大**。

### 利好 4：商城租金韧性⚡⚡

**依据**

- **商城租金1308元/㎡/月**（远高于写字楼613元）
- **商城出租率95.5%**（高于写字楼92.3%+公寓90.4%）
- **商城收入-1.38%**仅小幅下滑
- **奢侈品+餐饮+娱乐**全业态

**解读**

**商城业务相对写字楼+酒店表现更为韧性**+**租金高+出租率高+毛利高**+**CBD消费人群聚集+商务/高端消费**+**2026-2027消费回暖后可能率先反弹**。

### 利好 5：北京CBD长期价值+限供⚡⚡

**依据**

- 北京CBD写字楼新增供应有限
- 城市更新+存量改造为主
- 长期需求稳定（跨国+央企+金融）

**解读**

**北京CBD长期限供**+**高端物业稀缺性**+**即使短期供过于求**+**长期看供需逐步平衡**+**2027-2028年有望回暖**。

### 利好 6：低估值+典型REITs标的⚡⚡

**依据**

- 高分红率+低估值
- PB 1倍++PE 10倍+
- 类REITs投资标的

**解读**

**股价长期低估**+**对标海外物业REITs估值偏低**+**高股息+稳定分红**+**适合保险资金/年金/价值投资者配置**+**长期持有收益稳定**。

### 利好 7：酒店行业有望回暖⚡

**依据**

- 2026-2027高端酒店需求有望回暖
- 国际会议+商务出行恢复

## 7. 未来可能的利空

### 利空 1：公司2026年自身指引继续下滑⚠️⚠️⚠️

**依据**

- **公司预计2026年营收36.1亿**（同比-4.2%）
- **利润总额14.5亿**（同比-9.7%）
- **连续第三年下滑指引**

**解读**

**公司自身明确给出下滑指引**+**短期无反转催化**+**2026年业绩确定性下行**+**股价可能继续承压**+**投资者需耐心等待**。

### 利空 2：北京写字楼市场持续低迷⚠️⚠️⚠️

**依据**

- **北京写字楼新增供应持续**
- **新增需求疲弱**
- **CBD去化压力大**
- 租金持续承压

**解读**

**北京CBD写字楼市场**在**新增供应持续释放+新增需求疲弱**双重压力下+**租金持续下降+出租率下降趋势短期难以扭转**+**国贸写字楼作为行业标杆同受影响**。

### 利空 3：酒店毛利率大幅下滑⚠️⚠️⚠️

**依据**

- **酒店毛利率从7.27%→5.17%下滑2.10pct**
- **高端酒店竞争加剧**
- **北京新开高端酒店持续增加**

**解读**

**酒店业务盈利能力显著下滑**+**新开豪华酒店+竞争加剧**+**商务+旅游需求恢复缓慢**+**酒店业务短期无法成为增长引擎**。

### 利空 4：主营三大业务全线下滑⚠️⚠️

**依据**

- **写字楼-6.32%**
- **商城-1.38%**
- **酒店-5.98%**
- 三大支柱全部下滑

**解读**

**三大主营业务全部下滑**+**没有一个细分领域能够形成增长引擎**+**仅公寓/其他小幅正增长无法补位**+**业绩修复需要宏观环境支持**。

### 利空 5：经营现金流+总资产双下降⚠️⚠️

**依据**

- **经营现金流-12.32%**
- **总资产-7.07%**
- 可能反映资产处置+折旧+运营放缓

**解读**

**经营现金流下降**+**总资产下降**+**显示业务规模有收缩迹象**+**不利于长期增长**。
