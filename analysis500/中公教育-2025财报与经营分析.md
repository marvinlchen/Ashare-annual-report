> **数据来源说明**：
> 本报告基于**2025年三季报**（2025-10-31披露）、**2025年半年报**（2025-08-29披露）、**2025年一季报**、**2024年年报**、**2025年度业绩预告**（2026-01-31披露）及券商研究报告综合整理。
> 由于2025年年度报告尚未正式披露（预计2026-04下旬），部分全年财务数据暂以**业绩预告区间**及**前三季度累计数据**代替。

## 1. 业绩总结

### 1.1 收入与利润表现

- **全年营业收入表现**：根据业绩预告披露信息，2025年营收预期持续承压。**前三季度营收16.57亿元（同比 -21.09%）**，其中Q3单季营收5.02亿元（同比 -23.05%）。参照前三季度趋势，**预计全年营收约22-24亿元**（同比 -8%至-16%）。
- **全年营业利润表现**：**经营性利润承压但扣非后净利润降幅明显小于账面归母净利**，反映主业盈利能力基本稳定。
- **全年归母净利润表现**：根据**2025年度业绩预告**，**预计归母净利润4,250万元-5,500万元**（同比 **-70.06%至-76.86%**），上年同期为18,367.89万元；**扣非后净利润5,450万元-6,700万元**（同比 -15.18%至-31.00%）。**"归母降幅远大于扣非降幅"**反映非经常性损益的波动是核心原因。

### 1.2 增长与拖累因素

- **主要增长驱动**：
  1. **AI就业学习机**成为新增长点——自2025年7月发布以来，**截至三季度末累计销量突破万台**，硬件销售贡献增量。
  2. **H1"硬件+服务"双轨模式销售规模突破5,000万元**，成为新业务亮点。
  3. **毛利率显著提升**：H1毛利率59.61%（同比 +2.89pct），**成本持续优化**（H1营业总成本同比 -18.51%）。
  4. **连续7个季度实现盈利**，**经营性现金流已连续3个季度为正**，主业"造血"能力逐步恢复。
  5. **Q3净利润环比Q2增长39.34%**，**Q2环比Q1营收+0.90%**，环比连续改善。

- **主要拖累因素**：
  1. **非经常性损益大幅波动**：2025年预计非经常性损失-1,200万元，**而2024年同期为+10,468.95万元**，两者合计影响净利润下降约1.17亿元，占归母净利润下降幅度的**63.53%**。
  2. **全年新建135个教学服务中心**（H1新建45个），**战略从"阶段性整合"转向"有序扩张"**，业务拓展开支大幅增加，**营业总成本下降幅度收窄**。
  3. **传统业务持续萎缩**：公务员招录培训H1收入5.87亿元（同比 -19.41%）、事业单位招录培训H1收入1.54亿元（同比 -19.25%）、教师招录及教师资格培训H1收入8,736万元（同比 -37%）。
  4. **前三季度经营现金流净额3.34亿元**（同比 -41.40%）。

- **承压领域**：**公务员招录培训、事业单位招录培训、教师招录培训、综合培训**四大传统业务板块H1全面下滑，降幅均在19%-37%之间。

### 1.3 趋势判断

- **未来主要利好方向**：**AI就业学习机快速放量+教学服务中心有序扩张**有望成为2026年增长新引擎；**公考赛道长青**（2025年国考报名341.6万人，竞争比86:1）+"粉笔/华图/中公"三巨头格局稳定；**DeepSeek大模型私有化部署+云信自研模型**构筑AI先发优势。
- **未来主要压力方向**：**控股股东股权100%被冻结+股权拍卖**导致**控制权稳定性存在不确定性**；**协议班退费历史遗留问题**仍未完全出清（流动比率仅0.09）；公考市场份额被**粉笔/华图**分食。
- **整体趋势判断**：公司2025年呈现**"扣非净利基本企稳+业务结构转型+AI新业务放量"**的复杂格局。**"归母大幅下滑"主要由非经常性损益波动造成**，**主业已连续7个季度盈利**。**2026年看AI就业学习机能否形成规模、135家新增教学服务中心能否贡献业绩、公考市场能否企稳**三重关键。

## 2. 主营业务

### 2.1 公司简介

中公教育是**国内招录考试培训领域龙头企业**，公考"三巨头"之一，聚焦**公务员招录、事业单位招录、教师招录、综合培训**四大核心赛道，2025年起全面布局**AI就业服务**新业务，拥有"云信"自研大模型+DeepSeek私有化部署。

### 2.2 主营业务构成

1. **公务员招录培训**：H1收入5.87亿元，占比**50.85%**（同比 -19.41%）——**核心支柱**，面向国考、省考、选调生、军转干等全口径考试。
2. **综合培训**：H1收入2.98亿元，占比**25.75%**（同比 -18.58%）——包括研究生考试、医疗卫生、金融银行、IT互联网、学历提升等多元化培训业务。
3. **事业单位招录培训**：H1收入1.54亿元，占比**13.32%**（同比 -19.25%）——面向全国各地事业编制招录考试。
4. **教师招录及教师资格培训**：H1收入8,736万元，占比约**7.56%**（同比 -37%）——覆盖教资证考试+教师编制招录，2024年市场占率提升至**29%**。
5. **AI就业服务（新业务）**：H1销售规模突破**5,000万元**，三季度末**AI就业学习机累计销量破万台**——**定价4,999元基础版/发布会首发价3,999元**，提供基础/专业/超强三档服务。

### 2.3 核心收入来源

公司收入**高度依赖公务员招录培训**（占比过半）。2025年上半年**正价课培训人次达45万人次**（同比 +18.42%），**培训规模显著扩大**但**客单价下降**导致收入同比下滑。**线下教学服务中心+线上AI+硬件设备**三位一体的交付模式正在形成。

**商誉风险判断**

**商誉规模较小**（9,987.67万元，不足1亿元），占总资产约1.67%，**减值风险较低**，对未来利润影响有限。

1. **AI就业服务新业务**：从零起步，H1销售规模突破5,000万元，**成为新增长点**。

**主要拖累因素**

1. **公务员招录培训收入下滑**：同比 -19.41%，**客单价下降**是主因（培训人次增长但收入下降）。

4. **协议班占比降低+非协议班转化不足**：行业"冷静期"特征。

**结构变化**

- **主业结构稳定**：公考+综合+事业单位+教师=传统四大板块占比超95%。
- **AI就业服务业务从无到有**：已占约4.5%，**2026年有望突破10%**。
- **收入结构更加多元**：硬件销售（AI就业学习机）+服务收入+培训收入三元结构开始形成。

4. **研发投入加码**：Q2研发投入环比 +8.36%，AI技术研发与应用的持续投入。

## 6. 未来可能的利好

### 利好 1：AI就业学习机快速放量+AI就业服务新业务放量

**依据**

1. **行业首款"AI就业学习机"**：2025年6月发布，**截至三季度末累计销量破万台**，H1销售规模突破5,000万元。
2. **双核驱动架构**：整合**"云信"垂直大模型+DeepSeek大模型**（2025年2月9日完成DeepSeek私有化部署，**就业服务领域首家**）。
3. **一站式就业护航模式**：内置AI智能助理"小鹿老师"+7大AI工具组合（AI申论批改/AI人岗精准匹配等），覆盖职业测评、选岗规划、备考学习、考试计划全程。
4. **内容丰富**：**20,000+学时精讲内容+30,000+核心知识点**，涵盖国考、省考、事业单位、教师招聘、研究生、英语四六级等8大类考试。
5. **三档产品定位**：基础版4,999元（专注AI智能规划）/专业版（人工在线服务）/超强版（全面人工专业支持）。
6. **销售渠道覆盖**：天猫、京东、抖音等主流电商平台上线。

**解读**

AI就业学习机**填补了公考培训硬件产品市场空白**，是中公教育**从"卖课时"到"卖硬件+服务"模式转型的战略级创新**。**首发价3,999元+发布首季度累计破万台**验证市场接受度。对标教育硬件赛道（科大讯飞AI学习机2024年销售规模20亿+），若**2026年销量达到5-10万台**，将带来**3-5亿元增量收入+10%以上净利率**，**有望成为公司第二增长曲线**。**DeepSeek先发优势+云信自研大模型**构建**技术护城河**。**"AI+职业教育"融合**符合国家政策导向，长期成长空间广阔。

### 利好 2：公务员考试长青赛道+三巨头格局稳定

**依据**

1. **2025年国考共招录3.97万人，再创新高**。
2. **报名阶段341.6万人通过资格审查，竞争比约86:1**，考公热度持续。
3. **公考培训市场规模**：2025年有望**突破280亿元**，2024年招录考培市场规模约400亿元（公务员培训270亿元）。
4. **三巨头格局**：中公教育、粉笔、华图山鼎（华图教育）主导，**CR3超70%**。
5. **2024年各公司H1数据**：粉笔营收16.31亿元/净利润2.8亿元，中公教育营收14.47亿元/净利润1.16亿元，华图山鼎营收15.06亿元/净利润1.22亿元——**三家规模接近**。
6. **教师招录培训市场占率**：中公教育2024年提升至**29%**。

**解读**

公考培训是**典型的长青赛道**——就业压力越大，考公热度越高。**三巨头格局稳定**意味着公司无需担心被颠覆性新进入者取代。**280亿市场+稳定增长**为公司提供**长期可持续现金流**。**中公教育作为老牌龙头**，虽然市场份额有所下降，但**品牌积累+渠道覆盖+教研体系**仍具优势。**135个新教学服务中心+45万正价课人次+18.42%**表明**公司已重新回到正常增长轨道**。

### 利好 3：业务已企稳+连续7个季度盈利+扣非净利基本稳定

**依据**

1. **连续7个季度实现盈利**（从2023Q4至2025Q3）。
2. **经营性现金流已连续3个季度为正**（主业"造血"能力逐步恢复）。
3. **2025Q3净利润环比Q2增长39.34%**，**Q2环比Q1营收+0.90%**。
4. **毛利率持续提升**：H1毛利率59.61%（同比 +2.89pct）。
5. **管理费用Q1-Q3同比 -29.37%**，组织架构精简显效。
6. **2025年扣非净利润预计5,450-6,700万元**，同比 -15.18%至-31.00%，**降幅远小于归母净利的-70.06%至-76.86%**——主业盈利能力基本稳定。
7. **员工数从2021年4.5万人降至2024年9,515人（-82%）**，**线下分支从1,669家降至720家（-57%）**，**降本增效已基本完成**。
8. **所有者权益Q3末8.16亿元**（同比 +12.70%）——财务结构改善。

**解读**

中公教育**已完成"刮骨疗毒"式的痛苦调整**——**"大规模退费+大规模裁员+大规模关店"**阶段已基本结束。**连续7个季度盈利+连续3个季度经营现金流为正+毛利率持续提升**表明**主业经营质量实质改善**。**归母净利润70%+的下滑主要来自非经常性损益的剧烈波动**（从+1.05亿转为-0.12亿，变动-1.17亿），**并非主业恶化**。**扣非净利润仅下降15-31%**更能反映真实经营状况。**"整合到扩张"战略转折**+**135个新中心**为2026年重回增长通道奠定基础。

## 7. 未来可能的利空

### 利空 1：控股股东股权100%被冻结+股权拍卖+控制权稳定性风险

**依据**

1. **李永新个人所持8.87亿股已100%处于被冻结或标记状态**。
2. **累计被质押股份7.07亿股**（与一致行动人合计），**占其所持股份的64.78%**。
3. **累计被轮候冻结股份3.66亿股**（占其所持股份的33.36%）。
4. **2025年12月，李永新持有的4,800万股被司法拍卖**，成交金额1.01亿元，**完成过户后持股比例从17.70%降至16.92%**。
5. **公司公告表示**：控股股东及其一致行动人**面临债务压力较大**，公司**控制权稳定性面临不确定性风险**。
6. **2025年7月再度司法冻结**：李永新4.5亿股被冻结。
7. **2025年11月**：李永新1.84亿股遭轮候冻结，涉案金额3亿元。

**解读**

控股股东股权100%冻结+频繁司法拍卖是**悬在公司头上的达摩克利斯之剑**。**债务压力巨大**可能导致**更大规模的股权拍卖+控制权转移**。一旦控制权发生变更，将对**公司战略连续性、管理层稳定性、员工信心**造成严重冲击。**协议班退费的历史包袱**未完全消化，**李永新个人债务问题**与**公司经营**深度绑定（此前曾通过个人向公司借款），**形成持续流动性压力**。**这是公司最大的不确定性风险**，短期难以根本解决，可能长期压制估值。

### 利空 2：短期偿债压力巨大+流动性风险

**依据**

1. **2025Q3末货币资金仅1.04亿元**，较年初 **-64.42%**，主因支付上年末工资和归还股东借款。
2. **流动比率仅0.09**（流动资产37.26亿 / 流动负债43.46亿），**远低于健康水平**（通常建议>1.5）。
3. **有息负债17.66亿元**（处于较高水平）。
4. **2025Q1-Q3经营现金流3.34亿元**（同比 -41.40%），**创造现金能力下降**。
5. **2025前三季度预收款（合同负债）规模下降**——培训班业务预收款减少反映新招生不足。
6. **总负债51.67亿元**，资产负债率86.35%（仍较高）。

**解读**

**"货币资金只有1亿，但有息负债17.66亿，流动负债43.46亿"**——**账面现金远不足以覆盖短期债务**。虽然**连续3个季度经营现金流为正**，但**Q1-Q3现金流净额3.34亿元同比下降41.40%**，**偿债能力恢复缓慢**。**一旦发生**：①退费需求集中爆发；②控股股东个人债务引发股权变动影响融资；③主业客单价进一步下降——公司可能面临**严重的流动性危机**。**高财务杠杆+低流动性+历史退费问题**三重压力叠加，**短期偿债风险是核心脆弱点**。

### 利空 3：传统业务持续萎缩+行业竞争加剧+公考培训"冷静期"

**依据**

1. **2025H1四大传统业务全面下滑**：
   - 公务员招录培训：-19.41%
   - 综合培训：-18.58%
   - 事业单位招录培训：-19.25%
   - 教师招录培训：**-37.00%**
2. **行业进入"冷静期"**：消费热情下降、获客难度增加。
3. **2024年市场份额**：中公教育曾占比33%，因**协议班退费等问题市场份额下降**。
4. **粉笔2024H1营收16.31亿元/净利润2.8亿元**，**华图山鼎H1营收15.06亿元/净利润1.22亿元**——**均超越中公教育**。
5. **三巨头2024下半年业绩表现**：华图山鼎非学历培训业务规模已**超过中公教育**。
6. **粉笔追赶势头猛烈**+**华图转型基建+教培双主业**+**小机构降本增效加速**。
7. **2025年非经常性损益-1,200万元**（vs 2024年+1.05亿元），**历史拨备/资产处置收益不再贡献利润**。

**解读**

虽然公考赛道本身具有长期需求支撑，但**短期竞争格局对中公教育不利**：①**粉笔以"低价+AI"占领下沉市场**，**中公教育"高协议班+线下重资产"模式压力巨大**；②**华图山鼎借基建上市企业重组登陆A股**，**资源整合能力更强**；③**公考培训"冷静期"**意味着**整体市场增速放缓**（2025年280亿仅比2024年的270亿增长3.7%）。**公司H1四大传统业务全面下滑**+**教师招录培训-37%**+**非经常性损益转负**共同构成**戴维斯双杀**。**若2026年传统业务仍不能企稳，AI就业新业务的增长可能不足以对冲**。

## 8. 总结判断

**一句话结论**

2025年中公教育呈现**"归母利润大幅下滑（非经常性损益拖累）+扣非净利企稳+AI就业新业务放量+业务结构有序调整"**的复杂格局，**主业已完成最艰难的"刮骨疗毒"阶段**并**连续7个季度盈利**，**但控股股东股权100%冻结+流动性压力仍然巨大**，**是典型的"困境反转+AI新业务催化+历史包袱沉重"三重特征的转型期公司**。

**核心看点**

1. **AI就业学习机成为新增长引擎**——**DeepSeek+云信双大模型**+**硬件+服务双轨模式**+**三季度末累计销量破万台**，有望成为**2026年第二增长曲线**，对标科大讯飞AI学习机空间广阔。
2. **公考长青赛道+三巨头格局稳定+主业"造血"能力恢复**——**连续7个季度盈利+连续3个季度经营现金流为正+毛利率+2.89pct**，**扣非净利润降幅仅15-31%**反映主业实质企稳。
3. **战略转折从"整合"到"有序扩张"**——**135个新教学服务中心+45万正价课人次（+18.42%）+教师市场份额29%**表明公司**已回到增长通道**。

**核心风险**

1. **控股股东李永新股权100%被冻结+4,800万股司法拍卖+控制权稳定性风险**——**债务压力巨大**，**可能导致更大规模股权变动**，**是悬在公司头上的最大风险**。
2. **短期偿债压力巨大**——**货币资金仅1.04亿元（-64.42%）+流动比率0.09+有息负债17.66亿元**，**流动性风险突出**，历史退费问题未完全出清。
3. **传统业务持续萎缩+行业"冷静期"+竞争加剧**——**H1四大业务同比-18%至-37%**，**粉笔/华图山鼎规模已超中公**，**2025年非经常性损益从+1亿转为-0.12亿**共同构成戴维斯双杀压力。
