# 华能蒙电（600863.SH，内蒙华电）2025年财报整理

> 说明：截至当前，华能蒙电（内蒙华电）**2025年年度报告尚未发布**，本文件基于**2025年三季报+2025年半年报**整理经营现状。后续年报发布后将补充更新。所有数据以原始披露为准，不作主观换算。

## 1. 业绩总结

### 1.1 收入与利润表现

- **Q1-Q3营业收入 150.52亿元（-9.23%）** ⚠️ 下滑。
- **Q1-Q3归母净利润 22.08亿元（-10.60%）** ⚠️ 下滑。
- **Q1-Q3利润总额 32.81亿元（+2.34%）** ⚡ 逆势增长。
- EPS 0.33元（-10.81%）。
- **Q3单季营收52.25亿（-10.13%）**。
- **Q3单季归母6.51亿（-7.28%）**。
- Q1-Q3经营活动现金流44.77亿元（+0.19%）。
- 2025H1：营收98.27亿（-8.75%）、归母15.57亿（-11.91%）。
- 2025H1总资产405.93亿、资产负债率40.28%、加权ROE 8.91%。

### 1.2 业绩下滑原因

1. **发电量下降**：
   - Q1-Q3完成发电量**401.58亿千瓦时（-10.96%）**；
   - **火电发电量375.08亿千瓦时（-11.80%）**；
   - 主要因**新能源发电增加，导致火电机组调峰增加，利用小时下降**；
2. **煤炭销售价格下降**：
   - 煤炭销售均价**304.10元/吨（不含税），同比下降109.19元/吨（-26.42%）**；
3. **燃料采购成本下降**：
   - 入炉标煤单价**508.65元/吨（-12.15%）**；
   - 部分抵消售价下降影响。

### 1.3 积极因素

- **平均售电单价341.52元/千瓦时（+2.01%）** 电价上行；
- **经营现金流44.77亿（+0.19%）** 稳定；
- **利润总额+2.34%** 逆势增长（得益于入炉煤价降幅大于发电量降幅）；
- 资产负债率40.28%（稳健）。

### 1.4 增长与拖累因素

- **主要增长驱动**：
  1. **售电单价+2.01%**（电价上行红利）；
  2. **入炉标煤单价-12.15%**（燃料成本下降）；
  3. **利润总额+2.34%**（逆势增长）；
  4. **经营现金流+0.19%**（稳定）；
  5. **华能集团背景**（大型央企电力巨头）；
  6. **北方联合电力（华能系）持股51.24%**（控股稳定）；
  7. **煤电一体化**（同时有煤炭销售业务）；
  8. **内蒙古区位优势**（风光基地+煤炭资源富集）。
- **主要拖累因素**：
  1. **发电量-10.96%**（火电调峰增加）；
  2. **火电发电量-11.80%**（被新能源挤压）；
  3. **煤炭销售价格-26.42%**（煤价下跌）；
  4. **营收-9.23%+归母-10.60%**；
  5. **Q3单季归母-7.28%**（持续承压）；
  6. 电力市场化改革+新能源消纳竞争；
  7. 内蒙古区域电力供需宽松。
- **整体趋势判断**：华能蒙电（内蒙华电）作为"**华能集团内蒙古核心电力+煤电一体化**"标的，2025年呈现"**发电量下滑+煤价下跌+电价上行+成本改善**"的复杂局面：Q1-Q3营收-9.23%+归母-10.60%反映**火电被新能源挤压**（发电量-10.96%）+**煤炭销售收入下降**（煤价-26.42%），但**利润总额+2.34%**逆势增长显示**入炉煤价-12.15%的成本红利**部分抵消。核心矛盾是"**火电调峰量下降**"与"**新能源装机加速**"的结构性转换。未来看点在于**煤电一体化+新能源转型+电价市场化**的综合效益。属于"**煤电一体化+新能源转型+利润韧性**"组合标的。

## 2. 主营业务

### 2.1 公司简介

华能蒙电（600863.SH）全称**内蒙古蒙电华能热电股份有限公司**（简称"**内蒙华电**"），是**中国华能集团**旗下以内蒙古为主要运营区域的电力+煤炭一体化上市公司。**第一大股东为北方联合电力有限责任公司**（华能集团子公司，持股**51.24%**）。公司主营**火电发电+煤炭销售+新能源发电**，是"**西电东送**""**蒙电外送**"的核心参与者，**内蒙古电力外送华北电网**的重要力量。

### 2.2 主营业务构成（2025Q1-Q3）

1. **火电发电业务（核心主业）**：
   - Q1-Q3火电发电量375.08亿千瓦时（-11.80%）；
   - 包括坑口电厂+调峰电厂；
   - 主要向华北电网外送；
   - 受新能源挤压，利用小时下降。
2. **煤炭销售业务**：
   - Q1-Q3煤炭销售均价304.10元/吨（-26.42%）；
   - 入炉标煤单价508.65元/吨（-12.15%）；
   - 煤电联动的"**煤电一体化**"经营模式。
3. **新能源发电业务**：
   - 包括风电+光伏；
   - Q1-Q3总发电量401.58亿千瓦时中非火电占比约6.6%（约26.5亿千瓦时）；
   - 受益于内蒙古风光资源富集。
4. **供热业务**：
   - 热电联产；
   - 稳定业务收入。

### 2.3 核心收入来源

**火电发电**是绝对核心业务（占Q1-Q3总发电量的93.4%）。**煤炭销售**是煤电一体化的重要组成，煤价下跌拖累营收但降低燃料成本。**新能源**是未来增长方向。**"煤电+煤炭+新能源"三位一体**的经营结构使公司在电力市场化改革+双碳转型中具有独特优势。

## 4. 营收分布和增长

### 4.1 业务板块构成（2025Q1-Q3）

| 业务板块 | 占比/规模 | 同比 | 备注 |
|---|---:|---:|---|
| **火电发电** | 375.08亿千瓦时（占发电量93.4%） | **-11.80%** | 核心主业 |
| **煤炭销售** | 销售均价304.10元/吨 | **-26.42%** | 价格下跌 |
| **新能源发电** | ~26.5亿千瓦时（占发电量6.6%） | - | 增量方向 |
| 供热 | - | - | 稳定 |

### 4.2 营收分析

**营收-9.23%——发电量+煤炭双降拖累**

营收下滑主因：①**发电量-10.96%**（火电被新能源挤压）；②**煤炭销售价格-26.42%**（煤价下跌）。尽管**电价+2.01%**部分对冲，但总量和价格的双向下行难以避免营收下滑。

**发电量下降的结构性原因**

内蒙古风电+光伏装机加速，绿电消纳压力下**火电被迫调峰+让利**，利用小时数下降。这是双碳转型过程中传统火电的结构性压力。

**煤炭销售价格-26.42%——煤价周期下行**

煤价下跌拖累煤炭销售收入，但同时降低入炉标煤单价-12.15%，对火电业务是成本利好。**"煤电联动"机制下，煤价下行通常对煤电综合盈利有利**。

**入炉煤价-12.15%是核心改善因素**

燃料成本下降12.15%+电价上升2.01%=**煤电利润率显著改善**。这是利润总额+2.34%的核心驱动力。

**煤电业务盈利韧性凸显**

尽管发电量-11.80%，但通过"**量减价升+成本降**"的综合效应，煤电业务盈利韧性仍较强，显示公司定价+成本管理能力。

## 6. 未来可能的利好

### 6.1 利好 1：**华能集团背景——大型央企电力巨头**

**依据**

- 实际控制人**中国华能集团**（五大发电集团之一）；
- **北方联合电力**（华能系）持股51.24%；
- 是华能集团内蒙古核心电力平台。

**解读**

央企背景提供：①**信用评级高**（融资成本低）；②**监管支持**（电价政策倾斜）；③**资产注入预期**（华能集团内蒙古其他资产）；④**技术+人才+管理资源**。

### 6.2 利好 2：**煤电一体化——成本竞争优势**

**依据**

公司**同时经营火电发电+煤炭销售**，煤矿+电厂"坑口一体化"布局。

**解读**

坑口煤电一体化降低煤炭运输成本，提供**天然的成本优势**。在煤价波动时具有较强的利润平滑能力——煤价高时煤炭业务受益，煤价低时发电业务受益。

### 6.3 利好 3：**入炉煤价-12.15%+电价+2.01%——煤电价差扩大**

**依据**

Q1-Q3入炉标煤单价508.65元/吨（-12.15%）+平均售电单价341.52元/千瓦时（+2.01%）。

**解读**

**煤价降+电价升**的剪刀差是煤电企业盈利修复的核心逻辑。若2026年煤价继续下行+电价继续上行，**煤电业务盈利将显著改善**。

### 6.4 利好 4：**内蒙古区位优势——风光+煤炭资源富集**

**依据**

内蒙古是中国**风光资源最富集+煤炭储量最大**的省区，也是"**国家能源安全保障基地**"。

**解读**

公司坐拥：①**廉价煤炭**（蒙煤储量占全国25%+）；②**优质风光资源**（风电2,400小时+、光伏1,600小时+）；③**蒙电外送**通道（向华北电网外送）。

### 6.5 利好 5：**利润总额+2.34%——逆势增长韧性**

**依据**

Q1-Q3利润总额32.81亿元（+2.34%）逆势增长。

**解读**

在发电量-10.96%+煤炭-26.42%的不利局面下仍实现利润总额+2.34%，反映公司**成本管控+业务多元化**的抗周期能力。

### 6.6 利好 6：**经营现金流44.77亿+0.19%——现金流稳定**

**依据**

Q1-Q3经营现金流44.77亿元（+0.19%）。

**解读**

稳定的经营现金流为：①**分红**提供支撑；②**新能源转型投资**提供资金；③**降杠杆**提供能力。

### 6.7 利好 7：**新能源转型+装机加速**

**依据**

内蒙古作为新能源装机大省，公司受益于风光大基地建设。

**解读**

虽然短期火电被新能源挤压利用小时下降，但公司自身也在**加速发展风光新能源装机**，未来"**火电稳底+新能源增量**"将成为新的盈利模式。

### 6.8 利好 8：**资产负债率40.28%——财务稳健**

**依据**

2025H1资产负债率40.28%。

**解读**

在电力行业中40.28%属于较低水平，财务杠杆稳健，为后续新能源转型投资提供空间。

## 7. 未来可能的利空

### 7.1 利空 1：**发电量-10.96%——火电被新能源挤压**

**依据**

- Q1-Q3总发电量-10.96%；
- **火电-11.80%**。

**解读**

内蒙古新能源装机加速（2024年末风光装机已超火电），火电调峰压力大幅上升，**利用小时数下降是长期趋势**。若新能源继续大规模并网，火电发电量可能继续下降。

### 7.2 利空 2：**营收-9.23%+归母-10.60%——下行压力**

**依据**

Q1-Q3营收-9.23%、归母-10.60%，且Q3单季仍下滑。

**解读**

如果2026年煤价+电价+发电量无法企稳，公司业绩可能继续承压。下行趋势尚未扭转。

### 7.3 利空 3：**煤炭销售价格-26.42%——煤炭业务拖累**

**依据**

Q1-Q3煤炭销售均价304.10元/吨（-26.42%）。

**解读**

煤价下跌虽然降低入炉成本，但煤炭销售业务收入端损失明显。"煤电一体化"在煤价下跌时**煤炭业务拖累更明显**。

### 7.4 利空 4：**电力市场化改革+新能源消纳竞争**

**依据**

电力市场化改革深化+绿电交易+新能源挤压带来的电价波动风险。

**解读**

**煤电作为调峰电源**，定位从"主力电源"向"托底电源"转变，盈利模式可能发生结构性变化。短期内利用小时下降+辅助服务收入增加存在不确定性。

### 7.5 利空 5：**内蒙古区域电力供需宽松**

**依据**

内蒙古装机规模大+新能源+区域电力相对宽松。

**解读**

供需宽松导致电价市场化后有下行压力。即使Q1-Q3电价+2.01%，未来可能受区域竞争影响出现回落。

### 7.6 利空 6：**双碳政策下火电长期面临转型压力**

**依据**

"**3060双碳目标**"下火电长期面临转型压力，新装机受限+老旧机组退役加速。

**解读**

长期看火电资产将面临减值风险，公司需加速新能源转型。若转型节奏慢于行业，竞争力将下降。

### 7.7 利空 7：**归母/利润总额比下降——少数股东分享利润**

**依据**

Q1-Q3归母/利润总额比从77.0%降至67.3%。

**解读**

少数股东分享利润比例上升，反映新项目少数股权较多，**母公司股东实际分享的利润下降**。

## 8. 总结判断

**一句话结论**

华能蒙电（内蒙华电，600863.SH）作为"**华能集团内蒙古核心电力+煤电一体化**"标的，2025年呈现"**发电量下滑+煤价下跌+电价上行+成本改善**"的复杂局面：Q1-Q3营收-9.23%+归母-10.60%反映**火电被新能源挤压**（发电量-10.96%）+**煤炭销售收入下降**（煤价-26.42%），但**利润总额+2.34%**逆势增长显示**入炉煤价-12.15%+售电单价+2.01%的剪刀差成本改善**。核心矛盾是"**火电调峰量下降**"与"**新能源装机加速**"的结构性转换。未来看点在于**华能集团央企背景+煤电一体化成本优势+内蒙古风光资源富集+新能源转型**的综合效益。属于"**煤电一体化+新能源转型+利润韧性**"组合标的，适合关注**煤电价差修复+新能源转型节奏**的价值投资者。

**核心看点**

1. ⚡ **华能集团央企背景+北方联合电力控股51.24%**
