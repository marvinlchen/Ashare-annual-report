# 联创光电2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

联创光电（600363.SH）**2025年年度报告尚未披露**，但已于**2026年1月24日发布业绩预增公告**：预计2025年全年归母净利润**4.35-5.32亿元**（同比**+80.36%~+120.57%**），扣非净利润**3.83-4.92亿元**（同比**+100.70%~+157.82%**）。Q1-Q3实际已实现营业收入**25.03亿元（+2.85%）**、归母净利润**4.00亿元（+19.37%）**、扣非净利**3.64亿元（+16.79%）**，其中Q3单季归母**1.37亿元（+28.31%）**创单季新高。业绩预告对应Q4将大幅放量，呈现**"传统主业稳健+激光业务大幅放量+超导业务进入规模化订单期"**的典型硬科技转型格局。

### 1.2 增长与拖累因素

**主要增长驱动**：（1）**激光业务（子公司中久光电）爆发式增长**——H1激光业务营收**+176.87%**，中久光电H1收入**1.14亿（+265%）**，"光刃系列"激光反无人机装备**完成首单出口**+阿布扎比防务展获**70+家意向客户**；（2）**背光源业务大幅减亏**（2024年拖累的黑点转正）；（3）**超导业务（参股子公司联创超导）订单密集落地**——"星火一号"聚变-裂变混合堆磁体订单约**50亿元**+感应加热设备+硅单晶炉等；（4）**高温超导磁控单晶炉**：与盈谷股份签订**百余台设备批量化订单**，处于产业化批量交付阶段；（5）**传统电线电缆+智能控制主业稳健经营**。**主要拖累**：（1）2024年基数较低（归母仅2.41亿，-27.86%）；（2）Q3经营现金流**2661.69万元（-20.11%）**——回款节奏需关注；（3）毛利率19.73%（整体偏低，因传统业务+新业务规模化初期）。

### 1.3 趋势判断

联创光电是**"激光+高温超导"双核心驱动的硬科技转型龙头**，背靠江西电子集团（江西省国资委）。公司正处于**"传统主业利稳+激光业务爆发+超导商业化加速"**的交汇点。2025年核心看点：**中久光电激光业务+177%高增长+光刃出口突破+联创超导"星火一号"50亿磁体订单+商业航天切入+可控核聚变产业化前夜**。**2026年关键看激光业务持续放量+超导订单持续兑现+可控核聚变进入规模交付**。整体判断为**"业绩拐点确立+硬科技估值切换"**双重逻辑，2025年是业绩爆发元年。

## 2. 主营业务

### 2.1 公司简介

联创光电（600363.SH）是**江西电子集团（江西省国资委）旗下上市公司**，是**国内知名光电通信产品制造企业**，经过近年战略转型，已形成**"以智能控制产业为基础，重点突出激光和高温超导两大产业"**的业务格局，定位**"激光+超导"双核心硬科技转型**。

### 2.2 主营业务构成

1. **激光业务（子公司中久光电）** ⚡⚡⚡：核心增量业务，主要产品为激光反无人机装备"光刃系列"，覆盖高中低三层次产品体系（光刃-ⅠB/Ⅱ/ⅡA/ⅡC/Ⅲ）。H1营收+176.87%，首单出口完成。
2. **高温超导业务（参股子公司联创超导）** ⚡⚡⚡：核心突破业务，产品覆盖：
   - **可控核聚变磁体**（"星火一号"项目约50亿元订单）
   - **高温超导磁控单晶炉**（与盈谷股份签订百余台设备批量化订单）
   - **商业航天电磁弹射**（2025年7月中标资阳项目1960万元）
   - **感应加热设备**
   - **聚变-裂变混合堆磁体**
   - 截至2024年累计54项超导产业专利
3. **智能控制业务**：公司战略基础，传统主业
4. **电线电缆业务**：传统主业，提供现金流
5. **背光源业务**：2024年亏损拖累业务，2025年大幅减亏
6. **LED业务**：传统业务
7. **激光军民融合+核聚变+电磁弹射+单晶炉**："激光+超导"双核心业务板块

### 2.3 核心收入来源

公司正处于**业务结构剧烈切换期**：
- **传统主业（电线电缆+智能控制+LED）**：贡献稳定现金流和约70%+收入
- **激光业务（中久光电）**：H1+177%高增长，成为**业绩增长最大引擎**
- **高温超导业务（联创超导）**：参股子公司，通过投资收益+协同订单贡献利润
- **背光源**：减亏贡献增量利润

## 6. 未来可能的利好

### 利好 1：激光业务爆发式增长+中久光电持续放量 ⚡⚡⚡

**依据**

- H1激光业务营收**+176.87%**
- **中久光电H1收入1.14亿（+265%）**
- 光刃系列**完成首单出口**
- 2025年阿布扎比防务展**获70余家意向客户**关注
- 产品矩阵完整：光刃-ⅠB/Ⅱ/ⅡA/ⅡC/Ⅲ五大型号高中低三层次
- 完成**14件知识产权申报**

**解读**

激光反无人机装备是当前军贸+民用安防最火的赛道之一：
1. **军贸市场**：俄乌冲突+无人机威胁全球上升，激光反无人机需求爆发
2. **出口突破**：完成首单出口+70+家意向客户说明市场认可度高
3. **产品成熟**：高中低三层次产品完整覆盖
4. **技术自主**：图像识别+伺服系统闭环控制等核心技术自主化

### 利好 2：联创超导"星火一号"50亿磁体订单+可控核聚变产业化 ⚡⚡⚡

**依据**

- **"星火一号"聚变-裂变混合堆磁体总订单约50亿元**
- 完成**国内首个基于高温超导缆线的D型线圈20K温区低温实验**
- 液氮温区稳态运行**电流突破1.5kA**
- 中标**中核二三系统+中国科学院合肥物质科学研究院**超导磁体项目
- 累计**54项超导产业专利**

**解读**

可控核聚变是国家级战略级项目：
1. **50亿订单是重磅利好**：对应5-10年业绩释放
2. **技术领先**：国内首个高温超导D型线圈实验
3. **客户覆盖国家队**：中核二三+中科院合肥物质科学研究院
4. **专利壁垒**：54项专利构筑技术护城河

### 利好 3：高温超导磁控单晶炉百余台批量订单+商业化加速 ⚡⚡⚡

**依据**

- 与**盈谷股份签订百余台设备批量化订单**
- 处于**产业化批量交付阶段**
- 完成**光伏级和电子级磁控硅单晶生长装置**用超导磁体研发

**解读**

磁控单晶炉是高温超导在工业领域的**首个大规模商业化应用**：
1. **百余台订单**：对应数亿元收入
2. **产业化阶段**：已从研发进入批量交付
3. **光伏+电子**：应用场景广泛
4. **验证技术成熟**：为其他工业应用打下基础

### 利好 4：商业航天电磁弹射切入 ⚡⚡⚡

**依据**

- 2025年7月**中标资阳商业航天发射技术研究院项目**

商业航天是未来5-10年千亿级市场：
1. **电磁弹射是SpaceX等新一代火箭发射的核心技术**

3. **技术协同**：高温超导+电磁弹射高度协同
4. **国家级战略**：商业航天国家大力支持

### 利好 5：背光源业务大幅减亏 ⚡⚡

**依据**

业绩预告明确指出**背光源业务板块大幅减亏**是业绩预增重要原因之一。

背光源业务2024年亏损拖累公司整体业绩，2025年大幅减亏直接贡献利润增量。如果完全扭亏，将进一步改善利润表。

省国资委背景提供稳定融资渠道+政府支持+订单资源。硬科技战略清晰，是国有资本+高科技的典型代表。

### 利好 7：业绩拐点确立+估值切换 ⚡⚡

**依据**

- 2024年归母2.41亿（-27.86%） → 2025年预告4.35-5.32亿（**+80~+121%**）
- 扣非增速（+100.70%~+157.82%）更高

2025年是公司业绩拐点确立之年。市场预期将逐步从传统估值切换到**硬科技/激光/超导估值体系**，估值溢价空间大。

- 研发投入大

**解读**

硬科技业务前期毛利率受产能爬坡影响：
1. **激光业务毛利率需要规模效应**
2. **超导业务研发投入大**
3. **背光源/传统业务毛利率偏低**
4. **结构优化需要时间**

### 利空 3：50亿订单执行+产能交付不确定性 ⚠️⚠️

**依据**

- "星火一号"50亿订单需多年交付
- 产能扩张+供应链是否能跟上
- 可控核聚变技术仍在发展阶段

**解读**

50亿订单是好事但也是挑战：
1. **产能能否及时扩张**
2. **技术标准是否稳定**
3. **客户验收节奏**
4. **首次大规模交付的经验不足**

### 利空 4：军贸出口+地缘政治风险 ⚠️⚠️

**依据**

- 光刃激光反无人机**首单出口**
- 军贸受地缘政治影响大
- 海外客户国别风险

**解读**

军贸出口业务具有：
1. **地缘政治不确定性**
2. **客户集中度风险**
3. **出口管制风险**
4. **外汇风险**

### 利空 5：高估值风险+硬科技估值泡沫 ⚠️⚠️

**依据**

- 2024年业绩低迷，2025年业绩大幅反弹
- 市场对激光+超导+核聚变+商业航天概念追捧
- 估值可能已反映较多预期

**解读**

估值风险：
1. **概念驱动估值可能脱离基本面**
2. **如果新业务增长不及预期**估值将大幅回撤
3. **硬科技估值具有波动性**

### 利空 6：联创超导参股方式+利润贡献间接性 ⚠️

**依据**

- 联创超导为**参股子公司**
- 公司以**投资收益+协同订单**方式体现
- 非完全并表

**解读**

参股方式意味着：
1. **50亿订单利润并非全额归属公司**
2. **取决于公司持股比例**
3. **与直接控股相比利润确认复杂**

### 利空 7：2024年基数较低+高增速具有一次性 ⚠️

**依据**

- 2024年归母2.41亿（-27.86%）基数较低
- 2025年预告+80~+120%部分源于低基数

**解读**

高增速有低基数因素。2026年需要继续维持高增长，否则增速可能明显回落。

## 8. 总结判断

**一句话结论**

联创光电2025年**"硬科技转型业绩爆发元年+激光+超导双轮驱动"**——业绩预告归母**4.35-5.32亿（+80.36%~+120.57%）**+扣非**3.83-4.92亿（+100.70%~+157.82%）**+Q1-Q3归母**4.00亿（+19.37%）**+Q3单季**1.37亿（+28.31%）**创新高。**激光业务（中久光电）H1+176.87%+光刃首单出口+阿布扎比防务展70+意向客户**+**高温超导（联创超导）"星火一号"50亿磁体订单+磁控单晶炉百台订单+商业航天电磁弹射1960万+国内首个D型线圈20K温区低温实验**。背靠**江西电子集团省国资委**。**传统主业稳健+激光爆发+超导商业化+背光源减亏**四大动能叠加，2025年业绩拐点确立。

**核心看点**

1. **⚡⚡⚡ 激光业务H1 +176.87%+中久光电H1 1.14亿（+265%）**
2. **⚡⚡⚡ "光刃系列"激光反无人机首单出口+70+家海外意向客户**
3. **⚡⚡⚡ "星火一号"聚变-裂变混合堆磁体50亿订单**
4. **⚡⚡⚡ 高温超导磁控单晶炉百余台批量订单（盈谷股份）**
5. **⚡⚡⚡ 商业航天电磁弹射切入（资阳项目1960万）**
6. **⚡⚡⚡ 国内首个高温超导D型线圈20K温区低温实验（电流1.5kA）**
7. **⚡⚡⚡ 业绩预告归母+80~+121%+扣非+101~+158%**
8. **⚡⚡ 背光源业务大幅减亏**
9. **⚡⚡ 江西电子集团央企背景+"激光+超导"双核心战略**
10. **⚡⚡ Q3单季归母+28.31%创新高**
11. **⚡⚡ 54项超导产业专利构筑技术壁垒**

**核心风险**

1. **⚠️⚠️⚠️ Q1-Q3经营现金流-20.11%+回款节奏**
2. **⚠️⚠️⚠️ 毛利率19.73%偏低+规模化初期**
3. **⚠️⚠️ 50亿订单执行+产能交付不确定性**
4. **⚠️⚠️ 军贸出口+地缘政治风险**
5. **⚠️⚠️ 高估值风险+硬科技概念泡沫**
6. **⚠️ 联创超导参股方式+利润贡献间接性**
7. **⚠️ 2024年基数较低+高增速一次性**

**投资逻辑**

**激光业务+177%爆发+光刃出口+70+意向客户+星火一号50亿订单+磁控单晶炉百台+商业航天切入+背光源减亏+业绩预告+80~+121%+江西省国资委背景+"激光+超导"双核心** vs **现金流-20%+毛利率偏低+订单执行不确定+军贸地缘政治+硬科技估值泡沫+参股间接性+低基数效应**。2026关键看**激光业务持续放量+超导订单持续落地+可控核聚变规模交付+海外出口突破+估值切换完成**。**硬科技转型+业绩爆发+估值切换**三重逻辑，2025年是业绩拐点元年。
