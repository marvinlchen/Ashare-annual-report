# 中直股份2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

中直股份2025年实现营业收入**290.86亿元**（同比 -2.28%），归母净利润**6.52亿元**（同比 +17.27%），扣非净利润**5.61亿元**（同比 +16.34%），基本每股收益**0.7951元**（同比 +9.91%）。营业收入全年微降2.28%，但归母净利润实现**双位数增长17.27%**，净利率由1.87%提升至2.25%（+0.43pct）。**Q4单季强势回暖**：Q4营收140.39亿元（同比 +12.62%）、归母净利润约2.79亿元，呈现"Q1-Q3承压、Q4集中确认"的军工典型节奏。

### 1.2 增长与拖累因素

**主要驱动**：①**期间费用 -15.88%**（财务费用 -81.85%、研发费用 -30.18%、管理费用压降），为核心利润弹性来源；②**其他收益 +88.66%**（政府补助增加）；③**产品结构优化**，AR-E800/AR-E3000成功首飞、eVTOL产品研制进展；④**合同负债59.03亿元（+32.77%）**，订单储备充足反映需求旺盛；⑤**Q4集中交付**带动全年收入向上修正。

**主要拖累**：①全年收入 -2.28%，主因最终客户交付节奏放缓（Q3单季营收仅48.1亿，同比 -47.3%）；②**经营现金流 -107.76亿元**，严重为负，军工客户账期拉长、资金占用巨大；③航空产品毛利率约6-10%、净利率仅2.25%，盈利空间极薄；④研发费用大幅削减（-30.18%），短期改善利润但可能影响中长期技术竞争力。

### 1.3 趋势判断

中直股份作为**中国直升机产业唯一上市平台**，具有极强的战略稀缺性。2025年受交付节奏影响收入微降，但**费用管控 + 政府补助 + 产品结构优化**推动利润增长17%。核心矛盾是经营现金流巨额为负（-107.76亿），反映军工行业"先生产后回款"的资金占用特征。**未来核心利好**：①十五五（2026-2030）军机订单释放；②哈飞集团吸并哈飞航空（合并后总资产超420亿元），组织结构优化；③AC332等民机交付加快（订单24架），低空经济+应急救援打开民品增量。**未来主要压力**：经营现金流持续为负、军品定价机制下的低利润率、交付节奏不确定性。**整体判断**：稀缺直升机平台 + 十五五订单预期 + 低空经济概念，但短期业绩受交付节奏和现金流拖累。

## 2. 主营业务

### 2.1 公司简介

中直股份（600038.SH）是中国航空工业集团旗下的直升机产业上市平台，**中国唯一的直升机整机及零部件上市公司**，同时是中国直升机产业化、市场化平台。产品涵盖**军用直升机、民用直升机、通用飞机（运12系列）、航空零部件、无人机、eVTOL**等。

### 2.2 主营业务构成

1. **航空产品（99.41%）**：以直升机整机及零部件为主，包括军用直升机（主体）、民用AC系列直升机（AC311/AC311A/AC312E/AC332/AC352/AC313A）、通用飞机（运12/运12F）、航空转包生产、客户化服务。
2. **先进制造（0.38%）**：精密制造、特种加工等。
3. **其他（0.21%）**：其他业务。

### 2.3 核心收入来源

航空产品贡献近100%收入，其中**军用直升机是绝对主体**（占营收主要部分），民用直升机和通用飞机正在积极拓展。产品覆盖**1吨至13吨全系列直升机**，满足军用、应急救援、通用航空、警用航空、海洋/海上救援等多场景需求。核心子公司为**哈飞集团**（哈尔滨飞机工业集团）和**昌飞集团**（景德镇昌河飞机工业集团）。

## 4. 营收分布和增长

### 4.1 按业务划分

| 业务板块 | 占比 | 备注 |
|---|---:|---|
| **航空产品** | **99.41%** | 直升机+通用飞机（军品为主） |
| 先进制造 | 0.38% | |
| 其他 | 0.21% | |

### 4.2 按客户/产品分析

**军用直升机**（主体）：产品覆盖军用运输直升机、武装直升机、侦察直升机、舰载直升机等，最终客户为军方。2025年交付节奏调整，Q1-Q3同比下降显著。

**民用直升机**：AC系列（AC311/AC311A/AC312E/AC313A/AC332/AC352）加速交付：
- **AC313A**（13吨级）：2025年6月完成局部改进及复飞工作
- **AC332**：新增**18架销售合同+6架意向**（合计24架），建三江开展自驾系统调参试飞
- **AC311A**：新增3架订单（2销+1租），完成11架采购选型论证
- **AC352**（7吨级）：持续交付

**通用飞机**：运12/运12F系列继续交付。

**航空转包**：为空客、波音等提供零部件转包生产。

### 4.3 营收分析

**主要增长来源**：Q4集中交付（+12.62%）、民用直升机AC系列订单加速、转包业务稳定。

**主要拖累**：军用直升机交付节奏调整（Q1-Q3 -13.02%、Q3单季 -47.3%），受十四五收官阶段订单结构调整影响。

**结构变化**：低空经济、应急救援、eVTOL等新兴领域占比逐步提升，民品结构改善。

## 5. 利润分布和增长

### 5.1 整体利润表现

| 指标 | 2025年 | 2024年 | 同比增速 | 备注 |
|---|---:|---:|---:|---|
| 归母净利润 | 6.52亿元 | ~5.56亿元 | +17.27% | 费用压缩驱动 |
| 扣非净利润 | 5.61亿元 | ~4.82亿元 | +16.34% | |
| 净利率 | ~2.25% | ~1.87% | +0.43pct | 薄利改善 |

### 5.2 利润驱动分析

**利润增长驱动**：
1. **期间费用 -15.88%**（财务费用 -81.85%、研发费用 -30.18%）——核心降本
2. **其他收益 +88.66%**（政府补助增加）——非经常性贡献
3. **产品结构优化**——民用直升机、高附加值产品占比提升
4. **Q4毛利率改善**——规模效应显现

**利润承压因素**：
1. 军品定价机制下毛利率天花板低（6-10%）
2. 净利率仅2.25%，远低于工业平均水平
3. 研发投入削减30%，可能影响中长期竞争力

### 5.3 分部利润

年报未单独披露分部利润，但**净利率提升0.43pct**、**Q3单季净利率大幅改善**反映产品结构优化成效。

## 6. 未来可能的利好

### 利好 1：十五五军工订单释放

**依据**

2025年是十四五收官之年，十五五（2026-2030）新一轮装备采购周期即将开启。Q3末合同负债59.03亿元，同比 +32.77%。

**解读**

作为中国**唯一直升机整机上市平台**，军用直升机需求确定性高。合同负债大幅增长是在手订单（预收款）增加的直接反映，十五五军机订单释放将带来收入中枢抬升。

### 利好 2：哈飞吸并哈飞航空+组织架构优化

**依据**

2025年11月公告：哈飞集团拟吸收合并哈飞航空，合并后总资产将超过**420亿元**。控股股东中航科工累计增持股份约**5002万元**。

**解读**

吸并将优化组织架构、减少内部交易、提升经营效率，并利于低空经济领域整合。控股股东增持体现对公司长期价值的信心。

### 利好 3：低空经济+eVTOL商业化

**依据**

AR-E800、AR-E3000成功首飞。eVTOL新兴低空航空器发展推动应用场景拓展。AC311直升机在应急救援中表现突出（27小时内完成7架次飞行救援任务）。

**解读**

低空经济是国家战略新兴产业，2024年首次写入政府工作报告。公司在eVTOL领域的布局（AR-E800/AR-E3000）可能带来民品增量，应急救援、通用航空等新兴市场打开第二增长曲线。

### 利好 4：AC系列民用直升机订单加速

**依据**

AC332新增18架销售合同+6架意向（合计24架），AC311A新增3架订单，AC313A完成改进并复飞。

**解读**

民用直升机应用于**应急救援、通用航空、低空物流、警航、海上救援**等场景，市场空间广阔，有望成为第二增长曲线。国产替代空间大。

### 利好 5：首次发布国际标准

**依据**

2025年首次发布2项国际标准。

**解读**

从"跟跑"到"并跑"甚至"领跑"，反映公司在直升机领域的技术话语权提升，有望带来技术溢价。

## 7. 未来可能的利空

### 利空 1：经营现金流巨额为负

**依据**

2025年经营现金流**-107.76亿元**，严重为负。

**解读**

最终客户（主要为军方）付款周期严重拉长，资金占用巨大。这是军工行业的通病，但-107亿的规模仍值得高度警惕，将持续对公司资金链和财务费用形成压力。

### 利空 2：极薄的利润率

**依据**

全年净利率仅~2.25%，航空产品毛利率6-10%。

**解读**

军品定价机制（成本加成5%）导致利润率天花板偏低，且短期难以改善。与国外直升机龙头（贝尔、西科斯基等15-20%利润率）差距大。

### 利空 3：交付节奏不确定性

**依据**

2025年Q1-Q3收入 -13.02%，Q3单季收入 -47.3%。

**解读**

军工产品交付受客户计划影响大，收入波动不可控。Q4集中交付模式导致全年业绩季节性明显，单季数据难以线性外推。

### 利空 4：研发费用大幅削减

**依据**

研发费用 -30.18%。

**解读**

短期利润改善靠削减研发，但可能影响中长期技术竞争力，特别是与国外直升机巨头的技术追赶进程。建议关注2026年研发投入能否恢复。

### 利空 5：估值偏高+短期缺乏催化

**依据**

军工板块整体估值处于历史中枢以上，稀缺性溢价已充分反映。

**解读**

十五五订单释放节奏仍需观察，若订单释放不及预期，估值可能面临调整压力。

## 8. 总结判断

**一句话结论**

中直股份2025年营收290.86亿元（-2.28%）、归母净利6.52亿元（+17.27%），**利润增长主要靠费用压缩（-15.88%）和政府补助（+88.66%）**，Q4集中确认贡献全年业绩。**经营现金流-107.76亿元**是核心财务风险，**合同负债+32.77%**则是积极订单储备信号。作为**中国唯一直升机整机上市平台**，稀缺性极强，十五五军机订单、低空经济、AC系列民机订单释放是中长期看点。

**核心看点**

1. **十五五军用直升机订单释放**：合同负债+32.77%已提前反映
2. **哈飞吸并+组织优化**：合并后总资产420亿元+效率提升
3. **低空经济/eVTOL商业化**：AR-E800/AR-E3000首飞+民品增量
4. **AC系列民用直升机**：AC332订单24架+AC313A复飞+AC311A加速
5. **经营现金流改善**：军方回款节奏（最关键观察指标）
6. **Q4集中确认效应**：2026年持续观察是否延续

**核心风险**

1. **经营现金流-107.76亿元**（军工行业通病但规模过大）
2. **净利率极薄（~2.25%）**，军品定价机制下难以改善
3. **交付节奏受客户计划影响**，Q1-Q3波动剧烈
4. **研发费用-30.18%**，短期利润改善但影响中长期竞争力
5. **十五五订单释放节奏不确定**，若不及预期可能影响估值
