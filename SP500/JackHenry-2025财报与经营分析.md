## 1. 业绩总结

### 1.1 收入与利润表现（FY2025）

- 全年 **GAAP 营收 $2,375.3M**（约 $23.8 亿美元），同比 **+7.2%**（FY24 $2,215.5M）；**Non-GAAP Adjusted Revenue $2,341.4M**（剔除 deconversion），同比 **+6.5%**
- 全年 **GAAP 营业利润 $568.7M**，同比 **+16.2%**（FY24 $489.4M），**营业利润率 23.9%**（FY24 22.1%，**+1.8pp**）；**Non-GAAP Adjusted Operating Income $541.1M**，+9.8%，利润率 23.1%（FY24 22.4%）
- 全年 **GAAP 净利润 $455.7M**，同比 **+19.4%**（FY24 $381.8M），**摊薄 EPS $6.24**（FY24 $5.23，**+19.3%**）；**Non-GAAP Adjusted Net Income $434.7M**，+13.1%
- **Non-GAAP EBITDA $745.8M**，+7.7%；**自由现金流（Free Cash Flow）$410.3M**，+21.9%（FY24 $336.5M）
- **Free Cash Flow Conversion 90.0%**（FY24 87.9%），**ROIC 22.1%**（FY24 19.7%），**ROE 22.9%**（FY24 22.1%）

### 1.2 增长与拖累因素

- **主要增长驱动**：① **Services & Support（服务与支持）+$85.8M / +6.7%**（占总营收 57.3%），其中 Cloud Hosting + Data Processing **+12.0%**（核心引擎）、Consulting + Release Revenue +9.6%、Deconversion Revenue +$17.4M / +104.9%；② **Processing（支付处理）+$74.0M / +7.9%**（占 42.7%），Transaction + Digital +13.0%、Payment Processing +9.4%、Card +6.6%；③ 运营杠杆明显：Cost of Revenue 占收入比 57.3%（FY24 58.7%，**-1.4pp**），SG&A 占比 11.9%（FY24 12.6%，**-0.7pp**）；④ 利息支出 -36.3%（$16.4M → $10.4M）+ 利息收入 +11.0% = 净利息收益翻倍（$8.6M → $17.3M）；⑤ 有效税率下降至 22.2%（FY24 23.3%）
- **主要拖累因素**：① **License + Hardware 收入 -25.2%**（客户转向云订阅，许可证 / 硬件销售长期下滑趋势）；② R&D 开支 +9.8%（+$14.5M），反映平台能力建设投入加大；③ **Corporate & Other 分部营收 -1.8%**（GAAP）或 -1.9%（Non-GAAP），第三方协议重构对 FY26 造成 $16M 反转利空（FY26 Q1 $12M 冲击）；④ FY24 基数含 $16.4M VEDIP 裁员费用，对 FY25 增速有 ~2pp 贡献基数效应（剔除后 FY25 Non-GAAP 营业利润增长约 6-7%）
- **结构视角**：四大业务分部 FY25 增速分化——**Complementary（补充解决方案）+9.2% 领跑**（$618M → $675M），Core +7.0%（核心系统），Payments +6.8%（支付），Corporate & Other -1.8%；**Non-GAAP 调整后** Complementary +8.5%、Payments +6.2%、Core +6.0%、Corp & Other -1.9%

### 1.3 趋势判断（含 Q2 FY2026 最新动态）

- **Q1-Q2 FY2026 延续高增长**：H1 FY26 GAAP 营收 **+7.6%**、营业利润 **+25.1%**、GAAP EPS **$3.70（+24.5%）**；Q2 单季度 GAAP 营收 **+7.9%**、营业利润 **+29.4%**、EPS $1.72（**+28.6%**）
- **未来主要利好**：① **现金流结构性增强**：美国新税法（R&D 全额费用化 + 100% bonus depreciation）预计 FY26 FCF conversion **回升至 85-100%**；② Q2 FY26 赢得 **22 个 core 新签**，产品组合 Jack Henry Platform / Tap2Local / Rapid Transfers 持续放量；③ 行业并购整合虽冲击 deconversion，但也使 Jack Henry 获客加速（"Jack Henry to Jack Henry" 并购锁定长期收入）；④ 债务归零 + $102M 现金 = 资本配置灵活性大
- **未来主要压力**：① **银行合并潮**加速 deconversion（客户流失）抬升短期收入流失（FY26 EPS 增速从 19% 降至 +1-3% 指引区间中值）；② FY25 上半年若干大客户续约出现 price compression（价格压缩），直接影响 FY26 前期增长；③ Payments 分部受第三方风控收入下降 + 第三方协议重组影响，$16M 年度收入逆风；④ 银行 / 信用合作社账户数增速放缓带来基本面拖累
- **整体判断**：Jack Henry 是美国社区银行 / 信用合作社 **IT 核心供应商中的三巨头之一（与 Fiserv、FIS 并列）**，**FY25 GAAP EPS +19.3% / Free Cash Flow +21.9% / 债务归零**代表其**现金牛 + 防御性 SaaS 转型**模式已经成熟；2026 年虽有行业并购整合和 deconversion 技术性扰动，但底层 Core + Processing 业务年化 7-8% 有机增速稳健；**适合作为 Fintech-Utility 型防御配置**

## 2. 主营业务

### 2.1 公司简介

Jack Henry & Associates（NASDAQ: JKHY）是美国领先的**金融科技公司**，为约 **7,400 家美国银行和信用合作社**提供核心银行系统、支付处理和补充服务解决方案，S&P 500 成分股，1976 年成立于密苏里州 Monett，拥有近 50 年服务社区金融机构经验。

### 2.2 主营业务构成

按**服务类别**分为两大板块：

1. **Services & Support（服务与支持，FY25 营收 $1,361.7M / 57.3%）**：核心系统许可、云托管（Private / Public Cloud）、数据处理、一般维护与实施、咨询、工单和版本升级、Deconversion（一次性客户流失费）、License 和 Hardware 销售
2. **Processing（处理业务，FY25 营收 $1,013.6M / 42.7%）**：信用卡 / 借记卡交易处理、数字银行、电子支付（ACH / Bill Pay / Wire / FedNow）、商户收单（Tap2Local）、风控服务

按**业务分部（Segment）**分为四个运营板块：

1. **Core（核心银行系统，FY25 $739.3M / 31.1%）**：SilverLake、CIF 20/20、Core Director 三大核心平台，服务全 US 银行；**非 GAAP 分部利润率 59.2%**
2. **Payments（支付解决方案，FY25 $873.5M / 36.8%）**：PayCenter、iPay、JHA Payment Solutions、EPS 等；**非 GAAP 分部利润率 46.7%**
3. **Complementary（补充解决方案，FY25 $675.2M / 28.4%）**：Banno Digital Platform（数字银行）、欺诈检测、文档管理、CRM 等；**非 GAAP 分部利润率 60.5%（最高）**
4. **Corporate & Other（企业与其他，FY25 $87.3M / 3.7%）**：不可分配收入与费用（主要为资金管理相关）

### 2.3 核心收入来源

- **Recurring Revenue（经常性收入）占比约 92-93%**：Cloud 托管、数据处理、许可证续约、支付处理、维护服务等订阅 / 交易类收入
- **按客户规模**：以**中小型银行 + 信用合作社**为主，同时近年来获得**更多大型金融机构**（大客户签约增加）
- **按交付模式**：Public Cloud 和 Private Cloud 合计占 Data Processing 收入 **+12.0% YoY 增长**，代表 Jack Henry 的 **Cloud-First** 战略已进入收入兑现阶段
- **新产品引擎**：FY25 发布 **Jack Henry Rapid Transfers™**（实时转账，与 Fed FedNow 对接）、**Tap2Local™**（移动支付 + 商户收单）、**Jack Henry Platform™**（下一代云原生核心系统）

### 4.3 营收分析

**主要增长来源**

- **Cloud Hosting & Data Processing +12.0%**（最大增长引擎）：反映 Jack Henry 客户从本地部署向 Private Cloud / Public Cloud 迁移的结构性趋势；
- **Transaction + Digital +13.0%**：数字银行（Banno Digital Platform）渗透率提升 + 交易量增长；
- **Payment Processing +9.4%**：ACH、Wire、Bill Pay 等电子支付量增长；
- **Consulting + Release Revenue +9.6%**：客户升级、迁移项目增多；
- **Deconversion Revenue +$17.4M / +104.9%**：虽为流失费（客户并购后转投其他系统），但为短期 GAAP 收入贡献。

**主要拖累因素**

- **License + Hardware Revenue -25.2%**：客户转向云订阅 → 本地许可证收入长期下滑（此为行业共性趋势，Fiserv / FIS 亦面临）；
- **Corporate & Other -1.8%**：Fin Products 和 Fin Services 部分第三方协议重构（FY26 将造成 $16M 收入逆风）；
- **Payments 第三方风控收入下降**：行业 KYC / 反欺诈市场竞争加剧。

**结构变化**

- **Processing 占比上升**（42.7% vs FY24 42.4%，+0.3pp）：Jack Henry 业务模式从"一次性许可证 + 维护"向"持续性交易处理 + 订阅"转型持续；
- **Complementary 成为增长最快分部**（+9.2%）：Banno Digital 数字银行平台推动，反映社区银行追赶大行数字化能力的需求；
- **Cloud 渗透率继续提升**：Private + Public Cloud 合计 +12% 远超整体增速 7.2%，**长期 ARR 结构改善明显**。

- ② **成本控制**：Cost of Revenue 占比下降 1.4pp，SG&A 占比下降 0.7pp（主要因 FY24 基数高 VEDIP 费用）；
- ③ **净利息收益翻倍**（$8.6M → $17.3M）：偿还 $150M 债务后利息支出减少 $6M + 现金余额上升带动利息收入 +$2.7M；
- ④ **有效税率下降**至 22.2%（-1.1pp），贡献约 $5-6M 净利润；
- ⑤ **三大业务分部利润均 +9-12%**：Core +9.5% / Payments +10.0% / Complementary +11.8%，反映结构性营运杠杆释放。

**利润承压因素**

- ① **R&D 投入加大**（+9.8%）：新产品 Jack Henry Platform、Tap2Local、Rapid Transfers 等的预投入；
- ② **License + Hardware 收入 -25.2%**：高毛利产品下降，对整体毛利率有结构性拖累（虽然 Cloud 订阅毛利率也高，但 License 一次性结算影响短期）；
- ③ **FY2026 潜在压力**：第三方协议重构 -$16M 收入（对营业利润影响 $10-15M），将部分抵消 FY26 运营杠杆。

## 6. 未来可能的利好

### 利好 1：Cloud 转型兑现 + 新产品管道（Rapid Transfers / Tap2Local / Jack Henry Platform）

**依据**

- FY25 **Private + Public Cloud 数据处理收入 +12.0%**（远超整体 +7.2%）；Transaction & Digital 收入 +13.0%；
- FY25 **Rapid Transfers™ 上线**（实时转账，与 Fed FedNow 对接）、**Tap2Local™ 商户收单解决方案上线**、**Jack Henry Platform™（云原生下一代核心）**投产；
- Q2 FY26 **22 个 core 新签**（FY25 全年 51 个 core win）持续强劲；
- 新客户管道（pipeline）保持"非常健康"状态（管理层表述）。

**解读**

- Jack Henry 的 Cloud-First 战略开始进入**收入规模化兑现期**，远高于行业平均 5-6% 的核心系统增速；
- Rapid Transfers（FedNow 对接）卡位美国**实时支付赛道（FedNow + RTP）**，未来 3-5 年**交易量和手续费收入** 有望高速增长；
- Tap2Local 提供小额商户收单（SoftPOS），直接扩大 Jack Henry 在**商户支付生态**的份额，突破传统"银行 BaaS"定位；
- Jack Henry Platform 代表**云原生核心系统**，可替换 SilverLake / CIF 20/20 等老平台，为未来 10 年的 Core 市场做长期铺垫；
- 22 个 Q2 新签 + 51 个 FY25 新签说明 **Jack Henry 在竞品并购浪潮中继续获客**（如 Fiserv 的 Finxact、FIS 的 Modern Banking Platform 等），市场份额稳中有升。

### 利好 2：美国新税法带来 FCF 结构性提升 + 债务归零 + $20B 资本回报能力

**依据**

- 美国 IRC 174 R&D 全额费用化 + 100% "bonus" 税务折旧，预计 FY26 **FCF Conversion 回升至 85-100%**（FY25 90% 已达上限，税务利好将带来结构性强化）；
- FY25 末**有息负债归零**（FY24 $150M），现金 $102M；
- FY25 经营现金流 $641.5M（+12.9%）、自由现金流 $410.3M（+21.9%）；
- 2026 FY 继续回购、分红扩大空间：已授权 $0.58/股 季度分红 + 按需回购。

**解读**

- 新税法意味着 Jack Henry 可在 FY26 一次性兑现过去几年累积的 R&D 税务抵免 + 软件资本化的税务折旧，**FCF 或一次性增加 $50-100M**；
- 债务归零 + 现金增加 = **$650M+ 即时可支配资本**（含 $600M Revolving 额度），可用于 M&A、加大回购或维持 **20+ 年连续加息的股息纪录**（Dividend Aristocrat 成员）；
- ROIC 从 19.7% 上升至 22.1%，反映**资本效率持续改善**；股东回报（Dividend + Buyback）$200M / 年稳健。

### 利好 3：银行业并购浪潮虽短期冲击 Deconversion，但长期 "Jack Henry to Jack Henry" 并购锁定核心客户

**依据**

- 管理层表述："Recent industry consolidation continues... Several convert/merges are Jack Henry to Jack Henry thus limiting uplift in short-term revenue but securing long-term revenue opportunities"；
- 并购潮下，**大客户向 Jack Henry 迁移**（FY25 获大客户新签增加）；
- FY25 A Bank 数字银行平台 Banno 用户数继续快速增长（具体数字未披露，但驱动 Complementary +9.2%）。

**解读**

- **美联储利率降低 + 监管宽松 = 美国社区银行并购提速**，短期内被收购方流失带来 deconversion 一次性费用；
- 但若收购方是 Jack Henry 客户（"Jack Henry to Jack Henry"），则核心系统合并后 Jack Henry **仍保留收入**，形成**并购消化后的长期订阅收入锁定**；
- 这是美国 FinTech 供应商独特的"**护城河效应**"——头部 Core 供应商（Jack Henry / Fiserv / FIS）在行业集中度提升中实际上**市场份额稳中有升**，小型供应商会被挤出；
- 当前 Jack Henry 有~$250M + 累计 deconversion 收入储备（合同义务），**未来 2-3 年可持续贡献一次性收入**。

## 7. 未来可能的利空

### 利空 1：FY26 GAAP EPS 增速大幅放缓至 +1-3%，市场期望下调

**依据**

- FY26 GAAP EPS 指引 **$6.32-$6.44**（中值 $6.38），**同比仅 +1.3% ~ +3.3%**（vs FY25 的 +19.3%）；
- 指引放缓原因包括：① **Deconversion revenue 基数高**（FY25 $33.9M 为历史高位，管理层谨慎指引 FY26 从低基数起步，但每 $1M 偏差会影响 $0.16 EPS）；② 第三方协议重构 -$16M 收入；③ FY25 有 $16.4M VEDIP 基数效应（FY26 比较基数正常化）；
- FY26 Q1 预计营收**高于指引中值 +100bp**，Q2 低于 -100bp（Connect 大型客户大会从 Q2 挪至 Q1），**Q3 略弱、Q4 略强**（季度 cadence 异常）。

**解读**

- **市场对 Jack Henry 的预期已经锚定 10-15% EPS 增长**，若 FY26 仅交付 +1-3%（即使是指引的"保守"性），**股价短期承压是大概率**；
- 当然，若 industry consolidation 在 FY26 下半年加速，Deconversion revenue 会超预期，则 EPS 可能上修至 +5-8%；但初始指引保守说明管理层对未来 12 个月环境不乐观；
- Q2 FY26 EPS **+28.6%** 增长看似亮眼，但主要来自 Connect 会议移动的**季度错配**（Q1 +20% vs Q2 +28.6%）；H1 合计 +24.5%；管理层仍维持保守指引暗示 H2 增速可能放缓。

### 利空 2：Core / Processing 市场竞争加剧，大客户续约出现 Price Compression

**依据**

- 管理层公开表述："in the first half of fiscal year 2025 we had several key large customer contracts that renewed with price compression that is contributing to the revenue headwinds in the short term"；
- 竞争对手 Fiserv（FISV）推出 Finxact 云原生核心、FIS 推出 Modern Banking Platform、TCS BaNCS、Thought Machine Vault 等**新一代云原生核心系统**涌入美国市场；
- 行业整体账户增长率放缓（银行与信用合作社整体账户数增速降低）直接影响按账户定价的合约规模。

**解读**

- 美国 Core Banking 市场处于**云原生迁移窗口期**，每次 5-10 年大客户续约都是一次竞争性激烈的价格谈判；
- **Price compression** 意味着 Jack Henry 即使保留客户也无法完全通过涨价传导通胀 + 成本上涨；
- 长期看，若 Jack Henry Platform™ 不能快速追平 Finxact / Thought Machine 的云原生技术水位，可能在**未来 5-10 年的新建核心替换周期**中处于相对守势；
- 银行账户总数增速放缓（~1-2% YoY，远低于往年 3-5%）对支付和补充服务的基础量也形成压力。

### 利空 3：Payments 分部第三方风控 / 协议变更 + 账户账户合计增速放缓

**依据**

- 管理层表述："The Payments segment is under pressure from lower growth in risk management and third-party revenue"；
- "The restructure of a third-party agreement has resulted in a $16 million fiscal year-over-year revenue headwind, with $12 million of that coming in the first quarter [of FY26]"；
- 行业内风控 / 反欺诈赛道涌入 Plaid、Unit21、Sardine 等新竞品，Jack Henry 传统第三方合作模式面临毛利压缩。

**解读**

- **Payments 分部是 Jack Henry 最大业务**（36.8% 收入），任何 margin compression 都会显著影响整体利润率；
- $16M 年度收入逆风 + Q1 FY26 集中释放 $12M，**FY26 Q1 Payments 段同比或仅 +2-3%**（vs FY25 FY +6.8%）；
- 若后续第三方协议继续重构，则收入结构长期需要重塑——不过管理层指出**利润影响 minimal**（成本同步下降），实际盈利冲击有限；
- 银行账户总数增速放缓也直接影响 Jack Henry 按账户 / 按交易收费的支付处理业务，长期基本面 TAM 受限。

## 8. 总结判断

**一句话结论**

- 2025 财年 Jack Henry 交出 **GAAP 营收 +7.2% / 营业利润 +16.2% / EPS +19.3% / FCF +21.9% / 债务归零**的成熟 SaaS 转型答卷，**Non-GAAP EBITDA 利润率 31.4% / ROIC 22.1% / FCF Conversion 90%**；2026 财年受 deconversion 基数 + 第三方协议重组影响，GAAP EPS 增速将大幅放缓至 +1-3%，但 H1 FY26 已交付 **EPS +24.5% 强劲开局**；**Jack Henry 是美国社区银行 / 信用合作社 Core+Payments 龙头之一（与 Fiserv / FIS 并列），具备 Dividend Aristocrat 特质、高 ROIC 和长期防御性**。

**核心看点**

1. **Cloud 转型 + 新产品兑现**：Cloud Data Processing +12.0% / Rapid Transfers / Tap2Local / Jack Henry Platform 构成长期 ARR 增长引擎，行业并购潮中持续获客（FY25 51 个 core win + Q2 FY26 22 个）。
2. **现金牛 + 资本回报模式**：FY25 FCF $410M / 债务归零 / 现金 $102M / ROIC 22.1%，**美国新税法 R&D 全额费用化**将带来 FY26 FCF 结构性提升至 85-100% conversion；连续 35 年提高股息的 Dividend Aristocrat。
3. **并购行业整合 + 大客户获取**：美国社区银行并购浪潮下，"Jack Henry to Jack Henry" 并购锁定长期收入，大客户新签加速（FY25 大型金融机构新签显著增加）。

**核心风险**

1. **FY26 GAAP EPS 增速放缓**（指引 +1-3%）带来短期市场预期下修压力；Deconversion revenue 基数不确定 + Connect 大会季度错配导致季度 cadence 波动大。
2. **Core 市场 Price Compression 压力**：FY25 大客户续约已出现定价压力；云原生核心（Finxact / Thought Machine / FIS Modern Banking Platform）竞争加剧，可能影响未来 5-10 年核心续约节奏。
3. **Payments 分部压力 + 行业账户增速放缓**：第三方风控协议重构 -$16M 收入、银行账户总数增速降至 1-2%，对按账户 / 交易定价的 Payments 业务形成中期天花板效应。
