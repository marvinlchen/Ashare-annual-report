# NetApp, Inc. (Nasdaq: NTAP) 2025年财报整理

> 本报告基于 NetApp 的 **Fiscal Year 2025 年度业绩公告**（2025 年 5 月 29 日新闻稿、SEC Form 8-K Exhibit 99.1），**FY2025 财年截至 2025 年 4 月 25 日**；并整合其后公司 Form 10-K（2025 年 6 月 SEC 提交）和投资者沟通材料。
> 所有金额均按原始披露币种（美元）和单位（百万美元 $M），所有同比增速以"同比 +X.X% / -X.X%"格式呈现。
> **特别说明**：NetApp 财年**4 月底结束**，**FY2025 覆盖 2024/4/27 - 2025/4/25**。"2025 年财报"即 FY2025 Results。

## 1. 业绩总结

### 1.1 收入与利润表现

- **全年净收入 $6,572M，同比 +5%**（FY2024 $6,268M）——**创历史新高**，主要由 All-Flash Array + Public Cloud 服务强劲增长驱动；产品收入 **$3,040M（+6.7%）**，服务收入 **$3,532M（+3.3%）**。
- **GAAP 营业利润 $1,337M（+10%）**；**GAAP 营业利润率从 19.4% 提升至 20.4%（+100bp）**——**公司历史最高**。**Non-GAAP 营业利润 $1,861M，Non-GAAP 营业利润率 28%**，均创历史记录。
- **GAAP 净利润 $1,186M，同比 +20%**（FY2024 $986M）；**GAAP 稀释 EPS $5.67，同比 +22%**；**Non-GAAP 稀释 EPS $7.25，同比 +12%**（FY24 $6.46）。**Billings $6,780M，同比 +8%**（超过收入增速，反映未来收入动能）；**账面价值增长 + 减少股数回购贡献 EPS 放大**。

### 1.2 增长与拖累因素

- **主要增长驱动**：(1) **All-Flash Array ARR $4.1B，同比 +14%**：市场份额持续抢占，FY25 刷新历史纪录；(2) **First-Party and Marketplace Public Cloud services revenue $416M，同比 +43%**：与 Azure NetApp Files、Amazon FSx for NetApp ONTAP、Google Cloud NetApp Volumes 深度合作放量；(3) **Public Cloud 分部毛利率从 66.8% 升至 75.2%（+840bp）**：剥离 Spot by NetApp 后聚焦核心存储服务的收益；(4) **AI 数据基础设施业务放量**：NVIDIA DGX SuperPOD + NVIDIA Cloud Partners + NVIDIA-Certified Systems 认证 + NVIDIA AI Data Platform 集成；(5) **定价纪律 + 产品组合改善**：整个系统产品组合 FY25 完成刷新；(6) **Non-GAAP 营业利润率 28% 创历史记录**：规模效应 + 运营纪律双重改善。
- **主要拖累因素**：(1) **产品毛利率 -230bp 至 58.0%**（FY24 60.3%）：部件成本上升 + 降价策略夺取 All-Flash 市场份额；(2) **Hybrid Cloud 分部毛利率从 72.3% 降至 70.6%（-170bp）**：价格折让 + 技术换代成本；(3) **重组费用 +88.6% 至 $83M**（FY24 $44M）：企业架构调整 + 裁员；(4) **经营现金流 -10.7% 至 $1,511M**（FY24 $1,691M）：部分反映应收账款激增 + 重组支付；(5) **股东权益 -9.2% 至 $1,040M**：大额回购导致；(6) **长期债务 +24.8% 至 $2,485M**（FY24 $1,992M）：新发债务支持回购。
- **增长或承压主要来自**：**All-Flash Array + Public Cloud 服务 + AI 数据基础设施**三大引擎，带来 +5% 收入增长和 +10% 营业利润增长；**产品毛利率 -230bp + 重组费用 +88.6%**是内部调整代价；**Hybrid Cloud 分部毛利率下滑**是需要关注的结构性压力。

### 1.3 趋势判断

- **未来主要利好方向**：(1) **AI / 数据基础设施爆发性机会**：NetApp 通过 NVIDIA DGX SuperPOD 认证 + NVIDIA AI Data Platform 集成 + Domino Volumes（Enterprise AI Platform），**已成为 AI 数据基础设施（智能数据基础设施）的头部供应商**；(2) **All-Flash Array $4.1B ARR 持续扩张**：管理层强调 +14% 同比 + 市场份额扩张；(3) **Public Cloud 服务 +43% 加速增长**：First-Party（Azure NetApp Files + Amazon FSx + Google Cloud NetApp Volumes）深度合作；(4) **FY2026 全年指引上调**：收入 $6.625-6.875B（中点 $6.75B，同比 +2.7%），Non-GAAP EPS $7.60-7.90（中点 $7.75，同比 +7%），Non-GAAP 营业利润率 28.8-29.8%（继续扩张）；(5) **后量子密码 + 勒索软件保护**等新功能扩展安全市场；(6) **NFL + NVIDIA + Microsoft + Google + AWS 一系列重大生态伙伴关系**；(7) **产品组合刷新完成后**，FY2026 可望进一步提升价格动能。
- **未来主要压力方向**：(1) **Hybrid Cloud 分部毛利率持续压力**：产品毛利率 -230bp + 市场价格竞争；(2) **存储市场整体温和 +5% 增长**：NetApp 成长性受行业天花板限制；(3) **股东权益 -9% 降至 $1,040M**：积极回购 + 加大债务支撑，**财务杠杆持续上升**（债务/总资产从 24.2% 升至 29.9%）；(4) **公共部门收入从 11% 降至 11% 持平**（但 Q4 FY25 降至 9%）：**联邦政府采购可能受特朗普政府支出调整影响**；(5) **Pure Storage + Dell + HPE 持续竞争**：存储行业高度集中化，价格战持续；(6) **宏观经济 + 关税 + 美中紧张**：IT 基础设施支出在宏观紧缩时承压；(7) **重组 $83M + 裁员相关费用**：FY26 可能还有进一步重组。
- **整体趋势判断**：NetApp FY2025 在**存储行业温和 +5% 增长大背景**下实现了**收入创新高、营业利润率创新高、All-Flash Array ARR 创新高、Public Cloud 服务 +43% 加速、Non-GAAP 营业利润率 28% 创新高**的全面强劲业绩；**核心战略定位 "智能数据基础设施"**在 AI 浪潮下具有高度差异化；**FY2026 指引 Non-GAAP 营业利润率进一步扩张至 28.8-29.8%**，显示管理层对继续扩张盈利能力的强信心；**未来核心看点是 AI 数据基础设施份额扩张 + Public Cloud 服务放量 + All-Flash Array 市场主导地位**。

## 2. 主营业务

### 2.1 公司简介

NetApp, Inc.（Nasdaq: NTAP）是**全球领先的企业数据管理和智能数据基础设施（Intelligent Data Infrastructure）公司**，总部位于美国加州 San Jose，员工约 12,000 人。公司成立于 **1992 年**，核心产品是**存储系统（All-Flash / Hybrid Flash）+ 云存储服务 + 数据管理软件**，服务于**混合云、多云、AI 数据管理**场景。S&P 500 成员。主要竞争对手是 **Pure Storage、Dell PowerStore、HPE Alletra、Hitachi、IBM Storage**。

### 2.2 主营业务构成

NetApp 按**两大业务分部**划分：

1. **Hybrid Cloud（混合云）**：FY2025 收入 **$5,907M（+4.4%）**，占总收入 **~90%**。包含：
   - **产品（存储系统）**：$3,040M（+6.7%）——NetApp ASA（块存储）、AFF（All-Flash FAS）、EF（高性能 Flash）、C-Series（低成本 Flash）、FAS（混合 Flash）；FY2025 刷新了**整个系统产品组合**；
   - **支持服务（Support）**：$2,315M（+1% 估算）——年度硬件 + 软件支持合同；
   - **专业服务（Professional and Other Services）**：$94M（+22%）——部署 + 咨询服务。
2. **Public Cloud（公有云）**：FY2025 收入 **$665M（+8.8%）**，占总收入 **~10%**。包含：
   - **First-Party 和 Marketplace 公有云服务 $416M（+43%）**：Azure NetApp Files、Amazon FSx for NetApp ONTAP、Google Cloud NetApp Volumes（AWS + Azure + GCP 深度集成）；
   - **Cloud Operations / Observability**：CloudOps 服务（Spot by NetApp 已出售给 Flexera）；
   - **Keystone 存储即服务**：基于订阅的混合云存储模式；
   - **Cloud Volumes ONTAP**：基于 AWS / Azure / GCP 的 NetApp ONTAP 云版本。

### 2.3 核心收入来源

- **按业务分布**（FY2025）：Hybrid Cloud $5,907M（90%）+ Public Cloud $665M（10%）= $6,572M；
- **按收入类别**：产品收入 $3,040M（46%）+ 服务收入 $3,532M（54%，含支持 + 专业服务 + 公有云服务）；
- **按地区**：**美洲 51%（美洲商业 40% + 美国公共部门 11%）+ EMEA 34% + APAC 15%**；
- **按客户**：财富 500 强 + 政府 + 电信运营商 + 金融服务 + 医疗等企业客户，无单一大客户集中；
- **核心产品组合**：
  - **All-Flash Array**（全闪存阵列）：FY25 ARR 达 **$4.1B（+14%）**——**NetApp 最核心的增长引擎**；
  - **NetApp ONTAP 数据管理 OS**：行业领先的企业存储操作系统，支持混合云、容器化、AI 工作负载；
  - **StorageGRID**：对象存储（用于大规模非结构化数据）；
  - **BlueXP**：统一数据管理控制平面，跨本地 + 多云；
  - **SnapLock / SnapCenter**：数据保护 + 合规。

**资产负债率变化**

NetApp 的债务/总资产率从 **24.2% 升至 29.9%（+570bp）**，主要由**新发 $750M + $493M 长期债务**（合计 $1.24B 新增债务）支持积极股份回购。尽管如此，公司仍处于 **净现金状态**（现金 $3,850M > 总债务 $3,235M）。相较于半导体、软件行业的大型并购带来的极端杠杆（如 Skyworks-Qorvo 将增加 $3B+ 债务），NetApp 的财务杠杆仍处于健康水平。

**偿债压力判断**

- **信用评级**：**BBB+ (S&P) / Baa1 (Moody's)**，长期稳定；
- **利息覆盖率**：GAAP Op Income $1,337M / 利息估算 $100M = **13x**，健康；
- **流动性**：$3,850M 现金 + 未用循环信贷 = 充足流动性；
- **债务期限**：$750M 即将到期（FY26）；
- **偿债压力可控**，但需关注 FY26 债务再融资。

**商誉风险判断**

- **商誉 + 无形资产**：**未明确披露**，但从资产负债表推算 ~$2.5-3B（主要来自 Solidfire、CloudJumper、Spot.io 等 2017-2020 年收购）；
- FY2025 **出售 Spot by NetApp 给 Flexera**，实现了部分商誉退出；
- **商誉进一步减值风险中等**：Public Cloud 分部表现较好，但 Spot 剥离表明管理层承认前期扩张的"未能兑现价值"。

### 4.3 营收分析

**主要增长来源**

1. **All-Flash Array ARR $4.1B（+14%）**：FY25 市场份额持续抢占，**是 NetApp 最核心的增长引擎**；

6. **Billings $6,780M（+8%）**：超过收入增速 +3pt，**反映未来 2-3 季度收入动能强**。

**主要拖累因素**

1. **Hybrid Cloud 分部毛利率 -170bp**：产品毛利率 -230bp，反映部件成本上升 + 抢占市场份额的价格策略；

3. **经营现金流 -10.7%**：重组支付 + 营运占用；
4. **Q4 FY25 公共部门从 11% 降至 9%**（季度占比）：美联邦政府采购可能出现暂停；
5. **美洲收入占比 51%，但美国公共部门从 11% 降至 9%（Q4）**：预示 FY26 可能进一步下滑。

**结构变化**

- **Public Cloud 占比从 9.8% 升至 10.1%**：结构性提升；
- **Public Cloud 毛利率从 66.8% 升至 75.2%（+840bp）**：产品组合大幅改善；
- **Product 毛利率从 60.3% 降至 58.0%**：价格战压力。

## 6. 未来可能的利好

### 利好 1：AI 数据基础设施爆发性机会 —— NVIDIA 深度合作伙伴

**依据**

**解读**

NetApp 通过与 **NVIDIA 的深度集成**，已将自己定位为 **AI 训练 + 推理基础设施的关键数据层**。这是 FY2025 最重要的战略胜利，也是 FY2026+ 最大的增长催化剂。企业客户在构建私有 AI 集群时，NetApp ONTAP + NVIDIA GPU 的组合正成为**事实标准**。管理层预计 AI 相关业务 FY2026 将带来**$100-200M 增量**，长期可能达到 $500M+ 规模。估值角度，**AI 基础设施纯粹玩家** 在 2024-2025 年估值溢价显著，NetApp 有望享受这一红利。

### 利好 2：All-Flash Array 市场份额持续扩张

**依据**

- All-Flash Array **ARR $4.1B，同比 +14%**，FY25 历史新高；
- FY2025 **刷新整个系统产品组合**：ASA（block storage）扩展至远程分支办公室、EF 引入 30TB / 60TB QLC 硬盘、FAS50 新入门级系统；
- 30TB / 60TB QLC 硬盘适合媒体娱乐、HPC、AI、高性能数据库等工作负载；
- NetApp 在全闪存市场与 **Pure Storage、Dell PowerStore、HPE Alletra** 并列前三；
- 根据 IDC 数据，全球 All-Flash Array 市场年均 +8-10% 增长，NetApp +14% 超行业增速。

**解读**

All-Flash Array 是数据存储市场的"圣杯"：更快、更可靠、更能耗经济。NetApp ARR +14% 超过行业 +8-10% 意味着**持续抢占市场份额**。刷新后的产品组合（ASA + AFF + EF + C-Series + FAS）覆盖从入门级到高端的全价位，加上 ONTAP 操作系统的软件价值，形成了**差异化护城河**。未来 3-5 年内，All-Flash Array 市场份额提升有望为 NetApp 带来**每年 +$500-800M 增量收入**。

### 利好 3：Public Cloud 服务 +43% 加速增长 + First-Party 深度绑定

**依据**

- **First-Party and Marketplace Public Cloud services 收入 $416M，同比 +43%**；

**解读**

Public Cloud 服务是 NetApp 的**"云原生转型"的核心证据**。与 AWS、Azure、GCP 形成 First-Party 合作伙伴关系意味着：(1) **云厂商直接销售 NetApp 服务**（不是转售），大幅降低获客成本；(2) **客户可以通过云厂商的账单统一付费**（减少采购摩擦）；(3) **深度技术集成**（比如 Azure NetApp Files 已经是 Azure 的原生服务）。+43% 增速远超 Public Cloud 市场整体 +25-30%，反映了 NetApp 的差异化优势。**Public Cloud 收入 $665M + 毛利率 75.2% = 毛利贡献 $500M**，已经成为公司重要的毛利增长引擎。

## 7. 未来可能的利空

### 利空 1：Hybrid Cloud 分部毛利率持续压力

**依据**

- **Hybrid Cloud 分部毛利率从 72.3% 降至 70.6%（-170bp）**；
- **产品毛利率从 60.3% 降至 58.0%（-230bp）**；
- 部件成本（NAND flash, DRAM）2024-2025 年上升 10-15%；
- Pure Storage 在 FY25 保持产品毛利率 70%+（NetApp 的 58% 形成反差）；
- Dell PowerStore、HPE Alletra 持续降价抢份额；
- FY26 行业毛利率预期继续承压。

**解读**

Hybrid Cloud 分部占 NetApp 收入 90%，其毛利率下滑直接影响整体盈利能力。产品毛利率 -230bp 是一个**结构性警告信号**。除非 NetApp 能：(1) 在 QLC 硬盘 + NVMe / PCIe Gen 5 等新技术上获得价格溢价；(2) 提升软件订阅比例（ONTAP 数据管理软件更高毛利）；(3) 扩大 Public Cloud 服务占比（75% 毛利率），否则**整体毛利率可能继续承压 50-100bp**。管理层在 FY2026 指引中维持 Non-GAAP 毛利率 71-72%，反映挑战仍在。

### 利空 2：美联邦政府采购 + 宏观经济风险

**依据**

- 美国公共部门收入 **全年占比 11%，但 Q4 FY25 降至 9%**（季度占比下降 2pt）；
- 特朗普政府 2025 年 1 月上台后推行 **"政府效率部门（DOGE）"**，可能削减联邦 IT 支出；
- 2025 年美国联邦 IT 预算（FITARA）面临审计 + 裁员压力；
- NetApp 在国防部、退伍军人事务部（VA）、医疗保险（CMS）等部门有大型合同；
- 2026 联邦财政年度（FY26 10/1/25 开始）存在政府关门风险。

**解读**

美国公共部门业务 11% 占比 + 核心存储合同（军工、情报、医疗等）使 NetApp 对**美国联邦 IT 支出削减**有直接暴露。Q4 FY25 已经出现 2pt 占比下滑，预示 FY2026 可能进一步下降。如果美联邦支出削减**25% 的存储采购**，NetApp 将损失 ~$150M 年收入（约 2%）。加上宏观经济 + 关税 + 美中紧张，企业 IT 基础设施支出在 2026 年可能面临短期压力。

### 利空 3：重组费用 + 组织架构调整的执行风险

**依据**

- FY2025 重组费用 **$83M，同比 +88.6%**（FY24 $44M）；
- Q4 FY25 重组费用 $45M（Q4 FY24 为 $0）；
- 管理层在 FY2024 后期进行**"Public Cloud 分部聚焦"**（Spot 剥离）+ **裁员**；
- Frank Pelzer（Spotnana COO）新加入董事会，Beth O'Callahan 升任 Chief Administrative Officer；
- **FY2026 可能继续进行组织架构优化**。

**解读**

重组费用 +88.6% 表明 NetApp 正在经历**深度组织架构调整**。虽然从 FY25 营业利润率 +100bp 看这些重组已经带来效果，但**执行风险依然存在**：(1) 裁员可能影响员工士气 + 创新能力；(2) 管理层调整（Pelzer + O'Callahan）需要时间磨合；(3) Public Cloud 聚焦后，Spot 剥离可能带来一次性收入下滑（FY25 已显现 -22% "Other Public Cloud"）；(4) FY2026 可能还有 $50-80M 重组费用。

### 利空 4：债务 +35.2% 显著上升 + 股东权益 -9.2%

**依据**

- 总债务 **$3,235M，同比 +35.2%**（FY24 $2,392M）；
- 长期债务 +24.8% 至 $2,485M（新发债）；
- 当期长期债务 +87.5% 至 $750M（FY26 到期）；
- 股东权益 **$1,040M，同比 -9.2%**（FY24 $1,146M）；
- 债务/股东权益 = 3.1x（显著高于软件行业平均 0.8x）；
- 债务/总资产 29.9%（+570bp YoY）。

**解读**

NetApp 通过**加大债务 + 股份回购**来放大 EPS 增长，是**财务工程 + 实际业务增长的混合**。$3,235M 总债务 vs $1,040M 股东权益，**Debt/Equity 3.1x 远高于软件行业均值 0.8x**。虽然现金 $3,850M > 总债务，仍处于净现金状态，但**财务弹性已经显著下降**。如果未来：(1) 利率上升；(2) 股票估值下跌（回购回报下降）；(3) 商誉减值；(4) 重大并购需要资金，NetApp 的财务结构可能受到挑战。

### 利空 5：存储行业温和增长 + Pure Storage / Dell / HPE 竞争加剧

**依据**

- 全球企业存储市场 FY25 **温和 +5% 增长**（vs AI 芯片 +50%+ 增长）；
- Pure Storage FY25 收入 +11% 超越 NetApp +5%；
- Dell PowerStore + HPE Alletra 持续降价抢份额；
- **Hyperscalers（AWS / Azure / GCP）的自研存储**（比如 AWS EFS、Azure Files）持续挤压 NetApp 的高端企业存储；
- 2024-2025 年，部分企业将数据迁移到 S3 / Azure Blob / GCS 等**对象存储**（降低 NetApp 文件存储业务）。

**解读**

NetApp 的**行业增长天花板**是长期挑战。即使 NetApp 份额扩张 + Public Cloud +43% 加速，整体业务仍受存储市场 +5-8% 增长限制。**Pure Storage 作为纯粹全闪存玩家持续 +10-15% 增长**，NetApp 5% 的总体收入增长显得相对疲软。未来 NetApp 需要：(1) 加速 AI 基础设施业务；(2) Public Cloud 从 10% 提升至 20%；(3) 产品组合持续创新，才能维持股价上涨动能。

### 利空 6：宏观经济 + 关税 + 美中紧张的间接冲击

**依据**

- NetApp 供应链依赖**台湾 TSMC、中国 / 韩国的内存 + 硬盘制造**；
- 美国对华 2025 年新关税政策 + 出口管制加码；
- 美洲（美国 + 加拿大 + 拉美）收入占 51%，但其中**联邦 / 州政府 11%**；
- EMEA 34% 部分在俄乌战争影响下；
- APAC 15% 部分在美中紧张中受冲击；
- FY25 有效税率 14.2% 比 FY24 21.9% 显著下降 —— **未来可能反弹**（如果税务 safe harbor 结束）。

**解读**

NetApp 作为**全球 IT 基础设施供应商**，对宏观经济和地缘政治敏感。(1) 关税上调可能推高硬盘 / 内存成本，进一步压制毛利率；(2) 美中紧张可能限制 NetApp 对中国银行 / 央企的销售；(3) 欧洲地缘政治 + 经济增长乏力影响 EMEA 34% 业务；(4) FY25 有效税率 14.2% 低于正常水平，FY26 可能反弹至 19-21%（拖累 EPS +5-7%）。

## 8. 总结判断

**一句话结论**

NetApp FY2025 在**存储行业温和 +5% 增长大背景**下实现了**收入 $6.572B 创历史新高（+5%）、GAAP EPS $5.67（+22%）、Non-GAAP 营业利润率 28% 创历史新高**的全面强劲业绩；**核心战略 "智能数据基础设施" 在 AI 浪潮下具有高度差异化**，**NVIDIA 深度合作 + Public Cloud 服务 +43% 加速 + All-Flash Array ARR $4.1B（+14%）** 构成三大核心增长引擎；**FY2026 指引 Non-GAAP EPS +7% + 营业利润率进一步扩张至 29.3%**，显示管理层高度信心；**主要风险是 Hybrid Cloud 毛利率压力 + 联邦政府采购下滑 + 财务杠杆上升**。

**核心看点**

1. **NVIDIA 深度合作 + AI 数据基础设施业务爆发**：DGX SuperPOD + NVIDIA AI Data Platform + Domino Volumes；
2. **All-Flash Array ARR $4.1B（+14%）持续市场份额扩张**：刷新整个产品组合（ASA / AFF / EF / C-Series / FAS / FAS50）；
3. **Public Cloud 服务 +43% 加速**：First-Party 合作（Azure NetApp Files + Amazon FSx + Google Cloud NetApp Volumes），毛利率升至 75.2%（+840bp）；
4. **Non-GAAP 营业利润率 28% 创历史新高**：FY2026 指引进一步扩张至 29.3%；
5. **$1.57B 股东回馈 + 回购加速 + 股息稳定**；
6. **重大生态伙伴**：NFL、NVIDIA、Microsoft、Google、AWS、Domino。

**核心风险**

1. **Hybrid Cloud 分部毛利率 -170bp + 产品毛利率 -230bp**：价格竞争 + 部件成本双重压力；
2. **美联邦政府采购削减**：Q4 FY25 公共部门占比从 11% 降至 9%；
3. **重组费用 +88.6% 至 $83M**：FY26 可能还有组织架构调整代价；
4. **总债务 +35.2% 至 $3.24B + 股东权益 -9.2%**：财务杠杆快速上升；
5. **存储行业天花板 +5-8% 温和增长**：Pure Storage +11% 增速对比 NetApp +5% 显疲软；
6. **宏观经济 + 关税 + 美中紧张**：企业 IT 基础设施支出短期承压。
