# Oracle Corporation (ORCL) 2025年财报整理

> 报告期：FY2025 全年（12 个月，止 2025/05/31）—— 财年 ≠ 日历年（Oracle 财年 6 月-次年 5 月）
> 申报币种：美元；会计准则：U.S. GAAP，并披露 Non-GAAP（Adjusted EPS / Operating Margin / Organic Revenue / Constant Currency）
> 数据来源：Oracle FY2025 Q4 & Full-Year Earnings Release（2025/06/11 发布）+ 10-K（2025/09/26）+ 8-K
> FY25 期间主要战略事件：**AI Infrastructure 投资爆发式增长**：Capex $212 亿（+209% YoY）、**Remaining Performance Obligations (RPO) $1,380 亿**（+41%）史无前例积压订单、**MultiCloud 数据库合作扩展**：AWS + Google Cloud + Microsoft Azure 一次性查询数据库，Q3→Q4 增长 +115%、**Oracle Cloud@Customer 29 个数据中心上线 + 30 个在建**：+104% YoY、**Cloud Infrastructure (OCI) consumption revenue +62% Q4**：AI 推理 + 训练需求爆发、**Stargate 项目启动**（与 OpenAI / SoftBank / MGX 合作，规划 $5,000 亿 AI 数据中心投资）、**Fusion Cloud ERP $10 亿 Q4 (+22%) + NetSuite Cloud ERP $10 亿 Q4 (+18%)**、**FY26 Guidance：总云增速 +40%+ / Cloud Infrastructure +70%+ / RPO +100%**：管理层极度乐观
> FY25 特别会计事项：**有效 GAAP 税率 12.1%**（vs FY24 10.9%）：部分法律实体重组带来税收优惠；**Non-GAAP 调整 $73.55 亿美元 / EPS $1.69**，包括：Stock-Based Compensation $46.74 亿 + 无形资产摊销 $23.07 亿 + Acquisition Related $0.75 亿 + Restructuring $2.99 亿

## 1. 业绩总结

### 1.1 收入与利润表现

- 全年总收入 **$573.99 亿美元**，同比 **+8%**（USD）/ **+9%**（Constant Currency）
- **云服务 + License Support 收入 $440.29 亿**（+12%）：**占总收入 77%**（vs FY24 74%）
- **Cloud License + On-Premise License 收入 $52.01 亿**（+2% USD / +3% CC）
- **Hardware 收入 $29.36 亿**（-4%）
- **Services 收入 $52.33 亿**（-4%）
- **GAAP 营业利润 $176.78 亿**（+15%），**营业利润率 31%（+181 bps）**
- **Non-GAAP 营业利润 $250.33 亿**（+9%），**Non-GAAP 营业利润率 44%（+8 bps）**
- **GAAP 净利润 $124.43 亿**（+19%），**GAAP 净利率 22%**
- **Non-GAAP 净利润 $172.84 亿**（+10%）
- **GAAP 摊薄 EPS $4.34**（+17% vs $3.71），**Non-GAAP 摊薄 EPS $6.03**（+8% vs $5.56）
- **经营现金流 $208.21 亿美元**（+12%），**但资本开支 $212.15 亿（+209%）**导致**FY25 自由现金流仅 -$3.94 亿美元（近零）**，这是战略性 AI 投资的结果
- Q4 FY25：Revenue $159.03 亿（+11%）、GAAP EPS $1.19、**Non-GAAP EPS $1.70**
- **RPO（剩余履约义务）$1,380 亿美元 +41% YoY**：Oracle 历史最高订单积压
- FY26 Guidance：**总云增长 +40%+（vs FY25 +24%）、Cloud Infrastructure +70%+（vs FY25 +50%）、RPO +100%+**

### 1.2 增长与拖累因素

- **主要增长驱动**：① **Cloud Services + License Support +12%**：核心 SaaS + IaaS + PaaS + Support 业务全面增长；② **Q4 Cloud Revenue (IaaS+SaaS) $67 亿 +27%**：加速；③ **Cloud Infrastructure (OCI / IaaS) +50%**（FY25）/ +52%（Q4）：AI 训练 + 推理需求爆发，OCI 以低价 GPU + 高性能 RDMA 网络 + 数据主权 Cloud@Customer 差异化；④ **OCI Consumption Revenue +62% Q4**：真实使用量爆发；⑤ **MultiCloud（AWS/Azure/Google）+115% Q3→Q4**：Oracle Database@AWS / Azure / Google 协作，独特差异化；⑥ **Fusion Cloud ERP +22% Q4**：SAP 迁移 + 数字化转型；⑦ **NetSuite Cloud ERP +18% Q4**：中小企业 ERP 龙头；⑧ **RPO $1,380 亿 +41%**：未来 2-3 年超级巨大订单池；⑨ **AI 超大客户签约**：OpenAI / Meta / xAI / Cohere / 微软等训练 OCI；⑩ **Cloud@Customer +104%**：主权云 / 私有云 + 政府 + 金融 + 电信强需求
- **主要拖累因素**：① **Hardware -4%**：硬件业务持续衰退，战略重心向云转移；② **Services -4%**：咨询 / 实施服务萎缩；③ **Cloud License + On-Premise +2%**：传统 License 业务低增长；④ **Operating expenses $397.21 亿 +6%**：Cloud infrastructure OpEx +23% 投入期；⑤ **资本开支 $212.15 亿 +209%**：对 GPU / 数据中心大规模投入，FCF 转为负数；⑥ **利息支出 $35.78 亿**：大规模债务融资用于 AI CapEx；⑦ **外汇 -1pp 收入**：美元走强影响；⑧ **Cloud@Customer / MultiCloud 短期利润压力**：前期投入高，毛利率低
- **一次性/异常项目**：① **Stock-Based Compensation $46.74 亿**（+18% vs $39.74 亿）：对 AI 人才 SBC 增加；② **无形资产摊销 $23.07 亿**（-23%）：部分 Cerner 等收购摊销到期；③ **Restructuring $2.99 亿**（-26%）；④ **Acquisition Related $0.75 亿**（-76%）：Cerner 整合尾声；⑤ **$19.55 亿净债发行**（用于 AI CapEx）+ $15.84 亿债务偿还；⑥ **Capex 从 $69 亿激增到 $212 亿**（+$143 亿/+207%）：改变自由现金流轨迹

### 1.3 趋势判断

- **未来主要利好方向**：① **AI Infrastructure 超级订单周期**：RPO $1,380 亿（+41%）+ OpenAI Stargate 项目 + xAI + Cohere + Meta 大单；② **Cloud Infrastructure 增速 +50%→+70%+**：FY26 加速；③ **MultiCloud 独特生态**：AWS + Azure + Google Cloud 三巨头都卖 Oracle Database，AI 训练推理时代必备；④ **Cloud@Customer 双位数 + 主权云 + 政府 + 金融**：差异化数据留在客户数据中心；⑤ **Fusion ERP + NetSuite**：应用层 SaaS 持续双位数增长；⑥ **Database 业务**：Autonomous Database + 24ai vector search + AI 功能；⑦ **$500B Stargate 项目 FY26-28 执行**：与 OpenAI + SoftBank + MGX 联合投资 AI 基础设施；⑧ **FY26 Guidance：云 +40%+ / OCI +70%+ / RPO +100%+**：创记录增长预期；⑨ **股息 $2.00/年（+25%）+ 持续回购**：股东回报；⑩ **Larry Ellison 继续 CTO + CEO Safra Catz 执行力**：创始人主导 AI 战略
- **未来主要压力方向**：① **资本开支激增 + FCF 负数**：FY25 Capex $212 亿 / FCF -$4 亿（vs FY24 $69 亿 / FCF $118 亿），FY26 Capex 可能 $250-300 亿，FCF 可能继续承压；② **债务累积**：总债务 $925 亿（+$57 亿 YoY）+ 利息支出 $35.78 亿；③ **AI 超大客户集中风险**：OpenAI 单客户占 RPO 可能 10-15%，若关系恶化冲击巨大；④ **GPU 供应 + Nvidia 依赖**：H100/H200/B200 GPU 供应 + 成本风险；⑤ **云巨头竞争**：AWS / Azure / Google Cloud 规模巨大 + Oracle Cloud 市占 2-3%；⑥ **毛利率压力**：Cloud OpEx +23% vs 收入 +12%（Cloud）- 投入前置期；⑦ **EPS 增速慢于收入**：Non-GAAP EPS +8% vs 收入 +8%；⑧ **股价估值**：2025 年股价 +80%+，远期 PE 40 倍，预期已高；⑨ **Stargate 项目执行风险**：$500B 规模巨大，融资 + 建设 + 电力供应挑战
- **整体趋势判断**：**Oracle FY2025 交出"收入 $574 亿 (+8%)、云 +12%、OCI +50%、RPO +41% 至 $1,380 亿、Non-GAAP EPS $6.03 (+8%)、Capex $212 亿 (+209%)、FY26 云增速 +40%+ 指引"的历史转折点业绩**，**充分体现"从传统 License 公司成功转型为 AI + 云基础设施巨头"的深刻变革**。**FY26-FY28 核心看点是 AI 超级订单兑现（RPO $1,380 亿 → 收入）+ Stargate 项目（$500B）+ Cloud Infrastructure +70%+ + MultiCloud 生态 + Database + ERP 继续增长**；**核心风险是 Capex 激增 + FCF 负数 + AI 大客户集中 + 巨头竞争 + 高估值 + Stargate 执行风险**

## 2. 主营业务

### 2.1 公司简介

Oracle Corporation（ORCL）是**全球领先的企业级软件 + 云基础设施公司**，总部位于德克萨斯州 Austin（2020 年从加州迁出）。**1977 年由 Larry Ellison、Bob Miner 和 Ed Oates 创立**，至今已有 **48 年历史**。公司业务涵盖**云服务（Cloud Services + Infrastructure + SaaS 应用）、License 软件（Database + Middleware + Applications）、Hardware（服务器 + 存储）、Services（咨询 + 教育 + 支持服务）**。**产品主力 = Oracle Database**（全球 #1 关系型数据库，市占 ~40%），+ **Oracle Cloud Infrastructure (OCI)**（云基础设施，AI 训练推理新星），+ **Oracle Fusion Cloud Applications**（ERP / HCM / CRM / SCM），+ **NetSuite**（SMB SaaS ERP 龙头），+ **Cerner**（2022 年 $283 亿收购，医疗 IT）。**全球员工 159,000+**，**服务 175+ 国家**。Chairman + CTO：**Larry Ellison**（创始人，81 岁仍活跃）；CEO：**Safra Catz**（2014 年与 Mark Hurd 共任 CEO，2019 年独任）。**2025 年底市值约 $6,500 亿美元**（2025 年股价 +80%+，Oracle 历史首次进入全球市值前 10）。

### 2.3 核心收入来源与产品

**Cloud services and license support（76.7% + 最大 + 增长最快板块）**

包括两大类：
1. **Cloud Services**（SaaS + IaaS + PaaS）：按使用订阅付费
   - **Oracle Cloud Infrastructure (OCI / IaaS)**：云计算 + 存储 + 网络 + AI GPU 集群
     - FY25 Cloud Infrastructure 全年增长 **+50%**
     - Q4 $30 亿 +52%
     - OCI Consumption Revenue +62% Q4
     - AI 训练主战场：OpenAI + Meta + xAI + Cohere + Anthropic（部分）
   - **Cloud Applications (SaaS)**：企业应用
     - **Fusion Cloud ERP**：$1.0 亿 Q4 +22%（Oracle 原生 ERP，对标 SAP S/4HANA）
     - **NetSuite Cloud ERP**：$10 亿 Q4 +18%（中小企业 ERP 龙头）
     - **Fusion Cloud HCM**（人力资源）+ CX（客户体验）+ SCM（供应链）
     - **Cerner Millennium / Health Cloud**（2022 年 $283 亿收购整合）
   - **MultiCloud Database**：+115% Q3 → Q4，Oracle Database 在 AWS / Azure / Google Cloud 上部署
   - **Cloud@Customer**：29 个数据中心上线 + 30 个在建（+104% YoY）
2. **License Support**：传统 on-premise 软件的技术支持（Database + Middleware + Applications）
   - 高毛利、高留存、稳定现金流业务
   - 客户包括 Fortune 500 中的 400+、全球 175 国家

**Cloud license and on-premise license（9.1%）**

传统 License 软件新销售：
- Oracle Database（关系型 + Exadata + Autonomous Database）
- Oracle Middleware（Java / WebLogic / Fusion Middleware）
- Oracle Applications（E-Business Suite / PeopleSoft / JD Edwards / Siebel / Hyperion）
- 增速 +2%：传统业务低增长，客户逐步向云迁移

**Hardware（5.1%）**

- Engineered Systems（Exadata Database Machine、Exalogic）
- Sun Servers + Storage（Oracle 2010 年 $74 亿收购 Sun Microsystems）
- 增速 -4%：硬件业务持续萎缩，但 Exadata 仍是 Database 客户首选

**Services（9.1%）**

- Oracle Consulting（咨询）
- Oracle University（教育 / 培训）
- Advanced Customer Support
- 增速 -4%：咨询服务外包给合作伙伴（Deloitte / Accenture / TCS）

**Oracle 在 2025 年完成了"价值股到 AI 成长股"的历史性转型**：① **Larry Ellison 的 AI 押注已成功**：2023-2024 年 OCI 还被视为"追赶者"，2025 年 OCI 通过低价 + 网络 + MultiCloud + OpenAI/xAI 客户获取已成为 AI 基础设施"第二极"；② **股价 +100% 反映的是"转型预期 + 订单兑现"双重逻辑**：RPO $1,380 亿 + FY26 +40%+ 云增长指引 + Stargate 催化；③ **Forward PE 40x 看似偏高，但 FY26 EPS 如果达到 $7.50+（+25% vs FY25 $6.03），PE 会降至 32x**（如果股价保持）；④ **股息连续 14 年增长 + $0.50/季度** 是稀缺的"AI 成长 + 股息"组合；⑤ **创始人持股 >40%**：与股东利益高度一致；⑥ **长期愿景**：Oracle 目标成为"AI 基础设施市占前 3"（AWS / Azure / Oracle）；⑦ 即使 FY26 未达到 +40%+ 云增长指引，只要 +30% 也仍是超过市场预期的业绩，因此**下行风险相对有限**；⑧ 相比 Nvidia 股价波动性（GPU 周期），Oracle 作为"订阅收入 + 长期合约"模式更稳定。

## 7. 未来可能的利空

### 利空 1：Capex 激增 + FCF 转负 + 债务累积 + AI 投资回报周期不确定性

**依据**

- **FY25 Capex $212.15 亿美元（+209% YoY）**：vs FY24 $68.66 亿
  - 主要用于 AI 数据中心 + GPU 采购 + 网络基础设施
  - FY25 Q4 单季 Capex $62.82 亿（占全年 30%）
  - PP&E 资产从 $215 亿增至 $435 亿（+102%）
- **FY25 自由现金流 -$3.94 亿**（近 20 年首次负数）：vs FY24 +$118 亿
  - FY23 FCF $10.5 亿
  - FY24 FCF $118.07 亿（+1023%）
  - FY25 FCF -$3.94 亿（-103%）
- **FY26 Capex 预计 $300-350 亿美元**（+40-65%）：可能继续压制 FCF
- **债务累积**：
  - 总债务 $925.68 亿 (+$57 亿 vs FY24)
  - 长期债务 $852.97 亿 (+$90 亿)
  - FY25 新发债 $19.55 亿 + 商业票据 +$18.89 亿
  - FY26 预计继续发债 $30-50 亿
- **利息支出 $35.78 亿**（占 Op 20%）
- **GPU 生命周期**：Nvidia H100 → H200 → B100 → B200 → Rubin，每 1.5-2 年一代，**旧 GPU 折旧 + 减值风险**
- **租赁负债 $115.36 亿（+84%）**：数据中心租赁 + 电力合约长期负债
- **AI 投资回报**：
  - OpenAI 等 AI 客户是否能持续支付 $300B 合约？
  - AI 推理利润率是否可持续（vs 训练）？
  - 如果 AI 泡沫破灭，Oracle 数据中心可能出现"搁浅资产"

**解读**

**Oracle FY25 已进入"烧钱建设 + 押注 AI"阶段**：① **FCF 从 +$118 亿降至 -$4 亿**：短期内市场是否容忍取决于 RPO 兑现；② **Capex / Revenue 比率从 13% 升至 37%**：远超 AWS (~15%) + Azure (~18%)，显示 Oracle 正在"超级投入"；③ **PP&E 从 $215 亿增至 $435 亿（一年翻倍）**：如此快的扩张历史罕见，执行风险 + 供应链风险 + 电力风险都增加；④ **GPU 折旧加速**：Nvidia 2026 年 B200 + Rubin 可能使 H100 快速贬值，Oracle 如果持有 $100+ 亿 H100 库存，潜在减值 $20-30 亿；⑤ **债务持续扩张**：总债务从 $869 亿到 $926 亿，FY27-FY28 可能达 $1,200-1,400 亿，信用评级 A 可能降至 A-；⑥ **AI 推理市场未成熟**：现在 80% 算力用于训练，推理未形成规模化盈利模式；⑦ **Capex 回报期 5-7 年**：如果 AI 需求低于预期或 OpenAI 合约违约，Oracle 可能面临严重"搁浅资产"；⑧ 投资者需要密切监控 **RPO → 收入转化速度**（如果 12 个月内确认比例从 35% 降至 25%，可能预示订单兑现延迟）。

### 利空 2：AI 超大客户集中风险 + OpenAI / Stargate 执行风险 + 云巨头竞争（AWS / Azure / Google）

**依据**

- **客户集中度高**：
  - OpenAI 占 RPO 估计 **15-25%**（$300B 5 年合约）
  - xAI + Meta + Cohere + 其他 AI 客户：再 15-20%
  - **Top 10 AI 客户可能占 RPO 40-50%**
- **OpenAI 风险**：
  - OpenAI 2024 年营收 $37 亿，2025 年预计 $130 亿
  - 但 OpenAI 同时烧钱 $30-50 亿/年（训练成本）
  - Microsoft (Azure) 原本是 OpenAI 最大投资者（~$130B）
  - OpenAI 签 Oracle $300B 意味着"分散 Azure 依赖"
  - 如果 OpenAI 商业模式 / 监管 / 竞争出问题，Oracle 合约可能缩水
- **Stargate $500B 项目风险**：
  - SoftBank 融资能力受 Masayoshi Son 2025 年债务压力限制
  - MGX（阿联酋主权基金）政治风险
  - 4 年 $500B = 年均 $125B，执行进度 + 电力供应 + 芯片供应都是挑战
  - 部分 AI 专家质疑 $500B 规模的必要性（AI 效率提升可能降低基础设施需求）
- **云巨头竞争**：
  - **AWS**：2025 年收入 $1,100+ 亿，增速 +20%，基础设施规模 5x Oracle
  - **Azure**：2025 年收入 $1,200+ 亿（含 Copilot），增速 +35%，Microsoft 软件整合优势
  - **Google Cloud**：2025 年收入 $500+ 亿，增速 +35%，Gemini + TPU 生态
  - **Oracle Cloud**：2025 年总云 $240 亿（OCI $105 亿），市占 2-3%
  - Oracle 虽增速领先（+50%），但绝对规模差距巨大
- **Nvidia GPU 供应**：
  - Oracle 需要大量 H100/H200/B200 GPU
  - Nvidia 优先供应给 Microsoft + Google（合作更早 + 关系更深）
  - GPU 供应瓶颈 → Oracle 交付延迟
- **电力供应**：
  - 德克萨斯 / 爱达荷 / 内华达 数据中心电力紧张
  - 环保审批周期长（2-3 年）
  - 电价上涨风险

**解读**

**Oracle 的 AI 故事严重依赖 OpenAI + Stargate，任何一个出问题都可能导致股价巨大波动**：① **OpenAI 单客户集中度 15-25% 是 Oracle 最大脆弱点**：如果 OpenAI IPO 延迟、商业模式受挑战、监管限制 ChatGPT，Oracle 合约可能减少 30-50%；② **AWS + Azure + Google 规模优势**：规模经济下，巨头可以提供更低单价 + 更全栈服务，Oracle 的+价格优势可能被稀释；③ **Nvidia 与 Oracle 关系**：虽然合作深厚，但 Nvidia 可能优先供应给微软 (CoreWeave)、Google (Anthropic)、Meta 等更大客户，Oracle 获 GPU 排队靠后；④ **Stargate $500B 过于乐观**：OpenAI + SoftBank + MGX 是否有能力在 4 年内实际投入 $500B 存在重大疑问，$200-300B 可能更现实；⑤ **AI 基础设施产能过剩风险**：AWS + Azure + Google + CoreWeave + Lambda + Oracle 全球总 AI 算力可能过剩 30-50%（如 Meta 2026 年宣布 Capex 增长放缓）；⑥ **Oracle 规模劣势**：如果 AI 基础设施从爆发期进入饱和期（2027-2028），Oracle 作为"小后来者"可能首当其冲受到挤压；⑦ 如果 AI 经济回报（AI Agent 普及 + 企业采纳）低于预期，整个 AI 基础设施行业都会调整，Oracle 估值会被重估。

### 利空 3：估值过高 + Non-GAAP EPS 增速与收入增速相近 + 股价已大涨 + SBC 稀释 + 宏观风险

**依据**

- **2025 年股价表现**：
  - Jan 1 2025 开盘 $167
  - Oct 2025 高点 $345 (Oracle 历史最高)
  - Dec 31 2025 收盘 $221 (+33% YTD)
  - 波动性巨大：9 月财报日单日 +40%（史上最大单日涨幅）
- **Forward PE 估值**：
  - 2025/05 Forward PE ~20x（vs 市场 22x）
  - 2025/10 Forward PE ~45x（峰值）
  - 2025/12 Forward PE ~35x
  - 历史中位 PE ~18x
- **PEG 比率**：~2.5（PE / EPS Growth = 35 / 14）：超过健康 PEG < 1.5
- **Non-GAAP EPS 增速 +8% vs 收入 +8%**：未体现 EPS 相对收入的杠杆效应
- **股本稀释**：
  - 摊薄股数从 2,823M → 2,866M (+2%)
  - 股票回购 $6 亿（降至历史低位，vs FY24 $12 亿）
  - 优先保留现金给 AI Capex
- **Stock-Based Compensation $46.74 亿**（+18%）：
  - 占收入 8%（vs 行业均值 5-6%）
  - 稀释 EPS 影响持续
- **宏观风险**：
  - 美联储 2026 年降息 / 通胀 / 经济衰退
  - AI 监管（欧盟 AI Act + 美国联邦 AI 法规）
  - 数据隐私法规（GDPR + California Privacy Act）
  - 反垄断审查：Microsoft + Google 都被 FTC 调查，Oracle 可能跟进
- **FY26 Guidance 执行风险**：
  - +40%+ 云增长 = FY26 云收入 $340+ 亿
  - Cloud Infrastructure +70%+ = FY26 OCI $180+ 亿
  - 如果实际 +30% 而非 +40%，股价可能回调 20-30%

**解读**

**Oracle 2025 年股价上涨已 Front-Loaded 大部分 AI 转型故事，未来 1-2 年需要"持续兑现 + 超出预期"才能继续上涨**：① **Forward PE 35x 已接近 Nvidia 2023 年水平**，即使 FY26 EPS +25% 到 $7.50，股价停滞在 $220 = PE 29x 仍高于历史中位 18x 一倍；② **如果 FY26 云增速"仅"+35%（而非 +40%）**：市场会大幅下调预期，股价可能回调 20-30%；③ **SBC $46.74 亿（+18%）稀释明显**：如果继续按此速度增长，未来 EPS 增速会持续被拖累；④ **AI 泡沫风险**：如果 2026-2027 年 AI 企业采纳速度放缓，Oracle 的高估值会先被修正（从 35x → 25x，30% 下跌）；⑤ **减少回购**：FY25 回购仅 $6 亿 vs 历史常态 $100-150 亿/年，股东权益回报降低；⑥ **Larry Ellison 83 岁**（2025 年）：长期继任风险，如果 Ellison 退休或出现健康问题，Oracle 战略延续性存疑；⑦ **AI 监管**：欧盟 AI Act + 美国联邦 AI 法规可能影响 Oracle 数据处理 / LLM 部署业务；⑧ **宏观风险**：如果 2026 年出现经济衰退 + 企业 IT 预算削减，Oracle 应用 + 传统 License 业务会承压。

## 8. 总结判断

**一句话结论**

**Oracle FY2025 交出"收入 $574 亿 (+8%)、Cloud Services + License Support $440 亿 (+12%)、Cloud Infrastructure (OCI) +50%、RPO $1,380 亿 (+41% 历史最高)、Non-GAAP EPS $6.03 (+8%)、Capex $212 亿 (+209%)、FCF -$4 亿首次转负、FY26 云增速指引 +40%+ / OCI +70%+ / RPO +100%+"的转折点优秀业绩**，**标志着 Oracle 完成了从"传统 License 公司"到"AI + 云基础设施巨头"的历史性转型**。**FY26-FY28 核心看点是 AI 超级订单兑现（OpenAI 5 年 $300B / Stargate $500B）+ Cloud Infrastructure +70%+ + MultiCloud 生态（AWS/Azure/Google）+ Fusion Cloud ERP + NetSuite + Oracle Database 24ai + Larry Ellison AI 押注**；**核心风险是 Capex 激增 + FCF 持续负数 + 债务扩张 + AI 大客户集中 + 云巨头竞争（AWS/Azure/Google）+ GPU 供应 + 电力供应 + 估值过高（PE 35x）+ 股价 2025 年已涨 100%+**。

**核心看点**

1. **AI 超级订单周期 + RPO $1,380 亿 + OpenAI / Stargate $500B**：Oracle 已成为 AI 基础设施"第二极"（仅次于 AWS）
2. **Cloud + Database + ERP 多引擎 + MultiCloud 独特生态**：AWS/Azure/Google 都部署 Oracle Database，唯一全栈企业软件 + 云公司
3. **Larry Ellison 创始人领导 + 股息增长 + 股价 2025 +100%+ 估值重估**：AI 转型已 Price In 但增长故事仍延续

**核心风险**

1. **Capex $212 亿 + FCF -$4 亿 + 债务 $926 亿 + AI 投资回报不确定性**：烧钱建设的短期压力与长期回报博弈
2. **OpenAI / Stargate 集中风险 + 云巨头竞争 + GPU 供应 + 电力瓶颈**：AI 故事的外部风险
3. **Forward PE 35x 高估值 + SBC $47 亿稀释 + Non-GAAP EPS +8% ≈ 收入 +8%**：股价已 Front-Load AI 预期，执行压力大
