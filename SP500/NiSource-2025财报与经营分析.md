# NiSource Inc. (NI) 2025年财报整理

> 报告期：**FY2025 全年（止 2025/12/31）** —— NiSource 财年与自然年一致
> 数据来源：NiSource Inc. Form 10-K for Fiscal Year Ended December 31, 2025（SEC 2026/02/11 提交，Acc-no 0001111711-26-000027）、Q4 & FY2025 Earnings Press Release（2026/02/11 发布，Exhibit 99.1，Acc-no 0001111711-26-000026）
> 货币：美元（USD），除非特别说明单位为百万美元（millions）。所有指标默认 GAAP 口径；**Adjusted Net Income / Adjusted EPS / Base Plan Adjusted EPS / Consolidated Adjusted EPS / FFO/Debt** 为公司定义的 Non-GAAP 指标（剔除天气对经营收入影响 + 优先股赎回溢价 + 其他一次性项目）。**自 2026 年起**公司将新增 **"Base Plan Adjusted EPS" 和 "Consolidated Adjusted EPS"** 两个口径，前者剔除数据中心业务影响以体现基础业务增长。
> **重大事项**：FY25 **与 Amazon 签署里程碑式的数据中心供电合同（Contract Assets ~$6-7B 投资至 2032）+ 返还超过 $10 亿电费节约给印第安纳州零售客户**；**GAAP Net Income +25.7%、Adj EPS +8.6% 超指引上限**；**FY26-2030 资本计划提高至 $28.0B**（含 $7.0B 数据中心专项）；**FY26-2033 Consolidated Adjusted EPS CAGR 指引 8-9%** + **Rate Base CAGR 9-11%**。公司由 CEO Lloyd Yates 领导，总部位于印第安纳州 Merrillville。

## 1. 业绩总结

### 1.1 收入与利润表现

- **全年营业收入 $6,642M（同比 +21.8%，+$1,187M）创历史新高** —— 主要由 Columbia Operations +$627M（+23.1%）+ NIPSCO Operations ~$560M 增长驱动，背后是 **rate case 胜诉 + 电力需求增长 + 基础费率上调**。
- **GAAP 营业利润 $1,835M（+26.1%，+$379.8M）**；**Consolidated 净利润 Available to Common $929.5M（+25.7%）**，**GAAP Diluted EPS $1.95（vs FY24 $1.62）**。
- **Non-GAAP Adjusted Net Income $905.2M（+13.3%）**，**Adj EPS $1.90（+8.6%，超 $1.85-1.89 指引上限）** —— 剔除 FY24 $60.4M "好天气" 利好 + $14.0M 优先股赎回溢价 + FY25 -$32.6M "暖冬不利" 后更能体现运营真实增长。
- **FFO/Debt 比率 16.1%**（超 14-16% 指引上限），资产负债表质量持续改善。

### 1.2 增长与拖累因素

- **主要增长驱动**：
  1. **Columbia Operations 营业收入 +23.1%（+$627M）**：主要来自 Ohio、PA、VA、KY、MD 五州的**住宅费率上调**（rate case 获批）+ 系统现代化回收；
  2. **NIPSCO Electric 需求增长**：**16,538.8 GWh 电力销售**（FY25），受**Amazon 数据中心合同**预热 + 印第安纳州工业用电回升驱动；
  3. **基础费率调整**（RNG + ADIT）定期回收 + **TDSIC/CSIA 系统改善跟踪条款**自动回收资本投入；
  4. **关闭煤电 → 风能 / 太阳能 / 电池储能转型**的成本节省（NIPSCO Your Energy, Your Future 计划）；
  5. **Capital Expenditures $3,721M（NIPSCO $2,509M + Columbia $1,213M）**的 rate base 增长带来的回收；
  6. **数据中心合同**预期的增量收入（虽然 2025 主要是筹备阶段）。
- **主要拖累因素**：
  1. **FY25 "较常年天气偏冷 / 暖" 不利 -$32.6M**（vs FY24 +$60.4M 正贡献），对比正常天气口径实际 Adj Net Income 增长更加稳健；
  2. **摊销稀释股本 474.5M（+4.1%，FY24 456.0M）**：公司为资助数据中心 / 系统现代化 $28B 资本计划**发行普通股** → Adj EPS $1.90 增长 +8.6% 低于 Adj Net Income 增长 +13.3%；
  3. **利率上升环境**：长期债务利息支出 + 净利息支出高于 FY24；
  4. **MISO 容量认定 / 电力市场法规变化风险**；
  5. **与 Amazon 数据中心合同相关的 FY25 前期开发成本** + 法律 / 咨询费用。

### 1.3 趋势判断

- **未来主要利好方向**：
  1. **FY26 Consolidated Adj EPS 指引 $2.02-$2.07（+7.4%-8.9%）** + **Base Plan Adj EPS 新口径公开披露**（剔除数据中心影响后看"基础业务"增长）；
  2. **FY26-2033 CAGR 8-9% Adj EPS + 9-11% Rate Base + 6-8% Base Plan Adj EPS**（**7 年可预测增长路径**，公用事业中较强）；
  3. **FY26-2030 $28B 资本计划**（vs 前计划 $19-20B，大幅上调）—— 含 **$7B 专项数据中心基础设施 + $21B 基础业务 + $2B 上行机会**；
  4. **Amazon 数据中心合同**（Contract Assets 至 2032 约 $6-7B 投资）+ **第二 / 第三个潜在数据中心客户**（管理层已暗示 pipeline 活跃）；
  5. **天然气分销系统现代化（主干管道更换 + 漏气检测 + 测量技术）** 持续获批回收；
  6. **风能 / 太阳能 / 电池储能转型完成后** —— NIPSCO 清洁能源组合 + Your Energy, Your Future 计划后期降低长期燃料成本；
  7. **股息增长 +7.1%（2026）+ 投资级信用评级（Moody's Baa1、S&P BBB+）+ 目标 TSR 11-12%**。
- **未来主要压力方向**：
  1. **Amazon 数据中心合同执行风险**：建设进度（2027-2032 分阶段）+ 资本回收不达预期 + 客户履约风险（虽然 Amazon 信用良好）；
  2. **融资压力**：$28B 资本计划需要**持续股权 + 债务融资**（股东稀释 / 杠杆增加风险）；
  3. **MISO 容量认定变化**（MISO accreditation of capacity resources）：可能影响 NIPSCO 电力容量收入；
  4. **利率风险**：公用事业行业对利率高度敏感，10Y Treasury 5% 以上将压制 P/E；
  5. **监管风险**：印第安纳、Ohio、PA、VA、KY、MD 六州 UTC / PUC 费率审批节奏与批准率；
  6. **环保 / 气候成本**：NIPSCO 净零 2040 目标（Scope 1+2）+ 煤电关闭成本（尚未完全结束的煤电转型期）。
- **整体趋势判断**：NiSource 已从传统"稳健配息公用事业"进化为"**AI 数据中心资本密集投入 + Scaled 长期增长**"典型案例 —— **Amazon 合同** + **$28B 资本计划** + **8-9% CAGR** 让其成为**公用事业板块中少数能匹配 AI 主题但仍享公用事业防御属性**的标的。短期催化包括 FY26 指引执行 + 第二个数据中心合同公告 + Base Plan Adj EPS 新口径披露；**长期关键 = Amazon 建设进度 + 融资条款 + MISO 电力市场规则**。

## 2. 主营业务

### 2.1 公司简介

**NiSource Inc. (NI)** 是**全美最大的完全受监管（fully-regulated）公用事业公司之一**，总部位于印第安纳州 Merrillville，服务 **6 个州（Indiana, Ohio, Pennsylvania, Virginia, Kentucky, Maryland）** 共 **约 330 万天然气客户 + 50 万电力客户**（合计服务约 380 万客户），7,700 名员工。公司通过两大品牌运营：**Columbia Gas**（5 个州天然气分销业务，2.43M 客户）+ **NIPSCO（Northern Indiana Public Service Company）**（印第安纳州天然气 877,875 + 电力 496,534 客户，双能源公用事业）。FY25 总营收 $6.6B，Consolidated Operating Income $1.84B，**是当前美国最快成长的大型公用事业公司之一**（FY26-2033 EPS CAGR 指引 8-9%）。NiSource 2004 年上市，**Dow Jones Sustainability - North America Index** 成员，2040 年 Net Zero（Scope 1+2）目标。

### 2.2 主营业务构成

NiSource 划分为 **2 个 reportable segments**：

1. **Columbia Operations（Columbia Gas 天然气分销，营收占比 50.3%，营业利润占比 48.8%）**：
   - **5 州天然气分销业务**：Ohio、Pennsylvania、Virginia、Kentucky、Maryland；
   - **2.43 百万客户**（住宅 + 商业 + 工业）；
   - 业务覆盖**管道系统维护 + 更换 + 扩展 + 智能表计（AMI）+ 测量 + 漏气检测（ALDR Advanced Leak Detection & Repair）**；
   - FY25 营业收入 $3,343M（+23.1%，+$627M），营业利润 $895.1M（+22.9%）；
   - **监管回收机制**：IRP（Infrastructure Replacement Program）、CEP（Capital Expenditure Program）、费率调整 tracker。

2. **NIPSCO Operations（NIPSCO 天然气 + 电力双能源，营收占比 49.8%，营业利润占比 51.1%）**：
   - **印第安纳州独立运营 NIPSCO** —— **天然气分销 877,875 户 + 电力分销 / 发电 496,534 户**；
   - **电力端**：16,538.8 GWh 发电 / 销售（FY25），包括**煤电（关闭中）+ 风能 + 太阳能 + 电池储能 + 天然气发电**（Your Energy, Your Future 计划，2028 完成煤电退役）；
   - **天然气端**：印第安纳北部 + 其他区域分销；
   - FY25 营业收入 $3,308M（其中 Gas $1,084M + Electric 约 $2,224M），营业利润 $938.1M；
   - **监管机制**：TDSIC / CSIA（Transmission, Distribution, Storage Improvement Charge / Clean Energy Improvement Charge）—— 资本投入自动 tracker 回收；
   - **2025 Amazon 数据中心合同核心对接方**（合同标的 Contract Assets 约 $6-7B 至 2032）。

### 2.3 核心收入来源

- **按业务类型（FY25）**：**Columbia Gas 分销 50.3% + NIPSCO Gas & Electric 49.8%**；
- **按客户类别**（Columbia）：**住宅 68.3%（$2,284M）+ 商业 23.0%（$768M）+ 工业 5.1%（$168.8M）+ 其他 3.6%**；
- **按地域**：**Indiana 49.8%（NIPSCO） + Ohio / PA / VA / KY / MD 50.3%**（Columbia）；
- **按能源类型**：**天然气（Columbia 全部 + NIPSCO 约 30%）~65% + 电力（NIPSCO 约 70%）~35%**；
- **未来新收入**：**数据中心专用电力合同（Contract Assets）—— 预计 2027-2032 分阶段贡献**（FY25 尚未开始贡献运营）。

**资产负债率变化**

资产负债率保持 **70-73% 稳定区间**。公用事业行业特有的**"高杠杆、稳定现金流、受监管投资回报（ROE ~9.5-10%）"**模式使得高负债本身是商业模式核心。关键监控指标是 **FFO/Debt（16.1%，健康）**、**Debt/Rate Base（约 58-62%，合理）**。**FY26-2030 的 $28B 资本计划需要约 $14-16B 新债发行 + $6-8B 股权融资**，因此资产负债率预计略有上升至 73-75%，但由于 Rate Base 同步增长（9-11% CAGR），FFO/Debt 预计保持 14-16% 指引区间。

**偿债压力判断**

NiSource 偿债压力**中等但受控**：
- **投资级评级**：穆迪 Baa1、S&P BBB+、Fitch BBB+，**公用事业中游水平**；
- **FFO/Debt 16.1%** 高于投资级门槛（公用事业通常 14%+）；
- **Amazon 合同现金流增量**有望支持未来 FFO 增长；
- **主要压力**：$28B 资本计划的融资需求 —— 如果利率大幅上涨（10Y 超 5%），债务成本上升可能侵蚀 Adj EPS 20-40 bp；
- **短期债务滚动 / 再融资**：常态化管理，公用事业常有 commercial paper + revolver 备用；
- **股权融资稀释**：为避免评级下调，每年可能发行 $400-700M 新股 → 稀释率约 0.7-1.5% 年；
- **无重大短期债务到期**（一般至 2027-2030 梯度分布）。

**商誉风险判断**

商誉约 **$1,690M**（占总资产 5.0%），主要来自 Columbia Gas 历史收购（2000 年 NiSource 收购 Columbia Energy Group $10B 时形成）。**FY25 无减值**，减值风险**低**，因：
1. 受监管业务回报确定（ROE 9.5-10%）；
2. Rate Base 持续增长，现金流稳定；
3. 两大分部均持续盈利且营收增长。

## 7. 未来可能的利空

### 利空 1：Amazon 数据中心合同执行风险 + 融资规模巨大 + 股权稀释

**依据**

- **Amazon 合同 Contract Assets $6-7B** 建设周期长（2025-2032），面临：
  - **建设延期风险**：发电 / 输电 / 变电站项目 6-9 年建设周期，任何环节延期都可能推迟收入确认；
  - **成本超支风险**：通胀环境下 EPC（工程 / 采购 / 建设）合同可能溢出 10-20%；
  - **Amazon 提前终止 / 减少容量承诺**：虽然信用良好，但如果 AI 热潮退潮或 Amazon 战略变化，合同可能变更；
- **$28B 5 年资本计划的融资需求**：
  - 假设 55-60% 债务、40-45% 股权 → **需要约 $11-13B 新股权融资 + $16-18B 新债**；
  - FY25 已发行 ~$600M 新股 + 债务滚动；
  - **每年稀释约 1-1.5% 股本** = **对 Adj EPS 增长造成 50-80bp 年化压力**；
- **投资级评级维护**：
  - 穆迪 Baa1 / S&P BBB+，如果 FFO/Debt 低于 14% 可能降级；
  - **管理层在 forward-looking statements 中特别强调"maintain our investment grade credit ratings as we finance and pursue our data center strategy"** —— 暗示**评级压力真实存在**；
- **MISO 容量认定变化**（前瞻性声明明确提及）：
  - MISO 可能调整容量资源认定方法，影响 NIPSCO 电力业务收入。

**解读**

Amazon 数据中心合同是**"双刃剑"**：
1. **情景 1（乐观）**：Amazon 合同按时完成 + 第二 / 第三客户签约 → **Adj EPS CAGR 10-12% + Rate Base CAGR 12-14%** → **TSR 13-15%**；
2. **情景 2（中性）**：Amazon 按时完成但无新客户 → **Adj EPS CAGR 8-9% + 稳定估值倍数**（当前指引情景）；
3. **情景 3（悲观）**：Amazon 合同建设延期 / 成本超支 / 提前终止 + 融资成本上升 → **Adj EPS CAGR 6-7% + 评级下调压力 + 估值压缩**；
4. **关键监控指标**：
   - 每季度 Contract Assets 建设进度披露；
   - FFO/Debt 是否维持 14-16%；
   - 股权发行节奏（避免过度稀释）；
   - MISO 容量市场规则变化。

### 利空 2：利率敏感度 + 公用事业板块估值压力 + 气候 / 监管 / 环保风险

**依据**

- **公用事业对利率高度敏感**：
  - 主要投资逻辑是"债券替代品"（Bond Proxy），10Y Treasury 每上升 100bp，公用事业 P/E 通常下降 2-3x；
  - 当前 10Y ~ 4-4.5%，如果 2026 年升至 5%+（通胀反复 + 财政赤字），**公用事业整体估值可能压缩 10-15%**；
- **NiSource 历史估值 Range**：P/E 18-22x；当前 2026 指引 $2.02-2.07 → P/E ~16-18x（相对合理，但如果利率升至 5%，可能降至 15x）；
- **煤电退役成本**：
  - NIPSCO 2028 完成煤电退役（Schahfer、Michigan City）；
  - 退役成本 / 资产减值可能额外 $200-500M；
  - 监管回收过程中可能面临反对；
- **气候 / 天气波动风险**：FY25 -$32.6M 天气损失即是案例；全球变暖可能使暖冬更常见 → 天然气消费下降 → 对营收造成持续轻微拖累；
- **环保监管升级**：
  - EPA 新规（甲烷排放、空气质量、水质）增加合规成本；
  - 部分州"脱气"（de-gas）激进议程（如 Massachusetts、California 路线）虽目前不影响 NiSource 六州，但长期主题存在；
  - 甲烷排放披露 / 减排要求；
- **Indiana Utility Regulatory Commission (IURC) 监管环境变化**：
  - IURC 监管透明度高但 **rate case 周期延长 / ROE 批准率下降**风险；
  - 如果 Amazon 合同招致其他客户反对，可能政治压力；
- **Columbia Gas Massachusetts 2018 爆炸事件**（已由分拆出售解决，但公司 legacy 风险管理仍被监管机构关注）。

**解读**

NiSource 作为**受监管公用事业**，在**利率 / 监管 / 气候**三维度面临不可控风险：
1. **利率每升 100bp → P/E 压缩 2-3x → 股价压力 -10-15%**；
2. **Indiana IURC 的数据中心合同审批成功是关键**：若未来第二 / 第三合同审批失败 → Adj EPS CAGR 可能降至 6-7%（失去增长溢价）；
3. **煤电退役 + 环保合规**短期 $200-500M 不确定性；
4. **天气风险**：FY25 -$32.6M 实际天气损失 → FY26 持续风险 ±$30-60M 波动；
5. **EPA 甲烷规则**：Columbia Gas 管道甲烷泄漏监控 / 修复成本增加。

### 利空 3：高 CapEx 扩张期 FCF 为负 + 股息支付率较高 + 稀释压力

**依据**

- **FY25 CapEx $3,721M** vs OCF 约 $2,200-2,400M → **FCF 约 -$1,400-1,500M**；
- **FY26-2030 5 年 CapEx $28B** vs OCF 约 $13-15B → **累计 FCF 缺口约 -$13-15B**；
- **每年股权融资 $1-1.5B + 债务融资 $2-3B + 股息 ~$550M 支付**；
- **股息派息率**：FY26 股息 ~$1.14/股（+7.1%），Adj EPS $2.02-2.07 → **派息率 55-57%**（在 55-65% 目标内，但已处中枢）；
- **股本摊薄 FY25 +4.1%**（474.5M vs 456.0M），**FY26-2030 累计可能摊薄 15-20%**；
- **长期债务 $14-15B 中未来 5 年到期规模**：
  - 平均每年需要滚动 $1.5-2B 债务；
  - 如果当前 10Y 5% + 信用利差 1.5% = 新发债务成本 6.5%（vs 老债务 4-5%），年化增加利息支出 $30-40M。

**解读**

NiSource 处于**"高投入 / 高增长 / 低 FCF"阶段**：
1. **FCF 持续为负**是公用事业 CapEx 扩张期正常现象，但依赖持续融资；
2. **股息 + 资本计划的"资本支付"并举**增加融资压力：FY26 股息 ~$570M + CapEx $5.6-6.0B = 约 $6.2B 资本支出，OCF 约 $2.5-2.8B → 需融资约 $3.5-3.7B；
3. **稀释风险**：每年发行 3-4% 新股，Adj EPS 增速 8-9% 中已包含稀释影响；
4. **派息率接近中枢**：如果 FY26 实际 Adj EPS 低于指引中点 $2.045，派息率将达 56-57%，接近 60% 上限；未来股息增速可能需要从 7.1% 下调至 4-5%；
5. **股东回购缺位**：NiSource 基本不回购股票（公用事业典型），不提供 buyback 防御垫；
6. **与 AI / 科技股竞争资金**：在 AI 热潮中，公用事业作为"防御品种"可能持续吸引避险资金流入，但在 risk-on 时可能表现落后。

## 8. 总结判断

**一句话结论**

NiSource Inc.（NI）是**美国最大的受监管公用事业公司之一**（330 万天然气 + 50 万电力客户，6 个州，Columbia Gas + NIPSCO 双品牌），FY25 营收 $6.64B（+21.8%）+ GAAP Net Income $929.5M（+25.7%）+ **Adj EPS $1.90（+8.6%，超指引上限）** + FFO/Debt 16.1% 超指引；**Amazon 数据中心合同（$6-7B Contract Assets 至 2032）+ $28B 五年资本计划（含 $7B 数据中心）+ FY26-2033 Adj EPS CAGR 8-9% + Rate Base CAGR 9-11%** 构成**公用事业板块中最稀缺的"AI 电力基础设施 + 稳健配息"双主题标的**；适合**寻求"AI 算力红利 + 公用事业防御 + 长期可视性"的混合型配置投资者**（核心风险：Amazon 合同执行 + 融资压力 + 利率敏感 + 稀释）。

**核心看点**

1. **Amazon 数据中心合同 + AI 算力基础设施红利**：NIPSCO 为 Amazon AWS 建设 $6-7B Contract Assets 至 2032，是**美国公用事业中最具标志性的 AI 算力合同之一**；管理层明示"pipeline of opportunities remains robust"，**第二 / 第三个数据中心客户签约可能将 Adj EPS CAGR 从 8-9% 上调至 10-12%**；FY26 新披露 **Base Plan / Consolidated Adj EPS 双口径**可能大幅提升数据中心业务估值可见性。
2. **FY26-2033 长期增长可视性 + 领先同业的 Rate Base CAGR 9-11%**：$28B 5 年资本计划（vs 前次 $19-20B 上调 +40%）+ 监管回收机制成熟（TDSIC / CSIA / CEP 等 tracker）+ FY25 Adj EPS $1.90 超指引 + FFO/Debt 16.1% 超指引均已证明管理层执行力；长达 7 年的 Adj EPS CAGR 8-9% 增长路径在公用事业中属 top 5。
3. **稳健配息 + 目标 TSR 11-12% + 投资级评级（Baa1/BBB+）**：FY26 股息 +7.1% 至约 $1.14/股；55-65% 派息率健康区间；股息 + 资本增值 = 长期 TSR 目标 11-12%，在公用事业中属行业前列；ESG 评级高（Dow Jones Sustainability Index + Net Zero 2040）。

**核心风险**

1. **Amazon 数据中心合同执行风险 + $28B 资本计划融资压力 + 股权稀释**：Contract Assets 建设 6-9 年周期面临延期 / 成本超支 / MISO 容量规则变化风险；**融资需 $11-13B 新股权 + $16-18B 新债**，每年稀释 3-4% 股本；投资级评级维护压力（FFO/Debt 需保持 14%+）。
2. **利率敏感度 + 公用事业板块估值压力**：10Y Treasury 每升 100bp → P/E 压缩 2-3x → 股价压力 -10-15%；如果 2026 利率升至 5%+，**NiSource 相对 AI / 成长股可能阶段性跑输**；公用事业"债券替代品"属性在高利率环境下不利。
3. **监管 / 气候 / 天气风险**：Indiana IURC、Ohio PUC 等 rate case 审批节奏与批准率；煤电退役成本 $200-500M 不确定；天气波动（FY25 -$32.6M）+ EPA 甲烷 / 环保新规增加合规成本；"脱气"激进州政策长期风险（不直接影响六州但威胁天然气行业长期前景）。
