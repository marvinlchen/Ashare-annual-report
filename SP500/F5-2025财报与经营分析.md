## 1. 业绩总结

### 1.1 收入与利润表现

- **全年营业收入表现**：**总营收 $3,088.1M**（FY24 $2,816.1M，**同比 +9.7%**，公司对外传达为 **10%**，10 年来最强年度增长）——**Product Revenue $1,508.6M**（同比 +18.5%）**Software $803M（+9%）+ Systems $706M（+31%）**；**Services Revenue $1,579.4M**（同比 +2.3%）；**Q4 单季 $810M（同比 +8.5%）+ 产品端 +16%（Systems +42%，Software +0.3%）**。
- **全年营业利润表现**：**GAAP Operating Profit $765.9M**（FY24 $658.6M，**同比 +16.3%**，GAAP Operating Margin 24.8% vs 23.4%，**+140bp**）；**Non-GAAP Operating Profit $1,086.4M（同比 +14.9%，历史首次突破 $1B）**，Non-GAAP Operating Margin 35.2% vs 33.6%（**+160bp**）。营业杠杆显著（收入 +9.7% / 运营利润 +16.3%）。
- **全年归母净利润表现**：**GAAP Net Income $692.4M**（FY24 $566.8M，**同比 +22.2%**）；**GAAP EPS $11.80（+23.6%）**；**Non-GAAP Net Income $928M（+16.9%）**；**Non-GAAP EPS $15.81（+18.3%，对外沟通+18%）**——EPS 增长快于净利润（回购减少股本）。

### 1.2 增长与拖累因素

- **主要增长驱动**：**(1) Systems 硬件爆发（FY25 +31%，Q4 +42%）** —— 客户大规模数据中心重投资周期 + 混合多云架构投入 + AI 基础设施推动应用交付需求；**(2) Software Revenue $803M（+9%）** —— F5 BIG-IP 软件订阅 + F5 Distributed Cloud Services（SaaS 订阅）+ F5 NGINX Plus 持续拓展；**(3) 毛利率改善**（GAAP Gross Margin 81.4% vs 80.2%，+120bp），高价系统产品组合改善 + 产品定价提升；**(4) 费用控制** —— 销售费用几乎持平（$860M +3.4%）、研发费用 $540M（+10.1%，对齐营收），G&A $322M（+19.9%，因一次性项目）；**(5) 新产品 ADSP 平台** —— 2025 年推出 F5 Application Delivery and Security Platform，整合高性能流量管理 + API 安全 + 混合多云；**(6) CalypsoAI 收购**（2025/09 完成，$145.2M）—— 加强 AI Inference 安全能力。
- **主要拖累因素**：**(1) 网络安全事件（最大冲击）** —— 2025/10/15 披露 UNC5221 国家级黑客入侵超 12 个月，盗取部分 BIG-IP 源代码、未公开漏洞信息、部分客户配置数据；CISA 紧急指令要求联邦机构限期打补丁；**一次性"Cyber Incident Costs" Q4 $3.4M + 未来 FY26 持续增加专业服务和响应开支**；**(2) 销售周期扰动** —— 客户被迫先评估、修复环境，新项目 orders 可能递延；**(3) Services Revenue 仅 +2%** —— 服务收入增长放缓，反映客户基础数 vs 服务渗透未匹配；**(4) Q4 Software 增长仅 +0.3%** —— 软件订阅转型节奏仍有波动，部分客户合同递延；**(5) Restructuring Charges $25.5M**（FY25）+ Acquisition-related $11.5M，均较 FY24 明显增加（重组成本 +194%）；**(6) Al Higginson 退休 + 董事会治理架构过渡**——带来短期人事稳定性担忧。
- **增长或承压主要来自哪些业务、产品、区域或成本因素**：**Systems 硬件（+31%）为核心驱动**（客户数据中心重投资 + AI 基础设施）；**Software +9% 略低于预期但保持稳健**；**Services +2% 相对疲弱**（客户续保但支持和专业服务增长放缓）；**区域未在 ER 单独披露**（10-K 按 Americas/EMEA/APAC 三区报告，未披露具体金额）；**网络安全事件对 FY26 的 orders disruption 是短期核心压制因素**。

### 1.3 趋势判断

- **未来主要利好方向**：**(1) AI/混合多云驱动应用交付 + API 安全需求结构性加速**；**(2) ADSP 平台整合（BIG-IP + NGINX + Distributed Cloud + CalypsoAI）形成完整"Deliver & Secure Every App/API"能力**；**(3) FY25 $906M FCF + $1.34B 现金 + 无债务 Balance Sheet → 支持持续回购（$502M/年）+ 并购**；**(4) 新产品 F5 XOPS（近 900 客户）+ F5 Distributed Cloud（Top 1,000 客户中 26%）加速推进**。
- **未来主要压力方向**：**(1) 网络安全事件余波 → FY26 营收增长仅 0-4%（vs FY25 +10%）**；**(2) 客户可能从 F5 BIG-IP 环境迁移至云原生替代方案**（如 AWS ALB、Cloudflare、Kong、HashiCorp Consul）；**(3) Services 增速持续放缓**（+2%）—— 反映传统 Hardware/Services 业务模式面临 SaaS 转型压力；**(4) Non-GAAP 运营利润率可能下降 80-170bp**（FY26 指引 33.5-34.5% vs FY25 35.2%）；**(5) CISA ED 26-01 紧急指令的监管和法律负担 + 潜在集体诉讼**。
- **整体趋势判断**：**FY25 是 F5 历史最佳年度（10% 营收增长、18% EPS 增长、35.2% Operating Margin、首破 $1B），但 FY26 面临明显逆风 —— 安全事件导致的销售周期扰动为短期（H1 FY26 压力最大，H2 有望正常化）+ 核心资产（ADSP 平台、$2.0B Deferred Revenue、$1.3B 现金、强 FCF 机器）保持韧性。中长期看 AI + 混合多云 + API 安全仍是 F5 护城河；但需要 2-3 个季度兑现 FY26 恢复叙事。当前估值（Fwd P/E 约 18-20x）已定价大部分不确定性。**

## 2. 主营业务

### 2.1 公司简介

**F5, Inc.（NASDAQ: FFIV）** 是全球领先的 **应用交付与安全解决方案（Application Delivery & Security Solutions）** 供应商，总部位于美国华盛顿州西雅图，成立于 1996 年，核心使命为 **"Deliver and Secure Every App and API, Anywhere（在任何地方交付并保护每一个应用和 API）"**。公司提供 **BIG-IP 应用交付控制器 + NGINX 开源/商用网络负载均衡 + F5 Distributed Cloud Services SaaS**，服务 Fortune 500 企业、电信运营商、云服务商及政府机构。

### 2.2 主营业务构成

F5 的营收按产品类型可划分为三大分部（FY25 披露口径，未使用业务分部口径）：

1. **Systems（系统硬件）**：FY25 $706M（+31% Y/Y，占总营收 22.9%）—— **BIG-IP i-Series / r-Series 硬件设备**，用于企业数据中心的负载均衡、流量管理、SSL 加速、应用安全（WAF）、DDoS 防护；**Q4 单季 $186M（+42%）—— 最强季度**。客户大规模数据中心重投资 + AI 基础设施建设推动。高毛利硬件（~70% 毛利率）。
2. **Software（软件订阅）**：FY25 $803M（+9% Y/Y，占总营收 26.0%）—— **BIG-IP Next / BIG-IP 虚拟版 + NGINX Plus + NGINX App Protect + F5 Distributed Cloud Services（SaaS）**；以 **订阅（Subscription）和永久授权（Perpetual License）** 两种方式销售；**SaaS 和 Managed Services 贡献增长**。**F5 Distributed Cloud Services** 提供跨公有云、私有云、边缘的应用交付 + API 安全 + Bot 防护 + WAAP（Web App & API Protection）+ DDoS + CDN 整合。
3. **Global Services（全球服务）**：FY25 $1,579M（+2% Y/Y，占总营收 51.1%，**最大分部**）—— **技术支持、维护、专业服务、培训**，按订阅续费制；高毛利（~85%）；属于 Recurring Revenue 但增速较慢，反映硬件装机基数和维护周期。

**产品平台统一（2025 年核心战略）**：
- **F5 Application Delivery and Security Platform (ADSP)** —— 2025 年推出的旗舰平台，整合 BIG-IP + NGINX + F5 Distributed Cloud + 数据分析 + AI 推理安全（通过 CalypsoAI 收购），提供 **"One Platform for Every App, Every API, Anywhere"** 的完整方案。
- **F5 XOPS** —— 新一代运营/DevOps 功能，截至 FY25 Q4 **近 900 个企业客户已采用**；**Top 1,000 客户中 26% 使用 F5 Distributed Cloud Services**（客户采纳率同比大幅提升）。
- **主要客户类型**：Fortune 500 企业（金融服务、电信、医疗、科技、政府、能源）+ 云服务商（AWS、Azure、GCP 合作生态）+ 运营商（Verizon、AT&T、DoCoMo）。
- **销售渠道**：主要通过全球分销伙伴 Westcon（亚洲核心）、Exclusive Networks（EMEA）、Synnex/TD Synnex（北美），约 **50% 以上营收来自分销渠道**（10-K 未直接披露 FY25 占比，历史数据）。

### 2.3 核心收入来源

- **产品结构**：**Services $1.58B（51%）+ Product $1.51B（49%）**；其中 Product 内部 Software 53% / Systems 47%。
- **Recurring Revenue**：**Deferred Revenue $1.99B（短 $1.21B + 长 $0.79B，同比 +12.9%）**——反映订阅合同积压 + 未确认服务收入，Q4 FY25 Operating Cash Flow 大幅增长主因即 Deferred Revenue +$200M。订阅和订阅式服务占比持续提升。
- **地域分布**（10-K 按 Americas/EMEA/APAC 划分，未单独披露 FY25 分项金额，管理层表述）：Americas ~55-58%、EMEA ~25-27%、APAC ~15-18%（与历史水平一致）；**US 国内营收占比约 45-50%**。
- **客户集中度**：**无 >10% 的单一客户**（10-K 披露），但分销渠道集中（前三分销商占比约 40-45%）。
- **AI 基础设施驱动**：Locoh-Donou 管理层明确表示 **"AI-driven infrastructure 是 FY25 增长的核心驱动之一"**；AI 集群 + GPU 集群对数据中心流量管理、安全、负载均衡的需求激增。

### 4.3 营收分析

**主要增长来源**

1. **Systems 硬件（+31%）为第一增长引擎** —— FY25 增量 $167M（产品增量总 $236M 的 71%）；**客户数据中心大规模刷新周期 + AI 基础设施对 10/25/100G 高性能网络设备的需求爆发**；Q4 单季 +42% 是历史最强硬件增长；**BIG-IP r-Series 新产品周期** + **产品定价提升** 贡献主要增量。
2. **Software +9%** —— BIG-IP Software 订阅 + F5 Distributed Cloud SaaS + NGINX Plus 订阅合计贡献 $69M 增量；**近 900 客户采用 F5 XOPS** + **Top 1,000 客户中 26% 使用 F5 Distributed Cloud Services**。
3. **AI + 混合多云成为新增长叙事** —— 管理层强调"AI-driven infrastructure 驱动对应用交付 + API 安全的需求"；2025 年推出 **F5 Application Delivery and Security Platform (ADSP)** 整合 BIG-IP + NGINX + Distributed Cloud + CalypsoAI。
4. **客户装机基数扩大** —— 内生增长（非一次性并购驱动）是 FY25 核心叙事；10% 增长 vs FY24 的 4.4% 大幅加速。

**主要拖累因素**

1. **Services 仅 +2%** —— 反映传统 Hardware Support 续费增长放缓 + 客户转向 SaaS 订阅（部分从 Services 转入 Software 口径）；对整体增长拖累最大（51% 占比仅贡献 $36M 增量）。
2. **Q4 FY25 Software +0.3%** —— 反映订阅合同递延或部分客户因早期评估期暂停签约（Late-Q4 重叠网络安全事件披露窗口前）。
3. **网络安全事件（2025/10/15 披露，已在 FY26 影响）** —— UNC5221 黑客入侵被披露后，客户被迫暂停新项目评估、先修复 BIG-IP 环境，对 FY26 Q1-Q2 新签订单形成明显压制。
4. **区域平衡**：EMEA 和 APAC 增速（~3-5%）低于 Americas（~12-13%），反映欧洲经济疲软 + 亚太客户资本开支节奏放缓。

**结构变化**

- **收入结构更加均衡**：Product 占比从 45% 提升至 49%，Services 从 55% 下降至 51%，反映硬件回归 + 软件订阅渗透提升；长期看 Software + Services 作为 Recurring Revenue 合计占比 77%（vs FY24 81%），硬件周期性回归。
- **F5 Distributed Cloud Services 崛起**：SaaS 订阅占 Software 比重快速提升（具体金额 F5 未单独披露，但管理层明确表述"占 Software 增长的大部分"）。
- **Deferred Revenue +11.2%（$2.0B）**——显示订阅合同积压和 Recurring Revenue 可见性大幅提升，未来 12-24 个月营收可见性强。
- **平台化战略（ADSP）**——推动客户从"单产品"采购转向"平台订阅"，提升 ARPU 和客户黏性。

## 6. 未来可能的利好

### 利好 1：AI 基础设施 + 混合多云浪潮驱动核心业务结构性加速

**依据**

- FY25 Systems 硬件 **+31%（Q4 +42%）**，反映客户数据中心 + AI GPU 集群对 **高性能应用交付（10/25/100G 负载均衡、SSL 加速）+ API 安全 + 流量可观测** 的需求激增。
- 管理层明确表述：**"Market dynamics are moving in a direction where F5's solutions are more essential than ever. The accelerated adoption of hybrid multicloud architectures and AI-driven infrastructure is driving demand for advanced application delivery and security solutions"**（CEO Locoh-Donou，Q4 FY25 电话会）。
- 2025 年推出的 **F5 Application Delivery and Security Platform (ADSP)** 整合 BIG-IP + NGINX + Distributed Cloud + CalypsoAI（AI Inference 安全），成为完整"一平台交付和保护每个 App/API"的方案。
- **F5 XOPS 近 900 客户采用**，**Top 1,000 客户中 26% 使用 F5 Distributed Cloud Services**，显示平台化渗透快速提升。

**解读**

AI 负载（训练 + 推理）+ 混合多云部署对应用交付层的需求从"可选"变为"关键基础设施"。F5 作为行业龙头（WAAP 领域 IDC 认可的领导者）具有结构性受益。**长期看 AI 应用爆发期将为 F5 提供 3-5 年 CAGR 可能升至 8-12% 的潜力**（远高于 FY25 前 5 年平均 3-5% 的传统水平）。若 FY26 恢复正常后，FY27-FY30 增长有望回归中单-双位数。

### 利好 2：强现金流 + 零债务资产负债表支持持续回购和战略并购

**依据**

- **FY25 FCF $906M（+19%）**、**经营现金流 $950M（+20%）**、**现金及投资 $1,344M**、**零有息债务**。

- **Deferred Revenue $2.0B（+11%）**——提供未来营收可见性。

**解读**

在全行业充满估值泡沫（尤其 AI 软件公司）的背景下，F5 的**强 FCF + 零债务 + 持续回购 + 并购能力**形成独特资本配置优势。**FY26 即使营收仅 +0-4%（指引下限），FCF 仍可维持 $850-900M 级别，继续支持 $400-500M 回购 + 小型并购**。这使 F5 在周期性低谷期仍能为股东创造价值，护城河较强。

### 利好 3：网络安全事件或成为"转危为机"——强化平台护城河

**依据**

**解读**

短期（FY26 Q1-Q2）销售周期扰动 + 客户评估期是不可避免的，但中期看：**(1) F5 通过透明披露和积极响应已经最大化信任恢复**；**(2) 客户在修复周期内反而增加与 F5 的专业服务接触频次，有利于后期 SaaS + Platform 上量**；**(3) ADSP 定位进一步强化（"Deliver & Secure" 品牌心智）**。**H2 FY26 恢复叙事一旦验证，FY27 营收增速有望回归中单位数以上，Non-GAAP Operating Margin 有望回到 35%+**。这为当前低估（Fwd P/E ~18-20x）提供显著估值修复空间。

## 7. 未来可能的利空

### 利空 1：网络安全事件对 FY26 销售周期的持续影响（指引大幅下调）

**依据**

- **FY26 营收增长指引：0-4%（vs FY25 +10%，大幅放缓）**。
- **FY26 Non-GAAP 运营利润率：33.5-34.5%（vs FY25 35.2%，-80bp 至 -170bp）**。
- **FY26 Non-GAAP EPS：$14.50-15.50（vs FY25 $15.81，中位数 $15.00 意味着 EPS -5.1%）**。
- **Q1 FY26 营收：$730-780M（vs Q1 FY25 $740M，中位数 $755M 仅 +2%）**。
- 管理层明确：**"any demand impacts expected to be more pronounced in the first half, before normalizing in the second half"**。
- **Cyber Incident Costs FY25 仅 $3.4M，FY26 将大幅上升**（调查 + 修复 + 法律 + 保险免赔额 + 潜在集体诉讼和解）。

**解读**

FY26 是**明确的放缓年**，市场对增长预期需要重新校准。**最坏情况（营收 0% 增长 + Non-GAAP EPS $14.50）** 意味着 EPS -8%，是 F5 近 5 年最大 YoY 下滑。**主要风险节点**：
- **H1 FY26（2026/Q1-Q2）** 可能出现 orders 递延的"破窗效应"——如果多个大客户项目被暂停，营收可能跌入负区间。
- **客户流失风险**：部分客户可能趁机评估云原生替代（AWS Application Load Balancer、Cloudflare、Kong、HashiCorp Consul），小概率发生部分客户永久性迁移。
- **集体诉讼和监管调查**：股东集体诉讼（Hagens Berman 等律所已启动）+ SEC 调查可能带来数千万至上亿美元的长期法律成本。

### 利空 2：云原生替代方案 + API 网关新兴竞争者的结构性威胁

**依据**

- **F5 BIG-IP 主要竞争对手**：
  - **云原生负载均衡**：AWS ALB/ELB、Azure Application Gateway、Google Cloud Load Balancer —— 客户"Lift & Shift"迁移至云时，往往同时替换 F5。
  - **开源 + SaaS 新势力**：HAProxy、Kong、Istio、Envoy、Cloudflare、Fastly、Akamai。
  - **云原生应用安全**：Cloudflare（WAF + DDoS + Bot）、Imperva、Akamai Prolexic、Wiz、Snyk。
  - **API 网关**：Kong、Apigee（Google）、AWS API Gateway、MuleSoft（Salesforce）。
- **Services 仅 +2% 增长** —— 反映传统 Hardware Support 装机基数开始承压。
- **10-K 风险因素**：**"increased sales discounts" + "new product and marketing initiatives by competitors"** 被列为主要竞争风险。

**解读**

F5 的核心 BIG-IP 硬件业务从长期看将面临两个结构性压力：**(1) 企业工作负载上云后，硬件 ADC 需求下降；(2) 云原生替代方案在价格、简化部署、集成生态方面具有优势**。**F5 的防御策略（ADSP 平台化 + Software 订阅 + F5 Distributed Cloud SaaS）虽然正确，但转型节奏较慢（Software 仅 26% 营收）**。**中期威胁窗口 3-5 年**：如果 Software + SaaS 增速长期停留在 +5-10%，无法弥补 Services/Systems 的结构性放缓，整体营收 CAGR 可能被压制在 2-4%。

### 利空 3：商誉高企 + FY26 利润率下降 + 监管和法律成本上行

**依据**

- **商誉 $2,444M（占总资产 38.7%、占股东权益 68.0%）**——主要来自历史收购（NGINX $670M、Volterra $500M、Shape Security $400M+、Threat Stack、Lilac Cloud、CalypsoAI $145M 等）。FY25 未计提减值，但监管环境恶化下需持续关注。
- **FY26 Non-GAAP 运营利润率指引 33.5-34.5%**（vs FY25 35.2%）——**-80 至 -170bp 压缩**，反映：
  - **Cyber Incident 相关费用上升**（法律 + 专业服务 + 免赔额 + 安全投入）；
  - **销售费用难以同比例下降**（保留销售团队 + 加大 customer success 投入）；
  - **R&D 投入维持 +10%**（安全能力 + AI 平台）。
- **集体诉讼风险**：截至 2025/11，**Hagens Berman 律师事务所等多家律所已发起股东集体诉讼邀请**，质疑 F5 2024-2025 年信息披露充分性。可能带来数千万至上亿美元的法律和解 + 保险免赔额。
- **SEC / DOJ / CISA 可能持续调查**：F5 作为关键基础设施供应商，受监管关注度极高。

**解读**

FY26-FY27 是**财务逆风 + 法律成本 + 商誉减值测试三重压力年**：
- **如果 FY26 销售周期扰动超出指引**（营收实际 -3-5%），**Volterra + Shape Security 两大历史收购可能触发商誉减值测试** —— 潜在一次性减值 $200-500M；
- **Non-GAAP 利润率可能跌破 33%**，若 Non-GAAP EPS 跌破 $14，股价存在进一步压力；
- **集体诉讼和解 + 保险免赔额 + 强化合规成本**可能累计 **$50-150M** 影响 FY26-FY27 净利润。
- **整体利润风险**：Non-GAAP Net Income 可能出现连续 2 年下滑（FY24 → FY25 上升，FY25 → FY26 下降 6-10%）。

## 8. 总结判断

**一句话结论**

2025 年 F5 创下历史最佳年度（营收 +10%、Non-GAAP EPS +18%、Non-GAAP 运营利润率首次突破 35%、Non-GAAP 运营利润首破 $1B、FCF $906M），但 10 月披露的国家级黑客入侵 BIG-IP 源代码盗取事件将 FY26 推入明显的放缓周期（营收增长指引 0-4%、EPS 潜在 -5 到 -8%）；**中期看 AI + 混合多云 + API 安全叙事依然完整，零债务 + 强 FCF 提供深度防御，但需要 2-3 季度兑现 FY26 恢复叙事**。当前估值（Fwd P/E ~18-20x）已 price-in 大部分不确定性，属于"风险定价充分的防御性品种"。

**核心看点**

1. **AI + 混合多云 + API 安全三重叙事长期完整** —— Systems +31%（AI 基础设施驱动）、Software +9%（SaaS/订阅加速）、ADSP 平台（Deliver & Secure 一体化）构筑护城河；**Top 1,000 客户中 26% 已用 F5 Distributed Cloud Services + 近 900 客户用 F5 XOPS**，平台渗透率快速提升。
2. **堡垒式资产负债表 + 持续回购 + 并购能力** —— $1,344M 现金 + 零有息债务 + $906M FCF + $502M 年度回购（3 年累计减少股本 2.7%）+ CalypsoAI $145M 收购强化 AI 安全。FY26 即使营收 0-4%，FCF 仍可维持 $850-900M。
3. **FY26 H2 恢复叙事兑现 → 估值修复空间可观** —— 网络安全事件为一次性冲击，客户修复周期约 6-9 个月后新订单预计恢复；**Non-GAAP Operating Margin 有望在 FY27 重回 35%+**；**Fwd P/E ~18-20x 历史性低位**（过去 5 年均值 22-25x），估值修复潜力 20-30%。

**核心风险**

1. **网络安全事件对 FY26 销售周期的持续扰动风险** —— 最坏情况（营收 0% + Non-GAAP EPS $14.50）意味着 EPS 同比 -8%，是近 5 年最大 YoY 下滑；集体诉讼、SEC 调查、CISA 持续监管可能带来 $50-150M 法律 + 专业服务成本。
2. **云原生替代和 API 网关新兴竞争的长期威胁** —— 客户"Lift & Shift"上云时可能同时替换 BIG-IP（改用 AWS ALB、Cloudflare、Kong、Istio）；Services 仅 +2% 增长反映装机基数开始承压；**3-5 年转型窗口**：Software + SaaS 若长期停在 +5-10%，整体 CAGR 将被压至 2-4%。
3. **商誉减值 + FY26 利润率压缩风险** —— 商誉 $2.44B（占总资产 39%，主要为 Volterra + Shape Security + NGINX 历史收购）若业务放缓触发减值可一次性冲击 $200-500M；FY26 Non-GAAP 运营利润率指引 33.5-34.5%（-80 至 -170bp）；**股东权益 $3.6B 中 68% 为商誉**，下行保护较弱。
