# Danaher Corporation (DHR) 2025年财报整理

> 报告期：FY2025 全年（12 个月，止 2025/12/31）—— 财年与日历年一致
> 申报币种：美元；会计准则：U.S. GAAP，并披露 Non-GAAP（Adjusted EPS / Core Revenue Growth / Adjusted Op Margin）
> 数据来源：Danaher Q4 & Full-Year 2025 Earnings Release（2026/01/28 发布）+ 8-K + 10-K
> **注意**：所有结果仅反映持续经营业务（Continuing Operations），**Veralto 于 2023/09 分拆独立**（工业水处理 + 产品识别业务）已从持续经营中剔除
> FY25 期间主要战略事件：**Bioprocessing 业务持续复苏**：Biotechnology +6.5% core，Cytiva 生物制药订单加速、**Diagnostics 改善动量**：Cepheid + Beckman Coulter + Leica + Radiometer + Mammotome 四大品牌强势、**2025 Capital Allocation**：$31 亿股票回购（回购 1,200 万股）+ $9 亿股息 + $0 并购 - 等待合适标的、**商誉 / 商标减值 $562M**：Life Sciences trade name $432M + Biotechnology 工厂 $101M + Diagnostics $15M + Life Sciences 工厂 $14M、**新品发布**：Cepheid Xpert GI Panel（11 种胃肠道病原体 PCR 检测）+ Cytiva Xcellerex X-platform 500L / 2,000L 生物反应器 + SCIEX ZenoTOF 8600 高分辨质谱仪 + Beckman Coulter DxI 9000 神经退化疾病检测、**60,000 全球员工**、**FY26 Guidance**：Core Revenue +3-6% + Adjusted EPS $8.35-$8.50（+7-9%）
> FY25 特别会计事项：**GAAP 净利润因减值 $562M 受到压制**：减值 $562M / $420M 税后 / EPS $0.78、**Non-GAAP 调整 $27.75 亿美元 / EPS $2.77**，包括：无形资产摊销 $16.97 亿（最大） + 投资减值 $2.48 亿 + 商誉减值 $5.62 亿 + 其他 $0.04 亿 - 税务效应 $6.82 亿 - 离散税项 $1.54 亿

## 1. 业绩总结

### 1.1 收入与利润表现

- 全年总收入 **$245.68 亿美元**，同比 **+3.0%**（GAAP Sales Growth）/ **+2.0%**（Core Revenue Growth，非 GAAP）
  - 收并购贡献 +0% / 汇率 -1%
- **GAAP 营业利润 $46.90 亿美元**，同比 **-3.6%**（FY24 $48.63 亿）
- **GAAP 营业利润率 19.1%**（FY24 20.4%，**-130 bps**）—— 主要受减值 $562M 影响
- **Non-GAAP 调整后 Op Margin ~28.4%**（FY25 Q4 Adj Op Margin ~28.0%，FY26 Guidance ~28.5%）
- **GAAP 净利润 $36.00 亿美元**（持续经营），同比 -7.7%（FY24 $38.99 亿）
- **GAAP 摊薄 EPS $5.03**（-4.9% vs FY24 $5.29）
- **Non-GAAP Adjusted 摊薄 EPS $7.80**（+4.5% vs FY24 $7.48）—— **管理层核心指标**
- 经营现金流 $64.16 亿美元（-4.1% vs FY24 $66.88 亿）
- **自由现金流 $52.93 亿美元**（vs FY24 $53.09 亿，基本持平）
- **FCF / Net Earnings 转化率 1.47x**（vs FY24 1.36x，持续强劲现金转化）
- Q4 FY25：Revenue $68.38 亿（+4.5% / +2.5% core）、Non-GAAP EPS $2.23（+4.2%）
- FY26 Guidance：Core Revenue **+3% - +6%** + Adjusted EPS **$8.35-$8.50（+7-9%）**

### 1.2 增长与拖累因素

- **主要增长驱动**：① **Biotechnology 板块 +8.0% (+6.5% core)**：生物制药 Bioprocessing 强劲复苏，Cytiva Xcellerex bioreactor + 抗体 + 基因治疗 + 细胞治疗工艺；② **Biotechnology Op Profit +10.6%**（$1,864M vs $1,685M），**Op Margin 25.6%（+70 bps）**：规模恢复 + 运营杠杆；③ **Diagnostics +1.6% (+1.5% core)**：Cepheid 呼吸道检测（Q4 $5 亿）+ Beckman Coulter DxI 9000 + Leica 组织病理 + Radiometer 床旁检测；④ **Cepheid Xpert GI Panel FDA 批准**（11 种胃肠道病原体 PCR 检测）；⑤ **Cytiva Xcellerex 500L / 2,000L 生物反应器**：提高生物制药效率；⑥ **SCIEX ZenoTOF 8600 质谱仪**：加速药物开发；⑦ **Diagnostics Q4 Op Margin 26.2%**：改善动量明显；⑧ **Danaher Business System (DBS)**：持续运营效率改善；⑨ **现金流稳健**：FCF $53 亿，FCF / Net Earnings 转化率 1.47x
- **主要拖累因素**：① **Life Sciences 板块 0%（-1.5% core）**：仪器业务疲软 + 中国市场复苏缓慢 + 政府客户预算压力；② **Life Sciences Op Profit -40.8%**（$520M vs $879M），**Op Margin 7.1%（-490 bps）**：主要受 $432M Trade Name 减值影响（Aldevron）+ 工厂减值 $14M；③ **减值损失 $562M**（5.6 亿美元）：Life Sciences trade name $432M（Aldevron 商标）+ Biotechnology 工厂 $101M + Diagnostics 商标 $15M + Life Sciences 工厂 $14M；④ **Diagnostics Op Profit +1.0%**（$2,650M vs $2,625M），**Op Margin 26.7%（-10 bps）**：COVID-19 Cepheid 测试增长放缓，Q4 Cepheid 呼吸道季节性 $500M；⑤ **汇率 -1%** 拖累全年收入；⑥ **销售 / 管理费用 +6.1%**（$82.35 亿 vs $77.59 亿）：高于收入 +3%；⑦ **利息净额 $2.35 亿**（利息支出 $265M - 利息收入 $30M）
- **一次性/异常项目**：① **$562M 减值**：主要为 Aldevron（Life Sciences 2021 年 $96 亿收购的基因治疗工艺品牌）Trade Name 贬值 $432M，反映基因治疗市场预期调低；② **$248M 投资公允价值净损失**：权益投资减值（可能为 Beyonder / 生命科学 / 风投组合）；③ **$14M Life Sciences 工厂出售收益** + $10M 并购 Contingency 解除收益 + $9M 产品线出售收益；④ **$3,088M 股票回购**（1,200 万股，均价 ~$257），回购力度减少 48% vs FY24 $5,979M（因未能找到合适的大额并购标的）；⑤ **$1,556M 新发债** + $500M 债务偿还：净发债 $1,056M 支持回购

### 1.3 趋势判断

- **未来主要利好方向**：① **Biotechnology / Bioprocessing 持续复苏**：Cytiva 订单加速，FY26 Guidance +High-single digit Q1 / +6% 全年；② **Diagnostics 改善动量**：Q4 Op Margin 26.2%（+250 bps vs FY24 Q4 23.7%），Cepheid 新品（GI Panel）+ Beckman Coulter 神经退化测试；③ **GLP-1 药物生物制造**：Novo Nordisk / Eli Lilly 扩产，Danaher Cytiva 是主要工艺供应商；④ **细胞 + 基因治疗**：~30% 生物制药新品进入临床，CGT 工艺需求增长；⑤ **AI 药物研发**：SCIEX + Beckman Coulter 提供 AI 驱动科研工具；⑥ **Danaher Business System (DBS)**：50 年历史的持续改善系统，全球 60,000 员工；⑦ **FY26 Guidance $8.35-$8.50 EPS (+7-9%)**：稳健增长；⑧ **资产负债表健康**：Cash $46 亿 + Long-term Debt $184 亿，净债务可承受；⑨ **$3-5 亿美元/年 并购资金**：2026-2027 可能有大额并购（Life Sciences 或 Diagnostics）；⑩ **股票回购 + 股息增长**：2025 年 $31 亿回购 + $9 亿股息
- **未来主要压力方向**：① **Life Sciences 仪器业务疲软**：中国市场 + 政府客户 + 学术研究预算压力，FY26 Q1 Guidance Flat / -Low-single digit；② **中国 VBP（集中采购）**：Diagnostics 部分产品受压，Cepheid / Beckman Coulter 中国业务受影响；③ **并购整合 Aldevron 不及预期**：2021 年 $96 亿收购的基因治疗品牌，FY25 已减值 $432M，累计减值 $601M；④ **关税 / 贸易冲突**：Trump 2025 年关税，Danaher 有爱尔兰 / 德国 / 丹麦 / 美国多元供应链，短期挑战；⑤ **GLP-1 周期**：如果 Novo / Lilly 扩产放缓，Cytiva 订单可能调整；⑥ **AI 药物研发替代风险**：AlphaFold 3 + LLM 可能降低部分传统实验室需求；⑦ **美国政府医疗 / 科研预算**：NIH / CDC / FDA 预算调整风险（特朗普新政）；⑧ **股票回购力度减少**：FY25 $31 亿 vs FY24 $60 亿，EPS 支撑减弱；⑨ **竞争加剧**：Thermo Fisher（仪器）+ Abbott / Roche（诊断）+ Sartorius（生物反应器）
- **整体趋势判断**：**Danaher FY2025 交出"收入 $246 亿 (+3.0% / +2.0% core)、Biotechnology +8.0% +6.5% core 加速复苏、Diagnostics +1.6% Q4 OpM 26.2%、Adjusted EPS $7.80 (+4.5%)、FCF $53 亿、$562M 减值拖累 GAAP"的稳健复苏业绩**，**充分体现"Bioprocessing 复苏 + Diagnostics 韧性 + Danaher Business System 执行 + DHR 多元化 Life Sciences / Diagnostics 平台"的结构优势**。**FY26 核心看点是 Biotechnology +6% (Cytiva 持续订单) + Diagnostics 改善 + Adjusted EPS $8.35-$8.50 (+7-9%) + 可能的并购 + GLP-1 与 CGT 终端需求**；**核心风险是 Life Sciences 仪器持续疲软 + Aldevron 整合 + 中国 VBP + 关税 + AI 药物研发替代 + 美国政府科研预算压力**

## 2. 主营业务

### 2.1 公司简介

Danaher Corporation（DHR）是**全球领先的生命科学 + 诊断创新公司**，总部位于首都华盛顿特区。**1969 年由 Steven 和 Mitchell Rales 兄弟创立（以"水利 + 工业品"起家）**，至今已有 **57 年历史**。**2022-2023 年完成战略转型**：**2022 年分拆 Envista**（牙科业务 IPO）、**2023 年 9 月分拆 Veralto**（工业水处理 + 产品识别）。**2025 年 Danaher 已专注于"生命科学 + 诊断"三大板块**：Biotechnology（生物技术）、Life Sciences（生命科学）、Diagnostics（诊断）。公司以**Danaher Business System (DBS)** 闻名 —— 基于 Toyota Production System 的持续改善运营体系，已实施 **50 年**。**全球员工 60,000+**，**服务全球 160+ 国家**。Chairman：**Steve Rales**（创始人之一）；CEO：**Rainer M. Blair**（2020 年 9 月上任）。**关键并购历史**：2020 年 $213 亿收购 Cytiva（GE Life Sciences 生物技术业务）、2021 年 $96 亿收购 Aldevron（基因治疗 CDMO）、2023 年 $57 亿收购 Abcam（抗体）、2011 年 $68 亿收购 Beckman Coulter（诊断）、2016 年 $40 亿收购 Cepheid（分子诊断）。**2025 年底市值约 $1,400-1,600 亿美元**。

### 2.3 核心收入来源与产品

**Biotechnology（29.7%）—— 最快增长板块**

**主要产品**：
- **Cytiva（$213 亿 2020 年从 GE 收购）**：生物制药工艺设备（生物反应器、层析纯化、过滤），88% 经常性收入
  - **Xcellerex bioreactor**：500L / 2,000L 新规格（FY25 发布）
  - **ÄKTA chromatography**：蛋白质纯化系统
  - **HyClone culture media**：细胞培养基
- **Pall Life Sciences**：过滤 + 分离技术
- **Aldevron（$96 亿 2021 年收购）**：基因治疗 CDMO，mRNA / DNA / 蛋白生产
  - **FY25 Trade Name 减值 $432M**（累计 $601M 减值）

**收入构成**：88% 经常性（耗材 + 服务）+ 12% 非经常性（设备）

**客户**：Top 20 全球制药公司 + 生物技术公司 + CDMO

**Life Sciences（29.9%）—— 最大减值压力板块**

**主要产品**：
- **Beckman Coulter Life Sciences**：流式细胞仪 + 离心机 + 粒子计数
  - **CytoFLEX flow cytometry**：流式细胞仪 #1
- **SCIEX**：质谱仪（蛋白组学 + 代谢组学）
  - **ZenoTOF 8600**（FY25 新品）：高分辨率质谱
- **Leica Microsystems**：显微镜 + 病理学
  - **Aperio GT 450 DX**：数字病理扫描仪
- **Molecular Devices**：细胞成像 + 高通量筛选
- **IDBS**：生物制药数据管理软件
- **Phenomenex**：色谱分析 + 样本前处理
- **Abcam（$57 亿 2023 年收购）**：抗体产品（1,300,000+ 抗体目录）

**收入构成**：66% 经常性（耗材 + 试剂）+ 34% 非经常性（仪器）

**客户**：制药 + 学术研究 + 政府实验室 + 临床实验室

**Diagnostics（40.5%）—— 最大板块**

**主要产品**：
- **Cepheid（$40 亿 2016 年收购）**：分子诊断 PCR（GeneXpert 系统）
  - **Xpert GI Panel**（FY25 FDA 批准）：11 种胃肠道病原体 PCR
  - **Xpert Xpress 呼吸道 4-Plex**：COVID-19 + 流感 A/B + RSV
  - **Xpert MTB/RIF**：结核病 + 利福平耐药
- **Beckman Coulter Diagnostics（$68 亿 2011 年收购）**：免疫 / 临床化学 / 血液学
  - **DxI 9000 immunoassay platform**：FY25 扩展神经退化疾病检测
  - **DxA 5000 total lab automation**：实验室自动化
- **Leica Biosystems**：组织病理 + 癌症诊断
- **Radiometer**：床旁血气分析
- **Mammotome**：乳腺活检设备

**收入构成**：89% 经常性（耗材 + 试剂）+ 11% 非经常性（仪器）

**客户**：医院 + 诊断实验室 + 基层医疗

## 7. 未来可能的利空

### 利空 1：Life Sciences 仪器持续疲软 + Aldevron 整合不及预期 + $562M 减值连续 + 中国 VBP + 美国政府科研预算

**依据**

- **Life Sciences FY25 Core -1.5%**：全年疲软
- **Life Sciences Q4 Core +0.5%**：改善但仍低迷
- **FY26 Guidance Life Sciences**：Flat（全年）/ Flat / -Low-single digit（Q1）
- **Life Sciences Op Margin 7.1%**（FY25）vs 12.0%（FY24）：**-490 bps**
- **$562M 减值**（FY25）：
  - Life Sciences trade name $432M（Aldevron）
  - Biotechnology 工厂 $101M
  - Life Sciences 工厂 $14M
  - Diagnostics 商标 $15M
- **Aldevron 整合问题**：
  - 2021 年 $96 亿收购（基因治疗 CDMO）
  - FY25 Trade Name 减值 $432M
  - 累计 Aldevron 减值 $601M（FY22 $169M + FY24 无 + FY25 $432M）
  - 基因治疗市场预期低于收购时
- **中国 VBP（集中采购）**：
  - Beckman Coulter 临床化学 + 免疫诊断
  - SCIEX 质谱仪 + 色谱
  - Leica 病理学
  - Cepheid 分子诊断
  - 每轮招标价格下降 30-50%
- **美国政府科研预算**：
  - NIH 预算 FY25 $47 亿（横盘）
  - CDC 预算可能被削减
  - FDA 检查费用上升
  - 新政府（2025/01 Trump 上任）可能削减 NIH / NSF 预算
- **学术 + 政府客户占 Life Sciences 约 30%**：风险集中

**解读**

**Life Sciences 业务面临多重压力的"三重衰退"风险**：① **仪器业务衰退**：SCIEX + Beckman + Leica 的高价仪器依赖客户资本开支，2024-2025 年制药 + 学术预算紧张 + 中国需求疲软；② **Aldevron 减值连续**：$96 亿收购 → 累计 $601M 减值（6.3% 减值率），如果 FY26-27 基因治疗市场继续放缓，可能再次发生 $3-5 亿减值；③ **中国 VBP 持续**：中国占 Danaher 约 12% 收入，每年下降 2-3%；④ **NIH 预算风险**：如果 Trump 政府削减 NIH 15%（约 $70 亿），Life Sciences 仪器业务可能额外下滑 5-8%；⑤ **Abcam 整合**：2023 年 $57 亿收购，FY25 表现未披露，市场关注整合进度；⑥ **Op Margin 7.1% 过低**：Life Sciences 按行业标准应达到 15-20%，表明成本结构或需要 2-3 年重组；⑦ Life Sciences 占 Danaher 30% 收入，如果继续疲软，可能拖累整体 EPS 增速从 +7-9%（Guidance）降至 +4-5%。

### 利空 2：GLP-1 周期不确定性 + Bioprocessing 订单可能放缓 + 竞争加剧 + 关税风险

**依据**

- **Bioprocessing 订单特性**：
  - 非线性，往往 1-2 年一个大周期
  - FY22-23 疫苗 / 抗体 destocking（客户去库存）
  - FY24 开始复苏

  - Novo Nordisk / Lilly 扩产高峰 2025-2027
  - 如果 GLP-1 需求增长放缓（例如竞争对手进入、保险限制、替代药物）
  - 或 Novo / Lilly 自己产能扩建完成
  - Cytiva 订单可能从 FY28 开始回落
- **竞争加剧**：
  - **Sartorius**（德国）：生物反应器直接竞争，FY25 增速 +8%
  - **Thermo Fisher Life Sciences**（生物制药 / BioProduction）：$120+ 亿业务
  - **Merck Millipore**（德国 / Merck KGaA）
  - **Repligen**（蛋白过滤）
  - **Agilent**（分析仪器）

  - 2023-2024 年炒作过高
  - 部分 CGT 产品商业化失败（Bluebird Bio / bluebird）

  - 如果征收 10-25% 关税，可能增加 $1-3 亿美元成本

**解读**

**Bioprocessing 业务的"周期性 + 竞争性"是潜在风险**：① **GLP-1 周期可能在 FY28 转向**：Novo + Lilly 产能扩建完成后，新设备订单下降，虽然耗材仍增长但整体增速可能从 +6-8% 降至 +3-5%；② **Sartorius 直接竞争**：德国公司 Sartorius 在欧洲 + 亚洲市占率上升，FY25 Cytiva 失去部分市场；③ **Thermo Fisher 规模优势**：Thermo Fisher $450 亿营收（vs Danaher $246 亿），规模 + 并购能力更强；④ **CGT 市场不及预期**：Aldevron 减值反映市场预期调低，FY26-27 如果 CGT 产品商业化继续受挫，Cytiva 新订单可能下滑；⑤ **关税风险**：虽然 Danaher 有 60% 国际收入，但关键零部件 / 试剂仍来自美国 + 欧洲，关税成本可能侵蚀 Op Margin 50-100 bps；⑥ **外汇波动**：2025 年美元走强拖累 1%，2026 年 Guidance 预期美元走弱 +1%，但宏观不确定性高；⑦ 如果 FY26 实际 Core Revenue 仅 +2-3%（而非 Guidance 中位 +4.5%），Adjusted EPS 可能低于 $8.35。

## 8. 总结判断

**一句话结论**

**Danaher FY2025 交出"收入 $246 亿 (+3.0% GAAP / +2.0% core)、Biotechnology +8.0% (+6.5% core) 加速复苏、Diagnostics 改善动量 (Q4 Op Margin 26.2%)、Adjusted EPS $7.80 (+4.5%)、FCF $53 亿 (转化率 1.47x)、$562M 减值拖累 GAAP、股票回购 $31 亿"的稳健复苏业绩**，**充分体现"Bioprocessing 周期回暖 + Cytiva 产品管线 + Diagnostics 现金流韧性 + Danaher Business System 运营效率 + 聚焦生命科学/诊断平台"的结构优势**。**FY26 核心看点是 Biotechnology +6% (Cytiva 持续订单 + GLP-1 + CGT) + Diagnostics 改善 + Adjusted EPS $8.35-$8.50 (+7-9%) + 潜在 $30-50 亿并购 + Cepheid Xpert GI Panel 商业化 + Beckman Coulter 神经退化检测**；**核心风险是 Life Sciences 仪器持续疲软 + Aldevron 整合 + $562M 减值可能继续 + 中国 VBP + 美国 NIH 预算 + GLP-1 周期 + Sartorius / Thermo Fisher 竞争 + AI 药物研发替代 + 估值 25x Forward PE**。

**核心看点**

1. **Biotechnology / Bioprocessing 复苏 + GLP-1 + CGT + Cytiva 独特生态**：FY26+Cytiva 订单加速，Biotechnology 可能占比升至 35%
2. **Diagnostics 改善动量 + Cepheid 新品 + Beckman 神经退化 + Leica AI 病理**：稳定现金流 + 新增长引擎
3. **DBS + 强资产负债表 + 潜在并购 + 股东回报 + 股息 +14.8%**：Hidden Compounder，30 年年化 15%+

**核心风险**

1. **Life Sciences 仪器疲软 + Aldevron 连续减值 + $562M FY25 + NIH 预算 + 中国 VBP**：Life Sciences 利润率 7.1% 过低，2-3 年重组期
2. **GLP-1 周期不确定 + Sartorius / Thermo 竞争 + 关税 + 外汇**：Bioprocessing 增速可能 FY28 转向
3. **商誉 $431 亿 (52% 总资产) + AI 药物研发替代 + Forward PE 25x + 股票回购力度减少**：估值 + 长期颠覆风险
