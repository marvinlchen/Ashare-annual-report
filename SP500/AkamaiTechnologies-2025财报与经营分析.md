# 【Akamai Technologies, Inc. (NASDAQ: AKAM)】2025 年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

- **全年营业收入 $4,208.2M（同比 +5.4%，汇率调整后 +5%）**；Q4 单季 $1,094.9M（同比 +7%），加速增长。
- **GAAP 营业利润 $566.9M（同比 +6.3%）**，GAAP 营业利润率 13%（持平）；**Non-GAAP 营业利润 $1,253.6M（同比 +7.4%）**，Non-GAAP 营业利润率 **30%**（+1pp）。
- **GAAP 归母净利润 $452.0M（同比 -10.5%）**，GAAP 稀释 EPS **$3.07（同比 -6.1%）**，主要受 Q4 $55M 重组费用和有效税率从 14% 升至 25% 拖累；**Non-GAAP 净利润 $1,046.4M（同比 +5.1%）**，**Non-GAAP 稀释 EPS $7.12（同比 +9.9%）**。

### 1.2 增长与拖累因素

- **主要增长驱动**：
  - **Security 全年 $2,243.4M（同比 +9.8%）** - 已占收入 53.3%，成为第一大业务；其中 **Guardicore 微分段 + API Security $293M（同比 +43%）** 加速成长。
  - **Cloud computing 全年 $708.1M（同比 +12.3%）** - 其中 **Cloud Infrastructure Services（CIS，含 Linode+EdgeWorkers）$314M（同比 +36%）** 是最高增速业务。
  - 国际业务 $2,069.0M（同比 +8%，汇率调整后 +7%），国际收入增速超过美国 3%。
- **主要拖累因素**：
  - **Delivery（内容分发 CDN）全年 $1,256.7M（同比 -4.7%）**，长期结构性衰退（2024 年同比 -15%，2025 年降幅已大幅收窄）。
  - Q4 $55M 重组费用（裁员 + 无形资产减值）、GAAP 有效税率从 14% 升至 25% 挤压 GAAP EPS。
  - 年内资本开支从 17% 升至 19%（$818M vs. 去年 $676M），反映 AI/Compute 基础设施投资加大。

### 1.3 趋势判断

- **未来主要利好方向**：AI 推理云（Akamai Inference Cloud）开始商业化，CIS 增速 +36%+，Security 的 Guardicore + API Security 保持 40%+ 高增；公司向"安全 + 云 AI 基础设施"双引擎战略转型效果明显。
- **未来主要压力方向**：Delivery 业务仍将小幅萎缩；债务大幅扩张（可转债净增 ~$2.6B）用于回购和 AI 基建投资；竞争对手 Cloudflare、Fastly、AWS/Azure/GCP 持续挤压。
- **整体趋势判断**：收入结构从 CDN 向 Security + Compute 转型已基本完成，FY2026 收入指引 $4.40-4.55B（+4.6% ~ +8.1%），Non-GAAP EPS 指引 $6.20-7.20，预示 AI 基建投入期对 EPS 有一定拖累但长期复利可期。

## 2. 主营业务

### 2.1 公司简介

**Akamai Technologies, Inc.** 是全球领先的**网络安全和云计算公司**，总部位于美国马萨诸塞州剑桥市，业务核心是通过分布式边缘云平台保护和加速全球企业的在线业务。截至 FY2025 年末员工 **11,382 人**（同比 +634 人）。

### 2.2 主营业务构成

1. **Security（安全）**：全年收入 $2,243M（占比 **53.3%**），同比 +9.8%。核心产品包括 Web 应用防火墙（WAF）、DDoS 防护、Bot Manager、Zero Trust（基于 2021 年 Guardicore 收购）、API Security（基于 2023 年 Noname Security 收购）等。
2. **Delivery（内容分发）**：全年收入 $1,257M（占比 **29.9%**），同比 -4.7%。这是 Akamai 的传统 CDN 业务，是公司起家业务但长期面临结构性萎缩。
3. **Cloud Computing（云计算）**：全年收入 $708M（占比 **16.8%**），同比 +12.3%。包含 **CIS（Cloud Infrastructure Services，基于 2022 年收购的 Linode）**、EdgeWorkers 和合作伙伴方案等。2025 年 11 月推出 **Akamai Inference Cloud**（AI 推理云），主打 AI 应用和 AI 智能体的低延迟推理服务。

### 2.3 核心收入来源

- **Security 已成为 Akamai 最大的单一业务**，FY2025 Q4 占季收入 54.1%，同比 +11%；其中 Guardicore 微分段 + API Security 单季 $90M（+36%）。
- **Cloud Computing 是增速最快的业务**（Q4 +14%），其中 CIS 单季 $94M（+45%）增速最高。
- **国际业务占比 49.2%**（$2,069M），地理分布均衡，国际增速 +8% 超过美国 +3%。

## 6. 未来可能的利好

### 利好 1:AI 推理云(Inference Cloud)带来全新增长引擎

**依据**:管理层 CEO Tom Leighton 在电话会议中强调:"CIS 增速受益于 **Akamai Inference Cloud** 和 AI 应用/智能体的整体增长"。Akamai 拥有全球最分布式的边缘云平台(>4,100 PoP 节点,>130 个国家),天然适合低延迟的 AI 推理场景。

**解读**:边缘推理是 AI 应用商业化落地的关键环节之一,尤其是 AI 智能体和实时 AI 应用。Akamai 凭借已有的分布式边缘基础设施,相比 AWS/GCP/Azure 在推理延迟和成本上具有差异化优势。FY2026 CIS 收入有望加速至 +40-50% 增速。

### 利好 2:网络安全微分段 + API Security 双轮驱动

**依据**:Guardicore 微分段 + API Security 全年收入 $293M(+43%),Q4 $90M(+36%)。这两个产品均是企业 Zero Trust 和 API 安全的核心方向,FY2026 有望继续维持 30-40% 增速。API Security 受益于 Noname Security 收购后的产品整合。

**解读**:网络安全市场规模大且增速高,Zero Trust 微分段和 API Security 是企业刚需。按目前增速,到 FY2027 这两个产品线年收入有望超过 $550-600M,成为 Security 板块的核心增量。

### 利好 3:业务结构转型成功,Delivery 拖累淡化

**依据**:Delivery 收入降幅从 FY2024 的 -15% 大幅收窄至 FY2025 的 -5%(Q4 仅 -2%),下行接近尾声;Security + Cloud Computing 合计已占 70% 收入。管理层 FY2026 指引收入 $4.40-4.55B(+4.6% ~ +8.1%),体现增速有望小幅加速。

**解读**:市场长期担心 Akamai 的 CDN 老业务拖累整体增速,但 FY2025 已证明转型成功。随着 Delivery 占比继续下降(预期 FY2026 降至 28% 左右),整体增速有望从 +5% 升至 +6-8%,对估值形成支撑。

### 利好 4:充裕现金流 + 回购持续

**依据**:全年经营现金流 $1.52B、Adjusted EBITDA $1.80B、现金及有价证券 $1.92B;FY2025 回购 $800M(1,000 万股,均价 $79.77/股);虽然 Q4 暂停回购,但回购总额远超去年 $557M。

**解读**:公司资本返还能力强,在股价回撤时有能力持续回购。截至 FY2025 末股数 1.45 亿股(vs. 年初 1.53 亿股),年减约 4-5%,每股价值提升。

### 利好 5:国际业务成为收入第二引擎

**依据**:国际收入 $2,069M(+8%),占比 49.2% 接近半壁江山;国际增速连续多年超越美国。Akamai 平台在全球 130+ 国家的优势难以复制。

**解读**:国际市场潜力大,尤其是 APAC、拉美等新兴市场。国际业务多元化也降低了地缘政治集中风险。

## 7. 未来可能的利空

### 利空 1:Cloudflare、AWS/Azure/GCP 等竞争加剧

**依据**:Cloudflare 在 CDN、边缘安全和边缘计算全面对标 Akamai,FY2025 增速 >25%;AWS CloudFront、Azure Front Door 绑定大客户云合同,分食市场;Fastly 在现代应用 CDN 领域持续蚕食。

**解读**:Akamai 的 Delivery 业务结构性下行虽已收窄,但长期仍将继续。Security 板块面临 Cloudflare 和专业安全厂商(Palo Alto Networks、Zscaler)的双重挤压;Cloud Computing 面临超大规模云厂商的价格战和规模战。Akamai 虽然在分布式边缘云有差异化,但市场占有率仍较低。

### 利空 2:债务扩张 + Capex 加大压制自由现金流

**依据**:可转债总额从 $3,546M 升至 $4,105M(+$559M);Capex 从 $676M 升至 $820M(+$144M,占收入 17%→19%);FY2026 指引 Capex 占收入 **23-26%**,远高于 FY2025 的 19%。

**解读**:FY2026 Capex 将达到 $1.0-1.2B,同比 +25-50%,自由现金流(FCF)预计从 FY2025 的 ~$700M 降至 $500-600M。公司处于 AI 基建投入期,短期 FCF 承压。若 Inference Cloud 商业化不及预期,投资回报率将受质疑。

### 利空 3:重组和 SBC 持续压制 GAAP 利润

**依据**:FY2025 重组费用 $58M(其中 Q4 $55M,裁员 + 无形资产减值);FY2024 重组费用 $95M;SBC 从 $393M 升至 $459M(+17%)。

**解读**:公司持续调整组织结构(从 CDN 向 AI + Security 转型),重组费用可能继续出现;AI 人才战导致 SBC 占比持续提升。GAAP EPS 与 Non-GAAP EPS 的缺口($7.12 vs. $3.07,差额 $4.05)主要来自 SBC 和重组,市场对这部分的接受度有限。

### 利空 4:FY2026 Non-GAAP EPS 指引 $6.20-7.20(中枢持平甚至下降)

**依据**:FY2026 Non-GAAP EPS 指引 $6.20-7.20,中枢 $6.70 **低于 FY2025 实际的 $7.12**(-5.9%);指引下限 $6.20 较 FY2025 实际下降 13%。核心原因是 Capex 占比上升(折旧增加)、SBC 增加、预期回购放缓。

**解读**:指引的 EPS 下行是市场最大的担忧。管理层解释为"AI 投资期"的短期阵痛,但若投资回报率兑现不及时,股价将面临压力。即使增长结构优化,利润持平意味着估值扩张空间有限。

### 利空 5:Delivery 业务加速下行风险

**依据**:Q4 Delivery -2%(YoY)已经大幅好于 Q1 的 -6%,但仍为负增长;AI 应用让传统静态内容分发需求进一步减少(生成式 AI 动态内容占比上升);大客户(如 Netflix 等)自建 CDN 趋势。

**解读**:若下行再次加速(如 -5% 以上),将部分抵消 Security 和 CIS 的增速,拖累整体增速。管理层在指引中已留出余地($4.40-4.55B 跨度较大)。

### 利空 6:商誉和无形资产减值风险

**依据**:商誉 $3,207M(占总资产 27.9%),无形资产 $614.5M;FY2025 Q4 $55M 重组费用中包含部分无形资产减值;Noname Security(2023 年以约 $450M 收购)等近期并购尚在整合期。

**解读**:若 Security 或 Cloud Computing 某条产品线表现不及预期,可能触发更大规模商誉减值,冲击 GAAP 利润。

## 8. 总结判断

**一句话结论**

**FY2025 Akamai 完成从 CDN 向"安全 + AI 云基础设施"的关键转型,Security 和 Cloud Computing 占比升至 70%,但 FY2026 Non-GAAP EPS 指引 $6.20-7.20 中枢持平,反映 AI 基建投入期对短期利润的压制,长期成长性与估值将取决于 Akamai Inference Cloud 的商业化兑现速度。**

**核心看点**

1. **AI 推理云(Akamai Inference Cloud)新成长引擎** - Q4 CIS +45%,边缘推理差异化优势。
2. **Guardicore 微分段 + API Security +43%** - 网络安全刚需领域持续高增。
3. **Security 占比 53%,业务结构转型成功** - Delivery 下降幅度收窄至 -5%,拖累淡化。
4. **充裕现金流 + 1,920M 现金** - 全年回购 $800M(vs. $557M),股数减少 ~5%。
5. **国际业务 +8% 贡献 49.2% 收入** - 地理多元化。
6. **Non-GAAP 营业利润率 30%(+1pp)** - 规模效应和产品组合优化。

**核心风险**

1. **FY2026 Non-GAAP EPS 指引中枢持平** - Capex 占比升至 23-26%,压制自由现金流和 EPS,市场担忧 AI 投资回报率。
2. **Cloudflare、AWS、Cloudflare、Palo Alto Networks 等全面竞争** - Security 和 Cloud Computing 赛道竞争激烈。
3. **可转债扩张(+559M)+ 重组费用 + SBC 增加** - 压制 GAAP EPS,GAAP 与 Non-GAAP 缺口扩大。
4. **Delivery 业务仍将持续下行** - 若加速,将拖累整体增速回升。
5. **商誉 $3.2B + 无形资产减值风险** - 近期并购整合期的不确定性。
6. **股价估值扩张空间有限** - Non-GAAP 指引持平前提下,PE 扩张缺乏催化。
