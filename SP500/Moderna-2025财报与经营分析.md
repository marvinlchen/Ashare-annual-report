# Moderna Inc（MRNA）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

Moderna 2025 年全年总收入 $1,944M（同比 -39.9%），其中净产品销售 $1,818M（-41.5%），主要由 COVID 疫苗（Spikevax/mNEXSPIKE）驱动。GAAP 净亏损 $(2,822)M（EPS $(7.26)），较 2024 年亏损 $(3,561)M 收窄 20.7%。全年运营费用大幅缩减 $2.2B（-30%），超额完成降本目标——R&D 从 $4,543M 降至 $3,132M（-31%），SG&A 从 $1,174M 降至 $1,018M（-13%）。年末现金及投资余额 $8.1B，提供约 3 年的运营跑道。经营性现金流 $(1,873)M，较上年 $(3,004)M 改善 37.6%。

### 1.2 增长与拖累因素

主要增长/改善因素：（1）运营费用削减 $2.2B，R&D 降幅 31%，反映大型 Phase 3 呼吸道项目收尾和组合优化；（2）第三款产品 mRESVIA（RSV 疫苗）获批并开始贡献收入；（3）国际战略合作拓展（墨西哥、台湾长期协议），mNEXSPIKE 获加拿大和澳洲批准；（4）Q4 收入 $678M 高于预期上限。

主要拖累因素：（1）COVID 疫苗市场持续萎缩——接种率走低，产品销售同比 -42%；（2）库存减记 $291M 占 COGS 的 33.5%，反映 COVID 疫苗需求预测过高；（3）mRNA-1010（流感疫苗）和 mRNA-1083（流感+COVID 组合疫苗）均收到 FDA Refusal-to-File 通知，美国上市时间表不确定；（4）公司仍在深度亏损，年烧钱约 $1.9B。

### 1.3 趋势判断

Moderna 正处于从"单一 COVID 疫苗公司"向"mRNA 平台公司"转型的关键阶段。2025 年是转型最痛苦的一年——COVID 收入加速下滑，而新产品（RSV、流感、肿瘤）尚未形成规模。$8.1B 的现金储备（约 3 年跑道）是命脉。2026 年指引收入增长最高 10%（~$2.1B），但运营支出 ~$4.9B 意味着仍将亏损 ~$2.5-3.0B。真正的转折点取决于：（1）流感疫苗能否在 2026-2027 年获批；（2）mRNA-4157（肿瘤疫苗）Phase 3 数据是否积极；（3）诺如病毒疫苗 Phase 3 结果。如果这些管线成功，Moderna 有望在 2028-2029 年实现盈亏平衡；如果失败，公司将面临严峻的融资和生存压力。

## 2. 主营业务

### 2.1 公司简介

Moderna 成立于 2010 年，总部位于马萨诸塞州剑桥，是全球 mRNA 医药技术的先驱和领导者。公司通过其 mRNA 技术平台开发传染病疫苗、肿瘤治疗和罕见病疗法。截至 2025 年底，公司拥有三款已获批产品和一个涵盖传染病、肿瘤和罕见病的深厚管线。全球团队约 4,000+ 人（降本后规模），运营三个国际制造基地。

### 2.2 主营业务构成

1. **COVID-19 疫苗（Spikevax/mNEXSPIKE）**：核心收入来源。Spikevax 是原始 COVID-19 mRNA 疫苗，mNEXSPIKE 是 2025 年获批的下一代 COVID 疫苗。受全球接种率持续走低影响，收入大幅下滑。
2. **RSV 疫苗（mRESVIA）**：2025 年获批的第三款产品，面向 60 岁以上成人的呼吸道合胞病毒疫苗。市场与 GSK（Arexvy）和 Pfizer（Abrysvo）竞争。
3. **其他收入（$126M）**：包括政府拨款、合作/许可费、特许权使用费和 stand-ready 制造收入（来自国际战略合作伙伴的长期协议）。
4. **研发管线**：流感疫苗（mRNA-1010）、流感+COVID 组合疫苗（mRNA-1083）、诺如病毒疫苗（mRNA-1403）、个性化癌症疫苗（mRNA-4157/intismeran，与 Merck 合作）、癌症抗原疗法（mRNA-4359）、罕见病（丙酸血症 mRNA-3927、甲基丙二酸血症 mRNA-3705）。

### 2.3 核心收入来源

产品销售（$1,818M，93.5%）几乎全部来自 COVID 疫苗。其他收入（$126M，6.5%）来自合作和制造合同。地理分布：美国 ~$1,200M（62%），国际 ~$745M（38%）。

**利息收入提供重要缓冲**

$314M 的利息收入（-26%）来自 $8B+ 的现金和投资组合，相当于收入的 16%。随着现金持续消耗和利率可能下行，利息收入将逐年递减，对净亏损的缓冲效果将减弱。

**经营性现金流改善显著**

经营现金流从 $(3,004)M 改善至 $(1,873)M（+38%），主要受益于运营费用缩减和营运资本释放（应收回收 $156M、预付费减少 $153M）。资本支出从 $1,051M 大幅降至 $192M（制造基地建设基本完成）。

## 6. 未来可能的利好

### 利好 1：流感疫苗（mRNA-1010）和组合疫苗（mRNA-1083）——最大的近期收入机会

**依据**

mRNA-1083（流感+COVID 组合疫苗）正在欧洲和加拿大审批中；mRNA-1010（流感疫苗）在欧洲、加拿大和澳大利亚审批中，2026 年有望获批。全球流感疫苗市场规模约 $7-8B/年（Sanofi、CSL Seqirus、GSK 三分天下），mRNA 技术在匹配性和制造灵活性上具有潜在优势。

**解读**

如果流感疫苗在欧洲获批并在 2026-2027 年秋冬季上市，Moderna 将从"COVID 单品公司"升级为"呼吸道疫苗组合平台"。组合疫苗（流感+COVID 一针搞定）是最大的差异化卖点——目前没有竞品能做到。但美国 FDA 拒绝受理是重大不确定性，如果 mRNA-1083 的美国上市推迟至 2027-2028 年，收入增长将显著延迟。

### 利好 2：个性化癌症疫苗（mRNA-4157/intismeran）——潜在突破性疗法

**依据**

与 Merck 合作的 mRNA-4157 正在 8 项 Phase 2/3 临床试验中推进，覆盖黑色素瘤、非小细胞肺癌、膀胱癌和肾细胞癌。Phase 2b 辅助治疗黑色素瘤的 5 年数据显示，与 KEYTRUDA 联用将复发或死亡风险降低 49%。Phase 3 黑色素瘤数据可能在 2026 年读出。

**解读**

个性化癌症疫苗是 mRNA 技术平台最具革命性的应用之一——根据每个患者的肿瘤基因突变定制疫苗。如果 Phase 3 数据积极并获批，mRNA-4157 的市场潜力可能达到 $5-10B+/年（覆盖多个癌种的辅助治疗）。这将从根本上改变 Moderna 的估值逻辑——从"疫苗公司"变为"肿瘤免疫公司"。但 Phase 3 成功率存在不确定性，且商业化需要大量投资。

### 利好 3：诺如病毒疫苗（mRNA-1403）——潜在的 First-in-Class 产品

**依据**

mRNA-1403 Phase 3 试验已完成全部入组（第二个北半球季节 2025-2026），数据预计 2026 年读出。目前全球没有已获批的诺如病毒疫苗。全球诺如病毒每年导致约 6.85 亿例疾病和 20 万例死亡。

**解读**

如果 Phase 3 成功，mRNA-1403 将成为全球首款诺如病毒疫苗，具有 First-in-Class 优势和巨大市场潜力（估计 $2-4B/年的峰值销售）。这将证明 mRNA 平台在 COVID 之外的广泛适用性。竞争对手（Vaxcyte、Hillevax）进度落后，Moderna 有望获得先发优势。

## 7. 未来可能的利空

### 利空 1：COVID 疫苗市场持续萎缩——核心收入将进一步下滑

**依据**

COVID 疫苗接种率持续走低，2025 年产品销售已降至 $1.8B（-42%）。美国秋冬接种率约 20-25%，远低于疫情期间水平。2026 年指引收入增长最高 10%（~$2.1B），意味着 COVID 收入下滑被新产品部分抵消。

**解读**

COVID 最终可能成为类似流感的年度接种市场，但市场规模可能仅 $3-5B/年（vs 2022 年峰值 $35B+），Moderna 份额约 30-40%（~$1-2B）。如果接种率进一步走低或定价压力增加，COVID 收入可能跌破 $1B。新产品（RSV、mNEXSPIKE 海外）能否在 2026 年弥补 COVID 下滑是关键。

### 利空 2：年烧钱 ~$1.9B，$8.1B 现金约 3 年跑道——融资压力迫在眉睫

**依据**

2025 年经营现金流 $(1,873)M。2026 年指引运营费用 ~$4.9B（R&D $3.0B + SG&A $1.0B + COGS $0.9B），收入最高 ~$2.1B，暗示运营亏损 ~$2.8B。扣除利息收入和税后，净亏损仍可能达 ~$2.5B。年末现金指引 $5.5-6.0B。公司已抽取 $600M 信贷额度。

**解读**

按当前烧钱速度，$8.1B 现金可支撑约 3 年（至 2028 年）。如果管线项目（流感疫苗、癌症疫苗）在 2026-2027 年未能如期获批并产生收入，公司可能在 2027-2028 年面临融资需求。届时可能需要发行股权或债务，在当前 $200 亿+市值下，大规模股权融资将严重稀释股东。

### 利空 3：FDA Refusal-to-File——流感疫苗美国上市不确定性

**依据**

FDA 拒绝受理 mRNA-1010（流感疫苗）的上市申请，公司已请求 Type A 会议了解改进路径。mRNA-1083（流感+COVID 组合疫苗）也在等待 FDA 关于重新提交的指导。

**解读**

Refusal-to-File 意味着 FDA 认为提交材料存在重大缺陷，需要补充数据或重新提交。这可能将美国流感疫苗上市推迟 1-2 年（至 2027-2028 年）。美国是全球最大的流感疫苗市场，延迟将直接影响收入增长轨迹和投资者信心。虽然欧洲/加拿大/澳洲的审批正在推进，但这些市场规模较小且竞争激烈。

## 8. 总结判断

**一句话结论**

Moderna 2025 年在收入大降 40% 的逆境中成功削减运营费用 $2.2B、将净亏损从 $3.6B 收窄至 $2.8B，但 $8.1B 现金仅 3 年跑道，公司命运取决于流感疫苗和癌症疫苗管线能否在 2026-2027 年兑现。

**核心看点**

1. 运营费用大降 30%（$2.2B），经营现金流改善 38% 至 $(1.9B)，展示了管理层的降本决心
2. 管线丰富——流感疫苗（欧洲审批中）、癌症疫苗（Phase 3 数据 2026 年）、诺如病毒（Phase 3 数据 2026 年）提供多个催化剂
3. $8.1B 现金储备提供约 3 年缓冲，mNEXSPIKE 和国际合作有望在 2026 年推动 10% 收入增长

**核心风险**

1. COVID 疫苗市场持续萎缩，2025 年产品销售 -42%，核心收入正在侵蚀
