## 2. 主营业务

### 2.1 公司简介

Philip Morris International Inc.（PMI/PM）是**全球最大的国际烟草 + Smoke-Free 消费品公司**，总部位于美国康涅狄格州 Stamford。**2008 年从 Altria Group 拆分独立**（Altria 保留美国本土业务，PMI 专注国际市场），**自 2008 年在纽交所 (NYSE) 上市**。**2021 年 PMI 以 $160 亿美元收购 Swedish Match AB**（瑞典 Match，ZYN 尼古丁袋 + General snus 的创造者），这是公司历史上最大规模并购，彻底奠定全球 Smoke-Free 领导地位。**核心产品组合**：**Combustibles（传统香烟）**：**Marlboro**（全球最畅销香烟品牌，PMI 旗舰）、**L&M**、**Parliament**、**Virginia Slims**、**Chesterfield**、**Philip Morris**、**Bond Street** 等。**Smoke-Free Products (SFP)**：**Heated Tobacco（加热不燃烧）**：**IQOS**（行业领导者，全球 HTU 市占 76%）、**TEREA**（HTU 烟弹）、**HEETS**、**IQOS ILUMA**（新一代无刀片系统）；**Oral SFP（口含烟）**：**ZYN 尼古丁袋**（全球尼古丁袋第一品牌）、**General snus**（瑞典烟草，全球销量第一）；**E-vapor（电子烟）**：**VEEV**（闭合式电子烟）。**Wellness + Beyond Tobacco/Nicotine**：**Aspeya**（呼吸系统健康）、**Vectura**（2024 年出售）、**Fertin** 等。**2025 年 SFP 业务营收 $169B = 41.5% 总收入**，**SFP GP 占比 43%**，**目标 2030 年 SFP 占比 >67%**。**全球员工 ~80,000**，**服务 180+ 市场**，**主要制造基地**：瑞士（Neuchâtel IQOS 全球总部）、德国（工厂重组中）、瑞典（Swedish Match 总部）、意大利（Bologna IQOS）、美国（多个）、波兰、印度尼西亚、墨西哥等。**2025 年底市值约 $2,800 亿美元**（股价 ~$180 YoY +25%）。**Chair + CEO**：**Jacek Olczak**（2021 年接任，之前任 COO）；**CFO**：**Emmanuel Babeau**（2018 年加入）；**Chair Emeritus**：**Louis Camilleri**（2018 年退休 CEO 任 Chair）。**核心战略**：**"Vision of a Smoke-Free Future" + Smoke-Free Transformation** —— 目标是 PMI 成为基本不依赖 Cigarettes 的国际消费品公司。

**资产负债率变化**

**偿债压力判断**

偿债压力中等：① **Swedish Match 2022 收购产生 $16B+ 债务**，使 Net Debt / EBITDA 峰值达 3.5-4.0x；② FY24-25 去杠杆顺利，FY26 目标 2.0x；③ **CFFO $13.5B FY26**：有能力覆盖股息 $9B + Capex $1.5B + 还债 $3-5B；④ **信用评级 A-/A3**（稳定）：标普 + 穆迪 + Fitch；⑤ FY25 利息支出 ~$1.7B；⑥ **股息支付比率 ~80%**（股息 $9B / Adj 净利润 $11.8B）；⑦ 如 Adj EBITDA 下滑，可能压力上升，但 SFP 增长抵消；⑧ **2026 股票回购 = $0**：公司主动选择，反映去杠杆优先级。

**商誉 / 无形资产风险**

商誉 **~$13B + 无形资产 ~$10-11B = 总计 $23-24B**（占总资产约 36%）。**主要来源 Swedish Match 2022 年 $160 亿收购**。**减值风险评估：中等**：① Q4 FY25 有 $41M 商誉减值（小规模，可能 Wellness / Vectura 相关）；② Wellness 业务 Equity Investment Impairment $0.09/股 = $140M 估；③ **ZYN（尼古丁袋）增长 +36.6%**：证明 Swedish Match 收购成功，商誉覆盖价值不断上升；④ Intangible Amortization $1,003M/年：15-17 年完全摊销；⑤ **Vectura 出售损失 $199M**（FY24）：部分已消化；⑥ 整体 Swedish Match 商誉健康。

  - Turkey / Russia / 东欧高通胀税率压力
  - 新兴市场（印度 / 越南 / 泰国）SFP 禁令可能持续
- **反烟组织压力**：
  - 世界卫生组织 (WHO) 反对 HTU（认为仍有健康风险）
  - 学校 / 医疗 / 公共场所 SFP 限制
- **竞争加剧**：
  - **British American Tobacco (BAT)**：Velo 尼古丁袋 + Vuse 电子烟 + Glo HTU
  - **Altria**：on! 尼古丁袋 + NJOY 电子烟
  - **Japan Tobacco International (JTI)**：Ploom X HTU
  - **Imperial Brands**：blu 电子烟
  - 中国电子烟品牌（ELFBAR, SMORE 等）
- **Wellness 业务不确定**：
  - Aspeya 减值 $0.09/股（$140M 估算）
  - Fertin Pharma 整合不达预期
  - Wellness Revenue 小（<5% 总 Revenue）
  - 如果继续亏损，可能再有 Impairment
- **估值压力**：
  - 2025 年底 PM 股价 ~$180（YTD +25%）
  - Forward PE 21-22x（FY26 Adj EPS $8.50）
  - 历史均值 16-18x
  - **溢价 20-30%**
- **货币风险**：
  - FY25 FX 有利 +$461M
  - 如美元升值 FY26 可能 -$200-400M
  - 新兴市场货币（Turkey Lira、Russian Ruble、Argentine Peso、印尼 Rupiah）贬值风险

**解读**

**PMI 面临全球监管 + 竞争 + 估值的多重风险**：① **欧盟 TPD 3 最大监管风险**：如果 2026-2027 对 SFP 口味 / 税务 / 广告限制，PMI 欧洲业务受冲击大（占 42% Revenue + 48% OI）；② **美国 FDA 监管变化**：虽然 Trump 政府可能整体宽松，但青少年 Nicotine Pouch 使用率上升可能导致针对性限制；③ **ZYN 在美国的定价权受挑战**：BAT Velo + Altria on! 竞争加剧；④ **欧洲 IQOS 竞争**：JTI Ploom X + BAT Glo 在意大利 / 德国加速；⑤ **Wellness Aspeya 减值**：Fertin 整合不达预期，FY26 可能再有 Impairment $100-200M；⑥ **Forward PE 21-22x 溢价 20-30%**：反映了 Smoke-Free 转型乐观预期，如 FY26 EPS 未达 $8.38-8.53 上限 (+13%)，股价可能回调 10-15%；⑦ **货币风险**：FY26 美元可能升值（美联储慢于欧洲央行降息），会稀释 1-2pp EPS 增长；⑧ 如果多个风险同时出现（欧盟 TPD 3 + FDA 回撤 + 货币不利），FY26 Adj EPS 可能仅 +7-8%（低于 Guidance +9.5-13.1%）。

**Philip Morris International FY2025 交出"Net Revenue $406 亿首次破 $400 亿 (+7.3% Reported / +6.5% Organic)、Smoke-Free Business 近 $170 亿 (+15.0% / 41.5% 占比近半)、Adj Operating Income $164.28 亿 (+11.8% / +10.6% Organic)、Adj Op Margin 40.4% (+160 bps)、Adj EPS $7.54 (+14.8% / CN +14.2%)、Total Shipment Volume +1.4% (连续 5 年正增长)、IQOS HTU 全球 76% 份额 + ZYN 美国 +37% / 全球 +35%、3 年目标 2 年超额完成、股息连续 17 年加息年化 $5.88 (+11%)"的历史性转型优秀业绩**，**充分体现"从传统烟草公司向全球 Smoke-Free 领导者成功转型"的战略执行力 + Smoke-Free 规模化盈利兑现（GP 占比 43% 超过收入占比 41.5%）+ Swedish Match 收购价值释放（ZYN 美国爆发 +37%）+ IQOS 全球持续扩张（106 市场 43M+ 消费者 +4.5M）+ 新组织架构 2026/01/01 支持 IQOS 美国重启 + 2026-2028 新增长目标 +6-8% Revenue / +8-10% OI / +9-11% EPS CAGR + 净债务 / EBITDA 持续去杠杆至 2.0x + Jacek Olczak CEO 战略延续"的全球消费品龙头优势**。**FY26 核心看点是 Smoke-Free Revenue 突破 45% 占比 (预期 $190-200B) + ZYN 全球出货可能突破 1,100-1,200M cans + IQOS 美国独立板块 2026/01/01 开启重启 + VEEV E-vapor 继续翻倍 + Adj EPS $8.38-8.53 (+11.1-13.1%) / ex-FX $8.11-8.26 (+7.5-9.5%) + CFFO $13.5B + Capex $1.4-1.6B + 净债务 / EBITDA 达 2.0x 目标**；**核心风险是 Cigarettes Volume FY26 加速下降 -3% (历史 -1.5%) + Turkey / Poland / France 等主要市场份额压力（Turkey -5.6pp 最大）+ Americas 地区利润承压（ZYN 美国定价 + 投资）+ Germany 诉讼 + Restructuring 持续 + Wellness Aspeya 减值 + 欧盟 TPD 3 监管风险 + 竞争加剧 (BAT Velo / Vuse / Glo + Altria NJOY / on!) + Forward PE 21-22x 溢价 20-30% + FY26 No Share Repurchases + 货币风险**。
