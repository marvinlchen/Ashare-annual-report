> 报告期：**FY2025 全年（止 2025/12/31）** —— Pentair 财年与自然年一致
> 数据来源：Pentair plc Form 10-K for Fiscal Year Ended December 31, 2025（SEC 2026/02/24 提交，Acc-no 0000077360-26-000007）、Q4 & FY2025 Earnings Press Release（2026/02/03 发布，Exhibit 99.1）

## 1. 业绩总结

### 1.1 收入与利润表现

- **全年营收 $4,176.0M（+2.3%，Core 同比 +1.9%）**：Pool $1,558.8M（**+8.5%**，Core +6.5%，第一大板块）+ Flow $1,553.6M（+2.6%，Core +0.8%）+ Water Solutions $1,062.1M（**-6.1%**，Core -2.6%，拖累项），收购 +0.5% / 货币 +0.5% / 剥离 -0.1% 共同构成 +0.4% 非 Core 贡献。
- **GAAP 经营利润 $857.5M（+6.7%）**，**GAAP ROS 20.5%（+80bp）**；**Adjusted 经营利润 $1,053.5M（+9.8%）**，**Adjusted ROS 25.2%（+170bp）** —— **运营效率持续显著扩张**（Pentair Business System + Transformation + 80/20 策略驱动）。
- **GAAP Diluted EPS $3.93（+5.1%）** + **Adjusted Diluted EPS $4.92（+13.6%，创历史新高）**；**持续经营净利润 $649.5M（+3.8%），Adjusted Net Income $814.7M（+12.6%）**。

### 1.2 增长与拖累因素

- **主要增长驱动**：
  1. **Pool 板块 +8.5%**（Core +6.5%）—— 美国消费端维修 / 升级（aftermarket IoT 连网产品）+ 商用需求 + 新 SKU 推广；Pool 分部利润 $527.1M（+10.6%），ROS 33.8%（+60bp），为公司最高利润率业务；
  2. **Flow 板块利润率大幅扩张 +230bp**（ROS 21.0% → 23.3%）—— Residential Flow 渠道优化 + 商业 / 工业水泵业务定价纪律；Flow Q4 Core +4%、Q4 ROS +240bp 达 22.8%；
  3. **收购贡献 +0.5%** —— FY25 收购支出 $292.1M（vs FY24 $108.0M，+170% 加速）；
  4. **Transformation / 80/20 策略持续发酵**：Corporate 成本基本持平（$89.6M vs $90.5M）即使营收增长；
  5. **利息支出下降 22%**（$69.4M vs $88.6M，降息环境 + 债务结构优化）。
- **主要拖累因素**：
  1. **Water Solutions 全年 -6.1%（Core -3.7%）**：商业 Water Solutions 下滑最大（Q4 -15%），Pool Chemistry 消费支出疲软 + 渠道去库存；
  2. **FY25 Q2 剥离业务损失 $26.3M** + **资产减值及核销 $49.1M**（FY24 $17.6M）+ 法律和解 $11.6M（Q4）+ 交易费 $4.1M + 重组 $31.3M + Transformation $40.8M；
  3. **SG&A $736.9M（+5.1%）**：销售增长投入 + 激励薪酬；
  4. **汇率变动 -$24.9M 现金影响**；
  5. **商誉从 $3,286.6M 升至 $3,538.1M（+$251.5M）**—— 主要源于收购，短期无减值风险。

### 1.3 趋势判断

- **未来主要利好方向**：
  1. **FY26 Adj EPS 指引 $5.25-$5.40（+7~10%）+ Adj Operating Income +5~8% + ROS 继续扩张**（隐含 Adj ROS 约 26.0%，+80bp）；
  2. **50 年连续股息增长**：2026 年股息从 $0.25 → $0.27/Q（+8%），年度 $1.08（+8%）—— 显示管理层对现金流长期稳定的信心；
  3. **2026/03/04 Investor Day**：更新长期 EPS / ROS 目标 + Transformation Program 进展 + 新产品路线图；
  4. **Residential Flow + Water Solutions 渠道对齐**：新任高管职位（包括 Nick Brazis CFO）+ 组织架构重组预期带来 1-2% 额外协同；
  5. **Pool 业务强劲持续性**：Q4 Core +8.9% 延续；美国住房 + 老化泳池升级 + IoT 连网产品（VariablePro VSFHP）+ automation software（Pentair Pool IQ）；
  6. **$1.0B 股票回购剩余授权**（年回购 $225M 后）+ **收购管线活跃**。
- **未来主要压力方向**：
  1. **Water Solutions 持续疲软**：商业板块 Q4 -15%、住宅 chemistry -3.7% 核心销售，**转正时间点未明**；
  2. **潜在关税 / 供应链风险**：10-K 风险披露"changes in laws, regulations... impact trade agreements and tariffs"；公司在中国 / 墨西哥有制造基地，关税直接敞口；
  3. **住房需求 / 利率敏感度**：Pool 和 Flow 住宅业务与美国新房 + 住宅维修改造（R&R）市场高度相关；
  4. **收购整合风险**：FY25 $292M 收购加速，商誉 +8%；
  5. **汇率风险**：国际业务 ~25-30%。
- **整体趋势判断**：Pentair 已从"组合重塑"阶段（出售非核心业务 + 聚焦水资源）过渡至"**运营杠杆 + Transformation Program + 50 年股息连增**"的**稳健复利成长**阶段；**Pool 强劲 + Flow 利润率扩张 + Transformation 持续**支撑 FY26 指引 +7-10% Adj EPS 增长；**Water Solutions 弱势 + 关税风险**是 2026 主要监控点。

## 2. 主营业务

### 2.1 公司简介

**Pentair plc (PNR)** 是**爱尔兰注册、总部位于英国伦敦 / 运营中心位于美国明尼苏达 Golden Valley 的全球水资源技术领导者**，使命是"**帮助世界可持续地移动、改善、享受水 —— 生命最essential 资源**"。公司 **2026 年迎来 60 周年 + 50 年连续股息增长**（S&P 500 Dividend Aristocrat），全球 9,000 名员工服务超过 150 个国家，FY25 营收 $4.2B，业务涵盖住宅 + 商业 + 工业水资源应用（池 / 水处理 / 水泵 / 过滤）。

### 2.2 主营业务构成

Pentair 划分为 **3 个 reportable segments + 1 个 Corporate**：

1. **Pool（泳池，37.3% 占比，$1,558.8M，ROS 33.8%）**：**公司最核心且利润率最高的业务** —— 住宅泳池 + 商用泳池 + SPA 水疗的**水泵 / 过滤器 / 加热器 / 消毒器 / 灯 / 自动化控制器**完整解决方案。**IoT 连网产品（VSF VariablePro + Pool IQ automation）+ aftermarket 维修 / 更换**是核心利润来源。Q4 +11.2%（Core +8.9%，收购 +2.2%）；全年 +8.5%（Core +6.5%）。
2. **Flow（流量 / 水泵，37.2% 占比，$1,553.6M，ROS 23.3%，+230bp）**：**面向住宅 + 商业 + 市政 + 工业 + 农业水泵及相关流量系统** —— Residential Flow（家庭井水泵、增压泵、地下室排水泵 / sump pump）+ Commercial Flow（消防喷淋、HVAC 循环水）+ Industrial Flow（市政供水、工业过程水）。**FY25 Q4 Core +4% / 利润率扩张 +240bp**。
3. **Water Solutions（水处理，25.4% 占比，$1,062.1M，ROS 23.9%，+130bp 但销售 -6.1%）**：**水处理及水品质改善产品** —— 商用 Water Solutions（商业建筑、酒店、餐饮过滤系统）+ Residential Water Solutions（家用软水机、反渗透、过滤器）+ Pool Chemistry（泳池消毒剂 / 化学品）。**Q4 销售 -9.9%**（Core -3.7%，收购/剥离 -7.3%）；全年 -6.1%（Core -2.6%），**Water Solutions 是当前最大拖累项**。

### 2.3 核心收入来源

- **按产品类型**：**Pool 37.3% + Flow 37.2% + Water Solutions 25.4%** 三足鼎立；
- **按利润率**：**Pool 33.8% > Flow 23.3% > Water Solutions 23.9%** —— Pool 为公司核心利润引擎；
- **按客户类型**：**住宅 aftermarket 主导**（Pool + Residential Flow + Residential Water Solutions）+ 商业 + 工业；
- **按地区**：公司定义"全球"，主要收入来自美国 + 欧洲 + 其他（具体区域未在业绩公告披露，10-K 详细披露 US ~70%+、其他区域约 30%）。

**偿债压力判断**

Pentair 偿债压力**非常低**：
- **长期债务 $1,638.6M** + **$1.0B 股份回购剩余授权**；

- **投资级评级**（穆迪 Baa2 / S&P BBB）；
- **Net Debt / Adj EBITDA ≈ 1.3x** 优秀水平；
- 长期债务基本已将短期部分降至 $0（vs FY24 $9.3M）；
- FY26 预计 $800-825M OCF + $60-65M CapEx → FCF 约 $745-775M；
- **50 年连续股息增长**证明长期现金流稳定性。

**商誉风险判断**

商誉 **$3,538.1M**（占总资产 51.5%），主要来自历史收购（nVent 分拆前累积 + 最近 Manitowoc Ice / Manitowoc Foodservice + Aquion + 2025 新收购）。**FY25 新增 $251.5M**（8% 商誉增长），主要来自 FY25 $292M 收购（Pool / Water Solutions 业务增强）。**FY25 无商誉减值**（FY24 也无）。**短期减值风险低**，因：
1. 三大业务板块 Adj ROS 均 23%+，持续盈利；
2. FCF 强劲（$748M）；
3. 但**Water Solutions 疲软**若持续 2-3 年，**该板块商誉**可能面临减值测试压力。

### 4.4 营收分析

**主要增长来源**

1. **Pool +8.5%（贡献 +3.0% 整体增长）**：
   - Core +6.5%（量 + 价 + mix 综合驱动）
   - 收购 +2.0%（FY25 Pool 相关收购）
   - IoT 连网 VSF VariablePro 变频水泵 + Pool IQ automation 自动化产品

2. **Flow +2.6%（贡献 +1.0%）**：
   - Core +0.8%（温和增长）
   - 收购 +0.7% / 货币 +1.1%
   - Residential Flow 渠道优化 + Commercial / Industrial 定价纪律
3. **一次性收购贡献**：FY25 加速收购（$292M vs FY24 $108M）

## 8. 总结判断

**一句话结论**

Pentair plc（PNR）是**全球水资源技术领导者**（S&P 500 Dividend Aristocrat + 2026 年迎来 60 周年 + **50 年连续股息增长**），FY25 营收 $4.18B（+2.3%）+ Adj ROS 扩张 +170bp 至 25.2% + Adj EPS $4.92（+13.6%，创新高）+ FCF $748M（创新高）；**Pool 板块 +8.5% + Flow +2.6% / +230bp ROS + Pentair Business System + Transformation 三驱动** 支撑 FY26 Adj EPS +7-10% 增长指引；**Water Solutions -6.1% 疲软**是当前最大拖累；适合**寻求"稳健复利 + 连续股息增长 + 水资源长期主题"的价值型成长投资者**（核心风险：Water Solutions 延续疲软 + 关税 + 收购整合）。

**核心看点**

1. **FY26 Adj EPS 指引 $5.25-$5.40（+7-10%）+ Adj ROS 继续扩张（~26%）+ 50 年连续股息增长（2026 升 8% 至 $1.08）+ Investor Day（2026/03/04）**：可能催化长期目标（2027-2030 Adj ROS 27-28%、Adj EPS CAGR 8-10%）+ Multiple 重估。
2. **Pool 业务 33.8% ROS + 持续 8%+ 增长 + IoT / aftermarket 双驱动 + Pool 双寡头市占 70%**：如按 Pool Corp / Hayward 对标，仅 Pool 板块估值约 $10B，占 Pentair 当前市值 60%，其余 Flow + Water Solutions 则可获得行业估值补全；Flow 利润率扩张 +230bp 至 23.3% 加速贡献 EBIT。
3. **创纪录 FCF $748M（+8%）+ OCF / Net Income 转化率 125% + $1.0B 回购授权 + 投资级评级（Baa2/BBB）+ Net Debt/EBITDA 1.3x**：FY25-FY28 每年预计 $700-850M FCF 支持 $250-350M 回购 + $170-200M 股息 + $300-400M 收购，形成稳健资本返还 + 有机增长 + Bolt-on 并购的复利机器。

**核心风险**

1. **Water Solutions 持续疲软（FY25 -6.1% / Core -2.6%，Q4 商业 -15%）+ 管理层重组需 2-3 季度见效 + 可能 FY26 下半年才转正**：可能引发管理层下修指引或商誉减值压力（该板块商誉 $800-1000M）。
2. **关税 / 供应链 / 住房与利率敏感度**：2026 特朗普关税政策如对中国零部件加 20-30%，可能短期压制 Adj GM 100-200bp；美国房贷利率高位可能延缓住宅 Pool / Flow 新订单增长；铜 / 铝 / 钢原材料价格波动。
3. **收购加速整合风险（FY25 $292M，商誉 $3.5B，占资产 51.5%）+ 估值 P/E 19-22x + FY26 +8% EPS 增长低于 S&P 500 +10-12%**：相对增长劣势 + 宏观 risk-off 可能触发 multiple 从 20x 压缩至 17-18x → 股价 -15-20%；AI / 生物药等高增长板块分流资金。
