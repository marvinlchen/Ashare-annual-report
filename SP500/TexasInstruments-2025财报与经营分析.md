## 1. 业绩总结

### 1.1 收入与利润表现

- 全年总收入 **$176.82 亿美元**，同比 **+13%**（FY24 $156.41 亿，+$20.41 亿）—— 半导体周期复苏

- **R&D $20.83 亿美元**，同比 **+6.3%**（FY24 $19.59 亿）

- **Operating Profit $60.23 亿美元**，同比 **+10.2%**（FY24 $54.65 亿，+$5.58 亿）
- **Operating Margin 34.1%**（vs FY24 34.9%，-80 bps）
- **Other Income Net (OI&E) $2.30 亿美元**（vs FY24 $4.96 亿，-54%）—— 现金投资收益下降
- **利息支出 $5.43 亿美元**（vs FY24 $5.08 亿，+7%）—— 新发债增加
- **税前利润 $57.10 亿美元**，同比 **+4.7%**（FY24 $54.53 亿）
- **所得税 $7.09 亿美元**（有效税率 12.4% vs FY24 12.0%）—— CHIPS Act ITC 持续

### 1.2 增长与拖累因素

- **主要增长驱动**：① **工业 + 汽车半导体周期复苏**：FY24 工业市场经历深度库存调整后，FY25 Q1 开始逐步复苏，Q2-Q3 加速，Q4 季节性放缓但 YoY +10%；② **Analog 板块 +15%**（$14.006B）：核心业务强劲，Op Income +17%（$5.412B），Op Margin 38.6%（+70 bps）；③ **Analog Q4 YoY +14% / Op Income +13%**：持续高增长；④ **Embedded Processing +6%**（$2.697B）：Embedded Q4 YoY +8%；⑤ **Other +3%**（$979M）：小幅增长；⑥ **FCF +96%**：FCF $29.38 亿 vs $14.98 亿，FCF Margin 从 9.6% 升至 16.6%；⑦ **CFFO +13% 至 $71.53 亿**：CFFO Margin 40.5% 持续高位，证明 300mm 制造优势；⑧ **Capex -6%**：可能已过 Capex 峰值（FY24 $48.20 亿 → FY25 $45.50 亿 → FY26 $45-50 亿预期），未来 FCF 显著改善空间；⑨ **股票回购 +59% 至 $14.77 亿**：管理层对估值信心回归；⑩ **股息 +4.6% 连续 22 年加息**：可靠股东回报；⑪ **CHIPS Act ITC $670M**（+$82M vs FY24）：政府激励持续；⑫ **Haviv Ilan 战略延续**：保持 Rich Templeton 的资本回报 + 300mm 长期战略
- **主要拖累因素**：① **GAAP Gross Margin -110 bps（58.1% → 57.0%）**：300mm 新工厂折旧 + 产能爬坡 + SM1 / RFAB2 投产初期成本；② **Op Margin -80 bps**：GM 下降 + Restructuring $117M 一次性（FY24 $(124)M 基数反差）；③ **Embedded Processing Op Margin -260 bps**（FY24 13.9% → FY25 11.3%）：Embedded 新平台转型过渡期，Op Income $304M -14% YoY；④ **Other Op Income -39%**（$307M vs $505M）：DLP + 计算器 + 传统业务毛利率下降；⑤ **Other Revenue Q4 -34%**：一次性业务波动；⑥ **OI&E -54%**（$230M vs $496M）：现金 + 短期投资收益下降，因为现金余额 -36%（$48.81 亿 vs $75.80 亿）；⑦ **Q4 EPS -2%**（$1.27 vs $1.30）：Q4 含 $0.06 非 guidance 减少；⑧ **Q1 FY26 Guidance $4.32-4.68B / EPS $1.22-1.48**：中位 $1.35 vs FY25 Q1 实际 $1.28 → YoY +5.5% 但 QoQ 基本持平，反映半导体周期温和复苏；⑨ **利息支出 +7%**：$5.43B 绝对值仍高，长期债务 $13.548B；⑩ **Inventory +6% 至 $4.804B**：生产扩张但需谨慎避免过剩；⑪ **现金流 / 投资 -36% YoY**（$48.81B vs $75.80B）：资本返还超过 CFFO（$64.76B vs $71.53B）
- **一次性/异常项目**：① **Restructuring / Other $117M**（FY25 Q1-Q4 合计）：Q4 $32M + Q3 $35M（估）+ 此前小额：新厂房 / 业务重组；② **FY24 -$124M 含 $127M 资产出售收益**：比较基数差异 $241M 正面影响（但 FY25 基数比 FY24 低）；③ **CHIPS Act ITC $335M**（FY25 vs FY24 $588M）—— 政府激励下降 $253M；④ **CHIPS Act Proceeds $335M cash**（FY25 新增）vs FY24 $0；⑤ **Discrete Tax Items -$126M**（FY25 税收抵免）vs FY24 -$89M，+$37M 正面；⑥ **新发债 $12 亿**（FY25 新增长期债）vs $29.80 亿（FY24）；⑦ **TI 股票回购 $14.77 亿**（vs FY24 $9.29 亿，+59%）—— 战略性提高

### 1.3 趋势判断

- **未来主要利好方向**：① **工业 + 汽车半导体超级周期**：FY24 触底后，FY25-28 持续 +10-15%/年 增长，TI Industrial 占 70%+ + Automotive 占 17%；② **300mm 产能扩张完成 + Capex 周期下降**：SM1 / SM2 / LFAB1 / LFAB2 / RFAB2 投产后，FY27-28 Capex 从 $45-50 亿降至 $30-35 亿，**FCF 可能达 $80-100 亿**；③ **Analog 业务持续扩张**：FY25 +15% → FY26-27 +10-12%/年，占比从 79% 升至 82%+；④ **Capital Management Plan 资本返还**：FY25 股东回报 $64.76 亿 = 11 年累计 $550 亿+，FY26-28 可能 $70-80 亿/年；⑤ **CHIPS Act 持续激励 + Trump 政府制造业回流**：未来 3 年预计获得 $15-20 亿 CHIPS Act Direct Funding（+$5 亿以上 ITC）；⑥ **AI 加速+EV+ 工业自动化**：AI Server 电源 (AVP $8-20/Server) + EV 电动化（每辆 $500-1,000 半导体）+ 工业 IoT；⑦ **China AI + 工业本地化**：TI 在中国市占持续上升（~35%），华为 / 比亚迪等扩张带来的需求；⑧ **股息连续 22 年加息 + 回购加速**：未来 2-3 年可能 $20-30 亿/年回购；⑨ **CEO Haviv Ilan 延续战略**：保持长期资本回报 + 300mm 战略；⑩ **FY26-28 Op Margin 恢复 36-40%**：Capex 下降 + 折旧绝对值峰值过去；⑪ **Embedded Processing 重塑后 FY26-27 OP Margin 回到 15%+**；⑫ **AI Server 电源芯片 AVP**（Advanced Voltage Regulator）：FY25 新产品加速

## 2. 主营业务

### 2.1 公司简介

Texas Instruments Incorporated（TXN）是**全球最大的模拟半导体 (Analog) 公司**，总部位于德克萨斯州 Dallas。**1930 年由 Geophysical Service Incorporated (GSI) 创立**（最早是地球物理勘探公司），1951 年更名 Texas Instruments，至今已有 **96 年历史**。**1958 年 Jack Kilby 发明集成电路 (IC)**（诺贝尔物理学奖 2000 年），是半导体行业的先驱之一。公司是**全球模拟 + 嵌入式半导体龙头**，业务涵盖**模拟芯片 (Power Management + Signal Chain) + 嵌入式处理器 (MCU + SoC) + DLP 数字光学处理器 + 计算器**。**全球员工 34,000+**，**服务 100,000+ 客户**（分销 + 直接），**产品组合 80,000+ SKU**。**客户分布**：工业 70%+、汽车 17-20%、个人电子 10-15%、通信 / 企业计算 3-5%。**制造优势**：**300mm 晶圆制造**（行业领先，模拟芯片领域少有 300mm 能力）、**80% 内部制造**、**50% 封测内部**。**主要工厂**：**Sherman Fab 1 (SM1，2025 投产)**、Sherman Fab 2 (SM2，2026 H2 投产)、Lehi Fab 1 (LFAB1，2023 投产)、Lehi Fab 2 (LFAB2，2026-2027)、Richardson Fab 2 (RFAB2，2022 投产)、DFAB、DMOS5、DMOS6 等。**Chair + CEO**：**Haviv Ilan**（2024/04 接任 Rich Templeton，先前任 COO）；**CFO + SVP**：**Rafael Lizardi**（2013 年加入 TI）；**Chair (Retired CEO)**：**Rich Templeton**（2004-2024 年任 CEO，20 年任期）。**2025 年底市值约 $1,800-1,900 亿美元**（2025 年股价持平）。**核心战略**：**Capital Management Plan** —— 通过 300mm 制造优势 + Analog 长期增长趋势 + 严格资本返还，实现持续复利。

## 6. 未来可能的利好

**依据**

  - 电动汽车每辆半导体价值 $1,000（vs 传统车 $300-500）
  - 美国 + 欧洲制造业回流（CHIPS Act + 欧洲 Chips Act）
- **TI 300mm 制造优势**：
  - **300mm 晶圆模拟芯片生产**：行业 TI 独有优势（ADI / Infineon / ST 仍以 200mm 为主）
  - **300mm vs 200mm 成本优势**：每颗芯片制造成本 -40%
  - **SM1 (Sherman Fab 1)**：2025 投产，满产能 $10 亿/年
  - **SM2**：2026 H2 投产
  - **LFAB1 (Lehi)**：2023 投产，2024 年爬坡
  - **LFAB2**：2026-2027 投产
  - **RFAB2 (Richardson)**：2022 投产
  - **总计 6 个 300mm Fab**：全球最大模拟半导体制造网络
- **Capex 周期尾声**：
  - FY23 Capex $50.71 亿 → FY24 $48.20 亿 → **FY25 $45.50 亿**（-6%）
  - FY26 预期 $45-50 亿（持平 / 略下降）
  - FY27 预期 $35-40 亿（开始下降）
  - **FY28-30 Capex 稳定在 $25-35 亿/年**
- **FCF 改善**：
  - FY23 FCF $14.05 亿 → FY24 $14.98 亿 → **FY25 $29.38 亿**（+96%）
  - **FY26 FCF $40-50 亿预期**（Capex 持平 + CFFO 增长）
  - **FY27 FCF $50-70 亿预期**（Capex 下降 + CFFO 继续增长）
  - **FY28 FCF $80-100 亿预期**

## 7. 未来可能的利空

### 利空 1：Gross Margin 持续压力 + 300mm 折旧峰值 + Capex 周期不确定

**依据**

- **Gross Margin 下降趋势**：
  - FY22 67.6% → FY23 59.2% → FY24 58.1% → FY25 57.0%（**每年 -100-200 bps**）
  - FY25 H2 特别承压（Q4 56%+）
- **300mm 新工厂折旧爆发**：
  - FY23 折旧 $1,113M → FY24 $1,508M → **FY25 $1,918M (+27%)**
  - FY26 预期 $2,300-2,500M
  - FY27 预期 $2,800-3,200M（峰值）
  - FY28 可能开始下降
- **产能爬坡成本**：
  - SM1 / LFAB1 / RFAB2 爬坡期毛利率 < 50%
  - 产能利用率逐步提升（从 60-70% → 85-90%）
- **Capex 周期不确定**：
  - FY25 -6% 是首次下降，但 FY26 可能重新上升（SM2 投产 + LFAB2 启动）
  - **FY25-27 Capex 可能持续 $45-50 亿/年**
  - 如果 FY28 继续高位，FCF 改善幅度低于预期
- **产能过剩风险**：
  - 如果工业 + 汽车复苏不及预期，TI 可能面临 300mm 产能过剩
  - 产能利用率下降 10pp = GM -300-400 bps
- **SM2 + LFAB2 推迟风险**：
  - 如果 SM2 / LFAB2 投产推迟，FY27-28 增长可能受限
- **中国本土制造替代**：
  - 中芯国际 (SMIC) 300mm 模拟半导体扩张
  - 比亚迪半导体 + 华为 HiSilicon 扩张
  - 长期可能侵蚀 TI 中国市占

**解读**

**TI Gross Margin 持续压力是未来 3-5 年最大风险**：① **GM 每年 -100-200 bps**：从 FY22 67.6% 降至 FY25 57.0%（-10.6pp 已经发生），FY26-27 可能继续降至 55-56%；② **300mm 折旧 FY25-27 爆发**：从 FY24 $1.5B 升至 FY27 $3B+，累计 $1.5B/年 GM 压力；③ **产能利用率风险**：如果工业 / 汽车复苏不足，产能 70-75% 利用率 = GM 只能 55-56%；④ **Capex 周期延长**：FY25 -6% 不代表峰值已过，SM2 + LFAB2 可能使 FY26-27 Capex 继续 $45-50 亿；⑤ **FY28-29 才开始 FCF 大幅改善**：假设 Capex 降至 $30-35 亿 + CFFO 升至 $90-100 亿 → FCF $55-70 亿；⑥ **中国制造替代 + 地缘政治**：中芯国际 + 比亚迪半导体扩张，长期 TI 中国市占可能下降；⑦ 如果 FY26 GM 降至 55%，Op Margin 可能降至 32-33%，对 EPS 负面影响 -5-10%。

## 8. 总结判断

**一句话结论**

**Texas Instruments FY2025 交出"Revenue $176.82 亿 (+13% 周期复苏)、Analog $140.06 亿 (+15%) + Op Income $54.12 亿 (+17%) 强劲核心、Embedded Processing $26.97 亿 (+6% 但 Op Income -14%)、GAAP EPS $5.45 (+4.8%)、CFFO $71.53 亿 (+13% Margin 40.5%)、FCF $29.38 亿 (+96% Margin 16.6%)、Capex -6% 至 $45.50 亿 (可能已过峰值)、股东回报 $64.76 亿 (股息 $50 亿连续 22 年加息 + 回购 $14.77 亿 +59% 激活)"的周期拐点 + Capex 尾声优秀业绩**，**充分体现"Analog 核心业务重回增长（79% 收入）+ 300mm 制造优势持续扩大（6 个 Fab）+ CHIPS Act 激励 $670M + Capex 周期进入尾声 + 股票回购激活 + 连续 22 年股息加息 + Capital Management Plan 长期复利"的全球模拟半导体龙头优势**。**FY26 核心看点是 Analog 继续 +10-12% 增长 + Embedded Processing 重塑 Op Margin 回到 15%+ + Capex 平稳下降至 $40-45 亿 + FCF 改善至 $40-50 亿 + AI Server AVP 加速（Nvidia B100/Rubin 电源）+ 股东回报 $70-80 亿 + 股息连续 23 年加息**；**核心风险是 Gross Margin 继续下降（57.0% → 可能 55-56%）+ 300mm 折旧峰值 FY26-27（$2.5-3B/年）+ 中国地缘政治 + 出口管制 + 竞争加剧（ADI + Infineon + 中国本地 SMIC / 比亚迪）+ 工业库存再次调整（Q4 QoQ -7% 信号）+ 汽车 EV 放缓 + Forward PE 32-33x 溢价 50%+ 估值压力**。

**核心看点**

1. **工业 + 汽车半导体超级周期 + 300mm 制造优势 + Capex 周期尾声 = FCF 爆发**：FY28 FCF 可能 $80-100 亿
2. **Capital Management Plan + 股息连续 22 年加息 + 回购激活 +59% + 股东回报 $65 亿**：长期复利典范
3. **AI Server AVP + EV + 工业 4.0 + Embedded Processing 重塑 + CEO Haviv Ilan 战略延续**：多引擎驱动

**核心风险**

1. **Gross Margin 持续压力（FY25 57% → FY27 55-56%）+ 300mm 折旧峰值 + Capex 周期不确定**：盈利能力结构性承压
2. **Embedded Processing 盈利压力 + Other 结构性下降 + 工业库存再次调整 + 汽车 EV 放缓**：FY26 增速可能放缓
3. **中国地缘政治 + 出口管制 + 中国本土替代（SMIC + 比亚迪 + HiSilicon）+ 竞争加剧（ADI + Infineon）+ Forward PE 32-33x 溢价 50%+ 估值压力**：长期份额 + 估值双重风险
