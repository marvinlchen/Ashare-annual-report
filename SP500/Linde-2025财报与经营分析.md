# Linde plc (LIN) 2025年财报整理

> 报告期：FY2025 全年（12 个月，止 2025/12/31）—— 财年与日历年一致
> 申报币种：美元；会计准则：U.S. GAAP，并披露 Non-GAAP（Adjusted Operating Profit / Adjusted EPS / Adjusted EBITDA / FCF / ROC / 扣除 Linde AG 并购购买会计影响）
> 数据来源：Linde Q4 & Full-Year 2025 Earnings Release（2026/02/05 发布）+ 8-K + 10-K
> FY25 期间主要战略事件：**CEO Sanjiv Lamba 第 4 个完整年 + ROC 达 24.2%** + **项目储备 $100 亿美元（其中销售天然气合同储备 $71 亿）+ 扩产以电子 / 清洁氢为主**、**全球重组费用 $2.73 亿（其中 Engineering 业务大规模重组 $2.55 亿 Q4 单季）+ 剥离小型业务**、**股东回报 $74 亿（$46 亿回购 + $28 亿股息）**、**Capex $53 亿（+17%）支持合同储备交付**、**股息每股 $6.00（+7.9%）连续第 32 年增长**
> 关键 Non-GAAP 事项：**Linde AG 并购购买会计调整 $9.41 亿**（2018 年合并遗留，其中 $7.77 亿折旧摊销 + $1.64 亿其他），是 GAAP vs Adj 差异的主要来源；**Cost reduction 程序 $2.73 亿**（Q4 集中 $2.29 亿）

## 1. 业绩总结

### 1.1 收入与利润表现

- 全年总收入 **$339.86 亿美元**，同比 **+3%**（FY24 $330.05 亿）—— 其中 +2% 价格 + 1% 补强并购，销量持平
- 底层销量（underlying sales）同比 **+2%**（2% 价格 + 0% 销量）
- **GAAP 营业利润 $89.23 亿美元**，同比 **+3.3%**（FY24 $86.35 亿）
- **Adjusted 营业利润 $101.37 亿美元**，同比 **+4.3%**（FY24 $97.20 亿）
- **GAAP 营业利润率 26.3%**（+10 bps）；**Adjusted 营业利润率 29.8%**（+30 bps），创历史新高
- **GAAP 净利润 $68.98 亿美元**，同比 **+5.1%**（FY24 $65.65 亿）
- **Adjusted 净利润 $77.72 亿美元**，同比 **+4.0%**（FY24 $74.75 亿）
- **GAAP 摊薄 EPS $14.61**（+7.3%）；**Adjusted EPS $16.46**（+6.1%）
- **Adjusted EBITDA $133.51 亿美元**（+4.1%），Margin 39.3%（+50 bps）
- **经营现金流 $103.50 亿美元**（+9.8%）
- **自由现金流 $50.89 亿美元**（+4.3%）
- 股东回报 **$74 亿美元**（$46.01 亿回购 + $28.11 亿股息）
- 股本 -2.1%（Diluted 472.2M vs FY24 482.1M）
- Q4 FY25：Revenue $87.64 亿（+5.8%）、GAAP EPS $3.26（-9.4%）、Adj EPS $4.20（+5.8%）、Adj Op Margin 29.5%（-40 bps）

### 1.2 增长与拖累因素

- **主要增长驱动**：① **价格实现 +2%（全年）/ +2%（Q4）**：定价能力强大持续；② **项目启动贡献销量 +1% Q4**：主要来自电子 + 清洁能源新项目上线；③ **Americas 全年收入 +5.3% 至 $152.08 亿**：电子行业需求 + 价格上升；④ **EMEA 利润率 35.7%（+240 bps）**：成本效率大幅改善 + 节约举措见效；⑤ **补强并购 +1%**：FY25 投入 $4.12 亿用于 bolt-on acquisitions；⑥ **项目储备 $100 亿美元**：其中 $71 亿为合同销售天然气储备（长约 15+ 年）；⑦ **电子（半导体）行业需求增长**：特种气体 + 超高纯度气体用于芯片制造；⑧ **生产力举措（Productivity initiatives）**：跨所有细分地区持续改善成本结构
- **主要拖累因素**：① **EMEA Q4 销量 -3%**：化工 / 能源领域终端市场需求疲软；② **Engineering 业务销量 -2% Q4 / -3.1% FY**：订单减少 + Q4 单季 $4.34 亿订单（环比下滑）+ 第三方销售设备储备降至 $27 亿；③ **Engineering 大规模重组**：Q4 单季严重 $2.55 亿遣散费，主要针对 Engineering 业务；④ **Capex 增长 +17% 至 $53 亿**：合同销售气体储备交付所需 + 短期 FCF 承压；⑤ **净债务 +31% 至 $220 亿**：支持 Capex 扩张 + $74 亿股东回报；⑥ **Adjusted ROC -170 bps 至 24.2%**（FY24 25.9%）：资本支出增加导致资本基数扩大；⑦ **利息费用 +持平** $2.55 亿：债务增加但利率环境中性
- **一次性/异常项目**：① **Cost reduction 程序 $2.73 亿**（FY25 全年，税后 $1.98 亿）：包括全球遣散 $3.08 亿 - 剥离 $3500 万 benefit；Q4 集中 $2.29 亿（Engineering 重组为主）；② **Linde AG 并购购买会计调整 $9.41 亿**（无现金流，2018 合并遗留）：包括 $7.77 亿折旧摊销 + $1.64 亿其他费用 - $16 亿税前税收回馈；③ **Pension settlement 费用 $2 百万美元**（可忽略）；④ Q4 其他费用净额 $(1.05) 亿美元（FY24 Q4 为 $7,400 万）vs FY25 Q4 下降主要是 Linde AG merger-related 购买会计

### 1.3 趋势判断

- **未来主要利好方向**：① **项目储备 $100 亿**：合同销售天然气 $71 亿将在未来 2-5 年转化为收入，提供高可见度 + 稳定增长；② **清洁氢 / CCUS 机会**：IRA（美国 Inflation Reduction Act）+ 全球脱碳趋势，Linde 是氢能 leader；③ **电子行业（半导体）气体需求**：AI 芯片 + 先进制程 + 面板产能扩张推动 UHP gases + specialty gases 长期需求；④ **ROC 24.2% 全球最佳**：资本配置纪律 + 网络密度优势使 Linde 持续高 ROIC；⑤ **2026 EPS Guidance +6-9%**：连续 10+ 年兑现 10%+ 复合增长；⑥ **Capex $5.0-5.5 亿美元**：继续合理投入，支持储备项目交付；⑦ **连续 32 年股息增长（Dividend Aristocrat / King 标的）**：股东友好承诺；⑧ Engineering 业务重组后 FY26 利润率预计改善
- **未来主要压力方向**：① **EMEA 化工/能源行业需求疲软**：欧洲经济持续弱势，高能源成本削弱工业需求；② **Engineering 业务持续下滑**：订单 $4.34 亿 Q4 + 第三方储备 $27 亿（历史低位）；③ **关税 / 贸易冲突风险**：Trump 关税政策可能影响全球供应链 + 客户资本支出；④ **高利率持续**：$260 亿美元总债务 + 5-6% 再融资成本增加利息费用；⑤ **汇率波动**：美元走弱为 2026 EPS 提供 3% 顺风，但也意味着强美元会逆转；⑥ **ROC 下降 170 bps**：Capex 扩张可能使 ROC 继续下降；⑦ **Hydrogen Hub 商业化进度缓慢**：美国 7 个 Hydrogen Hub 项目资金政治风险
- **整体趋势判断**：**Linde FY2025 交出"收入 $340 亿 (+3%)、Adj EPS $16.46 (+6%)、Adj Op Margin 29.8% (+30 bps) 创新高、ROC 24.2% 全球最佳、项目储备 $100 亿创新高、Capex $53 亿 (+17%)、股东回报 $74 亿"的稳健业绩**，**体现"工业气体龙头 + 高壁垒 + 高 ROIC + 强现金流"的优质特质**。**FY26 核心看点是项目储备交付 + 电子 / 清洁氢驱动 + 2026 EPS Guidance +6-9% + 股息连续第 33 年增长 + Engineering 重组后利润率改善**；**核心风险是 EMEA 化工需求疲软 + Engineering 业务持续下滑 + 净债务 +31% + ROC 下降 + 高利率 + 全球贸易 / 关税风险**

## 2. 主营业务

### 2.1 公司简介

Linde plc（LIN）是**全球最大的工业气体公司**（按收入、利润、市值综合排名），总部位于英国沃金（Woking, UK），同时在纽约纳斯达克（LIN）和法兰克福证券交易所（LIN）双重上市。公司主要产品：**氧气（O2）、氮气（N2）、氩气（Ar）、二氧化碳（CO2）、氢气（H2）、氦气（He）、特种气体、工业气体（乙炔、氩气等）+ 工程与流程解决方案**。**2018 年由 Praxair 和 Linde AG 50:50 换股合并**，形成当前 Linde plc 架构，CEO：**Sanjiv Lamba**（2022 年 3 月上任）。**服务 10+ 个终端市场**：化学品 & 能源、食品 & 饮料、电子（半导体）、医疗保健、制造业、金属 & 采矿等。全球运营 **80+ 个国家**，**4 大地区报告段**（Americas、EMEA、APAC、Engineering）+ Other。**Linde 是 Dividend Aristocrat / King 标的**，**连续 32 年提升股息**（包括 Praxair 历史）。**2025 年底总市值约 $2,100 亿美元**。

### 2.2 主营业务构成

按 4 大地区段 + Other + Engineering 划分（FY25）：

| 业务段 | 主要内容 | 终端市场 |
|---|---|---|
| **Americas** | 美国、加拿大、墨西哥、南美 | 化工 / 能源 / 电子 / 医疗 / 制造 |
| **EMEA** | 欧洲、中东、非洲 | 化工 / 能源 / 食饮 / 医疗 / 制造 |
| **APAC** | 中国、印度、韩国、日本、东南亚、澳新 | 电子 / 化工 / 金属 / 制造 |
| **Engineering** | 工厂设计 / 建造 / 模块化项目 | 氢气、LNG、气体分离、合成气 |
| **Other** | Global Helium business + 企业中心 | 医疗、电子、特种气体 |

### 2.3 核心收入来源

按产品类型细分（管理层估计）：

- **On-Site（管道气 + 大气分离装置）约 35% 收入**：长期合同（15-20 年）+ 稳定现金流 + Take-or-Pay 条款 → 高可见度
- **Bulk / 液态气体（罐装运输）约 35% 收入**：中短期合同（1-5 年）+ 区域分发网络（Linde 网络密度全球最强）
- **Packaged Gas（钢瓶气体）约 20% 收入**：小客户 + 零售网络 + 本地化定价
- **Engineering 约 7% 收入**：气体工厂设计 / 建造 / 工艺技术转让
- **Other（Helium + 企业中心）约 4% 收入**：高附加值特种气体

按终端市场细分：

- **Chemicals & Energy 约 35-40%**：石化 + 炼化 + LNG + 清洁氢
- **Electronics / Semiconductor 约 15%**：超高纯度 + 特种气体（FY25 增长 +10%+）
- **Food & Beverage 约 10%**：保鲜 + 碳酸化 + 冷冻
- **Healthcare / Medical 约 10%**：医用氧气 + 麻醉气体
- **Manufacturing / Metal / Mining 约 25%**：焊接 + 切割 + 钢铁冶炼

## 6. 未来可能的利好

### 利好 1：项目储备 $100 亿美元（$71 亿合同销售气体）+ 可见度 2-5 年 + 电子 / 清洁氢新项目管道

**依据**

- **全年项目储备 $100 亿美元**（创历史新高）：① 销售气体合同储备 $71 亿美元（15-20 年期长约）；② 第三方销售设备储备 $27 亿美元（Engineering 段）；③ 其他 $2 亿美元
- **Q4 FY25 签署新合同 + 项目启动贡献销量 +1%**
- **2026 Capex Guidance $5.0-5.5 亿美元**：继续支持合同储备交付
- **清洁氢 / CCUS 领域 Linde 为全球龙头**：
  - 已签署多个 20 MW-500 MW 级别绿氢项目（美国 + 欧洲 + 中东）
  - OCI Global 蓝氢项目（美国德州）
  - Woodside Clean Hydrogen 项目（澳洲 + 美国）
  - ExxonMobil Baytown 低碳氢 500+ tpd 项目
- **电子 / 半导体需求**：
  - TSMC、Samsung、Intel、Micron 等芯片厂产能扩张推动 UHP gases 需求
  - AI 芯片（HBM + CoWoS）使特种气体消耗量增加
  - Linde 是全球电子气体第一梯队（与 Air Liquide 齐名）
- **长期合同**：15-20 年 "Take-or-Pay" 条款，无论用户用量如何，最低收入有保障
- 2025-2027 全球半导体工厂建设超 40 个，Capex 累计 $6,000 亿美元+

**解读**

**Linde 的商业模式是工业界最具护城河的 subscription economy**：① **长约 + 网络密度 + 规模效应**构成三重护城河；② **$71 亿销售气体合同储备**意味着未来 2-5 年将有 $71 亿额外收入叠加，带来 **3-5% 额外销量增长 / 年**；③ **清洁能源 / 氢能**是长期（10-20 年）故事，但近期（2026-2028）已有可见性的项目启动；④ **电子行业**是 Linde 最核心的高增长领域（FY25 +10%+），**半导体扩产 + AI 芯片 + 先进制程**都直接利好 Linde specialty gases 销售；⑤ 即使全球经济衰退，Linde 凭借长约 + 医疗 / 食饮 / 电子等"抗周期"业务，业绩波动远低于其他周期股；⑥ **管理层对 2026 EPS 指引 +6-9%**（即使假设 EUR / USD 反转），体现储备项目转化信心。

### 利好 2：Adjusted Op Margin 29.8% 创新高 + ROC 24.2% 全球最佳 + 成本纪律 + Engineering 重组后改善

**依据**

- **Adjusted Op Margin 29.8%**（+30 bps）**创历史新高**
- 10 年 Adjusted Op Margin 变化：FY15 约 20% → FY20 约 25% → FY25 29.8%
- **Adjusted EBITDA Margin 39.3%**（+50 bps）
- **Adjusted ROC 24.2%**（全球工业气体公司最高，Air Liquide 约 10-11%，Air Products 约 10-12%）
- **EMEA Op Margin 35.7%（+240 bps）**：成本节约举措 + 组合优化 + 生产力
- **销售成本增长 +1.4% 远低于收入 +3.0%**：成本纪律优秀
- **Cost reduction 程序 FY25 $2.73 亿**：主要 Engineering 业务重组（Q4 $2.55 亿遣散）
- **FY26 预计 Engineering 利润率改善**：重组后固定成本下降
- **网络密度**：美国 + 欧洲 + 中国等地 Linde 拥有密集的气体管道 + 液态气体罐车网络，后进入者难以复制
- CEO Sanjiv Lamba："Operating margins expanded to 29.8%, ROC reached 24.2% and EPS grew 6%. These results underscore the strength of our operating model and the exceptional execution by the Linde team worldwide."

**解读**

**Linde 的"运营效率 + 资本纪律"是业内标杆**：① **29.8% 的 Adj Op Margin 处于工业气体行业绝对领先位置**：同行 Air Liquide 约 19%、Air Products 约 21%、Nippon Sanso 约 11%，Linde margin 领先 8-18 个百分点；② **ROC 24.2% 在 S&P 500 中也是前 10%**：表明公司对股东资本的使用效率极高；③ **Margin 逐年提升**：从 2018 年合并后 20% → 2025 年 29.8%，7 年提升 约 10pp，主要来自：规模效应 + 合并协同 + 定价 discipline + 重组举措；④ **EMEA +240 bps 的 margin 扩张**证明即使在弱势市场环境下也能通过运营优化提升盈利能力；⑤ **Engineering 重组**虽然短期产生 $2.55 亿一次性费用，但 FY26+ 将带来结构性 margin 改善；⑥ 长期看 Linde 的 Adj Op Margin 有望突破 30%（公司内部目标），即达 $120 亿级别 Op Profit。

### 利好 3：股东回报 $74 亿 + 股息连续 32 年增长 + 2026 EPS Guidance +6-9% + 资本纪律

**依据**

- **FY25 股东总回报 $74.12 亿美元**（$46.01 亿回购 + $28.11 亿股息）
- **年度股息每股 $6.00**（+7.9%），连续第 32 年股息增长（包括 Praxair 历史）
- Dividend Aristocrat / King 标的（美股股息连续 25+ 年增长列入 Aristocrat，50+ 年为 King）
- **FY25 股本 -2.1%**：回购 $46.01 亿（vs FY24 $44.82 亿）
- **2026 Adj EPS Guidance $17.40-17.90**（+6-9%）+ 1Q26 $4.20-4.30（+6-9%）
- **2026 Capex Guidance $5.0-5.5 亿美元**：维持合理水平
- CEO：Linde "is well positioned to capture high-quality wins in 2026 and continue to create shareholder value regardless of macroeconomic uncertainties"
- 股息支付率（$6.00 / $16.46）= 36.4%，留下大量现金用于 Capex + 回购 + M&A
- 调整后净债务 / Adj EBITDA = 1.64x（健康水平）
- 10 年 EPS CAGR 约 12%（包括 Praxair 历史）
- A / A2 信用评级（标普 / 穆迪），借款成本低

**解读**

**Linde 是 "Growth + Quality + Income" 的典范标的**：① **$74 亿资本回报**相当于当前市值 $2,100 亿的 **3.5%**，股息 + 回购 + EPS 增长（内生）提供长期 double-digit 股东总回报；② **股息 32 年连续增长**是美股 Top 20 的稀缺标的（Dividend Aristocrats 内部 Tier 1），在经济衰退 / 利率走高 / 通胀等任何环境下都保持增长纪录；③ **2026 EPS Guidance +6-9%** 体现管理层对项目储备 / 电子 / 清洁氢交付的信心；④ **10 年 EPS CAGR 12%**（包括 Praxair 合并前）+ **Dividend Yield 1.2% + Buyback Yield 2.2% ≈ 总股东回报 15%/年**：Linde 是典型 "compounder" 标的；⑤ **估值角度**：当前 FY26 P/E 约 25x，相对 10 年历史均值 22x 略有溢价，但反映：产品组合改善 + 电子比重提升 + 清洁氢期权；⑥ **Linde 在过去 10 年的累计总回报（价格 + 股息 + 回购）超过 600%**，远超 S&P 500 约 250%，充分体现 "工业界守护神" 的长期价值。

## 7. 未来可能的利空

### 利空 1：EMEA 化工 / 能源需求疲软 + Engineering 业务持续下滑 + 欧洲经济结构性压力

**依据**

- **EMEA Q4 FY25 销量 -3%**：化工 / 能源终端市场需求疲软
- **EMEA Q4 销售同比虽 +6%（货币 +7% 提振），但底层 -2%**
- **Engineering FY25 收入 -3.1% 至 $22.50 亿**：订单减少
- **Engineering Q4 订单仅 $4.34 亿美元**（历史低位）
- **Engineering 第三方销售设备储备降至 $27 亿**（从 FY24 $30+ 亿）
- **Engineering 重组 Q4 $2.55 亿遣散费**：大规模裁员应对需求下滑
- **欧洲工业产出 2025 年持续弱势**：
  - 德国 2025 年 GDP 0% 增长
  - 欧元区制造业 PMI 多月处于收缩区间（45-48）
  - 高能源成本使化工 / 钢铁 / 金属工业竞争力下降
  - 俄乌战争 + 中东冲突使欧洲能源进口成本高企
- **EMEA 的终端市场结构**：
  - 化工 + 能源占 EMEA 收入约 40%（最大客户行业）
  - 德国、意大利、法国化工工业萎缩
  - 英国 / 欧盟工业电价是美国的 2-3 倍
- BASF、Covestro、Arkema 等欧洲主要化工客户持续削减产能

**解读**

**欧洲工业结构性压力是 Linde 的长期拖累因素**：① **EMEA 占 Linde 总收入 25.2%**，欧洲经济疲软直接影响 Linde 业绩；② **化工 / 能源行业"去工业化"**：由于能源成本 + 监管 + 绿色转型，欧洲化工 / 钢铁 / 铝工业持续萎缩，部分工厂迁至美国 / 亚洲；③ **Linde EMEA 的销量增长近 3-5 年持续低于全球平均**：FY25 底层 -1%，FY24 约 0%，FY23 约 -1%；④ **Engineering 业务是强周期业务**：订单 / 储备下滑预示 2027-2028 收入下降，即使当前储备 $27 亿仍可交付 2-3 年；⑤ **欧洲清洁氢 / CCUS 投资可能延迟**：财政压力 + 政治不稳定（德国、法国政府变动）+ 能源价格波动可能使 IRA 风格补贴难以在欧洲落地；⑥ 虽然 EMEA margin 在 FY25 提升 +240 bps 展现韧性，但长期看如果销量持续 -1-3%，EBITDA 也会承压；⑦ Linde 应对策略：加大 Americas + APAC 投入，逐步调整地区组合。

### 利空 2：Capex +17% 至 $53 亿 + 净债务 +31% + ROC 下降 170 bps + 高利率再融资压力

**依据**

- **FY25 Capex $52.61 亿美元**（+17.0% vs FY24 $44.97 亿）
- **2026 Capex Guidance $50-55 亿美元**：继续维持高位
- **调整后净债务 $219.30 亿美元**（+30.8% vs FY24 $167.69 亿）
- **总债务 $269.89 亿美元**（+24.8%）：支持 Capex + $74 亿股东回报
- **Adjusted ROC 24.2%**（-170 bps vs FY24 25.9%）：资本扩张 → 资本基数扩大
- ROC 趋势：FY22 25.6% → FY23 25.9% → FY24 25.9% → FY25 24.2%
- **长期债务 +34.8% 至 $207 亿**
- **利息费用保持 $2.55 亿**：目前低位，但未来再融资成本将上升
- 5 年内到期债务约 $100 亿美元+，需要以 4-5% 利率再融资（vs 原 2-3% 票息）
- 美国 10 年期国债收益率 FY25 末仍在 4.2-4.5%
- **项目储备 $100 亿需要额外资本投入**：交付期间 Capex 增加
- CFFO $103.5 亿 / Capex $52.6 亿 = 1.97x（之前年份 2.1-2.2x），现金覆盖率下降

**解读**

**资本支出加速 + 债务扩张是 Linde 的中期压力点**：① **FY25 Capex +17%** 是由 **合同销售气体储备交付**推动，属于"良性成长性"资本支出，但短期压制 FCF（FCF +3.3% 远低于 CFFO +9.8%）；② **ROC 从 25.9% 下降至 24.2%**：虽然仍是全球最佳，但趋势下滑反映 Capex 增加 → 资本基数扩大 → 同等规模的利润贡献相对资本比例下降；③ **调整后净债务 +31%** 意味着净债务 / Adj EBITDA 从 1.3x → 1.6x，虽然仍健康但杠杆上升；④ **利息费用上升压力**：当前 $2.55 亿极低（由 2-3% 原票息支撑），但 2025-2028 $100 亿美元到期债务再融资将以 4-5% 票息重置，年化增量利息费用 $2-3 亿美元，侵蚀 EPS 约 $0.30-0.45；⑤ **如果项目储备交付延迟**（如 Hydrogen Hub 政治风险 + 电子客户产能建设推迟），Capex 转化为收入的周期可能延长 1-2 年，进一步降低 ROC；⑥ **Capex / Revenue = 15.5%**（vs 历史平均 12-13%），处于高位；⑦ 应对策略：通过定价实现 + 产量贡献推动 ROC 回升，长期 Linde 目标 ROC 维持 25%+。

### 利空 3：关税 / 贸易冲突风险 + 全球经济不确定性 + 汇率波动 + 清洁氢商业化进度不确定

**依据**

- **Trump 政府关税政策**（2025 实施，2026 可能扩大）：
  - 对中国商品 60% 关税
  - 对墨西哥 / 加拿大 25% 关税（USMCA 重议）
  - 对欧盟 10-20% 关税可能
  - 对钢铁 / 铝 25% 关税（扩展品类）
- **Linde 客户受影响**：化工、金属、电子、汽车行业的客户可能降低资本支出 / 产能调整
- **全球经济增长预期**：IMF 2026 全球 GDP 3.0-3.2%，相对 2021-2022 的 5% 水平大幅放缓
- **美元走弱预期**：公司 2026 EPS Guidance 含 3% 货币顺风（Q1）+ 1% 货币顺风（全年）
- **货币波动**：美元 / 欧元、美元 / 人民币、美元 / 印度卢比 / 巴西雷亚尔等影响 EMEA + APAC + 南美业绩
- **清洁氢商业化不确定性**：
  - 美国 45V 清洁氢税收抵免规则（最高 $3/kg）尚未最终确定
  - 7 个 Hydrogen Hub 项目（美国能源部）资金审批可能延迟
  - 绿氢 LCOE 仍高于灰氢 + 蓝氢 2-3 倍
  - Linde 投入清洁氢项目 Capex 回收周期 10-15 年
- **电网 / 电力成本**：数据中心 + 工业需求推高电价，影响 Linde 主要成本项（电力）
- 2026 全球通胀 2.5-3%，高于之前的 1.5-2% 常态
- 风险披露（10-K）：关税 / 贸易冲突被重点强调为新增风险

**解读**

**宏观与政策不确定性是 Linde 不可控的外部风险**：① **关税影响客户 Capex → Linde 销量**：如果客户因关税调整生产计划 / 推迟扩产，Linde 工业气体需求增长可能放缓；② **美元走弱 vs 走强的不确定性**：FY25 全年货币中性，但 FY26 Guidance 含 3% 顺风；如果美元反转走强，EPS 可能比预期低 3-5%；③ **EMEA 货币（欧元）**：欧元走势对 Linde 影响最大（EMEA 占 25% 收入）；④ **清洁氢商业化进度**：Linde 是全球氢能 leader，但清洁氢项目从"签约"到"运营"周期 3-5 年，短期收入贡献有限，同时需要前期资本投入 → 短期财务负担 + 长期不确定回报；⑤ **电力成本上升**：工业气体生产主要成本是电力（O2 / N2 生产），如果电价持续上涨（AI 数据中心 + 脱碳推动），Linde 需要提价转嫁 → 可能损失销量；⑥ **全球经济衰退风险**：如果 2026-2027 出现全球经济衰退（制造业 PMI 全面 <50），Linde 销量可能 -5-10%（2008-2009 金融危机 Linde 销售 -10%+）；⑦ 虽然 Linde 业绩韧性在周期股中排名前列（长约 + 医疗 / 食饮占比），但仍无法完全免疫宏观冲击。

## 8. 总结判断

**一句话结论**

**Linde FY2025 交出"收入 $340 亿 (+3%)、Adj EPS $16.46 (+6%)、GAAP EPS $14.61 (+7%)、Adj Op Margin 29.8% 创新高 (+30 bps)、Adj EBITDA $134 亿 (+4%)、ROC 24.2% 全球最佳、项目储备 $100 亿创新高、股东回报 $74 亿、股息连续第 32 年增长"的卓越业绩**，**充分体现"工业气体龙头 + 高壁垒 + 高 ROIC + 长期合同驱动 + 定价纪律"的优质特质**。**FY26 核心看点是项目储备交付（$71 亿合同销售气体）+ 电子 / 清洁氢新项目 + 2026 EPS Guidance +6-9% + EMEA 重组后 margin 改善 + Engineering 重组后 FY26+ 利润率回升**；**核心风险是 EMEA 化工 / 能源需求疲软 + Engineering 业务持续下滑 + Capex +17% 与 ROC 下降 170 bps + 高利率再融资压力 + 关税 / 贸易冲突 + 清洁氢商业化不确定**。

**核心看点**

1. **项目储备 $100 亿（$71 亿合同销售气体）+ 2-5 年可见度 + 电子半导体 + 清洁氢长期增长**：15-20 年长约 + take-or-pay 条款提供稳定现金流，电子行业 FY25 +10%+ 驱动短期成长
2. **Adj Op Margin 29.8% 创新高 + ROC 24.2% 全球最佳 + Engineering 重组后 margin 改善**：运营效率 + 资本纪律行业标杆，margin 持续扩张体现结构性改善
3. **股东回报 $74 亿 + 股息连续第 32 年增长 + 2026 EPS Guidance +6-9% + 10 年 EPS CAGR 12%**：Dividend King / Aristocrat 标的，Growth + Quality + Income 典范

**核心风险**

1. **EMEA 化工 / 能源需求疲软 + Engineering 业务持续下滑 + 欧洲经济结构性压力**：EMEA 底层 -1% + Engineering -3%，长期去工业化风险
2. **Capex +17% 至 $53 亿 + 净债务 +31% + ROC 下降 170 bps + 高利率再融资压力**：短期 FCF 承压 + FY26+ 利息费用上升
3. **关税 / 贸易冲突 + 全球经济不确定性 + 汇率波动 + 清洁氢商业化进度**：Trump 关税影响客户 Capex，清洁氢项目从签约到运营需 3-5 年
