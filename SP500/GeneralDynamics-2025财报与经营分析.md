# General Dynamics Corp (GD) 2025 年财报整理

> 数据口径：本报告基于 General Dynamics Corporation 于 2026 年 1 月 28 日发布的 2025 财年全年及第四季度财务业绩公告（FY2025 = 2025-01-01 至 2025-12-31，自然年），币种美元（USD），按美国 GAAP 准则编制。自由现金流（Free Cash Flow）和投入资本回报率（ROIC）为非 GAAP 口径。General Dynamics 主要按**四大业务分部**披露：**Aerospace（Gulfstream 公务机）**、**Marine Systems（海军造船）**、**Combat Systems（陆战系统 + 弹药）**、**Technologies（IT 与任务系统）**。所有金额保留原始披露币种并注明单位。

## 1. 业绩总结

### 1.1 收入与利润表现

- **全年营业收入**：FY2025 总营收 **$52,550M（约 $525.5 亿美元）**，**同比 +10.1%**（FY2024 $47,716M）；**四大分部均实现正增长，Aerospace 和 Marine Systems 双位数增长驱动**；Q4 单季营收 **$14,379M（+7.8%）**。
- **全年营业利润**：GAAP Operating Earnings **$5,356M（+11.7%）**，**营业利润率 10.2%**（FY2024 10.1%，**+10bps**）；Q4 单季营业利润 $1,452M（+2.0%）。
- **全年归母净利润**：Net Earnings **$4,210M（+11.3%）**，**稀释 EPS $15.45（+13.4%）**（EPS 增速高于净利润主要因**连续回购**，FY2025 稀释股数从 277.5M 降至 272.4M，-1.8%）；Q4 EPS $4.17（+0.5%）。
- **订单和积压（Backlog）**：**Q4 2025 新订单 $22.4B**，**Q4 Book-to-Bill 1.6x / FY 1.5x**；**年末 Backlog 达 $1,180 亿美元（+30% YoY）**，**总合同预计价值 $1,790 亿美元（+24%）**；**Combat Systems 积压 +60%**，**Marine Systems 积压 +31%**，**订单创公司历史纪录**。
- **现金流与资本结构优化**：**经营现金流 $5,120M（+24.5%，占净利润 122%）**；自由现金流 **$3,959M（+23.9%）**；资本开支 $1,161M（+27%，加速投资产能）；**总债务降至 $8,013M（-8.5%）**，**净债务降至 $5,680M（-19.6%）**；**股东权益从 $22.1B 增至 $25.6B（+16%）**。

### 1.2 增长与拖累因素

- **主要增长驱动**：
  - **Aerospace 创纪录交付 158 架 Gulfstream**（FY2024 136 架，**+22 架 / +16.2%**）：**G700 进入规模量产 + G800 / G400 陆续量产**，大型公务机（Large-cabin）交付 **136 架（FY2024 118 架，+15%）**、中型公务机（Mid-cabin）**22 架（FY2024 18 架，+22%）**；Aerospace 营收 **$13,110M（+16.5%）**。
  - **Marine Systems 营收 +16.6% 至 $16,723M**：**最快增长的分部之一**，主要受益：(1) **哥伦比亚级核潜艇（Columbia-class SSBN）量产**（首艘 DC 已完成主体建造，准备下水）；(2) **弗吉尼亚级核潜艇（Virginia-class SSN）产能爬升**（美国海军需求目标达 Block V/Block VI 每年 2+ 艘）；(3) **AUKUS 联盟对核潜艇建造能力的长期投资需求**；(4) **盟军合同（英国、澳大利亚）+ 美国海军 Maintenance、Repair 业务**。
  - **Combat Systems 营收 +2.8% 但积压 +60%**：**订单积压创纪录 $27.2B**（FY2024 $17.0B）；俄乌冲突延续 + 美国陆军现代化 + 欧洲盟国补充库存 + 中东紧张推动**主战坦克（Abrams M1A2 SEPv3/v4）、装甲战车（Stryker、AJAX、LAV 6.0）、弹药（155mm 炮弹 MASS 产能翻倍扩张）**等订单；GD 是**北美唯一 155mm 大型炮弹供应商**，持续扩产。
  - **Technologies 营收 +2.6% 至 $13,471M**：**IT 服务、云迁移、网络安全、任务系统**增长，但**美国政府 CR（Continuing Resolution）+ 部分合同转交影响**；年末积压 $16.7B（+18%）。
  - **AUKUS 联盟 + 国防预算增长 + 地缘政治**：FY2025 美国国防预算约 $895B（+2-3%），欧洲重新武装（德国、波兰、英国增加支出），多国海军 + 陆军升级周期共振。
  - **强劲订单转化 + Book-to-Bill 1.5x 全年**：保障未来 3-5 年收入可见度；**$1,180 亿积压约等于 2.2 年的营收**。
- **主要拖累因素**：
  - **Q4 Aerospace 营业利润 -17.8%**（$481M vs $585M，Op Margin 12.7% vs 15.6%）：**G700 新品量产爬坡相关的制造成本 + 供应链问题 + 产品组合短期变化**；FY2025 全年 Aerospace 利润率仍扩张 +30bps 至 13.3%，Q4 的短期压力主要与 G800 首交付 + G400 量产前期投入相关。
  - **Q4 Technologies 营业利润 -9.1%**（$290M vs $319M，Op Margin 9.0% vs 9.8%）：部分大型合同转接 + 产品组合变化 + 政府 CR 影响；FY 全年利润率 9.5%（-10bps）。
  - **Q4 税率上升**：Q4 有效税率 18.3%（FY2024 Q4 16.1%），影响 Q4 净利润同比小幅下滑（-0.4%）；但 FY2025 全年有效税率 17.5%（FY2024 16.7%），仍处于较低水平。
  - **资本开支 +27%**：$1,161M（FY2024 $916M），**FY2025 对 Gulfstream 产能、Marine Systems 潜艇建造厂（Electric Boat、NASSCO）、Combat Systems 弹药产能**大规模投资；短期对自由现金流有压制，但长期对未来 5-10 年产能保证至关重要。
- **结构性观察**：FY2025 是 GD **"四大分部齐增长 + 积压创纪录 + 资本返还加速 + 长期需求确定性强"**的标志性年份；**公司从周期性防务承包商转型为"国防长期建仓 + 公务机高端制造"双引擎 Quality Compounder**；FY2026-2030 收入增速有望保持 +8-10%，EPS 增速 +10-13%。

### 1.3 趋势判断

- **未来主要利好方向**：
  - **国防预算长期高位 + AUKUS + 欧洲重新武装**：美国 FY2026 国防预算 +2%、FY2027-2030 年均 +3-5%；德国、波兰、英国、日本、韩国、澳大利亚国防支出持续上升，直接利好 GD 四大分部。
  - **Marine Systems 超级周期**：哥伦比亚级（12 艘 SSBN 订单 / 单艘 $15B+ / 2025-2042 交付）+ 弗吉尼亚级（Block V/VI 每年 2-3 艘）+ AUKUS 澳大利亚核潜艇（$2,390 亿长期合同），**Marine Systems 分部未来 15-20 年可见度无出其右**。
  - **Combat Systems 订单积压 $27.2B（+60%）**：Abrams 现代化（M1A2 SEPv4）、Stryker 升级、欧洲 AJAX + LAV 6.0 合同、弹药产能翻倍（155mm MASS 产能到 2026 年末达 100,000 发/月），长期高位订单。
  - **Aerospace Gulfstream 新机型周期**：G800（2025 H1 认证交付）、G400（2025 H2 交付）、G500/G600 持续销售；**Aerospace Book-to-Bill 1.2x**，订单 Backlog $21.8B（+11%）；高端公务机市场景气度 2025-2028 年延续。
  - **Technologies 向数字化转型**：**IT 现代化 + 云迁移 + 网络安全 + AI + 国防通信**是美国政府长期优先级，GD Technologies 是主要承包商之一（仅次于 Leidos、Booz Allen、CACI）。
  - **稳健资本返还 + 投资加速**：**FY2025 股息 $1,593M（+4%）+ 回购 $637M = $2.23B 返还**；连续 28 年分红增长；FY2025 净债务 -19.6% 显著改善资本结构。
  - **Return on Invested Capital（ROIC）14.2% / Return on Equity（ROE）17.9%**：持续改善的资本效率。
- **未来主要压力方向**：
  - **美国政府 Continuing Resolution（CR）+ 政府停摆风险**：2025 Q4 已有短暂政府停摆，影响部分 Technologies 合同；若 2026-2027 年再次陷入政府关门，Technologies 和 Aerospace 部分合同会受影响。
  - **Aerospace 短期利润率压力**：G700 爬坡 + G800/G400 首批交付成本 + 供应链依然存在问题；FY2026 Aerospace 利润率可能 **temporary** 承压。
  - **关税和供应链**：虽然 GD 主要是美国本土生产，但部分零部件、原材料（钢、铝、电子零件）依赖全球供应链；若美国新关税政策 + 中美脱钩加剧，Combat Systems 和 Marine Systems 成本可能上升。
  - **竞争加剧 + 新兴防务创业公司威胁**：Anduril、Palantir、SpaceX 等科技驱动防务公司正在挑战传统防务巨头；GD 需在 AI、无人系统、自主作战等新兴领域加快投入。
  - **估值偏高**：FY2025 末股价约 $380，Forward P/E ~23x（基于 FY2026 预期 EPS $16.50 中值），处于历史偏高位置。
- **整体趋势判断**：**FY2026-2028 GD 将是国防工业板块 "最确定的长期增长 + 最稳健的资本返还" 组合**；$1,180 亿积压 + AUKUS + 欧洲重新武装 + Gulfstream 新机型 = **营收 CAGR 8-10% / EPS CAGR 10-13%**；GD 的 Marine Systems 分部（哥伦比亚级 + 弗吉尼亚级 + AUKUS）构成美国国防工业**最长远的收入可见度**（到 2045 年）；估值虽然偏高但可接受，**是长期防务配置的核心仓位**。

## 2. 主营业务

### 2.1 公司简介

General Dynamics Corporation (NYSE: GD) 是**全球最大的航空航天与防务公司之一**，总部位于美国弗吉尼亚州 Reston，成立于 1952 年。公司**全球员工超过 11 万人**，FY2025 营收 $52.6 亿美元，是 Fortune 100 和 S&P 500 成分股，具有**完整产业链覆盖**（包括空中、海上、陆上、IT/任务系统四大领域），主要客户包括**美国国防部（约 60-65% 营收）、美国联邦政府（Technologies 客户）、北约盟国、商业 + 私人公务机市场（Gulfstream）**。

### 2.2 主营业务构成（四大分部）

1. **Aerospace（航空，FY2025 营收 $13,110M，占总营收 24.9%）**：
   - **核心产品**：**Gulfstream 系列公务喷气机**（G800、G700、G650ER、G600、G500、G400、G280），全球大型公务机市场领导者；
   - **服务业务**：**Jet Aviation**（全球公务机维护、管理、包机服务）；
   - **FY2025 关键数据**：
     - **Gulfstream 交付 158 架**（FY2024 136 架，**+22 架 / +16.2%**）
     - 大型机 **136 架**（含 G800 首批交付、G700 量产）
     - 中型机 **22 架**（G280 + G400）
     - **Book-to-Bill 1.2x**
     - 订单积压 **$21.8B（+11% YoY）**

2. **Marine Systems（海洋系统，FY2025 营收 $16,723M，占总营收 31.8%，**最大分部**）**：
   - **核心业务**：**核潜艇设计和制造**（Electric Boat 电船分部）+ **水面舰艇设计和制造**（NASSCO）+ **舰船维修、大修、升级**；
   - **主要项目**：
     - **哥伦比亚级战略核潜艇（Columbia-class SSBN）**：12 艘订单 / 单艘 $13-15B / 2025-2042 交付
     - **弗吉尼亚级攻击核潜艇（Virginia-class SSN）**：Block V/VI 每年 2-3 艘
     - **AUKUS 澳大利亚核潜艇合同**
     - 海军补给舰（TAKE、TAO、T-ESB）
     - 海军维护和现代化（MRA）
   - **FY2025 关键数据**：
     - 营收 **+16.6%**，最快增长分部之一
     - 订单积压 **$52.3B（+31.4% YoY，创纪录）**

3. **Combat Systems（陆战系统，FY2025 营收 $9,246M，占总营收 17.6%）**：
   - **核心产品**：
     - **主战坦克**：M1 Abrams（美国陆军 M1A2 SEPv3/v4 升级）
     - **装甲战车**：Stryker（8x8 装甲运兵车）、LAV 6.0、AJAX（英国陆军）、Piranha、ASCOD
     - **弹药**：155mm 炮弹（MASS 产能到 2026 年底达 100,000 发/月）、Hydra-70 火箭弹、中小口径炮弹
     - **武器系统**：炮塔、主炮、火控系统
   - **主要客户**：美国陆军、北约盟国（英国、德国、波兰、西班牙、意大利）、中东盟国（沙特、UAE、埃及、以色列）；
   - **FY2025 关键数据**：
     - 营收 **+2.8%**（相对温和，因为部分合同是从 2024 年延续）
     - **订单积压 $27.2B（+60.3% YoY，创纪录）**
     - **弹药产能扩张**：美国陆军 MASS 计划预期投资 $8B，GD 为主要受益者

4. **Technologies（技术，FY2025 营收 $13,471M，占总营收 25.6%）**：
   - **核心业务**：**IT 服务 + 任务系统 + 网络安全 + 通信**；
   - **主要子公司**：
     - **General Dynamics Information Technology (GDIT)**：美国联邦政府 IT 服务领导者之一，主要客户：国防部、国土安全部、社保局、VA、FBI、国务院等；
     - **General Dynamics Mission Systems (GDMS)**：军用通信、情报、侦察、指挥与控制（C4ISR）系统、加密通信、传感器；
   - **FY2025 关键数据**：
     - 营收 **+2.6%**（温和增长，受政府 CR 影响）
     - 订单积压 **$16.7B（+18% YoY）**

### 2.3 核心收入来源

- **按客户类型**：美国国防部（DoD）~55-58%、美国联邦政府其他部门（GDIT 主要客户）~15-18%、北约盟国 ~10-12%、商业 + 私人（Gulfstream）~15-17%、其他 ~1-3%。
- **按地域**：美国约 75-78%、欧洲 + 中东 15-18%、其他地区（亚太、加拿大）5-8%。
- **按合同类型**：成本加成（Cost-Plus）~35-40%、固定价格（Fixed-Price）~55-60%、T&M（Time and Materials）~5-10%。
- **按业务模式**：产品（Production）~75-80%，服务（Services）~20-25%。

  - **芬兰、瑞典加入 NATO** 后的武器采购；
- **155mm 炮弹产能扩张（MASS 计划）**：
  - 美国陆军 MASS（Modular Artillery System Solution）计划投资 **$8 亿美元**；
  - GD OTS（Ordnance & Tactical Systems）产能从 FY2024 24,000 发/月扩至 **FY2026 末 100,000 发/月**；

## 6. 未来可能的利好

### 利好 1：Marine Systems 超级周期 —— 哥伦比亚级 + 弗吉尼亚级 + AUKUS 构成 20 年可见度

**依据**

- **哥伦比亚级战略核潜艇（Columbia-class SSBN）**：
  - 美国海军订单 **12 艘**（单艘 $13-15B），**总订单约 $1,600-1,800 亿美元**；
  - 交付时间表：首艇 2027 年服役 → 每年 1 艘左右 → 2042 年前 12 艘全部交付；
  - GD Electric Boat 是**主承包商**（与 Huntington Ingalls/Newport News 共同建造）；
  - 替代即将退役的俄亥俄级 SSBN，是**美国核威慑三位一体的海基支柱**，**政治上不可替代**；
- **弗吉尼亚级攻击核潜艇（Virginia-class SSN）**：
  - Block V（2024-2029 年）+ Block VI（2026 年开始）；
  - 产能目标 **2-3 艘/年**（vs 当前 1.5 艘/年），**美国海军和国会正在投资扩产**；
  - 单艘约 **$3.5-4.5B**；
- **AUKUS 澳大利亚核潜艇合同**：
  - 2021 年美国、英国、澳大利亚三国签署 AUKUS 协议；
  - 澳大利亚将购买 **3-5 艘弗吉尼亚级 SSN（2032 年起）+ 自建 SSN AUKUS（2040 年起）**；
  - 长期合同价值 **$2,390 亿美元**（澳大利亚国防部估算）；
  - GD Electric Boat 提供**设计、建造支持、培训**；
- **Marine Systems 积压 $52.3B（+31%）已反映部分**，但**哥伦比亚级剩余订单 + AUKUS 长期合同尚未完全进入 Backlog**，实际可见收入可达 **$200B+**。

**解读**

- **Marine Systems 是 GD **最具长期确定性的分部**：
  - **20 年可见度**：到 2045 年有明确订单；

## 7. 未来可能的利空

### 利空 1：美国政府 Continuing Resolution + 政府停摆风险

**依据**

- **FY2025 Q4 美国政府曾短暂停摆**（几周），影响 Technologies 合同执行；
- **美国政府长期 CR（Continuing Resolution）**：2020 年以来几乎每年都陷入 CR 状态，无法正常执行新预算；
  - CR 期间：部分新项目无法启动，已有合同维持但扩展受限；
  - 若 2026-2027 年继续陷入 CR，Technologies 分部收入增速可能停留在 +2-3%；
- **政治不确定性**：美国两党对国防支出的不同态度可能影响 FY2027+ 预算通过速度；
- **GDIT 客户风险**：GDIT 主要客户是美国联邦政府民用部门（社保局、VA、FBI），受 CR 影响较多。

**解读**

- **基准情景**：FY2026 美国预算在 9-12 月通过，Technologies 分部回归 +4-6% 增速；
- **悲观情景**：若继续 CR 全年，Technologies 分部 +1-2% 增速，整体 GD 营收增速可能 +4-5%（vs 共识 +6-8%）；
- **缓冲因素**：
  - Aerospace、Marine Systems、Combat Systems 主要不受 CR 直接影响（长期合同已签）；
  - Technologies 分部占 GD 营收 26%，有限敞口；
- **长期观察**：美国国防 + 联邦政府 IT 预算结构性增长趋势不变，CR 只是短期扰动。

### 利空 2：Aerospace 短期利润率承压 + Gulfstream 新品量产风险

**依据**

- **Q4 2025 Aerospace 利润率 12.7%**（FY2024 Q4 15.6%，-290bps）；
- **主要原因**：
  - **G800 首批交付相关成本**：新机型量产初期成本较高（供应链、质量控制、客户定制化）；
  - **G400 量产爬坡**：新机型学习曲线期；
  - **供应链问题**：航空制造业（发动机供应商、航电系统）仍有压力；
- **市场对 FY2026 Aerospace 利润率担忧**：可能短期降至 **12.5-13.0%**（vs FY2025 13.3%），temporary 下降；
- **Gulfstream 品牌历史上经历过多次**量产爬坡期利润率承压**（如 G700 2023-2024 量产初期）**。

**解读**

- **基准情景**：FY2026 Aerospace 利润率 12.8-13.3%（与 FY2025 持平），FY2027+ 回升至 14.0%+；
- **悲观情景**：若 G800/G400 供应链问题延续，利润率可能停留在 12.5% 或更低，**影响 FY2026 Aerospace 营业利润 -$100-200M**；
- **缓冲因素**：
  - Gulfstream 长期竞争优势稳固（市占率 45-50%）；
  - 订单积压强劲（Book-to-Bill 1.2x）；
  - 售后服务（Jet Aviation）高利润率稳定贡献；
- **长期看**：Gulfstream 新品周期通常 3-4 年，2027-2028 年 G800/G400 进入成熟期后利润率将回归历史高位（14-15%）。

### 利空 3：关税和供应链风险 —— 钢铁、铝、电子零部件依赖全球供应链

**依据**

- GD 主要制造在美国本土，但以下品种存在供应链敞口：
  - **钢材（潜艇、坦克、装甲战车）**：部分高端特种钢从欧洲、日本进口；
  - **铝材（公务机机身）**：部分合金从加拿大、德国进口；
  - **电子零部件（雷达、通信、电子设备）**：部分来自中国、墨西哥、马来西亚；
  - **电机、液压系统**：部分来自欧洲；
- **美国新关税政策**：可能对中国（+25%）、墨西哥（+10-25%）、欧盟（+10-20%）商品加征关税；
- **成本影响估算**：
  - 若关税全面施加，GD 短期成本可能上升 **$200-400M 年化**；
  - 长期可部分转嫁给客户，但时间滞后 12-24 个月；
- **俄乌冲突 + 中东紧张**：能源成本上升、航运受阻（红海）。

**解读**

- **基准情景**：FY2026 关税影响利润率 -30-50bps，**对 EPS 负面影响 $0.20-0.40**；
- **悲观情景**：若中美全面关税战 + 供应链重组 + 通胀加剧，FY2026 EPS 可能受影响 $0.50-0.80；
- **缓冲因素**：
  - GD 75%+ 营收来自美国国防部 + 联邦政府，成本加成（Cost-Plus）合同可转嫁部分成本；
  - GD 主要制造在美国本土，**受益 Reshoring（生产回流）趋势**；
  - 长周期合同（如核潜艇、哥伦比亚级）有通胀调整机制；
- **长期看**：美国国防制造供应链本土化是长期趋势，GD 作为美国本土大型承包商将最终受益。

  - Marine Systems 哥伦比亚级不可替代；
- **长期投资者**：逢低加仓，GD 是防务板块 Quality Compounder，持有 10 年复合回报 11-14%。

### 利空 6：商誉 $21B + 并购依赖 + 养老金风险

**依据**

- **商誉 $21.0B 占总资产 37%**：
  - 主要来自 CSRA（2018 年 $97 亿并购）+ Gulfstream（2001 年）+ 历史并购；
  - 若 Technologies 分部持续低增长，可能产生商誉减值压力；
- **养老金负债**：
  - Other Liabilities 包括 Defined Benefit 养老金；
  - FY2025 Other Liabilities $7.8B（FY2024 $8.7B，大幅减少），主要因精算调整；
  - 若未来利率下降或股市下跌，养老金负债可能重新上升；
- **并购依赖**：GD 历史上通过 CSRA（2018）、Bath Iron Works（1995）等并购扩张，但近年并购节奏放缓；若 Technologies 分部不通过并购扩张，可能增速持续温和。

**解读**

- **商誉减值风险可控**：
  - Technologies 分部订单积压 +18% 显示健康，未出现明显衰退信号；
  - FY2026-2027 若美国预算 CR 延续可能触发减值评估，但概率较低；
- **养老金风险**：
  - FY2025 Accumulated Other Comprehensive Loss 从 -$1,518M 改善至 -$483M（+$1,035M），主要因养老金精算利好；
  - 若未来利率大幅下降，养老金负债可能重新上升 $500-1,000M，影响股东权益；
- **并购预期**：未来 2-3 年 GD 可能寻求 $1-3B 规模的 Technologies 领域并购（AI、网络安全、云服务），完善 Technologies 分部的长期增长动力。

## 8. 总结判断

**一句话结论**

General Dynamics FY2025 录得**营收 $525.5 亿（+10.1%，四大分部均正增长）**、**营业利润 $53.6 亿（+11.7%）**、**净利润 $42.1 亿（+11.3%）**、**稀释 EPS $15.45（+13.4%）**、**经营现金流 $51.2 亿（+24.5%，占净利润 122%）**、**自由现金流 $39.6 亿（+23.9%）**、**ROIC 14.2%（+100bps）、ROE 17.9%**；**Q4 新订单 $22.4B / Book-to-Bill 1.6x / FY Book-to-Bill 1.5x、年末 Backlog 达 $1,180 亿（+30%）、总合同预计价值 $1,790 亿（+24%）**，**Marine Systems +16.6%（积压 $52.3B / +31%）+ Aerospace +16.5%（交付 158 架 Gulfstream / +16%）+ Combat Systems 积压 $27.2B（+60%）**三线强劲引擎，**Technologies 受 CR 影响 +2.6%**；**资本结构大幅优化**（净债务 -19.6% 至 $5.68B、Debt-to-Equity -840bps 至 31.3%、Net Debt/EBITDA 从 1.16x 降至 0.90x）；**股息 $1.6B（连续 28 年增长）+ 回购 $637M = 资本返还 $2.2B**。GD 已成为**防务板块最具长期确定性的 Quality Compounder**，**哥伦比亚级核潜艇 + AUKUS + 欧洲重新武装 + 弹药超级周期 + Gulfstream 新机型** 五大长周期催化确保 FY2026-2030 营收 CAGR 8-10% / EPS CAGR 10-13%。

**核心看点**

1. **Marine Systems 超级周期（到 2045 年）**：哥伦比亚级 12 艘 SSBN（$160-180B 订单）+ 弗吉尼亚级 Block V/VI（每年 2-3 艘）+ AUKUS 澳大利亚核潜艇（$2,390 亿合同）；积压 $52.3B（+31%）+ Marine Systems 利润率从 7.0% 升至 8-9%。
2. **Combat Systems 订单积压 $27.2B（+60%）+ 弹药产能翻倍**：俄乌 + 欧洲重新武装 + 美国陆军现代化驱动；**155mm 炮弹产能从 24,000 发/月扩至 100,000 发/月**；Combat Systems 利润率 14.4% 持续高位。
3. **FY2025 Book-to-Bill 1.5x + Backlog $118B（+30%）+ 连续 28 年股息增长**：2.2 年收入可见度 + 防务板块投资级 A 信用评级 + ROIC 14.2%；FY2026-2030 EPS CAGR 10-13% 确定性极高。

**核心风险**

1. **美国政府 CR 延续 + Technologies 分部持续温和 + 政府停摆风险**：FY2025 Technologies 增速仅 +2.6%，若 FY2026 继续 CR 增速可能停留在 +2-3%，拖累整体 GD 营收增速至 +4-5%（vs 共识 +6-8%）。
2. **Aerospace 短期利润率承压（Q4 -290bps）+ Gulfstream G800/G400 新品量产风险 + 关税和供应链风险**：新机型量产初期成本压力 + 美国新关税政策 + 中美脱钩可能影响 FY2026 EPS $0.20-0.60。
3. **Forward P/E ~22x 处于历史 75-80 百分位 + 防务板块整体估值高位 + 新兴科技防务公司竞争**：相对 Lockheed Martin/Northrop 溢价 10-20%；若俄乌停火或宏观恶化可能 -10-20% 回调；Anduril、Palantir 等新兴竞争者可能挑战 Technologies 分部。
