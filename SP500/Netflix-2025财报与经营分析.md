## 1. 业绩总结

### 1.1 收入与利润表现

- 全年总收入 **$451.83 亿美元**，同比 **+15.9%**（FY24 $390.01 亿）/ **FX-Neutral +17%**
- **GAAP 营业利润 $133.27 亿美元**，同比 **+27.9%**（FY24 $104.18 亿）
- **GAAP 营业利润率 29.5%**，同比 **+280 bps**（FY24 26.7%）
- **GAAP 净利润 $109.81 亿美元**，同比 **+26.1%**（FY24 $87.12 亿）
- **GAAP 摊薄 EPS $2.53**（+27.8%）（FY24 $1.98）—— 已按 10:1 拆股重新调整
- **经营现金流 $101.49 亿美元**（+37.9%）（FY24 $73.61 亿）
- **自由现金流 $94.61 亿美元**（+36.7%）（FY24 $69.22 亿）
- **Content Investment（Cash spend）$170.97 亿美元**（+5.4%）
- **Content Amortization $164.22 亿美元**（+7.3%）
- 股本 -1.1%（Diluted 4,343,863K vs FY24 4,392,608K 经 10:1 split 后）
- **广告收入超过 $15 亿美元**（+2.5x vs FY24）
- **期末付费会员 3.25 亿 +**（突破 3.25 亿里程碑，Q4 期间）
- Q4 FY25：Revenue $12.05B（+17.6%）、Op Income $2.96B（+30.1%）、Op Margin 24.5%（+230 bps）、Net Income $2.42B（+29.4%）、Diluted EPS $0.56（+30.2%）、FCF $1.87B（+35.8%）

### 1.2 增长与拖累因素

- **主要增长驱动**：① **广告收入 +2.5x 至 $15 亿美元**：Ad Tier 会员（$7.99/月）成为新用户主要入口 + ad sales 规模效应 + AI-powered custom ads；② **订阅价格上涨**：美国 Standard 会员 $15.49 → $17.99（+16%），Standard with Ads $6.99 → $7.99；③ **会员增长 325M+**（2024 年底约 301M，FY25 增加约 24M 净增）；④ **US TV 时长份额创新高 9.0%**（12 月 Nielsen 数据，+50 bps YoY）；⑤ **H2 2025 Engagement +2%**：Branded Originals 观看时间 +9%；⑥ **内容热门**：Stranger Things 最终季 120M 视图、Wake Up Dead Man: A Knives Out Mystery 66M、Frankenstein 102M、A House of Dynamite 78M、Nobody Wants This S2 31M；⑦ **K-Pop Demon Hunters 文化现象**（动画电影 + 商品 + 剧场版）；⑧ **UCAN 收入 +14.8% 至 $199.57 亿**（占比 44.2%）：最大单一区域；⑨ **EMEA +16.8% 至 $145.15 亿**（占比 32.1%）：第二大增长引擎；⑩ **Live events**：NFL Christmas Day + WWE + Anthony Joshua vs Jake Paul 拳击（330M 平均观众 Live+1）创签约 / 参与高峰
- **主要拖累因素**：① **非品牌内容观看时长下滑**：2023-2024 WGA 编剧罢工结束后许可内容供应量下降；② **Content Amortization +7.3%**（高于收入 +5% 的目标比率）；③ **Q4 FY25 有 $60M 桥接贷款利息费用**：WBD 收购相关，非经营性；④ **LATAM 报告收入增长 +10%**（FX-Neutral +22%）：货币疲软拖累美元报告增长；⑤ **APAC 报告 +18% / FX-Neutral +21%**（日元 / 人民币等亚洲货币走弱）；⑥ **利息费用 +8.0% 至 $7.77 亿**：债务增加 + 新增桥接贷款利息；⑦ **有效税率 13.7%**（FY24 12.6%，上升 110 bps）；⑧ **股权激励费用 +35.2% 至 $3.68 亿**：人才保留成本
- **一次性/异常项目**：① **10 for 1 股拆股（2025/11/14）**：所有每股数据重新调整；② **WBD 收购相关融资费用 ~$60M**（Q4）：桥接贷款相关利息费用；③ **外汇重计量损失 $72M**：债务外币重计量；④ **巴西税务诉讼 ~$700M 押金延迟**：从 2025 年移至 2026 年，导致 FCF 超指引 $5 亿

### 1.3 趋势判断

- **未来主要利好方向**：① **WBD 收购（$720 亿美元全现金）**：将 HBO Max + HBO + Warner Bros. 影视库（DC Universe / Lord of the Rings / Harry Potter / Game of Thrones 等）+ Warner Bros. 电影公司纳入 Netflix；② **广告收入 2026 年 ~doubling**（管理层指引）：~$30 亿美元，占比 >6%；③ **2026 Revenue Guidance $50.7-51.7B**（+12-14%）+ **Op Margin 31.5%**（+200 bps）：健康扩张路径；④ **付费会员突破 3.25 亿 + US TV 时长份额 9%**：仍远低于 40%+ 线性电视，增长空间巨大；⑤ **Live events 扩张**：2026 年 World Baseball Classic（日本 47 场全部独播）+ MLB Opening Night + Home Run Derby + WWE + NFL Christmas Day；⑥ **Video Podcasts 新品类**：Spotify/The Ringer + iHeartMedia + Barstool Sports 合作 + Pete Davidson / Michael Irvin 原创；⑦ **云游戏战略**：Cloud-delivered TV-based Party Games（Boggle, Pictionary, Lego Party, Tetris）已向 ~1/3 会员推出，2026 年扩大（含 FIFA 重新想象版）；⑧ **Pay 1 全球授权协议**：与 Sony Pictures 从美国扩展至全球，Universal 动画 / 真人电影授权；⑨ **Netflix Houses + Tudum 粉丝经济**：IP 变现 + 商品销售；⑩ 连续第 3 年 Op Margin 扩张：FY23 21.1% → FY24 26.7% → FY25 29.5% → FY26E 31.5%
- **未来主要压力方向**：① **WBD 收购整合风险**：$720 亿美元大规模并购，整合 HBO + WBD 文化 + IT + 人才挑战巨大；② **$422 亿美元桥接贷款 → 长期债务**：将使 Netflix 总债务从 $145 亿增至 $400 亿 +，利息费用可能翻番；③ **股票回购暂停**：为收购积累现金，短期 EPS 受压；④ **流媒体竞争加剧**：Disney+ / Hulu 合并、YouTube 进入 TV（2029 起 Oscars 全球独播）、Amazon Prime Video 体育投入（NBA + Thursday Night Football）+ Apple TV+ 原创内容；⑤ **内容投入持续**：$170 亿/年 Capex，2026 年 Cash Spend to Amortization 1.1x（上升）；⑥ **美元走强风险**：2026 Guidance 基于 1/1/2026 汇率，如果美元反转走强，EPS 可能低 3-5%；⑦ **印度 / LATAM 市场增长放缓**：APAC 报告 +18% vs 过去 +25%，增速回落；⑧ **监管风险**：欧盟 Digital Services Act + 英国 Online Safety Act + 印度内容审查 + 美国 FTC 并购审查（WBD 反垄断风险）；⑨ **AI 生成内容对创作者经济影响**：可能引发工会 / 创作者反弹
- **整体趋势判断**：**Netflix FY2025 交出"收入 $452 亿 (+16%)、Op Margin 29.5% (+280 bps)、EPS $2.53 (+28%)、FCF $95 亿 (+37%)、广告收入 +2.5x 至 $15 亿、会员 3.25 亿 + 突破里程碑"的突破性业绩**，**充分体现"全球流媒体龙头 + 内容飞轮 + 广告变现 + 国际扩张 + 资本纪律"的完美结合**。**FY26 核心看点是 WBD $720 亿美元收购完成（重塑行业格局）+ 广告收入翻倍 + Op Margin 31.5% + 会员突破 3.5 亿 + 新内容品类（Live/Podcasts/Games）扩张**；**核心风险是 WBD 整合复杂度 + $400 亿美元债务 + 回购暂停 + 流媒体竞争 + AI 时代内容成本结构 + 监管风险**

## 2. 主营业务

### 2.1 公司简介

Netflix, Inc.（NFLX）是全球领先的**付费流媒体娱乐服务**，总部位于加利福尼亚 Los Gatos。**1997 年创立**（初始为 DVD 邮寄租赁业务），**2007 年推出流媒体服务**，**2013 年启动原创内容**，**2022 年推出 Ad Tier**，**2024 年关闭 DVD 业务**。联合 CEO：**Ted Sarandos（内容）+ Greg Peters（产品 / 商业）**；CFO：**Spencer Neumann**；创始人 **Reed Hastings**（2023 年卸任，现任主席）。**2025 年底全球付费会员 3.25 亿+，全球覆盖 190+ 国家**，**支持 30+ 种语言字幕**，**2025 年 US TV 时长份额 9.0%**。主要商业模式：**订阅付费（SVOD）+ 广告支持订阅（AVOD）+ 未来的 WBD 并购后 PVOD/影院窗口**。**拟定于 2026 年完成 WBD $720 亿收购**（2025/12 宣布，2026/01 修订为全现金 $27.75/股），将 HBO Max + HBO + Warner Bros. Pictures / Television / Studios 纳入。**2025/11/14 执行 10:1 拆股**。

### 2.3 核心收入来源

- **订阅收入约 96%**：3.25 亿会员 × 加权平均 ARPU ~$11.6/月 ≈ $45 亿/月 ≈ $540 亿/年（毛），扣除不激活月、免费试用、Legacy 折扣等 ≈ $436 亿/年 订阅部分
- **广告收入约 3.3%**：$15 亿+（FY25），2026E ~$30 亿（~6% 占比）
- **Live/PPV/其他约 1%**：体育 + 综艺 live + DVD（停运中）

**关键价值驱动**：
1. **ARPU（Average Revenue Per Member）**：不断提价 + Ad Tier 新会员 + Premium 升级
2. **Member Growth**：会员总量增长（FY25 +24M 净增至 325M+）
3. **Engagement**：H2 2025 96B 小时（+2%）—— 决定保留率 + 定价能力
4. **Content ROI**：内容投入 / 观看小时数 / 会员价值
5. **Operating Leverage**：内容摊销增速 < 收入增速（FY26 预计 +10% vs 收入 +12-14%）

  - 顶级内容（Stranger Things / Squid Game / Bridgerton 等）
  - 全球覆盖（不同于 Hulu 美国本土）
- 2026 World Baseball Classic（日本 47 场）+ MLB Opening Night + Home Run Derby 提供体育广告库存

**解读**

**广告业务是 Netflix 未来 5 年最重要的增长引擎**：① **$15 亿 → $30 亿（1 年 100% 增长）**：在仅 3-4 年历史的业务中，绝对增速居美股 Top 5%；② **Netflix 的广告 TAM 巨大**：美国线性电视广告 ~$65B，Netflix 当前份额 <5% → 目标 20-30%；③ **单位经济效益（Unit Economics）**：Ad Tier 用户 ARPU = $7.99 订阅 + $3-5 广告 = $11-13（vs Standard 无广告 $17.99）—— Ad Tier 用户实际 ARPU 接近 Standard 但获客成本低；④ **AI 驱动广告**：Netflix 拥有 Content + User Data 双重优势，AI 可以创建 "episode-aware" 广告（如在 Wednesday 特定场景后出现 DC Comics 广告），CPM 可能提升至 $60-80；⑤ **体育 + Live events 广告库存**：独特的高价值库存（拳击 + NFL + WBC + 棒球）；⑥ **2027+ 广告收入可能达 $50-60 亿**（10%+ 占比），显著推动 Op Margin 扩张；⑦ Netflix 将广告视为"纯增量利润"：增量 Op Margin 可能 >60%，远高于订阅的 ~30%。

### 利好 3：2026 Op Margin 31.5% + FCF $110 亿 + 内容飞轮 + 产品创新 + 股东回报

**依据**

- **2026 Revenue Guidance $50.7-51.7B**（+12-14% YoY）
- **2026 Op Margin Guidance 31.5%**（+200 bps vs FY25 29.5%）
- **2026 FCF Guidance ~$11B**（+16% vs FY25 $9.46B）
- Content Spend / Amortization Ratio ~1.1x（vs FY25 1.04x）：投资高于摊销 → 内容库存加大
- **2026 资本分配优先级**：
  - 第一：业务再投资（有机 + M&A）
  - 第二：保持流动性
  - 第三：回馈股东（回购）
  - **回购暂停**：为 WBD 并购积累现金（FY25 回购 $91 亿，FY26 暂停）
- **新内容品类扩张**：
  - **Live Events**：WBC Japan 全 47 场 + MLB Opening Night + Home Run Derby + Star Search + Skyscraper Live
  - **Video Podcasts**：Spotify/The Ringer + iHeartMedia + Barstool + Pete Davidson / Michael Irvin 原创
  - **Cloud Games**：FIFA 重新想象版 + Boggle / Pictionary / Lego Party / Tetris
- **2026 Content Slate 亮点**：
  - Bridgerton S4 / ONE PIECE S2 / The Night Agent S3 / BEEF S2 / Stranger Things 2.0 世界观扩展
  - Greta Gerwig 的 Narnia + Something Very Bad Is Going to Happen（Duffer brothers）

  - Ben Affleck + Matt Damon The Rip + Charlize Theron Apex + Enola Holmes 3
- 已签署与 Universal（真人 + 动画）+ Sony Pictures 全球 Pay 1 协议 + Paramount ~20 shows 授权
- **Netflix Houses**：Dallas + King of Prussia 开业 → IP 变现 + 粉丝经济
- **Tudum**：232M 访问量（+18%）
- 10 年累计总回报 720%（vs S&P 500 298%）

**解读**

**Netflix 的"内容 + 产品 + 商业模式"是美股最完美的 compounder**：① **Op Margin 29.5% → 31.5% → 长期 35%+**：Op Leverage 仍有巨大空间；② **FCF $95 亿 → $110 亿**：高资本效率的自循环机器；③ **Content Flywheel**：内容吸引会员 → 会员提供数据 → 数据改进内容推荐 → 推荐提高参与度 → 参与度延长订阅 → 规模扩大内容预算；④ **产品创新速度**：从 Ad Tier (2022) → Live events (2024) → Games (2024) → Podcasts (2026) → Cloud Games (2026)，每 12-18 个月推出新品类；⑤ **国际扩张**：LATAM + APAC 仍有 2-3 倍增长空间（当前仅占 24%）；⑥ **股价长期表现**：10 年 720% vs S&P 500 298%，证明 "user-centric + long-term thinking" 策略的成功；⑦ **AI 时代护城河**：庞大的用户数据 + 内容 IP + 技术平台 + 人才团队 → 即使 AI 内容生成成熟，Netflix 仍保持独占性优势；⑧ 即使不完成 WBD 并购，Netflix 单独也是美股最优质的 compounder 之一。

## 7. 未来可能的利空

### 利空 1：WBD 收购整合风险 + $422 亿桥接贷款 + 股票回购暂停 + 监管审批不确定

**依据**

- **WBD 收购对价 $720 亿美元全现金**（$27.75/股）
- **桥接贷款 $422 亿美元**（2025/12 初始 $59B → 2025/12/19 调整为 $34B → 2026/01/19 增至 $42.2B）
- **股票回购暂停**：FY25 回购 $91 亿（平均 $300+/股，2025 年 11 月 10:1 拆股后等价 $30+），FY26 暂停积累现金
- **整合复杂度**：
  - WBD 员工 ~27,000 人（vs Netflix ~13,000 人）—— 合并后员工可能 40,000+
  - Discovery Global 剥离（CNN / TNT / Discovery / HGTV 等）需要分开运营后剥离
  - HBO Max 技术平台与 Netflix 技术平台整合
  - 品牌策略（HBO vs Netflix，独立还是合并）
  - 国际市场策略（WBD 在某些国家独家协议 vs Netflix）
- **监管审批**：
  - 美国 FTC：流媒体市场集中度 + 内容供应商合并
  - 欧盟 DG COMP：Media pluralism + 内容多样性
  - 英国 CMA：市场份额
  - 澳洲 / 日本 / 印度等其他市场
  - **预计审批时间 12-18 个月**（最早 2026 Q4 关闭，可能延迟至 2027 Q2）
- **财务影响**：
  - 合并后总债务可能 $400-450 亿（vs 当前 $145 亿）
  - 净债务 / EBITDA 从 0.4x 升至 2.5-3.0x
  - 利息费用可能从 $8 亿 / 年翻倍至 $15-20 亿 / 年
  - 信用评级可能从 BBB+ 下调至 BBB
- **交易风险**：
  - WBD 股东投票：需要多数通过
  - 反垄断诉讼：竞争对手 / 州检察官可能起诉
  - 分手费：如果 Netflix 退出可能面临 $30 亿 + 分手费
- **CEO 股东信警告**（risks）：
  - "(iii) Netflix's and WBD's ability to implement their business strategies"
  - "(vi) the risk that disruptions from the proposed transaction will harm Netflix's or WBD's business"
  - "(xv) failure to receive the approval of the stockholders of WBD"

**解读**

**WBD 收购是 Netflix 历史上最大的战略赌注，风险与机会并存**：① **流媒体行业巨型并购的成功率低于 50%**：Disney-Fox（2019）+ AT&T-Warner（2018）+ WBD-Discovery（2022）都经历了整合困难 + 股价下跌；② **$422 亿桥接贷款**意味着 Netflix 从"净债务 $54 亿"变为"净债务 $380+ 亿"，财务柔性大幅降低；③ **回购暂停**：FY25 回购 $91 亿对 EPS 贡献约 1-2%，FY26 暂停 → EPS 增长依赖于 Op 增长；④ **监管风险**：美国流媒体市场前三（Netflix、Amazon、Disney）合计占 >70% 份额，Netflix + WBD 合并可能面临类似 2021 AT&T-Discovery 的监管挑战；⑤ **整合期（2026-2028）可能有 $10-20 亿重整费用 + 临时 Op Margin 压力**（2026 Guidance 已含 $275M 收购相关费用）；⑥ **文化冲突**：Netflix "flat + data-driven + big-bets" 文化 vs WBD 传统"hierarchical + IP-centric"文化；⑦ **关键员工流失风险**：HBO 的 Casey Bloys + WBD 高管可能离职；⑧ 投资者应警惕 FY26-27 是"整合消化期"，真正协同效应可能 2028+ 才显现；⑨ 如果监管失败或 WBD 股东投票否决，Netflix 股价可能短期承压。

### 利空 2：流媒体竞争加剧 + YouTube 威胁 + Disney/Amazon/Apple/ATT 投入 + AI 时代内容成本

**依据**

- **全球流媒体市场增长放缓**：2025 年全球 SVOD 订阅增速 <10%（vs 2020-2022 年 20-30%）
- **主要竞争对手最新动态**：
  - **YouTube**：
    - 2025 年占美国 TV 份额 13%（vs Netflix 9.0%），**YouTube 仍是第一**
    - 2029 年起奥斯卡全球独播
    - NFL Sunday Ticket 独家（2023-2030）
    - 已成为全球第二大流媒体
  - **Disney+/Hulu**：
    - 2024 年合并成为单一 app
    - 2025 年付费会员 >250M（含 ESPN+ + Hulu）
    - 内容投入 $20+ 亿/年
  - **Amazon Prime Video**：
    - Thursday Night Football + 完整 NBA slate（$1.9B/年）
    - MGM Studios 并入后 IP 库庞大
    - 1.8 亿 Prime 会员（含免费）
  - **Apple TV+**：
    - 每年 $8-10 亿内容投入
    - 原创内容（Ted Lasso, Severance, Shrinking 等）
  - **Paramount+**：
    - NFL + CBS 内容
    - Taylor Sheridan universe（Yellowstone, 1883, Tulsa King 等）
- **内容成本上升压力**：
  - 体育权益每 5 年续约 +50-100%（NBA 11 年 $77B、NFL $110B/11 年）
  - 好莱坞明星片酬膨胀（$20-30M/片）
  - AI 时代创作者 / 工会可能提高版权费用
- **"Peak TV" 可能已过**：2024 年美国原创剧数量 -15%（vs 2022 年高点）
- **US TV 时长份额**：Netflix 9.0% vs YouTube 13% vs 线性电视 40%+，份额争夺激烈

**解读**

**流媒体竞争已进入"头部集中 + 整合"阶段**：① **YouTube 是 Netflix 最大的挑战**：YouTube 拥有用户 20 亿 + 免费 + UGC + Shorts + TV + Podcasts + Live Sports，全方位覆盖，Netflix 在大部分地区仍无法与 YouTube 直接竞争；② **Amazon + Disney + Apple + Paramount** 形成"第 2 梯队"，合计投入 $50+ 亿/年挑战 Netflix；③ **体育权益竞争**：NFL、NBA、MLB、Premier League 续约时，YouTube + Amazon + Apple 直接抬价，Netflix 难以竞争；④ **AI 时代内容成本**：如果编剧 / 演员工会因 AI 版权问题再次罢工（2023 WGA/SAG 罢工前例），Netflix 内容供应链会中断 + 成本上升；⑤ **"Peak TV" 退潮**：美国原创剧从 2022 年 600+ 降至 2025 年 ~500，表明行业在减量增效，但明星级内容的价格进一步抬升；⑥ **新平台涌现**：TikTok 可能进入 long-form、Instagram Reels on TV 挑战用户时长；⑦ **Netflix 优势**：最大会员基础 + 全球发行 + 数据 + 原创能力 + 制作基础设施，但每一项优势都在被逐步蚕食；⑧ 长期看，流媒体市场可能从"Netflix + 5 个对手"变为"Netflix + WBD 合并后 + YouTube + Amazon"四足鼎立局面。

### 利空 3：WBD 并购后债务扩大 + 利息翻倍 + 监管风险 + 国际扩张放缓

**依据**

- **WBD 并购完成后（2026 或 2027）**：
  - Netflix 总债务可能从 $145 亿增至 $400-450 亿
  - 净债务 / EBITDA 从 0.4x 升至 2.5-3.0x
  - **利息费用可能翻倍至 $15-20 亿/年**
  - 信用评级可能从 BBB+ 下调至 BBB
- **WBD 现有债务 $400 亿美元**（FY25 末）：Netflix 并购 WBD 后需要承担 / 再融资
- **美国 10 年期国债收益率 4.2-4.5%**（FY25 末）：债券发行成本 5-6%
- **回购暂停至 WBD 并购完成**：FY26 + FY27 股本增长几乎为零
- **监管合规成本**：
  - 欧盟 Digital Services Act：平台透明度 + 内容审查
  - 英国 Online Safety Act：年龄验证 + 内容分级
  - 印度内容审查：政治 / 宗教敏感内容
  - 美国 FTC：并购审批 + 数据隐私
- **国际扩张放缓信号**：
  - APAC 增长从 +26% (Q4 FY24) 降至 +17% (Q4 FY25)
  - LATAM 增长从 +35% FX-Neutral 降至 +20% FX-Neutral
  - 印度 + 日本 + 韩国市场饱和度上升
- **汇率风险**：2026 Guidance 基于 1/1/2026 汇率，美元走强可能拖累 EPS 3-5%
- **地缘政治风险**：
  - 俄罗斯市场（2022 关闭）
  - 中国大陆市场（从未进入）
  - 中东 / 伊朗 / 沙特市场敏感内容
  - 美中关系对 APAC 业务的影响

**解读**

**Netflix 面临"从低杠杆成长股向高杠杆整合股"的转型风险**：① **$400+ 亿债务意味着 Netflix 需要持续 $80+ 亿 FCF 用于还本付息 + Capex + 内容投入**：任何单年 FCF 下滑 20%+ 都会产生压力；② **信用评级下调**：从 BBB+ 至 BBB 导致借款成本上升 0.5-1%，年化增量利息 $2-4 亿；③ **回购暂停 2-3 年**：对"多年来股价上涨主要来自 EPS 增长（Op 增长 50-60% + 回购 40-50%）"的投资模式产生重大影响；④ **国际扩张放缓**：APAC + LATAM 从"20-30% 增长"降至"10-20% 增长"，意味着 Netflix 需要更多依赖 UCAN + EMEA 的成熟市场价格上涨 + 广告；⑤ **监管合规成本增加 $5-10 亿/年**：合规团队 + 内容审查 + 年龄验证 + 数据本地化（印度、欧盟）；⑥ **AI 时代内容成本重构**：如果 AI 生成内容被工会抵制 → 内容成本上升；如果 AI 生成内容普及 → 传统制作商价值下降，Netflix 可能面临估值压缩；⑦ 长期看，Netflix 估值从当前 $1,100/股（FY26 P/E 约 33x）可能面临"估值重估"压力，尤其是如果 WBD 并购失败或监管审批受阻。

## 8. 总结判断

**一句话结论**

**Netflix FY2025 交出"收入 $452 亿 (+16%)、Op Margin 29.5% (+280 bps)、EPS $2.53 (+28%)、FCF $95 亿 (+37%)、广告收入 +2.5x 至 $15 亿、会员突破 3.25 亿、股票回购 $91 亿、US TV 份额创新高 9%、Stranger Things 最终季 120M 视图、WBD $720 亿美元全现金收购宣布、10:1 拆股完成"的突破性业绩**，**充分展现"全球流媒体龙头 + 内容飞轮 + 广告加速变现 + 产品创新 + 国际扩张 + 资本纪律"的完美结合**。**FY26 核心看点是 WBD 并购完成（重塑行业格局）+ 广告收入翻倍至 $30 亿 + Op Margin 扩张至 31.5% + 会员突破 3.5 亿 + 新品类（Live/Podcasts/Games）扩张 + FCF 达 $110 亿**；**核心风险是 WBD 整合复杂度（$422 亿桥接贷款 + 债务扩大 + 回购暂停）+ 流媒体竞争加剧（YouTube + Amazon + Disney）+ AI 时代内容成本结构 + 国际扩张放缓 + 监管审批不确定**。

**核心看点**

1. **WBD 收购（$720 亿美元全现金）+ HBO Max + Warner Bros. 影视库 + 行业整合**：重塑全球流媒体格局，合并后全球会员可能突破 400M+ / 年营收突破 $500 亿
2. **广告收入 +2.5x 至 $15 亿 + 2026 年翻倍至 $30 亿 + AI-driven 广告平台**：Ad Tier >70M 用户，广告 TAM 巨大，增量 Op Margin >60%
3. **2026 Op Margin 31.5% + FCF $110 亿 + 内容飞轮 + 产品创新（Live/Podcasts/Games）**：Netflix 是美股最完美的 compounder，10 年累计回报 720%

**核心风险**

1. **WBD 收购整合风险 + $422 亿桥接贷款 + 股票回购暂停 + 监管审批不确定（12-18 个月）**：成功率 <50%，财务柔性大幅降低
2. **流媒体竞争加剧 + YouTube 威胁（US TV 份额 13% vs Netflix 9%）+ Disney/Amazon/Apple 内容投入 + AI 时代内容成本**：竞争格局恶化
3. **WBD 并购后债务扩大至 $400 亿+ + 利息费用翻倍至 $15-20 亿 + 信用评级下调 + 国际扩张放缓**：估值重估风险
