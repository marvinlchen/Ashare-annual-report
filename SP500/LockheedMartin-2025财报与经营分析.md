# 【Lockheed Martin Corporation (NYSE: LMT)】2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

- **全年 Sales**：**$750.48 亿美元**，同比 **+6%**（vs FY24 $710.43 亿）；**Q4 $203.21 亿（+9% YoY）**。
- **全年 Business Segment Operating Profit**：**$67.43 亿**（+11% vs $60.83 亿）；**Q4 $20.58 亿（+383% vs $4.26 亿）** —— Q4 大幅反弹因 FY24 Q4 有 $17 亿 Classified Program Losses 拖累。
- **全年 Consolidated Operating Profit**：**$77.31 亿**（+10%）；**Q4 $23.31 亿（+235%）**。
- **全年 Consolidated Operating Margin**：**10.3%**（vs FY24 9.9%，+40 bps）；**Q4 11.5%**（vs 3.7%，+780 bps 反弹）。
- **全年 Pre-tax Income**：$59.22 亿（-5%）—— Pension Settlement Charge $479 M + 高利息 + 非服务 FAS Pension 费用 $874 M（vs FY24 $62 M 收益）拖累。
- **全年 Net Earnings**：**$50.17 亿**（vs FY24 $53.36 亿，**-6%**）—— 含 **Pension Transfer Settlement Charge $479 M（$377 M after-tax，$1.63 EPS）**。
- **全年 Diluted EPS**：**$21.49**（vs FY24 $22.31，**-3.7%**）；**Q4 $5.80**（vs $2.22，+161%）。
- **Effective Tax Rate**：15.3%（vs FY24 14.2%，+110 bps）—— One Big Beautiful Bill Act (Tax Act) FDII 扣除降低；Q4 IRS 审计和解 $109 M 税收收益。
- **全年 Cash from Operations**：**$85.57 亿**（vs FY24 $69.72 亿，**+23%**）。
- **全年 Free Cash Flow**：**$69.08 亿**（+31%）—— 含 **$860 M 养老金提前出资**（完成 FY26 义务）。
- **Backlog 创纪录 $1,940 亿**（FY25 末）：极其强劲的订单储备。
- **FY2026 Outlook**：**Sales $775-800 亿（+3-7%）、Diluted EPS $29.35-30.25（+37-41%）、Business Segment Op Profit $84.25-86.75 亿（+25%）、Free Cash Flow $65-68 亿**。

### 1.2 增长与拖累因素

- **主要增长驱动**：
  - **Missiles and Fire Control (MFC) 爆发式增长**：全年营收 **$144.50 亿（+14%）**、Op Profit **$19.89 亿（+382%）**、Margin **13.8%（vs 3.3%）**：
    - **Tactical & Strike Missiles +$14 亿**：JASSM、LRASM、Precision Fires；
    - **Integrated Air & Missile Defense +$4.5 亿**：**PAC-3 防空导弹**（乌克兰、以色列、中东、欧洲多国订单）；
    - FY24 的 $14 亿 Classified Program Loss 比较基数效应；
    - **MFC 已成为公司最强增长引擎**，市场需求远超产能；
  - **Aeronautics 营收增长稳健**：$302.57 亿（+6%）：
    - **F-35 项目 +$19 亿**：生产 + 维护合同持续增长（2025 交付 ~145 架 F-35）；
    - Classified Contract 负面影响部分抵消；
  - **Space 分部 Op Profit +10%**：
    - **NGI (Next Generation Interceptor) + FBM (Fleet Ballistic Missiles) +$3.8 亿**：战略弹道导弹防御；
    - **Orion 载人航天器 +$2.55 亿**：NASA Artemis 计划加速；
    - Margin 从 9.8% → 10.3%（+50 bps）；
  - **PAC-3 七年框架协议**（2026/Q1）：Department of War (DoW) Acquisition Transformation Strategy 标杆合同，**预计合同价值 $150-250 亿 +**，锁定长期产能；
  - **Cash from Ops +23%**：工作资本大幅释放（Accounts Payable +$13.4 亿、Contract Liabilities +$12.2 亿）+ **Tax Act R&D 扣除** 降低税款支出；
  - **Backlog $1,940 亿创历史新高**：**2.58x Sales Ratio**（健康水平 FY24 $1,760 亿）；
  - **FY26 EPS 指引 $29.35-30.25（+37-41%）**：反映 FY25 Reach-forward Losses $16.15 亿 已一次性消化；
  - **Q4 IRS 审计和解 $109 M 税收收益**；
  - **Q4 2026 Operation Absolute Resolve**：F-35、F-22、RQ-170、Sikorsky Black Hawk 参与美国军事行动，**实战验证 Lockheed 武器系统**。
- **主要拖累因素**：
  - **Reach-forward Losses $16.15 亿 Q2 2025 集中计提**（after-tax $12.13 亿 / $5.17 EPS）：
    - **Aeronautics Classified Program $9.50 亿**（$7.13 亿 after-tax / $3.04 EPS）；
    - **RMS CMHP (Canadian Maritime Helicopter Program) $5.70 亿**（$4.28 亿 after-tax / $1.83 EPS）；
    - **RMS TUHP (Turkish Utility Helicopter Program) $9,500 万**（$7,100 万 after-tax / $0.30 EPS）；
    - **Aeronautics Fixed Asset Write-off $6,600 万** + 其他；
  - **Rotary and Mission Systems (RMS) 营收持平、Op Profit -31%**：
    - RMS Op Profit $13.23 亿（vs $19.21 亿）、Margin 7.6%（vs 11.1%，**-350 bps**）；
    - CMHP + TUHP Reach-forward Losses 为主因；
    - Seahawk + Littoral Combat Ship (LCS) 销量下降；
  - **Q4 Pension Settlement Charge $479 M**（$377 M after-tax / $1.63 EPS）：
    - 转让 **$9.43 亿养老金义务及资产到保险公司**（pension risk transfer）；
    - 虽一次性损失但**降低长期养老金风险**；
  - **Non-service FAS Pension 费用 $8.74 亿**（vs FY24 收益 $0.62 亿）：利率上升导致养老金负债贴现率变化；
  - **Interest Expense +7.9% 至 $11.18 亿**：Long-term Debt +$9.05 亿；
  - **Aeronautics Op Margin 下降至 6.9%**（vs 8.8%）：Classified Program Loss + C-130 项目不利调整 $140 M；
  - **ULA (United Launch Alliance) 股权收益下降 -$40 M**：波音 + Lockheed 50/50 合资；
  - **Tax Rate 上升 +110 bps**：Tax Act 降低 FDII 扣除；
  - **FY25 EPS -3.7%** 是 Lockheed 15 年来少数下降年份，但 **基本面 FY26 强势回升**。
- **资本 / 股东回报**：
  - **FY25 Dividends Paid $31.31 亿**（+2.4% vs FY24 $30.59 亿）；
  - **Q4 Dividend Rate $3.45/股**（年化 $13.80，FY25 总 $13.20 估算）；
  - **Stock Repurchases $30 亿**（6.6M 股，均价 ~$454）—— 减少 2.8% 流通股；
  - **Total Capital Return $61.31 亿** = Net Earnings 122%（超过净利回馈）；
  - **2 万亿美元以上资产的军工龙头公司 + 23 年连续股息增长**（Dividend Aristocrat 候选）；
  - **Pension Contribution $860 M** 提前完成 FY26 义务 —— 2025 年资产负债表优化；
  - **$2 亿 Long-term Debt 净发行**：继续维持 Investment Grade 信用（A3/A-）。

### 1.3 趋势判断

- **未来主要利好方向**：**Trump 2.0 国防预算上升**（FY26 国防预算 $8,950-9,000 亿 +5-7%）、**Golden Dome Missile Defense 计划**（Trump 2025 签署 Executive Order，$100-200 亿 长期投入）、**PAC-3 七年框架协议 $150-250 亿**、**F-35 F-47 下一代战斗机延续**、**乌克兰 + 以色列 + 中东 + 欧洲 PAC-3/THAAD/HIMARS 国际需求爆发**、**NGI / FBM 战略核威慑现代化**、**Orion 载人航天 + Artemis 计划加速**、**Sikorsky Black Hawk 替代 + TUHP 合同完善**、**BrainSense aDBS 神经调节 + 医疗技术投资**（通过 Sikorsky）、**Backlog 创纪录 $1,940 亿提供强可见度**、**FY26 EPS 指引 +37-41% 反映 Reach-forward Losses 已一次性消化**。
- **未来主要压力方向**：**固定价格合同 Cost Overrun 风险持续**（FY25 $16 亿 Reach-forward Losses，FY24 $20 亿，已成为重大风险）、**F-35 Block 4 + TR-3 升级延迟 + 成本上升压力**、**Department of War Acquisition Transformation Strategy 变革风险**（合同模式变化）、**RMS Sikorsky 业务盈利能力持续弱化**、**Classified Aeronautics 项目持续亏损**（FY24 $5.55 亿 + FY25 $9.50 亿 累计 $15 亿）、**Pension 风险 + 加息导致 Non-service Pension Expense 高位**、**Supply Chain 稀土 + 关键组件短缺**（钛金属 + 半导体 + 集成电路）、**中国 / 俄罗斯军工技术赶超**、**FTC/DOJ 反垄断审查 + Congressional F-35 预算审议**、**劳工短缺 + IT 网络攻击 + Cyber Security 风险**。
- **整体趋势判断**：Lockheed Martin FY2025 录得 **Sales +6%、Business Segment Op Profit +11%、Backlog $1,940 亿创纪录、Free Cash Flow +31% 至 $69 亿**，但 **EPS -3.7% 至 $21.49** 受 **Reach-forward Losses $16.15 亿 + Pension Settlement $479 M** 拖累。**核心业务基本面极其强劲**：**MFC 营收 +14%、Op Profit +382%、PAC-3 七年框架协议** 锁定长期产能。**FY26 Guidance EPS $29.35-30.25（+37-41%）+ Business Segment Op Profit +25% + Sales +3-7%** 反映业绩正在强势回归。**Forward P/E 15-17x** 处于近年周期性偏低位（S&P 500 22x），反映市场对 Fixed-price Contract 风险的担忧。**但考虑 Backlog $1,940 亿（2.58x Sales）+ 23 年连续股息增长 + $200-300 亿 Golden Dome 机会 + PAC-3/JASSM/LRASM/THAAD 需求爆发**，**FY27-28 EPS 有望突破 $35+**，股价当前 **$450-480 对应 FY27 EPS $35 = 13-14x P/E** 极具吸引力。**CEO Jim Taiclet 领导下 "21st Century Security + All-Domain Mission" 战略执行** 提供长期愿景。**Operation Absolute Resolve 的实战表现** 是 FY26 Q1-Q2 关键催化剂。**国防股中最具防御属性 + 最强 Backlog 支撑 + 最高 Capital Return 的核心配置**。

## 2. 主营业务

### 2.1 公司简介

Lockheed Martin Corporation（NYSE: LMT）成立于 **1912 年**（Lockheed Aircraft Company 前身），**1995 年 Lockheed Corporation + Martin Marietta 合并成立**，总部位于美国 **Maryland Bethesda**，是 **全球最大的国防承包商 + 航空航天公司**。**董事长、总裁兼 CEO James D. "Jim" Taiclet**（2020/6 接任 Marillyn Hewson）、**CFO Evan Scott**（2024/11 接任 Jay Malave）、**COO Frank A. St. John**。全球员工约 **122,000 人**（FY25 末），**80% 营收来自美国政府（Department of War, formerly Department of Defense）**，**15% 来自国际客户**，**5% 来自商业 + 民用政府客户**。市值约 **$1,000-1,100 亿**（FY25 末），**S&P 500 成员、Dow Jones Industrial Average 组成公司**（2022 加入）。

**2025 年 10 月美国国防部更名为 "Department of War" (DoW)**。**Trump 2.0 "Golden Dome" Missile Defense Shield**（2025/1 Executive Order）成为 Lockheed 未来 5-10 年最大增长催化剂。**CEO Jim Taiclet 提出 "21st Century Security" 和 "All-Domain Mission" 战略**，强调跨空中 + 陆地 + 海上 + 太空 + 网络空间的多域集成解决方案。Lockheed **2025 投资 $35 亿** 于生产扩能和下一代技术，**IRAD (Independent Research and Development) $20 亿**。

### 2.2 主营业务构成

Lockheed Martin FY25 分为 **4 大 Reportable Segments**：

1. **Aeronautics (空中平台, 占 FY25 Sales 40.3%)**：
   - **F-35 Lightning II**（**核心旗舰产品**，美国、英国、日本、澳大利亚、韩国、以色列、荷兰等 20+ 国家）；
   - **F-22 Raptor** (空优战斗机, 已停产, 现为维护/升级业务)；
   - **F-16 Fighting Falcon** (多国采购 + 升级)；
   - **C-130J Super Hercules** (战术运输机)；
   - **C-5M Super Galaxy** (战略运输机, 升级维护)；
   - **U-2 Dragon Lady** (侦察机) + Classified (分类保密) 项目；
   - **Skunk Works** (先进计划开发, 开发 Next Generation Air Dominance F-47)；
2. **Missiles and Fire Control (MFC, 占 FY25 19.3%)** —— **增长最快分部**：
   - **PAC-3 (Patriot Advanced Capability-3)** 防空反导系统；
   - **THAAD (Terminal High Altitude Area Defense)** 末端高空防御；
   - **HIMARS (High Mobility Artillery Rocket System)** 多管火箭炮 + Extended Range；
   - **JASSM (Joint Air-to-Surface Standoff Missile)** + **LRASM (Long Range Anti-Ship Missile)**；
   - **Precision Fires (Hellfire, Javelin, LMAMS)**；
   - **Hypersonic Weapons (CPS, LRHW, ARRW)** 高超音速武器；
   - **MFC 受益于乌克兰战争 + 中东冲突 + 印太威慑 + 台海局势** 的导弹需求爆发；
3. **Rotary and Mission Systems (RMS, 占 FY25 23.1%)**：
   - **Sikorsky Helicopters**：
     - **UH-60 Black Hawk** (美国陆军 + 国际 100+ 用户)；
     - **MH-60 Seahawk** (美国海军)；
     - **CH-53K King Stallion** (美国海军陆战队重型运输)；
     - **S-92** + **S-97 Raider X** (商业 + 军用)；
   - **Integrated Warfare Systems and Sensors (IWSS)**：
     - **Aegis Combat System** (舰载防空反导)；
     - **AN/SPY-1 / SPY-6 雷达系统**；
     - **Littoral Combat Ship (LCS)** (美国海军) + **River Class Destroyer (RCD)** (加拿大皇家海军 CSC)；
   - **Training and Logistics Solutions (TLS)**；
   - **Cyber and Information Management**；
4. **Space (占 FY25 17.4%)**：
   - **战略 + 导弹防御**：
     - **NGI (Next Generation Interceptor)** 下一代拦截器；
     - **FBM (Fleet Ballistic Missiles)** 三叉戟潜射弹道导弹；
   - **国家安全太空 (National Security Space)**：
     - **OPIR (Overhead Persistent Infrared)** + **Transport Layer** (SDA);
     - **GPS III** 导航卫星；
   - **商业 + 民用太空**：
     - **Orion** (NASA Artemis 载人航天器);
     - **Commercial Communications Satellites**;
   - **United Launch Alliance (ULA, 50% 合资 with Boeing)** 火箭发射服务。

### 2.3 核心收入来源

- **F-35 + F-16 + F-22 等战斗机平台 ~$200 亿/年（27%）**：美国 + 盟国主力；
- **PAC-3 + THAAD + HIMARS + JASSM + LRASM 导弹系统 ~$145 亿/年（19%）**：**增长最快**；
- **Sikorsky Black Hawk + Seahawk + CH-53K 直升机 ~$85 亿/年（11%）**；
- **IWSS + Aegis + LCS/RCD ~$80 亿/年（11%）**；
- **NGI + FBM + GPS + Orion + OPIR ~$130 亿/年（17%）**；
- **Classified 保密项目 ~$100 亿/年（13%）**（Skunk Works + Special Projects）；
- **F-35 Sustainment & Upgrades ~$40 亿/年（5%）** -- 长期高毛利收入来源。

  - 2025 $35 亿 投资 产能扩张 + 下一代技术；
  - 2025 $20 亿 IRAD；
  - 2025 $8.60 亿 Pension Contribution 提前完成 FY26 义务；
  - 2025 LT Debt Issuance $19.85 亿 维持资本灵活性；
- **FY26 Free Cash Flow Guidance $65-68 亿**（略低 vs FY25 $69 亿，主因 Capex +52-70% 至 $25-28 亿扩产）；
- **Pension De-risking**：
  - $9.43 亿 Pension Transfer 到保险公司（Q4 2025）；
  - Accrued Pension Liabilities -18% 至 $39.15 亿；
  - 长期降低 Non-service Pension Expense 波动。

**解读**

- **短期（FY26）**：
  - Free Cash Flow $65-68 亿，Cash from Ops $91-94 亿（+7-10%）；
  - Capital Return 预计 $60-70 亿（股息 +5% + 回购 $30-35 亿）；
  - **Dividend +5-6% 大概率**；
- **中长期（FY27-30）**：
  - **Backlog 提供 2.5+ 年可见度**；
  - FY30 Free Cash Flow 有望达到 $75-85 亿；
  - **累计股东回馈 FY26-30 $300-350 亿**；

## 7. 未来可能的利空

### 利空 1：Fixed-price Contract 成本超支 + Reach-forward Losses 持续风险

**依据**

- **FY25 Reach-forward Losses $16.15 亿**（$12.13 亿 after-tax / $5.17 EPS 拖累）：
  - **Aeronautics Classified Program $9.50 亿**；
  - **RMS CMHP $5.70 亿**；
  - **RMS TUHP $0.95 亿**；
  - 加上 Fixed Asset Write-off $66 M = **$16.81 亿 Consolidated Op Profit 负面影响**；
- **FY24 Reach-forward Losses $19.65 亿**：
  - MFC Classified Program $14.10 亿；
  - Aeronautics Classified Program $5.55 亿；
- **累计 FY24-25 $35.80 亿 Reach-forward Losses** —— 占 FY24-25 Business Segment Op Profit 总额 $127.26 亿 的 **28%**；
- **FY25 Aeronautics Classified Program Margin 5.4% → 6.9%** 反映公司进展有限；
- **C-130 项目 FY25 $1.40 亿不利调整**；
- **CMHP 加拿大海事直升机**（Sikorsky CH-148 Cyclone）：固定价格合同 + 技术复杂度超预期 + 加拿大采购成本上涨；
- **TUHP 土耳其通用直升机**（S-70 Black Hawk International）：地缘政治 + 供应链复杂；
- **Aeronautics Classified Program**（估计为 Next Generation Air Dominance 隐身战斗机 + 无人机项目）：
  - 技术创新 + 固定价格合同 + 成本估算困难；
  - **未来 FY26-27 可能继续计提 $3-5 亿 Reach-forward Losses**。

**解读**

- **短期（FY26）**：
  - FY26 Guidance 已反映基本情景，**Reach-forward Losses 可能收敛至 $5-8 亿**；
  - 如 Classified Aeronautics 再次计提 $5 亿+，**FY26 EPS 可能低于指引下限**；
- **中期（FY27-28）**：
  - Department of War Acquisition Transformation Strategy 将转向 **更多 Cost-Plus 合同**，**降低 Fixed-price 风险**；
  - 但老 Fixed-price 合同（CMHP、TUHP、某些 Classified）仍需完成；
  - **预计 FY26-28 累计 $10-20 亿 Reach-forward Losses**；
- **Mitigation**：
  - Lockheed 持续改善 Program Management + Cost Estimation；
  - 新合同 Cost Transparency 和 Profit Sharing 条款；
  - 但**类似 RMS Sikorsky 业务结构性问题**（多年连续亏损）需要长期解决；
- **对 Op Margin 压力**：如果 Reach-forward Losses 每年 $5-8 亿，则对 Business Segment Op Margin 产生 **70-100 bps 拖累**。

### 利空 2：Department of War 预算 + Acquisition 模式变化风险

**依据**

- **FY26 美国国防预算 $8,950-9,000 亿**：
  - 虽然总量增长 +5-7%，但 **资源优先级调整**；
  - **Trump 2.0 优先级**：Golden Dome、Space Force、Hypersonic、AI/Autonomous；
  - **降低优先级**：部分 Classified + Legacy Programs（F-22 退役延迟 + C-130 订单调整）；
- **Department of War Acquisition Transformation Strategy (2025-)**：
  - 强调 **"Faster, Better, Cheaper" 获取模式**；
  - **Fixed-price + Milestone-based 合同** 成为主流（对供应商更不利）；
  - **Open Architecture + Competition** 打破 Lockheed 传统独家供应商地位；
  - **Software-centric acquisition** 可能降低硬件溢价；
- **Continuing Resolution (CR) + Government Shutdown 风险**：
  - 2025 年已有短期政府关门事件；
  - **2026 Congressional 预算谈判** 仍有不确定性；
- **Congressional F-35 审议**：
  - F-35 项目总成本 $1.7 万亿（生命周期），**持续受到国会审查**；
  - **Sustainment 成本超预期**可能导致交付量调整；
- **Executive Order 风险**：
  - Trump 政府可能通过 Executive Order 调整采购政策；
  - Mass Layoffs / Program Cancellations 风险。

**解读**

- **短期（FY26-27）**：
  - 美国国防预算总量支持，Lockheed 核心项目（F-35、PAC-3、NGI、Orion）安全；
  - **Golden Dome Executive Order 提供新增长机会**；
  - **FY26 Outlook Excludes Government Shutdown Impact**（管理层谨慎）；
- **中期（FY28-30）**：
  - **Acquisition Transformation 长期影响**：Fixed-price 风险增加、Margin 可能压缩 50-100 bps；
  - **Open Architecture 可能增加新竞争者**：Boeing, Raytheon RTX, General Dynamics 争夺；
  - **Software-centric + AI + Commercial-off-the-shelf (COTS)** 打破 Lockheed 硬件护城河；
- **Mitigation**：
  - Lockheed 80 年政府采购经验 + 关系；
  - Backlog $1,940 亿锁定 2.5+ 年；
  - 技术创新 + Skunk Works 保持前沿地位；
- **风险敏感度**：
  - 如果 DoW 预算削减 10%，Lockheed 营收可能下降 5-8%；
  - EPS 影响 -$5-8。

### 利空 3：Supply Chain 稀土 + 关键组件 + 劳工 + Cyber Security

**依据**

- **钛金属**（F-35 / F-22 主要原料）：
  - 全球 40% 钛来自俄罗斯（VSMPO-AVISMA）；
  - 制裁后 Lockheed 转向美国 TIMET + Allegheny Technologies；
  - **价格上涨 30-50%**（2022-2025）；
- **稀土元素**（永磁体 + 传感器）：
  - **全球 75% 稀土加工在中国**；
  - 2024-2025 中国逐步收紧稀土出口限制；
  - **Gallium、Germanium、Neodymium** 对 Lockheed 电子系统关键；
  - 美国 Mountain Pass Mine + Pentagon 战略储备 部分抵消；
- **半导体 + 集成电路**：
  - **军用级芯片供应链薄弱**；
  - TSMC Arizona 工厂投产支持，但 2027-2028 才全面产能；
  - **商用级芯片军用化替代** 风险；
- **劳工短缺 + IAM 罢工风险**：
  - Lockheed 122,000 员工中 ~30,000 IAM (International Association of Machinists) 工会；
  - 2024 Boeing 机械师罢工 53 天影响行业；
  - **FY26 Lockheed 工会合同谈判** 可能导致薪资上涨 +15-20%；
- **Cyber Security 风险**：
  - Lockheed 长期是 APT 组织（国家级黑客）攻击目标；
  - F-35 + Classified Programs 数据泄露风险；
  - 2024-2025 多起供应商 Cyber 事件；
- **中国 + 俄罗斯军工技术赶超**：
  - **中国 J-20 + J-35 隐身战斗机**；
  - **俄罗斯 S-400 / S-500 防空导弹**；
  - **Hypersonic 武器竞赛**：中国 DF-17、俄罗斯 Kinzhal。

**解读**

- **短期（FY26）**：
  - Supply Chain 问题已部分缓解（2023-2024 高峰后）；
  - 美国本土钛产能扩张；
  - 但稀土依赖 + 半导体脆弱性持续；
- **中期（FY27-30）**：
  - 如果中美关系恶化 + 稀土完全断供，Lockheed 生产成本 +10-15%；
  - 军用级芯片供应链重建需要 5-10 年；
  - **Cost Inflation 压力持续 +3-5%/年**；
- **Mitigation**：
  - **$35 亿/年 Production Capacity 投资**；
  - **Nearshoring + Friend-shoring**（美国、加拿大、日本、韩国、欧洲）；
  - **Government Strategic Reserve 储备**；
  - **AI + Automation 降低劳工依赖**；
- **长期 Resilience**：
  - Lockheed 规模 + 政府优先级 使其在 Supply Chain 困境中相对优势；
  - 但**中小型 Supplier 破产风险** 影响 Lockheed 交付节奏。

## 8. 总结判断

**一句话结论**

Lockheed Martin FY2025（2025/1/1-2025/12/31）录得 **Sales $750.48 亿 (+6%)、Business Segment Op Profit $67.43 亿 (+11%)、Backlog $1,940 亿创纪录、Free Cash Flow $69.08 亿 (+31%)**，但 **Net Earnings $50.17 亿 (-6%)、EPS $21.49 (-3.7%)** 受 **Reach-forward Losses $16.15 亿（Aeronautics Classified + RMS CMHP/TUHP）+ Pension Settlement Charge $479 M** 拖累。**MFC 分部爆发式增长 +14% 营收 + Op Profit $19.89 亿 (+382%) + Margin 13.8%**（乌克兰、以色列、印太需求 + PAC-3/JASSM/LRASM/Precision Fires），**Q4 FY25 Sales +9% + Business Segment Op Profit +383%** 反弹强劲。**FY26 Guidance Sales $775-800 亿 (+3-7%)、Diluted EPS $29.35-30.25 (+37-41%)、Business Segment Op Profit +25%** 反映业绩正在强势回归。**Golden Dome Missile Defense + PAC-3 七年框架协议 + F-35/F-47 延续 + 国际订单爆发** 是核心增长驱动。**Backlog / Sales Ratio 2.58x + 23 年连续股息增长 + Capital Return 122% 净利回馈** 提供财务安全边际。**Forward P/E 15-17x** 处于近年周期性偏低，**CEO Jim Taiclet "21st Century Security + All-Domain Mission" 战略** 执行良好。**长期定位：国防股中最强 Backlog + 最高 Capital Return + 最稳 Defensive 属性的核心配置，FY27-28 EPS 有望突破 $35+**。

**核心看点**

1. **Golden Dome Missile Defense + PAC-3 七年框架协议**：Trump 2.0 EO + DoW 采购改革，FY26-30 累计 $300-500 亿 Lockheed 订单，MFC 营收 CAGR +10-15%。
2. **F-35 + F-47 + 国际订单爆发**：FY26 交付 156 架、国际订单 (波兰、希腊、罗马尼亚、新加坡、捷克) 锁定，Sustainment 市场 $500-1,000 亿生命周期价值。
3. **MFC 爆发式增长 +382% Op Profit**：JASSM + LRASM + Precision Fires + PAC-3 + THAAD + Hypersonic 六大产品线全线爆发，乌克兰/以色列/台海消耗持续驱动。
4. **Backlog 创纪录 $1,940 亿 + Contract Liabilities +17%**：订单储备提供 2.5+ 年可见度，客户预付款大幅增加反映需求强劲。
5. **Capital Return 122% 净利 + 23 年连续股息增长**：Dividend Yield ~2.9% + Buyback Yield ~1.5% + EPS FY26 +37-41%，Total Shareholder Yield ~4-5%。

**核心风险**

1. **Fixed-price Contract 成本超支**：FY24-25 累计 $35.80 亿 Reach-forward Losses (Aeronautics Classified + CMHP + TUHP)，FY26-27 可能继续 $5-8 亿/年计提。
2. **Department of War Acquisition Transformation**：Fixed-price + Milestone-based 合同成为主流，Open Architecture 增加竞争，Lockheed Margin 可能压缩 50-100 bps。
3. **F-35 Block 4 + TR-3 升级延迟 + Classified Program 延期**：技术复杂度 + 软件问题 + 国会审议，可能影响未来订单节奏。
4. **Supply Chain 稀土 + 钛 + 半导体 + 劳工**：中美关系恶化 + 稀土断供风险 + IAM 工会合同谈判 + 军用级芯片脆弱性；Cyber Security APT 攻击持续。
5. **Pension + Non-service FAS Expense 波动**：FY25 $8.74 亿 Non-service Pension 费用（vs FY24 $0.62 亿收益），利率敏感；FY26 Pension Transfer 后风险下降但非零。
