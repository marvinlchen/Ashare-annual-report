# McDonald's Corporation (MCD) 2025年财报整理

> 报告期：FY2025 全年（12 个月，止 2025/12/31）
> 申报币种：美元；会计准则：U.S. GAAP，并披露 Non-GAAP（恒定汇率、剔除重组费用的调整后 EPS）
> 数据来源：McDonald's Q4 & FY2025 Earnings Release（2026/02/11 发布的 Form 8-K Exhibit 99.1）
> FY25 期间主要战略事件："Accelerating the Organization" 重组项目持续（FY25 税前重组费用 $2.29 亿）、Q4 强劲复苏（全球可比销售 +5.7%）、季度股息 +5% 至 $1.86/股（连续 49 年增长）、70 个市场忠诚度计划会员销售额突破 $370 亿（+20%）

## 1. 业绩总结

### 1.1 收入与利润表现

- 全年总收入 **$268.85 亿美元**，同比 +4%（恒定汇率 +2%）；全年系统销售额 **突破 $1,390 亿美元 +7%（恒定汇率 +5%）**
- 全年合并营业利润 **$123.93 亿美元 +6%（恒定汇率 +4%）**；**营业利润率 46.1%**（+90 bps）—— 行业内顶级水平
- 全年 GAAP 归母净利润 **$85.63 亿美元 +4%**；**GAAP 摊薄 EPS $11.95 +5%**；**调整后 EPS $12.20 +4%**（剔除 $0.25 重组费用）
- **Q4 强劲复苏**：单季营收 $70.09 亿 +10%、营业利润 $31.56 亿 +10%、全球可比销售 **+5.7%（vs Q4 2024 +0.4%）**
- 经营现金流 CFFO **$105.51 亿美元 +11.7%**；FCF $71.86 亿 +7.7%；股东回报 $71.71 亿（分红 $51.15 亿 + 回购 $20.56 亿）
- 季度股息 +5% 至 **$1.86/股（连续 49 年增长）**

### 1.2 增长与拖累因素

- **主要增长驱动**：① **Q4 全球可比销售 +5.7%**（美国 +6.8%、IOM +5.2%、IDL +4.5%），创年内最强单季复苏；② **全球忠诚度会员销售额 +20% 至 $370 亿美元**，活跃用户 +19% 至 2.1 亿人（70 个市场）；③ **美国分部 Q4 +6.8%**（vs 2024 Q4 -1.4% 的深度反转）—— "Value Leadership" 价值策略 + McValue Menu + $5 套餐见效；④ **IOM 领涨市场**：英国、德国、澳大利亚强劲恢复；⑤ IDL 日本市场突出；⑥ 加盟餐厅收入 +5% 至 $165.48 亿美元，特许经营利润率提升
- **主要拖累因素**：① 全年可比销售 +3.1%（主要受上半年拖累）：美国 +2.1% 为三大分部最弱；② **重组费用 $2.29 亿美元**（"Accelerating the Organization"内部组织现代化项目持续）；③ **公司自营餐厅销售 -1% 至 $96.90 亿美元**：汇率 + 部分市场（如以色列）运营收缩 + 自营转加盟比例提升；④ 利息支出 +5% 至 $15.82 亿美元（利率 + 新增债务）；⑤ 有效税率 +50 bps 至 21.4%；⑥ **股东权益为负 -$17.91 亿美元**（长期回购累积的财务结构特征）
- **一次性/异常项目**：重组费用 $2.29 亿美元（$0.25/股；FY24 $2.91 亿 $0.33/股）；FY24 包含韩国业务出售净收益 + 以色列业务收购交易费用 $7,000 万；其他营业收入/支出净额 FY25 仅 $200 万（vs FY24 $1.39 亿）

### 1.3 趋势判断

- **未来主要利好方向**：① **Q4 复苏势头延续至 FY26**：全球可比 +5.7% + 各分部全面正增长显示价值策略已产生实质性拉动；② **忠诚度计划放量**：$370 亿忠诚销售仅占系统销售 27%，未来 3 年有望突破 50%；③ **"Accelerating the Organization"组织重组完成后**将释放约 $5 亿年化效率红利；④ **全球门店扩张**：目标 2027 年达到 50,000 家门店（FY25 超 45,000 家）；⑤ 美国市场 $5 Meal Deal + 数字化业务持续发力；⑥ 中国市场（CITIC 持股 52%）通过 FCA 特许经营模式快速扩张
- **未来主要压力方向**：① **低收入消费者持续承压**：美国 <$50K 年收入客群快餐支出压力，McDonald's 作为平价品牌受益但也面临 Wendy's、Burger King、Raising Cane's 等竞争；② **GLP-1 减肥药影响**：Ozempic、Wegovy 长期可能降低快餐消费频次（约 2-5% 影响）；③ **Israel / 中东市场持续压力**：2023/10 起抵制运动对 IOM 增长仍有负面影响；④ **最低工资上涨**：加州 FAST Act ($20/hr) + 其他州联动对加盟商利润率压力；⑤ 竞争对手：Chipotle、Starbucks、Wendy's（Baconator）、Chick-fil-A、Starbucks 等
- **整体趋势判断**：**McDonald's FY25 交出"营收 +4%、EPS +5%、Q4 可比 +5.7%、系统销售 $139B、忠诚度会员 +20%、现金流 $106 亿、股息连续 49 年增长"的标杆业绩**，证明了**价值领导战略 + 全球加盟网络 + 数字化忠诚度**三位一体的商业模式韧性。Q4 的强劲复苏（尤其美国 +6.8%）是**最重要的经营拐点**。**作为标普 500 消费必需品/可选消费中最具防御性 + 股息持续增长（Dividend Aristocrat）的标的**，适合长期持有

## 2. 主营业务

### 2.1 公司简介

McDonald's Corporation 是全球最大的快餐餐厅连锁公司，拥有超过 **45,000 家门店，分布在 100 多个国家**，**约 95% 为特许经营**（franchisee-operated）。公司采用"**资产较轻 + 高利润率**"的特许经营 + 房地产模式：总部位于美国芝加哥，1955 年由 Ray Kroc 在伊利诺伊州 Des Plaines 开出第一家餐厅。核心资产是**Golden Arches 金拱门品牌 + Big Mac/McNuggets/McFlurry 等经典 SKU + 遍布全球的房地产组合**。

### 2.2 主营业务构成

按 3 大分部（地理）划分：

1. **U.S.（美国分部）- FY25 系统销售约占全球 38%**：
   - 约 13,500+ 家门店（其中 95% 为加盟），是 McDonald's 最重要的单一市场
   - FY25 可比销售 +2.1%（全年）/ Q4 +6.8%（显著反弹）
   - 核心策略：**Value Leadership + McValue Menu + $5 Meal Deal**；数字化订餐 + Drive-Thru + 配送 3 大渠道
   - 主要竞争：Burger King（RBI）、Wendy's、Chick-fil-A、Sonic、Jack in the Box

2. **International Operated Markets (IOM 国际直营市场) - FY25 系统销售约占全球 38%**：
   - 涵盖 **英国、德国、法国、加拿大、澳大利亚、意大利、荷兰、西班牙**等 8 大核心市场
   - FY25 可比销售 +3.2%（全年）/ Q4 +5.2%；英国、德国、澳大利亚领涨
   - 混合门店模式：Australia（约 80% 加盟）、UK（约 70% 加盟）、Germany（多数加盟）

3. **International Developmental Licensed Markets (IDL 国际发展特许市场) - FY25 系统销售约占全球 24%**：
   - 包含 **中国（与 CITIC/凯雷合资，CITIC 持股 52%）、日本（持股 35%）、韩国、印度、拉美、中东、非洲**等 100+ 国家
   - FY25 可比销售 +4.6%（全年）/ Q4 +4.5%；**日本市场领涨**
   - **中国市场**通过 FCA（Franchising Company Agreement）模式快速扩张，5,000+ 家门店
   - 2023/12 收购以色列业务；2023 出售韩国业务

### 2.3 核心收入来源

McDonald's 营收 **高度依赖特许经营业务（61.5%）**，公司自营餐厅（36.0%）和其他收入（2.4%）共同构成。收入构成如下：

| 收入类型 | FY25 金额 | 占比 | 同比增速 | 核心驱动 |
|---|---:|---:|---:|---|
| **加盟餐厅收入** | $16,548M | **61.5%** | +5% | 系统销售 +7% + 特许费率 |
| **公司自营餐厅销售** | $9,690M | 36.0% | -1% | 门店数量减少 + 韩国 / 以色列影响 |
| **其他收入** | $647M | 2.4% | +53% | 科技授权 + 数字服务 |
| **总收入** | **$26,885M** | 100% | +4% | |

**关键商业模式点**：McDonald's 不是传统"卖汉堡的公司"，而是**全球最大的商业地产出租 + 特许经营费收集者**。**加盟餐厅营业费用** $2,618M（FY25）包含特许房地产租金 + 特许费用，利润率极高（加盟收入毛利 $13,930M / 收入 $16,548M = **84.2%**）。

**资产负债率变化**

McDonald's 的财务结构极具特色 —— **股东权益为负值**（-$17.91 亿美元），但这 **并非财务困境** 而是长期股票回购积累的结果：① McDonald's 累计回购了远超其累积利润的股票，导致账面权益为负；② 这反映了公司"利用高 FCF + 稳定低杠杆债务 + 持续回购"的长期资本回报策略；③ FY25 权益赤字从 -$38 亿缩窄至 -$18 亿（+52.8% 改善），反映 FY25 净利润 $86 亿 - 股息 $51 亿 - 回购 $21 亿 = 积累约 $14 亿；④ 对于这种"资产较轻 + 高 FCF + 稳定特许费收入"的轻资产模式，传统资产负债率指标意义不大。**实际偿债能力应看 CFFO / 长期债务比率 = $10,551M / $39,973M = 26.4%，属于极健康水平**。

**偿债压力判断**

偿债压力极低：① **CFFO $10,551M / 长期债务 $39,973M = 26.4%**（5 年内可偿还全部债务）；② 标普 / 穆迪 **BBB+ / Baa1** 评级；③ FY25 利息覆盖倍数 = $12,393M 营业利润 / $1,582M 利息支出 = **7.8x**，健康水平；④ 每年 CFFO $100+ 亿美元提供充足灵活性；⑤ 长期债务持续 refinancing 利率环境下小幅上升可控；⑥ 经营租赁负债 $14,147M（门店长期租约）带来稳定性而非压力。

**商誉风险判断**

McDonald's 资产负债表中**没有显著的商誉单独披露**（新闻稿未披露细项），**商誉风险极低**：① 主要资产是全球自持物业 / 门店 / 长期经营租赁使用权 ($14,606M) + 加盟品牌无形资产；② FY25 未计提商誉减值；③ 韩国业务剥离（FY24）和以色列业务并购（FY24）已完成，无持续整合风险。

**数据口径说明**

币种：美元百万；会计准则：美国通用会计准则（U.S. GAAP）；利润口径：归属母公司股东净利润；**McDonald's 财年 = 日历年**（1/1-12/31）；Non-GAAP 调整剔除重组费用 $2.29 亿（"Accelerating the Organization"）和 FY24 韩国出售 / 以色列并购相关项目。

### 4.5 营收分析

- **其他收入 +53%**：数字化 + 科技授权服务增长

**主要拖累因素**

- **美国全年可比 +2.1%**（低于 IOM +3.2% 和 IDL +4.6%）：上半年消费疲软 + 高收入客户为 2024 "McDouble/McFlurry 通胀"抱怨拉低；下半年 Value Menu 恢复后 Q4 明显反弹
- **公司自营餐厅销售 -1%**：韩国业务出售 + 部分市场战略性调整 + 加盟转移
- **Israel / 中东抵制残留影响**：IOM 全年 +3.2% 但部分市场仍低迷
- **重组费用 $2.29 亿美元**（"Accelerating the Organization"）拖累 EPS $0.25
- **汇率逆风 -2pp**：美元走强拖累国际分部报告增速

**结构变化**

- **Q4 成为年内最强单季**（美国 +6.8% vs Q1 -3.6%）—— 经营拐点显现
- **加盟收入占比提升至 61.5%**（vs 60.6% FY24）—— 轻资产模式进一步深化
- **忠诚度计划从"会员身份"转向"实际销售渠道"**：$370 亿销售占系统销售 27%
- **McValue / $5 Meal Deal 成为标准配置**：几乎所有分部都在推广价值菜单

## 6. 未来可能的利好

### 利好 1：Q4 美国可比 +6.8% 反转 + Value Leadership 战略验证

**依据**

- Q4 2025 美国可比销售 **+6.8%**（vs Q4 2024 -1.4%），**单季反转达 +8pp**
- Q4 全球可比 **+5.7%**（vs Q4 2024 +0.4%）
- Q4 全球系统销售 +11%（恒定汇率 +8%）
- CEO Chris Kempczinski："McDonald's value leadership is working. By listening to customers and taking action, we have improved traffic and strengthened our value & affordability scores."
- **McValue Menu + $5 Meal Deal** 已扩展至全美所有门店
- 美国分部访客数（guest count）正增长 —— 不仅是涨价推动
- IOM 分部 Q4 +5.2% 也创年内新高

**解读**

**这是 FY25 最重要的战略转折点**：① 美国分部在 FY24 连续 4 个季度负增长后实现 Q4 +6.8% 的深度反转，**证明 $5 Meal Deal + McValue Menu 的价值策略已成功重建品牌对低收入消费者的吸引力**；② 客流量（traffic）正增长意味着 **不是靠涨价，而是真正赢得份额**；③ 如果这一势头延续至 FY26，全年可比销售可能从 FY25 的 +3.1% 加速至 +4-6%，进而推动 EPS 增速从 +5% 加速至 +8-10%；④ 全球化联动：IOM +5.2% + IDL +4.5% 显示策略在多市场同步见效；⑤ McDonald's 作为 "Always Recession-Proof + Value Leader" 的双重定位在 2026 美国消费降级 / GLP-1 普及的大环境下更加凸显优势。

### 利好 2：忠诚度会员 2.1 亿人 + $370 亿会员销售（+20%）+ 数字化护城河

**依据**

- FY25 全球忠诚会员系统销售 **$370 亿美元（+20%）**（70 个市场）
- **90 天活跃会员 2.1 亿人（+19%）**
- 忠诚销售占系统销售约 27%
- 比较：Starbucks Rewards 约 3,800 万活跃会员（美国），Chipotle Rewards 约 4,000 万
- McDonald's 忠诚会员规模是 Starbucks Rewards 的 5 倍+，行业绝对领先
- 管理层目标：中长期忠诚销售占比达 50%+
- 每月新增约 200-300 万活跃用户

**解读**

**忠诚度计划是 McDonald's 最被低估的战略资产**：① **2.1 亿活跃用户**（约占全球人口 2.6%，美国 1.4 亿人口的 30%）是互联网级别的用户规模；② $370 亿会员销售以 +20% 增速持续扩张（vs 整体系统销售 +7%），意味着 **忠诚会员 ARPU（每用户年均销售）** 约 $176，且持续上升；③ 忠诚度会员的复购率 / 客单价显著高于非会员，每增加 1pp 忠诚销售渗透率 = 系统销售 +约 0.5%；④ **数据资产价值**：2.1 亿用户的消费数据可以支撑 AI 个性化推荐、动态定价、精准营销，这是没有 Rewards 的竞争对手无法复制的；⑤ 忠诚度计划直接降低营销成本（从传统广告转向会员直接触达）；⑥ 从营收规模看，McDonald's 的 Rewards 业务单独拆分可能估值超 $500 亿美元。

### 利好 3：股息连续 49 年增长 + 资本回报稳定 + 负股东权益下的财务工程

**依据**

- FY25 季度股息 +5% 至 **$1.86/股**（年化 $7.44）—— **连续 49 年增长**
- FY25 股东回报 $71.71 亿美元（分红 $51.15 亿 + 回购 $20.56 亿）
- FY25 CFFO $105.51 亿（+11.7%）+ FCF $71.86 亿
- **Dividend Aristocrat**（标普 500 中连续 25+ 年增长股息的公司），**接近 Dividend King 地位**（50 年标准）
- 股东权益 -$17.91 亿美元 vs FY24 -$37.97 亿 —— 改善 $20 亿
- 分红回购总额占 FY25 净利润 83.7%
- 全球 45,000+ 家门店 + 庞大房地产资产支撑长期稳定现金流

**解读**

**McDonald's 是 Dividend 投资的教科书级标的**：① **FY26 有望创造"50 年连续股息增长"里程碑**，进入 Dividend King 行列（目前仅有 54 家美股达到），进一步强化防御属性；② 负股东权益反映的是"极致股东回报"战略 —— 只要 FCF 持续 $70+ 亿/年，权益赤字不构成财务风险；③ **股息收益率约 2.3-2.6%**（$7.44 / $290-320 股价）在高利率环境仍有吸引力，加上股息持续 +5-10%/年增长，长期回报稳定；④ McDonald's 的全球物业组合（约 45,000 家门店中 2/3 自持物业）为债权人提供强大担保，即使 CFFO 暂时承压也能通过物业融资灵活应对；⑤ 在当前"AI 成长 + 大盘波动"的市场环境下，McDonald's 属于 **稀缺的"现金流防御 + 股息增长 + 低波动"三重属性**标的，适合退休账户 / 保守配置。

## 7. 未来可能的利空

### 利空 1：GLP-1 减肥药普及 + 长期消费习惯变迁 + 健康饮食趋势

**依据**

- Ozempic、Wegovy、Mounjaro 等 GLP-1 减肥药在美国普及率持续提升（预计 2026 年约 1,500-2,000 万人使用）
- GLP-1 用户的食物消费量平均下降 20-30%，尤其减少高热量、高糖、高脂肪食物
- Walmart 已发布数据显示 GLP-1 用户食品支出下降 5-10%
- 年轻消费者对"植物性食品"、"健康食品"的偏好提升
- Chipotle、Sweetgreen、Cava 等健康快餐品牌增速超过 McDonald's
- 美国消费者平均每周快餐消费次数从 2019 年 2.1 次下降至 2025 年约 1.7 次
- GLP-1 对"快餐 / 垃圾食品"品牌构成长期需求结构性压力

**解读**

**GLP-1 是餐饮行业未来 5-10 年最大的结构性变量**：① 虽然 McDonald's 当前可以通过 Value Menu 抵消部分负面影响，但长期来看 **"量"的下降是不可逆的**；② 如果 GLP-1 用户从 2,000 万上升至 8,000 万（美国 25%+ 人口），McDonald's 美国业务系统销售长期可能面临 3-5% 的结构性压力；③ McDonald's 已在尝试推出更健康的菜单选择（McWrap Chicken、McPlant 等），但客户认知仍停留在"汉堡 + 薯条 + 可乐"的核心组合；④ 竞争对手 Chipotle、Sweetgreen 等在"高端健康餐"市场快速扩张；⑤ 这是一个 5-10 年慢变量，不会立即显现，但需要纳入长期估值考虑。

### 利空 2：Israel / 中东抵制残留 + 最低工资上涨 + 加盟商利润承压

**依据**

- **Israel / Gaza 冲突持续**：2023/10 起中东 / 穆斯林市场抵制运动对 IOM 部分市场仍有残留负面影响
- 印尼、马来西亚、土耳其、摩洛哥等多个市场 FY25 可比销售仍低迷
- 美国加州 FAST Act 自 2024/4 起要求快餐工人最低工资 **$20/小时**（vs 联邦 $7.25）
- 加州自营 / 加盟餐厅 FY25 利润率压缩约 300-500 bps
- 其他州（纽约、伊利诺伊、华盛顿）陆续跟进最低工资上涨
- 加盟商（95% 门店）面临利润压缩 —— 2025 年多家加盟主公开诉苦
- 原材料（牛肉、鸡肉、奶酪）成本仍在高位
- 保险、公用事业、房地产税成本持续上升

**解读**

**McDonald's 面临"加盟商 Squeeze"的长期压力**：① 加州 $20/hr 最低工资实施以来，加州加盟商利润率已从 15-18% 压缩至 10-13%，部分加盟主开始 **涨价或削减员工** 引发口碑下滑；② 如果其他州跟进，全美系统 **加盟商利润池可能压缩 10-20%**，长期可能导致加盟商投资意愿下降、扩张放缓；③ **Israel / 中东地缘**：IOM 和 IDL 中的穆斯林市场 FY25 可比销售低于其他市场，若冲突长期未解决，这些市场的品牌修复周期可能 3-5 年；④ McDonald's 企业端可以通过特许费调整 / 营销支持缓解加盟商压力，但会影响自身利润率。

### 利空 3：竞争加剧（Wendy's, Burger King, Chick-fil-A, Chipotle, Raising Cane's）+ 门店密度饱和

**依据**

- **Wendy's** 2025 推出 Baconator + Breakfast Menu 扩展，可比销售 +3.2%
- **Burger King（RBI）**：2024/25 推出 "Reclaim the Flame" 战略，可比销售 +4.5%
- **Chick-fil-A**：美国第三大快餐品牌，单店销售 $6.7M（vs McDonald's $3.2M 美国平均）
- **Chipotle**：FY25 可比销售 +6-8%（远超 McDonald's +2.1%）
- **Raising Cane's**：美国最快扩张的快餐品牌，2024/25 开店超 100 家
- **Taco Bell**（YUM）：$5 Value Menu 直接竞争 McDonald's $5 Meal Deal
- 美国快餐市场密度饱和（每 3,000 人一家快餐店）
- 在线订餐 + 配送（DoorDash、UberEats）改变行业竞争格局
- Starbucks、Dunkin' 早餐市场持续竞争

**解读**

**McDonald's 虽然仍是全球第一，但竞争对手不断收窄差距**：① **Chick-fil-A** 在美国市场的单店销售已是 McDonald's 的 2 倍，显示品牌粘性和产品差异化；② **Chipotle** 在"年轻 + 健康 + 高端"市场持续抢占 McDonald's 潜在客户；③ **Burger King 和 Wendy's** 在 $5 Value Menu 上快速跟进，McDonald's 的价值领导优势被快速复制；④ 美国快餐市场每年新增门店约 3-5%，但人口 + 消费增长仅 1-2%，**市场竞争本质上是零和博弈**；⑤ 在线订餐平台（DoorDash、UberEats）的崛起对 Drive-Thru 渠道构成冲击，McDonald's 需要持续投入数字化以维持优势；⑥ 咖啡早餐市场（McCafe vs Starbucks）的持续竞争可能限制美国分部早餐增长。

## 8. 总结判断

**一句话结论**

**McDonald's FY2025 交出"营收 $268.85 亿（+4%）、系统销售 $1,390+ 亿（+7%）、GAAP EPS $11.95（+5%）、调整后 EPS $12.20（+4%）、合并营业利润率 46.1% 行业顶级、CFFO $105.51 亿（+11.7%）、忠诚会员 2.1 亿（+19%）、股息连续 49 年增长"的标杆业绩**，**Q4 美国分部可比销售 +6.8% 的深度反转验证了"Value Leadership + McValue Menu"战略的成功**。**FY26 核心看点是 Q4 复苏势头延续 + 忠诚度会员突破 3 亿 + Dividend King 达成；核心风险是 GLP-1 结构性压力 + 加盟商最低工资压力 + Chipotle/Chick-fil-A 竞争**。

**核心看点**

1. **Q4 全球可比 +5.7% 深度反转**：美国 +6.8%、IOM +5.2%、IDL +4.5%，全面正增长 —— **McValue Menu + $5 Meal Deal 策略验证成功**
2. **忠诚会员 2.1 亿 + $370 亿销售 +20%**：70 个市场数字化护城河，中长期目标占系统销售 50%+
3. **连续 49 年股息增长 + FY26 有望达成 Dividend King**：标普 500 稀缺防御型资产 + 股东回报 $72 亿
4. **全球 45,000+ 门店网络 + 轻资产加盟模式**：营业利润率 46.1% 行业顶级，每年 CFFO $100+ 亿
