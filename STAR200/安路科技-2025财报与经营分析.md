# 安路科技（688107）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

2025年安路科技实现营业收入5.20亿元，同比 -20.22%，连续两年收入下滑。归母净利润-2.72亿元，亏损较2024年的-2.05亿元进一步扩大。综合毛利率43.00%，同比 +8.62pp，毛利率大幅改善但不足以覆盖高额研发支出（3.45亿元，占营收66.34%）。

### 1.2 增长与拖累因素

拖累因素：1）部分终端行业客户需求阶段性波动导致收入下滑；2）研发投入3.45亿元（占收入66%），远超收入规模所能承受的水平；3）累计未弥补亏损达-6.24亿元，不分红不转增。积极因素：1）毛利率大幅改善+8.62pp至43%，产品结构和定价优化效果显著；2）自Q2起季度收入环比增长，显示复苏态势；3）全年订单量和终端客户数同比增长，客户覆盖超2000家；4）库存下降11%至5.03亿元，库存消化效果明显。

### 1.3 趋势判断

公司处于FPGA国产替代的战略投入期，以"亏损换份额"的模式持续投入研发。收入下滑但订单和客户数增长是积极信号，说明产品力在提升。关键拐点在于收入何时恢复增长以摊薄高额研发费用。若2026年收入回到6-7亿元水平且毛利率维持43%+，亏损有望显著收窄。但持续亏损和6.24亿累计亏损是现实风险。

## 2. 主营业务

### 2.1 公司简介

安路科技是一家专注于FPGA（现场可编程门阵列）芯片及相关EDA工具研发的集成电路设计企业，是国内FPGA领域的主要玩家之一。产品广泛应用于工业控制、音视广播、数据中心、消费电子、机器视觉、新能源、汽车电子等领域。

### 2.2 主营业务构成

1. **FPGA芯片**：核心产品，占收入约89%，包括SALPHOENIX系列（高性能）、SALELF系列（新产品）等
2. **FPSoC芯片**：现场可编程系统级芯片，SALDRAGON系列集成ARM/RISC-V硬核处理器，占收入约7%
3. **技术服务及其他**：EDA软件工具及IP授权等，占收入约4%

### 2.3 核心收入来源

FPGA芯片是绝对核心收入来源（约89%），FPSoC产品增长迅速。客户覆盖工业控制、电力新能源、机器视觉、汽车电子等多个领域，服务客户超2000家。

| 归母净利润 | -2.72亿元 | -2.05亿元 | 亏损+32.7% | |
| 毛利率 | 43.00% | 34.38% | +8.62pp | |
| 净利率 | -52.3% | -31.4% | -20.9pp | |

### 5.2 利润分析

**亏损核心原因**

1. **研发费用过高**：研发投入3.45亿元，占收入66.34%，远超毛利润2.24亿元
2. **收入规模不足**：5.20亿元的收入尚不能摊薄固定研发投入
3. **以亏损换份额**：FPGA国产替代需要持续高研发投入以保持技术竞争力

**利润改善因素**

1. 毛利率大幅提升+8.62pp至43%，产品定价能力和结构改善
2. 研发费用绝对额同比-5.2%（3.45亿 vs 3.64亿），显示一定的费用管控
3. 库存下降11%，减值压力缓解

**盈亏平衡估算**

毛利率43%，研发3.45亿，假设管理+销售费用约1.0亿，则盈亏平衡点收入约（3.45+1.0）/0.43 ≈ 10.35亿元，约为当前收入的2倍。

## 6. 未来可能的利好

### 利好 1：FPGA国产替代战略机遇

**依据**

FPGA是半导体领域技术壁垒最高的品类之一，全球市场被Xilinx（AMD）和Intel（Altera）两家垄断。国内FPGA国产化率极低，国产替代空间巨大。公司已服务超2000家客户，产品线覆盖从低端到中高端。

**解读**

在中美科技博弈背景下，FPGA国产替代是国家战略需求。安路科技作为国内少数具备中高端FPGA设计能力的企业，有望持续受益于国产替代浪潮。

### 利好 2：订单和客户数增长预示收入拐点

**依据**

2025年全年订单量和终端客户数均同比增长，Q2起季度收入环比增长。电力新能源领域+20%，汽车和智算中心实现突破。SALDRAGON系列已导入超200个客户项目。

**解读**

订单增长但收入下滑可能反映新产品导入期的时间差。随着新产品放量和客户扩展，2026年收入有望恢复增长。若收入回到7亿+水平，亏损将显著收窄。

### 利好 3：毛利率持续改善至43%

**依据**

毛利率从2024年的34.38%大幅提升至43.00%（+8.62pp），产品结构优化和定价能力增强效果显著。

**解读**

43%的毛利率已达到较好水平。若收入恢复增长（毛利率43%+收入7亿=毛利3亿），研发费用保持在3.5亿左右，则亏损可大幅收窄至数千万级别。

## 7. 未来可能的利空

### 利空 1：持续亏损，累计亏损6.24亿元

**依据**

公司已连续三年亏损（2023年-1.97亿、2024年-2.05亿、2025年-2.72亿），累计未弥补亏损达-6.24亿元。不分红不转增，现金储备下降42%。

**解读**

持续亏损侵蚀净资产和现金储备，若2026-2027年仍无法扭亏或大幅减亏，可能面临融资需求。股权激励也难以达标，影响核心团队稳定性。

### 利空 2：收入持续下滑，盈亏平衡点遥远

**依据**

收入从2022年的10.42亿元持续下滑至2025年的5.20亿元，三年腰斩。估算盈亏平衡点约10.35亿元，当前收入仅为其一半。

**解读**

即使2026年收入恢复增长，要回到盈亏平衡点需翻倍，短期内难以实现。这意味着公司可能还要经历2-3年的亏损期。

### 利空 3：国际巨头竞争和技术代差

**依据**

全球FPGA市场由AMD（Xilinx）和Intel（Altera）两家寡头垄断，技术代差仍然显著。国际巨头在先进制程和高端产品上的领先优势短期内难以逾越。

**解读**

虽然国产替代提供了市场机会，但在高端市场上安路科技仍面临技术差距。若国际关系缓和或供应链限制放松，国产替代需求可能减弱。

## 8. 总结判断

**一句话结论**

2025年安路科技收入-20%至5.20亿、亏损扩大至-2.72亿，但毛利率+8.62pp至43%和订单/客户增长释放出复苏信号，核心看点是FPGA国产替代能否推动收入拐点到来。

**核心看点**

1. FPGA国产替代战略赛道，客户超2000家，订单量和客户数同比增长
2. 毛利率大幅改善至43%（+8.62pp），产品定价能力增强
3. Q2起季度收入环比增长，电力新能源+20%，汽车/智算突破

**核心风险**

1. 连续三年亏损，累计亏损6.24亿元，盈亏平衡点（~10亿收入）遥远
2. 收入三年腰斩（10.42亿→5.20亿），需求复苏速度不确定
3. 研发占收入66%，持续烧钱模式对现金储备消耗大
