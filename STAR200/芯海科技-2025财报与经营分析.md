# 芯海科技（688595）2025年财报整理

> 数据来源：2025年年度报告 + 2025年三季报

## 1. 业绩总结

### 1.1 收入与利润表现

2025年芯海科技实现营业收入8.49亿元，同比增长20.82%；归母净利润-1.05亿元，亏损同比收窄6,743万元；扣非净利润-1.15亿元，亏损收窄6,696万元。2023-2025年营收复合增长率达40%，收入保持高速增长，但仍处于亏损阶段。模拟信号链芯片（+65.20%）成为增长主引擎，MCU与AIoT芯片（+2.91%）平稳增长。亏损收窄主要得益于规模效应释放和成本结构优化。

### 1.2 增长与拖累因素

**主要增长驱动**：(1) 模拟信号链芯片2.99亿元（+65.20%），BMS芯片、USB HUB芯片快速放量；(2) Q3单季营收2.41亿元（+46.85%），增速加快；(3) 毛利率提升至35.46%（+2.18pp）；(4) 新品（PPG芯片、EC芯片）进入规模出货。

**主要拖累**：(1) 仍处亏损（全年-1.05亿），Q4亏损约4,200万元；(2) 研发费率32.81%处于高位；(3) MCU与AIoT芯片增速仅+2.91%。

### 1.3 趋势判断

芯海科技是国内模拟信号链+MCU融合芯片设计公司，正从消费电子向汽车、服务器、AI领域拓展。营收两年复合增长40%，亏损持续收窄，处于从亏损向盈利跨越的关键阶段。模拟信号链芯片的爆发增长和EC芯片获Intel/AMD双认证是重要里程碑。

## 2. 主营业务

### 2.1 公司简介

芯海科技专注于高精度ADC、模拟信号链芯片和高可靠性MCU的研发与销售，产品广泛应用于智能手机、笔记本电脑、智能穿戴、智能家居、汽车电子、服务器等领域。公司是国内唯一通过Intel和AMD双认证的EC（嵌入式控制器）芯片厂商。

## 4. 营收分布和增长

### 4.1 产品收入分析

**模拟信号链芯片（35.2%，+65.20%）**

增速最快的品类，包括BMS芯片、USB HUB芯片、高精度ADC等。BMS芯片受益于新能源汽车和储能需求，USB HUB芯片受益于PC/服务器需求。

**MCU与AIoT芯片（61.6%，+2.91%）**

核心业务，增长平稳。包括EC芯片（已获Intel/AMD双认证，在联想、荣耀、小米量产）、健康测量MCU（PPG芯片在头部穿戴客户规模出货）、智能仪表MCU等。

### 4.2 增长趋势

两年复合增长率40%，模拟信号链芯片从"第二增长曲线"逐步成为主引擎。Q3增速加快至+47%表明新品放量加速。

## 5. 利润分布和增长

### 5.1 整体利润表现

| 指标 | 2025年 | 同比变化 | 备注 |

| 归母净利润 | -1.05亿元 | 减亏6,743万 | |
| 扣非净利润 | -1.15亿元 | 减亏6,696万 | |

### 5.2 利润分析

**亏损收窄的驱动**

1. **收入增长20.8%**：规模效应开始释放
2. **毛利率提升2.18pp至35.5%**：产品结构优化和成本管控
3. **费用率下降**：管理效率提升

**仍处亏损的原因**

1. **研发费率32.8%**：2.78亿元研发投入用于车规MCU、AI芯片、服务器EC等前瞻布局
2. **新品尚处放量早期**：部分产品线毛利率尚未达到盈利水平

## 6. 未来可能的利好

### 利好 1：EC芯片获Intel/AMD双认证

**依据**

国内唯一通过Intel和AMD双认证的EC芯片厂商，已在联想、荣耀、小米等品牌量产。

**解读**

笔记本电脑EC芯片国产替代空间巨大，双认证意味着公司产品已达到国际一线水平，后续有望在更多PC品牌放量。

### 利好 2：模拟信号链芯片爆发增长

**依据**

模拟信号链芯片+65.20%至2.99亿元。

**解读**

BMS芯片受益于新能源汽车和储能爆发，USB HUB芯片受益于AI PC/服务器升级，均为高增长赛道。

### 利好 3：车规MCU突破

**依据**

首款ASIL-D功能安全等级车规MCU完成流片点亮。

**解读**

ASIL-D是最高安全等级，如果通过认证并量产，将打开汽车电子广阔市场。

### 利好 4：亏损持续收窄，接近盈亏平衡

**依据**

亏损从~1.72亿收窄至1.05亿，减亏39%。

**解读**

按当前减亏趋势，2026年有望实现扭亏为盈或接近盈亏平衡。

## 7. 未来可能的利空

### 利空 1：仍处亏损状态

**依据**

全年亏损1.05亿元，Q4亏损约4,200万元。

**解读**

尽管亏损收窄，但何时盈利仍有不确定性。如果新品放量不及预期或行业景气度下行，扭亏时间可能延后。

### 利空 2：研发费率32.8%持续高企

**依据**

研发投入2.78亿元，占营收32.81%。

**解读**

高研发投入支撑长期竞争力，但短期严重压制利润。如果营收增速放缓，研发费率难以下降。

### 利空 3：MCU业务增长乏力

**依据**

MCU与AIoT芯片仅+2.91%，占比61.6%。

**解读**

核心业务增长几乎停滞，如果模拟信号链芯片增速回落，整体增长可能放缓。

### 利空 4：净资产持续下降

**依据**

净资产7.47亿元，同比-1.09%，连续因亏损侵蚀。

**解读**

持续亏损导致净资产缩水，如果不能尽快盈利，可能面临融资压力。

## 8. 总结判断

**一句话结论**

芯海科技2025年营收8.49亿元（+20.82%），归母净利润-1.05亿元（减亏39%），模拟信号链芯片爆增65%成为主引擎，EC芯片获Intel/AMD双认证，处于从亏损向盈利跨越的关键时期。

**核心看点**

1. 模拟信号链芯片2.99亿（+65%），BMS/USB HUB快速放量
2. 国内唯一Intel+AMD双认证EC芯片，在联想/荣耀/小米量产
3. 营收两年复合增长40%，毛利率35.5%（+2.2pp）
4. 亏损收窄39%，接近盈亏平衡
5. 首款ASIL-D车规MCU完成流片

**核心风险**

1. 仍亏损1.05亿元，扭亏时间不确定
2. 研发费率32.8%持续高企
3. MCU核心业务增长仅+3%
4. 净资产因亏损持续缩水
