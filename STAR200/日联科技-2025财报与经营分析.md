# 日联科技（688531）2025年财报整理

> 数据来源：2025年度业绩快报 + 2025年三季报 + 2025年半年报

## 1. 业绩总结

### 1.1 收入与利润表现

2025年日联科技实现营业收入10.71亿元，同比增长44.88%；归母净利润1.75亿元，同比增长21.81%；扣非净利润1.45亿元，同比增长50.85%；经营性现金流1.91亿元，同比增长509.37%。公司收入实现近45%的高增长，扣非利润增速50.85%快于收入增速，但归母净利润增速21.81%低于收入增速，主要受非经常性损益变化影响。经营现金流大幅改善是一大亮点。

### 1.2 增长与拖累因素

**主要增长驱动**：(1) X射线源产品全谱系覆盖，微焦点射线源大规模出货；(2) 纳米级开管射线源和大功率射线源实现产业化；(3) 半导体检测、新能源电池检测等新应用场景拓展；(4) 并购整合（收购珠海九源55%、新加坡SSTI 66%）；(5) AI智能检测软件和3D/CT检测技术突破。

**主要拖累**：(1) 并购带来的费用和整合成本；(2) 非经常性损益减少导致归母利润增速低于扣非增速；(3) 应收账款规模较大。

### 1.3 趋势判断

日联科技是国内X射线智能检测装备龙头，正处于快速成长期。公司通过"X射线源自研+下游应用拓展+并购整合"三条路径实现高增长。半导体检测和新能源电池检测是两大新增长极，海外布局（马来西亚、匈牙利、美国）打开全球市场空间。

## 2. 主营业务

### 2.1 公司简介

日联科技是国内领先的X射线智能检测装备企业，产品涵盖工业X射线源、X射线智能检测装备和AI检测软件。应用领域包括集成电路与电子制造、新能源电池、铸件焊件及材料、食品异物检测等。

### 2.2 主营业务构成

1. **X射线智能检测装备**（核心业务）
   - 集成电路及电子制造检测设备
   - 新能源电池检测设备
   - 铸件焊件及材料检测设备
   - 食品异物检测设备
2. **X射线源**（上游核心零部件）
   - 闭管微焦点射线源
   - 开管射线源（纳米级）
   - 大功率射线源
3. **AI智能检测软件及服务**
4. **并购业务**
   - 珠海九源：新能源电池检测
   - 新加坡SSTI：半导体检测诊断与失效分析

## 4. 营收分布和增长

### 4.1 按应用领域划分

| 领域 | 趋势 | 备注 |
|---|---|---|
| 集成电路及电子制造 | 高增长 | 开管射线源获头部客户认可 |
| 新能源电池检测 | 快速增长 | 收购珠海九源强化布局 |
| 铸件焊件及材料检测 | 稳定 | 传统优势领域 |

### 4.2 营收分析

1. X射线源自研能力突破，微焦点射线源大规模出货
2. 纳米级开管射线源产业化，打开半导体检测高端市场
3. 新能源电池检测需求旺盛（半固态电池等新工艺）
4. 并购珠海九源并表贡献增量（2025年6月起）
5. AI智能检测软件+3D/CT技术增加客户粘性和客单价

### 5.2 利润分析

**扣非利润增速快于归母的原因**

1. 2024年可能有较大非经常性收益（政府补助等），2025年减少
2. 扣非利润+50.85%更真实反映经营质量，高于收入增速，显示经营杠杆效应

**净利率下降的原因**

1. 并购整合相关费用（中介费、尽调费等）
2. 研发投入持续加大（新产品开发）
3. 海外工厂建设期费用（马来西亚、匈牙利、美国）

## 6. 未来可能的利好

### 利好 1：半导体检测市场爆发

**依据**

收购新加坡SSTI（深耕半导体检测30年），开管射线源获头部电子半导体客户认可。

**解读**

全球半导体先进封装（Chiplet、3D封装）对X射线检测需求快速增长。SSTI的技术积累+日联科技的射线源自研能力，有望在半导体检测领域实现协同突破。这是一个百亿级别的全球市场。

### 利好 2：X射线源自研替代进口

**依据**

公司已实现闭管微焦点、开管纳米级、大功率射线源的全谱系覆盖，2025上半年新增8款射线源产品。

**解读**

X射线源是检测设备的核心零部件，过去长期依赖进口。日联科技的自研突破不仅降低成本、提升毛利率，还能实现"源+设备+软件"一体化解决方案，提高竞争壁垒和客户粘性。

### 利好 3：新能源电池检测需求增长

**依据**

已向半固态电池厂商提供检测设备，收购珠海九源55%股份强化布局。

**解读**

固态/半固态电池是下一代电池技术方向，对检测精度要求更高。公司提前布局，有望在新能源电池检测领域建立先发优势。

### 利好 4：经营现金流大幅改善

**依据**

2025年经营性现金流1.91亿元，同比+509.37%。

**解读**

现金流改善说明公司收入质量提高、回款加速，有助于支撑后续研发和并购投入。

## 7. 未来可能的利空

### 利空 1：并购整合风险

**依据**

2025-2026年密集收购珠海九源和新加坡SSTI。

**解读**

并购整合存在文化差异、管理协调、商誉减值等风险。特别是跨国收购SSTI，涉及不同国家的法律法规和市场环境。如果整合不及预期，可能拖累利润。

### 利空 2：应收账款规模较大

**依据**

三季报显示应收账款体量较大。

**解读**

高速增长期应收账款增长属正常现象，但需关注坏账风险。如果下游客户（特别是新能源企业）出现经营困难，可能导致回款风险。

### 利空 3：利润率承压

**依据**

归母净利率约16.3%，较2024年的19.5%下降3.2pp。

**解读**

并购整合费用、海外工厂建设、研发投入持续加大，短期内可能继续压制利润率。需要关注规模效应能否在中期改善盈利能力。

## 8. 总结判断

**一句话结论**

日联科技2025年营收10.71亿元（+44.88%），扣非净利润1.45亿元（+50.85%），X射线检测装备龙头进入高速成长期，"射线源自研+应用拓展+并购整合"三轮驱动。

**核心看点**

1. 收入+44.88%高增长，扣非利润+50.85%更快增长
2. X射线源全谱系自研突破，进口替代空间巨大
3. 半导体检测（收购SSTI）+新能源电池检测双新增长极
4. 经营现金流+509.37%，收入质量显著提升
5. Q4单季营收3.34亿元创历史新高

**核心风险**

1. 并购整合风险（珠海九源、新加坡SSTI）
2. 应收账款规模较大
3. 短期净利率承压（并购费用+海外扩张+研发投入）
