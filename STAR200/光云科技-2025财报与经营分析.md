# 光云科技（688365）2025年财报整理

> 数据来源：2025年度业绩快报 + 2025年三季报 + 2024年年报

## 1. 业绩总结

### 1.1 收入与利润表现

2025年光云科技实现营业收入5.65亿元，同比增长18.33%；归母净利润-2638万元（亏损），同比减亏67.85%；扣非净利润-2975万元，同比减亏59.24%。基本每股收益-0.06元。总资产17.53亿元（+9.33%），归母净资产9.44亿元（-1.23%）。前三季度毛利率70.31%（+8.06pp），经营现金流5161万元（+90.82%）。公司收入保持近20%增长，亏损大幅收窄，盈利拐点渐近。

### 1.2 增长与拖累因素

**核心驱动**：1）多平台布局成效显著——抖音平台中小商家SaaS收入+78.41%，多平台收入占比提升至49.14%；2）大商家SaaS产品稳定增长，线下地推覆盖30城；3）电商供应链SaaS（逸淘）成为新增长点；4）毛利率70.31%高位，AI赋能（快麦小智客服提升GMV 20%）提升产品附加值；5）经营现金流+90.82%大幅改善。

**拖累因素**：1）仍处亏损状态（-2638万），SaaS企业盈利周期较长；2）三费占比53.92%偏高（虽已下降5.35pp）；3）净资产微降（-1.23%），累计亏损侵蚀净资产。

### 1.3 趋势判断

光云科技是国内电商SaaS龙头，覆盖超120万付费用户、日均处理包裹3000万+。2025年收入+18.33%/亏损收窄67.85%体现"多平台扩张+大客户升级+供应链延伸"三重增长逻辑。毛利率70%+说明SaaS商业模式成立，关键是费用率能否进一步下降至盈亏平衡点。若2026年收入维持15%+增长+费用率继续下降，有望实现扣非转正。

## 2. 主营业务

### 2.1 公司简介

光云科技是服务于电商商家的SaaS企业，为淘宝、京东、拼多多、抖音、微信等全平台商家提供店铺管理、订单处理、客服、ERP、供应链等SaaS产品和增值服务。累计覆盖超120万付费用户，日均处理包裹量超3000万。

### 2.2 主营业务构成

1. **中小商家SaaS**：超级店长、快递助手等，覆盖多平台（抖音收入+78.41%），多平台收入占比49.14%
2. **大商家SaaS**：快麦ERP等，面向大型电商商家，线下地推覆盖30城，收入稳定增长
3. **AI赋能产品**：快麦小智客服机器人（提升GMV 20%）、深绘美工机器人
4. **电商供应链SaaS**：逸淘供应链（新增长点），进军采购及供应链管理
5. **企服类产品**：有成财务、有成CRM、有成报销等

### 2.3 核心竞争优势

- 全平台覆盖（淘宝/京东/拼多多/抖音/微信），多平台占比已达49%
- 120万+付费用户，日处理包裹3000万+，规模优势明显
- 毛利率70%+，典型SaaS商业模式
- AI技术深度赋能产品（客服机器人提升GMV 20%）

## 5. 利润分布和增长

### 5.1 整体利润表现

| 指标 | 2025年 | 2024年 | 同比增速 | 备注 |
|---|---:|---:|---:|---|
| 归母净利润 | -2638万元 | ~-8206万元 | 减亏67.85% | |
| 扣非净利润 | -2975万元 | ~-7299万元 | 减亏59.24% | |
| 毛利率（Q1-Q3） | 70.31% | 62.25% | +8.06pp | |

### 5.2 利润分析

**亏损大幅收窄原因**

1. 收入+18.33%带来规模效应，毛利率提升至70%+
2. 三费率下降5.35pp至53.92%，费用管控有效
3. 多平台收入高增长（抖音+78%）带来边际利润改善
4. 经营现金流+90.82%，回款质量提升

**距盈亏平衡的差距**

- 2025年收入5.65亿×毛利率70% = 毛利约3.96亿
- 三费约3.05亿（53.92%×5.65亿）
- 毛利-三费 = 约0.91亿，理论上应盈利，但可能存在减值/其他费用
- 若2026年收入+15%至6.5亿且三费率下降至50%，有望实现扣非转正

## 6. 未来可能的利好

### 利好 1：多平台收入占比49%+，抖音平台+78%爆发

**依据**

多平台收入占比提升至49.14%，抖音平台中小商家SaaS收入+78.41%。

**解读**

电商格局从淘宝一家独大转向多平台并存，光云科技成功实现多平台布局，抖音/拼多多等新平台增速远超传统平台。多平台收入占比近50%意味着公司已摆脱对单一平台的依赖，增长更健康可持续。

### 利好 2：AI赋能产品提升ARPU，快麦小智提升GMV 20%

**依据**

快麦小智客服机器人帮助商家提升GMV 20%。

**解读**

AI是SaaS企业的核心竞争壁垒和提价杠杆。通过AI客服、AI美工等产品，公司可以提升单用户付费（ARPU），从"工具型SaaS"升级为"增值型AI SaaS"，打开客单价上限。

### 利好 3：亏损大幅收窄+经营现金流+90%，盈利拐点渐近

**依据**

归母减亏67.85%，扣非减亏59.24%，经营现金流+90.82%。

**解读**

SaaS企业的典型路径是"亏损扩张→亏损收窄→盈利"。光云科技2025年已进入"亏损大幅收窄"阶段，毛利率70%+说明商业模式成立，费用率持续下降。若2026年延续减亏趋势，有望在年内或2027年实现扣非转正。

## 7. 未来可能的利空

### 利空 1：仍处亏损状态，累计亏损侵蚀净资产

**依据**

归母-2638万，净资产-1.23%。

**解读**

虽然亏损大幅收窄，但连续亏损仍在消耗净资产。若盈利拐点迟迟不到，公司可能需要再融资，稀释股东权益。

### 利空 2：电商SaaS行业竞争激烈，客户流失风险

**依据**

三费率53.92%仍偏高。

**解读**

电商SaaS领域有赞、微盟等竞争对手众多，中小商家对价格敏感，转换成本低。高销售费用率说明获客/留客仍需大量投入，若竞争加剧可能限制费用率下降空间。

### 利空 3：淘宝平台收入增长放缓，传统基本盘承压

**依据**

淘宝/天猫占比下降至约51%。

**解读**

虽然多平台布局成功，但淘宝仍是最大收入来源。若淘宝商家活跃度下降或平台生态变化，将直接影响公司传统基本盘收入。

## 8. 总结判断

**一句话结论**

2025年光云科技营收5.65亿元（+18.33%）/归母-2638万元（减亏67.85%）/扣非-2975万元（减亏59.24%），多平台收入占比49%+抖音+78%+毛利率70%+经营现金流+91%，亏损大幅收窄、盈利拐点渐近；但仍处亏损+三费率偏高+电商SaaS竞争激烈。AI+多平台+供应链SaaS是三大增长引擎。

**核心看点**

1. 多平台收入占比49%，抖音+78%爆发增长
2. 毛利率70%+，AI赋能提升产品附加值（GMV+20%）
3. 亏损减少67.85%+经营现金流+91%，盈利拐点渐近
4. 供应链SaaS（逸淘）成为新增长点

**核心风险**

1. 仍处亏损状态，净资产持续被侵蚀
2. 电商SaaS竞争激烈，客户转换成本低
3. 淘宝传统基本盘增长放缓
