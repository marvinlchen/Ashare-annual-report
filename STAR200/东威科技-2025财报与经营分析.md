# 东威科技（688700）2025年财报整理

> 数据来源：2025年度业绩快报 + 2025年三季报

## 1. 业绩总结

### 1.1 收入与利润表现

2025年东威科技实现营业收入11.07亿元，同比增长47.65%；归母净利润1.29亿元，同比增长86.81%；扣非净利润1.25亿元，同比增长103.52%；营业利润1.45亿元，同比增长96.61%；EPS 0.43元。公司收入和利润均实现大幅增长，利润增速（87%/104%）远超收入增速（48%），反映出经营杠杆释放和盈利效率改善。Q3单季营收3.14亿元（+67.2%），环比+35.2%，增长加速态势明显。

### 1.2 增长与拖累因素

**主要增长驱动**：(1) PCB电镀设备订单持续增长，受益于PCB东南亚投资潮和AI算力需求；(2) 脉冲VCP设备获台湾知名科技企业突破一亿元订单；(3) 水平镀三合一设备获海外订单；(4) Q3毛利率环比+1.1pp至36.1%，海外高毛利业务占比增加。

**主要拖累**：(1) 毛利率同比下降4.80pp至34.0%，产品结构和竞争压力；(2) Q3毛利率同比仍-3.1pp，但环比已改善。

### 1.3 趋势判断

东威科技是PCB电镀设备龙头，核心受益于AI算力/高端PCB需求爆发和全球PCB产能向东南亚转移。公司在高端VCP/水平镀领域持续突破，国产替代加速推进。收入和利润双高增，Q3加速趋势明确，盈利拐点已确认。

## 2. 主营业务

### 2.1 公司简介

东威科技是一家专注于电镀设备研发、生产和销售的高新技术企业，主要产品包括PCB电镀设备（VCP设备、水平电镀线）、五金电镀设备和复合铜箔设备等。公司产品广泛应用于PCB制造、半导体封装、新能源等领域，是国产高端PCB电镀设备的领先企业。

### 2.2 主要产品线

| 产品类别 | 特点 | 市场方向 |
|---|---|---|
| 脉冲VCP设备 | 线宽/线距最小8μm | 高端HDI/IC载板 |
| 水平TGV电镀线 | 半导体封装级 | 先进封装 |
| 水平镀三合一设备 | 已获海外订单 | PCB全球产能转移 |
| HVLP铜箔设备 | 高频超低轮廓铜箔 | AI PCB高频高速 |
| 五金电镀设备 | 传统业务 | 通用工业 |

## 4. 营收分布和增长

### 4.1 营收分析

1. **PCB电镀设备是核心增长引擎**，受益于AI算力PCB需求和东南亚PCB投资潮
2. **高端产品突破**：脉冲VCP获台湾大客户亿元订单，水平镀三合一获海外订单
3. **Q3加速明显**：单季+67%，环比+35%，订单转化加速
4. **Q4推算约3.50亿元**（11.07-7.57），同比增长约100%+，为全年最高季度
5. **海外业务拓展**：水平镀产品已打入海外市场

### 4.2 收入增长逻辑

- **短期**：PCB电镀设备订单持续增长，Q3/Q4放量
- **中期**：东南亚PCB投资潮+AI高端PCB需求+国产替代
- **长期**：半导体封装、复合铜箔、HVLP铜箔等新产品线扩展

## 5. 利润分布和增长

### 5.1 整体利润表现

| 指标 | 2025年 | 同比 | 备注 |
|---|---:|---:|---|
| 归母净利润 | 1.29亿元 | +86.81% | |
| 扣非净利润 | 1.25亿元 | +103.52% | 翻倍 |
| 营业利润 | 1.45亿元 | +96.61% | |

### 5.2 利润分析

**利润翻倍增长的原因**

1. **收入高增长+经营杠杆释放**：收入+48%但利润+87-104%
2. **净利率提升**：从约9.2%提升至约11.7%
3. **Q3毛利率环比改善**：海外高毛利订单占比增加
4. **扣非利润增速（+104%）高于归母（+87%）**：核心经营改善

**关注点**

毛利率同比仍在下降（-4.8pp），需观察高端产品放量后毛利率能否企稳回升。

## 6. 未来可能的利好

### 利好 1：AI算力驱动高端PCB电镀需求爆发

**依据**

AI服务器对高层数HDI板和IC载板的需求激增，公司脉冲VCP设备线宽可达8μm，已获台湾大客户亿元订单。

**解读**

AI算力建设是确定性趋势，高端PCB电镀设备需求将持续增长，公司作为国产替代龙头直接受益。

### 利好 2：PCB产能向东南亚转移带来新机遇

**依据**

全球PCB产能加速向东南亚转移，公司水平镀三合一设备已获海外订单。

**解读**

东南亚PCB新建产能需要配套电镀设备，公司凭借性价比和技术优势有望获得可观的海外增量订单。

### 利好 3：半导体封装和新产品拓展

**依据**

水平TGV电镀线应用于半导体封装，HVLP铜箔设备已接到客户订单。

**解读**

半导体封装和新型铜箔设备打开新成长空间，产品线从PCB向半导体和新能源延伸。

### 利好 4：盈利拐点确认

**依据**

扣非净利润翻倍增长，Q3毛利率环比改善，净利率提升。

**解读**

经营杠杆开始释放，规模效应逐步显现，盈利能力进入上升通道。

## 7. 未来可能的利空

### 利空 1：毛利率同比仍在下降

**依据**

Q1-Q3毛利率34.0%（-4.8pp），Q3毛利率36.1%（-3.1pp）。

**解读**

虽然环比改善，但同比仍有较大降幅，行业竞争和产品结构变化可能继续压制毛利率。

### 利空 2：客户集中度和订单波动风险

**依据**

脉冲VCP单笔订单超过一亿元，大客户依赖度可能较高。

**解读**

高端设备订单金额大但客户数量少，单一大客户的订单波动可能导致业绩不稳定。

### 利空 3：行业竞争加剧

**依据**

PCB电镀设备领域竞争者增多，日本和中国台湾厂商在高端市场仍有优势。

**解读**

国产替代虽在推进，但高端市场份额争夺仍面临外资竞争压力。

## 8. 总结判断

**一句话结论**

东威科技2025年营收11.07亿元（+47.65%），归母净利润1.29亿元（+86.81%），扣非翻倍增长，PCB电镀设备龙头受益于AI算力和东南亚PCB投资潮，Q3加速增长，盈利拐点确认。

**核心看点**

1. 收入+48%，扣非利润翻倍（+104%），盈利拐点确认
2. Q3营收+67%、环比+35%，加速趋势明确
3. 脉冲VCP获台湾大客户亿元订单，高端产品突破
4. 水平镀获海外订单，PCB东南亚投资潮受益
5. 半导体封装+HVLP铜箔新产品线拓展

**核心风险**

1. 毛利率同比下降4.8pp，能否企稳需观察
2. 大客户依赖和订单波动风险
3. 高端市场外资竞争
