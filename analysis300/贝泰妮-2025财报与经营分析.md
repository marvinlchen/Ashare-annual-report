# 贝泰妮 2025年财报整理

> **说明**：贝泰妮（300957.SZ）**2025 年年度报告尚未披露**（预计 2026 年 4 月下旬披露，**业绩预告/快报亦未发布**）。本文基于 **2025 年三季报（2025-10-27 披露）+ 2025 年半年报 + 2025 年一季报 + 机构研报** 整理，年报未披露的字段统一标注"年报未披露"。

## 1. 业绩总结

### 1.1 收入与利润表现

贝泰妮 2025 年 Q1-Q3 实现营业收入 34.64 亿元，**同比 -13.78%**；归母净利润 2.72 亿元，**同比 -34.45%**；扣非净利润 1.84 亿元，**同比 -46.17%**。Q1-Q3 毛利率 **74.33%（+0.60pct）** 维持高位，净利率 7.97%（-2.10pct）。**Q3 单季归母 2,522 万（同比 +136.55%）+ Q2 净利率 15.3%（环比 +12.35pct）**——**业绩拐点已现**。**经营现金流 4.42 亿元（同比 +6,772.14%，暴涨 67 倍）**——回款质量大幅改善。

- Q1-Q3 营业收入 34.64 亿元（同比 -13.78%）
- Q1-Q3 营业利润 年报未披露（全年）
- Q1-Q3 归母净利润 2.72 亿元（同比 -34.45%）

### 1.2 增长与拖累因素

- **主要增长驱动**：①**瑷科缦 AOXMED H1 +93.90%**（高端抗衰爆发）；②薇诺娜宝贝 +8.62%（婴童稳增）+ 浴见（抖音双十一前十）；③**Q3 归母 +136.55%**（低基数下拐点信号）；④**经营现金流 +6,772%（4.42 亿）**；⑤毛利率维持 74-78% 高位；⑥H1 抖音 +7.41%
- **主要拖累因素**：①**薇诺娜主品牌 H1 -18.4%**（19.5 亿，主动调整）；②**线下渠道 H1 -41.58%**（严重萎缩）；③**Q1 归母 -83.97%**（深度承压）；④化妆品行业价格战 + 流量成本上升；⑤净利率 -2.10pct
- **增长主要来自**：瑷科缦 / 薇诺娜宝贝 / 浴见等新品牌；**承压主要来自**：薇诺娜主品牌 + 线下渠道

### 1.3 趋势判断

- **未来主要利好方向**：Q2/Q3 拐点延续 + 瑷科缦高端化爆发 + 多品牌矩阵开花 + 现金流改善 + 薇诺娜主品牌企稳
- **未来主要压力方向**：薇诺娜持续下滑 + 线下渠道萎缩 + 化妆品价格战 + 流量红利消退 + 竞争对手（珀莱雅 / 可复美 / 华熙等）蚕食份额
- **整体趋势判断**：**2026 年是贝泰妮穿越调整期、多品牌矩阵跑通的关键之年**。若拐点延续 + 子品牌提速 + 主品牌企稳，**有望复制珀莱雅式多品牌增长路径**。华源证券预测 2025-2027 归母 **6.0 / 7.7 / 8.8 亿**

## 2. 主营业务

### 2.1 公司简介

贝泰妮是**A 股功能性护肤品龙头 + 国货敏感肌护肤绝对第一**，正从单品牌向"薇诺娜 + 瑷科缦 + 薇诺娜宝贝 + 浴见 + 菁白 + 悦芙媞 + 姬芮 + 泊美"多品牌矩阵转型。**薇诺娜品牌连续 5 年蝉联中国敏感肌护肤销售额第一**。

### 2.2 主营业务构成

1. **薇诺娜 Winona（主品牌）**：敏感肌修护领导品牌，H1 **19.5 亿（-18.4%）**，占营收 80%+，连续 5 年敏感肌护肤销售第一
2. **瑷科缦 AOXMED（高端品牌）**：高端抗衰护肤，H1 **0.5147 亿（+93.90%）**，天猫美妆国货新品牌第二，绿安缦系列爆款
3. **薇诺娜宝贝**：婴童护肤，H1 1.09 亿（+8.62%），婴童洗护赛道前列
4. **浴见**：身体洗护品牌，抖音双十一身体洗护前十
5. **姬芮 Za / 泊美 Pure & Mild**（资生堂品牌收购）：2023 年完成收购，2024-2025 年逐步整合（大众品护肤 + 彩妆）
6. **薇诺娜菁白 + 悦芙媞**：美白 / 功效护肤新品

### 2.3 核心收入来源

**薇诺娜（主品牌）**仍是绝对大头（占收入 80%+）；**瑷科缦 + 薇诺娜宝贝 + 浴见**多品牌矩阵快速放量中；**线上渠道占比超 73%**（H1 17.43 亿），抖音 / 天猫 / 京东 / 拼多多全电商布局；国内市场为主，海外（东南亚）小规模拓展。

## 6. 未来可能的利好

### 利好 1：Q3 归母 +136.55% + Q2 净利率 +12.35pct 拐点

**依据**

Q3 单季归母 **+136.55%** 同比大幅转好；Q2 净利率 **15.3%**（环比 Q1 +12.35pct）；Q2 环比 Q1 营收 +50% + 归母 +668%；H1 扣非 -56.48% → Q1-Q3 扣非 -46.17%（降幅收窄）。

**解读**

Q2 / Q3 连续两个季度盈利能力大幅改善 + 同比降幅持续收窄 → **业绩拐点已现**。若 2026 Q1 在低基数（2025 Q1 归母仅 2,834 万）下，**有望迎来 100%+ 的业绩弹性**，2026 年业绩修复可期。

### 利好 2：经营现金流 +6,772% 暴涨 67 倍

**依据**

Q1-Q3 经营现金流 **4.42 亿（+6,772.14%）**，回款大幅改善；Q1-Q3 现金流超归母净利润（归母 2.72 亿 < OCF 4.42 亿）。

**解读**

经营现金流暴涨 67 倍 + 远超净利润 → **回款质量极高 + 存货 / 应收优化 + 业务扎实**。说明虽然利润下滑，但**现金创造能力显著增强**，为后续分红 / 新品投入 / 并购提供弹药。

### 利好 3：瑷科缦 +93.90% + 多品牌矩阵开花

**依据**

**瑷科缦（AOXMED）H1 +93.90%** + **天猫美妆国货新品牌第二** + **绿安缦系列爆款**；薇诺娜宝贝 +8.62% + 婴童洗护赛道前列；浴见抖音双十一身体洗护前十；姬芮 + 泊美收购整合推进；薇诺娜菁白 + 悦芙媞美白 / 功效新品。

**解读**

**多品牌矩阵正在跑通**，高端（瑷科缦）/ 婴童（薇诺娜宝贝）/ 身体护理（浴见）三条新曲线同步爆发。若 2026-2027 新曲线继续翻倍增长，**多品牌协同贡献 20-30 亿营收规模**，将实质性扭转主品牌依赖。

### 利好 4：薇诺娜主品牌品牌力稳固 + 敏感肌赛道龙头

**依据**

薇诺娜**连续 5 年中国敏感肌护肤销售额第一**；天猫美妆预售榜单前十；H1 薇诺娜毛利率 **78.3%（+4.2pct）**。

**解读**

薇诺娜核心品牌力 + 敏感肌用户心智仍稳固。H1 调整主要为主动去库存 + 产品矩阵升级 + 价格带调整。若 2026 年完成调整 + **主品牌回归低个位数增长** + 多品牌矩阵继续 20%+ 增长 → **双轮驱动业绩修复**。

### 利好 5：高端化 + 瑷科缦 AOXMED 爆发

**依据**

瑷科缦 H1 +93.90% + 绿安缦系列爆款 + 天猫美妆国货新品牌第二 + 高端抗衰护肤（客单价高）。

**解读**

高端化是国货美妆穿越周期核心抓手。瑷科缦 + 绿安缦等**高端产品 + 高毛利 + 高客单价** → 带动整体毛利率维持 74-78% 高位。若 2026 年瑷科缦突破 5 亿+ 规模，**将贡献核心利润增量**。

## 7. 未来可能的利空

### 利空 1：薇诺娜主品牌持续下滑

**依据**

主品牌薇诺娜 H1 **-18.4%（19.5 亿）**，占总营收 80%+。敏感肌赛道竞争加剧（润百颜 / 可复美 / 薇婷等）。大单品特护霜 / 清痘霜等迭代压力。

**解读**

薇诺娜作为收入核心基本盘 -18.4% → **整体业绩直接承压**。若 2026 年持续下滑 + 多品牌增量不足以对冲，整体业绩修复节奏将延后。**薇诺娜能否企稳是最核心观察点**。

### 利空 2：化妆品行业价格战 + 流量成本高企

**依据**

国货美妆品牌众多（珀莱雅 / 毛戈平 / 韩束 / 可复美等）；抖音 / 小红书 / 天猫流量成本持续上升；直播电商价格战；海外品牌（雅诗兰黛 / 兰蔻 / 欧莱雅）高端品压制。

**解读**

化妆品行业**流量成本年均 +15-20%** + 终端价格战 + 营销投放压力持续，毛利率虽维持高位但**销售费用率被动提升** → 净利率承压。

### 利空 3：线下渠道严重萎缩

**依据**

H1 线下 **4.24 亿（-41.58%）**，占比降至 17.9%。屈臣氏 / CS 店 / 连锁药房渠道下滑，KA 商超萎缩。

**解读**

线下加速萎缩反映**传统美妆渠道彻底瓦解** + 品牌心智切换至线上 + 线下高毛利渠道消失可能影响整体毛利结构。

### 利空 4：Q3 营收仍同比 -10% + 环比 Q2 -23%

**依据**

Q3 营收 10.9 亿（-10%）；Q3 环比 Q2（14.23 亿）-23%；Q3 单季营收并未迎来强劲反弹。

**解读**

Q3 归母同比 +136.55%（低基数）但营收仍 -10% + 环比下滑 23% → **收入端尚未真正反弹**。双十一 Q4 表现 + 2026 年增速修复仍需验证。

### 利空 5：竞争对手蚕食份额 + 并购整合风险

**依据**

珀莱雅 / 可复美 / 可丽金 / 毛戈平 / 润百颜等快速增长；华熙生物 / 韩束 / 敷尔佳等功能性护肤全面竞争；外资品牌仍在加码中国市场。2023 年收购姬芮 + 泊美（资生堂品牌），2024-2025 年逐步整合 + 投入期。

**解读**

功能性护肤 / 敏感肌护肤赛道竞争加剧，公司份额若被蚕食 + 薇诺娜作为标杆品牌受冲击，长期增长面临挑战。姬芮 / 泊美被收购品牌整合周期长 + 管理模式差异 + 短期可能拖累盈利，若未跑通则存在减值风险。

## 8. 总结判断

**一句话结论**

贝泰妮 2025 Q1-Q3 呈现"营收 34.64 亿（-13.78%）+ 归母 2.72 亿（-34.45%）+ 扣非 1.84 亿（-46.17%）+ 毛利率 74.33% + 薇诺娜主品牌 H1 -18.4% + 瑷科缦 +93.90% + 线下 -41.58% + Q1 归母 -83.97% + Q2 净利率 15.3%（+12.35pct）+ Q3 归母 +136.55% + 经营现金流 4.42 亿（+6,772%）"的"**主品牌深度调整 + 子品牌爆发 + Q2/Q3 拐点 + 现金流暴涨 67 倍**"格局；2025 年年报尚未披露，2026 关键看薇诺娜主品牌企稳 + 瑷科缦持续爆发 + 多品牌协同兑现。

**核心看点**

1. **Q3 归母 +136.55% + Q2 净利率 +12.35pct 业绩拐点**
2. **瑷科缦 +93.90% + 多品牌矩阵（薇诺娜宝贝 / 浴见 / 菁白 / 悦芙媞）开花**
3. **经营现金流 +6,772% 至 4.42 亿**（回款质量大幅改善）

**核心风险**

1. **薇诺娜主品牌 H1 -18.4% 持续下滑**（占营收 80%+）
2. **化妆品价格战 + 流量成本高企** + 线下渠道萎缩 41.58%
3. **珀莱雅 / 可复美 / 毛戈平**等竞争对手蚕食份额 + 姬芮 / 泊美整合风险
