# 华利集团 2025年财报整理

> **说明**：华利集团（300979.SZ）**2025 年年度报告已于 2026 年 4 月正式披露**；业绩快报于 2026-03-10 披露。本文基于**2025 年年度报告 + 业绩快报 + 券商研报**整理。

## 1. 业绩总结

### 1.1 收入与利润表现

华利集团 2025 年实现营业收入 249.80 亿元，**同比 +4.06%**；归母净利润 32.07 亿元，**同比 -16.50%**；扣非归母净利润 32.57 亿元，**同比 -13.87%**。**销售毛利率 21.89%（-4.92pct）**—— 新工厂爬坡 + 产能调配导致单位成本上升。**期间费用率 4.51%（-1.49pct）**（销售 + 管理费用合计 -33%）—— 费用管控表现优秀。全年**销售运动鞋 2.27 亿双（+1.59%）**，均价约 **110 元/双（+2.22%）**。经营活动现金流 **37.43 亿（-18.93%）**。分红 **24.5 亿元（每 10 股派 21 元）**，**分红率 76%**（股息率 ~4.5%）。

- 营业收入 249.80 亿元（同比 +4.06%）
- 营业利润 41.73 亿元（同比 -15.98%）
- 归母净利润 32.07 亿元（同比 -16.50%）

### 1.2 增长与拖累因素

- **主要增长驱动**：①**新客户订单大幅增长**（**New Balance 跻身前五大客户**）+ Hoka / On / Lululemon 等新兴品牌放量；②**欧洲 +56.71%** + **其他地区 +52.80%**（地区结构优化）；③均价 **+2.22%**（产品结构升级）；④费用率 **-1.49pct** + 销售 + 管理费用 **-33%**（降本增效）；⑤**前五大客户占比从 79.13% → 72.55%**（客户集中度下降）
- **主要拖累因素**：①**毛利率 -4.92pct 至 21.89%**（新工厂爬坡 + 印尼工厂尚未完全盈利）；②**北美 -4.87% 至 194.12 亿**（老客户订单下滑）；③Q4 利润同比 -22.66%（逐季走弱）；④**美国对越南 20%、印尼 19% 加征关税**扰动订单节奏；⑤经营现金流 **-18.93%**
- **增长主要来自**：新客户 + 欧洲 / 其他地区 + 费用管控；**承压主要来自**：新工厂爬坡 + 美国关税 + 北美下滑

### 1.3 趋势判断

- **未来主要利好方向**：新工厂全面渡过爬坡期 + 毛利率修复 + 新客户（New Balance / Hoka / On）订单持续高增 + 欧洲 + 其他地区高速扩张 + 高分红回报（76%）
- **未来主要压力方向**：美国关税政策反复 + 北美需求疲软 + 新工厂爬坡节奏 + 行业竞争（丰泰、裕元等东南亚同业扩产）
- **整体趋势判断**：**2026 年核心看毛利率修复 + 新工厂盈利兑现**。公司处于短期压力期但长期竞争力稳固，运动鞋代工龙头地位 + 深度绑定 Nike/Adidas + 极高分红 + 客户结构优化 + 地区多元化，长期成长逻辑未变

## 2. 主营业务

### 2.1 公司简介

华利集团是**全球领先的运动鞋专业制造商（OEM/ODM）**，主要为国际头部运动品牌提供运动鞋产品开发设计、生产与销售服务。生产基地主要位于**越南和印尼**，销往全球。

### 2.2 主营业务构成

1. **运动鞋制造（核心主业）**：为 Nike、Adidas、Converse、Vans、UGG、Hoka、On、**New Balance（新晋前五大客户）**、Puma、Asics、Under Armour、Reebok、Lululemon 等品牌代工生产运动鞋
2. **产品开发设计**：提供从设计到生产的全链条服务，增强客户粘性
3. **鞋材及模具生产**：配套鞋材和模具制造，提升供应链自主可控能力

### 2.3 核心收入来源

公司收入几乎全部来自运动鞋制造，下游客户为全球头部运动品牌。**前五大客户占比从 2024 年 79.13% 降至 2025 年 72.55%**（客户集中度下降，新客户放量）。生产基地主要位于越南和印尼，产品销往北美（194.12 亿，占比 77.7%）、欧洲（+56.71%）、其他地区（+52.80%）等全球市场。收入取决于销量（2.27 亿双）和均价（约 110 元/双）。

## 6. 未来可能的利好

### 利好 1：新工厂渡过爬坡期释放利润弹性

**依据**

2025 年 Q3 已有 3 家新工厂实现盈利；公司预计 2025-2026 年产量复合增长率有望达 **15%** 左右；新工厂产能利用率将逐步提升。

**解读**

**毛利率 -4.92pct 至 21.89% 的核心原因是新工厂爬坡期的低产能利用率 + 折旧 + 人员培训成本**。一旦新工厂满产运转，毛利率将回升至正常水平（25%+）甚至更高（规模效应 + 新设备效率）。2026 年是**毛利率修复 + 利润恢复增长**的关键之年，若能回到 25% 毛利率水平，对应 **利润弹性可达 +8-10 亿**（相当于归母 +25-30%）。

### 利好 2：客户结构优化，New Balance 跻身前五

**依据**

**New Balance 跻身前五大客户**（前五大客户包括 Nike / Adidas / Deckers(UGG/Hoka) / Vans / New Balance）；**前五大客户占比从 79.13% 降至 72.55%**（-6.58pct）；Hoka / On / Lululemon 等新兴高增长品牌订单大幅增长。

**解读**

**客户集中度从 79% 降至 72.55%**，**客户结构多元化进入实质阶段**，大幅降低了对单一品牌的依赖风险。New Balance 跻身前五后，若后续订单增量持续释放，**2026-2027 年收入增速有望重回 10%+**。Hoka、On 等快速增长品牌的订单增量是未来 2-3 年核心增长来源。

### 利好 3：欧洲 +56.71% + 其他地区 +52.80% 地区多元化

**依据**

2025 年**欧洲地区收入 +56.71%** + **其他地区 +52.80%**，而北美 -4.87%。

**解读**

**地区结构从北美独大向多元化转变**，大幅降低了对美国关税政策的敏感度。若欧洲 / 其他地区持续高速增长 + 2026 年北美企稳，**公司整体收入增速有望重回 +10%**。这也是应对美国 "对越南 20% / 印尼 19% 关税" 的最有效对冲。

### 利好 4：高分红回报率（76%）彰显经营信心

**依据**

2025 年全年**分红 24.5 亿元**（每股 2.1 元）+ **分红率 76%**（股息率约 4.5%）；公司资产负债率仅 23.38%，经营现金流 37.43 亿元充裕。

**解读**

**分红率 76%** 在制造业中极为罕见 + **股息率 4.5%** 具备较强配置价值。体现了管理层对公司现金流和长期盈利能力的信心。对于注重股息回报的投资者具有较强吸引力，为估值提供"底部支撑"。

### 利好 5：费用管控优秀 + 研发持续投入

**依据**

**期间费用率 4.51%（-1.49pct）** + 销售 + 管理费用合计 **-33%**；**研发费用率 1.66%（+0.10pct）** 持续提升。

**解读**

销售 + 管理费用 -33% 反映公司对成本的极强管控能力 —— 在毛利率承压的情况下通过费用端"挤出"大量利润空间。研发费用率持续提升保证产品迭代能力。**"降费增效"是穿越行业下行期的核心能力**。

## 7. 未来可能的利空

### 利空 1：美国关税政策冲击

**依据**

美国对**越南加征 20% 关税**、对**印尼加征 19% 关税**；公司产品主要出口美国市场（北美占比 77.7%）；虽然公司可根据订单销售目的地调配产地，但短期内影响客户订单节奏和利润率。

**解读**

关税加征可能导致品牌客户重新评估供应链布局，短期内影响订单节奏。若关税政策反复或进一步升级，可能压制下游品牌需求或转嫁成本压力至代工厂。**北美 -4.87%** 已反映部分影响。

### 利空 2：新工厂爬坡期持续拖累盈利

**依据**

多家新工厂处于投产初期，印尼新工厂尚未完全盈利；产能调配给毛利率带来短期压力；**毛利率 -4.92pct**；Q4 净利润同比 -22.66%（利润逐季走弱）。

**解读**

若新工厂爬坡速度慢于预期（人员招聘 / 培训、良率提升、设备调试等），**毛利率承压时间可能延长至 2026 H2**，2026 年利润恢复节奏可能不及预期。需密切跟踪 **新工厂盈利进度 + 季度毛利率修复节奏**。

### 利空 3：北美下滑 + 下游品牌需求波动

**依据**

**北美 -4.87%（194.12 亿，占比 77.7%）**；部分老客户订单下滑；全球运动鞋市场增速放缓，部分品牌进入库存调整周期。

**解读**

若全球消费需求持续疲软或主要品牌客户（Nike / Adidas 等）订单进一步缩减，公司收入增速可能继续放缓。北美作为最大市场（77.7%），其波动直接影响整体业绩。

### 利空 4：行业竞争加剧

**依据**

制鞋行业竞争者（**丰泰、裕元**等）也在东南亚扩产；同业扩产可能加剧订单竞争和压制代工费用率。

**解读**

运动鞋代工行业进入产能释放期，若同业扩产速度过快 + 行业供给大于需求，**代工费用率可能被动下降**，压制公司议价能力和毛利率修复节奏。

### 利空 5：Q4 利润加速下滑

**依据**

**Q4 利润同比 -22.66%**（全年 -16.50% 的基础上加速下滑）；全年增速从 Q1（+12.34%）逐季放缓至 Q4（-3.0%）。

**解读**

Q4 利润加速下滑反映 **2026 年初仍面临压力** + 关税扰动 + 新工厂爬坡延续。若 2026 Q1 继续下滑，业绩弹性兑现时点将延后。

## 8. 总结判断

**一句话结论**

华利集团 2025 年呈现"营收 249.8 亿（+4.06%）+ 归母 32.07 亿（-16.50%）+ 毛利率 21.89%（-4.92pct）+ 新客户订单大增 + New Balance 跻身前五 + 前五大客户占比 79.13% → 72.55% + 欧洲 +56.71% + 其他地区 +52.80% + 北美 -4.87% + 费用率 -1.49pct + 分红率 76% + 股息率 4.5%"的"**新工厂爬坡 + 关税扰动 + 客户多元化 + 高分红**"格局；2026 年核心看**毛利率修复 + 新工厂盈利 + 欧洲持续高增**。

**核心看点**

1. **新工厂 2026 年全面渡过爬坡期，毛利率修复至 25%+**（利润弹性 +8-10 亿）
2. **客户集中度下降（79.13% → 72.55%）+ New Balance 跻身前五 + 新客户订单持续高增**
3. **欧洲 +56.71% + 其他地区 +52.80% 地区多元化 + 高分红率 76%**

**核心风险**

1. **美国关税政策反复**（越南 20% / 印尼 19%）+ **北美需求疲软**（-4.87%）
2. **新工厂爬坡慢于预期**（毛利率修复延迟）
3. **Q4 利润加速下滑 -22.66%** + 同业扩产加剧（丰泰、裕元）
