# 中国交建（601800）2025年财报整理

> **数据来源：** 中国交建（601800）2025年年度报告（2026年3月30日发布），数据完整。营收-5%、归母-37%大幅下滑。海外+9%是唯一亮点。新签合同1.88万亿基本持平。

## 1. 业绩总结

### 1.1 收入与利润表现

2025年中国交建营业收入7,311.09亿元，同比 -5.29%；归母净利润147.51亿元，同比 -36.92%；扣非归母111.98亿元（-43.48%）。利润降幅远大于收入降幅，毛利率-0.92ppt至11.37%，净利率-1.09ppt至2.85%。新签合同18,837亿元（+0.13%）基本持平，在手订单34,460亿元。

- 全年营业收入7,311.09亿元，同比 -5.29%
- 全年归母净利润147.51亿元，同比 -36.92%
- 扣非归母净利润111.98亿元，同比 -43.48%

### 1.2 增长与拖累因素

海外新签合同+9%（3,924亿元，占比21%）是唯一亮点。基建建设占营收88%，该板块收入下降是营收下滑的主因。利润降幅（-37%）远大于收入降幅（-5%），说明利润率压缩严重——毛利率-0.92ppt、净利率-1.09ppt。应收账款1,552亿元增长较快，回款压力增大。

- 主要增长驱动：海外+9%、新签合同持平
- 主要拖累因素：营收-5%、归母-37%、毛利率-0.92ppt
- 增长或承压主要来自：基建投资下行+利润率压缩

### 1.3 趋势判断

中国交建是全球最大的交通基础设施建设企业之一，业务涵盖公路、桥梁、港口、疏浚等领域。2025年营收-5%、归母-37%是近年最差表现，反映了国内基建投资增速放缓和利润率承压。但34,460亿在手订单（约4.7年收入）提供了收入保障，海外（21%占比）是相对亮点。

- 未来主要利好方向：海外扩张、新兴业务、在手订单转化
- 未来主要压力方向：基建投资下行、利润率压缩、回款压力
- 整体趋势判断：行业性下行期，海外是突围方向

## 2. 主营业务

### 2.1 公司简介

中国交通建设股份有限公司（601800.SH / 1800.HK）是全球最大的港口设计建设企业、公路桥梁设计建设企业和疏浚企业之一。业务涵盖基建建设、基建设计、疏浚工程、港口建设等，海外业务规模居建筑央企首位。

### 2.2 主营业务构成（收入占比）

1. **基建建设**：占营收88.48%（公路、桥梁、铁路、市政等）
2. **疏浚工程**：占营收6.92%
3. **基建设计**：占营收4.08%
4. **其他**：占营收3.63%

### 2.3 核心收入来源

基建建设是绝对主体（88%），疏浚和设计是差异化竞争优势。海外占新签合同21%，是建筑央企中国际化程度最高的。

## 4. 营收分布和增长

### 4.1 基建建设——88%收入贡献

**占营收88.48%**

基建建设是绝对主体。国内基建投资增速放缓是收入-5%的直接原因。新签合同17,224亿元在基建业务中基本持平，说明未来收入有保障但增长有限。

### 4.2 疏浚——差异化优势

**占营收6.92%，新签1,069亿元**

疏浚是中国交建的核心差异化业务——在全球疏浚市场拥有领先地位。疏浚业务的利润率通常高于基建建设。

### 4.3 海外——唯一亮点+9%

**海外新签3,924亿元（+9.09%），占比21%**

海外业务是中国交建最大的增长亮点。+9%的增速在建筑企业中表现突出。一带一路相关合同占海外的78%，覆盖非洲、东南亚、中东等市场。海外业务的利润率有望高于国内。

### 5.1 整体利润表现

| 毛利率 | 11.37% | -0.92ppt | |
| 净利率 | 2.85% | -1.09ppt | |

### 5.2 利润分析

**利润降幅（-37%）远大于收入降幅（-5%）**

收入仅-5%但归母-37%，利润弹性极大。原因：
1. 毛利率-0.92ppt：低毛利项目占比增加、竞争加剧导致中标价格下降
2. 净利率-1.09ppt：毛利率下降叠加期间费用相对刚性
3. 应收账款增长较快（回款变慢）可能导致坏账计提增加
4. 有息负债7,583亿（+13%）带来的财务费用增加

**净利率2.85%——接近盈亏平衡**

从3.9%降至2.85%，万亿级收入规模下任何成本变动都会放大利润影响。如果净利率继续下降至2%以下，公司将面临严峻的盈利压力。

## 6. 未来可能的利好

### 利好 1：34,460亿在手订单——4.7年收入保障

**依据**

在手未完工合同34,460亿元，约为年收入的4.7倍。

**解读**

充裕的在手订单提供了收入的确定性。即使新签合同增速放缓，在手订单的转化也能支撑未来几年的收入水平。

### 利好 2：海外+9%——全球化领先

**依据**

海外新签3,924亿元（+9%），占比21%，建筑央企首位。

**解读**

海外市场是突围方向。一带一路沿线国家的基建需求仍然旺盛。如果海外占比从21%提升至30%，将显著改善整体增长和利润率。

### 利好 3：新兴业务6,655亿——转型发力

**依据**

新兴业务领域新签合同6,655亿元。

**解读**

新兴业务（新能源、生态环保、智慧交通等）的拓展有助于降低对传统基建的依赖，打开新的增长空间。

## 7. 未来可能的利空

### 利空 1：归母-37%，利润率严重承压

**依据**

归母147.51亿元（-37%），毛利率-0.92ppt，净利率-1.09ppt至2.85%。

**解读**

利润降幅远大于收入降幅是最大的警示信号。如果2026年利润率不能企稳，归母可能继续下降至120亿以下。2.85%的净利率在大型建筑企业中已处于低位。

### 利空 2：应收账款和有息负债增长

**依据**

应收账款增长较快，有息负债7,583亿元（+13%）。

**解读**

回款压力增大和融资成本上升是双重打击。在基建行业景气度下行期，地方政府的支付能力减弱，应收账款可能进一步增加。有息负债+13%带来的利息支出增加将直接侵蚀利润。

### 利空 3：分红率仅21.5%——偏低

**依据**

分红31.72亿元，分红率21.5%。

**解读**

21.5%的分红率在大型央企中偏低。如果利润继续下降，分红金额可能进一步缩减，影响投资者信心。

## 8. 总结判断

**一句话结论**

2025年中国交建营收7,311亿元（-5%）、归母148亿元（-37%）大幅下滑；基建建设占88%，新签1.88万亿（+0.1%），在手3.45万亿；海外+9%（3,924亿，占21%）是唯一亮点；毛利率-0.9ppt至11.4%，净利率-1.1ppt至2.85%。

**核心看点**

1. 在手订单34,460亿元，约4.7年收入保障
2. 海外新签+9%（3,924亿），建筑央企首位
3. 新兴业务新签6,655亿元
4. 疏浚业务全球领先

**核心风险**

1. **归母-37%**至148亿元，利润大幅下滑
2. **扣非-43%**至112亿元
3. **净利率仅2.85%**（-1.1ppt），接近盈亏边缘
4. 应收账款增长+有息负债+13%至7,583亿
5. 分红率仅21.5%
