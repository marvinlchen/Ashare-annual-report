# 中国平安 2025年财报整理

> **数据来源：** 中国平安（601318.SH / 02318.HK）2025 年年度报告已于 **2026-03-26** 正式披露（A+H 股，收盘后），数据完整。**审计机构：安永华明会计师事务所（特殊普通合伙）**，出具标准无保留意见审计报告。**关键审计事项：保险合同负债计量、发放贷款及垫款和债权投资的减值评估**。**综合金融集团，保险主业强劲复苏**。

# 中国平安（601318）2025年财报与经营分析

## 1. 业绩总结

### 1.1 收入与利润表现

2025年中国平安营业收入10,505亿元，同比 +2.1%；归母净利润1,348亿元，同比 +6.5%；归母营运利润1,344亿元，同比 +10.3%（Q4单季+35.3%加速）。保险主业全面向好：寿险NBV+29.3%，财险COR降至96.8%（承保利润+96%）。归母净资产首次突破万亿（10,004亿元，+7.7%），每股股息2.70元（+5.9%）。

- 全年营业收入10,505.06亿元，同比 +2.1%
- 全年归母净利润1,347.78亿元，同比 +6.5%
- 归母营运利润1,344.15亿元，同比 +10.3%

### 1.2 增长与拖累因素

保险主业（寿险+财险）是利润增长的核心驱动力。寿险NBV+29.3%体现了代理人产能提升和银保渠道爆发。财险承保利润+96%因综合成本率大幅改善。但银行业务营运利润-4.2%拖累，资管业务仍在减亏中。扣非净利润+22.5%（1,438亿元）显著高于归母+6.5%，说明非经常性损失较大。

- 主要增长驱动：寿险NBV+29.3%、财险COR 96.8%（-1.5pct）、综合投资收益率6.3%（+0.5pct）
- 主要拖累因素：银行营运利润-4.2%、资管亏损38亿（虽减亏68%）
- 增长或承压主要来自：保险主业全面向好，投资端改善叠加

### 1.3 趋势判断

中国平安正从"低谷期"走出。2020-2023年因寿险改革阵痛和投资端承压经历了利润低迷，2024-2025年保险主业全面复苏、投资回暖推动利润重回增长轨道。营运利润+10.3%是更能反映核心盈利趋势的指标。NBV连续高增长意味着未来数年利润释放的确定性在增强。

- 未来主要利好方向：NBV持续高增长、投资端配置优化、分红持续提升
- 未来主要压力方向：银行业务承压、利率下行影响保险投资、资管扭亏尚需时日
- 整体趋势判断：保险主业复苏确认，估值修复空间依然存在

## 2. 主营业务

### 2.1 公司简介

中国平安（601318.SH / 2318.HK）是中国最大的综合金融集团，以保险为核心，涵盖银行、证券、信托、资产管理、科技等业务。总资产超12万亿元，个人客户2.51亿。旗下平安寿险、平安产险、平安银行均为各自领域的头部公司。

### 2.2 主营业务构成

1. **寿险及健康险**：核心利润来源，营运利润998亿元（+2.9%），NBV 369亿元（+29.3%）
2. **财产保险**：车险+非车险，原保费3,432亿元（+6.6%），营运利润169亿元（+13.2%）
3. **平安银行**：营运利润247亿元（-4.2%）
4. **资产管理**：仍在减亏中，亏损38亿元（减亏68.2%）
5. **科技及其他**：金融赋能等

### 2.3 核心收入来源

寿险及健康险是绝对利润核心（营运利润998亿元，占比约74%）。财险（169亿，13%）和银行（247亿，18%）分列二三。资管和科技尚未贡献正利润。

## 4. 营收分布和增长

### 4.1 保险主业全面向好

**寿险NBV+29.3%——复苏确认**

369亿元的NBV创近年新高，+29.3%的增速体现了寿险改革成效。个险渠道NBV+10.4%是基本盘，银保渠道+138%（贡献25.5%）是增量引擎。代理人虽减少3.3%至35.1万人，但人均产能大幅提升。NBV利润率23.4%（+4.9pct）说明产品结构持续优化。

**财险COR 96.8%——承保利润接近翻倍**

综合成本率96.8%（-1.5pct）是近年最好水平，其中车险95.8%（-2.3pct）表现出色。承保利润107亿元（+96.2%）接近翻倍。原保费+6.6%至3,432亿元，规模稳步增长。

### 4.2 银行业务承压

平安银行营运利润247亿元（-4.2%），受行业息差下行影响。在平安集团的综合金融体系中，银行主要扮演渠道（银保联动）和客户获取的角色。

### 4.3 投资端改善

综合投资收益率6.3%（+0.5pct），受益于2025年权益市场回暖。股票配置比例提升至14.8%（+7.2pct），其中FVOCI占57%，通过FVOCI方式配置权益可以平滑利润波动。

### 5.2 利润分析

**营运利润+10.3%更反映核心趋势**

归母净利润+6.5%低于营运利润+10.3%，主因非经常性损失（可能包括投资波动或资管相关减值等）。扣非净利润+22.5%（1,438亿元）显著高于归母，说明非经常性项目是负面的。

营运利润的三驾马车：寿险998亿（+2.9%）、银行247亿（-4.2%）、财险169亿（+13.2%）。寿险虽然营运利润增速仅+2.9%，但NBV+29.3%意味着未来数年利润释放将加速。

## 6. 未来可能的利好

### 利好 1：寿险NBV持续高增长，未来利润释放确定性强

**依据**

NBV 369亿元（+29.3%），银保渠道+138%，NBVM 23.4%（+4.9pct）。代理人产能持续提升。

**解读**

NBV是寿险业务未来利润的先行指标——今天的新业务价值将在未来15-20年内逐步释放为利润。29.3%的高增速意味着平安寿险的"利润蓄水池"正在快速扩大。如果NBV能在2026-2027年保持15-20%的增长，寿险营运利润的增速将从2.9%逐步提升至8-12%。银保渠道的爆发（+138%）为增长提供了新引擎。

### 利好 2：财险承保利润翻倍，COR持续优化

**依据**

承保利润107亿元（+96.2%），COR 96.8%（-1.5pct），车险COR 95.8%（-2.3pct）。

**解读**

财险承保利润从约55亿翻倍至107亿，体现了平安在风险定价和成本控制上的领先能力。如果COR能维持在97%以下，按3,432亿保费计算，承保利润稳定在100亿元以上。加上投资收益，财险板块的ROE将继续改善。

### 利好 3：净资产首破万亿，估值仍有修复空间

**依据**

归母净资产10,004亿元（+7.7%），寿险EV+11.2%，集团EV+5.7%。分红2.70元/股（+5.9%）。

**解读**

万亿净资产标志着平安的资本实力达到新高度。寿险EV+11.2%的增长远超市场预期，为股价提供了坚实的估值锚。如果当前股价对应P/EV低于0.8倍，则估值修复空间仍然充足。

## 7. 未来可能的利空

### 利空 1：利率下行影响投资收益和保险负债

**依据**

净投资收益率3.7%（-0.1pct），持续小幅下降。长端利率下行增加保险负债的再投资风险。

**解读**

利率下行是所有保险公司面临的长期挑战。净投资收益率从过去的4.5%+降至3.7%，意味着新增保费的投资回报在降低。如果利率继续下行，可能需要调整分红保单预定利率，影响产品吸引力和NBV增长。

### 利空 2：银行业务持续承压

**依据**

平安银行营运利润247亿元（-4.2%），受行业息差下行影响。

**解读**

银行贡献了约18%的营运利润，其持续下降对集团利润构成拖累。如果2026年银行息差继续收窄，银行营运利润可能进一步下降至230-240亿元。在寿险和财险增长的背景下，银行的拖累效应可能被对冲，但如果降幅超预期，会影响整体增速。

### 利空 3：资管业务扭亏时间不确定

**依据**

资管业务亏损38亿元（虽减亏68.2%），仍未实现盈利。

**解读**

资管业务（主要是平安信托的不动产相关业务）的亏损反映了此前房地产项目的遗留风险。虽然亏损大幅收窄，但不动产市场的低迷可能导致扭亏时间推迟。如果2026年仍亏损20-30亿元，将继续侵蚀集团利润。

## 8. 总结判断

**一句话结论**

2025年中国平安营收10,505亿元（+2%）、归母营运利润1,344亿元（+10%），保险主业全面复苏——寿险NBV+29.3%、财险COR 96.8%（承保利润翻倍）、综合投资收益率6.3%，净资产首破万亿；每股股息2.70元（+5.9%），估值修复空间仍在。

**核心看点**

1. **寿险NBV 369亿元（+29.3%）**，银保渠道+138%，复苏确认
2. **财险COR 96.8%（-1.5pct）**，承保利润107亿元（+96%）
3. 归母营运利润+10.3%，Q4加速至+35.3%
4. 净资产首破万亿（10,004亿元，+7.7%），寿险EV+11.2%
5. 综合投资收益率6.3%（+0.5pct），投资端改善
6. 每股股息2.70元（+5.9%），分红率36.4%

**核心风险**

1. 利率下行影响投资收益和保险负债
2. 银行营运利润-4.2%，持续拖累
3. 资管亏损38亿，扭亏时间不确定
4. 归母净利润+6.5%低于营运利润+10.3%，非经常性因素影响
