# 豪恩汽电（301488）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

2025年豪恩汽电实现营业收入17.88亿元，同比 +26.26%；归母净利润7495万元，同比 -24.68%；扣非净利润6442万元，同比 -23.78%。典型的"增收不增利"格局：收入高速增长但利润显著下滑。

### 1.2 增长与拖累因素

收入增长主要来自智能驾驶感知系统（超声波/毫米波雷达、视觉系统、域控系统）在国内外整车厂的放量，以及机器人/低空经济新业务初步贡献。利润下降的核心原因是毛利率持续下降（2023年22.71%→2024年20.93%→2025年预计进一步下滑）、研发投入加大（占收入12.21%，约2.18亿元）、以及管理和销售费用增长。

### 1.3 趋势判断

豪恩汽电处于"智能驾驶主业放量+机器人/低空经济第二曲线孵化"阶段。短期利润承压（新业务投入大、行业降价压力），中期看点在于机器人感知产品量产（与智元合作）和eVTOL产品2026年规模化交付。需要关注毛利率能否止跌以及新业务何时贡献利润。

---

## 2. 主营业务

### 2.1 公司简介

豪恩汽电是国内领先的汽车智能驾驶感知系统供应商，为主流整车厂提供超声波雷达、毫米波雷达、视觉感知和域控系统一体化解决方案，同时布局机器人感知和低空经济（eVTOL）领域。

### 2.2 主营业务构成

1. **智能驾驶感知系统**：核心主业，涵盖超声波雷达、毫米波雷达、车载摄像系统（视觉感知），覆盖L2-L3级自动驾驶
2. **域控系统**：智能驾驶域控制器，与感知系统协同
3. **机器人感知产品**：已量产交付，基于英伟达Jetson Thor芯片的机器人域控系统（2070TOPS算力），与智元合作超声波传感器
4. **低空经济（eVTOL/无人机）**：为eVTOL和工业无人机提供感知与决策系统，已获项目定点，2026年规模化量产

### 2.3 核心收入来源

汽车智能驾驶感知系统是核心收入来源，客户为国内外主流整车厂。机器人和低空经济为新增业务，收入贡献尚小。研发团队546人，占员工总数34%。

## 4. 营收分布和增长

### 4.1 按业务划分

年报未单独披露分产品收入明细。主要产品包括：

| 产品类别 | 备注 |
|---|---|
| 车载摄像系统（视觉感知） | 核心产品 |
| 超声波雷达系统 | 传统优势产品 |
| 毫米波雷达系统 | 增长产品 |
| 域控系统 | 高附加值产品 |
| 机器人感知产品 | 新业务，已量产 |
| 低空经济产品 | 新业务，2026年放量 |

### 4.2 营收分析

**主要增长来源**

1. 智能驾驶感知系统在国内外整车厂的渗透率提升（L2+渗透率快速增长）
2. 毫米波雷达和域控系统等高价值产品放量
3. 机器人感知产品开始量产交付（增量贡献）

**主要拖累因素**

行业降价压力持续（整车厂成本压缩传导至零部件供应商），新业务（机器人、eVTOL）处于投入期尚未贡献利润。

**结构变化**

从超声波雷达单一产品向"超声波+毫米波+视觉+域控"全感知体系拓展，同时向机器人和低空经济延伸。

### 5.1 整体利润表现

| 净利率 | 4.19% | ~7.03% | ~-2.84pp | |

### 5.2 利润下降原因分析

**毛利率下降**

汽车零部件行业整体降价压力大，整车厂持续向供应链压价。毛利率从2023年22.71%降至2024年20.93%，2025年预计进一步下降。

**费用率上升**

研发投入12.21%（约2.18亿元）为机器人和低空经济等新业务投入。管理和销售费用增长也快于收入增速。

**新业务投入期亏损**

机器人和eVTOL业务处于研发投入和早期量产阶段，尚未贡献正向利润，反而增加了费用。

## 6. 未来可能的利好

### 利好 1：机器人感知产品放量

**依据**

机器人感知产品已实现量产交付。基于英伟达Jetson Thor芯片打造2070TOPS算力的机器人域控系统。与智元机器人合作研发超声波传感器。

**解读**

人形机器人量产浪潮中，感知系统（超声波+视觉+域控）是核心组件。公司在汽车感知领域的技术积累可以直接迁移到机器人，如果头部机器人客户放量，将打开高增长空间。

### 利好 2：低空经济eVTOL产品2026年规模化交付

**依据**

已获得国内领先低空飞行器企业的项目定点，产品覆盖eVTOL和工业无人机的感知与决策系统，2026年规模化量产交付。

**解读**

低空经济是政策风口（2025年"低空经济元年"），eVTOL对感知系统的需求类似于汽车ADAS但要求更高。先发定点优势有望带来可观的增量收入。

### 利好 3：收入高速增长，市占率持续提升

**依据**

收入+26.26%，显著高于汽车行业平均增速。覆盖国内外主流整车厂。

**解读**

L2+智能驾驶渗透率快速提升，作为感知系统供应商直接受益。26%的收入增速说明市占率在提升，规模效应未来有望改善利润率。

## 7. 未来可能的利空

### 利空 1：毛利率持续下降，"增收不增利"困境

**依据**

毛利率从2023年22.71%→2024年20.93%→2025年预计继续下降。收入+26%但利润-25%。净利率从7%降至4.2%。

**解读**

整车厂持续压价是结构性问题。如果毛利率不能止跌，收入增长将无法转化为利润增长。4.2%的净利率已经很薄，进一步下滑将威胁盈利能力。

### 利空 2：研发高投入但回报周期长

**依据**

研发占收入12.21%（~2.18亿），研发团队546人占34%。机器人和eVTOL业务尚在投入期。

**解读**

新业务（机器人、eVTOL）研发投入大但收入贡献小，投入回报周期可能较长。如果新业务放量不及预期，高研发费用将持续压制利润。

### 利空 3：现金流偏弱，资产负债率上升

**依据**

2024年经营现金流仅0.81亿元（同比大降），资产负债率从37%升至41%。

**解读**

现金流偏弱+负债率上升说明公司在快速扩张中资金压力加大。如果收入增长放缓或客户回款不畅，可能面临流动性压力。

---

## 8. 总结判断

**一句话结论**

2025年豪恩汽电收入+26%至17.88亿但利润-25%至7495万，"增收不增利"因毛利率下降+新业务投入，机器人和eVTOL布局打开第二曲线想象空间。

**核心看点**

1. 收入+26%，智能驾驶感知系统市占率提升
2. 机器人感知产品量产+智元合作，2070TOPS域控系统
3. eVTOL已获定点，2026年规模化交付
4. 研发投入12.21%，技术壁垒持续加深

**核心风险**

1. 毛利率连续3年下降，净利率已降至4.2%
2. 新业务投入回报周期长，短期压制利润
3. 现金流偏弱，资产负债率上升
