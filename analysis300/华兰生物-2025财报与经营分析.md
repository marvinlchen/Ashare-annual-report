# 华兰生物 2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

- 全年营业收入 **45.95 亿元**，同比 **+4.93%**；
- 归母净利润 **9.40 亿元**，同比 **-13.55%**；扣非归母净利润 **9.37 亿元**，同比 **-4.51%**（降幅显著小于归母）；
- 基本每股收益 **0.52 元**；经营现金流 **8.78 亿元**（-17.12%）。

### 1.2 增长与拖累因素

- 主要增长驱动：① **血液制品业务 33.87 亿元（+4.35%）**，占比 73.71%，依然稳健；② **白蛋白收入 14.18 亿元（+15.40%）**，量价稳步增长；③ **疫苗业务 11.95 亿元（+7.00%）**；④ 流感疫苗 11.21 亿元（+4.43%）保持领先地位；⑤ **分红 18.27 亿元（每 10 股派 5 元）**，股利支付率超 97%，股东回报突出。
- 主要拖累因素：① **静注丙球收入 8.45 亿元（-8.94%）**，景气下行拖累；② 归母净利润 -13.55% 明显大于营收 +4.93%，说明**成本端与费用端压力加大**（研发投入增加、生物药投入等）；③ 经营现金流 -17.12%（回款节奏偏慢）；④ **筹资活动现金流大降 -155.89%**（分红等资金流出大）；⑤ 扣非 -4.51% < 归母 -13.55%，说明约 0.09 亿元的非经常性项目微扰动，但整体下滑仍主要在经营层面。
- 整体呈现"**血制品稳健增长 + 疫苗微增 + 静丙承压 + 生物药培育期**"的血液制品龙头典型特征。

### 1.3 趋势判断

- 未来主要利好方向：浆站扩容与采浆量增长（血制品产能利用率提升）+ 流感疫苗国际化拓展 + 生物药（华兰生物工程）管线兑现 + 高股利持续回报股东；
- 未来主要压力方向：静丙价格与需求下行 + 血制品"量增价稳"天花板 + 生物药研发投入侵蚀利润 + 疫苗集采与批签发周期波动 + 海外竞争加剧；
- 整体趋势判断：华兰生物 2025 年正处于"**血制品+疫苗双主业稳定 + 生物药第三曲线培育**"的转型期。短期利润承压（生物药投入）但长期看生物药（华兰基因工程）为公司打开第二增长空间；高分红（97% 股利支付率）凸显价值属性。

## 2. 主营业务

### 2.1 公司简介

华兰生物（002007.SZ）是国内血液制品行业龙头之一，核心业务为血液制品（白蛋白、静丙、凝血因子等）和疫苗（以流感疫苗为核心），通过子公司华兰生物工程（华兰疫苗）和华兰基因工程开展疫苗与生物药（重组蛋白、单抗等）业务。

### 2.2 主营业务构成

1. **血液制品**（核心业绩底盘）：2025 年收入 **33.87 亿元（+4.35%）**，占比 **73.71%**；包括人血白蛋白、静注丙球、凝血因子、特殊免疫球蛋白等；公司拥有多家浆站，年采浆量处于行业前列。
2. **疫苗制品**（成长稳定引擎）：2025 年收入 **11.95 亿元（+7.00%）**，占比 26.01%；主打流感疫苗（11.21 亿元，+4.43%），也有甲肝、乙肝等其他疫苗；子公司华兰疫苗是国内流感疫苗龙头之一。
3. **生物药**（第三增长曲线）：子公司**华兰基因工程**布局单抗、重组蛋白等生物创新药，管线覆盖肿瘤、自免、代谢等领域；短期仍在研发投入期。
4. **其他业务**：2025 年 0.13 亿元（微量）。

### 2.3 核心收入来源

- **血液制品是绝对核心**（收入占比 73.71%）：
  - 白蛋白 14.18 亿元（+15.40%）——增长最快的血制品大单品；
  - 静丙 8.45 亿元（-8.94%）——景气下行的拖累；
  - 其他血液制品 11.23 亿元（+3.20%）；
- **疫苗业务**（26.01%）以流感疫苗为主（流感占疫苗收入 93.8%）；
- **华兰疫苗**（子公司）是国内流感疫苗龙头（与赛诺菲巴斯德、科兴并列三大主流厂家）；
- **华兰基因工程**是未来中长期增长引擎，管线价值尚未充分释放。

## 6. 未来可能的利好

### 利好 1：血液制品业务稳健增长 + 白蛋白量价齐升

**依据**

2025 年血液制品收入 33.87 亿元（+4.35%）；白蛋白 14.18 亿元（+15.40%）量价齐升；公司是国内血制品行业龙头（与天坛生物、上海莱士并列三大主流厂家）。

**解读**

- 血液制品是医药行业中**供给端受限、需求端稳定**的稀缺资源品类；
- 国家对新浆站审批严格，存量浆站牌照具有稀缺性与政策壁垒；
- 公司拥有多家浆站，年采浆量处于行业前列；
- 白蛋白用于肝病、肾病、营养补给等多个场景，长期需求稳定；
- 国内白蛋白仍有约 50% 靠进口，未来国产替代空间大；
- 预计 2026-2028 年血制品业务保持 5-10% 稳定增长。

### 利好 2：生物药（华兰基因工程）管线进入兑现期

**依据**

华兰基因工程布局单抗、重组蛋白等生物创新药管线（肿瘤、自免、代谢等领域）；2025 年加大研发投入。

**解读**

- 生物药是未来 5-10 年中国医药行业最重要的增长赛道；
- 华兰基因工程有多款生物类似药和创新药处于临床阶段；
- 若管线获批上市，将显著打开公司估值空间（类似信达、百济神州路径）；
- 生物药毛利率普遍在 80-90%，远高于血制品和传统疫苗；
- 这是公司从"**传统血制品 + 疫苗**"向"**生物创新药龙头**"跨越的关键；
- 2026-2028 年是华兰基因工程管线逐步兑现的关键窗口。

### 利好 3：高股利支付率 + 极低资产负债率的价值属性

**依据**

2025 年派息 18.27 亿元（每 10 股派 5 元），**股利支付率 97.2%**；资产负债率约 24.6%（行业最低水平）；基本无有息负债；经营现金流 8.78 亿元（虽然下降但仍正向）。

**解读**

- 股利支付率接近 100%，显示公司对现金流稳定性充满信心；
- 低杠杆 + 稳定分红 + 血制品稀缺资源属性 = "**医药行业中的公用事业**"；
- 在利率下行环境中，高股息（以 2025 年 EPS 0.52 元、派息 0.5 元计算）成为稀缺配置；
- 稳定分红可吸引长线资金（社保、保险、外资等）；
- 股东回报质量显著高于 A 股医药行业平均水平；
- 若估值持续合理，股价下行空间相对可控。

## 7. 未来可能的利空

### 利空 1：静丙业务持续承压 + 血制品"量增价稳"天花板

**依据**

2025 年静丙收入 8.45 亿元（-8.94%）；血液制品整体增速仅 +4.35%；净利率从 24.82% 降至 20.46%（-4.36pct）。

**解读**

- 静丙用于免疫缺陷、自免疾病等，医院端使用量结构性调整；
- 静丙价格长期承压，对公司毛利率形成拖累；
- 血液制品整体受采浆量限制（新浆站审批慢），难以高增长；
- 白蛋白虽然增长快，但毛利率低于静丙（拖累综合毛利率）；
- 2026 年若静丙继续下行，血制品整体增长可能进一步放缓至 3-5%。

### 利空 2：生物药研发投入大 + 商业化不确定性

**依据**

华兰基因工程处于研发投入期，对利润形成阶段性侵蚀；归母净利润 -13.55% 显著大于扣非 -4.51%，显示研发等费用扩张压力。

**解读**

- 生物药研发周期长（通常 8-10 年）、投入大（单个品种数亿元）；
- 临床试验失败、审批延迟、商业化不及预期等风险显著；
- 国内生物药竞争激烈（恒瑞、信达、百济、康方等强敌林立）；
- 华兰基因工程的管线进展慢于信达、百济等头部企业；
- 若研发转化不及预期，前期投入可能成为长期利润压力；
- 需要持续跟踪华兰基因工程的临床数据与 NDA/BLA 进展。

### 利空 3：疫苗集采/批签发 + 海外竞争风险

**依据**

流感疫苗收入 11.21 亿元（+4.43%）增速放缓；国内流感疫苗行业竞争激烈（赛诺菲巴斯德、科兴、国药等多厂商并存）；医保/集采政策变化。

**解读**

- 流感疫苗属于"**非免疫规划疫苗**"（自费疫苗），需求与居民支付能力、健康意识强相关；
- 经济下行可能抑制自费疫苗接种率；
- 若流感疫苗纳入医保或集采，价格将面临大幅压力；
- 海外厂商（赛诺菲巴斯德）产品线更丰富（鼻喷、四价等），竞争持续；
- 公司流感疫苗海外拓展仍在早期阶段，面临认证、渠道、本地化等多重挑战；
- 疫苗批签发周期性波动（夏秋为旺季）使业绩季节性明显。

## 8. 总结判断

**一句话结论**

2025 年华兰生物呈现"**营收 +4.93% + 归母净利 -13.55% + 扣非净利 -4.51% + 血制品 +4.35% + 疫苗 +7.00% + 分红 18.27 亿（97.2% 股利支付率）**"的血液制品龙头典型特征，是**血制品+疫苗双主业稳定 + 生物药第三曲线培育期 + 高股利价值属性突出**的医药核心资产。

**核心看点**

1. **血液制品业务稳健增长** + **白蛋白 +15.40%** 量价齐升 + 浆站资源壁垒；
2. **生物药（华兰基因工程）**管线进入兑现期，未来 3-5 年估值重塑关键；
3. **股利支付率 97.2%** + 资产负债率约 24.6%（行业最低）+ 无有息负债，价值属性突出。

**核心风险**

1. 静丙业务持续承压（-8.94%）+ 血制品"量增价稳"天花板，净利率下行 -4.36pct；
2. 生物药研发投入侵蚀利润 + 商业化不确定性 + 管线进度慢于头部创新药企；
3. 疫苗集采/批签发 + 海外竞争风险 + 经济下行对自费疫苗的抑制。
