## 1. 业绩总结

### 1.1 收入与利润表现

- **2025 年全年营业收入（业绩预告）**：**429.68-466.71 亿元**，同比 **+16%~+26%**（2024 年 370.40 亿元）
- **2025 年全年营业利润**：年报未披露具体金额（利润增速与收入增速基本同步）
- **2025 年全年归母净利润（业绩预告）**：**49.71-54.00 亿元**，同比 **+16%~+26%**（2024 年 42.85 亿元）
- **扣非归母净利润（业绩预告）**：**47.62-51.66 亿元**，同比 **+18%~+28%**（扣非增速略高，核心经营质量更好）

### 1.2 增长与拖累因素

- **主要增长驱动**（业绩预告 + Q3 数据）：①**新能源汽车及轨交业务** Q1-Q3 收入 148 亿元（**+38%**）、联合动力 145 亿（+36%），规模放量 + 海外出海；②**通用自动化（含工业机器人）** Q1-Q3 收入 131 亿元（**+20%**），工控业务恢复；③**数字化 + 能源管理 + 机器人**新业务在培育第二增长曲线
- **主要拖累因素**：①**新能源车毛利率承压** Q3 单季 14.5%（**-2.9pct**），客户结构调整 + 价格战影响；②**整体毛利率** Q1-Q3 29.27%（**-1.75pct**）、Q3 单季 27.51%（-2.18pct）；③**Q3 单季归母 +4.04%**（明显放缓，利润端压力显现）；④**智慧电梯业务**受地产拖累，Q1-Q3 36 亿（同比基本持平）
- **增长 vs 承压的核心矛盾**：收入高增长（+24.67%）但毛利率下降（-1.75pct），量增价减

### 1.3 趋势判断

- **未来主要利好**：①**新能源汽车全球化**（联合动力出海）；②**工业自动化恢复 + 工业机器人放量**；③**数字化 / 能源管理 / AI+工控**第二增长曲线；④**扣非净利润增速（+18~28%）高于归母（+16~26%）**显示核心经营质量在改善
- **未来主要压力**：①新能源车价格战 + 客户集中度调整继续压制毛利；②地产下行对电梯业务的持续拖累；③通用自动化增速回归常态（+20%）后能否持续加速
- **整体判断**：**"高成长 + 毛利率阶段性承压 + 结构转型"**，新能源汽车规模效应 + 工控业务恢复 + 机器人 / 数字化新业务共同支撑长期增长（年报预计 2026-04 下旬披露，届时更新）

## 2. 主营业务

### 2.1 公司简介

汇川技术（300124）是中国工业自动化与新能源汽车电控领域的龙头企业，专注于**工业自动化控制（通用变频器、伺服、PLC&HMI、工业机器人）、新能源汽车电控系统（电机电控 + 联合动力）、智慧电梯控制系统、轨道交通牵引系统**。产品线覆盖"电机 + 电控 + 传感 + 算法"的完整闭环，是国内工控领域少数具有全球竞争力的平台型企业。

### 2.2 主营业务构成

1. **通用自动化（含工业机器人）**（Q1-Q3 占 41.4%，131 亿元 / +20%）
   - **通用变频器**：Q1-Q3 40.5 亿元
   - **通用伺服系统**：Q1-Q3 **53.4 亿元**（最大品类）
   - **PLC & HMI**：Q1-Q3 12.9 亿元
   - **工业机器人**：Q1-Q3 8.7 亿元（培育中）
2. **新能源汽车和轨道交通**（Q1-Q3 占 46.7%，148 亿元 / **+38%**）
   - **新能源汽车电控系统**（联合动力为主体）：Q1-Q3 145 亿元（+36%）
   - 轨道交通牵引系统
3. **智慧电梯**（Q1-Q3 占 ~11.4%，36 亿元 / 持平）
   - 电梯一体化 / 专用变频器 / 物联网解决方案
   - 2025 全年预计 ~50 亿元
4. **其他新业务**
   - 数字化与能源管理
   - AI + 工业控制平台
   - 海外 / 出海业务

### 2.3 核心收入来源

**新能源汽车 + 通用自动化**（合计约 88%）是两大核心引擎。新能源汽车是增长第一引擎（+38%），通用自动化是利润质量和基本盘。智慧电梯受地产周期拖累，但海外 + 旧梯更新提供缓冲。

## 6. 未来可能的利好

### 利好 1：新能源汽车业务规模化 + 全球化

**依据**

- 2025Q1-Q3 新能源汽车业务收入 **148 亿元（+38%）**，联合动力 145 亿元（+36%）
- 2024 年全年该业务约 150 亿元，2025 年预计 200 亿左右

**解读**

联合动力是国内领先的新能源汽车电控平台，客户覆盖主流车企。全球化布局（出海）提供了**第二增长曲线**。未来随着规模扩大 + 产能利用率提升 + 海外高毛利订单，新能源车毛利率有望从底部恢复。

### 利好 2：工业自动化恢复 + 工业机器人放量

**依据**

- 2025Q1-Q3 通用自动化收入 **131 亿元（+20%）**
- **工业机器人** Q1-Q3 8.7 亿元（培育期）
- 通用伺服 53.4 亿元、通用变频器 40.5 亿元、PLC&HMI 12.9 亿元 —— 全线品类恢复增长

**解读**

工业自动化是公司**利润质量最好的核心底盘**。工控下游恢复 + 汇川伺服 / 变频器持续提升国产替代份额 + 工业机器人逐步放量，将共同构成长期利润基石。工业机器人是未来 3-5 年的重要看点。

### 利好 3：扣非净利润增速高于归母 —— 核心经营质量在提升

**依据**

- 业绩预告扣非净利润增速 **+18%~+28%**，高于归母 +16%~+26%
- Q3 累计归母 +26.84%（Q1-Q3），表明前三季度核心经营非常健康

**解读**

扣非 > 归母意味着**非经常性损益项减少**，公司业绩主要来自主营业务。长期来看，这种质量提升比"帐面增长"更有价值，估值中枢有望逐步抬升。

### 利好 4：数字化 + 能源管理 + AI + 工控第二增长曲线

**依据**

- 公司在业绩预告中明确"数字化、能源管理、机器人"等新业务在培育
- "AI + 工业控制平台"是长期战略方向

**解读**

未来 3-5 年，如果数字化 / 能源管理 / AI + 工控等新业务从培育期进入放量期，将形成**第二增长曲线**，为公司打开新的成长空间。

## 7. 未来可能的利空

### 利空 1：新能源汽车价格战 + 客户结构集中压制毛利

**依据**

- 2025Q3 单季新能源车毛利率 **14.5%（-2.9pct）**，大幅下滑
- 公告明确"主要因产品客户结构调整"
- 新能源车行业价格战持续

**解读**

新能源车下游价格战压力会**持续向上游电控零部件传导**。同时大客户集中（比亚迪 / 理想 / 小鹏等）会压缩议价能力。若 2026 年价格战继续加剧，新能源车毛利率可能进一步探底至 13% 附近。

### 利空 2：智慧电梯业务持续受地产周期拖累

**依据**

- Q1-Q3 电梯业务 36 亿元同比基本持平
- 全年预计 50 亿左右

**解读**

中国地产下行周期尚未结束，新梯销量持续疲软。虽然**旧梯更新 + 海外市场**提供缓冲，但整体电梯业务难以重回高增长。占公司收入 ~11% 的业务可能在未来 2-3 年继续拖累综合增速。

### 利空 3：Q3 单季归母增速骤减至 +4.04%

**依据**

- 2025Q3 单季归母净利润 12.86 亿元，**同比仅 +4.04%**（Q1-Q3 累计 +26.84%）
- Q3 毛利率 27.51%（-2.18pct）

**解读**

Q3 单季利润增速断崖式下滑，如果趋势延续到 Q4，全年利润可能**靠近业绩预告区间下限（+16%）**。如果 Q4 没有明显反弹，2026 年利润率压力会持续。

### 利空 4：通用自动化增速回归常态后能否持续加速

**依据**

- Q1-Q3 通用自动化 +20%，已经是从 2023-2024 年低基数上的恢复性增长
- 下游制造业景气度能否持续是关键变量

**解读**

如果 2026 年制造业景气度回落（如 PMI 重新跌破 50），通用自动化增速可能从 +20% 回落至 +10% 附近，拖累公司整体增速。

## 8. 总结判断

**一句话结论**

**"高成长 + 毛利率阶段性承压 + 结构转型"**：汇川技术 2025 年业绩预告显示收入 +16~26%（429-467 亿）、归母 +16~26%（50-54 亿）、扣非 +18~28%，核心经营质量提升；新能源汽车 +38% 是增长第一引擎，通用自动化 +20% 恢复增长，但新能源车毛利率 Q3 单季 -2.9pct + Q3 归母仅 +4.04% 显示利润端压力加大。2026 年需关注新能源车毛利率能否企稳 + 工控业务能否持续加速 + 机器人 / 数字化新业务放量。

**核心看点**

1. **新能源汽车全球化** + 联合动力 2026 年规模效应是否修复毛利率
2. **通用自动化 + 工业机器人** 能否持续高增长 + 机器人放量节奏
3. **扣非 > 归母增速** 延续性 + 新业务（数字化 / 能源管理 / AI + 工控）兑现

**核心风险**

1. **新能源车价格战 + 客户结构调整**持续压制毛利（Q3 毛利率 -2.9pct）
2. **Q3 单季归母 +4.04% 断崖式下滑**若延续至 Q4 则全年靠下限
3. **地产周期持续拖累电梯业务**
4. **制造业景气度回落**压制通用自动化增速
