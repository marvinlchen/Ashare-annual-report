## 1. 业绩总结

### 1.1 收入与利润表现

- **2025 年前三季度营业收入**：174.84 亿元，同比 +7.25%（全年数据年报未披露）
- **2025 年前三季度归母净利润**：31.15 亿元，同比 **-9.76%**
- **2025 年前三季度扣非归母净利润**：31.19 亿元，同比 +0.20%（核心经营基本稳住）
- Q3 单季收入 59.77 亿元（+3.83%），归母 10.64 亿元（**-24.12%**）、扣非 10.80 亿元（-18.70%）

### 1.2 增长与拖累因素

- **主要增长驱动**：①**屈光项目 +11.14%**（H1 数据，全光塑 + 全飞秒 4.0 等高端术式推广 + 平均单价提升）；②**视光服务 +14.73%**（H1，近视防控品牌 + "5P 近视防控模式"）；③门诊量 +16.47%、手术量 +7.63%（H1 数据）；④**全球医疗网络覆盖**：内地 811 家 + 欧洲 137 家 + 东南亚 17 家 + 港 8 家 + 美国 1 家
- **主要拖累因素**：①**毛利率 49.27%（-1.74pct）**，Q3 单季毛利率 50.65%（-3.26pct）；②**Q3 归母 -24.12%**（较 Q2/Q1 明显恶化，警示信号）；③**归母与扣非背离**：归母 -9.76% vs 扣非 +0.20%，非经常损益影响大；④白内障业务受医保支付改革 + 高值耗材集采影响
- **趋势切换**：从"收入增长、利润率稳定"→"收入增长、利润率承压"阶段

### 1.3 趋势判断

- **未来主要利好**：近视防控 + 老龄化 + 消费升级长期驱动；全光塑 / 全飞秒 4.0 高端术式 + AI 眼科 + 数字化转型；全球网络规模护城河
- **未来主要压力**：屈光 / 视光毛利率边际回落；医保支付改革 + 高值耗材集采；Q3 利润大幅下滑是否持续；可选医疗消费敏感度
- **整体判断**：**"长期逻辑未坏、短期盈利走弱"**，业务量仍在扩张但利润率压力需要继续观察 2025 全年结果（2026-04-24 年报披露后更新）

## 2. 主营业务

### 2.1 公司简介

爱尔眼科（300015）是全球最大的眼科医疗连锁集团之一，主营眼科疾病诊疗、手术服务及医学验光配镜。截至 2024 年末，医疗网络覆盖中国内地 811 家、欧洲 137 家、东南亚 17 家、中国香港 8 家、美国 1 家。

### 2.2 主营业务构成

1. **屈光项目**（Q1-Q3 占 **40.13%**）：近视、远视、散光矫正手术，含全光塑、全飞秒 4.0、全飞秒 Pro 等高端术式，**H1 毛利率 56.12%**
2. **视光服务项目**（Q1-Q3 占 23.64%）：青少年近视防控、医学验光配镜、"5P 近视防控模式"，**H1 毛利率 54.21%**
3. **白内障项目**（Q1-Q3 占 15.48%）：白内障手术 + 高端晶体（多焦点、飞秒白内障），H1 毛利率 36.34%
4. **眼前段项目**（Q1-Q3 占 8.96%）：角膜、虹膜、晶状体疾病，H1 毛利率 39.62%
5. **眼后段项目**（Q1-Q3 占 6.82%）：视网膜、玻璃体、黄斑、糖尿病眼病，H1 毛利率 30.09%
6. **其他项目**（Q1-Q3 占 4.67%）：斜弱视、眼表、青光眼等

### 2.3 核心收入来源

**屈光 + 视光**（合计 ~63.77%）是公司收入和利润的"双引擎"，两者 H1 毛利率均 >54%，是利润质量的核心。白内障 + 眼前段 + 眼后段是面向老龄化、综合眼病的稳健收入来源。

## 6. 未来可能的利好

### 利好 1：眼科需求长期景气 + 近视防控 + 老龄化红利

**依据**

- 中国近视人口 7 亿+，近视率高企，青少年近视防控需求旺盛
- 老龄化加速，白内障、黄斑病变等老年眼病需求持续扩大
- 消费升级 + 眼健康意识提升，屈光手术渗透率持续提升

**解读**

爱尔眼科所处的眼科医疗赛道是**需求长期确定性最高的医疗细分之一**，近视防控 + 老龄化 + 消费升级三大红利叠加，长期需求空间广阔。

### 利好 2：屈光 + 视光双引擎 + 高端术式放量

**依据**

- 屈光项目 **+11.14%**，毛利率 56.12%（H1）
- 视光服务 **+14.73%**，毛利率 54.21%（H1）
- **全光塑、全飞秒 4.0、全飞秒 Pro、飞秒白内障**等高端术式持续推广，带动平均单价提升
- "5P 近视防控模式"品牌影响力扩大

**解读**

屈光 + 视光合计占收入 63.77%，是爱尔眼科的"黄金双引擎"，高毛利率 + 高增长为公司提供长期护城河。高端术式 + 品牌溢价能力是对抗价格竞争的关键武器。

### 利好 3：全球最大眼科连锁网络 + AI + 数字化

**依据**

- 医疗网络覆盖全球 974 家（内地 811 + 欧洲 137 + 东南亚 17 + 香港 8 + 美国 1）
- **"AI + 眼科"** 应用布局加快
- 数智化转型推进（排班、诊疗、患者管理、质量控制）

**解读**

全球规模网络构成品牌、人才、学科、科研和管理体系的多重壁垒，新进入者难以复制。AI + 数字化有望提升效率、降低成本、改善患者体验，是未来利润率恢复的重要抓手。

### 利好 4：业务量仍在稳健扩张

**依据**

- 2025H1 门诊量 **924.83 万人次（+16.47%）**
- 2025H1 手术量 **87.90 万例（+7.63%）**

**解读**

业务量（尤其是门诊量）仍保持两位数高增长，说明终端需求和品牌力并未削弱。"价格下行但量增长"提供了护城河韧性，中长期利润率随规模效应有望恢复。

## 7. 未来可能的利空

### 利空 1：2025 年利润端显著承压

**依据**

- 2025 年前三季度归母净利润 **-9.76%**
- 2025Q3 单季归母 **-24.12%**
- 2025 年前三季度毛利率 49.27%（-1.74pct）

**解读**

Q3 单季归母 -24.12% 是明显的警示信号，若趋势持续，2025 全年利润可能大幅下滑。费用端 + 政策端 + 价格端多重压力叠加。2026 年净利率能否稳住在 18%+ 是关键观察点。

### 利空 2：医保支付改革 + 高值耗材集采

**依据**

白内障业务增长在医保支付方式改革和高值耗材集采背景下仅 +2.64%，远低于屈光 / 视光增速。

**解读**

医保支付方式改革（DRG / DIP）+ 高值晶体集采持续推进，压缩白内障 / 眼前段 / 眼后段等医保覆盖业务的利润空间。若集采进一步扩围，将对公司非消费医疗板块利润率产生结构性压力。

### 利空 3：商誉减值风险

**依据**

公司长期通过"医院合伙人 + 并购连锁"扩张，历史商誉规模约 80-90 亿元级别（2024 年末）。

**解读**

在医保集采 + 利润下行压力下，部分并购医院可能出现经营不及预期，引发商誉减值计提。商誉减值若集中在 2025 年度报告计提，将对归母净利润造成一次性冲击（需密切关注 2026-04-24 年报结果）。

### 利空 4：屈光 / 视光高毛利受价格 + 竞争压制

**依据**

屈光 + 视光毛利率虽仍高，但 2025H1 均较上年同期有所下降。私立眼科、公立医院私立化、OK 镜等近视防控赛道竞争持续加剧。

**解读**

爱尔眼科面临来自华厦眼科、何氏眼科、普瑞眼科等眼科连锁 + 公立医院的双重竞争，价格体系持续承压，高毛利护城河可能被逐步侵蚀。

### 利空 5：可选医疗消费对经济环境敏感

**依据**

屈光手术、部分高端视光（蔡司/尼德克 / OK 镜等）、高端白内障晶体本质为**可选医疗消费**。

**解读**

若宏观消费意愿转弱，客单价较高的项目（全光塑、全飞秒 Pro 等）可能阶段性回落，对收入和毛利率形成双重压力。

## 8. 总结判断

**一句话结论**

**"长期逻辑未坏、短期盈利走弱"**：爱尔眼科 2025 年报**尚未披露（2026-04-24 披露）**，从最新可得前三季度数据看，业务量仍稳健增长（门诊量 H1 +16.47%、手术量 +7.63%），但利润率出现明显承压（毛利率 -1.74pct、Q3 归母 -24.12%），需要重点关注 2025 全年结果和商誉减值情况。

**核心看点**

1. **屈光 + 视光双引擎**：毛利率 >54%，全光塑 / 全飞秒 4.0 高端术式放量
2. **全球 974 家连锁网络**：全球最大眼科连锁，品牌 + 人才 + 学科护城河
3. **AI + 眼科 + 数字化**：效率提升 + 成本优化新抓手

**核心风险**

1. 2025 年利润端显著承压（Q3 -24%，商誉减值风险）
2. 医保支付改革 + 高值耗材集采对白内障 / 眼底业务持续压制
3. 屈光 / 视光毛利率边际回落 + 私立眼科连锁 + 公立竞争加剧
