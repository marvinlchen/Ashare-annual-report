# 拓普集团（601689）2025年财报整理

> **数据来源：** 拓普集团（601689）2025年年度报告（2026年3月23日发布），数据完整。营收+11%逼近300亿，但归母-7%增收不增利。汽车电子+52%最亮眼，人形机器人和液冷是新赛道。

## 1. 业绩总结

### 1.1 收入与利润表现

2025年拓普集团营业收入295.81亿元，同比 +11.21%，连续六年增长逼近300亿大关；但归母净利润27.79亿元，同比 -7.38%；扣非净利润26.11亿元，同比 -4.30%。增收不增利的原因是毛利率下降（-1.37ppt至19.43%）、海外新产能爬坡成本、研发投入增加（+22%）。Q4出现积极信号：营收+19%、扣非+13%。

- 全年营业收入295.81亿元，同比 +11.21%
- 全年归母净利润27.79亿元，同比 -7.38%
- 扣非净利润26.11亿元，同比 -4.30%

### 1.2 增长与拖累因素

汽车电子（+52%至27.69亿）是最大增长引擎。内饰功能件（96.72亿）和底盘系统（87.22亿）是收入基本盘。减震器（-3.3%）和热管理（-2.3%）小幅下降。利润下降的核心原因是：下游车企压价导致毛利率-1.37ppt，海外新工厂爬坡期固定成本高，研发投入+22%，管理费用+24%。

- 主要增长驱动：汽车电子+52%、底盘系统增长、海外扩张
- 主要拖累因素：毛利率-1.37ppt、新产能爬坡、研发+22%
- 增长或承压主要来自：战略投入期，以短期利润换长期增长

### 1.3 趋势判断

拓普集团是中国领先的汽车零部件平台型企业，横跨减震器、底盘、内饰、热管理、汽车电子五大领域，核心客户包括特斯拉、赛力斯、吉利等。2025年是战略投入年——人形机器人执行器（特斯拉Optimus供应商）、液冷技术（15亿首批订单）等新赛道布局加速。Q4改善信号明显。

- 未来主要利好方向：人形机器人、液冷、汽车电子放量、海外产能释放
- 未来主要压力方向：毛利率下降、车企降价压力、新业务盈利周期
- 整体趋势判断：战略投入期，Q4拐点初现

## 2. 主营业务

### 2.1 公司简介

宁波拓普集团股份有限公司（601689.SH）是中国领先的汽车零部件平台型企业，总部位于浙江宁波。以NVH减震系统起家，逐步拓展至底盘系统、内饰功能件、热管理系统、汽车电子五大业务领域。是特斯拉全球核心供应商之一，同时也是特斯拉Optimus人形机器人的核心零部件供应商。

### 2.2 主营业务构成

1. **内饰功能件**：收入96.72亿元，占比32.7%，最大业务线
2. **底盘系统**：收入87.22亿元，占比29.5%
3. **减震器**：收入42.56亿元（-3.33%），占比14.4%，传统优势领域
4. **汽车电子**：收入27.69亿元（+52.11%），占比9.4%，增长最快
5. **热管理系统**：收入20.91亿元（-2.26%），占比7.1%

### 2.3 新兴业务

- **人形机器人执行器**：特斯拉Optimus核心供应商，2025年营收1,359万元
- **液冷技术**：首批15亿元订单，应用于储能、AI算力服务器

## 4. 营收分布和增长

### 4.1 内饰功能件——最大业务线

**收入96.72亿元，占比32.7%**

内饰功能件包括仪表板骨架、门护板骨架、立柱护板等。作为最大的收入来源，为特斯拉Model 3/Y等车型供货。

### 4.2 底盘系统——第二大业务

**收入87.22亿元，占比29.5%**

底盘系统包括副车架、控制臂、转向节等关键零部件。这是拓普从NVH减震向平台型供应商拓展的核心成果。

### 4.3 汽车电子——增长最快，+52%

**收入27.69亿元（+52.11%），占比9.4%**

汽车电子是增速最快的业务，涵盖智能驾驶相关控制器等产品。+52%的高增速体现了电动化和智能化趋势下的需求爆发。虽然目前占比仅9.4%，但如果保持50%+增速，2-3年后将成为与减震器同等规模的业务线。

### 4.4 减震器和热管理——传统业务微降

**减震器42.56亿元（-3.3%），热管理20.91亿元（-2.3%）**

减震器是拓普的传统优势业务，小幅下降可能与行业竞争和产品价格压力有关。热管理-2.3%反映了新能源车热管理市场的竞争加剧。

## 5. 利润分布和增长

### 5.1 整体利润表现

| 指标 | 2025年 | 同比变化 | 备注 |
|---|---:|---:|---|
| 归母净利润 | 27.79亿元 | -7.38% | |
| 扣非净利润 | 26.11亿元 | -4.30% | |
| 净利率 | ~9.4% | -1.9ppt | |
| 毛利率 | 19.43% | -1.37ppt | |
| 经营现金流 | 44.82亿元 | +38.50% | |

### 5.2 利润分析

**增收不增利的四重原因**

1. **毛利率-1.37ppt至19.43%**：下游车企持续压价，特别是新能源车企的降本需求传导至零部件供应商。三年累计毛利率下降3.2ppt。
2. **海外新产能爬坡**：新建工厂处于产能爬升阶段，固定成本摊销高而产出尚未达标。
3. **研发投入+22%至14.96亿元**：空气悬架、线控制动、智能座舱、人形机器人、液冷等新赛道的研发投入大幅增加。
4. **管理费用+24%**：全球化布局导致管理体系扩张。

**Q4拐点信号**

Q4扣非+12.71%是积极信号——说明新产能逐步释放、成本控制见效。如果2026年Q1继续环比改善，则利润拐点得到确认。

**经营现金流+38.5%是最大亮点**

44.82亿元的经营现金流（+38.5%）远超归母净利润27.79亿元，说明公司的现金创造能力强于账面利润表现。

## 6. 未来可能的利好

### 利好 1：人形机器人——特斯拉Optimus核心供应商

**依据**

2025年人形机器人执行器营收1,359万元。拓普是特斯拉Optimus的核心零部件供应商。

**解读**

虽然1,359万元的营收微不足道，但作为特斯拉Optimus的核心供应商，一旦人形机器人进入量产阶段，拓普将是最直接的受益者。如果Optimus年产1万台，每台零部件价值假设5-10万元，仅此一项就能贡献5-10亿元收入。这是未来3-5年最大的想象空间。

### 利好 2：液冷技术——15亿首批订单

**依据**

液冷技术取得首批15亿元订单，应用于储能和AI算力服务器。

**解读**

液冷是AI算力时代的关键基础设施。15亿元订单仅是起步，全球AI数据中心液冷市场预计2030年达千亿级别。拓普的热管理和精密制造能力可以自然延伸到液冷领域。如果能在3年内做到年收入50亿+，将成为重要的第二增长曲线。

### 利好 3：汽车电子+52%——新增长引擎确立

**依据**

汽车电子收入27.69亿元（+52.11%），增速远超其他业务。

**解读**

汽车电子受益于电动化和智能化的结构性趋势。+52%的增速如果保持3年，到2028年可达100亿元级别（27.69→42→64→97亿），成为与底盘系统同等规模的业务。智能驾驶控制器的单车价值量高，利润率也有望高于传统零部件。

## 7. 未来可能的利空

### 利空 1：毛利率持续下降——三年-3.2ppt

**依据**

毛利率19.43%（-1.37ppt），三年累计下降3.2ppt。

**解读**

毛利率从2022年的约22.6%持续下降到19.4%。如果继续下降至18%以下，净利率将被压缩到8%以下。下游车企的降本需求是结构性趋势（特别是特斯拉的激进降价策略），零部件供应商很难完全抵消。只有通过产品结构升级（更多高附加值产品）或规模效应才能止住下滑。

### 利空 2：客户集中度高——特斯拉依赖

**依据**

特斯拉是核心客户之一（具体营收占比未披露）。

**解读**

如果特斯拉销量不及预期或转换供应商，将对拓普造成显著影响。特斯拉的降价策略直接传导为供应商的毛利压力。同时，特斯拉的供应链管理以严苛著称，供应商的议价能力有限。客户多元化（赛力斯、吉利等）是降低集中度风险的方向。

### 利空 3：海外产能爬坡期——短期拖累利润

**依据**

海外新建工厂处于爬坡阶段，固定成本高。

**解读**

全球化布局是正确的战略方向，但短期内新工厂的低产能利用率会拖累利润。从建成到满产通常需要1-2年。如果2026年仍未达到满产水平，利润压力将持续。

## 8. 总结判断

**一句话结论**

2025年拓普集团营收296亿元（+11%）六连增逼近300亿，但归母28亿元（-7%）增收不增利；汽车电子+52%是新引擎，人形机器人执行器（特斯拉Optimus）和液冷（15亿首批订单）是新赛道；Q4扣非+13%出现拐点信号；经营现金流45亿（+38%）显著改善。

**核心看点**

1. 营收296亿（+11%），六连增逼近300亿
2. 汽车电子+52%至27.69亿，增长最快
3. 特斯拉Optimus人形机器人核心供应商
4. 液冷技术首批15亿元订单（储能+AI算力）
5. 经营现金流45亿（+38%），远超净利润
6. Q4扣非+13%，拐点信号初现
7. 研发+22%至14.96亿，布局空悬/线控制动/智能座舱

**核心风险**

1. **归母-7.38%**，增收不增利
2. **毛利率-1.37ppt至19.43%**，三年累计-3.2ppt
3. 客户集中度高（特斯拉依赖）
4. 海外新产能爬坡拖累短期利润
5. 管理费用+24%，全球化管理成本上升
