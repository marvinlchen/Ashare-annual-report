# 上海机场 2025年财报整理

> **说明**：上海机场（600009.SH）**2025 年年度报告尚未披露**（预计 2026 年 4 月下旬披露，**业绩预告/快报亦未发布**）。本文基于 **2025 年三季报（2025-10-31 披露）+ 2025 年半年报（2025-08-30 披露）+ 2025 年一季报 + 券商研报** 整理，年报未披露的字段统一标注"年报未披露"。

## 1. 业绩总结

### 1.1 收入与利润表现

上海机场 2025 年 Q1-Q3 实现营业收入 97.1 亿元，**同比 +5.7%**；归母净利润 16.3 亿元，**同比 +36%**。Q3 单季营收 33.6 亿元（**+7.5%**），归母净利润 **5.9 亿元（+53%）**——**盈利弹性显著加速**。H1 盈利 10.4 亿（+28%），利润增速 > 收入增速 > 旅客吞吐量增速，反映**国际航线占比提升 + 免税收入改善 + 成本管控**三重共振。H1 中期现金分红比例 **50.05%**。

- Q1-Q3 营业收入 97.1 亿元（同比 +5.7%）
- Q1-Q3 营业利润 年报未披露（Q1-Q3 合计利润总额估 ~22 亿）
- Q1-Q3 归母净利润 16.3 亿元（同比 +36%）

### 1.2 增长与拖累因素

- **主要增长驱动**：①**Q3 国际航线旅客吞吐量 +15.1%**（浦东 +17.4% / 虹桥 +17%）—— 国际航线高毛利恢复核心；②**Q3 免税合同收入 3.14 亿（+17.6%）**；③H1 航空性业务收入 29.2 亿（+8.35%）；④成本管控到位，利润增速远高于收入增速（Q1-Q3 +36% vs +5.7%）；⑤Q3 浦东+虹桥国内旅客吞吐 2,419.49 万人次（+2.8%）
- **主要拖累因素**：①H1 免税收入 6.28 亿（**-3.1%**）+ 广告收入 2.97 亿（**-8.0%**）+ 商业餐饮 10.9 亿（-1.9%）—— 传统非航业务 H1 承压；②H1 非航业务收入 34.4 亿（仅 +1.93%）；③Q3 广告合同收入 1.7 亿（-2%）；④国际线恢复程度仍低于 2019 年 + 国人"免税消费偏好"尚未全面回暖
- **增长主要来自**：国际航线恢复 + 免税收入 Q3 转正 + 成本管控；**承压主要来自**：H1 非航业务 + 广告收入 + 免税消费恢复慢

### 1.3 趋势判断

- **未来主要利好方向**：①国际航线持续修复（Q3 +15.1% 已超 17% 拐点）+ 2026 年免税消费回暖预期；②新一轮免税补充协议（与日上免税行）巩固口岸免税龙头地位；③Q3 盈利弹性加速 + 2026 年业绩高增可期
- **未来主要压力方向**：①免税租金定价权弱化（新协议下保底租金大幅下调）；②行业竞争（广州/北京/深圳等机场分流国际航线）；③国际消费力复苏节奏 + 国人境外免税替代（海南/市内免税）
- **整体趋势判断**：**2026 年是上海机场业绩弹性兑现关键期**。Q3 单季归母 +53% 信号强劲，若 Q4 和 2026 国际线延续 +15% 以上增速 + 免税合同收入 +18% 趋势，2026 年归母有望冲击 30-35 亿规模

## 2. 主营业务

### 2.1 公司简介

上海机场是**中国最大的国际航空枢纽之一**，运营管理**浦东国际机场 + 虹桥国际机场**，是**全国唯一一家"一市两场"的上市机场**。公司主营航空性业务（起降费、停场费、客桥费、旅客服务费等）+ 非航空性业务（免税商业、广告、物流、餐饮、零售等）。

### 2.2 主营业务构成

1. **航空性业务**：起降费、停场费、客桥费、旅客服务费、安检费等，与旅客吞吐量和飞机起降架次强相关；**2025 H1 收入 29.2 亿（+8.35%）**
2. **非航空性业务**：
   - **免税业务**：与日上免税行合作的免税品销售提成（保底租金 + 流水抽成），历史上是上海机场**核心利润来源**；**2025 H1 免税收入 6.28 亿（-3.1%）**；**Q3 免税合同收入 3.14 亿（+17.6%）**
   - **广告业务**：机场灯箱、媒体广告等；**2025 H1 2.97 亿（-8.0%）**
   - **商业餐饮**：机场零售、餐饮租金分成；**2025 H1 10.9 亿（-1.9%）**
   - **物流服务**：货运地面服务；**2025 H1 8.3 亿（+0.76%）**
   - **其他非航**：办公楼租赁等；**2025 H1 15.1 亿（+2.3%）**

### 2.3 核心收入来源

**航空性收入**（~46%，随国际航线修复持续增长）+ **免税收入 / 商业租金**（历史上是利润核心，受中国消费偏好影响大）+ **物流 / 广告 / 餐饮**等稳健收入。**浦东机场是核心利润来源**，国际线复苏是最关键变量。

## 6. 未来可能的利好

### 利好 1：国际航线持续修复 + 2026 年免税消费回暖

**依据**

Q3 浦东+虹桥国际及地区航线旅客吞吐量 **1,099.13 万人次（+15.1%）**；**浦东国际 +17.4% / 虹桥国际 +17%**；Q3 免税合同收入 **3.14 亿（+17.6%）**；Q1-Q3 免税合同收入累计同比 +18%。

**解读**

国际航线修复是上海机场业绩弹性**最核心驱动**（国际线客流的人均消费力 + 免税消费 ≫ 国内线）。若 Q4 + 2026 年国际线延续 **+15%+** 增速 + 免税合同 **+18%+** 趋势，**2026 年归母有望冲击 30-35 亿规模**，对应利润同比 +50%+。

### 利好 2：Q3 单季归母 +53% 盈利弹性加速

**依据**

Q3 单季营收 33.6 亿（+7.5%）+ **归母 5.9 亿（+53%）**；Q1-Q3 归母 +36%（> H1 +28.14% > 2024 全年 +107% 延续）；**净利率 Q1-Q3 ~16.79%**（显著高于 2024 全年 ~15.64%）。

**解读**

**Q3 盈利弹性加速**反映成本管控 + 高毛利业务（国际航线 / 免税）结构性改善。经营杠杆显著（利润增速 5-10 倍于收入增速），若 Q4 + 2026 延续，业绩弹性极大。

### 利好 3：与日上免税行签订补充协议巩固口岸龙头

**依据**

公司与**日上免税行签订补充协议**巩固口岸免税市场领先地位；上海机场是中国最大的免税口岸之一（浦东机场免税销售占全国口岸免税 40-50%）。

**解读**

新协议虽然保底租金可能下调，但**流水抽成比例 + 销售额基数增长**有望带来长期稳定收益。若国际旅客持续回流 + 人均免税消费回升，免税收入将继续成为**利润核心引擎**。

### 利好 4：上海"一市两场"国际枢纽地位稳固

**依据**

公司运营**浦东国际机场 + 虹桥国际机场**（全国唯一"一市两场"）；2024 年两场年旅客吞吐量超 **1.24 亿人次**（历史新高）；年货邮吞吐量超 **420 万吨**（超 2019 水平）。

**解读**

**长三角 + 全国商务/金融中心**的区位优势 + 国际航线枢纽地位 + 进博会 / 世博等国际重大活动支撑 + 上海自贸港政策红利，**上海机场的中长期流量基本盘稳固**。

### 利好 5：中期分红率 50.05% 体现股东回报

**依据**

2025 年 **H1 中期现金分红比例 50.05%**；公司资产负债率偏低 + 经营现金流稳定。

**解读**

**50%+ 分红率**在机场行业中较为可观，体现了管理层对公司现金流和长期盈利能力的信心。对于注重股息回报的投资者具有较强吸引力。

## 7. 未来可能的利空

### 利空 1：免税合同新模式下租金定价权弱化

**依据**

2022 年公司与日上免税行重新签订免税协议，**保底租金大幅下调**（从原高保底+流水抽成变更为较低保底 + 销售额提成）。2025 年新补充协议延续该模式。H1 免税收入 -3.1%。

**解读**

新模式下上海机场的**租金收入与日上销售额深度绑定**，**不再享有高保底的旱涝保收**。若免税销售因国际客流恢复慢、国人境外 / 海南 / 市内免税替代等因素持续低于预期，**免税收入增速将受压制**。

### 利空 2：国人免税消费"替代效应"（海南/市内/跨境）

**依据**

海南离岛免税 + 市内免税店 + 跨境电商 + 海外奢侈品代购等**多渠道分流**口岸免税；国人对海外购物偏好下降。

**解读**

**口岸免税的核心竞争对手从"境外免税"转向"海南/市内免税"**。若 2026 年继续下调海南免税限额 / 市内免税试点扩大 + 其他口岸机场（首都 / 广州 / 深圳）加码免税，**上海机场免税恢复斜率将低于预期**。

### 利空 3：国际航线恢复节奏 + 航线分流风险

**依据**

Q3 国际航线 +15.1% 恢复强劲但**仍未完全达到 2019 年峰值**；广州 / 北京 / 深圳等机场加大国际航线投入；商务出行需求恢复节奏不均。

**解读**

若 2026 年全球经济 / 国际航空市场 / 签证政策 / 汇率等因素影响国际旅客恢复斜率，**上海机场业绩弹性可能打折扣**。同时国际航线在其他枢纽（广州 / 北京等）的分流也会削弱上海机场的"枢纽溢价"。

### 利空 4：H1 非航业务整体承压

**依据**

H1 非航业务收入 34.4 亿（仅 **+1.93%**），其中免税 -3.1% + 广告 -8.0% + 商业餐饮 -1.9%；Q3 广告仍 -2%。

**解读**

非航业务曾是上海机场的**核心利润来源**（~60% 利润）。若非航业务尤其是广告 / 商业餐饮持续低迷 + 免税反弹不及预期，**利润修复将不够有力**。广告 -8% 反映品牌广告投入仍然谨慎。

### 利空 5：资本开支与扩建计划

**依据**

机场基础设施需持续投入（跑道 / 航站楼扩建 / 数字化 / 新设备等）；2024 年总资产 700.46 亿。

**解读**

机场重资产属性导致**折旧 / 维修费用持续增长**。若新增产能爬坡慢 / 客流增速低于扩建进度，**折旧拖累**可能在特定年份压制利润。

## 8. 总结判断

**一句话结论**

上海机场 2025 Q1-Q3 呈现"营收 97.1 亿（+5.7%）+ 归母 16.3 亿（+36%）+ Q3 单季归母 5.9 亿（+53%）+ Q3 国际旅客 +15.1% + 浦东国际 +17.4% + 免税合同收入 +18% + 航空性 +8.35% + H1 非航 +1.93% + H1 中期分红率 50.05%"的"**国际线加速修复 + 免税拐点显现 + 盈利弹性加速 + 非航分化**"格局；2025 年年报尚未披露，2026 关键看**国际线延续 +15%+、免税合同 +18%+、非航业务整体回暖**。

**核心看点**

1. **Q3 国际旅客 +15.1% + 浦东国际 +17.4%**（国际线修复加速）
2. **Q3 免税合同收入 +17.6% + Q1-Q3 累计 +18%**（免税拐点显现）
3. **Q3 单季归母 +53% + Q1-Q3 归母 +36%**（盈利弹性加速）

**核心风险**

1. **免税合同新模式下租金定价权弱化**（保底租金大幅下调）+ 国人免税消费替代（海南/市内）
2. **国际航线恢复斜率 + 其他枢纽分流**（广州/北京/深圳加码国际）
3. **H1 非航业务 +1.93%**（广告 -8% + 商业 -1.9% 持续承压）
