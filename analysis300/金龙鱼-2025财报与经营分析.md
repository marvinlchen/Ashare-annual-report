# 金龙鱼 2025年财报整理

> **说明**：金龙鱼（300999.SZ）**2025 年年度报告已于 2026 年 4 月 18 日正式披露**；业绩快报于 2026-02-26 披露。本文基于**2025 年年度报告 + 业绩快报 + 券商研报**整理。

## 1. 业绩总结

### 1.1 收入与利润表现

金龙鱼 2025 年实现营业收入 2,451.26 亿元，**同比 +2.87%**；利润总额 48.89 亿元，**同比 +33.71%**；营业利润 52.38 亿元，同比 +44.35%；归母净利润 **31.53 亿元，同比 +26.01%**（含诉讼预计负债约 7.33 亿元）；**扣非归母净利润 28.45 亿元，同比暴增 +193.68%**——核心经营能力大幅修复。全年**产品销量 5,675.4 万吨（+6.06%）**。整体毛利率 **6.43%（+1.54pct）**，厨房食品毛利率 7.3%，**饲料原料及油脂科技毛利率 5%（同比翻倍）**。研发投入 2.88 亿元（+5.06%）。前三季度经营现金流 **275.97 亿元（+119.97%）**——现金流大幅改善。

- 营业收入 2,451.26 亿元（同比 +2.87%）
- 营业利润 52.38 亿元（同比 +44.35%）
- 归母净利润 31.53 亿元（同比 +26.01%）
- **连续 4 年利润下滑后首次实现正增长**

### 1.2 增长与拖累因素

- **主要增长驱动**：①**扣非净利润 +193.68%**（核心经营盈利大幅修复）；②**大豆压榨利润上涨**（南美大豆供应充足）；③**饲料原料及油脂科技毛利率翻倍至 5%**；④新工厂产能释放 + 销售渠道拓展；⑤面粉和大米业务原料价格下降改善盈利；⑥**全年销量 +6.06%（5,675.4 万吨）**
- **主要拖累因素**：①**子公司诉讼预计负债合计约 7.33 亿元**（广州益海 2.61 亿 + 东莞富之源 4.72 亿）—— 直接减少归母净利润；②收入增速仅 +2.87%（增长动能偏弱）；③粮油行业毛利率极低（6.43%），对原料成本高度敏感；④零售渠道消费不及预期 + 竞争加剧
- **增长主要来自**：压榨利润 + 销量增长 + 原料成本下降；**承压主要来自**：诉讼损失 + 消费疲软 + 毛利率结构低

### 1.3 趋势判断

- **未来主要利好方向**：大豆压榨利润持续 + 厨房食品渠道拓展 + 大健康 / 中央厨房等新业务布局 + 销量持续增长 + 毛利率结构优化
- **未来主要压力方向**：**诉讼二审不确定性**（广州益海 18.81 亿最大隐忧）+ 原料成本波动敏感（大豆 / 豆油占成本 90%）+ 消费疲软 + 零售竞争加剧
- **整体趋势判断**：**金龙鱼从 2024 年盈利低谷强势修复**（连续 4 年利润下滑后首次正增长），**扣非利润暴增 193.68% 反映核心经营能力恢复**。2026 年关键看压榨利润持续性 + 诉讼风险释放 + 新业务贡献增量

## 2. 主营业务

### 2.1 公司简介

金龙鱼（益海嘉里金龙鱼）是**中国最大的粮油食品企业之一**，主要从事厨房食品和饲料原料及油脂科技产品的研发、生产和销售，产品覆盖食用油（金龙鱼、胡姬花等品牌）、大米、面粉、调味品、饲料原料和油脂化学品等。**第一大股东：丰益营销（香港）有限公司，持股 89.99%**。

### 2.2 主营业务构成

1. **厨房食品（核心 C 端业务）**：食用油（金龙鱼、胡姬花等品牌）、大米、面粉及挂面、调味品等，面向 C 端消费者和 B 端餐饮；**2025 年收入 1,509.8 亿元，销量 2,518 万吨，毛利率 7.3%**
2. **饲料原料及油脂科技（核心 B 端业务）**：豆粕、麸皮、米糠粕等饲料原料；油脂基础化学品、油脂衍生化学品、营养品、日化用品等，面向 B 端客户；**2025 年收入约 924 亿元，销量 3,157 万吨，毛利率 5.0%（同比翻倍）**
3. **新兴业务**：大健康产业（DAG 油、植物甾醇等功能性产品）、中央厨房、新品类拓展

### 2.3 核心收入来源

**饲料原料及油脂科技**（约 924 亿，占 ~38%）是收入第二大板块，受益于大豆压榨利润上涨；**厨房食品**（1,509.8 亿，占 **~62%**）是品牌价值和消费者认知的核心，毛利率相对较高。公司依托**全球大豆、棕榈油**等大宗原料的采购和加工能力构建竞争壁垒。全年销量 5,675.4 万吨（+6.06%）。

## 6. 未来可能的利好

### 利好 1：核心经营盈利大幅修复 + 扣非暴增 193.68%

**依据**

2025 年**扣非净利润 28.45 亿元**（同比 **+193.68%**）；整体毛利率 6.43%（+1.54pct）；**前三季度经营现金流 275.97 亿（+119.97%）**；大豆压榨利润上涨 + 原料成本下降是主要驱动。

**解读**

扣非利润大幅改善说明**核心经营能力已走出 2024 年低谷**。若全球大豆供应持续充足、压榨利润维持在合理水平，**2026 年盈利有望继续改善**，这也是**连续 4 年利润下滑后首次正增长**，具有"拐点意义"。

### 利好 2：饲料原料及油脂科技毛利率翻倍

**依据**

2025 年饲料原料及油脂科技业务毛利率 **5.0%（同比翻倍）**，收入约 924 亿元，销量 3,157 万吨。南美大豆供应充足 + 大豆压榨利润上涨 + 下游养殖业需求良好。

**解读**

压榨利润翻倍是 2025 年利润修复核心引擎。虽然大豆压榨利润具有周期性，但**南美大豆持续增产 + 中国养殖业需求稳定**支撑压榨利润在 2026 年维持合理水平，**利润持续性可期**。

### 利好 3：厨房食品渠道拓展 + 新工厂产能释放

**依据**

厨房食品销量和利润同比增长，**收入 1,509.8 亿元 + 销量 2,518 万吨 + 毛利率 7.3%**。新工厂产能释放 + 销售渠道持续拓展。公司在面粉和大米业务中因原料降价实现盈利明显改善。

**解读**

作为**中国最大粮油品牌**，渠道优势和品牌粘性是长期竞争壁垒。新工厂投产和渠道下沉将进一步巩固市场地位。厨房食品作为 C 端业务盈利更稳定，是长期价值锚点。

### 利好 4：大健康和中央厨房等新业务布局

**依据**

公司积极布局**大健康产业**（DAG 油、植物甾醇等功能性产品），推进**中央厨房**项目，拓展新品类。功能性食品是消费升级方向。研发投入 **2.88 亿元（+5.06%）** 持续加大。

**解读**

新业务布局有望在中长期打开新的增长空间，**提升产品附加值和毛利率**，减少对低毛利大宗粮油业务的依赖。大健康 + 央厨赛道未来 3-5 年有望成为 100 亿+ 新业务板块。

### 利好 5：销量 +6.06% 持续增长 + 资产负债率改善

**依据**

2025 年全年销量 **5,675.4 万吨（+6.06%）**；资产负债率 ~57.84%（同比 -0.51pct）；归母净资产 964.32 亿（+2.88%）。

**解读**

**销量稳健增长（+6.06%）> 收入增长（+2.87%）** 反映公司在量端的扩张能力稳固；资本结构持续优化；前三季度经营现金流 +119.97% 显著改善，为后续扩产 / 新业务 / 分红提供弹药。

## 7. 未来可能的利空

### 利空 1：诉讼风险和不确定性

**依据**

- **子公司广州益海**因合同诈骗案被判需承担 **18.81 亿元连带赔偿责任**（已上诉），影响净利润 2.61 亿元
- **子公司东莞富之源**增值税纠纷被判补缴税款，影响净利润 4.72 亿元（已上诉）
- 合计诉讼预计负债 **7.33 亿元**

**解读**

两起诉讼均已上诉，最终结果存在不确定性。**若二审维持原判，可能进一步计提损失**。尤其广州益海案涉及金额高达 **18.81 亿元**（相当于 2024 年归母净利润的 ~75%），若全额赔偿将对 2026 年利润构成**重大打击**。

### 利空 2：原料成本波动高度敏感

**依据**

大豆、豆油等占经营成本近 **90%**。粮油行业毛利率仅 6.43%，极低的利润空间意味着原料成本波动对盈利影响巨大。大豆压榨利润具有周期性。

**解读**

若大豆价格因天气、政策（如中美贸易摩擦）等因素大幅上涨，而终端产品提价受阻，将导致利润快速收窄甚至亏损。这是**粮油行业的结构性风险**。2025 年压榨利润上涨的顺风可能在 2026 年减弱甚至反转。

### 利空 3：消费需求疲软和零售竞争加剧

**依据**

零售渠道消费不及预期，竞争加剧；收入增速仅 +2.87%（增长动能偏弱）；2024 年营收同比 -5.0%，行业景气度仍面临挑战。

**解读**

消费疲软环境下，食用油和大米等必需品虽然需求刚性，但**品牌溢价和渠道利润面临压力**。若经济恢复不及预期，收入增速可能继续低迷。零售竞争加剧（线上 / 渠道战）影响厨房食品毛利率。

### 利空 4：净利率极低 + 风险抵御能力偏弱

**依据**

整体净利率仅 **1.29%**；毛利率 6.43%（粮油行业结构性低）；单笔 7.33 亿诉讼损失即直接影响净利润 23%。

**解读**

净利率极低反映公司**风险抵御能力偏弱**。任何大宗原料价格波动、汇率变动、单一诉讼事件都可能对利润造成显著冲击。低净利率也限制了进一步提升分红 / 研发 / 扩产的空间。

### 利空 5：H 股上市与股东结构

**依据**

第一大股东丰益营销（香港）持股 89.99%（流通股较少）；新加坡丰益国际为实际控制方。

**解读**

极高的股东集中度（89.99%）意味着**流通盘小 + 外资母公司决策影响大**。若新加坡丰益国际在全球业务战略调整（如棕榈油等其他品类倾斜），可能影响金龙鱼在 A 股业务的投入节奏。

## 8. 总结判断

**一句话结论**

金龙鱼 2025 年呈现"营收 2,451.26 亿（+2.87%）+ 归母 31.53 亿（+26.01%，连续 4 年下滑后首次正增长）+ 扣非 28.45 亿（+193.68%）+ 销量 5,675.4 万吨（+6.06%）+ 整体毛利率 6.43%（+1.54pct）+ 饲料原料及油脂科技毛利率翻倍至 5% + 厨房食品 1,509.8 亿 + 前三季度经营现金流 +119.97% + 诉讼预计负债 7.33 亿"的"**盈利强势修复 + 压榨利润上涨 + 销量增长 + 诉讼风险**"格局；2026 年关键看**压榨利润持续性 + 诉讼二审结果 + 新业务（大健康 / 央厨）贡献**。

**核心看点**

1. **扣非净利润暴增 +193.68%**（核心经营盈利大幅修复 + 连续 4 年下滑后首次正增长）
2. **饲料原料及油脂科技毛利率翻倍至 5%**（大豆压榨利润上涨）+ 厨房食品 1,509.8 亿
3. **前三季度经营现金流 +119.97%** + 销量 +6.06% + 资产负债率改善

**核心风险**

1. **诉讼风险**（广州益海 18.81 亿 + 东莞富之源 4.72 亿），二审结果不确定
2. **原料成本占比 90%+** + 毛利率仅 6.43%（盈利对成本波动极度敏感）
3. **消费疲软 + 零售竞争** + 净利率仅 1.29%（风险抵御能力弱）
