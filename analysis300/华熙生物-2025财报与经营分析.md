# 华熙生物（688363）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

2025年华熙生物实现营业收入42.17亿元，同比 -21.49%；归母净利润2.91亿元，同比 +67.03%；扣非归母1.88亿元，同比 +75.28%。加权平均ROE 4.17%（+1.62ppt）。收入下降但利润大幅增长——"以退为进"的结构调整正在兑现盈利改善。

- 全年营业收入42.17亿元，同比 -21.49%
- 全年归母净利润2.91亿元，同比 +67.03%
- 全年扣非归母1.88亿元，同比 +75.28%

### 1.2 增长与拖累因素

收入下降是主动的战略选择——公司系统性压缩低效营销投放、减少投入产出比低的品牌及项目、将资源集中于核心品牌（润百颜、夸迪）。利润大幅增长则受益于：销售费用-30%+、管理费用-10%+、Q3销售费率降至34.26%（近五年最低）。

- 主要增长驱动：销售费用-30%+、管理费用-10%+、核心品牌聚焦
- 主要拖累因素：主动削减低效收入、品牌结构调整
- 利润增长主要来自：费用大幅压缩+核心品牌效率提升

### 1.3 趋势判断

华熙生物正从"规模扩张"向"质量增长"转型。2024年是转型起点（利润触底），2025年是"以退为进"的关键一年——收入牺牲换来利润翻倍。核心看点是收入何时企稳回升+利润率持续改善。

- 未来主要利好方向：核心品牌回归增长、润百颜三类械产品放量、合成生物技术
- 未来主要压力方向：收入规模持续缩减、功效护肤竞争加剧
- 整体趋势判断：从规模到质量的转型期，2026年有望收入企稳+利润继续增长

## 2. 主营业务

### 2.1 公司简介

华熙生物科技股份有限公司（688363.SH）是全球最大的透明质酸（玻尿酸/HA）原料供应商和国内领先的生物活性材料平台企业，拥有润百颜、夸迪、BM肌活等消费品牌，以及医美终端产品和生物原料业务。

### 2.2 主营业务构成

1. **皮肤科学创新转化业务**：功能性护肤品（夸迪、润百颜、米蓓尔等品牌）和医美终端产品（润百颜注射用HA、水光产品）
2. **营养科学创新转化业务**：营养健康食品及功能性食品
3. **生物活性物原料业务**：透明质酸原料、胶原蛋白原料等，全球领先
4. **医疗终端业务**：医疗器械类产品（骨科注射、眼科等）

### 2.3 核心收入来源

皮肤科学（护肤+医美）是最大收入来源。原料业务虽然体量相对较小但毛利率极高且全球领先。2025年的调整主要集中在C端品牌体系和产品结构优化。

## 4. 营收分布和增长

### 4.1 业务调整方向

业绩快报未披露详细的分业务收入数据，但揭示了以下结构性调整方向：

1. **压缩低效营销投放**——以销售额为单一指标的投放活动被大幅削减
2. **减少低效品牌/项目**——投入产出效率较低的品牌及项目被减少
3. **聚焦核心品牌**——资源集中于润百颜、夸迪等具备技术基础和产品积累的核心品牌
4. **品牌传播由规模→效率转型**——加强专业内容科普，推动传播体系效率导向

### 4.2 关键产品进展

### 4.3 营收分析

华熙生物在过去几年经历了"规模扩张→品牌泛化→效率下降"的困境。2025年的-21.5%收入下降不是被动的市场萎缩，而是管理层主动"砍掉"低效业务的结果。关键问题是：核心品牌（润百颜、夸迪）在调整后能否重回增长？

## 6. 未来可能的利好

### 利好 1：润百颜·玻玻——国内首款三类械水光

**依据**

自主研发的"润百颜·玻玻"成为国内首款三类械水光产品。

**解读**

三类医疗器械是医美水光产品的最高审批级别，合规壁垒极高。润百颜作为国内首款获批的三类械水光，具有先发优势和品牌背书。国内水光市场规模预计数百亿，此前以非法或二类械产品为主。三类械的获批将推动市场规范化，利好率先获批的企业。如果"玻玻"2026年放量，可为医美终端业务贡献显著增量。

### 利好 2：费用压缩效果持续——利润率有望继续提升

**依据**

2025年销售费用-30%+，Q3销售费率34.26%（五年最低），管理费用-10%+。

**解读**

费用压缩的效果已在2025年得到验证（利润+67%）。如果2026年收入企稳或回升，而费用继续保持低位，利润率有望进一步提升。历史上华熙生物的销售费率曾高达50%+，目前已降至34%——仍有下降空间。如果费率能进一步降至30%，在收入不变的情况下，利润可再增加1.7亿。

### 利好 3：营养科学首季盈利——新增长曲线拐点

**依据**

营养科学业务在收入保持增长的同时实现季度首次盈利。

**解读**

营养科学是华熙生物的"第二曲线"业务。从亏损到盈利是重要的里程碑。如果能持续盈利并扩大规模，将验证华熙生物从透明质酸向更广泛的合成生物材料拓展的战略可行性。

## 7. 未来可能的利空

### 利空 1：收入持续缩减——何时企稳是关键

**依据**

2025年营收42.17亿（-21.5%），2024年已从2023年的高峰大幅下降。

**解读**

收入连续大幅下降已持续2年。虽然管理层定义为"主动调整"，但市场会质疑：1）核心品牌（润百颜、夸迪）的市场竞争力是否真的在恢复？2）砍掉的低效收入是否反映了品牌力的下降？如果2026年收入继续下滑超10%，市场信心可能进一步恶化。

### 利空 2：功效护肤竞争白热化——品牌力挑战

**依据**

国内功效护肤赛道竞争激烈——珀莱雅、贝泰妮、巨子生物等快速崛起。

**解读**

功效护肤是过去几年国内化妆品增长最快的赛道，但竞争也最为激烈。珀莱雅已超越华熙生物成为国货护肤龙头，贝泰妮在敏感肌领域深耕，巨子生物在重组胶原蛋白领域异军突起。华熙生物需要在透明质酸的品牌认知基础上找到差异化定位，否则市场份额可能继续被侵蚀。

### 利空 3：ROE仍然偏低——资本效率有待提升

**依据**

ROE仅4.17%，远低于消费品行业平均水平。

**解读**

4.17%的ROE说明公司资产配置效率较低——大量资产（包括合成生物研发平台、生产基地等）尚未充分产出利润。如果2026年ROE不能提升至8%+，市场可能持续给予较低的估值倍数。

## 8. 总结判断

**一句话结论**

2025年华熙生物营收42.17亿（-21.5%），归母2.91亿（+67%！），扣非1.88亿（+75%），销售费用-30%+——"以退为进"的结构调整正在兑现：收入牺牲换来利润翻倍。润百颜·玻玻（首款三类械水光）+营养科学首季盈利——新增长点在酝酿。

**核心看点**

1. **归母+67%！扣非+75%！——利润大幅改善**
2. **销售费用-30%+，Q3费率34.26%（五年最低）**
3. **润百颜·玻玻——国内首款三类械水光**
4. **夸迪悬油次抛2.0畅销**
5. **营养科学首季盈利——新增长曲线拐点**
6. ROE 4.17%（+1.62ppt）——改善中
7. 合成生物技术平台深化
8. 3款化妆品新原料备案

**核心风险**

1. **营收-21.5%——连续两年大幅下滑**
2. **收入何时企稳？——2026年是关键验证年**
3. 功效护肤竞争白热化——珀莱雅、巨子生物等追赶
4. ROE仅4.17%——资本效率偏低
5. 品牌力是否真正恢复——待验证
