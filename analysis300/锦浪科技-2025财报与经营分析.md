# 锦浪科技 2025年财报整理

> **说明**：锦浪科技（300763.SZ）**2025 年年度报告尚未披露**（预计 2026 年 4 月下旬披露，**业绩预告/快报亦未发布**）。本文基于 **2025 年三季报（2025-10-15 披露）+ 2025 年半年报（2025-08 披露）+ 投资者交流（2026-03）+ 券商研报** 整理，年报未披露的字段统一标注"年报未披露"。

## 1. 业绩总结

### 1.1 收入与利润表现

锦浪科技 2025 年 Q1-Q3 实现营业收入 56.63 亿元，同比 **+9.71%**；归母净利润 8.65 亿元，同比 **+29.39%**；扣非净利润 8.02 亿元，同比 **+27.32%**。经营活动现金流 17.07 亿元（同比 +21.22%）。Q1-Q3 毛利率 **36.04%（+7.06pct）**、净利率 **15.28%（+17.94pct）**，**盈利能力显著改善**。H1 表现尤为突出（营收 +13.09% / 归母 +70.96% / 扣非 +71.80%），Q3 单季"增收不增利"—— 营收 18.69 亿（+3.43%）但归母仅 2.63 亿（**-16.85% 同比 / -35% 环比**），需关注。

- Q1-Q3 营业收入 56.63 亿元（同比 +9.71%）
- Q1-Q3 营业利润 年报未披露（全年）
- Q1-Q3 归母净利润 8.65 亿元（同比 +29.39%）

### 1.2 增长与拖累因素

- **主要增长驱动**：①**储能逆变器 H1 收入 7.93 亿（+313.51%）** 爆发式增长；②海外业务 H1 17.78 亿（+25.26%），占比 46.86%；③**毛利率大幅 +7.06pct**（价格修复 + 结构优化）；④大洋洲 / 中东 / 非洲等新兴市场高增；⑤逆变器均价提升（销量 -7% 但收入 +16.6%）
- **主要拖累因素**：①**并网逆变器 H1 -11.22%**（传统业务下滑）；②**逆变器总销量 46.62 万台（-6.98%）**；③**Q3 单季归母 -16.85%**（环比 -35%）；④**8 月欧洲出口 -1.6%**；⑤电芯价格上涨 + 成本传导滞后（3-6 个月）
- **增长主要来自**：储能业务爆发 + 海外高毛利市场；**承压主要来自**：并网逆变器传统业务下滑 + Q3 季度波动

### 1.3 趋势判断

- **未来主要利好方向**：储能系统 2026 核心战略增量（工商储 / 户储 / 地面储全场景）+ 东南亚 / 拉美产能 2026 投产规避关税 + 新兴市场开拓 + 毛利率持续修复
- **未来主要压力方向**：Q3 单季利润下滑信号 + 欧洲需求波动 + 电芯成本传导滞后 + 逆变器行业竞争加剧
- **整体趋势判断**：**2026 有望延续储能爆发 + 海外扩张 + 盈利修复**格局，关键看全年储能兑现（民生证券预测归母 12.31 亿、+78.1%）+ Q4 出货节奏

## 2. 主营业务

### 2.1 公司简介

锦浪科技是**全球光伏逆变器第三大制造商 + 全球户用单相组串式逆变器第一**，主营**并网逆变器 + 储能逆变器 + 储能系统 + 新能源电站运营**，覆盖户用 / 工商业 / 大型地面电站全场景。

### 2.2 主营业务构成

1. **并网逆变器**：户用单相 + 组串式 + 大功率，公司基本盘，**全球户用单相第一**
2. **储能逆变器**：户储 + 工商储逆变器，**2025-2026 核心增长引擎**（H1 +313.51%）
3. **储能系统**：**工商储（125kW/261度电）**（2025-01 批量发货）+ **户储系统**（2026-03 正式发货）+ **地面储能**（2026-05/06 上市）
4. **新能源电力生产**：通过子公司锦浪智慧进行光伏电站开发 / 建设 / 运营 + 户用光伏发电系统
5. **电池配套**：主要通过外部代工厂合作（轻资产模式）

### 2.3 核心收入来源

**并网逆变器 + 储能逆变器 + 海外市场**为当前三大支柱；**储能系统**是 2026 核心战略增量；**户用单相逆变器**保持全球第一优势；**海外市场占比 46.86%**。计划 **2027 年储能占比提升至 45%**（当前约 20%）。

## 6. 未来可能的利好

### 利好 1：储能系统 2026 核心战略增量

**依据**

- 2026-03 公司明确**将储能系统确定为 2026 年核心战略增量**
- **工商储系统**（125kW/261 度电）2025-01 批量发货
- **户储系统** 2026-03 底正式发货，Q1 已获数千套试订单
- **地面储能产品** 预计 2026-05/06 上市
- **储能逆变器 H1 收入 +313.51%（7.93 亿）**，毛利率 30.27%
- **计划 2027 年储能占比提升至 45%**（当前 ~20%）

**解读**

公司从"纯逆变器"向"**储能系统 + PCS + BMS + 电池整体解决方案**"转型，储能业务从 0 到 1 快速兑现 + 工商储 / 户储 / 地面储全场景覆盖，有望成为 **2026-2028 年核心增长引擎**。若 2027 年储能占比 45% 目标达成，将彻底改变公司估值逻辑和增长曲线。

### 利好 2：海外产能布局 2026 年投产

**依据**

公司计划在**东南亚和拉美建立本地生产基地**，**2026 年投产**；已通过可转债募集 16.77 亿元支持产能扩张。海外收入占比 46.86%（+25.26% 同比）。大洋洲 / 中东 / 非洲等新兴市场快速释放。

**解读**

东南亚 + 拉美产能投产将**降低关税成本 + 贴近客户 + 规避贸易壁垒**，特别是**美国本土化需求 + 欧洲储能爆发 + 新兴市场快速增长**三重利好。可有效对冲**欧洲出口波动 + 美欧贸易政策风险**。

### 利好 3：毛利率大幅修复 + 盈利能力结构性改善

**依据**

Q1-Q3 毛利率 **36.04%（+7.06pct）** + 净利率 **15.28%（+17.94pct）**；H1 并网逆变器毛利率 **26.12%（+7.62pct）**；储能逆变器毛利率 30.27%（+2.69pct）。

**解读**

毛利率 +7pct + 净利率 +17.94pct 反映**盈利质量显著改善** + 规模效应 + 高毛利储能业务占比提升 + 海外高端市场拓展。若 2026 继续保持 36%+ 毛利率，归母净利润有望持续高增。

### 利好 4：全球逆变器龙头 + 户用单相第一

**依据**

- **全球光伏逆变器第三大制造商**，2024 年全球市占率约 5%
- **户用单相组串式逆变器全球第一**
- 逆变器总收入 H1 **26.13 亿（+16.57%）**

**解读**

户用单相逆变器龙头地位稳固 + "**AI + 储能 + 户用光伏**"三大趋势共振。若全球户用光伏 + 储能市场 2026 爆发式增长，公司作为龙头将直接受益，市占率有望进一步提升。

### 利好 5：电站运营 + 轻资产储能电池模式

**依据**

通过子公司**锦浪智慧**从事新能源电力生产 + 户用光伏发电系统业务；电池产能主要通过外部代工厂合作（轻资产模式）。

**解读**

"**逆变器 + 储能系统 + 电站运营**"生态闭环 + 轻资产电池模式提升 ROE + 降低资本开支压力。

## 7. 未来可能的利空

### 利空 1：Q3 单季"增收不增利"信号

**依据**

Q3 单季营收 18.69 亿（+3.43%）+ 归母 **2.63 亿（-16.85% 同比 / -35% 环比）**。Q3 增速从 H1 的 +70.96% 骤降至 -16.85%。

**解读**

Q3 单季利润大幅下滑反映**储能备货 + 产能爬坡 + 费用增加 + 欧洲需求波动**等短期压力。若 Q4 延续 Q3 趋势，**全年利润兑现民生证券 12.31 亿预测存在不确定性**。

### 利空 2：欧洲市场需求波动

**依据**

- **8 月对欧洲出口 -1.6%**
- 欧洲户储渠道去库存周期延长
- 欧洲补贴退坡 + 电价波动 + 需求节奏不稳定

**解读**

欧洲是光伏 / 储能逆变器核心市场，补贴退坡 / 渠道去库存 / 地缘政治等因素导致**欧洲需求持续波动**。虽然大洋洲 / 中东 / 非洲新兴市场对冲，但欧洲地位短期难以替代。

### 利空 3：电芯价格上涨 + 成本传导滞后

**依据**

- 电芯供应面临价格上涨挑战
- 成本压力传导需 3-6 个月周期
- 储能系统业务电池成本占比高

**解读**

电芯价格上涨若持续 + 成本向终端传导滞后，**短期储能系统毛利率承压**，2026 Q1-Q2 储能业务利润可能低于预期。公司通过外部代工厂合作可缓解产能压力，但议价能力相对有限。

### 利空 4：并网逆变器传统业务下滑

**依据**

H1 并网逆变器收入 18.20 亿（**-11.22%**）；逆变器总销量 46.62 万台（-6.98%）；国内光伏行业产能过剩 + 价格战持续。

**解读**

并网逆变器仍占收入较大比例（H1 约 48%），若持续 -11%~-15% 下滑，储能爆发需更快以完全对冲。国内光伏新增装机增速放缓 + 价格战压力持续。

### 利空 5：逆变器行业竞争加剧

**依据**

阳光电源 / 华为 / 古瑞瓦特 / 德业等全球逆变器龙头竞争加剧；全球逆变器产能普遍过剩 + 户用市场价格战；储能系统领域竞争者众多（宁德时代 / 比亚迪 / 阳光电源 / 远景等）。

**解读**

逆变器行业**全球市场份额相对分散** + 龙头公司差异化竞争激烈 + 储能系统领域竞争加剧，公司份额扩张难度加大，价格战压力可能进一步传导至毛利率。

## 8. 总结判断

**一句话结论**

锦浪科技 2025 Q1-Q3 呈现"营收 +9.71%（56.63 亿）+ 归母 +29.39%（8.65 亿）+ 储能逆变器 H1 +313.51% 爆发 + 海外 46.86% + 毛利率 36.04%（+7.06pct）+ 可转债 16.77 亿 + 工商储批量发货 + 户储 2026 Q1 发货 + 东南亚/拉美 2026 投产 + 全球逆变器第三 + 户用单相全球第一"的多重向好格局，但 **Q3 单季归母 -16.85%** + 欧洲出口波动 + 电芯价格压力为隐忧，2026 关键看储能系统规模化 + 海外产能投产 + 欧洲需求回暖 + Q4 业绩兑现。

**核心看点**

1. **储能系统 2026 核心战略增量**（工商储 / 户储 / 地面储全场景 + 2027 占比 45% 目标）
2. **海外收入占比 46.86% + 东南亚 / 拉美产能 2026 投产**
3. **毛利率 +7.06pct + 净利率 +17.94pct 盈利能力结构性改善**

**核心风险**

1. **Q3 单季归母 -16.85% 同比 / -35% 环比**（增收不增利信号）
2. **欧洲出口 8 月 -1.6% + 需求持续波动** + 电芯价格上涨 + 成本传导滞后
3. **并网逆变器 H1 -11.22% 传统业务持续下滑** + 行业竞争加剧
