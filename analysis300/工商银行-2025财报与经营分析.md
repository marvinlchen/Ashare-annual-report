# 工商银行 2025年财报整理

> **数据来源：** 工商银行（601398.SH / 01398.HK）2025 年年度报告已于 **2026-03-27** 正式披露（A+H 股），数据完整。**审计机构：安永华明会计师事务所（特殊普通合伙，境内）/ 安永会计师事务所（境外）**，分别根据中国审计准则和国际审计准则审计，出具标准无保留意见审计报告。**全球首家总资产突破 50 万亿元的商业银行，盈利韧性稳中提质，全球系统重要性银行（G-SIBs）**。

# 工商银行（601398）2025年财报与经营分析

## 1. 业绩总结

### 1.1 收入与利润表现

2025年工商银行营业收入8,013.95亿元，同比 +1.9%；净利润3,707.66亿元，同比 +1.0%；归母净利润3,685.62亿元，同比 +0.74%。总资产53.48万亿元（+9.5%），成为全球首家突破50万亿元的银行。非利息收入1,663亿元（+11.8%）增速亮眼。全年派息1,106亿元，A股股息率4.22%。

- 全年营业收入8,013.95亿元，同比 +1.9%
- 全年净利润3,707.66亿元，同比 +1.0%
- 归母净利润3,685.62亿元，同比 +0.74%

### 1.2 增长与拖累因素

非利息收入+11.8%是营收增长的主要驱动力，有效对冲了利息净收入-0.4%的下降。NIM 1.28%（-14bp）是利息收入承压的直接原因。成本收入比29.48%保持稳定。个人贷款不良率1.58%（+0.43ppt）上升显著，反映了房地产调整对零售端的影响。

- 主要增长驱动：非利息收入+11.8%、总资产+9.5%、战略业务贷款高增长
- 主要拖累因素：NIM-14bp、利息净收入-0.4%、个人不良率+0.43ppt
- 增长或承压主要来自：以量补价保住利息收入、非息收入高增长补充

### 1.3 趋势判断

工商银行作为全球最大商业银行，53.48万亿总资产、3,686亿归母净利润、1,106亿分红的体量无可比拟。在息差持续收窄的大环境下，通过规模扩张（+9.5%）和非息收入增长（+11.8%）实现了营收利润双增。ROE 9.45%虽然有所下降但仍在合理水平。

- 未来主要利好方向：非息收入持续增长、战略贷款高增长、分红持续稳定
- 未来主要压力方向：NIM继续收窄、个人不良率上升、ROE下行
- 整体趋势判断：宇宙行稳如磐石，低增长高分红的典型代表

## 2. 主营业务

### 2.1 公司简介

中国工商银行（601398.SH / 1398.HK）是全球资产规模最大的商业银行，总资产53.48万亿元，2025年成为全球首家突破50万亿元的银行。客户基础庞大，在全球设有广泛的机构网络。以公司金融和个人金融为核心，辅以资金业务和综合化经营。

### 2.2 主营业务构成

1. **利息业务**：利息净收入6,351亿元（占营收79.3%），贷款30.51万亿、存款37.31万亿
2. **非利息业务**：非息收入1,663亿元（+11.8%），手续费佣金1,112亿元
3. **战略业务**：科技贷款6万亿（+19.9%）、绿色贷款6.7万亿（+19.1%）、普惠3.6万亿（+22.8%）
4. **养老金融**：管理养老金5.9万亿（+18.5%）

### 2.3 核心收入来源

利息净收入是绝对主体（79.3%），但非利息收入（20.7%）增速更快（+11.8%），收入结构正在优化。手续费佣金净收入1,112亿元是非息收入的主要组成部分。

## 4. 营收分布和增长

### 4.1 收入结构分析

**利息净收入-0.4%——以量补价**

利息净收入6,351亿元微降0.4%，NIM 1.28%（-14bp）的下行被贷款规模+7.5%的增长大部分对冲。利息收入-6.7%，利息支出-11.9%，负债端成本下降更快是维持NII稳定的关键。NIM降幅逐季收窄，Q4环比改善。

**非利息收入+11.8%——增长亮点**

非息收入1,663亿元（+11.8%）增速远超利息收入，成为营收增长的核心驱动。这体现了工行在财富管理、代理销售、投行咨询等领域的收入结构优化。

### 4.2 战略业务高速增长

科技贷款+19.9%（6万亿）、绿色贷款+19.1%（6.7万亿）、普惠+22.8%（3.6万亿）——三大战略业务增速均接近或超过20%，远高于全行贷款+7.5%的增速。这些业务不仅是响应政策要求，也正在成为工行的增长引擎。

### 5.2 利润分析

**利润增速虽低但正增长**

在NIM下降14bp的背景下，工行仍实现了净利润+1.0%的正增长。这得益于：1）规模扩张以量补价；2）非息收入高增长；3）拨备计提保持审慎但总额可控。

**ROE 9.45%和ROA 0.72%**

ROE和ROA均有所下降（反映了资产扩张速度快于利润增速），但仍处于大行合理区间。9.45%的ROE意味着工行的净资产每年能产生约9.45%的回报，对于低风险的大行而言仍具吸引力。

**成本管控有效**

成本收入比29.48%，营业费用+1.9%与营收增速基本持平。在加大科技投入和战略业务拓展的同时，保持了良好的费用纪律。

## 6. 未来可能的利好

### 利好 1：非利息收入持续高增长

**依据**

非息收入1,663亿元（+11.8%），占比提升至20.7%。

**解读**

如果非息收入能保持10%以上的增长，即使利息净收入持平，也能推动营业收入增长2%以上。财富管理、代销基金/保险、投行咨询等业务的深化将持续贡献非息收入增量。

### 利好 2：全球首破50万亿，规模效应持续

**依据**

总资产53.48万亿（+9.5%），贷款30.51万亿（+7.5%），存款37.31万亿（+7.1%）。

**解读**

规模是大行最大的护城河。53万亿的资产规模意味着：1）更强的议价能力；2）更低的单位运营成本；3）更多的交叉销售机会。贷款+7.5%的增速可以在一定程度上对冲NIM下行。

### 利好 3：千亿级分红，高股息率

**依据**

全年派息1,106亿元，A股股息率4.22%，H股5.99%。

**解读**

在低利率环境下，工行的稳定分红（1,106亿元/年）具有极高的配置价值。4.22%的A股股息率远超银行理财和国债收益率。如果利润保持正增长，分红有望持续提升。

## 7. 未来可能的利空

### 利空 1：NIM持续收窄

**依据**

NIM 1.28%（-14bp），净利差1.15%（-8bp）。

**解读**

如果2026年NIM继续下行10-15bp至1.13-1.18%，按53万亿资产计算，利息净收入可能下降500-800亿元。虽然可以通过规模增长部分对冲，但NIM每下降1bp对利息净收入的影响约53亿元。

### 利空 2：个人不良率大幅上升

**依据**

个人贷款不良率1.58%（+0.43ppt），零售资产质量短期进入上行通道。

**解读**

个人不良率+0.43ppt是一个显著的恶化信号，主要受房地产调整和消费信贷违约增加影响。如果2026年房地产市场继续调整，按揭贷款和消费贷的不良可能进一步暴露，增加拨备计提压力。

### 利空 3：ROE持续下行

**依据**

ROE 9.45%，有所下降。ROA 0.72%。

**解读**

资产扩张速度（+9.5%）远快于利润增速（+1.0%），导致ROE和ROA持续下行。如果资本回报率继续降低，长期来看会影响股东价值创造和估值水平。

## 8. 总结判断

**一句话结论**

2025年工商银行营收8,014亿元（+1.9%）、归母净利润3,686亿元（+0.74%），总资产53.48万亿全球首破50万亿；非息收入+11.8%是增长亮点，NIM 1.28%（-14bp）但逐季收窄；全年派息1,106亿元，A股股息率4.22%。

**核心看点**

1. **总资产53.48万亿（+9.5%）**，全球首家突破50万亿银行
2. 非利息收入1,663亿元（+11.8%），收入结构优化
3. 科技贷+20%、绿色贷+19%、普惠+23%，战略业务高增长
4. 全年派息1,106亿元，A股股息率4.22%、H股5.99%
5. 不良率1.31%（-0.03ppt），整体资产质量改善
6. 营收利润双增，盈利韧性确认

**核心风险**

1. NIM 1.28%（-14bp），息差持续收窄
2. 个人不良率1.58%（**+0.43ppt**），零售端显著恶化
3. ROE 9.45%、ROA 0.72%，资本回报率下行
4. 归母净利润仅+0.74%，增速非常缓慢
