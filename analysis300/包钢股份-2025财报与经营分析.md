# 包钢股份 2025年财报整理

> **说明**：包钢股份（600010.SH）**2025 年年度报告尚未披露**（预计 2026 年 4 月中下旬披露，**业绩预告/快报亦未发布**）。本文基于 **2025 年三季报（2025-10-27 披露）+ 2025 年半年报（2025-08-26 披露）+ 2025 年一季报 + 2024 年年报（2025-04-19 披露）** 整理，年报未披露的字段统一标注"年报未披露"。

## 1. 业绩总结

### 1.1 收入与利润表现

包钢股份 2025 年 Q1-Q3 实现营业收入 480.80 亿元，**同比 -3.58%**；归母净利润 2.33 亿元，**同比 +145.03%**（增加 7.49 亿元，**实现扭亏为盈**）；扣非归母净利润 1.86 亿元，**同比 +132.67%**。**毛利率 9.99%（+2.11pct）**，**三个季度连续上涨**；净利率 0.50%（+1.89pct）。H1 营收 313.29 亿（-11.02%），归母 1.51 亿（+39.99%），利润总额 1.68 亿（+148.57%）。在钢材价格指数同比 **-13.35%** 的行业低谷下**靠降本兑现利润修复**——H1 降本 **8.22 亿**，Q1-Q3 吨钢制造成本再降 **2.45%**（去年 -2.8% 基础上）。

- Q1-Q3 营业收入 480.80 亿元（同比 -3.58%）
- Q1-Q3 营业利润 年报未披露（H1 营业利润 1.68 亿，+148.57%）
- Q1-Q3 归母净利润 2.33 亿元（同比 +145.03%）

### 1.2 增长与拖累因素

- **主要增长驱动**：①**降本增效**（H1 降本 8.22 亿 + Q1-Q3 吨钢成本再降 2.45%）；②**品种钢高增**（Q1-Q3 品种钢产量 **711.14 万吨，+18.1%**）；③**稀土钢新材料 +8.28%**（同期新高）；④稀土精矿业务贡献稳定利润（H1 稀土精矿销售 20.49 万吨，均价 1.87 万元/吨）；⑤毛利率 +2.11pct 连续三季度上涨
- **主要拖累因素**：①**钢材价格指数 -13.35%**（行业深度承压）；②营收 -3.58% 规模效应下降；③钢铁行业产能过剩 + 需求疲软；④净利率仅 0.50%（风险抵御能力弱）；⑤H1 钢铁板块（板材 465.40 万吨 + 管材 87.13 万吨 + 型材 79.88 万吨 + 线棒材 61.76 万吨）普遍承压
- **增长主要来自**：降本 + 品种钢 / 稀土钢 / 稀土精矿；**承压主要来自**：钢价低迷 + 需求疲软 + 钢铁板块整体低位

### 1.3 趋势判断

- **未来主要利好方向**：①**稀土板块价值重估**（国家稀土战略 + 资源稀缺性）；②**稀土钢 + 品种钢占比持续提升**（高端化转型）；③2026 年钢铁行业供给侧政策（减碳 / 产能置换 / 反内卷）；④持续降本增效 + 吨钢成本仍有下降空间
- **未来主要压力方向**：①钢价若继续下探 + 需求持续疲软；②稀土精矿价格波动；③资产规模庞大 + 折旧负担；④净利率极低（风险抵御能力弱，原料 / 钢价任何冲击即可能由盈转亏）
- **整体趋势判断**：**包钢股份 2025 Q1-Q3 呈现"价跌量增+降本增效+稀土战略+品种钢升级+扭亏为盈"格局**。2026 年关键看钢铁行业触底回暖节奏 + 稀土板块重估 + 降本持续兑现。短期是"困境反转"初始阶段，长期依靠稀土资源独有性和稀土钢差异化竞争力

## 2. 主营业务

### 2.1 公司简介

包钢股份是**中国北方最大的钢铁联合生产企业之一 + 全球最大的稀土资源拥有企业**，隶属于内蒙古包钢集团（国有控股）。公司拥有世界级稀土矿藏——**白云鄂博矿区**（稀土储量占全国 83%+、全球 34%+），**稀土矿开采指标占全国 70%**（18.9 万吨/27 万吨，2024 年数据）。主营**钢铁产品 + 稀土精矿 + 萤石精矿**等。

### 2.2 主营业务构成

1. **钢铁产品（核心主业，占营收 85%+）**：
   - 板材：H1 产量 **465.40 万吨**（占钢铁主体）
   - 管材：H1 产量 **87.13 万吨**
   - 型材：H1 产量 **79.88 万吨**
   - 线棒材：H1 产量 **61.76 万吨**
   - **稀土钢新材料**（高端产品）：Q1-Q3 产量 **+8.28%**（创同期新高）；开发 6 个具有自主知识产权的新型稀土高端汽车板产品
   - **品种钢**（高附加值）：Q1-Q3 完成 **711.14 万吨（+18.1%）**
2. **稀土精矿（独有资源，战略地位极高）**：
   - H1 生产 **21.11 万吨**（干量，REO = 50%）
   - H1 销售 **20.49 万吨**，平均售价 **1.87 万元/吨**
   - **全国稀土矿开采占比 70%**（2024 年 18.9/27 万吨）
3. **萤石精矿及氟化工产品**：萤石精矿、氟化盐等，稀土矿伴生资源综合利用

### 2.3 核心收入来源

**钢铁产品**占营收绝对大头（~85%+），包括板材、管材、型材、线棒材；**稀土精矿**是**战略资源核心利润池**（高毛利，占利润比重远高于收入占比）；**品种钢 + 稀土钢新材料**是结构性升级方向，占比持续提升。公司是**"钢 + 稀土"双主业布局**，稀土战略地位日益突出。

## 6. 未来可能的利好

### 利好 1：稀土战略地位提升 + 资源独有性

**依据**

公司拥有**白云鄂博稀土矿**（稀土储量占全国 83%+、全球 34%+）；**2024 年稀土矿开采占全国 70%**（18.9/27 万吨）；H1 稀土精矿销售 20.49 万吨，均价 1.87 万元/吨。

**解读**

稀土是国家战略资源（新能源、永磁电机、半导体、军工等核心需求）。**全球稀土定价权 + 中国稀土供给收紧** + 需求持续增长 → **稀土精矿价格和销量有望稳步上行**。公司作为全国最大稀土资源提供方，**战略地位日益突出**，估值重塑空间大。

### 利好 2：降本增效持续兑现

**依据**

2024 年吨钢制造成本下降 2.8%；**2025 Q1-Q3 吨钢制造成本再降 2.45%**；H1 降本 **8.22 亿**；毛利率三季度连续上涨至 9.99%（+2.11pct）。

**解读**

在钢铁行业价格下行周期中，**包钢通过降本增效跑赢行业** + 盈利能力持续修复。若 2026 年钢价企稳 + 降本措施延续，**利润弹性将大幅释放**（同收入下利润可提升 5-10 亿）。

### 利好 3：品种钢 +18.1% + 稀土钢新材料 +8.28%

**依据**

**Q1-Q3 品种钢完成 711.14 万吨（+18.1%）**；**稀土钢新材料产量同比提升 8.28%**（创同期新高）；开发 6 个具有自主知识产权的**新型稀土高端汽车板产品**。

**解读**

品种钢 + 稀土钢高端化是钢铁行业**破内卷核心路径**（差异化竞争 + 高附加值 + 高毛利）。**稀土钢是包钢的独有差异化优势**（添加稀土元素改善钢材性能，应用于汽车、桥梁、船舶、军工等），若 2026 年订单持续放量 + 毛利率提升，将成为第二增长曲线。

### 利好 4：2026 年钢铁行业供给侧改革预期

**依据**

钢铁行业产能过剩 + "反内卷"政策预期 + 减碳 / 产能置换政策推进 + 需求端基建 / 地产企稳预期。

**解读**

**若 2026 年钢铁行业供给侧政策落地**（去产能 / 限产 / 产能置换 / 减排），**钢价有望企稳反弹**，包钢作为行业龙头将直接受益。国企背景 + 环保达标 + 稀土差异化，**在供给侧改革中居于优势地位**。

### 利好 5：扭亏为盈 + 经营拐点

**依据**

**Q1-Q3 归母 2.33 亿（+145.03%，扭亏为盈）** + 扣非 1.86 亿（+132.67%，扭亏）；毛利率 **三季度连续上涨**；H1 降本 8.22 亿。

**解读**

**扭亏为盈 + 毛利率连续上涨 + 降本持续兑现**是**经营拐点的明确信号**。若 Q4 + 2026 延续，**业绩弹性可期**。虽然净利率仅 0.50%，但边际改善显著，2026 年有望回到 1-2% 正常水平。

## 7. 未来可能的利空

### 利空 1：钢价持续下探 + 行业深度承压

**依据**

**2025 H1 钢材价格指数 -13.35%**；钢铁行业产能过剩 + 需求疲软持续；营收 Q1-Q3 -3.58%。

**解读**

钢价是影响包钢利润最核心变量。若 2026 年地产持续低迷 + 基建增速放缓 + 需求持续疲软，**钢价可能继续下探**，即使降本增效难以完全对冲，**由盈转亏风险仍存在**。

### 利空 2：净利率极低 + 风险抵御能力弱

**依据**

Q1-Q3 净利率仅 **0.50%**；2024 全年归母净利 2.646 亿（净利率 ~0.39%）；利润规模相对资产体量（~1,500 亿）极小。

**解读**

**钢铁行业利润率结构性偏低** + 公司净利率不足 1% → **风险抵御能力极弱**。任何原料涨价（铁矿石 / 焦炭 / 电力）、钢价波动、需求冲击都可能将利润由盈转亏。**利润稳定性差 + 盈利波动大**。

### 利空 3：资产规模庞大 + 折旧负担

**依据**

总资产 ~1,500 亿（钢铁重资产属性）；钢铁产能建设投资大，折旧周期长；设备老化需持续更新。

**解读**

**高资产负担导致高固定成本**，即使销量增长也难以带来显著利润释放。若未来继续扩产或进行重大技术改造，**折旧压力将继续压制利润**。

### 利空 4：稀土精矿价格波动风险

**依据**

H1 稀土精矿均价 1.87 万元/吨；稀土价格具有周期性；包钢稀土（子公司）与母公司存在关联交易。

**解读**

稀土精矿虽然是战略资源，但**价格仍具周期性波动**。若全球新能源需求放缓或下游库存高企，**稀土价格可能下行**，影响公司核心高毛利业务。此外，与集团内关联方（包头稀土高科）的交易价格受监管约束，定价空间受限。

### 利空 5：营收 -3.58% + 规模效应收缩

**依据**

Q1-Q3 营收 480.80 亿（-3.58%）+ H1 营收 313.29 亿（**-11.02%**）。

**解读**

营收下滑反映**下游需求疲软 + 钢价下行**双重压力。规模收缩 → 单位固定成本上升 → 盈利能力受限。若 2026 年继续下滑 + 需求未复苏，**利润修复斜率将低于预期**。

## 8. 总结判断

**一句话结论**

包钢股份 2025 Q1-Q3 呈现"营收 480.80 亿（-3.58%）+ 归母 2.33 亿（+145.03%，扭亏为盈）+ 扣非 1.86 亿（+132.67%）+ 毛利率 9.99%（+2.11pct，三季度连续上涨）+ 品种钢 711.14 万吨（+18.1%）+ 稀土钢新材料 +8.28% + 稀土精矿 H1 销售 20.49 万吨（均价 1.87 万元/吨）+ H1 降本 8.22 亿 + 吨钢成本再降 2.45%"的"**价跌量增 + 降本增效 + 稀土战略 + 品种钢升级 + 扭亏为盈**"格局；2025 年年报尚未披露，2026 关键看**钢铁行业触底回暖 + 稀土板块重估 + 降本持续兑现**。

**核心看点**

1. **稀土战略地位 + 资源独有性**（全国 70% 矿开采 + 白云鄂博 + 新能源需求）
2. **降本增效 + 毛利率三季度连续上涨**（H1 降本 8.22 亿 + 吨钢成本再降 2.45%）
3. **品种钢 +18.1% + 稀土钢 +8.28% 高端化升级**

**核心风险**

1. **钢价持续下探**（H1 钢材价格指数 -13.35%）+ 行业需求疲软
2. **净利率仅 0.50%**（风险抵御能力极弱）
3. **资产规模大 + 折旧负担** + 营收仍 -3.58%（规模收缩）
