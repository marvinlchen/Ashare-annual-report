# 西部证券2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

西部证券2025年实现营业收入59.85亿元，同比 -10.84%；归母净利润17.54亿元，同比 +24.97%；扣非归母净利润17.36亿元，同比 +25.36%。呈现"增利不增收"特征：营收下滑主因大宗商品贸易业务收缩，核心证券业务（财富管理+投行）高速增长驱动利润大幅提升。

- 全年营业收入59.85亿元，同比 -10.84%（2024年67.12亿元）
- 全年归母净利润17.54亿元，同比 +24.97%（2024年14.03亿元）
- 全年扣非归母净利润17.36亿元，同比 +25.36%（2024年13.85亿元）
- 基本每股收益0.39元，同比 +25.13%
- 手续费及佣金净收入同比 +34.98%
- 分红：每10股派0.90元，合计5.36亿元（分红比例30.54%）

### 1.2 增长与拖累因素

核心证券业务表现亮眼：财富管理板块营收同比 +30.77%、投行板块营收同比 +42.09%，手续费及佣金净收入同比 +34.98%。拖累因素主要是大宗商品贸易业务收缩导致营收下降，以及自营投资收入同比 -8.89%、公允价值变动收益亏损3.16亿元。

- 主要增长驱动：财富管理（+30.77%）、投行（+42.09%）
- 主要拖累：大宗商品贸易收缩、自营投资下滑（-8.89%）
- 并购国融证券纳入合并，推高管理费用（+10.51%）

### 1.3 趋势判断

公司正从"交易驱动"向"核心业务驱动"转型，收入结构持续优化。并购国融证券后规模扩张明显（总资产突破1200亿），但需关注整合风险和自营波动。

- 未来利好：核心业务高增、并购整合带来规模效应、拟设立香港子公司拓展海外
- 未来压力：自营投资波动、并购整合成本、经营现金流大幅下降
- 整体判断：增利不增收的转型期券商，核心业务增势良好

## 2. 主营业务

### 2.1 公司简介

西部证券是陕西省属国有控股券商，以财富管理、投行、资产管理、自营投资等为主营业务。2025年完成对国融证券的收购，总资产突破1200亿元。

### 2.2 主营业务构成

1. **财富管理**：经纪、代销金融产品等，营收同比 +30.77%
2. **投资银行**：股权融资、债券承销等，营收同比 +42.09%
3. **自营投资**：证券投资交易，收入18.82亿元，毛利率89.29%
4. **资产管理**：资管产品管理
5. **信用业务**：融资融券、股票质押

### 2.3 核心收入来源

手续费及佣金净收入是核心轻资产收入，同比 +34.98%。自营投资贡献收入18.82亿元但波动较大。融资融券余额123.15亿元（+24.39%）。

## 4. 营收分布和增长

### 4.1 按业务划分

| 业务板块 | 2025年营收 | 同比增速 | 备注 |
|---|---:|---:|---|
| 财富管理 | | +30.77% | 经纪+代销 |
| 投资银行 | | +42.09% | 股+债承销 |
| 自营投资 | 18.82亿元 | -8.89% | 毛利率89.29% |
| 资产管理 | | | |
| 信用业务 | | | 两融余额+24% |
| 手续费佣金净收入 | | +34.98% | 核心轻资产 |

### 4.2 营收分析

**主要增长来源**

财富管理（+31%）和投行（+42%）是核心增长引擎。手续费及佣金净收入+35%体现核心业务竞争力提升。

**营收下降原因**

大宗商品贸易业务收缩导致总营收下降10.84%。该业务贡献收入但毛利率极低，收缩反而有利于收入质量。

**结构变化**

低利润率贸易业务收缩+高利润率核心证券业务增长，收入结构显著优化。这是"增利不增收"的根本原因。

## 6. 未来可能的利好

### 利好 1：核心业务高速增长态势延续

**依据**

财富管理+30.77%、投行+42.09%、手续费佣金+34.98%，核心业务增长强劲。

**解读**

证券行业进入改革深化期，注册制深化和资本市场改革为投行业务提供增量空间。公司核心业务增速远超行业平均水平，市场份额持续提升。

### 利好 2：并购国融证券带来规模效应

**依据**

2025年完成收购国融证券，总资产突破1200亿元。并购带来分支机构网络扩展和业务协同。

**解读**

中小券商并购整合是行业趋势，公司借此实现规模跨越，有望获取更多业务牌照和市场份额。并购完成后的整合效果将在2026年进一步体现。

### 利好 3：拟设香港子公司开拓海外业务

**依据**

公司规划设立香港子公司，拓展海外业务。

**解读**

跨境业务和海外市场是国内券商新增长点。随着互联互通深化和中国资产国际化，海外布局具有战略意义。

### 利好 4：利息收支结构改善

**依据**

利息净收入由负转正（1.02亿元），主要因应付债券利息和回购利息支出减少，利息收入+8.28%。

**解读**

负债端成本下降改善净息差，对利润贡献将持续。

## 7. 未来可能的利空

### 利空 1：自营投资波动风险

**依据**

自营业务收入同比 -8.89%，公允价值变动收益亏损3.16亿元。自营投资规模扩大（占用经营现金流）加大波动风险。

**解读**

自营投资收入18.82亿元是重要利润来源，但市场波动可能导致大额亏损。OCI策略配置虽有助于平滑损益，但实际风险敞口仍需关注。

### 利空 2：经营现金流大幅下降81.64%

**依据**

经营活动现金流22.19亿元，同比 -81.64%（2024年120.89亿元）。主因自营投资规模扩大和融资融券余额增长。

**解读**

经营现金流大幅收缩可能影响公司资金调度能力和流动性安全边际，尤其在市场剧烈波动时可能面临流动性压力。

### 利空 3：并购整合风险

**依据**

并购国融证券后，管理费用+10.51%，业务范围和分支机构大幅增加，合规管理边界扩大。

**解读**

并购整合需要时间和资源，文化融合、系统对接、人员管理等方面可能出现摩擦。监管政策趋严下合规管控压力增加。

### 利空 4：大宗商品贸易业务不确定性

**依据**

大宗商品贸易业务收缩是营收下降的主因。

**解读**

该业务虽毛利率低，但曾贡献较大营收。若公司战略性退出该领域，需关注对合并营收和客户关系的影响。

## 8. 总结判断

**一句话结论**

"增利不增收"的结构优化型增长——核心业务（财富管理+31%/投行+42%）爆发驱动净利+25%，并购国融后总资产突破1200亿，但自营波动和经营现金流骤降(-82%)需警惕。

**核心看点**

1. 财富管理+30.77%、投行+42.09%、手续费佣金+34.98%，核心业务增长强劲
2. 扣非净利+25.36%高于净利+24.97%，盈利质量提升
3. 并购国融证券后总资产突破1200亿，规模跨越

**核心风险**

1. 经营现金流-81.64%（22.19亿 vs 120.89亿），流动性压力
2. 自营公允价值变动收益亏损3.16亿元，投资波动风险加大
3. 并购整合期管理费用+10.51%，合规管理压力
