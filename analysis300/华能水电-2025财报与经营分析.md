# 华能水电 2025年财报整理

> 说明：本文基于华能澜沧江水电股份有限公司2025年第三季度报告（2025年10月31日披露）及2025年半年度报告（2025年8月29日披露）整理。**截至当前，公司2025年年度报告尚未披露**（预计2026年4月下旬披露，尚未发布业绩预告/快报），全年数据、分业务/分地区收入明细、分部利润、股东分红方案等信息暂以"年报未披露"标注。2024年对比数据取自公司2024年年度报告。除特别说明外，金额单位均为人民币亿元。

## 1. 业绩总结

### 1.1 收入与利润表现

华能水电2025年前三季度实现营业收入206.41亿元，同比 +6.30%；归属于上市公司股东的净利润75.39亿元，同比 +4.34%；扣非净利润75.54亿元，同比 +4.62%。毛利率60.68%（同比 +0.85pct）；净利率39.52%（同比 -0.51pct）。基本每股收益0.40元；加权平均净资产收益率11.62%。Q1-Q3合并发电量同比 +11.90%。

**核心特征：**发电量高增 + 营收稳健增长 + 毛利率持续修复，但净利率小幅下滑主要因财务/所得税等非经营性因素。

### 1.2 增长与拖累因素

主要增长驱动：①澜沧江流域来水延续2024年偏丰态势，发电量同比 +11.90%，是营收和利润增长的最核心动力；②装机规模持续扩张（2024年末2,920.32万千瓦，云南省第一），托巴水电站140万千瓦机组2024年全部投产，贡献2025年完整年度电量；③新能源（风光）装机同步扩张；④云南省内用电需求持续增加，电价相对稳定。

主要拖累因素：①财务费用较高（在建项目融资 + 存量债务）；②部分电站平均电价结算价格波动；③净利率小幅下滑（-0.51pct）反映费用端压力。

### 1.3 趋势判断

短期看，澜沧江流域来水偏丰的红利有望延续至2026年，叠加托巴新机组贡献和新能源装机放量，水电+新能源双引擎将继续支撑利润增长。但水电业务核心受来水影响，若2026年出现枯水年或上游梯级电站检修，可能形成业绩冲击。整体判断：**"水电清洁能源稀缺性 + 来水偏丰 + 装机扩张"三轮驱动逻辑清晰，业绩持续性高**，全年归母净利润有望保持稳健增长。

## 2. 主营业务

### 2.1 公司简介

华能水电（华能澜沧江水电股份有限公司）是中国华能集团旗下核心水电上市公司，澜沧江流域水电资源开发主体，水电装机规模位居全国第二、云南省第一，主营水力发电、新能源发电（风电/光伏）及电力销售业务。

### 2.2 主营业务构成

1. **水力发电业务**：公司最核心业务，主营澜沧江流域梯级水电站群（小湾、糯扎渡、乌弄龙、托巴、景洪、大朝山等），合计装机超2,700万千瓦。2024年水电业务占主营收入94.31%。
2. **新能源发电业务**：包括风电和光伏项目，近年来装机快速增长，是公司"十四五"战略延伸方向。
3. **热力与供暖业务**：规模较小，区域性补充业务。
4. **其他业务**：含电站运维服务、工程建设及咨询等配套业务。

### 2.3 核心收入来源

华能水电核心收入来源是澜沧江流域梯级水电站的电力销售。主要客户为云南省电网公司，通过省内直供、省外西电东送（送往广东、广西、重庆等）及越南、老挝等境外市场销售电量。电价按水电基准电价结算，部分机组参与市场化电量交易。收入与来水情况（发电量）、结算电价、外送比例高度相关。澜沧江流域作为中国第一大水电资源富集流域，具备稀缺性。

### 4.3 营收分析

**主要增长来源**

- **澜沧江流域来水偏丰**：2024年乌弄龙+30%、小湾+31%、糯扎渡+29%（H1数据），2025年来水延续偏丰

**主要拖累因素**

- 电价市场化可能带来部分电量结算价格下行（云南省市场化比例提升）
- 外送电量比例与省内需求此消彼长，结构性波动影响收入组合
- 热力/其他业务增量贡献有限

**结构变化**

水电依然绝对主导（2024年占 94.31%），但新能源业务占比提升将是中长期结构变化方向。随着风电和光伏项目（部分位于云南省内）逐步并网，公司向"水风光储一体化"清洁能源运营商转型。

## 6. 未来可能的利好

### 利好 1：澜沧江流域来水持续偏丰 + 装机规模领先

**依据**

2024年澜沧江流域乌弄龙、小湾、糯扎渡等关键电站来水同比偏丰约3成；2025年Q1-Q3发电量同比 +11.90%（Q1更达 +21.84%），来水延续偏丰。截至2024年末装机2,920.32万千瓦，云南省第一，全国水电装机前列。

**解读**

水电是"靠天吃饭"业务，但澜沧江流域中长期来水趋势与"厄尔尼诺-拉尼娜"气候周期、青藏高原冰雪融水等相关。若偏丰态势延续，公司利润将持续释放。稀缺水电资源禀赋+头部装机规模构成深厚护城河。

### 利好 2：托巴水电站满年贡献 + 新机组持续投产

**依据**

托巴水电站（140万千瓦）2024年全部投产，2025年贡献完整年度电量。后续还有如美、古水等梯级电站规划建设中。

**解读**

水电项目建设周期长（8-12年）、投资回收期长，但一旦投产即产生长期稳定现金流。托巴及后续项目将持续抬升装机规模和盈利中枢。

### 利好 3：清洁能源稀缺性 + 电力市场改革红利

**依据**

水电是"零碳 + 灵活调节 + 长寿命资产"的稀缺品种，在能源转型和双碳目标下估值重塑逻辑清晰。云南省内硅、铝、新能源汽车等产业对绿电需求旺盛。

**解读**

在碳中和趋势下，水电的环境价值被低估。随着绿电交易、碳市场、可再生能源消纳等政策推进，水电有望享受"量价齐升"溢价。

### 利好 4：水风光储一体化与新能源装机扩张

**依据**

公司"十四五"战略布局新能源业务，风电和光伏装机快速增长，并规划水风光储一体化项目。

**解读**

新能源业务作为增量板块，能够对冲水电"丰枯"波动，并受益于新能源装机行业高景气。公司"水+风+光"协同调度将提升整体资产效率。

### 利好 5：高分红承诺 + 财务结构改善

**依据**

2024年分红比例42.42%，公司承诺**未来三年分红比例不低于50%**。2025年Q3末资产负债率61.28%，较上年同期下降0.65pct。总资产突破2,000亿元。

**解读**

高分红承诺带来稳定股息回报，适合中长期配置资金。负债率下降 + 融资成本下行空间，进一步提升ROE水平。

## 7. 未来可能的利空

### 利空 1：来水周期性波动风险

**依据**

水电业绩完全依赖来水量。2024年来水偏丰约3成，若2026年进入枯水周期，将对公司业绩形成直接冲击。历史上澜沧江流域曾出现多次枯水年。

**解读**

"靠天吃饭"是水电公司最大的变量。枯水年业绩波动幅度可能达到 -10%~-20%。投资者需对气候周期性保持警惕。

### 利空 2：电价市场化改革的结构性影响

**依据**

云南省电力市场化比例持续提升，2025年部分机组参与市场化电量交易。市场化电价相比基准电价波动更大。

**解读**

如果云南省内用电需求放缓或外送电价下行，市场化电量的单价可能低于基准价，对收入形成压力。

### 利空 3：在建项目投产风险与资本开支压力

**依据**

公司在建水电及新能源项目较多，需要持续资本投入。水电项目建设周期长、技术复杂、资本密集。

**解读**

如果新项目投产进度不及预期或出现工程问题，短期可能影响业绩增长兑现。资本开支大亦可能挤压自由现金流和分红能力。

### 利空 4：高财务费用压制净利率

**依据**

2025年Q1-Q3净利率39.52%，同比 -0.51pct，尽管毛利率提升但净利率下滑。公司有息负债规模较大，财务费用较高。

**解读**

虽然水电业务现金流稳定，但资产重、融资规模大，财务费用刚性较强。如果利率回升或融资规模增加，将持续压制净利率。

### 利空 5：上游梯级电站检修/调度协调风险

**依据**

澜沧江流域梯级电站联合调度复杂，上游电站检修或调度协调出现问题将影响整体发电效率。此外，跨境河流（澜沧江下游流入湄公河）面临国际合作和环保压力。

**解读**

梯级联合调度是优势也是风险，需要精细管理。国际公共资源属性也可能带来外部因素干扰。

## 8. 总结判断

**一句话结论**

2025年Q1-Q3华能水电受益于澜沧江流域来水偏丰 + 托巴等新机组贡献 + 新能源扩张，呈现营收稳增（+6.30%）+ 利润稳健增长（+4.34%）+ 毛利率修复（+0.85pct）的良性态势，**但2025年年报尚未披露**，全年数据待2026年4月确认。

**核心看点**

1. **Q1-Q3发电量 +11.90%**，澜沧江流域来水偏丰延续2024年的有利格局
2. **毛利率修复至60.68%（+0.85pct）**，水电高盈利能力稳固
3. **托巴水电站满年贡献 + 新能源装机扩张**，量增逻辑清晰
4. **装机2,920万千瓦（云南第一）**，稀缺水电资源禀赋 + 碳中和估值重塑
5. **承诺分红比例不低于50%**，股息稳定可期

**核心风险**

1. **来水周期性波动**（若2026年进入枯水周期，业绩将承压）
2. **电价市场化改革**带来结构性影响
3. **高财务费用**压制净利率
4. 资本开支压力 + 梯级联合调度风险
