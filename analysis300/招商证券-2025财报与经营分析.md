# 招商证券（600999）2025年财报整理

## 1. 业绩总结

### 1.1 收入与利润表现

2025年招商证券实现营业收入249.72亿元，同比 +19.53%，连续三年保持增长；归母净利润123.50亿元，同比 +18.91%，创历史新高。公司是"十四五"收官之年交出的一份稳健增长答卷，各核心业务板块全面增长。

- 全年营业收入249.72亿元，同比 +19.53%
- 全年归母净利润123.50亿元，同比 +18.91%
- 利润总额140.80亿元，同比 +25%

### 1.2 增长与拖累因素

业绩增长主要由轻资产业务（财富管理+机构业务）驱动，该板块收入 +35.10%，贡献超过一半营收。市场回暖带动经纪业务收入大增43.8%，股基交易量同比增长69.79%。但经营现金流由+547亿元剧烈转为-314亿元，是最大的财务隐忧。

- 主要增长驱动：市场交投活跃（股基交易量+69.79%）、经纪收入大增（+43.8%）、投行债券承销增长（+21.3%）
- 主要拖累因素：经营现金流-313.69亿元（上年+547.26亿元）、汇兑收益-73.39%、资管增速偏低（+3.29%）
- 增长或承压主要来自：市场景气度回升是核心驱动力，轻资产业务弹性充分释放

### 1.3 趋势判断

招商证券是行业头部券商之一，财富管理和数字化服务优势明显（APP人均使用时长行业第一）。在行业集中度提升的趋势下，公司有望受益于"建设一流投资银行"政策红利。但资管业务发展较慢、经营现金流剧烈波动是需要关注的短板。

- 未来主要利好方向：行业集中度提升利好头部券商、数字化财富管理领先优势、科创板做市和衍生品资格领先
- 未来主要压力方向：经营现金流剧烈波动、分红比例偏低（31.62%）、资管规模有待提升
- 整体趋势判断：稳健增长型头部券商，轻资产业务驱动增长可持续性较好，但需改善现金流管理

## 2. 主营业务

### 2.1 公司简介

招商证券（600999.SH / 6099.HK）是招商局集团旗下的大型综合性证券公司，A+H股上市，总部位于深圳。公司以财富管理为核心，在经纪、投行、自营、资管、衍生品做市等领域全面布局。截至2025年末，正常交易客户数约2,097万户，托管客户资产5.29万亿元。

### 2.2 主营业务构成

1. **财富管理和机构业务**：包括证券经纪、融资融券、基金代销、期货经纪等，2025年收入138.25亿元，占总营收55.36%
2. **投资及交易业务**：包括权益和固收自营投资、衍生品做市、证券做市等，收入69.40亿元，占27.79%
3. **投资银行业务**：包括股权融资、债券承销、并购重组、新三板等，收入10.31亿元，占4.13%
4. **投资管理业务**：包括招商资管和私募股权投资，收入9.54亿元，占3.82%
5. **其他业务**：收入22.22亿元，占8.90%

### 2.3 核心收入来源

财富管理和机构业务是绝对主力（55.36%），其中经纪业务贡献88.9亿元。投资及交易业务（27.79%）是第二大收入来源，自营投资收益（含公允价值变动）达97.9亿元。两大板块合计贡献超83%的收入，体现了"经纪+自营"双轮驱动的业务格局。

### 4.3 营收分析

**主要增长来源**

财富管理和机构业务是最大的增长引擎。受益于A股市场交投活跃（股基交易量+69.79%），经纪业务收入大增43.8%至88.9亿元。客户数增长8.67%至2,097万户，托管资产增长23.89%至5.29万亿元，体现了客户规模和质量的双提升。APP人均使用时长居行业首位，数字化获客和留客能力突出。

投行业务增长20.27%，债券承销5,175亿元（+21.3%），其中ABN和信贷ABS承销分别位列行业第一和第二。IPO家数排名行业第四。"羚跃计划"累计服务665家早期科创企业，超80家完成投行业务转化。

**主要拖累因素**

投资管理业务仅增长3.29%，是增速最低的板块。尽管公募基金规模从1亿元爆发至704亿元，但总体资管收入增长有限，反映资管业务尚处于布局期。投资及交易业务增长9.56%，自营投资收益仅+2.7%，增速低于整体营收增速。

**结构变化**

财富管理和机构业务占比从2024年的约49%提升至55.36%，成为更加突出的营收支柱。轻资产业务（经纪+投行+资管）合计占比约63%，高于重资产业务（自营投资约28%），收入结构较为健康。

## 6. 未来可能的利好

### 利好 1：行业集中度提升，头部券商持续受益

**依据**

招商证券是行业前十大券商之一，总资产7,535亿元，净资产约1,380亿元。在"建设一流投资银行"的政策导向下，头部券商在牌照、资本、客户等方面的优势持续扩大。

**解读**

行业并购整合加速（如中金+银河、国泰+海通等案例），但招商证券作为招商局集团旗下的核心金融平台，具有独特的股东背景优势。在监管鼓励做大做强头部券商的政策环境下，招商证券有望通过有机增长和潜在整合进一步提升市场地位。IPO家数行业第四、ABN承销行业第一、衍生品做市行业第一等多项排名体现了综合竞争力。

### 利好 2：数字化财富管理领先优势

**依据**

招商证券APP人均单日使用时长位居行业首位，月均活跃客户同比增长13.44%。客户数2,097万户（+8.67%），托管资产5.29万亿元（+23.89%）。"羚跃计划"累计服务665家早期科创企业。

**解读**

在券商佣金率持续下行的趋势下，客户黏性和资产规模才是财富管理的核心竞争力。招商证券在数字化服务上的领先优势，有助于实现从"交易佣金"到"AUM管理费"的商业模式转型。5.29万亿元的托管资产规模为未来基金代销、投顾服务、配资等业务提供了广阔的变现空间。

### 利好 3：公募资管业务爆发式启动

**依据**

招商资管公募基金规模从2024年末的1亿元增长至2025年末的704亿元，实现从0到700亿的突破。资管总规模2,610.50亿元。

**解读**

公募基金是券商资管转型的战略方向。招商资管在一年内实现公募规模从1亿到704亿的飞跃，体现了强大的产品发行和渠道能力。随着公募规模的持续增长，资管业务有望从当前的"拖后腿"（仅增3.29%）转变为新的增长极。参考同行经验，千亿级公募规模将带来显著的管理费收入增量。

## 7. 未来可能的利空

### 利空 1：经营现金流剧烈波动，从+547亿转为-314亿

**依据**

2025年经营现金流净额为-313.69亿元，较2024年的+547.26亿元大幅下滑157.32%。变动幅度超过860亿元，是财务指标中变动最为显著的项目。

**解读**

尽管券商经营现金流天然波动较大（受自营投资和客户保证金影响），但如此剧烈的逆转仍值得警惕。-314亿元意味着公司在经营层面大量消耗现金，主要用于融出资金和金融资产的扩张。如果市场出现下跌，这些扩张的资产可能面临减值压力，而现金流的紧张可能限制公司的应对能力。

### 利空 2：盈利对市场景气度高度敏感

**依据**

经纪业务收入88.9亿元占总收入35.6%，自营投资收益97.9亿元，两者合计占营收约75%。Q3市场最活跃时营收77.23亿元，远高于其他季度。

**解读**

经纪+自营两大板块占据约75%的收入，意味着业绩与A股市场走势高度相关。2025年的19%营收增长主要源于市场回暖（股基交易量+69.79%），如果2026年市场交投量回落，经纪和自营收入可能同步承压。轻资产业务的高弹性在市场上行时是优势，下行时则是风险。

### 利空 3：分红比例偏低，股东回报有待提升

**依据**

2025年分红39.05亿元，分红比例仅31.62%，低于许多同行券商40-50%的水平。每10股4.49元，以当前股价计算的股息率约2-3%。

**解读**

在高股息策略受到市场追捧的背景下，31.62%的分红比例对于追求分红回报的投资者吸引力不足。公司可能需要在业务扩张（保留资本）和股东回报之间做更好的平衡。同行中泰证券的分红比例为47.91%，招商证券在这方面有提升空间。

## 8. 总结判断

**一句话结论**

2025年招商证券受益于市场回暖和数字化财富管理优势，营收250亿元（+20%）、净利润123.5亿元（+19%）创历史新高，但经营现金流剧变为-314亿元暴露出资金管理的阶段性压力。

**核心看点**

1. 营收和净利润连续三年增长，2025年净利润创历史新高（123.5亿元）
2. 财富管理和机构业务收入+35%，经纪收入+43.8%，数字化获客优势突出
3. 衍生品做市资格124个行业第一，债券ABN承销行业第一，多项业务行业领先
4. 公募资管从1亿爆发至704亿，资管转型加速

**核心风险**

1. 经营现金流从+547亿剧变为-314亿，变动幅度超860亿，流动性管理需关注
2. 经纪+自营占收入约75%，业绩对市场景气度高度敏感
3. 分红比例仅31.62%，在高股息策略流行的市场环境中吸引力不足
4. 汇兑收益大降73%，国际化业务面临汇率风险
