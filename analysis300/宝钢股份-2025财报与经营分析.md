# 宝钢股份 2025年财报整理

> 说明：本文基于宝山钢铁股份有限公司2025年第三季度报告（2025年10月31日披露）及2025年半年度报告（2025年8月28日披露）整理。**截至当前，公司2025年年度报告尚未披露**（预计2026年4月下旬披露），全年数据、分业务/分地区收入、分部利润等信息暂以"年报未披露"标注。2024年对比数据取自公司2024年年度报告。除特别说明外，金额单位均为人民币亿元。

## 1. 业绩总结

### 1.1 收入与利润表现

宝钢股份2025年前三季度实现营业收入2,324.36亿元，同比 -4.29%；利润总额106.4亿元，同比 +29.4%；归属于上市公司股东的净利润79.59亿元，同比 +35.32%；扣非归母净利润80.36亿元，同比 +46.42%。经营活动现金流净额308.43亿元，同比 +97.23%。前三季度毛利率7.2%，较上年同期5.3%提升1.9pct。三季度单季营业收入810.64亿元（+1.83%），归母净利润30.81亿元（+130.31%），扣非30.81亿元（+163.14%），EPS 0.14元（+133.33%）。

**核心特征：前三季度归母净利润79.59亿已超过2024全年73.62亿**——典型"减收大增利"的行业盈利修复年份。

### 1.2 增长与拖累因素

利润大增的核心驱动：①钢材价格跌幅小于原料价格跌幅，购销差价扩大，毛利率明显抬升至7.2%；②"2+2+N"战略产品销量2,568.9万吨（+13.9%），高附加值产品放量；③出口1-9月接单516.9万吨（+10.9%），全年预计突破650万吨，海外市场对冲内需压力；④汽车板接单Q3量价齐升，新能源车需求拉动。

主要拖累因素：营业收入同比 -4.29%，钢材价格整体仍处下行通道；2024年行业整体毛利率仅5.5%的低基数；国内房地产需求持续承压；钢铁行业产能过剩问题尚未根本解决。

### 1.3 趋势判断

短期看，原料价格下行+产品结构升级+出口放量有望驱动利润修复势头延续，"2+2+N"战略产品和取向硅钢、汽车板等高端品类构成差异化壁垒。但钢价持续下行压制收入端、内需房地产下行以及出口市场的贸易摩擦/反倾销风险构成主要压力。整体判断：**盈利修复逻辑清晰、高端化/出口化进展良好、全年业绩大概率同比大幅改善**，但营收端恢复仍依赖行业景气度回升。

## 2. 主营业务

### 2.1 公司简介

宝钢股份是中国宝武钢铁集团核心上市子公司，全球领先的钢铁综合制造商，以碳钢板带材为核心，主营汽车板、硅钢、家电板、油气管道、金属包装、不锈钢、特钢及钢铁延伸服务。公司下辖宝山基地、东山基地、湛江基地、梅山基地、青山基地和新疆基地等，总产能约4,600万吨，为国内最大钢铁上市公司。

### 2.2 主营业务构成

1. **碳钢板带材业务**：公司最核心的业务，涵盖热轧板/冷轧板/镀层板/彩涂板/电工钢等，用于汽车、家电、能源、基建等下游。
2. **"2+2+N"战略产品**：含取向硅钢、汽车板两大"领跑者"产品，无取向硅钢、镀锡板两大"相对领先"产品，以及N种特色差异化品类（家电板、造船板、油气管线、桥梁结构钢等）。
3. **长材与钢管业务**：包括钢管、棒线材、异型材，主要面向能源（油气管道）、工程机械和建筑行业。
4. **不锈钢与特钢业务**：不锈钢板带、特殊钢制品，向高端制造和装备升级方向延伸。
5. **延伸加工与钢铁服务**：金属包装、深加工、物流配送、资源回收、工业互联网（欧冶云商）等。
6. **海外业务**：印度尼西亚不锈钢项目、沙特合资钢铁项目、东南亚镀锌基地等海外产能布局。

### 2.3 核心收入来源

宝钢股份收入核心来自碳钢板带材，2024年"1+1+N"产品销量3,059万吨（占总销量59.3%），2025年"2+2+N"销量2,568.9万吨（+13.9%），以汽车板、取向硅钢、镀锡板为代表的高附加值产品是核心利润来源。出口销量已成为重要增量，2024年出口606.7万吨创历史新高，2025年预计突破650万吨。公司收入与钢材价格、下游需求（汽车/家电/基建/能源）及原料（铁矿石、焦煤）价格高度相关。

### 4.3 营收分析

**主要增长来源**

2025年Q1-Q3在营收整体同比 -4.29% 的背景下，以下结构性亮点支撑了盈利改善：
- **"2+2+N"战略产品销量2,568.9万吨（+13.9%）**，高附加值品类持续放量
- **出口接单516.9万吨（+10.9%）**，全年预计突破650万吨
- **三季度汽车板接单量显著高于近年平均水平，实现"量价齐升"**，主要得益于新能源汽车需求拉动
- 取向硅钢技术领先（"激光耐热刻痕"项目获2025冶金科学技术奖特等奖），高端产品定价能力强

- 国内房地产需求持续低迷，建筑钢材/管材相关需求承压
- 普钢品类同质化竞争激烈，毛利空间有限

**结构变化**

收入结构进一步向"2+2+N"战略产品和出口市场倾斜。"2+2+N"产品（占总销量比例持续提升）的定价韧性和毛利贡献更好，出口业务成为增长第二引擎。海外基地（湛江、印尼、沙特合资）逐步投产将进一步改善结构。

## 6. 未来可能的利好

### 利好 1：钢价-原料价差扩大持续修复盈利

**依据**

2025年Q1-Q3毛利率提升至7.2%（同比 +1.9pct），Q1-Q3归母净利润79.59亿元（+35.32%）、扣非80.36亿元（+46.42%）。公告明确"钢材价格跌幅小于原料价格跌幅，购销差价有所扩大"。2024年全年降本50.1亿元，2025年持续推进降本增效。

**解读**

钢铁行业进入"以价换量"不再普遍适用、"降本保利"阶段，宝钢股份凭借规模、产线灵活性和采购议价能力，在价差扩大周期中具备领先的利润修复弹性。如果铁矿石价格延续偏弱、焦煤价格维持低位，毛利率有望进一步上行。

### 利好 2："2+2+N"战略产品高端化加速

**依据**

2025年Q1-Q3"2+2+N"产品销量2,568.9万吨，同比增长13.9%。取向硅钢领域，公司牵头的"激光耐热刻痕取向硅钢制造技术开发与应用"项目荣获2025年冶金科学技术奖特等奖。Q3汽车板接单量显著高于近年平均水平，实现"量价齐升"。

**解读**

公司依靠取向硅钢（变压器）、无取向硅钢（新能源电机）、汽车板（新能源/传统汽车）、镀锡板（金属包装）等高端品类构建差异化竞争力。随着新能源汽车渗透率提升、特高压建设加速、家电以旧换新政策延续，高附加值产品需求具备高景气度，有望持续提升吨钢盈利中枢。

### 利好 3：出口高增长打开第二增长曲线

**依据**

2025年1-9月出口接单516.9万吨（+10.9%），全年出口目标600万吨以上，预计突破650万吨。2024年全年出口606.7万吨创历史新高（+3.94%）。公司在印尼、沙特、东南亚布局海外产能，湛江钢铁基地对东南亚辐射能力强。

**解读**

出口是内需承压周期下的重要补偿渠道。宝钢产品定位高端、品牌力强、船运物流便利，是"一带一路"国家首选供应商。若海外基地逐步投产，将进一步改善出口结构并摊薄贸易摩擦风险。

### 利好 4：2026 年钢铁供给侧改革预期升温

**依据**

市场普遍预期2026年钢铁行业将有新一轮供给侧改革措施（粗钢控产、反内卷限产、碳中和减碳等）。宝钢作为行业龙头、央企控股，**受益最为明确**。

**解读**

若供给侧改革切实落地，将从行业供给端压制普钢产能过剩问题，有利于整体钢价企稳并拉高行业盈利中枢。宝钢凭借规模和高端占比，有望获得超额收益。

### 利好 5：高分红 + 低负债 + 低估值安全垫

**依据**

2024年每股现金分红0.21元（含税），分红比例高达61.34%，股息率居行业前列。2025年Q3末资产负债率40.65%（行业最低之一），经营现金流308.43亿（+97.23%）覆盖充分。

**解读**

优异的分红、极低的负债率和充沛现金流，构成强安全边际。在行业周期底部，公司具备"进可攻、退可守"的财务基础。

## 7. 未来可能的利空

### 利空 1：钢价企稳尚需时间，营收收缩持续

**依据**

2025年Q1-Q3营业收入2,324.36亿元，同比 -4.29%。房地产投资持续负增长、基建投资增速放缓，国内需求仍处弱势。

**解读**

钢价底部区间震荡可能持续至2026年，营收端恢复偏慢。如果下游需求进一步恶化，即使成本下行，盈利弹性也会受限。

### 利空 2：原料价格反弹风险

**依据**

2025年毛利修复的核心是钢价跌幅小于原料跌幅。如果2026年铁矿石、焦煤价格反弹（宏观刺激、发改委涨价、供给收缩等），将直接压制毛利率。

**解读**

钢铁行业利润波动的本质是钢价-原料价差。原料供给端（淡水河谷、必和必拓、力拓）集中度高，价格谈判权较弱，若铁矿供应中断或全球经济回暖，原料成本将迅速拉高。

### 利空 3：出口贸易摩擦与反倾销风险

**依据**

2025年Q1-Q3出口接单516.9万吨（+10.9%）。但欧盟、美国、土耳其、印度、巴西等国陆续对中国钢材启动反倾销调查或加征关税；东南亚国家（越南、印尼、泰国）也在跟进。

**解读**

出口规模越大，贸易摩擦风险越高。如果主要出口国集中加税，全年出口650万吨目标可能面临调整，海外收入和利润贡献将被压制。

### 利空 4：行业产能过剩与低端同质化竞争

**依据**

中国钢铁行业长期存在产能过剩，2024年粗钢产量约10.05亿吨。低端普钢产品同质化严重，价格竞争激烈。

**解读**

虽然宝钢以高端产品为主，但低端产线仍面临压价压力。如果供给侧改革不及预期，行业竞争格局难以真正改善，普通品类仍将持续承压。

### 利空 5：碳中和政策与环保投入增加

**依据**

中国"双碳"目标下，钢铁行业被列为重点减排领域。碳排放配额交易、超低排放改造、氢能炼钢转型等环保要求持续收紧。

**解读**

宝钢作为央企龙头，在碳减排和绿色制造方面投入巨大（湛江绿色基地、超低排放改造等），短期内环保投入将形成刚性成本项。如果碳配额逐步收紧，履约成本可能压制火电+钢铁组合的利润空间。

## 8. 总结判断

**一句话结论**

2025年Q1-Q3宝钢股份已呈现典型"减收大增利"的行业盈利修复特征（营收 -4.29%、归母 +35.32%，Q1-Q3归母79.59亿已超2024全年73.62亿），"2+2+N"高端化 + 出口放量 + 成本下行三轮驱动兑现良好；**2025年全年年报大概率交出同比大幅改善的答卷**（预计归母净利润超100亿元，同比增幅或超35%），但完整数据需等待2026年4月年报披露。

**核心看点**

1. **Q1-Q3归母79.59亿（+35.32%）已超2024全年**，利润修复斜率陡峭，Q4有望延续
2. **毛利率抬升至7.2%（+1.7pct）**，钢价-原料价差扩大 + "2+2+N"产品升级 + 出口放量三重驱动
3. **经营现金流308.43亿（+97.23%）**，现金创造能力几乎翻倍，利润质量优异
4. **2+2+N销量 +13.9%、出口 +10.9%**，高端化 + 出口双引擎驱动

**核心风险**

1. 钢价底部震荡拖累营收端恢复（Q1-Q3营收 -4.29%）
2. 原料价格反弹风险（若铁矿/焦煤价格反弹，价差收窄）
3. 出口反倾销风险（欧美、东南亚多国启动调查）
4. 国内房地产需求持续低迷 + 行业产能过剩 + 碳中和环保投入加大
