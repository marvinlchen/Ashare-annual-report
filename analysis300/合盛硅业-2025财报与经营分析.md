## 1. 业绩总结

### 1.1 收入与利润表现

2025 年合盛硅业经营承受行业周期底的沉重压力——营业收入 **204.99 亿元，同比 -23.20%**；**归母净利润 -29.91 亿元**（上市以来首次年度亏损，2024 年 +17.4 亿）；**扣非归母 -31.13 亿元**；基本 EPS -2.55 元、扣非 EPS -2.66 元。分业务看：工业硅营收 **90.77 亿（-34.05%）**、有机硅 **95.75 亿（-21.52%）**，两大传统主业虽盈利但规模大幅收缩；**光伏业务营收仅 8.99 亿（占 4.38%）但成为亏损主源**——多晶硅产线停产 + 组件产线产能利用率偏低 + 光伏资产减值 **8.70 亿**。各项减值合计 **侵蚀利润 16.99 亿**。经营现金流 40.13 亿（-11.16%，**仍保持净流入**）是财务健康的重要正面信号。

- 全年营业收入：**204.99 亿元**，同比 **-23.20%**
- 全年归母净利润：**-29.91 亿元**（由盈转亏，2024 年 +17.4 亿）
- 全年扣非归母：**-31.13 亿元**（上年 +15.4 亿）
- 全年营业利润 / 利润总额：年报摘要未单独披露（利润总额大幅转亏）

### 1.2 增长与拖累因素

业绩承压的"三重打击"：1）**工业硅价格同比 -27%**——核心产品价格暴跌，2024-2025 年间部分月份 553 金属硅跌破 10,000 元 / 吨后再下探至 7,000-8,000 元 / 吨；2）**有机硅 DMC 价格 -12%**——下游需求疲弱 + 产能集中投放压价；3）**光伏多晶硅产线停产 + 组件产线低利用率**——2023-2024 年新布局的光伏 N 型多晶硅项目遭遇行业最差时期，产生大额停工损失和运营亏损；4）**资产减值 16.99 亿**（光伏占 8.70 亿）——年末集中计提"洗大澡"。Q3 单季工业硅价格环比 +8.11% 曾带来短暂扭亏（扣非 +2.62 亿），但 Q4 因减值集中计提再度巨亏。

- 主要增长驱动：Q3 工业硅价格反弹（+8.11% QoQ）、经营现金流 40.13 亿仍为正、碳化硅等高端产品积蓄动能
- 主要拖累因素：工业硅 -27%、有机硅 -12%、光伏停产、减值 16.99 亿
- 增长或承压主要来自：**硅基材料全产业链价格下行 + 光伏投资失误带来的"学费"**

### 1.3 趋势判断

合盛硅业是中国最大的工业硅 + 有机硅一体化生产企业（工业硅产能约 130 万吨 / 年、有机硅约 173 万吨 / 年），具备新疆 / 云南低电价 + 煤炭资源的长期成本优势。2025 年的首次年度亏损主要是行业周期性下行 + 光伏业务战略失误共同导致。传统工业硅 + 有机硅业务**规模虽下滑但仍保持盈利**，底子未坏；光伏板块已通过大额减值 + 停产"出清"，若 2026 年不再有大额减值，公司有望重新回到盈利轨道。管理层在年报中定调"主业触底回暖、高端产品积蓄新动能"——碳化硅（SiC）、电子级硅片等高端方向是中长期看点。

- 未来主要利好方向：工业硅 / 有机硅价格触底反弹、光伏减值一次性出清、碳化硅 / 电子级硅等高端动能、现金流仍为正
- 未来主要压力方向：行业产能过剩延续、光伏残值继续拖累、高杠杆（资产负债率 64.96%）、股东套现压力
- 整体趋势判断：**硅基材料龙头的周期底部 + 光伏负担年**，2026 年有望实现减值出清后的同比大幅改善

## 2. 主营业务

### 2.1 公司简介

合盛硅业股份有限公司（603260.SH）是中国最大的工业硅和有机硅一体化生产企业，产业链覆盖从上游工业硅（金属硅）→ 中游有机硅单体（DMC）→ 下游硅橡胶 / 硅油 / 硅树脂 / 环体，以及新布局的光伏多晶硅 + 组件。总部位于浙江嘉兴，生产基地主要在**新疆石河子 / 鄯善、云南昭通**（依靠低电价 + 煤炭资源）。控股股东为罗立国、罗烨栋父子控制的合盛硅业（嘉兴）集团，控股股东 + 一致行动人合计持股 **73.52%**。

### 2.2 主营业务构成（2025 年）

1. **有机硅产品**：营收 **95.75 亿（-21.52%，占比 46.71%）**——硅橡胶、硅油、硅树脂、环体等下游深加工产品，2025 年第一大收入来源（超过工业硅）
2. **工业硅（金属硅）**：营收 **90.77 亿（-34.05%，占比 44.28%）**——传统核心业务，内部供应有机硅 + 外部销售，价格 -27% 导致收入大幅下滑
3. **光伏产品**：营收 **8.99 亿（占比 4.38%）**——多晶硅 + 组件，2023-2024 年新投入业务，**本年巨亏 + 减值 8.70 亿**
4. 其他业务（含碳化硅、电子级硅片等高端产品）——年报定调"高端产品积蓄新动能"

### 2.3 核心收入来源

**有机硅（46.71%）+ 工业硅（44.28%）合计 91%** 是绝对核心。光伏板块虽然仅 4.38%，但产生了绝大部分亏损（减值 8.70 亿 + 运营亏损）。新疆的低电价（自备电厂 + 煤电一体化）和煤炭资源是长期成本优势，行业底部时的"最后防线"。

## 6. 未来可能的利好

### 利好 1：光伏减值一次性"洗大澡"——2026 年有望实现同比大幅改善

**依据**

2025 年各项资产减值合计 **16.99 亿**（光伏资产减值 **8.70 亿**），多晶硅产线已停产。归母 -29.91 亿中减值贡献了约 17 亿亏损。

**解读**

减值是一次性事件，2026 年若不再有大额减值，仅此一项就能让利润基线**提升约 17 亿**。传统工业硅 + 有机硅业务在减值之外的"实际经营亏损"约 -12 至 -14 亿——若 2026 年工业硅 / 有机硅价格企稳回升 + 光伏停产止住"失血"，扣除非经因素后公司有望重新回到微利或小幅盈利。2026 年归母可能从 -29.91 亿改善至 -5 至 +10 亿的区间。

### 利好 2：工业硅 / 有机硅价格触底反弹 + 新疆低成本优势

**依据**

工业硅 553 年均价从 ~11,000 跌至 ~8,000 元 / 吨（-27%），Q3 环比 +8.11% 已现反弹迹象；有机硅 DMC 价格 -12% 但 Q4 已企稳。合盛在新疆依靠自备电厂 + 煤炭一体化，是行业成本最低的企业之一。

**解读**

硅基材料属于典型周期行业，价格长期支撑来自下游需求（有机硅、铝合金、光伏、碳化硅）。2026 年若部分高成本产能加速退出 + 下游需求复苏，工业硅价格有望回升至 9,000-10,000 元 / 吨，有机硅 DMC 有望回升 5-10%。每 1,000 元 / 吨的工业硅涨价对应约 13 亿收入 + 5-8 亿利润增量（按 130 万吨年产能）。合盛在新疆的低成本优势意味着行业底部时"最后倒下"，价格回升时"最先受益"。

### 利好 3：经营现金流 40.13 亿为正 + 碳化硅 / 电子级硅高端动能

**依据**

2025 年经营现金流净额 **40.13 亿（仍为正）**，研发费用 5.21 亿聚焦**有机硅和碳化硅**领域，年报定调"高端产品积蓄新动能"。

**解读**

尽管账面巨亏 -29.91 亿，但经营现金流仍保持 40.13 亿净流入（折旧摊销 + 应收 / 存货管理），说明公司**没有资金链断裂风险**，能"熬过"行业寒冬。研发聚焦碳化硅（SiC，电动车 / AI 服务器 / 光伏核心材料）和电子级硅片（半导体产业链），是中长期战略第二曲线。若 2027-2028 年 SiC 产能放量，有望从 8 英寸 SiC 衬底 / 外延 / 器件环节获取高毛利业务增量。

## 7. 未来可能的利空

### 利空 1：光伏业务可能继续产生损失 + 资产处置压力

**依据**

多晶硅产线已停产但未完全处置；光伏组件产线产能利用率偏低；2025 年减值 8.70 亿仅是光伏相关资产的第一轮减值。

**解读**

虽然 2025 年已大额减值，但**光伏资产的残值处置、维护费用、设备折旧、在建工程继续减值**都可能在 2026-2027 年继续产生损失。如果全球光伏供需持续过剩到 2027 年，合盛可能需要进一步计提减值或以亏损价格处置光伏资产。同时，光伏业务背后对应的有息负债（贷款 / 债券）仍需偿还，利息成本继续侵蚀利润。

### 利空 2：工业硅 / 有机硅产能过剩延续 + 价格反弹不确定

**依据**

工业硅全行业产能 > 700 万吨 / 年 vs 年需求约 400-450 万吨；有机硅 DMC 全行业产能 > 500 万吨 / 年 vs 年需求约 350-400 万吨。

**解读**

硅基材料行业的过剩问题不是短期能解决——即使 2025-2026 年部分高成本小产能退出，新疆 / 云南 / 四川仍有新增产能陆续投放（合盛自身也在继续扩产）。如果工业硅价格长期维持 7,000-8,000 元 / 吨、有机硅 DMC 维持 11,000-12,000 元 / 吨的低位，**合盛传统业务可能继续微利或小亏**，公司"熬"的时间可能延长至 2-3 年。

### 利空 3：高杠杆 + 大股东高持股套现风险 + 不分红

**依据**

资产负债率 **64.96%（+1.13ppt）**；控股股东 + 一致行动人合计持股 **73.52%**；**2025 年不分红不转增**（母公司累计未分配利润 -35.54 亿）。

**解读**

负债率提升至 64.96%，远高于行业平均水平，若工业硅价格长期低迷，财务费用负担将加重。73.52% 的高持股意味着流通盘只有 26.48%，**大股东减持压力**始终存在（2024 年曾有减持）。**不分红是重大负面**——对依靠股息回报的长期持有者而言是严重打击，且母公司累计未分配利润 -35.54 亿意味着**短期内（2026-2027 年）都难以恢复分红**。相比之下，同行业部分龙头（如东方希望 / 新疆其它有机硅厂）仍保持盈利。

## 8. 总结判断

**一句话结论**

2025 年合盛硅业营收 **204.99 亿（-23.20%）**、**归母 -29.91 亿（上市以来首亏）**、扣非 -31.13 亿；分业务看：有机硅 95.75 亿（-21.52%）、工业硅 90.77 亿（-34.05%）、光伏 8.99 亿（亏损主源）；**各项减值 16.99 亿**（光伏 8.70 亿）；经营现金流 40.13 亿仍为正；资产负债率 64.96%；**因亏损 2025 年不分红不转增**。2026 年减值出清后有望大幅改善。

**核心看点**

1. **经营现金流 40.13 亿为正**（-11.16%）——造血能力保留，没有资金链断裂风险
2. Q3 单季工业硅价格 +8.11% QoQ——触底反弹迹象
3. 新疆 / 云南低电价 + 煤炭一体化——行业底部时"最后倒下"
4. **光伏减值"一次性洗大澡"**（8.70 亿）——2026 年有望大幅改善
5. 有机硅营收规模超过工业硅——业务结构向下游深加工演进
6. 研发聚焦**碳化硅（SiC）+ 电子级硅片**——中长期高端动能
7. 天健会计师事务所审计，标准无保留意见
8. 管理层定调"主业触底回暖、高端产品积蓄新动能"

**核心风险**

1. **归母 -29.91 亿**——上市以来首次年度亏损
2. **各项减值 16.99 亿**（光伏 8.70 亿）——光伏资产后续仍可能继续减值
3. **2025 年不分红不转增**——母公司累计未分配利润 -35.54 亿，2026-2027 年或难恢复分红
4. 工业硅 -27%、有机硅 -12%——行业产能过剩延续
5. 资产负债率 64.96%（+1.13ppt）——利息负担加重
6. 光伏业务"学费"昂贵——战略失误代价仍在消化
7. 控股股东持股 73.52%——流通盘小 + 潜在减持压力
