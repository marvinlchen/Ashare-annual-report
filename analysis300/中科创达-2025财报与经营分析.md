# 中科创达（300496）2025年财报整理

> **说明**：中科创达 2025 年年度报告**尚未披露**（预计 2026 年 4 月下旬披露），**业绩预告/快报亦未发布**。本文基于 **2025 年三季报（2025-10-30 披露）+ 2025 年半年报 + 2024 年年报 + 券商研报** 整理，年报未披露的字段统一标注"年报未披露"。

## 1. 业绩总结

### 1.1 收入与利润表现

中科创达 2025 年前三季度实现营业收入 51.48 亿元，同比 **+39.34%**；归母净利润 2.29 亿元，同比 **+50.72%**；扣非净利润 2.00 亿元，同比 **+57.58%**。Q3 单季营收 18.48 亿（**+42.87%**）、归母净利 7057 万（+48.26%）、扣非 6362 万（**+64.76%**）加速增长。

- Q1-Q3 营业收入 51.48 亿元（同比 +39.34%）
- Q1-Q3 营业利润 年报未披露（全年）；Q3 单季加速
- Q1-Q3 归母净利润 2.29 亿元（同比 +50.72%）

### 1.2 增长与拖累因素

- **主要增长驱动**：①智能物联网 H1 收入 12.70 亿（同比 +136.14%）爆发（AI 机器人 / AI 眼镜 / AI PC / 可穿戴 / 智能视觉）；②滴水 AI OS 1.0 Evo 发布 + AI BOX 与吉利 + NVIDIA 合作 + 7B 车端大模型落地；③海外营收同比 +81.41%（泰国 / 越南 / 中东机器人落地）
- **主要拖累因素**：①Q1 归母同比仅 +2.00% 偏弱（季节性 + 费用投入）；②研发投入和人员扩张压制利润率；③部分传统智能软件业务竞争压力
- **增长主要来自**：智能物联网 + 智能汽车 + 海外业务，AI 终端爆发是核心弹性

### 1.3 趋势判断

- **未来主要利好方向**：滴水 AI OS 成为 AI 汽车操作系统核心标准、AI 终端（机器人 / 眼镜 / PC）爆发、高通 / NVIDIA 深度合作生态壁垒
- **未来主要压力方向**：Q1 归母偏弱、AI 汽车 OS 竞争激烈（华为 / 腾讯 / 特斯拉）、研发投入压制利润率、高通依赖风险
- **整体趋势判断**：中期"智能汽车 + 智能物联网 + 智能软件"三位一体 AI OS 龙头定位清晰，2026 年关键看滴水 OS / AI BOX 规模化量产 + 智能物联网持续高增长

## 2. 主营业务

### 2.1 公司简介

中科创达是中国领先的智能操作系统产品和技术提供商，是全球安卓系统第一大操作系统贡献者 + 高通全球核心战略合作伙伴。

### 2.2 主营业务构成

1. **智能汽车**：滴水 AI OS 1.0 Evo（面向中央计算的 AI 原生整车操作系统）、AI BOX（基于 NVIDIA DRIVE AGX 平台 + 7B 车端大模型）、智能座舱 / 智能驾驶 / 中央计算；客户包括高通 / NVIDIA / 吉利 / 丰田等
2. **智能物联网**：AI 机器人、AI 眼镜、AI PC、可穿戴设备、智能视觉、手持终端；H1 收入 12.70 亿（同比 +136.14%），多家世界 500 强客户落地
3. **智能软件**：Android / Linux 等操作系统内核、智能手机 / 平板 / AIoT 软件；高通核心战略合作伙伴（传统基本盘）

### 2.3 核心收入来源

**智能汽车 + 智能物联网**是双核增长引擎，**智能软件**为传统基本盘。2025 年最核心看点是**智能物联网爆发（+136%）+ 智能汽车 AI 战略落地**。

### 4.3 营收分析

**主要增长来源**

智能物联网业务 H1 收入 12.70 亿（+136.14%）贡献最大弹性，AI 机器人 + AI 眼镜 + AI PC + 可穿戴 + 智能视觉 + 手持终端多产品线齐头并进，多家世界 500 强客户落地。海外业务 +81.41%（泰国 / 越南 / 中东机器人落地）打开国际市场。智能汽车方面，滴水 AI OS 1.0 Evo + AI BOX（吉利 + NVIDIA + 7B 车端大模型）战略落地。

**主要拖累因素**

## 6. 未来可能的利好

### 利好 1：智能物联网业务爆发式增长

**依据**

2025 年 H1 智能物联网收入 12.70 亿（同比 +136.14%），涵盖 AI 机器人 / AI 眼镜 / AI PC / 可穿戴 / 智能视觉 / 手持终端；多家世界 500 强客户落地；海外（泰国 / 越南 / 中东）机器人落地。

**解读**

AI 终端爆发式增长与中科创达作为智能 OS 核心提供商的角色高度契合。多产品线齐头并进 + 海外市场加速将推动第二增长曲线，2026 年有望延续高增长。

### 利好 2：滴水 AI OS + AI BOX + 7B 车端大模型抢占 AI 汽车 OS 制高点

**依据**

2025 年 4 月发布滴水 AI OS 1.0 Evo（面向中央计算的 AI 原生整车操作系统 + 深度融合 AI 大模型技术）；2025 年德国慕尼黑车展与吉利 + NVIDIA 联合发布 AI BOX，首次实现 7B 大模型在车端实时流畅运行。

**解读**

AI 汽车 OS 是新一代智能汽车核心基础设施。中科创达抢占 AI 原生整车 OS 制高点，若 2026-2030 年 SDV（软件定义汽车）全面铺开，公司有望成为全球汽车 OS 核心标准之一，打开从"软件授权"到"OS 平台"的价值跃迁。

### 利好 3：海外业务高增长 + 高通 / NVIDIA / 火山引擎深度合作生态壁垒

**依据**

2025 年 H1 海外营收 +81.41%，机器人在泰国 / 越南 / 中东落地。高通核心战略合作伙伴（2025 高通汽车技术与合作峰会）、NVIDIA DRIVE AGX 平台合作、火山引擎 AI 云合作。

**解读**

海外市场（东南亚 + 中东 + 全球）高增长打开国际市场空间。与全球顶级芯片 / AI 厂商深度合作构建生态壁垒，高通 8295 / 8775 / 8295Pro 等新一代智能汽车 SoC 均与中科创达深度合作，不可替代性强，持续受益 AI 芯片 + 大模型发展。

## 7. 未来可能的利空

### 利空 1：AI 汽车 OS 竞争激烈

**依据**

华为鸿蒙智行、腾讯汽车云、特斯拉 OS、以及主机厂自研 OS（如蔚来 SkyOS、小鹏 XOS、理想星环 OS 等）竞争激烈。

**解读**

AI 汽车 OS 是全球顶级科技公司和车企必争之地。中科创达面临激烈竞争 + 客户流失风险。尤其在中国市场，华为鸿蒙生态和主机厂自研趋势对第三方 OS 供应商形成压力。

### 利空 2：智能物联网高增速的持续性存疑

**依据**

H1 +136.14% 的高增长基于去年较低基数（2024 H1 智能物联网仅约 5.38 亿）。AI 终端市场仍处于早期阶段，爆款产品尚未充分验证。

**解读**

若 AI 终端放量不及预期 + 基数抬高后，2026 年同比增速可能显著放缓。AI 眼镜 / AI PC / 机器人等新终端的渗透率和商业化节奏仍存在不确定性。

### 利空 3：研发投入 + 人员扩张压制利润率 + 高通依赖风险

**依据**

公司大力投入滴水 OS + AI 大模型 + AI 终端研发，人员规模持续扩张。净利率仅 4.45%（Q1-Q3），处于低位。中科创达长期深度绑定高通，高通是其最重要的合作伙伴。

**解读**

持续高研发投入 + 规模效应未完全释放前，利润率改善空间有限。若高通自身业务调整或 SoC 市场竞争格局变化（如联发科、NVIDIA、英伟达车端 SoC 崛起），可能影响中科创达业务稳定性。此外，Q1 归母 +2.00% 已经反映了费用前置压力。

## 8. 总结判断

**一句话结论**

中科创达 2025 Q1-Q3 呈现"营收 +39%（51.48 亿）+ 归母 +51%（2.29 亿）+ 扣非 +58%（2.00 亿）+ Q3 加速 + 智能物联网 H1 +136% + 海外 +81% + 滴水 AI OS / AI BOX 战略落地"的高增长格局；2025 年年报尚未披露，待年报进一步确认全年利润和分部数据。

**核心看点**

1. 智能物联网业务爆发式增长（H1 +136.14%）与 AI 终端时代契合
2. 滴水 AI OS + AI BOX + 7B 车端大模型抢占 AI 汽车 OS 制高点
3. 海外业务 +81% + 高通 / NVIDIA / 火山引擎深度合作生态壁垒

**核心风险**

1. AI 汽车 OS 竞争激烈（华为鸿蒙 / 腾讯 / 特斯拉 / 主机厂自研）
2. 智能物联网高增速持续性存疑（低基数效应 + AI 终端早期阶段）
3. 研发投入压制利润率 + 高通依赖风险
