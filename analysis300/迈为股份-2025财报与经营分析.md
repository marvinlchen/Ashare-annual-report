## 1. 业绩总结

### 1.1 收入与利润表现

迈为股份 2025 年 Q1-Q3 实现营业收入 62.04 亿元，**同比 -20.13%**；归母净利润 6.63 亿元，**同比 -12.56%**；扣非净利润 5.90 亿元，**同比 -13.98%**。Q3 单季营收 19.91 亿（**-31.30%**）、归母净利 2.69 亿（-9.42%）。**Q3 单季毛利率 39.81%（+9.62pct 同比 / +0.86pct 环比）** + 净利率 13.5%（+3.3pct 同比）—— **营收下滑但盈利能力显著改善**反映海外 HJT 整线订单 + 半导体业务高毛利贡献。

- Q1-Q3 营业收入 62.04 亿元（同比 -20.13%）
- Q1-Q3 营业利润 6.79 亿元（半年报 4.12 亿 + Q3 单季推算）
- Q1-Q3 归母净利润 6.63 亿元（同比 -12.56%）

### 1.2 增长与拖累因素

- **主要增长驱动**：①海外 HJT 整线订单（H1 海外收入 **8.2 亿 +143.8%**）；②半导体设备（前道刻蚀 + ALD 量产 + 后道激光开槽**市占第一**）；③Q3 毛利率 **+9.62pct** 大幅提升（高毛利业务结构优化）；④HJT 市占率超 **70%** + NBB 第二代无主栅技术（银浆 -30%）+ HJT 量产功率 760-780W；⑤研发投入 9.51 亿（+24.62%，40% 投向半导体及显示设备）
- **主要拖累因素**：①国内光伏电池片设备销量骤降；②Q3 单季营收 **-31.30%**；③**经营现金流 -10.42 亿（-728.06%）**；④2024 年计提资产减值 5.24 亿 + 2025 Q1 再计提 1.80 亿；⑤HJT 扩产节奏放缓；⑥存货 89.23 亿占总资产 37.43%
- **增长主要来自**：海外 HJT + 半导体新业务结构性贡献；**承压主要来自**：光伏行业深度调整 + 国内 TOPCon / HJT 扩产推迟

### 1.3 趋势判断

- **未来主要利好方向**：海外 HJT 整线订单持续扩张 + 半导体设备（刻蚀 / ALD / 激光开槽）规模化放量 + NBB / 780W 带动 HJT 渗透率提升
- **未来主要压力方向**：国内光伏主业见底节奏 + 现金流修复 + 减值持续风险 + TOPCon 仍为主流
- **整体趋势判断**：短期光伏主业深度调整 + 中长期"HJT + 半导体 + 钙钛矿 + OLED"多元化，2026 年关键看**光伏主业见底 + HJT 海外持续 + 半导体放量 + 现金流修复**

## 2. 主营业务

### 2.1 公司简介

迈为股份是 A 股**光伏设备龙头 + HJT（异质结）核心整线设备供应商**（市占率超 70%），同时布局**半导体设备（刻蚀 / ALD / 激光开槽）+ OLED 显示设备 + 钙钛矿设备**，丝网印刷设备全球第一。

### 2.2 主营业务构成

1. **光伏太阳能电池生产设备**（核心业务）：HJT 整线设备（市占率超 70%）、丝网印刷设备（全球第一）、NBB 无主栅技术、HJT 量产功率 760-780W
2. **半导体设备**：前道刻蚀设备（已量产）、前道 ALD 设备（已量产）、后道激光开槽设备（**市占率行业第一**）
3. **OLED 显示设备**：柔性 OLED、Mini LED、Micro LED 相关设备
4. **钙钛矿电池装备**：钙钛矿叠层电池整线设备布局
5. **海外 HJT 整线业务**：东南亚、印度、中东等海外光伏新兴市场

### 2.3 核心收入来源

**HJT + 丝网印刷 + 光伏设备**（约 60-70%，受光伏周期影响下降）+ **海外 HJT 整线**（快速增长，H1 海外占比大幅提升）+ **半导体设备**（新增长点，占比提升）+ OLED / 钙钛矿等新兴业务。客户覆盖通威、隆基、华晟、东方日升等光伏龙头及全球半导体客户。

## 6. 未来可能的利好

### 利好 1：HJT 市占率超 70% + 异质结技术龙头

**依据**

异质结设备市占率超过 **70%**，将双面微晶异质结高效电池整线装备的单线年产能升级至 **GW 级**；开发第二代无主栅技术 NBB，可将银浆耗量减少 **30% 以上**；2025 年底 HJT 量产功率目标突破 **760-780W**。

**解读**

HJT 是光伏下一代主流技术之一（25-30% 转换效率 + 更高发电量）；公司市占率超 70%、全球龙头地位 + 持续技术迭代（NBB + 双面微晶 + GW 级整线 + 780W 功率）构筑深厚技术壁垒。若 HJT 扩产周期重启，公司将直接受益。

### 利好 2：海外业务 H1 +143.8% + 海外 HJT 整线高毛利

**依据**

上半年海外收入 **8.2 亿元**，同比 **+143.8%**。

**解读**

海外 HJT 整线订单爆发（东南亚 / 印度 / 中东等新兴市场）+ 海外毛利率远高于国内 + 有效抵消国内光伏下行压力。**2026-2028 年海外有望成为最大增长点和盈利引擎**。

### 利好 3：Q3 单季毛利率 +9.62pct + 盈利能力大幅改善

**依据**

Q3 单季毛利率 **39.81%（+9.62pct 同比 / +0.86pct 环比）**；Q3 单季净利率 **13.5%（+3.3pct 同比 / +1.8pct 环比）**。

**解读**

Q3 毛利率 / 净利率双升反映海外 HJT + 半导体高毛利业务占比提升 + 产品结构优化。即使营收下滑，利润降幅远小于营收，盈利质量大幅改善，**业绩弹性大**。2026 年若延续趋势，利润有望持续修复。

### 利好 4：半导体设备多点开花 + 后道激光开槽市占第一

**依据**

半导体设备进展：前道刻蚀设备和 ALD 设备已实现多批次客户交付**进入量产**；后道激光开槽设备**市占率行业第一**。

**解读**

半导体设备是公司长期战略重点。在晶圆厂扩产 + 国产化浪潮下，半导体设备业务将成为**第二成长曲线**，2026-2028 年有望规模化贡献，为公司打开新的估值锚点。

### 利好 5：研发投入 9.51 亿 + 40% 投向半导体及显示设备

**依据**

2024 年研发支出 **9.51 亿元**（+24.62%），其中 40% 投向半导体及显示设备领域。

**解读**

大力度研发投入 + 40% 投向半导体 / 显示新赛道，为未来多元化成长打下坚实基础。技术护城河 + 新业务拓展同步推进。

## 7. 未来可能的利空

### 利空 1：Q3 单季营收 -31.30% + 光伏主业承压

**依据**

Q3 单季营收 19.91 亿（**-31.30%**），光伏行业产能过剩 + 电池片扩产节奏放缓 + HJT 扩产推迟。

**解读**

Q3 单季营收大幅下滑反映光伏设备订单显著不足。**2026 年若国内光伏主业不复苏，业绩压力持续**。半导体 / 海外新业务规模尚小，难以完全对冲国内光伏主业下行。

### 利空 2：经营现金流 -10.42 亿 +（-728.06%）

**依据**

Q1-Q3 经营性现金流 **-10.42 亿元**（同比 -728.06%）。

**解读**

经营现金流严重恶化，订单回款节奏 + 新签订单不足双重冲击。若 2026 年现金流持续负值，需关注资金安全 + 融资成本压力，也可能影响产能投入 / 研发投入节奏。

### 利空 3：资产减值持续 + 2024 年 5.24 亿 + 2025 Q1 再 1.80 亿

**依据**

**2024 年计提资产减值准备 5.24 亿**（应收账款坏账 3.88 亿 + 存货跌价 1.36 亿）+ **2025 Q1 再计提 1.80 亿**。2024 年存货 **89.23 亿**（占总资产 **37.43%**）。

**解读**

减值持续大幅计提反映光伏行业需求骤降 + 客户扩产缓滞 + 订单延迟 / 取消。应收账款风险 + 存货跌价压力持续。若光伏行业持续低迷，**2026 年可能继续计提，压制利润**。

### 利空 4：HJT 扩产节奏放缓 + TOPCon 仍为主流

**依据**

HJT 扩产低于预期，TOPCon 目前仍为电池片主流。

**解读**

TOPCon 产能过剩已影响整个行业 + HJT 扩产放缓。若 HJT 成本 / 效率优势未能充分兑现，或 TOPCon 持续降价，HJT 渗透率提升缓慢，**直接影响公司 HJT 设备订单**。

### 利空 5：半导体设备竞争加剧 + 客户导入周期长

**依据**

半导体设备行业竞争激烈（北方华创、中微、盛美、至纯、捷佳伟创等）。

**解读**

半导体设备行业壁垒高 + 客户导入周期长。公司虽然前道刻蚀 / ALD + 后道激光开槽市占第一，但**大规模放量仍需时间**，快速放量面临挑战。

## 8. 总结判断

**一句话结论**

迈为股份 2025 Q1-Q3 呈现"营收 -20.13%（62.04 亿）+ 归母 -12.56%（6.63 亿）+ Q3 单季毛利率 39.81%（+9.62pct）+ 海外 +143.8% + 半导体多元突破 + 光伏主业承压 + 经营现金流 -10.42 亿 + 减值持续"的"**营收端深度调整 + 毛利率大幅改善 + 海外 + 半导体多元突破 + 现金流恶化**"格局；2025 年年报尚未披露，待年报进一步确认全年利润和分部数据。

**核心看点**

1. HJT 市占率超 70% + 异质结技术龙头（NBB + 780W + GW 级整线）
2. 海外业务 H1 +143.8% + 半导体设备（刻蚀 / ALD / 激光开槽市占第一）双轮驱动
3. Q3 单季毛利率 +9.62pct + 盈利能力大幅改善（高毛利结构性变化）

**核心风险**

1. Q3 单季营收 -31.30% + 光伏主业承压（产能过剩 + 扩产放缓）
2. 经营现金流 -10.42 亿（-728.06%）（订单回款 + 新签不足双重冲击）
3. 资产减值持续（2024 年 5.24 亿 + 2025 Q1 再 1.80 亿）+ HJT 扩产放缓 + TOPCon 仍为主流
